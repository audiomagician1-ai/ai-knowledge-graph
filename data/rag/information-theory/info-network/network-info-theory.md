# 网络信息论基础

## 概述

网络信息论（Network Information Theory）是信息论在多用户、多节点通信系统中的延伸与深化，由 Cover 与 Thomas 在其经典教材《Elements of Information Theory》（1991, 2006）中系统整理，并由 El Gamal 与 Kim 在专著《Network Information Theory》（Cambridge University Press, 2011）中给出迄今最完整的数学体系。与香农（Shannon, 1948）单用户点对点信道容量理论不同，网络信息论研究的核心问题是：**当多个发送者与接收者共享同一物理介质时，整个系统的信息传输能力的极限是什么？**

这一问题之所以远比点对点情形困难，在于网络中存在干扰、协作、广播、中继等多种相互耦合的信息流，使得"容量区域"（Capacity Region）从单一实数扩展为高维凸集，其精确刻画至今仍有大量开放问题。1972年，Ahlswede 和 Liao 各自独立给出多址接入信道（MAC）的容量区域；1979年，Cover 提出广播信道的叠加编码方案；1979年，Cover 与 El Gamal 给出中继信道的编译码定理——这三个里程碑奠定了网络信息论的基石。

## 核心原理

### 多址接入信道（Multiple Access Channel, MAC）

多址接入信道是最简单的多用户模型：$K$ 个独立发送者各自编码，共同发送至单一接收者。以双用户为例，发送者1以速率 $R_1$ 发送，发送者2以速率 $R_2$ 发送，信道输出 $Y$ 的条件分布为 $p(y|x_1, x_2)$。

Ahlswede（1971）与 Liao（1972）证明，双用户离散无记忆 MAC 的容量区域 $\mathcal{C}_{\text{MAC}}$ 是满足以下不等式组的所有速率对 $(R_1, R_2)$ 的凸包：

$$R_1 \leq I(X_1; Y | X_2)$$
$$R_2 \leq I(X_2; Y | X_1)$$
$$R_1 + R_2 \leq I(X_1, X_2; Y)$$

其中联合分布满足 $p(x_1)p(x_2)p(y|x_1,x_2)$，即两发送者输入独立。第三个约束 $R_1 + R_2 \leq I(X_1, X_2; Y)$ 称为**求和速率约束**（Sum-Rate Constraint），刻画了整个信道的总吞吐量上界。

**例如**，考虑高斯 MAC：$Y = X_1 + X_2 + Z$，其中 $Z \sim \mathcal{N}(0, N)$，发送者功率约束分别为 $P_1, P_2$。则容量区域的顶角点（corner points）对应于连续干扰消除（Successive Interference Cancellation, SIC）解码策略，求和容量为：

$$C_{\text{sum}} = \frac{1}{2}\log\left(1 + \frac{P_1 + P_2}{N}\right)$$

这说明两个用户联合传输的总速率，等同于将两者功率合并后的单用户信道容量。SIC 解码先解码一个用户并从接收信号中减去其贡献，再解码另一用户，从而逼近区域的两个顶角点。

### 广播信道（Broadcast Channel, BC）

广播信道是 MAC 的对偶结构：单一发送者向 $K$ 个独立接收者发送不同的消息。Cover（1972）首先研究退化广播信道（Degraded BC），其中接收者的信道质量构成马尔可夫链 $X \to Y_1 \to Y_2$（接收者1信道更强）。

对于退化广播信道，叠加编码（Superposition Coding）达到容量区域：发送者用码字 $X^n(m_0)$ 携带公共消息 $m_0$（面向弱接收者），在其上叠加 $X^n(m_1|m_0)$ 携带私有消息 $m_1$（面向强接收者）。强接收者先解码私有消息再解码公共消息（类似 SIC），弱接收者将叠加信号视为噪声只解码公共消息。高斯退化广播信道的容量区域由以下参数化曲线给出（功率分配比 $\alpha \in [0,1]$）：

$$R_1 = \frac{1}{2}\log\left(1 + \frac{\alpha P}{\bar{\alpha} P + N_1}\right), \quad R_2 = \frac{1}{2}\log\left(1 + \frac{\bar{\alpha} P}{N_2}\right)$$

其中 $\bar{\alpha} = 1 - \alpha$，$N_1 < N_2$ 分别为两接收者的噪声功率，$P$ 为发送功率约束。一般（非退化）广播信道的容量区域至今仍是信息论中最著名的开放问题之一。

### 中继信道（Relay Channel）

中继信道由 van der Meulen（1971）首次形式化，包含发送者、中继节点和接收者三方。Cover 与 El Gamal（1979）在论文《Capacity Theorems for the Relay Channel》（IEEE Trans. Inf. Theory）中给出两个核心编码策略及其对应的可达速率下界：

**（1）解码转发（Decode-and-Forward, DF）**：中继完整解码发送者的消息，再重新编码转发。可达速率为：

$$R_{\text{DF}} = \max_{p(x_1, x_2)} \min\left\{ I(X_1, X_2; Y), \, I(X_1; Y_r | X_2) \right\}$$

其中 $X_1$ 为发送者输入，$X_2$ 为中继输入，$Y_r$ 为中继接收，$Y$ 为目的节点接收。DF 适用于中继距发送者较近（中继-源链路质量好）的场景。

**（2）压缩转发（Compress-and-Forward, CF）**：中继不解码消息，而是将其观测 $Y_r$ 量化压缩后转发给目的节点，目的节点联合两路信号进行解码。可达速率为：

$$R_{\text{CF}} = \max \, I(X_1; \hat{Y}_r, Y | X_2)$$

受约束于 $I(Y_r; \hat{Y}_r | Y, X_2) \leq I(X_2; Y)$，其中 $\hat{Y}_r$ 为中继对 $Y_r$ 的量化版本。CF 适用于中继距目的节点较近的场景。

中继信道的精确容量至今尚未完全确定，这一事实深刻揭示了网络信息论问题的内在困难性。

## 关键公式与模型

### 容量区域的凸性与时间共享

网络信息论中容量区域 $\mathcal{C}$ 总是凸集。原因在于**时间共享（Time Sharing）**原理：若 $(R_1^{(1)}, R_2^{(1)}) \in \mathcal{C}$ 且 $(R_1^{(2)}, R_2^{(2)}) \in \mathcal{C}$，则对任意 $\lambda \in [0,1]$，速率对 $(\lambda R_1^{(1)} + (1-\lambda) R_1^{(2)}, \lambda R_2^{(1)} + (1-\lambda) R_2^{(2)}) \in \mathcal{C}$。这一性质来源于用 $\lambda n$ 个信道使用时间传输第一个速率点、用 $(1-\lambda)n$ 个时间传输第二个速率点的简单策略。

### 干扰信道的汉斯-科斯-扬特曼（HK）内界

干扰信道（Interference Channel, IC）是网络信息论中难度最高的基本模型之一：两对发送-接收节点共享信道，各自产生干扰。Han 与 Kobayashi（1981）提出了著名的 HK 内界，将每个用户的消息分为公共部分（对方也解码）和私有部分（仅本方解码），其求和速率内界在弱干扰（$|a_{12}|^2 \leq 1$ 且 $|a_{21}|^2 \leq 1$）下渐近最优，且与外界的差距不超过 **1 bit**（Etkin, Tse, Wang, 2008）。

### 多跳网络的切割集界

对于一般网络，Cover 与 Thomas 给出**切割集界**（Cut-Set Bound）作为通用外界：对网络中任意将节点集 $\mathcal{V}$ 分成包含发送者的集合 $S$ 和包含接收者的集合 $S^c$ 的切割，总速率满足：

$$R \leq \max_{p(\mathbf{x}_S|\mathbf{x}_{S^c})} I(\mathbf{X}_S; \mathbf{Y}_{S^c} | \mathbf{X}_{S^c})$$

切割集界为网络容量提供了一个简洁但通常不紧的上界，其紧性条件至今仍是活跃研究方向。

## 实际应用

网络信息论的理论直接驱动了现代无线通信标准的设计：

- **LTE/5G 上行链路**：多用户 MIMO（MU-MIMO）的设计原理正是 MAC 容量区域理论的工程实现。基站配备大规模天线阵列，通过 SIC 或联合 MMSE 检测逼近 MAC 容量区域，3GPP 标准中的上行 SIC 接收机直接源于 Liao-Ahlswede 定理的启发。

- **协作通信与中继网络**：LTE-Advanced 引入了协作多点传输（CoMP）和中继站（Relay Node, RN），其系统设计中的 DF 中继方案直接对应 Cover-El Gamal DF 定理。

- **非正交多址（NOMA）**：5G NR 研究阶段深入探讨了 NOMA，其核心思想——允许多用户在同一时频资源上叠加发送，接收端用 SIC 解码——正是高斯 MAC 容量区域边界点的直接工程实现。

- **认知无线电**：干扰信道模型为主用户-次用户共存的频谱共享提供了理论基础，HK 编码策略指导了认知无线电中干扰管理方案的设计。

## 常见误区

**误区1：MAC 容量区域就是两个单用户容量的笛卡尔积。**
实际上，MAC 容量区域受到求和速率约束 $R_1 + R_2 \leq I(X_1,X_2;Y)$ 的限制，是一个五边形（对高斯信道而言），而非矩形。两个用户不能同时都达到各自的单用户最大速率，存在本质的资源竞争。

**误区2：中继信道的容量已由 DF 或 CF 达到。**
DF 和 CF 均为可达性（下界）结果，而非精确容量。Cover-El Gamal（1979）的定理给出的是两种策略的速率下界，中继信道的精确容量（除特殊退化情形外）至今未知。混淆可达速率与容量是学习网络信息论最常见的错误。

**误区3：广播信道叠加编码对所有信道均最优。**
叠加编码对**退化**广播信道（以及更广泛的"退化类"信道）达到容量区域，但对一般广播信道（如非退化 MIMO 广播信道），需要"脏纸编码"（Dirty Paper Coding, DPC；Costa, 1983）等更复杂方案。

**误区4：网络信息论问题仅关注渐近（$n\to\infty$）性能。**
有限码长（Finite Blocklength）网络信息论是近年兴起的前沿方向，Polyanskiy、Poor 与 Verdú（2010）在点对点信道建立的二阶速率理论正被推广至 MAC 和广播信道场景。

## 知识关联

**与香农信道容量的关系**：香农（1948）的单用户信道容量定理 $C = \max_{p(x)} I(X;Y)$ 是网络信息论在 $K=1$ 时的特例。MAC、BC 和中继信道在退化或特殊对称条件下均可化归为单用户问题。

**与率失真理论的关联**：中继信道中的 CF 策略本质上是率失真理论（Wyner-Ziv 编码，1976）的应用——中继将观测量化至满足功率约束的精度，目的节点利用自身接收信号作为边信息进行解码，体现了信源-信道编码联合设计的思想。
