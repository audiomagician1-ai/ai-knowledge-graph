# 信息论与密码学

## 概述

信息论与密码学的深层联系由克劳德·香农（Claude Shannon）在1949年发表的奠基性论文《保密系统的通信理论》（*Communication Theory of Secrecy Systems*，Bell System Technical Journal, Vol. 28）中首次系统阐明。这篇论文与香农1948年的信息论奠基论文《通信的数学理论》共同构成了现代通信科学的两大基石。在这篇1949年的论文中，香农将密码学从此前依赖直觉和经验的艺术，转变为可以用严格数学语言表达的科学，并引入了**完美保密（Perfect Secrecy）**的概念，从根本上回答了"什么样的加密系统是绝对安全的"这一核心问题。

香农的核心洞见在于：密码学的安全性本质上是一个关于**信息不确定性**的问题。密文能否泄露明文的信息，可以用条件熵来精确量化。这一视角将密码学分析从"攻击者能否破解"的计算复杂性问题，转化为"密文本身携带了多少关于明文的信息量"的信息论问题。

## 核心原理

### 完美保密的数学定义

香农将完美保密（Perfect Secrecy）定义为：加密系统对任意明文消息 $m$ 和密文 $c$，满足：

$$P(M = m \mid C = c) = P(M = m)$$

即观察到密文 $c$ 后，攻击者对明文 $m$ 的后验概率分布与先验概率分布完全相同。密文的出现丝毫不改变攻击者对明文的知识状态。换言之，密文与明文在统计意义上完全独立：

$$I(M; C) = 0$$

其中 $I(M; C)$ 为明文随机变量 $M$ 与密文随机变量 $C$ 之间的**互信息**。互信息为零意味着密文不包含任何关于明文的信息，这是可以用信息论给出的最强安全保证。

用条件熵语言表达同样的概念：完美保密等价于

$$H(M \mid C) = H(M)$$

即在已知密文 $c$ 的条件下，明文的不确定性与不知道密文时完全一样。攻击者无论截获多少密文，对明文的猜测能力没有任何提升。

### 完美保密的必要条件：密钥空间下界

香农在同一篇论文中证明了一个关键定理：**实现完美保密的充要条件是密钥空间的大小不小于明文空间的大小**。形式化地：

$$|K| \geq |M|$$

更精确地说，对于完美保密系统，密钥的熵必须不小于明文的熵：

$$H(K) \geq H(M)$$

这个不等式有深刻的实践含义：**密钥不能重复使用**。若用同一密钥加密两条明文 $m_1$ 和 $m_2$，得到密文 $c_1 = m_1 \oplus k$ 和 $c_2 = m_2 \oplus k$，则攻击者可以计算 $c_1 \oplus c_2 = m_1 \oplus m_2$，直接暴露两段明文的异或关系，严重泄露信息。二战期间苏联情报机构因重复使用一次性密钥（即所谓"维诺纳项目"VENONA所破获的材料）导致多名间谍暴露，正是这一数学原理的历史教训。

### 一次一密（One-Time Pad）与完美保密的实现

一次一密（One-Time Pad，OTP）是**唯一已被证明达到完美保密的实际加密方案**，由吉尔伯特·弗纳姆（Gilbert Vernam）于1917年发明，并于1919年获得美国专利（US Patent 1,310,719）。其加密操作为：

$$C = M \oplus K$$

其中 $M$ 为明文比特串，$K$ 为与 $M$ 等长的随机密钥比特串，$\oplus$ 为逐比特异或运算。解密操作对称地为 $M = C \oplus K$。

当且仅当满足以下三个条件时，OTP实现完美保密：
1. **密钥完全随机**：$K$ 的每一比特均匀独立地从 $\{0, 1\}$ 中选取，即 $H(K) = |K|$（比特）；
2. **密钥长度不小于明文长度**：$|K| \geq |M|$；
3. **密钥绝对不重复使用**：每次加密使用全新密钥。

验证其完美保密性：对任意固定明文 $m$ 和密文 $c$，存在唯一的密钥 $k = m \oplus c$ 使得加密成立；由于 $K$ 均匀分布，$P(C = c \mid M = m) = P(K = m \oplus c) = 2^{-n}$（对 $n$ 比特消息），这与 $m$ 的选取无关，因此条件分布 $P(M \mid C = c)$ 与先验分布 $P(M)$ 相同，完美保密成立。

## 关键公式与模型

### 不确定性分解与密钥等义度

香农还引入了**等义度（Equivocation）**的概念，定义为已知密文后密钥的条件熵 $H(K \mid C)$，用于衡量密码系统在给定一定量密文后剩余的密钥不确定性。对于长度为 $n$ 的密文序列，随着 $n$ 增大，等义度趋向于零意味着系统可被唯一破解（unicity point）。

香农定义**唯一解距离（Unicity Distance）** $U$ 为：

$$U \approx \frac{H(K)}{D}$$

其中 $H(K)$ 为密钥熵（比特），$D$ 为语言冗余度（redundancy），定义为 $D = \log_2 |\Sigma| - H_\infty$，其中 $|\Sigma|$ 为字母表大小，$H_\infty$ 为该语言的极限熵（比特/符号）。

例如，英语的极限熵约为 $H_\infty \approx 1.0 \sim 1.5$ 比特/字母（Shannon在1951年的论文《预测与英语熵》中通过人类被试实验估算），而 $\log_2 26 \approx 4.7$ 比特/字母，故英语冗余度 $D \approx 3.2$ 比特/字母。一个26个字母的简单替换密码密钥空间为 $26! \approx 2^{88}$，故 $H(K) \approx 88$ 比特，唯一解距离约 $U \approx 88 / 3.2 \approx 27$ 个字母——即理论上只需约27个英文字母的密文，就足以唯一确定替换密钥。这解释了为何简单替换密码在密文量充足时总是可破的。

### 信息论安全 vs 计算安全

信息论安全（Information-Theoretic Security）是比计算安全（Computational Security）更强的安全概念。计算安全假设攻击者的计算能力有限（如RSA、AES均属此类），一旦量子计算机成熟，这类假设可能失效。而信息论安全对攻击者的计算能力**无任何限制**，即使攻击者拥有无限算力也无法破解。

完美保密是信息论安全的最强形式，但其代价是密钥管理极其困难（密钥与明文等长且不可复用）。更宽松的信息论安全概念包括**统计安全（Statistical Security）**，允许攻击者以指数级小的概率 $\epsilon$ 获得额外信息，即要求：

$$|P(M = m \mid C = c) - P(M = m)| \leq \epsilon$$

对所有 $m, c$ 成立，其中 $\epsilon$ 是一个可接受的极小误差参数。

## 实际应用

**案例一：量子密钥分发（QKD）中的信息论安全**

量子密钥分发协议（如1984年Bennett和Brassard提出的BB84协议）利用量子力学的测量坍缩原理，使得任何窃听行为都必然在信道中留下可检测的扰动。其安全性的严格证明依赖于信息论中的保密容量（Secrecy Capacity）概念。QKD的目标是生成一个满足OTP条件的真随机密钥，从而实现端到端的完美保密通信。中国"墨子号"量子卫星（2016年发射）在2020年实现了北京与维也纳之间7600公里的量子加密视频通话，其底层安全性正是建立在香农完美保密定理之上。

**案例二：秘密共享方案（Secret Sharing）**

阿迪·沙米尔（Adi Shamir）于1979年提出的 $(t, n)$-门限秘密共享方案（Shamir's Secret Sharing）具有信息论安全性：任意 $t-1$ 个参与者持有的份额关于秘密的互信息为零，即 $I(S; \text{任意}\ t-1\ \text{份份额}) = 0$，而任意 $t$ 个参与者合作可完全恢复秘密。这是香农信息论直接催生的密码学构造之一。

**案例三：物理层安全（Physical Layer Security）**

无线通信中，Wyner（1975年）提出的**窃听信道模型（Wiretap Channel）**利用信道噪声差异实现信息论安全传输。定义保密容量（Secrecy Capacity）为：

$$C_s = \max[I(X; Y) - I(X; Z), 0]$$

其中 $X$ 为发送信号，$Y$ 为合法接收者接收信号，$Z$ 为窃听者接收信号。当合法信道质量优于窃听信道时，$C_s > 0$，意味着可以在信息论意义上安全地传输正速率的信息，而窃听者获得的互信息趋于零。

## 常见误区

**误区一：OTP不实用所以香农定理没有价值**

OTP由于密钥管理困难（密钥需安全预分发且等长于明文）在大多数场景下不实用，但这并不意味着香农的完美保密理论没有价值。该理论明确划定了信息论安全的**理论上界**，为所有密码系统提供了评价标准——任何密钥比明文短的系统都不能实现完美保密，这是设计密码系统时的根本约束。

**误区二："信息论安全"等同于"绝对无法破解"**

信息论安全的前提条件非常苛刻。OTP的完美保密依赖于真随机密钥——伪随机数生成器（PRNG）生成的密钥不满足此要求，因为PRNG的输出熵远小于其比特长度，$H(K) \ll |K|$。历史上多次"OTP"的安全失败（如VENONA案例）均源于密钥的非真随机性或重复使用，而非香农定理本身的缺陷。

**误区三：高熵密钥保证完美保密**

密钥熵高是必要条件，但非充分条件。加密函数的设计同样关键。例如，若加密函数 $E_k(m) = m$（忽略密钥），则无论密钥熵多高，密文直接暴露明文，完全不安全。完美保密要求加密函数、密钥分布、明文空间三者共同满足香农定理的条件。

## 知识关联

**与香农熵的关系**：完美保密的定义直接用互信息 $I(M; C) = H(M) - H(M \mid C)$ 和条件熵 $H(M \mid C)$ 表达，是香农熵理论的直接应用。没有对信息量的精确定义，完美保密就无法被数学化。

**与信道容量的关系**：保密容量（Secrecy Capacity）是信道容量（Channel Capacity）概念在窃听信道上的推广，将物理信道的噪声特性纳入密码学分析，形成了现代物理层安全理论的基础。

**与计算复杂性理论的关系**：信息论安全与计算安全构成密码学安全性的两个层次。现代公钥密码学（RSA、椭圆曲线等）属于计算安全，其安全性依赖于整数分解等问题的计算困难性假设，而非信息论意义上的不可能性。两者的区分对于后量子密码学设计至关重要。
