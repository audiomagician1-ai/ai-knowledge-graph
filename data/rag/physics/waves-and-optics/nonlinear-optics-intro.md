# 非线性光学简介

## 概述

非线性光学是研究强光场与介质相互作用时，介质极化强度对光场呈非线性响应的物理分支。在普通光强条件下，介质极化强度 **P** 与电场强度 **E** 成正比：P = ε₀χ⁽¹⁾E，其中 χ⁽¹⁾ 为线性电极化率。而当光场强度接近或超过原子内部电场（约 10⁸ V/m 量级）时，必须引入高阶项：P = ε₀(χ⁽¹⁾E + χ⁽²⁾E² + χ⁽³⁾E³ + …)，其中 χ⁽²⁾、χ⁽³⁾ 分别是二阶、三阶非线性极化率，正是这些高阶项催生了一系列独特的光学现象。

非线性光学的实验基础建立于1961年，即激光器诞生后仅一年。Peter Franken 等人将红宝石激光（694.3 nm）聚焦到石英晶体，首次观测到二次谐波（347.15 nm），这一实验标志着非线性光学作为独立学科的诞生。此后，Bloembergen 等人在1962年建立了系统的非线性光学理论，并因此获得1981年诺贝尔物理学奖。

非线性光学效应的实用价值在于它能将激光器无法直接输出的波长转换成目标波长，同时也是超短脉冲压缩、量子光学实验、全光信号处理的物理基础。没有非线性光学，现代工业激光系统中频繁使用的绿色激光（532 nm，由1064 nm Nd:YAG激光倍频所得）将无从实现。

## 核心原理

### 二次谐波产生（倍频）

二次谐波产生（Second Harmonic Generation，SHG）源于二阶非线性极化率 χ⁽²⁾。当角频率为 ω 的基频光入射时，二阶极化项 P⁽²⁾ = ε₀χ⁽²⁾E²cos²(ωt) = ½ε₀χ⁽²⁾E²[1 + cos(2ωt)]，其中振荡项 cos(2ωt) 辐射出频率为 2ω（波长减半）的倍频光。值得注意的是，χ⁽²⁾ 在具有中心对称性的介质（如液体、玻璃）中严格为零，因此 SHG 仅发生于非中心对称晶体，如磷酸二氢钾（KDP）、铌酸锂（LiNbO₃）、BBO（β-偏硼酸钡）等。

高效 SHG 的关键在于**相位匹配条件**：Δk = k(2ω) − 2k(ω) = 0，即 2ω 的波矢等于两倍基频波矢。由于正常色散导致折射率随频率增大，Δk 通常不为零，实验上常用双折射相位匹配：利用晶体的寻常光（o光）与非寻常光（e光）折射率随传播方向变化的特性，选取特定切割角使 n_e(2ω) = n_o(ω)，从而满足 Δk = 0，转换效率可从接近 0 提升至 80% 以上。

### 自聚焦效应

自聚焦来源于三阶非线性极化率 χ⁽³⁾ 引起的光克尔效应：介质折射率随光强变化，n = n₀ + n₂I，其中 n₀ 为线性折射率，n₂ = 3χ⁽³⁾/(4ε₀cn₀²) 为非线性折射率系数，I 为光强。由于高斯光束中心强度大于边缘，中心区域折射率更高，等效于一个正透镜，使光束向轴线汇聚——这就是自聚焦。

当光功率超过**临界功率** P_cr = λ²/(2πn₀n₂) 时（对常见玻璃约为数百千瓦量级），自聚焦效应将超过衍射发散，光束发生灾难性坍缩，导致介质损伤或产生超连续谱。飞秒激光在大气中传输时，自聚焦与等离子体散焦形成动态平衡，可以产生跨越数百米的"光丝"，这一现象在遥感大气成分中有重要应用。

### 光学参量效应

光学参量效应同样基于 χ⁽²⁾，但方向与 SHG 相反：一个高频泵浦光子（频率 ω_p）分裂为两个低频光子，满足能量守恒 ω_p = ω_s + ω_i，其中 ω_s（信号光）和 ω_i（闲频光）的频率由相位匹配条件决定，而非固定值。通过改变晶体温度或旋转角度调节相位匹配，可在宽范围内连续调谐输出波长。基于此原理构建的**光学参量振荡器（OPO）**能将固定波长激光转换为从近紫外到中红外的可调谐输出，覆盖许多气体分子的特征吸收峰，在分子光谱学中具有不可替代的作用。

## 实际应用

**倍频绿激光指示器**：消费级绿色激光笔的核心是将808 nm半导体泵浦激光激发Nd:YAG晶体产生1064 nm激光，再经KTP晶体（KTiOPO₄）相位匹配倍频，输出532 nm绿光，整个链路将电能→近红外→绿光，转换效率通常在10%–30%。

**超短脉冲测量**：自相关仪利用 SHG 测量飞秒脉冲宽度：将脉冲分为两束，改变延迟后在 BBO 晶体中产生倍频，通过扫描延迟测量倍频信号强度，重建脉冲时间轮廓，可分辨短至10飞秒（10⁻¹⁴ s）的脉冲。

**光纤通信中的四波混频**：单模光纤中的三阶非线性（χ⁽³⁾）导致四波混频（FWM），2ω₁ − ω₂ 处产生新频率成分，这是密集波分复用（DWDM）系统中串扰噪声的重要来源，设计时需通过引入色散管理抑制相位匹配以减小 FWM。

## 常见误区

**误区一：非线性光学效应只需要高光强，普通聚焦激光足够**。实际上，仅有高峰值强度（>10⁸ W/cm²量级）是必要条件之一，相位匹配才是决定转换效率的瓶颈。未满足相位匹配的倍频晶体中，倍频光功率随传播距离震荡而非单调增长，特征长度为相干长度 L_c = π/|Δk|，石英中典型值约为 20 μm，在此尺度内转换效率极低。

**误区二：χ⁽²⁾ 效应在所有非中心对称材料中同等显著**。χ⁽²⁾ 的量级在不同材料中差异超过三个数量级：气体中约为 10⁻²⁴ m/V，而优质非线性晶体如 GaAs 中可达 10⁻²¹ m/V 量级，BBO 的有效非线性系数 d_eff ≈ 2 pm/V，LiNbO₃ 的 d₃₃ ≈ 27 pm/V。选材不当将使效率相差数百倍。

**误区三：自聚焦总是有害的**。在超短脉冲激光加工与光丝应用中，自聚焦形成的细丝可以在空气中稳定传输数百米而不扩散，并同步产生覆盖从紫外到红外的超连续白光，这在激光雷达大气探测中被积极利用。自聚焦是否"有害"完全取决于峰值功率是否精确控制在 P_cr 附近的稳定窗口内。

## 知识关联

非线性光学以**激光**为直接前提：连续激光器的聚焦光强和脉冲激光的峰值功率（飞秒激光可达 10¹² W/cm²）是驱动非线性响应的能量来源，没有激光提供的相干高强光场，χ⁽²⁾ 和 χ⁽³⁾ 项在实验室条件下几乎无法被探测。量子光学角度看，光学参量下转换（SPDC）是 OPO 工作于单光子量级时的极限，能产生纠缠光子对，是量子密钥分发（QKD）和量子纠缠实验的核心光源。在凝聚态物理中，非线性极化率 χ⁽²⁾ 的对称性选择定则使 SHG 成为探测表面/界面结构、铁电畴翻转和手性分子的高灵敏探针，连接了光学与材料科学的前沿研究。
