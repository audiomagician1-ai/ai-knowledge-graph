# 可压缩流简介

## 概述

可压缩流是指流体密度随压力和温度变化而显著改变的流动状态。与不可压缩流（如低速水流或低速气流）不同，可压缩流中密度 ρ 不再是常数，必须作为独立变量参与所有方程的求解。气体在流速超过当地声速的约 30%（马赫数 Ma > 0.3）时，密度变化超过 5%，此时忽略可压缩性将带来不可接受的误差。

可压缩流理论的奠基工作可追溯到19世纪：1848年英国物理学家斯托克斯（G.G. Stokes）分析了声波传播方程；1887年奥地利物理学家恩斯特·马赫拍摄了第一张超声速子弹冲击波的阴影照片，揭示了激波结构；1908年普朗特与迈耶（Prandtl-Meyer）建立了超声速膨胀波的精确理论。二战期间喷气式飞机突破"音障"的需求推动了该领域的爆炸式发展。

理解可压缩流对航空航天工程至关重要。现代商用飞机巡航马赫数约为 0.85，超声速战斗机工作在 Ma > 1.5，火箭喷管出口马赫数可达 4～6。错误地将不可压缩伯努利方程套用于这些场合，会导致压力和推力计算严重偏差，因此需要专门的可压缩流动力学框架。

## 核心原理

### 马赫数与声速

马赫数定义为当地流速 V 与当地声速 a 之比：

$$Ma = \frac{V}{a}$$

理想气体中声速公式为：

$$a = \sqrt{\gamma R T}$$

其中 γ 为比热比（空气的 γ = 1.4），R 为气体常数（空气 R = 287 J/(kg·K)），T 为绝对温度（开尔文）。在海平面标准大气（T = 288.15 K）下，空气声速约为 340 m/s。值得注意的是，声速随温度升高而增大，与密度或压力无直接关系。

流动按马赫数分区：亚声速（Ma < 0.8）、跨声速（0.8 ≤ Ma ≤ 1.2）、超声速（1.2 < Ma < 5）和高超声速（Ma ≥ 5）。跨声速区内流场中同时存在亚声速和超声速区域，激波行为最为复杂。

### 等熵流关系式

等熵流（isentropic flow）是可压缩流的基准模型，假设流动绝热且无粘性（无摩擦），熵保持不变。在此条件下，滞止参数（总参数）与静参数的关系为：

$$\frac{T_0}{T} = 1 + \frac{\gamma - 1}{2} Ma^2$$

$$\frac{p_0}{p} = \left(1 + \frac{\gamma - 1}{2} Ma^2\right)^{\frac{\gamma}{\gamma-1}}$$

$$\frac{\rho_0}{\rho} = \left(1 + \frac{\gamma - 1}{2} Ma^2\right)^{\frac{1}{\gamma-1}}$$

其中下标 0 表示流体被绝热减速至静止时的滞止（总）状态。这三个公式将伯努利方程推广到了高速可压缩情形——当 Ma → 0 时，压力公式退化为不可压缩伯努利方程的近似形式。对于空气（γ = 1.4），当 Ma = 1 时，T/T₀ = 0.833，p/p₀ = 0.528，即临界截面处静压仅为滞止压的 52.8%。

等熵流还引出了**喉部面积比**概念：在收缩-扩张喷管（拉瓦尔喷管）中，喉部（最小截面）处流速恰好等于声速（Ma = 1），出口截面积 A 与喉部截面积 A* 之比决定出口马赫数：

$$\frac{A}{A^*} = \frac{1}{Ma}\left[\frac{2}{\gamma+1}\left(1 + \frac{\gamma-1}{2}Ma^2\right)\right]^{\frac{\gamma+1}{2(\gamma-1)}}$$

### 激波

激波是超声速流动中特有的薄层结构，厚度仅约几个分子自由程（在标准大气下约 0.1–1 μm）。通过正激波，流动由超声速骤变为亚声速，压力、密度、温度急剧升高，而速度骤降。正激波关系式（兰金-于戈尼奥条件）给出激波前（下标1）和激波后（下标2）的参数关系：

$$\frac{p_2}{p_1} = \frac{2\gamma Ma_1^2 - (\gamma-1)}{\gamma+1}$$

$$Ma_2^2 = \frac{(\gamma-1)Ma_1^2 + 2}{2\gamma Ma_1^2 - (\gamma-1)}$$

以 Ma₁ = 2、γ = 1.4 为例：激波后压力比 p₂/p₁ = 4.5，马赫数降至 Ma₂ ≈ 0.577。激波过程不等熵，熵增大、总压损失，这是超声速进气道设计中必须控制的核心损失来源。

## 实际应用

**拉瓦尔喷管（火箭发动机）**：SpaceX Merlin 发动机喷管利用收缩-扩张设计，使燃气在喉部达到 Ma = 1，随后在扩张段加速到 Ma ≈ 3–4，将热能高效转化为动能，提高比冲。设计时必须使用上述面积比公式而非伯努利方程。

**超声速进气道**：F-15 战斗机进气道设计为在 Ma = 2.5 巡航时通过一系列斜激波将来流减速，每道斜激波的总压损失远小于单道正激波，使总压恢复系数维持在约 0.95 以上，保障发动机进口气流质量。

**音爆（Sonic Boom）**：飞行器在超声速飞行时，机头和机尾产生的马赫锥（锥角 μ = arcsin(1/Ma)）扫过地面，造成可听到的双脉冲压力突变，即音爆。协和式超声速客机（Ma = 2.02）音爆约为 100 Pa 超压，是限制其陆上超声速飞行的主要原因。

## 常见误区

**误区一：以为超声速流动在管道收缩时加速**。对于亚声速流，管道截面减小确实使流速增大（连续性方程）；但超声速流动中面积-速度关系恰好相反——截面减小使超声速气流减速，截面扩大才使其加速。这源于可压缩流的连续性方程 dA/A = (Ma² - 1)dV/V，当 Ma > 1 时，面积与速度同向变化。

**误区二：以为激波前后总温不变而总压也不变**。正激波确实是绝热过程（总温 T₀ 保持不变），但激波内部存在强烈粘性耗散，熵增大，因此总压 p₀ 必然下降。以 Ma = 3 正激波为例，总压恢复系数仅约 0.328，意味着 67% 的总压被不可逆损失掉，这是超声速飞行器气动效率损失的根本原因之一。

**误区三：误用不可压缩伯努利方程计算高速气流**。不可压缩伯努利方程假设 ρ = const，在 Ma = 0.8 时用它计算动压会低估约 10%，在 Ma = 2 时误差超过 100%。正确做法是使用等熵流公式，以滞止压力与静压之比来反推流速。

## 知识关联

**与伯努利方程的关系**：不可压缩伯努利方程 p + ½ρV² = const 是等熵流公式在 Ma → 0 极限下的近似。将等熵压力公式对 Ma 展开泰勒级数，前两项恰好还原出伯努利方程的动压项，第三项为 ¼ρV²·Ma² 的可压缩修正量。因此伯努利方程是可压缩等熵流方程的低速特例，而非并列关系。

**与热力学的联系**：等熵流公式直接来自热力学第一定律（绝热）与第二定律（可逆，即 ds = 0）的联立，以及理想气体状态方程 p = ρRT。激波的不等熵特性则源于热力学第二定律：自然发生的过程熵只能增大，因此膨胀激波（密度降低的冲击波）在物理上不存在，自然界只允许压缩激波。

**延伸方向**：掌握本节内容后，可进一步学习斜激波与膨胀波（Prandtl-Meyer 膨胀）、准一维喷管流动的详细计算，以及高超声速流动中真实气体效应（离解与电离）对 γ 值的影响。
