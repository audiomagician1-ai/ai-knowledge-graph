# 放射性衰变

## 概述

放射性衰变是不稳定原子核自发地释放粒子或电磁辐射、转变为更稳定核素的过程。这一现象由贝可勒尔（Henri Becquerel）于1896年发现——他将铀盐放置在遮光的照相底片旁，发现底片仍然被感光，由此证明铀能自发辐射。随后居里夫人系统研究了钋（Po）和镭（Ra）的放射性，并于1898年命名了"放射性"（radioactivité）这一术语。

放射性衰变的本质是强核力与电磁力在核内竞争的结果。当质子数与中子数之比偏离稳定带时，原子核处于高能态，倾向于通过衰变降低总质量-能量（即结合能增大、质量亏损），以达到更稳定的核构型。质量亏损按爱因斯坦质能方程 $E = \Delta m c^2$ 转化为辐射能量或粒子动能，这是衰变释放能量的根本来源。

放射性衰变在核能、医学成像（PET/SPECT）、放射性年代测定和辐射防护等领域均具有直接应用价值。天然放射性同位素如 $^{238}$U、$^{14}$C 的衰变链，是地球内部热能和地质年代测定的重要依据。

## 核心原理

### α衰变

α衰变是重核（通常质量数 $A > 200$）释放一个α粒子（$^4_2\text{He}$ 核，含2个质子和2个中子）的过程。衰变方程的一般形式为：

$$^A_Z X \rightarrow \, ^{A-4}_{Z-2} Y + \, ^4_2\text{He} + Q$$

其中 $Q$ 为衰变能，由母核与子核、α粒子的质量差决定。以 $^{226}_{88}\text{Ra}$ 为例：

$$^{226}_{88}\text{Ra} \rightarrow \, ^{222}_{86}\text{Rn} + \, ^4_2\text{He}$$

α粒子是由于**量子隧穿效应**穿透库仑势垒而逸出的——经典力学无法解释α粒子逃脱，但量子力学给出隧穿概率，由伽莫夫（Gamow）于1928年用 WKB 近似计算，所得隧穿概率直接决定了α衰变的半衰期长短。α粒子质量大、电荷高，在空气中射程仅约3–8 cm，穿透力极弱，一张纸即可阻挡。

### β衰变

β衰变分为三种子类型：

- **β⁻衰变**：中子数过多时，一个中子转变为质子，同时释放电子（$e^-$）和反电子中微子（$\bar{\nu}_e$）：
  $$n \rightarrow p + e^- + \bar{\nu}_e$$
  核反应写法：$^A_Z X \rightarrow \, ^A_{Z+1} Y + e^- + \bar{\nu}_e$

- **β⁺衰变**：质子数过多时，一个质子转变为中子，释放正电子（$e^+$）和电子中微子（$\nu_e$）：
  $$p \rightarrow n + e^+ + \nu_e$$

- **电子俘获（EC）**：内层轨道电子被质子俘获，$p + e^- \rightarrow n + \nu_e$，是β⁺衰变的竞争过程。

β衰变的核心特征是发射粒子的**能量连续谱**，而非单一值。1930年泡利（Pauli）为解释这一连续谱提出了中微子的存在——能量守恒要求衰变总能量必须固定，因此多余部分由不可见的中微子带走。β粒子穿透力强于α粒子，在空气中射程可达数米，需铝板或塑料板屏蔽。

### γ辐射

γ辐射不改变原子核的质子数或中子数，仅是原子核从激发态跃迁到基态或较低能态时释放的高能光子。γ光子能量通常在0.01 MeV至数MeV之间，例如 $^{60}$Co 衰变后子核 $^{60}$Ni 释放 1.17 MeV 和 1.33 MeV 两条特征γ射线。γ射线是电磁辐射，穿透力最强，需数厘米铅板或数十厘米混凝土才能有效屏蔽。γ辐射通常伴随α或β衰变之后发生，是核激发态退激的标志。

### 衰变定律与放射性活度

放射性衰变是随机的量子过程，对大量原子核整体而言服从指数衰减规律：

$$N(t) = N_0 \, e^{-\lambda t}$$

其中 $N_0$ 为初始核数，$\lambda$ 为**衰变常数**（单位：s⁻¹），代表每个核单位时间内发生衰变的概率。放射性活度 $A$（即单位时间内衰变次数）定义为：

$$A(t) = \lambda N(t) = A_0 \, e^{-\lambda t}$$

活度的国际单位是**贝可勒尔（Bq）**，1 Bq = 每秒1次衰变；旧单位居里（Ci）定义为 $1 \, \text{Ci} = 3.7 \times 10^{10} \, \text{Bq}$，对应1 g镭的活度。

## 实际应用

**放射性年代测定**：$^{14}$C 的β⁻衰变（$\lambda = 1.21 \times 10^{-4} \, \text{yr}^{-1}$，半衰期5730年）被用于测定有机物年代。生物死亡后停止摄入碳，$^{14}$C 持续衰变，通过测量剩余 $^{14}$C 与 $^{12}$C 的比值即可推算死亡时间，可测范围约为300至50000年。

**核医学诊断**：PET（正电子发射断层扫描）利用β⁺发射体如 $^{18}$F（半衰期约110分钟）标记葡萄糖类似物注入体内，正电子与电子湮灭产生两个方向相反、能量各为0.511 MeV的γ光子，通过探测器重建三维代谢图像。

**核电站燃料与废料**：$^{235}$U 裂变产生的子核多为中子过剩的β⁻放射体，构成高放废料的主要辐射来源。了解各同位素的衰变类型和活度，是设计核废料储存方案的基础。

## 常见误区

**误区1：γ衰变改变了原子核的组成。**  
γ辐射不涉及质子数或中子数的变化，它只是核激发态向低能态跃迁的退激过程，类似原子中电子从高能级跃迁释放可见光，但能量尺度高出约百万倍。将γ辐射称为"γ衰变"在严格意义上不准确，它是伴随α/β衰变后的激发态退激，而不是独立的衰变类型。

**误区2：β衰变中电子来自核外电子云。**  
β⁻衰变释放的电子是在衰变过程中由核内中子通过弱相互作用新产生的，不是原子的核外电子。弱相互作用（W⁻玻色子作为媒介粒子）是β衰变的力学机制，与强核力和电磁力均无关。

**误区3：衰变常数λ越大，原子核越不稳定，衰变越慢。**  
恰恰相反：λ越大，表示单位时间衰变概率越高，原子核寿命越短（平均寿命 $\tau = 1/\lambda$），半衰期 $t_{1/2} = \ln 2 / \lambda \approx 0.693/\lambda$ 越小。例如 $^{212}$Po 的半衰期仅为 $2.99 \times 10^{-7}$ 秒，对应极大的λ值。

## 知识关联

**前置概念——原子核结构**：α、β衰变的发生条件直接取决于核内质子数（$Z$）与中子数（$N$）的比值是否偏离稳定带。质量亏损的计算依赖于核子结合能的知识，结合能曲线在α衰变（重核释放结合能更大的产物）中尤为关键。

**后续概念——半衰期**：衰变定律 $N(t)=N_0e^{-\lambda t}$ 中的 $t_{1/2}=\ln2/\lambda$ 将衰变常数转化为可直接测量的时间特征量，半衰期是衰变定律在实验和应用中的核心参数形式。

**后续概念——辐射防护**：α、β、γ三种辐射的穿透能力差异（分别被纸、铝、铅屏蔽）直接决定了防护材料和厚度的选择原则，这些规律完全源于各衰变类型的粒子属性。
