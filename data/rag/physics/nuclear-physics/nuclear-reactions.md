# 核反应

## 概述

核反应是指入射粒子（质子、中子、α粒子或γ光子等）与靶核相互作用，导致原子核发生变化并生成新核或新粒子的过程。核反应不同于化学反应——化学反应仅涉及核外电子重组，而核反应直接改变原子核的质子数或中子数，可释放或吸收约百万电子伏特（MeV）量级的能量，比化学反应能量高出约百万倍。

核反应的发现史始于1919年，卢瑟福首次用α粒子轰击氮-14原子核，生成了氧-17和质子，写作 ⁴He + ¹⁴N → ¹⁷O + ¹H，这是人类历史上第一个人工诱发的核反应。此后1932年查德威克发现中子，也正是通过核反应 ⁴He + ⁹Be → ¹²C + n 完成的。这两个里程碑事件奠定了整个核物理学的实验基础。

核反应在能源、医学和国防领域具有不可替代的地位。核裂变反应（如²³⁵U + n → 裂变碎片 + 2~3n）是核电站的能量来源；核聚变反应（如²H + ³H → ⁴He + n + 17.6 MeV）是太阳辐射的根本机制，也是人类追求的终极清洁能源。

## 核心原理

### 核反应方程与守恒定律

核反应方程必须同时满足四个守恒定律：**质量数守恒**（反应前后各粒子的质量数A之和相等）、**电荷数守恒**（质子数Z之和相等）、**能量守恒**以及**动量守恒**。以卢瑟福反应为例：

> ⁴₂He + ¹⁴₇N → ¹⁷₈O + ¹₁H

左侧：A = 4+14 = 18，Z = 2+7 = 9；右侧：A = 17+1 = 18，Z = 8+1 = 9，完全吻合。

书写核反应方程时，常见格式还有靶核记法：X(a,b)Y，其中X是靶核，a是入射粒子，b是出射粒子，Y是剩余核。例如 ¹⁴N(α,p)¹⁷O 与上述方程等价。这种简洁记法在文献中极为常用。

### Q值——核反应的能量账本

Q值（反应能）定义为核反应释放（或吸收）的净能量，计算公式为：

$$Q = (M_{\text{入射}} - M_{\text{出射}}) \times c^2$$

具体地：

$$Q = \left[(M_X + M_a) - (M_Y + M_b)\right] \times 931.5 \text{ MeV/u}$$

其中质量单位为原子质量单位u，1u对应931.5 MeV的静止能量。

- **Q > 0**：放热反应（exothermic），如²³⁵U裂变平均释放约200 MeV；反应无阈能，入射粒子仅需极低动能即可触发。
- **Q < 0**：吸热反应（endothermic），称为"核反应阈能"，入射粒子必须携带超过阈能 $E_{\text{th}} = |Q| \cdot \frac{M_X + M_a}{M_X}$ 的动能才能使反应发生。

以¹⁴N(α,p)¹⁷O为例，Q = -1.19 MeV，这意味着卢瑟福实验中α粒子需具备足够动能才能完成这一吸热反应。

### 核反应截面——反应概率的度量

截面（cross section）σ用来定量描述核反应发生的概率，其物理意义等价于靶核对入射粒子的"有效作用面积"。单位是**靶恩（barn，b）**，1 b = 10⁻²⁴ cm² = 10⁻²⁸ m²。

实验定义：若单位面积靶面上有n个靶核，入射粒子束流强度为I（单位时间内穿过单位面积的粒子数），则单位时间内发生反应的粒子数R为：

$$R = \sigma \cdot n \cdot I$$

截面大小与入射粒子能量密切相关，并非恒定值。热中子（能量约0.025 eV）与²³⁵U的裂变截面高达582 b，而快中子与²³⁵U的裂变截面仅约1~2 b，相差数百倍——这正是热堆核电站使用慢化剂（水或石墨）减慢中子速度的物理依据。

截面还可按反应类型细分为弹性散射截面、非弹性散射截面、俘获截面、裂变截面等，总截面等于各分截面之和：$\sigma_{\text{tot}} = \sigma_{\text{el}} + \sigma_{\text{inel}} + \sigma_{\text{cap}} + \sigma_{\text{f}} + \cdots$

## 实际应用

**核电站的链式裂变反应**：²³⁵U吸收一个热中子后裂变，平均产生2.43个快中子和约200 MeV能量。这些快中子经慢化后再次引发裂变，形成自持链式反应。临界条件由中子增殖因子k = 1决定；k > 1为超临界（核爆），k < 1为次临界（反应堆停堆）。

**放射性同位素生产**：医用¹³¹I由¹³⁰Te(n,γ)¹³¹Te经β衰变获得，¹⁸F（PET显像剂）通过¹⁸O(p,n)¹⁸F核反应在回旋加速器中生产，每次生产需精确控制质子束能量在10~18 MeV范围内。

**核聚变能源**：氘-氚反应 ²H + ³H → ⁴He（3.5 MeV）+ n（14.1 MeV），总释放17.6 MeV。14.1 MeV中子携带约80%的能量。托卡马克装置需将等离子体加热至约1.5×10⁸ K才能克服库仑势垒使反应持续发生。

## 常见误区

**误区1：认为核反应方程中质量守恒**

许多初学者将质量数守恒混淆为质量守恒。实际上，核反应前后各粒子的**静止质量之和并不相等**——质量亏损Δm转化为释放的能量，遵循爱因斯坦质能关系E = Δmc²。例如氘-氚聚变中，反应后⁴He和中子的总静止质量比反应前的²H和³H轻了约0.01887 u，正好对应17.6 MeV。守恒的是**质量数**（核子总数），而非精确的静止质量。

**误区2：Q值仅由初末态决定，与反应路径无关，就以为所有裂变产物释放能量相同**

Q值确实由初末态粒子的静止质量差决定，与路径无关。但²³⁵U裂变并非产生固定产物——裂变碎片分布遵循统计规律，最可能的碎片质量数在A≈95和A≈139附近（双峰分布），每次裂变的Q值因产物不同而在185~210 MeV范围内波动，平均才约200 MeV。认为每次裂变恰好释放200 MeV是过度简化。

**误区3：截面就是原子核的几何面积**

原子核的几何截面约为 πR² ≈ π(1.2×A^(1/3) fm)²，对²³⁵U约为1.7 b。但热中子与²³⁵U的裂变截面高达582 b，远超几何面积。截面是量子力学意义上的等效作用面积，与德布罗意波长、共振效应及核力本质密切相关，不能用经典几何图像理解。

## 知识关联

**依赖前置知识**：核反应方程的书写依赖对**原子核结构**（质子数Z、中子数N、质量数A = Z+N）的理解；Q值计算需要掌握原子质量单位u与MeV的换算（1u = 931.5 MeV/c²）以及质量亏损概念，这些在核结构部分已建立。核力的短程性（作用距离约1~3 fm）决定了为何带电粒子需要克服库仑势垒才能引发核反应，而中子由于不带电可更容易诱发反应。

**向衰变知识的延伸**：核反应产生的某些不稳定核素会进一步发生α衰变、β衰变或γ跃迁，这些衰变过程本质上也是特殊的核转变，其Q值同样由初末态核素的质量差决定，与核反应Q值计算方法完全一致。掌握核反应Q值计算，可直接推广至理解各种放射性衰变能量的来源。
