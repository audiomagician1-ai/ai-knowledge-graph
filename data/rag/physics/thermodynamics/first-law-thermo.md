# 热力学第一定律

## 概述

热力学第一定律是能量守恒定律在热力学系统中的具体表达，它定量描述了系统内能的变化量与系统吸收的热量及系统对外做功之间的数量关系。其核心数学表达式为 **ΔU = Q − W**，其中 ΔU 表示系统内能的变化量，Q 表示系统从外界吸收的热量（吸热为正，放热为负），W 表示系统对外界做的功（对外做功为正，外界对系统做功为负）。这一符号约定是物理学与工程热力学中最常见的约定，但须注意工程热力学有时将公式写为 ΔU = Q + W，此时 W 改为外界对系统做的功。

该定律的历史根源可追溯至19世纪中叶。德国物理学家亥姆霍兹（Hermann von Helmholtz）于1847年发表了系统阐述能量守恒的论文，而英国物理学家焦耳（James Prescott Joule）在1840年代通过著名的"焦耳热功当量实验"，测定了热功当量约为 **4.186 J/cal**，即1卡路里的热量等价于4.186焦耳的机械功。这一实验结果直接否定了当时流行的"热素说"，证明热量不是物质而是能量的传递形式。

热力学第一定律的意义在于它明确区分了热量、功与内能三个不同的物理量，并给出了三者之间的定量约束。具体而言，内能是系统的状态函数（只与当前状态有关），而热量和功都是过程量（与路径有关），这一区分对于分析任何热力学过程都至关重要。

## 核心原理

### 内能的本质与内能变化 ΔU

内能（Internal Energy，符号 U）是系统中所有微观粒子的动能与势能的总和，包括分子平动动能、转动动能、振动能以及分子间相互作用势能。对于理想气体，分子间无相互作用势能，内能仅由分子动能决定，故 **理想气体内能只与温度有关**，表达式为 U = nCᵥT，其中 n 为物质的量，Cᵥ 为摩尔定容热容，T 为热力学温度。ΔU 是状态函数的变化量，这意味着无论系统经历等压过程、等容过程还是任意可逆/不可逆路径，只要初末状态相同，ΔU 就完全相同。

### 热量 Q 的方向性与符号规定

热量是由于温度差而在系统与外界之间传递的能量，传递方向由高温流向低温。在热力学第一定律中，Q > 0 表示系统吸热（外界向系统传递能量），Q < 0 表示系统放热。热量是典型的过程量——同一初末状态，沿不同路径传递的热量可以完全不同。例如，理想气体从状态 A 沿等温路径到状态 B 与沿绝热路径到状态 B（若可达），吸收的热量截然不同，但 ΔU 相同。

### 功 W 的计算与意义

对于准静态过程，系统对外做的体积功（膨胀功）计算公式为：

**W = ∫P dV**

其中 P 为系统压强，dV 为体积微元。当气体膨胀（dV > 0）时，系统对外做正功；当气体压缩（dV < 0）时，外界对系统做正功，W < 0。在 P-V 图（压强-体积图）上，功的大小等于过程曲线下方的面积，这一几何意义使得比较不同过程做功的多少变得直观。除体积功外，实际系统还可能涉及电功、表面张力做功等，但在基础热力学中通常仅讨论体积功。

### 三种特殊过程中第一定律的简化

- **等容过程**（V = 常数）：W = 0，故 ΔU = Q。系统吸收的全部热量都转化为内能增加，此时 Q = nCᵥΔT。
- **等温过程**（理想气体，T = 常数）：ΔU = 0，故 Q = W。系统吸收的热量全部转化为对外做的功，或外界做的功全部以热量形式散出。
- **绝热过程**（Q = 0）：ΔU = −W。系统内能的减少量等于系统对外做的功，绝热压缩使内能升高（温度上升），绝热膨胀使内能降低（温度下降）。

## 实际应用

**内燃机气缸中的热力循环**：汽油发动机压缩冲程中，活塞压缩气体，外界对气体做功（W < 0），在接近绝热条件下（Q ≈ 0），由 ΔU = Q − W 得 ΔU > 0，气缸内温度急剧升高至约500°C以上，为点火做准备。这是热力学第一定律直接指导工程设计的典型案例。

**冰箱制冷的能量分析**：冰箱工作时，压缩机对制冷剂做功 W < 0（外界对系统做功），制冷剂从冷藏室吸热 Q > 0，然后在冷凝器向室内放热。对整个制冷剂循环一周，ΔU = 0（循环回到初始状态），由 Q_放 = Q_吸 + |W_压缩机| 可知，冰箱向室外散出的热量必然大于从冷藏室吸收的热量，这与直觉相符且由热力学第一定律严格保证。

**登山者的热量需求估算**：一名体重70 kg的登山者攀登高度差1000 m的山峰，克服重力做功约为 W = mgh = 70 × 10 × 1000 = 7 × 10⁵ J。考虑人体热效率约25%，实际需要消耗的食物能量约为 2.8 × 10⁶ J，约合670千卡，这一估算直接来源于热力学第一定律对能量转换的约束。

## 常见误区

**误区一：热量等于内能**。许多学生误以为"物体吸收了热量，内能就增加了相同的量"，忽略了系统同时可能在对外做功。例如，气体等温膨胀时吸热 Q > 0，但内能不变（ΔU = 0），吸收的热量全部转化为对外做的功。热量、功、内能变化三者须通过 ΔU = Q − W 联立考量，而非简单地将 Q 与 ΔU 画等号。

**误区二：功和热量都是系统的"储存量"**。功和热量都是能量传递的方式，是过程量，系统本身并不"含有"某个热量值或功值，只有内能才是系统的状态函数。说"系统含有多少热量"在热力学上是没有意义的表述；正确说法是"系统内能为多少"或"系统在某过程中吸收了多少热量"。

**误区三：第一定律允许第一类永动机**。有人认为只要维持能量"循环"就可以不消耗任何能量持续做功，实质上是混淆了"能量守恒"与"能量可无限循环利用"。第一类永动机（无需任何能量输入却持续对外做功的机器）违反热力学第一定律，因为持续做功意味着系统持续减少内能或持续吸热，两者都不能在没有能量来源的情况下永久维持。

## 知识关联

**与先修知识的联系**：热力学第一定律中的功 W 概念直接继承自力学中"功与能"的定义，体积功公式 W = ∫P dV 是力学中 W = ∫F·ds 在热力学场景的具体化。比热容与热容量的知识则提供了计算 Q 和 ΔU 的工具：等容过程中 Q = nCᵥΔT，等压过程中 Q = nCₚΔT，两者之差恰好对应对外做的功 nRΔT（理想气体），即 **Cₚ − Cᵥ = R**，这一关系是迈耶关系式，直接由热力学第一定律推导。

**向后续知识的延伸**：掌握 ΔU = Q − W 之后，可以系统分析等温、等压、等容、绝热四种热力学基本过程的能量特征，为学习卡诺循环及热力学第二定律奠定定量基础。焓的概念（H = U + PV）正是为了简化等压过程的计算而引入的状态函数，在等压过程中 ΔH = Qₚ，使得化学反应热与相变潜热的计算大为简便，是热力学第一定律在等压条件下的自然推广。
