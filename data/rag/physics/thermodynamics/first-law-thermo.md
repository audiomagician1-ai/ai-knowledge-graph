# 热力学第一定律

## 概述

热力学第一定律是能量守恒定律在热力学系统中的具体表达形式，它将系统内能的变化量与外界对系统传递的热量和系统对外做的功联系为一个精确的数学等式。其核心表达式为 **ΔU = Q - W**，其中 ΔU 表示系统内能的变化量，Q 表示系统从外界吸收的热量，W 表示系统对外界做的功。这一定律明确地断言：热和功都是能量传递的方式，而内能才是系统本身储存能量的状态量。

历史上，热力学第一定律的确立经历了数十年的争论。1842年，德国医生迈尔（Julius Robert von Mayer）首次从理论上提出热和功之间存在等价关系，并估算了热功当量。1843年至1849年间，英国物理学家焦耳（James Prescott Joule）通过著名的"搅拌水"实验精确测定热功当量为 **4.186 J/cal**（即1卡路里等于4.186焦耳），为第一定律提供了坚实的实验基础。1850年，克劳修斯将这些实验结论整理为严密的热力学理论框架，正式确立了这条定律。

热力学第一定律的重要性在于它彻底否定了"第一类永动机"的可能性——即任何无需能量输入却能持续对外做功的机器。由于 ΔU = Q - W 要求做功必须有相应的内能变化或热量输入，任何试图凭空产生功的设计都违反了这一等式，因而在原理上不可实现。

## 核心原理

### 内能（U）的本质

内能是系统内部所有微观粒子的动能与势能之和，是一个**状态函数**，其值只取决于系统当前的热力学状态（如温度、压强、体积），而与系统如何到达该状态的路径无关。对于理想气体，内能仅与温度有关，表达式为 **U = nCᵥT**，其中 n 为物质的量，Cᵥ 为定容摩尔热容，T 为热力学温度。这意味着同一种理想气体，只要温度相同，无论压强或体积如何变化，内能就相同。

### 符号约定与方程解读

使用 ΔU = Q - W 时，符号规定至关重要：
- **Q > 0**：系统从外界**吸热**；Q < 0：系统向外界**放热**。
- **W > 0**：系统对外界**做功**（如气体膨胀推动活塞）；W < 0：外界对系统做功（如气体被压缩）。

部分教材（尤其是化学热力学领域）采用 ΔU = Q + W 的形式，此时 W 定义为外界对系统做的功，符号相反。物理学中通常使用 Q - W 的形式，学生需注意所在领域的规范，避免混用导致计算错误。

### 功的计算：体积功

在热力学中，气体对外做的最常见形式是**体积功（膨胀功）**，其微元表达式为：

**dW = p dV**

对于有限过程：**W = ∫p dV**

这个积分的数值等于 p-V 图像上过程曲线与体积轴之间围成的面积。因为压强 p 在不同过程中与体积 V 的关系不同（等压、等温、绝热各有不同曲线），同样从状态 A 到状态 B，走不同路径做的功 W 一般不相等。这说明 W 是**过程量**，不是状态量——而 ΔU 作为状态量，其值对于相同的初末状态始终相同，因此 Q = ΔU + W 中的 Q 同样是路径相关的过程量。

### 第一定律在典型过程中的简化

| 热力学过程 | 条件 | 第一定律简化形式 |
|---|---|---|
| 等容过程（V不变） | dV = 0，W = 0 | ΔU = Q |
| 等压过程（p不变） | W = pΔV | ΔU = Q - pΔV |
| 等温理想气体（T不变） | ΔU = 0 | Q = W |
| 绝热过程（Q = 0） | 与外界无热交换 | ΔU = -W |

等容过程最直接地揭示了内能变化与热量的关系：在体积不变时，所有吸收的热量全部转化为内能增量，这正是通过定容热容 Cᵥ 来测量内能变化的物理依据。

## 实际应用

**内燃机的工作循环**：四冲程汽油发动机的做功冲程可视为近似绝热膨胀（Q ≈ 0），此时 ΔU = -W，即气体内能的减少完全转化为活塞的机械功。燃烧冲程则是系统快速吸热（Q 很大正值），而体积变化有限，导致内能急剧增加、温度升高。整个循环回到初始状态后 ΔU_循环 = 0，因此整个循环吸收的净热量等于对外输出的净功。

**绝热压缩升温**：柴油发动机不使用火花塞，而是依靠绝热压缩将空气内能提升到足以点燃柴油的温度（约550°C）。此过程中 Q = 0，外界对气体做正功（W < 0，即系统对外做负功），故 ΔU = -W > 0，内能增加，温度升高。压缩比约为14:1到25:1，远高于汽油机的8:1至12:1。

**生物体的能量代谢**：人体将食物中的化学内能（葡萄糖氧化：C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O，释放约2870 kJ/mol）转化为热量（维持体温，Q 向外）和机械功（肌肉做功，W 向外），符合 ΔU = Q - W。长期 Q + W > 化学能释放时体重下降，即 ΔU < 0。

## 常见误区

**误区一：热量就是内能**。许多学生认为"物体含有热量"，但根据第一定律，Q 是系统与外界之间传递的能量，是一个描述过程的量；U 才是系统储存的能量。"物体含有热量"的说法在热力学中是错误的，正确说法是"物体具有内能"，"系统吸收或释放热量"。

**误区二：等温过程中内能不变，所以不需要热量**。对于理想气体，等温过程确实 ΔU = 0，但由于气体膨胀对外做功（W > 0），由 Q = ΔU + W = W > 0 可知系统必须从外界吸热来补偿做出的功。忽视这一点会误以为等温膨胀不需要热源，这是理解热机效率的常见障碍。

**误区三：绝热过程与等温过程混淆**。绝热过程的条件是 Q = 0（与外界无热量交换），但温度一般会变化（ΔT ≠ 0）；等温过程的条件是 ΔT = 0（温度恒定），但通常伴随热量交换（Q ≠ 0）。两者在 p-V 图上的斜率不同：绝热曲线比等温曲线更陡，因为绝热压缩时内能增加导致温度上升，压强升高幅度更大。

## 知识关联

**与比热容的联系**：在等容过程中，ΔU = Q = nCᵥΔT，定容摩尔热容 Cᵥ 正是通过这一关系定义和测量的。对于单原子理想气体，Cᵥ = (3/2)R ≈ 12.47 J/(mol·K)；双原子气体 Cᵥ = (5/2)R，这些具体数值来自能量均分定理，是第一定律与微观统计物理的接口。

**为热力学过程分析奠定基础**：学习等温、等压、等容、绝热四类基本热力学过程时，每一种都是在 ΔU = Q - W 的框架下，通过固定某一变量简化方程后进行分析的。卡诺循环效率 η = 1 - T₂/T₁ 的推导同样依赖于对每个分过程中 Q 和 W 关系的精确计算。

**引出焓的概念**：在等压过程中，Q_p = ΔU + pΔV = Δ(U + pV)，这个组合量 H = U + pV 就定义为**焓**。焓的引入正是为了简化等压过程的热量计算，化学反应热（如标准生成焓 ΔHf°）本质上就是等压条件下第一定律的应用结果。
