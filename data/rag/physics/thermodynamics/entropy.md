# 熵

## 概述

熵（Entropy）是热力学和统计物理中最深刻也最常被误解的概念。在热力学中，熵由鲁道夫·克劳修斯于1865年引入，用于量化热力学过程的**不可逆程度**。在统计力学中，路德维希·玻尔兹曼赋予熵以微观意义：熵度量系统的**微观状态数**，即系统可能的微观排列方式有多少种。

热力学第二定律可以用熵来表述：**孤立系统的熵永远不减少**。这条定律给出了自然过程的方向性——热量自发地从高温流向低温、气体自发地扩散到整个容器、有序结构自发地走向无序。熵是唯一一个在基本物理定律中给出"时间箭头"的量。

学习熵需要先掌握热力学第一定律（能量守恒）和热力学基本概念（温度、热量、功）。掌握熵后，可以理解自由能、化学平衡、相变、信息论等广泛的应用领域。

## 核心知识点

### 克劳修斯定义（宏观热力学）

对于**可逆过程**，熵变定义为：

$$dS = \frac{\delta Q_{\text{rev}}}{T}$$

其中 $\delta Q_{\text{rev}}$ 是可逆过程中系统吸收的热量，$T$ 是绝对温度。

有限过程的熵变：

$$\Delta S = \int_i^f \frac{\delta Q_{\text{rev}}}{T}$$

SI 单位：J/K（焦耳每开尔文）。

关键性质：
- 熵是**状态函数**：只取决于系统的当前状态，与到达该状态的路径无关
- 计算任意过程的熵变时，都可以构造一条连接初态和末态的可逆路径来计算

### 玻尔兹曼定义（微观统计）

$$S = k_B \ln \Omega$$

其中 $k_B = 1.381 \times 10^{-23}$ J/K 是玻尔兹曼常数，$\Omega$ 是与宏观状态对应的**微观状态数**（multiplicity）。这个公式刻在玻尔兹曼的墓碑上。

**典型例子**：N 个气体分子在两半容器中的分布
- 全部在左半边：$\Omega = 1$，$S = 0$（极度有序）
- 均匀分布：$\Omega = \binom{N}{N/2} \approx 2^N$，$S = Nk_B\ln 2$（最大熵状态）

对于 $N \sim 10^{23}$（一摩尔气体），所有分子恰好跑到一半容器中的概率约为 $2^{-10^{23}}$——虽然不为零，但在宇宙年龄内也不会发生。

### 热力学第二定律的熵表述

**克劳修斯不等式**：对于任何循环过程，

$$\oint \frac{\delta Q}{T} \leq 0$$

等号对可逆过程成立。

推论——**孤立系统的熵增原理**：

$$\Delta S_{\text{total}} = \Delta S_{\text{system}} + \Delta S_{\text{surroundings}} \geq 0$$

- 可逆过程：$\Delta S_{\text{total}} = 0$（理想极限）
- 不可逆过程：$\Delta S_{\text{total}} > 0$（所有实际过程）

### 典型过程的熵变计算

| 过程 | 熵变公式 |
|:---|:---|
| 等温膨胀（理想气体） | $\Delta S = nR\ln(V_f/V_i)$ |
| 等压加热 | $\Delta S = nC_p\ln(T_f/T_i)$ |
| 等容加热 | $\Delta S = nC_v\ln(T_f/T_i)$ |
| 相变（恒温恒压） | $\Delta S = Q/T = nL/T$（$L$为摩尔潜热） |
| 两温度物体接触达平衡 | $\Delta S > 0$（总是正值，不可逆） |

## 关键要点

1. **熵是状态函数，不是过程量**：计算熵变只需知道初态和末态，可以用任何方便的可逆路径
2. **"熵增"是统计结论，不是绝对禁令**：从微观看，熵减少是可能的（涨落），但对于宏观系统，偏离平衡态的概率小到不可想象
3. **系统的熵可以减小**：只要环境的熵增加得更多即可。冰箱让食物降温（熵减），但向环境排热（熵增更多），总熵仍增加
4. **熵与信息有深刻联系**：香农信息熵 $H = -\sum p_i \log p_i$ 与玻尔兹曼熵形式相同。获取信息本质上是减少不确定性（减熵），但需要消耗能量

## 常见误区

1. **"熵就是无序度"**：这只是一种直觉近似。更准确的说法是：熵度量的是宏观状态对应的微观状态数的对数。有些看起来"有序"的状态（如晶体）熵也不为零，因为原子有热振动
2. **"热力学第二定律禁止局部熵减"**：不禁止。制冷机、生物体都是局部熵减的例子。第二定律只要求**宇宙总熵**不减
3. **"熵增意味着能量减少"**：熵增和能量守恒是两回事。不可逆过程中能量完全守恒（第一定律），但能量的"品质"降低了（高温热量变成低温热量），可用来做功的能力减少了

## 知识衔接

**先修概念**：热力学第一定律（能量守恒）、温度与热量、可逆过程与不可逆过程

**后续概念**：
- **热力学第二定律**：熵增原理的各种等价表述（克劳修斯、开尔文、卡诺）
- **自由能**：亥姆霍兹自由能 $F = U - TS$、吉布斯自由能 $G = H - TS$，判断过程自发性
- **统计力学**：从微观状态数推导宏观热力学量
- **信息论**：香农熵、信息与物理熵的关系（兰道尔原理）
- **黑洞物理**：贝肯斯坦-霍金熵 $S = kA/(4l_P^2)$，熵与面积成正比
