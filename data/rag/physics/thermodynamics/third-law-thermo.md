# 热力学第三定律

## 概述

热力学第三定律由德国化学家瓦尔特·能斯特（Walther Nernst）于1906年首次提出，因此又称为**能斯特定理**（Nernst Heat Theorem）。其核心表述有两个等价形式：第一，任何系统的熵在温度趋近于绝对零度（0 K）时趋向于一个与系统状态无关的常数，该常数被定义为零；第二，不可能通过有限步骤的物理或化学过程使任何物体的温度降低至绝对零度。这两个表述共同构成了第三定律的完整内容。

1912年，能斯特进一步将其严格化为"绝对零度不可达原理"：即使通过无限多次的绝热去磁或等温循环操作，也永远无法使系统精确达到 T = 0 K。这与热力学第二定律共同限定了自然界中热过程的方向性和极限。第三定律的意义在于为热力学系统的**绝对熵**提供了零点基准，使化学反应的标准摩尔熵可以被计算为一个具体的正数值，而不是像内能那样只能谈相对变化。

## 核心原理

### 绝对熵为零的微观解释

从统计力学角度，玻尔兹曼公式 S = k_B ln Ω 直接解释了第三定律。其中 S 为熵，k_B = 1.380649 × 10⁻²³ J/K 为玻尔兹曼常数，Ω 为系统的微观状态数（微态数）。当温度趋于 0 K 时，一个理想的纯净晶体中所有粒子都处于唯一的量子基态，Ω = 1，因此 S = k_B ln 1 = 0。这不是一个近似值，而是理论上的精确极限。第三定律的热力学表述与这一统计力学结论完全自洽，两者互相印证。

### 残余熵：第三定律的限定条件

第三定律并非对所有物质普遍成立，而是特指**完美有序的纯净晶体**。对于某些物质，即使在极低温度下，系统仍存在**残余熵**（Residual Entropy），即 T→0 时 Ω ≠ 1。典型例子是一氧化碳（CO）晶体：CO 分子的两端极性接近，晶格排列时 CO 和 OC 两种取向几乎等能，导致即使接近 0 K 仍有随机排列，残余摩尔熵约为 R ln 2 ≈ 5.76 J/(mol·K)，其中 R = 8.314 J/(mol·K) 为气体常数。冰（H₂O）也存在类似的质子无序残余熵，约为 3.4 J/(mol·K)。因此，第三定律的完整表述必须加上"完美晶体"这一限定。

### 绝对零度不可达的热力学证明

以绝热去磁制冷为例说明为何无法达到 0 K。每次循环包含两步：等温磁化（熵降低）和绝热退磁（温度降低）。设系统从温度 T₁ 出发，经过 n 次循环后到达 Tₙ，Tₙ 始终大于零，且每次降温的幅度会越来越小。这是因为当 T→0 时，系统热容 C→0（这本身由量子统计推导，与第三定律一致），导致相同的熵变所对应的温度变化 ΔT = T·ΔS/C 在 C→0 时并不趋于无穷。两个趋零量之比的极限由具体模型决定，但所有已知物理模型都给出有限的温度下界，永远无法在有限步骤内触及 0 K。

## 实际应用

**低温物理实验的温度极限**：现代稀释制冷机（Dilution Refrigerator）利用 ³He/⁴He 混合物在相分离时的吸热效应，可以将系统冷却至约 1–10 mK（毫开尔文），但永远无法达到 0 K，实践上验证了第三定律的不可达性。

**化学标准摩尔熵的计算**：由于第三定律给出了熵的绝对零点，化学家可以通过测量从 0 K 到目标温度的热容数据，积分计算物质的标准摩尔熵 S°(298 K)：
$$S°(298\,\text{K}) = \int_0^{298} \frac{C_p}{T} \, dT$$
例如，金刚石在 298 K 的标准摩尔熵为 2.38 J/(mol·K)，而石墨为 5.74 J/(mol·K)，这一差值可用于计算石墨→金刚石转化反应的熵变。

**量子计算与超导器件**：超导量子比特（Qubit）需要在约 10–20 mK 的极低温下运行，以减少热噪声对量子态的干扰。稀释制冷机是目前唯一实用的技术路线，其运作原理正是在不断接近但无法到达的 0 K 附近工作。

## 常见误区

**误区一：认为"熵趋于零"意味着所有物质在低温下行为相同**。实际上，第三定律仅保证完美晶体在 T→0 时的熵趋于零，残余熵的存在（如CO、NO、H₂O冰等）说明无序结构在极低温下仍保留有限熵值。不同物质在低温下的热容行为也截然不同：金属满足 C ∝ T（电子贡献为主），绝缘晶体满足德拜 C ∝ T³ 定律，这些差异正体现了第三定律适用条件的精确性。

**误区二：把"绝对零度不可达"理解为"技术限制"而非物理定律**。有些学生认为只要技术足够先进就可以达到 0 K，这是错误的。不可达性是由热力学第三定律从原理上保证的——任何有限步骤的准静态过程都无法将系统从有限温度带到 T = 0 K，这是物理规律，而非工程瓶颈。1989年，科学家曾将铑（Rh）核自旋系统冷却到 2.8 × 10⁻¹⁰ K 的极低温，但 0 K 依然未被触及。

**误区三：将第三定律与热力学第二定律混淆**。第二定律讨论的是熵在孤立系统中的增减方向，第三定律则确立了熵的绝对量级基准。前者说"熵不减"，后者说"熵在 0 K 时为零"。两者在逻辑上独立，但共同决定了低温热力学的完整框架。

## 知识关联

学习热力学第三定律的先决条件是深刻理解**熵**的概念，包括克劳修斯定义（dS = δQ/T）和玻尔兹曼统计定义（S = k_B ln Ω）。没有对微观状态数 Ω 的理解，就无法解释为何完美晶体在 0 K 时 S = 0，也无法理解残余熵的成因。量子力学中的能级离散性是解释低温热容趋零的必要背景——经典统计力学无法预测这一行为，这也是历史上能斯特定理促使物理学家转向量子统计的重要动力之一。

热力学三个定律共同构成完整体系：第一定律（能量守恒）、第二定律（熵增方向）与第三定律（熵的零点）三者缺一不可。第三定律在低温物理、量子信息、天体物理（如白矮星冷却模型）以及精密化学热力学数据库的构建中均有直接的定量应用，是热力学理论完备性的重要组成环节。
