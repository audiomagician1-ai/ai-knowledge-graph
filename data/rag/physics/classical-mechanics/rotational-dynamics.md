# 转动动力学

## 概述

转动动力学研究力矩如何产生角加速度，是牛顿第二定律在转动情形下的推广。其核心方程 τ = Iα 将净力矩 τ（单位：N·m）、转动惯量 I（单位：kg·m²）与角加速度 α（单位：rad/s²）联系起来，形式上与线动力学的 F = ma 完全对应。

这一体系的奠基工作由欧拉在18世纪完成。他于1765年在《刚体运动理论》（*Theoria Motus Corporum Solidorum seu Rigidorum*）中系统推导了转动方程，并引入了转动惯量的概念，将质量对旋转轴的分布正式数学化。这使人类第一次能够定量预测飞轮、陀螺等旋转物体在外力矩作用下的响应。

转动动力学的重要性在于，现实中绝大多数机械系统——发动机曲轴、风力涡轮机叶片、卫星姿态控制轮——都需要用 τ = Iα 来设计启动力矩和控制角加速度。纯粹用质点力学无法处理这些问题，因为物体的质量分布直接决定了它对同一力矩的响应速度。

## 核心原理

### 转动惯量 I 的计算

转动惯量是物体质量相对于转轴的分布度量，定义为：

$$I = \sum_i m_i r_i^2 \quad \text{（离散）} \quad \text{或} \quad I = \int r^2 \, dm \quad \text{（连续）}$$

其中 $r_i$ 是第 $i$ 个质元到旋转轴的垂直距离。几个常用结果需要熟记：
- **均匀细杆绕中心轴**：$I = \frac{1}{12}mL^2$
- **均匀细杆绕端点轴**：$I = \frac{1}{3}mL^2$
- **实心圆盘绕中心轴**：$I = \frac{1}{2}mR^2$
- **空心圆环绕中心轴**：$I = mR^2$

注意，同一根杆绕端点的转动惯量是绕中心的 4 倍，这意味着相同力矩作用下，绕端点转动的角加速度只有绕中心的四分之一。

### 平行轴定理

当旋转轴从质心轴平行移动距离 $d$ 时，转动惯量按以下公式增大：

$$I = I_{\text{cm}} + md^2$$

其中 $I_{\text{cm}}$ 是绕质心轴的转动惯量，$m$ 是物体总质量，$d$ 是两轴间距。这个定理由惠更斯于17世纪首先提出，斯坦纳（Steiner）后来给出了严格证明，因此又称**斯坦纳定理**。

例如，均匀细杆绕端点的 $I = \frac{1}{3}mL^2$ 可用平行轴定理从中心结果推导：$I_{\text{cm}} = \frac{1}{12}mL^2$，$d = L/2$，则 $I = \frac{1}{12}mL^2 + m\left(\frac{L}{2}\right)^2 = \frac{1}{12}mL^2 + \frac{1}{4}mL^2 = \frac{1}{3}mL^2$，验证一致。

### τ = Iα 的应用逻辑

τ = Iα 中的 τ 必须是相对于**同一旋转轴**的净力矩，I 也必须是关于该轴的转动惯量。解题时需要：
1. 明确旋转轴位置；
2. 用 $r \times F$ 计算各力的力矩并求代数和得 τ；
3. 用平行轴定理或标准公式计算 I；
4. 由 α = τ/I 求角加速度，再结合转动运动学（$\omega = \omega_0 + \alpha t$，$\theta = \omega_0 t + \frac{1}{2}\alpha t^2$）求其他量。

对于同时发生平动和转动的物体（如在斜面上滚动的圆柱），需要同时列 $F = ma_{\text{cm}}$ 和 $\tau = I_{\text{cm}}\alpha$，并利用纯滚动条件 $a_{\text{cm}} = R\alpha$ 联立求解。

## 实际应用

**飞轮储能设计**：工业飞轮（如汽车发动机飞轮）需要高转动惯量以平滑转速波动。设计时有意将质量集中在轮缘（近似空心圆环，$I = mR^2$），而非均匀分布（实心圆盘，$I = \frac{1}{2}mR^2$）。相同质量和半径下，轮缘设计的 I 是实心盘的 2 倍，储存同等角速度时能量 $E = \frac{1}{2}I\omega^2$ 也翻倍。

**门的铰链位置**：推门时若在距铰链 $r$ 处施力 F，则力矩 τ = Fr，由 τ = Iα 可知 α ∝ r。这解释了为什么门把手总设计在远离铰链的一侧——同样的推力，力臂越长，门板获得的角加速度越大，开门更省力。

**斜面滚动比较实验**：将实心球（$I = \frac{2}{5}mR^2$）与空心球（$I = \frac{2}{3}mR^2$）从同一斜面顶端同时释放，实心球先到达底部，因为其转动惯量更小，在相同重力分量下获得更大角加速度。这是 τ = Iα 的直接演示，与物体质量无关。

## 常见误区

**误区一：把 τ = Iα 中的力矩理解为单个力的力矩**
τ 必须是所有外力对旋转轴力矩的代数和（净力矩），遗漏任何一个力（如绳子张力、摩擦力）都会导致计算错误。例如，分析绕固定轴转动的滑轮时，两侧绳子张力的力矩方向相反，必须相减后才能代入方程。

**误区二：平行轴定理可以在任意两个平行轴间使用**
平行轴定理要求其中一个轴必须过质心。若要从轴 A 换算到轴 B，不能直接用 $I_B = I_A + md_{AB}^2$，必须先从 A 换算回质心轴，再从质心轴换算到 B，即 $I_B = I_{\text{cm}} + md_B^2$。

**误区三：转动惯量大则"更难被停下来"但"不影响最终速度"**
转动惯量 I 影响的是角加速度，即角速度变化的快慢，而非最终的角动量守恒结果。在能量守恒问题中，I 大的物体在滚动中分配给转动的动能更多，平动速度反而更小，这两点经常被混淆。

## 知识关联

**与力矩的关系**：力矩（τ = r × F）是转动动力学的输入量，若不能正确计算力矩的大小和方向，τ = Iα 就无从应用。特别是力矩的正负号约定（通常以逆时针为正）必须在代入方程前统一。

**与转动运动学的关系**：转动运动学提供了 α、ω、θ 之间的运动方程，转动动力学通过 τ = Iα 计算出 α 后，才能接入运动学公式求出任意时刻的 ω 和 θ。两者分工明确：动力学提供 α 的来源，运动学描述 α 作用的结果。

**通向角动量**：将 τ = Iα 改写为 τ = I(dω/dt) = d(Iω)/dt，右侧 Iω 正是角动量 L 的定义。当 τ = 0 时，L = Iω = 常数，这便是角动量守恒定律的直接推论。花样滑冰运动员收臂加速（I 减小，ω 增大）的现象，本质上是 τ = Iα 在零外力矩条件下的推论。
