# 转动动力学

## 概述

转动动力学研究力矩如何引起物体转动状态的变化，其核心方程 τ = Iα 是牛顿第二定律 F = ma 在转动领域的直接类比：合外力矩 τ（单位 N·m）等于转动惯量 I（单位 kg·m²）与角加速度 α（单位 rad/s²）的乘积。这一关系由欧拉在18世纪中叶系统化，是分析齿轮、电机、陀螺等一切旋转机械的理论基础。

转动惯量 I 是转动动力学区别于平动动力学的关键物理量。与质量只由物体本身决定不同，转动惯量同时取决于质量分布和转轴位置。相同的物体绕不同轴转动时，I 值可以相差数倍，这使得转动动力学的计算比平动复杂得多，也更为丰富。

## 核心原理

### 转动惯量的计算

对于质量连续分布的刚体，转动惯量定义为 I = ∫r²dm，其中 r 是质量元 dm 到转轴的垂直距离。常见刚体的转动惯量有精确解析结果：均匀细杆绕中心轴 I = (1/12)mL²，绕端点轴 I = (1/3)mL²；均匀实心圆盘绕中心轴 I = (1/2)mR²；均匀实心球绕直径 I = (2/5)mR²；均匀薄球壳绕直径 I = (2/3)mR²。这些系数（1/12、1/2、2/5等）直接反映了质量相对转轴的分布特征——质量越集中在轴附近，系数越小。

### 平行轴定理

当已知刚体绕过质心轴的转动惯量 I_cm，则绕任意平行轴的转动惯量为：

**I = I_cm + md²**

其中 m 是刚体总质量，d 是两平行轴之间的垂直距离。以均匀细杆为例：已知绕中心轴 I_cm = (1/12)mL²，利用平行轴定理，绕端点轴 I = (1/12)mL² + m(L/2)² = (1/3)mL²，与积分结果完全吻合。需要强调的是，平行轴定理只能从质心轴向外"移轴"，不能在两个任意非质心轴之间直接使用。

### τ = Iα 的应用逻辑

使用 τ = Iα 求解问题时，必须明确以下三点：①合外力矩 τ 必须相对同一转轴计算（逆时针为正或顺时针为正，保持一致即可）；②转动惯量 I 也必须对该同一转轴计算；③α 是刚体的角加速度，对于绕固定轴转动的刚体，各质点角加速度相同，但切向线加速度 a_t = αr 因 r 不同而不同。

以绕端点悬挂的均匀细杆在重力下从水平静止开始转动为例：重力 mg 作用在质心（距端点 L/2 处），力矩 τ = mg(L/2)；绕端点转动惯量 I = (1/3)mL²；代入 τ = Iα 得初始角加速度 α = (3g)/(2L)，端点处线加速度 a = αL = (3/2)g，竟大于自由落体加速度 g，这是转动动力学中一个反直觉但经过实验验证的结论。

### 垂直轴定理（补充）

对于薄板形物体（质量分布在 xy 平面内），绕 z 轴（垂直于板面）的转动惯量满足：

**I_z = I_x + I_y**

其中 I_x、I_y 分别是绕 x 轴和 y 轴的转动惯量。均匀圆盘绕任意直径轴的 I_x = I_y = (1/4)mR²，由此得 I_z = (1/2)mR²，与直接积分一致。

## 实际应用

**飞轮储能装置**：工业飞轮设计中，质量集中在轮缘（相当于薄环，I = mR²），在相同质量和半径下，其转动惯量是实心圆盘（I = mR²/2）的2倍，可储存更多转动动能 E_k = (1/2)Iω²，这正是飞轮轮辐设计得很细而轮缘很厚的物理原因。

**花样滑冰旋转**：选手收臂时将手臂质量的 r 减小，根据 I = Σmr²，转动惯量 I 减小，在合外力矩接近零（冰面摩擦极小）的条件下，τ ≈ 0 意味着角加速度趋近于零，而非速度不变——这里是角动量守恒（非转动动力学范畴），但理解 I 的变化机制来自转动动力学中对 ∫r²dm 的理解。

**汽车发动机启动**：曲轴从静止加速到怠速 800 rpm（约 84 rad/s），若曲轴组合转动惯量约 0.1 kg·m²，所需角加速度 α = 84/t，起动电机需提供的力矩 τ = 0.1α。这是工程师设计起动机扭矩规格的直接计算依据。

## 常见误区

**误区一：混淆转动惯量与质量**。学生常认为"质量大的物体转动惯量一定大"。实际上，将同样质量重新分布可以大幅改变 I——把质量集中在轴上（r→0）则 I→0，把质量移到轮缘则 I 最大化。转动惯量是质量关于轴的二阶矩，不是质量本身。

**误区二：错误使用平行轴定理的方向**。有学生将平行轴定理写成 I_cm = I_other - md²，试图从一个非质心轴的 I 推算质心轴的 I。该公式在形式上允许逆向推导（代数上成立），但实际中常出现的错误是：把非质心轴误认为质心轴，导致算出比 I_cm 更小的 I 值，违反了"质心轴对应最小转动惯量"的定理。

**误区三：对固定轴转动和平动叠加的情形套用 τ = Iα**。当物体既有平动又有转动（如在斜面上滚动的圆柱），不能简单地只列 τ = Iα，必须同时列平动方程 F = ma 并利用纯滚动约束 a = αR 联立求解。仅凭 τ = Iα 单独求解会遗漏平动自由度，得到错误结果。

## 知识关联

**前置概念衔接**：力矩 τ = r×F 的计算是 τ = Iα 左端的输入量；转动运动学提供了 ω、α 的定义及 ω² = ω₀² + 2αθ 等运动学方程，与 τ = Iα 联合可求解转动的完整时间演化。没有这两个前置概念，转动动力学方程无法被具体求解。

**后续概念延伸**：转动动力学直接导向角动量 L = Iω 的概念——当合外力矩为零时，dL/dt = τ = 0，即角动量守恒；刚体运动则将转动动力学与平动动力学整合，处理滚动、进动等复杂运动，其中转动惯量张量（3×3矩阵）是标量 I 在三维空间的推广。
