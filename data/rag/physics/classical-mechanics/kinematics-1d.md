# 一维运动学

## 概述

一维运动学研究物体沿**单一直线方向**运动时的位移、速度和加速度之间的数量关系，不涉及力的成因，只描述运动的几何和时间特征。"一维"意味着只需一个坐标轴（通常取为 $x$ 轴）就能完整描述物体的位置，所有运动量均可用正负号表示方向。

这一框架的系统化描述可追溯到伽利略·伽利雷（Galileo Galilei）在17世纪初的斜面实验。伽利略通过计时滚球在斜面上的位移，首次量化验证了匀加速运动中位移与时间的平方成正比——这正是今天教材中 $x = \frac{1}{2}at^2$ 的实验起源。牛顿后来在《自然哲学的数学原理》（1687年）中将这些关系整合进更广泛的力学体系。

学习一维运动学的价值在于它提供了分析所有复杂运动的基本语言。二维抛体运动、圆周运动乃至振动问题，都依赖将复杂运动分解为若干一维分量分别处理。掌握匀变速方程的推导逻辑，也是理解微积分在物理中应用的第一个具体范例。

## 核心原理

### 位移与路程的区别

**位移**（displacement）$\Delta x = x_f - x_i$ 是从初位置指向末位置的有向线段，单位为米（m）。它是向量，符号代表方向：$\Delta x > 0$ 表示沿 $x$ 轴正方向移动，$\Delta x < 0$ 表示反向。**路程**是物体实际经过轨迹的长度，始终为正值。一个物体从 $x = 0$ 运动到 $x = 5\,\text{m}$ 再返回 $x = 2\,\text{m}$，路程为 $8\,\text{m}$，而位移仅为 $+2\,\text{m}$。混淆两者是初学者最常见的错误来源。

### 速度与加速度的微分定义

**瞬时速度**定义为位移对时间的导数：
$$v = \frac{dx}{dt}$$
在 $x$-$t$ 图像中，瞬时速度等于该点切线的斜率。**平均速度**则为 $\bar{v} = \frac{\Delta x}{\Delta t}$，对应割线斜率。

**加速度**是速度对时间的导数：
$$a = \frac{dv}{dt} = \frac{d^2x}{dt^2}$$
在 $v$-$t$ 图像中，加速度等于直线的斜率，而图线与时间轴围成的**面积**等于位移 $\Delta x$——这一几何关系是推导匀变速方程的直觉来源。

### 匀变速运动的四个方程

当加速度 $a$ 为常数时，对 $v = v_0 + at$ 积分可得位移，共可导出如下四个方程（变量：初速度 $v_0$、末速度 $v$、加速度 $a$、时间 $t$、位移 $\Delta x$）：

| 方程 | 形式 | 缺少的量 |
|------|------|----------|
| ① | $v = v_0 + at$ | $\Delta x$ |
| ② | $\Delta x = v_0 t + \frac{1}{2}at^2$ | $v$ |
| ③ | $v^2 = v_0^2 + 2a\Delta x$ | $t$ |
| ④ | $\Delta x = \frac{v_0 + v}{2} \cdot t$ | $a$ |

方程③由①和②消去 $t$ 代入得到，特别适用于不知道运动时间的场景（如制动距离计算）。每个方程包含五个量中的四个，已知三个即可求解其余两个。

### 自由落体作为标准模型

自由落体是一维匀加速运动的最重要具体实例：竖直向下取正方向，$a = g \approx 9.8\,\text{m/s}^2$，初速度 $v_0 = 0$。代入方程②得 $h = \frac{1}{2}gt^2$，这正是伽利略斜面实验测量的关系。在地球表面，一个物体自由落下 $1\,\text{s}$ 后速度达到 $9.8\,\text{m/s}$，下落高度为 $4.9\,\text{m}$。

## 实际应用

**汽车制动距离**：一辆以 $v_0 = 30\,\text{m/s}$（约108 km/h）行驶的汽车，制动加速度为 $-6\,\text{m/s}^2$，用方程③令 $v = 0$：
$$0 = (30)^2 + 2(-6)\Delta x \implies \Delta x = 75\,\text{m}$$
若初速度翻倍为 $60\,\text{m/s}$，制动距离变为 $300\,\text{m}$，是原来的 **4倍**，体现了速度平方项的非线性危险性。

**电梯运动分析**：电梯从静止出发，以 $a = 1.5\,\text{m/s}^2$ 加速 $4\,\text{s}$，然后匀速运行，最后减速停止。加速阶段位移 $= \frac{1}{2}(1.5)(4^2) = 12\,\text{m}$，末速度 $= 1.5 \times 4 = 6\,\text{m/s}$。分段套用匀变速方程，各段独立计算后叠加，是处理非均匀运动的标准策略。

**运动传感器实验**：现代物理实验室用超声波测距仪以 $25\,\text{Hz}$ 采样频率记录小车位置，计算相邻采样点的 $\frac{\Delta x}{\Delta t}$ 近似瞬时速度，再对速度数据作差分得加速度，直接在计算机屏幕上呈现 $x$-$t$、$v$-$t$、$a$-$t$ 三图，使抽象的微分关系可视化。

## 常见误区

**误区一：速度大则加速度也大**。速度和加速度是完全独立的量。一辆以 $200\,\text{km/h}$ 匀速行驶的赛车加速度为零；一个从手中释放的静止小球（$v_0 = 0$）在释放瞬间加速度为 $9.8\,\text{m/s}^2$。混淆两者源于没有区分"运动有多快"（速度）与"速度如何变化"（加速度）这两个不同层次的描述。

**误区二：位移公式 $\Delta x = v_0 t + \frac{1}{2}at^2$ 中 $v_0$ 必须为正**。$v_0$ 可以是负值，代表初始运动方向与坐标轴正方向相反。例如一个向左以 $4\,\text{m/s}$ 运动的物体受到向右 $2\,\text{m/s}^2$ 的加速度，取右为正则 $v_0 = -4\,\text{m/s}$，代入公式仍完全有效。

**误区三：匀变速方程适用于所有运动**。这四个方程的推导前提是 $a = \text{常数}$。弹簧振子的加速度随位移变化（$a = -\frac{k}{m}x$），车辆受空气阻力时加速度随速度变化，这些情形均不能直接套用匀变速公式，需要用微分方程或能量方法处理。

## 知识关联

**前置概念衔接**：一维运动学中位移、速度、加速度均为有向量，直接应用了**物理中的向量**对正负号代表方向的约定。若不理解向量的符号规则，就无法正确判断速度与加速度同向（加速）还是反向（减速）的情形。

**通向二维运动学**：学习二维抛体运动时，水平方向（$a_x = 0$，匀速）和竖直方向（$a_y = -g$，匀加速）分别独立满足本章的一维方程。也就是说，二维运动学的核心方法论是"将运动分解为两个互相独立的一维问题"，因此本章方程的熟练度直接决定二维问题的解题效率。

**通向牛顿第一定律**：匀变速方程中令 $a = 0$ 得 $v = v_0 = \text{常数}$，$\Delta x = v_0 t$。这是一个纯运动学推论，而牛顿第一定律赋予它物理解释：合力为零的物体保持匀速直线运动。运动学描述"如何运动"，动力学解释"为何如此运动"，两者在 $a = 0$ 这一特殊情形处首次交汇。
