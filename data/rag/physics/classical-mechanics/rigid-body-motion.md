# 刚体运动

## 概述

刚体是指在受力过程中形状和大小保持不变的理想物体——任意两质点之间的距离始终保持固定。真实物体在小形变条件下均可近似为刚体，例如钢制齿轮、车轮、门轴等。刚体运动是经典力学对多质点系运动规律的集中体现，将质点力学中的平动方程推广为包含转动自由度的完整描述。

刚体运动的系统性研究可追溯至欧拉（Leonhard Euler）于18世纪中期的工作。1765年，欧拉在《刚体力学》（*Theoria Motus Corporum Solidorum seu Rigidorum*）中首次建立了刚体转动方程，引入了主轴与惯性张量概念，奠定了现代刚体动力学的基础。欧拉角至今仍是描述刚体空间姿态的标准工具。

刚体运动之所以重要，在于它将物体的**6个自由度**——3个平动自由度（x、y、z方向位移）与3个转动自由度（俯仰、偏航、滚转）——统一在一个理论框架下。工程中，从陀螺仪的稳定性分析、卫星姿态控制，到普通滚动轴承设计，都必须依赖刚体运动学与动力学。

## 核心原理

### 平动与转动的分解

刚体的任意运动可以被分解为**质心的平动**加上**绕质心的转动**，这一结论称为柯尼希定理（König's theorem）在运动学层面的推论。设刚体质心位置为 $\vec{r}_C$，刚体上任意一点 $i$ 的速度为：

$$\vec{v}_i = \vec{v}_C + \vec{\omega} \times \vec{r}'_i$$

其中 $\vec{v}_C$ 是质心速度，$\vec{\omega}$ 是角速度矢量，$\vec{r}'_i$ 是质点 $i$ 相对于质心的位矢。这意味着刚体的动能同样分解为两部分：

$$E_k = \frac{1}{2}mv_C^2 + \frac{1}{2}I_C\omega^2$$

第一项为质心平动动能，第二项为绕质心的转动动能，$I_C$ 是对质心轴的转动惯量。两部分相互独立，但共同决定刚体的完整运动状态。

### 定轴转动

当刚体绕固定轴转动时，运动自由度降为1，动力学方程简化为：

$$M = I\alpha$$

其中 $M$ 为合外力矩（单位 N·m），$I$ 为对该轴的转动惯量（单位 kg·m²），$\alpha = d\omega/dt$ 为角加速度。这一方程在形式上与牛顿第二定律 $F = ma$ 完全对应，$I$ 对应质量 $m$，$\alpha$ 对应线加速度 $a$，$M$ 对应合力 $F$。

转动惯量 $I$ 的计算依赖质量分布：对离散质点系 $I = \sum_i m_i r_i^2$，对连续体 $I = \int r^2 \, dm$。当转轴从质心轴平移距离 $d$ 时，适用**平行轴定理**：

$$I = I_C + md^2$$

例如，均匀圆盘对质心轴的转动惯量为 $\frac{1}{2}MR^2$，对过盘缘且平行的轴则为 $\frac{3}{2}MR^2$。

定轴转动的守恒量是**角动量** $L = I\omega$。当合外力矩为零时，$L$ 守恒——花样滑冰运动员收拢手臂后 $I$ 减小、$\omega$ 增大，正是这一原理的直观演示。

### 滚动运动

滚动是平动与转动的耦合形式，分为**纯滚动**（无滑动滚动）和**滑动滚动**。对于半径为 $R$ 的圆柱体在平面上纯滚动，质心速度 $v_C$ 与角速度 $\omega$ 满足约束条件：

$$v_C = R\omega$$

这是纯滚动的关键约束方程。与此同时，接触点的瞬时速度为零，即接触点为**瞬时速度中心**（瞬心）。

纯滚动时，静摩擦力做功为零（因接触点瞬时静止），因此可以用能量守恒分析。以质量为 $m$、半径为 $R$、转动惯量为 $I_C$ 的圆柱体沿倾角 $\theta$ 斜面纯滚而下为例，利用能量守恒：

$$mgh = \frac{1}{2}mv_C^2 + \frac{1}{2}I_C\omega^2$$

代入 $v_C = R\omega$ 及 $I_C = \frac{1}{2}mR^2$，可得质心加速度：

$$a_C = \frac{2}{3}g\sin\theta$$

相比在同斜面上无转动的质点（加速度 $g\sin\theta$），滚动圆柱的加速度仅为其 $\frac{2}{3}$，差额体现在转动动能的"分摊"上。不同形状物体的加速度比值由 $I_C/mR^2$ 决定：空心圆柱（$I_C = mR^2$）加速度为 $\frac{1}{2}g\sin\theta$，小于实心圆柱。

## 实际应用

**陀螺进动**：高速旋转的陀螺在重力矩作用下不倒反而进动，进动角速度 $\Omega = \frac{mgr}{I\omega}$，其中 $r$ 是质心到支点的距离。这是角动量矢量在力矩持续作用下方向改变的典型案例，广泛应用于惯性导航系统和陀螺稳定平台。

**车轮滚动摩擦**：汽车轮胎在路面上纯滚动时，驱动力由静摩擦提供，满足 $f \leq \mu_s mg$；一旦制动过猛发生滑动，变为滑动摩擦，制动距离反而增大，这是ABS防抱死系统的设计依据。

**转速测量与齿轮传动**：两啮合齿轮半径分别为 $R_1$ 和 $R_2$，定轴转动条件下，齿接触处线速度相同，故 $R_1\omega_1 = R_2\omega_2$，传动比 $i = \omega_1/\omega_2 = R_2/R_1 = z_2/z_1$（$z$ 为齿数），这是机械变速箱设计的基本方程。

## 常见误区

**误区一：混淆刚体的合外力与合外力矩**。学生常认为"合外力为零时刚体静止"，但实际上合外力为零只保证质心不加速，若合外力矩不为零，刚体仍会产生角加速度。正确表述：刚体处于平衡需同时满足 $\sum \vec{F} = 0$ 和 $\sum \vec{M} = 0$ 两个条件，共6个标量方程。

**误区二：认为滚动摩擦做功**。在纯滚动中，接触点瞬时速度为零，静摩擦力虽存在但对刚体做功为零，因此可以用机械能守恒计算纯滚动问题。而在有打滑的滚动中，动摩擦力做功不为零，必须用功能定理处理，机械能不守恒。

**误区三：转动惯量是固定常数**。转动惯量 $I$ 的值依赖转轴位置，同一刚体对不同轴的 $I$ 值不同——均匀细杆对端点轴的 $I = \frac{1}{3}mL^2$，对中点轴的 $I = \frac{1}{12}mL^2$，两者相差4倍。忽视转轴选取会导致计算错误。

## 知识关联

**前置知识**：转动动力学提供了力矩、角动量、转动惯量的基本定义和计算方法。刚体运动将这些概念组合应用于具有平动与转动双重自由度的系统，尤其是纯滚动问题中约束方程 $v_C = R\omega$ 的引入，是连接定轴转动与一般运动的关键步骤。

**横向联系**：刚体的平动方程 $\vec{F} = m\vec{a}_C$ 与转动方程 $\vec{M}_C = I_C\vec{\alpha}$ 共同构成完整的刚体动力学方程组，分别对应线动量定理和角动量定理在刚体上的具体形式。柯尼希定理将系统动能拆分为质心动能与相对质心动能之和，这一分解方式在处理碰撞、爆炸等问题时也同样有效。

**工程拓展**：在三维空间中，当转轴不固定时，需引入惯性张量（3×3对称矩阵）和欧拉动力学方程来处理刚体的自由转动，这属于高年级理论力学或工程力学的内容，是刚体运动分析的完整延伸方向。
