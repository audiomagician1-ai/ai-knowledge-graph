# 刚体运动

## 概述

刚体是指在外力作用下形状和大小保持不变的理想物体——即任意两质点间的距离始终恒定。刚体运动是经典力学中描述固体运动状态的完整框架，将一般质点系的运动分解为**质心平动**与**绕质心转动**两个相互独立的分量。这一分解由欧拉在18世纪中叶系统化，是分析复杂机械运动的理论基石。

刚体运动之所以区别于质点运动，在于刚体具有**空间延展性**：同一时刻，刚体上不同位置的速度方向和大小可以完全不同。一个自由刚体在三维空间中拥有6个自由度——3个平动自由度（质心位置 $x, y, z$）和3个转动自由度（欧拉角 $\phi, \theta, \psi$）。对刚体运动的研究覆盖了从齿轮传动到行星自转的广泛场景，也是工程机械设计的计算基础。

## 核心原理

### 平动与转动的分解

根据柯尼希定理，刚体的总动能可分解为质心平动动能与绕质心转动动能之和：

$$E_k = \frac{1}{2}Mv_C^2 + \frac{1}{2}I_C\omega^2$$

其中 $M$ 为刚体总质量，$v_C$ 为质心速度，$I_C$ 为绕质心轴的转动惯量，$\omega$ 为角速度。这一分解的关键在于：两项彼此独立，平动方程由合外力决定（$\vec{F} = M\vec{a}_C$），转动方程由对质心的合力矩决定（$\vec{\tau}_C = I_C\vec{\alpha}$）。

刚体上任意一点 $P$ 的速度可写为：
$$\vec{v}_P = \vec{v}_C + \vec{\omega} \times \vec{r}_{CP}$$
其中 $\vec{r}_{CP}$ 是从质心指向 $P$ 点的位矢。这说明刚体运动在任意瞬间可视为绕某一**瞬时轴**的纯转动。

### 定轴转动

定轴转动是刚体运动中约束最强的特例：转轴方向固定，刚体每个质点均做圆周运动，圆心在转轴上。定轴转动的动力学方程为：

$$\tau = I\alpha$$

其中 $\tau$ 是对转轴的合力矩，$I$ 是刚体对该固定轴的转动惯量，$\alpha$ 是角加速度。转动惯量 $I = \sum_i m_i r_i^2$（连续分布时为 $\int r^2 \, dm$），与质量分布和轴的位置密切相关。

**平行轴定理**提供了从质心轴转换到平行任意轴的方法：
$$I = I_C + Md^2$$
其中 $d$ 是两轴之间的距离。例如，均匀细杆绕端点的转动惯量为 $\frac{1}{3}ML^2$，而绕质心的转动惯量为 $\frac{1}{12}ML^2$，两者之差恰好满足平行轴定理（$d = L/2$）。

定轴转动中角动量守恒条件为合外力矩为零，这解释了花样溜冰运动员收拢手臂后旋转加速的现象：$I_1\omega_1 = I_2\omega_2$，质量向轴集中使 $I$ 减小，$\omega$ 增大。

### 滚动（纯滚动与滚动摩擦）

滚动是平动与转动的耦合运动。对于在固定平面上**纯滚动**（无滑动）的圆形物体，接触点的瞬时速度为零，因此满足约束条件：

$$v_C = R\omega$$

其中 $R$ 为滚动体半径。这一约束将平动和转动自由度关联起来，使系统从2个独立自由度退化为1个广义坐标。

纯滚动时，静摩擦力提供驱动力矩但不做功（接触点速度为零）。滚动物体沿斜面（倾角 $\theta$）的加速度为：

$$a = \frac{g\sin\theta}{1 + I_C/(MR^2)}$$

分母中的 $I_C/(MR^2)$ 反映了转动惯量的"阻碍"效应。均匀实心球的 $I_C = \frac{2}{5}MR^2$，代入得 $a = \frac{5}{7}g\sin\theta$；而薄壁空心球的 $I_C = \frac{2}{3}MR^2$，代入得 $a = \frac{3}{5}g\sin\theta$，明显更小——这正是滚动比赛中实心球领先的原因。

## 实际应用

**汽车车轮**：车轮在正常行驶时做纯滚动，$v_C = R\omega$ 使得里程表可通过转速直接计算行驶距离。轮胎与地面间的静摩擦提供前进的驱动力，若超过最大静摩擦力（急加速或制动）则发生滑动，此时动摩擦力介入，制动距离显著增加。ABS系统的原理正是维持车轮边缘接近纯滚动状态以保留最大静摩擦力。

**陀螺仪**：高速旋转的定轴刚体具有抵抗轴方向改变的特性，称为陀螺稳定性。角动量 $L = I\omega$ 越大，进动角速度 $\Omega = \tau/L$ 越小，轴方向越稳定。航空航天惯性导航系统中，陀螺仪转速通常高达 $24000$ rpm，转动惯量经精密设计以达到所需稳定精度。

**齿轮系统**：两个相互啮合的齿轮做约束定轴转动，节圆半径之比等于角速度之反比：$\omega_1 R_1 = \omega_2 R_2$。这是纯滚动约束在工程传动中的直接应用，变速箱通过多级齿轮组合实现扭矩与转速的精确调配。

## 常见误区

**误区一：认为滚动物体所受摩擦力总做负功**。纯滚动时接触点速度为零，静摩擦力对整个刚体做功为零，而非负功。只有发生相对滑动时，动摩擦力才耗散能量。混淆静摩擦与动摩擦在滚动中的角色会导致能量计算错误。

**误区二：将定轴转动的力矩方程 $\tau = I\alpha$ 直接套用于非定轴转动**。当转轴方向改变（如进动、章动）时，角动量方程必须写为向量形式 $\vec{\tau} = d\vec{L}/dt$，且转动惯量需用惯量张量（$3\times3$ 矩阵）描述，此时 $\vec{L}$ 与 $\vec{\omega}$ 不一定同向。标量公式只在主轴方向上成立。

**误区三：认为质心平动和绕质心转动可以单独分析任意点的受力**。刚体动力学的两个方程（质心运动定理和对质心的角动量定理）必须**联立求解**，尤其在存在未知约束力（如轴承反力、地面摩擦力）时，缺少任何一个方程都无法完整确定运动状态。

## 知识关联

刚体运动的分析直接依赖**转动动力学**的概念基础：转动惯量的计算方法、力矩的定义以及角动量定理均是本章的运算工具。若未掌握转动惯量的积分计算（如 $I = \int r^2\,dm$），将无法处理具有连续质量分布的刚体（如圆盘、球体、杆）的动力学问题。

在更广泛的力学体系中，刚体运动是连接**质点力学**与**分析力学**（拉格朗日、哈密顿力学）的关键桥梁。纯滚动约束是典型的**完整约束**，可自然引入广义坐标的概念——一个在斜面上纯滚的圆柱，其构型空间由单一广义坐标（质心位移或转角，二者通过 $v_C = R\omega$ 等价）完全描述，是学习分析力学约束理论的标准入门模型。
