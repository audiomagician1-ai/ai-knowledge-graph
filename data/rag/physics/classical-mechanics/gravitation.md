# 万有引力

## 概述

1687年，牛顿在《自然哲学的数学原理》中提出：宇宙中任意两个有质量的物体之间都存在相互吸引的力。这个统一了苹果落地与行星运动的理论，彻底改变了人类对自然界的认识。万有引力定律是经典力学的支柱之一，从卫星轨道设计到潮汐预测，它的影响遍及科学与工程的每个角落。

## 核心知识点

### 1. 万有引力定律

两个质量分别为 m1 和 m2 的质点，相距 r 时，彼此间的引力大小为：

**F = G * m1 * m2 / r^2**

其中 G 为万有引力常数，实验测定值为 6.674 x 10^(-11) N*m^2/kg^2。

关键特征：
- **平方反比律**：力与距离的平方成反比，这意味着距离加倍时引力降为1/4。
- **质量正比**：力与两个物体质量的乘积成正比。
- **普适性**：适用于所有有质量的物体，从原子尺度到星系尺度。
- **矢量形式**：F = -G*m1*m2 / r^2 * r_hat，负号表示引力始终为吸引力。

**实验验证**：1798年，亨利·卡文迪什用扭秤实验首次在实验室中测量了G的数值，其结果与现代测量值在1%以内吻合（Cavendish, Phil. Trans. 1798）。

### 2. 引力势能

将一个质量为 m 的物体从距质量 M 的中心 r 处移到无穷远所需要的功，定义了引力势能：

**U(r) = -G * M * m / r**

核心性质：
- 负号表示束缚态：物体在引力场中的势能为负值，意味着需要输入能量才能逃逸。
- 势能零点在无穷远处（r -> infinity 时 U -> 0）。
- **逃逸速度**：令动能等于势能绝对值，得 v_escape = sqrt(2GM/R)。地球表面的逃逸速度约为 11.2 km/s。
- **引力势**：phi = -GM/r，是单位质量的势能，常用于天体物理计算。

实例：地球质量 5.972 x 10^24 kg，半径 6371 km。在地表，一个 1 kg 物体的引力势能约为 -6.25 x 10^7 J。

### 3. 开普勒定律与引力的关系

开普勒在1609-1619年间基于第谷·布拉赫的观测数据总结出三大定律，牛顿后来证明它们都是万有引力定律的数学推论：

**第一定律（轨道定律）**：行星绕太阳运动的轨道是椭圆，太阳位于椭圆的一个焦点上。这是平方反比力场中二体问题的解析解。

**第二定律（面积定律）**：行星与太阳的连线在相等时间内扫过相等面积。这是角动量守恒的直接结果（dL/dt = r x F = 0，因为引力沿径向）。

**第三定律（周期定律）**：行星轨道半长轴 a 的立方与周期 T 的平方之比为常数：T^2 / a^3 = 4*pi^2 / (GM)。这个关系直接用于测量天体质量——通过观测卫星的轨道周期和半径即可推算中心天体质量。

### 4. 壳层定理与等效性

牛顿证明了两个关键的壳层定理（Principia, Book I, Propositions 70-71）：

- **外部等效**：均匀球壳对外部质点的引力等效于全部质量集中在球心的质点。这就是为什么我们可以把地球当作质点来计算地表引力。
- **内部为零**：均匀球壳对其内部质点的净引力为零。这导致了一个有趣的结论：如果在地球内部挖一条隧道，物体受到的引力只取决于比它到地心更近的那部分质量。

### 5. 引力场与潮汐力

引力场强度 g = -GM/r^2 * r_hat 描述单位质量受到的引力。地球表面 g 约为 9.81 m/s^2。

**潮汐力**是引力场的不均匀性（梯度）产生的差异力。月球对地球近端和远端的引力差异导致海洋潮汐。潮汐加速度正比于 d/r^3（d 为物体尺度，r 为中心距离），解释了为什么靠近黑洞时潮汐力会变得极端剧烈（"面条化效应"）。

## 关键要点

1. **F = Gm1m2/r^2** 是平方反比的普适吸引力，G = 6.674 x 10^(-11) N*m^2/kg^2
2. 引力势能 **U = -GMm/r** 为负值，逃逸速度 v = sqrt(2GM/R)
3. 开普勒三定律都是万有引力定律的数学推论，第三定律 T^2/a^3 = 4pi^2/(GM) 直接用于天体质量测量
4. 壳层定理使球对称质量分布可以简化为质点处理
5. 万有引力定律在弱场、低速条件下极其精确，但水星近日点进动等现象需要广义相对论修正

## 常见误区

1. **"太空中没有引力"**：国际空间站中的宇航员看似"失重"，实际上那里的引力仍有地表的约89%。失重感来源于自由落体——空间站和宇航员以相同加速度绕地球运动。
2. **混淆重力与万有引力**：地球表面的"重力"(g=9.81)并非纯引力，而是引力减去地球自转的离心加速度分量。赤道处两者差异约0.3%。
3. **忽略引力常数G的测量难度**：G是基本物理常量中测量精度最低的之一（相对不确定度约2.2 x 10^(-5)），因为引力相对于电磁力极其微弱。
4. **将开普勒定律视为独立定律**：它们实际上是万有引力在二体问题中的推论，而非独立的物理定律。

## 知识衔接

### 先修知识
- **牛顿第二定律** — F=ma 是理解引力加速度和轨道运动方程的基础
- **能量守恒** — 引力势能的定义和逃逸速度推导依赖能量守恒原理

### 后续学习
- **轨道力学** — 运用万有引力分析卫星轨道、转移轨道（霍曼转移）
- **潮汐力** — 引力梯度的物理效应，从海洋潮汐到洛希极限
- **广义相对论初步** — 引力作为时空弯曲的现代理解
