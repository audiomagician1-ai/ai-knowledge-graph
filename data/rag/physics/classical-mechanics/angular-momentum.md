# 角动量

## 概述

角动量（Angular Momentum）是描述旋转运动"运动量"的物理量，定义为物体的位置矢量与线动量的叉积：**L = r × p**，其中 **r** 是从参考点到质点的位置矢量，**p = mv** 是线动量。对于绕固定轴旋转的刚体，角动量简化为 **L = Iω**，其中 *I* 为转动惯量，*ω* 为角速度。角动量是矢量，其方向由右手定则确定，与 **r** 和 **p** 均垂直。

角动量的概念最早由牛顿在1687年的《自然哲学的数学原理》中隐含提出，开普勒第二定律（行星在等时间内扫过等面积）实质上正是行星绕太阳运动时角动量守恒的几何表达。19世纪欧拉与拉格朗日在刚体力学中将其系统化，使角动量成为分析陀螺仪、天体运动等复杂旋转问题的核心工具。

角动量的重要性在于它是自然界的守恒量之一。通过诺特定理，角动量守恒与空间各向同性（旋转对称性）直接对应——只要系统的物理规律在空间旋转下保持不变，该系统的总角动量就守恒。这一联系使角动量从经典力学延伸至量子力学和粒子物理，成为贯穿整个物理学的基础量。

## 核心原理

### 角动量的定义与计算

对单质点，角动量相对于参考点 O 定义为：

$$\mathbf{L} = \mathbf{r} \times \mathbf{p} = m(\mathbf{r} \times \mathbf{v})$$

其大小为 $L = mvr\sin\theta$，其中 $\theta$ 是 **r** 与 **v** 之间的夹角。当质点做圆周运动时 $\theta = 90°$，$L = mvr = mr^2\omega$，这正好等于 $I\omega$（因点质量的转动惯量 $I = mr^2$）。

对于由多个质点组成的系统，总角动量等于各质点角动量的矢量和：$\mathbf{L}_{total} = \sum_i \mathbf{r}_i \times \mathbf{p}_i$。对刚体绕固定轴旋转，不同轴对应不同的转动惯量（例如均匀实心球体绕直径轴 $I = \frac{2}{5}mr^2$，细棒绕端点 $I = \frac{1}{3}ml^2$），导致相同角速度下角动量大小不同。

### 角动量定理：力矩是角动量的时间变化率

角动量的时间导数等于作用于质点的合力矩：

$$\boldsymbol{\tau} = \frac{d\mathbf{L}}{dt}$$

这是角动量定理，是牛顿第二定律的转动版本。力矩 $\boldsymbol{\tau} = \mathbf{r} \times \mathbf{F}$，单位为 N·m。若合外力矩为零，则 $\frac{d\mathbf{L}}{dt} = 0$，即角动量不随时间变化——这正是角动量守恒的条件。需要特别注意，力矩和角动量必须相对同一参考点计算，混用不同参考点会导致错误。

### 角动量守恒定律

**当系统所受合外力矩为零时，系统总角动量守恒**，即 $\mathbf{L}_i = \mathbf{L}_f$（初末状态角动量相等）。对旋转刚体，这意味着 $I_1\omega_1 = I_2\omega_2$。

花样滑冰运动员收臂加速是这一定律最直观的例证：运动员伸臂旋转时转动惯量约为 $4\ \text{kg·m}^2$，角速度约为 $1\ \text{rad/s}$；收臂后转动惯量减小至约 $1\ \text{kg·m}^2$，角速度相应增大至约 $4\ \text{rad/s}$，总角动量 $L = 4\ \text{kg·m}^2/\text{s}$ 保持不变。天文尺度上，脉冲星（中子星）由于半径从约 $10^6\ \text{m}$ 的恒星塌缩至约 $10\ \text{km}$，转动惯量急剧减小，自转周期可短至 1.4 毫秒（如脉冲星 PSR J1748-2446ad），这是角动量守恒在宇宙级别的体现。

## 实际应用

**陀螺仪的稳定性**：高速旋转的陀螺具有大角动量，由于改变角动量矢量方向需要施加持续力矩，外界小扰动难以快速改变其转轴方向，这一特性被用于飞机、轮船和航天器的惯性导航系统，以及智能手机的姿态传感器。

**行星轨道与开普勒第二定律**：太阳系中行星所受引力指向太阳，对太阳的力矩为零，故行星绕太阳运动的角动量守恒。这直接导致近日点速度快、远日点速度慢的现象，以及面积速度恒定（$\frac{dA}{dt} = \frac{L}{2m} = \text{常数}$）。

**跳水与体操的空中动作**：跳水运动员离台后无外力矩（忽略空气阻力），角动量守恒。团身姿态转动惯量小（约 $3\ \text{kg·m}^2$），可实现高速翻转；展体时转动惯量增大（约 $12\ \text{kg·m}^2$），旋转减慢，可精确控制入水角度。

**子弹的膛线稳定**：枪管内的来复线（膛线）使子弹高速自旋，自旋角动量使弹头轴线保持稳定，抵抗飞行中的气动扰动，大幅提高射击精度。

## 常见误区

**误区一：角动量守恒等同于线动量守恒**。两者是独立的守恒律，适用条件不同。合外力为零时线动量守恒；合外力矩为零时角动量守恒。二者可以同时成立（如孤立系统），也可以只有一个成立——例如，物体受向心力（如行星绕日）时合外力不为零，线动量不守恒，但对力心点的合外力矩为零，角动量守恒。

**误区二：只有做圆周运动的物体才有角动量**。实际上，任何做匀速直线运动的质点相对于不在其运动直线上的点都具有非零角动量，且其大小 $L = mvd$（*d* 为参考点到运动直线的垂直距离）保持不变——这是匀速直线运动（合外力为零，力矩也为零）情形下角动量守恒的体现。

**误区三：角动量守恒意味着角速度不变**。角动量 $L = I\omega$ 守恒，但若转动惯量 *I* 发生改变（如质量分布改变），角速度 *ω* 必然随之变化。许多同学看到"守恒"便认为 *ω* 不变，忽略了转动惯量随形状变化的事实。

## 知识关联

**前置概念——转动动力学**：力矩、转动惯量、角加速度（$\tau = I\alpha$）是理解角动量的必要工具。转动动力学给出了 $\tau = \frac{dL}{dt}$ 的具体计算基础，没有对转动惯量的掌握，就无法量化地计算刚体角动量 $L = I\omega$。

**后续概念——自旋**：经典角动量是轨道角动量（质点绕外部参考点转动产生），而量子力学中的自旋是粒子的内禀角动量，没有经典的旋转对应，但其数学描述（算符对易关系 $[\hat{L}_x, \hat{L}_y] = i\hbar\hat{L}_z$）与经典角动量矢量代数直接类比。理解经典角动量的矢量性和守恒性，是进入量子力学自旋概念的必要铺垫。

**横向关联——刚体进动**：角动量矢量在外力矩作用下会发生进动（而非翻转），进动角速度 $\Omega = \frac{\tau}{L\sin\theta}$，这是陀螺仪和地轴进动（地球自转轴约26,000年完成一次进动周期）的共同机制，是角动量矢量性的直接物理后果。
