# 转动运动学

## 概述

转动运动学是描述物体绕固定轴旋转时位置、速度和加速度变化规律的学科分支，其核心变量是角位移 θ、角速度 ω 和角加速度 α。与直线运动学研究质点沿直线的位移 x 不同，转动运动学将分析对象从线量替换为角量，使得描述刚体绕轴旋转的数学结构与直线运动完全平行。

转动运动学的数学框架在18世纪由欧拉（Leonhard Euler）系统整理，他在1765年出版的《刚体力学》中明确引入了角速度作为描述旋转快慢的物理量，并建立了旋转运动的基本方程组。这一框架使得分析齿轮、陀螺仪、行星自转等复杂旋转现象成为可能。

转动运动学的重要性在于它与直线运动学之间存在精确的一一对应关系：线位移 x 对应角位移 θ，线速度 v 对应角速度 ω，线加速度 a 对应角加速度 α。掌握这组对应关系，不仅能高效求解旋转系统，还为后续研究转动动力学（引入转动惯量 I 和力矩 τ）奠定描述基础。

## 核心原理

### 角位移 θ

角位移是描述物体转过角度大小和方向的物理量，国际单位制中使用**弧度（rad）**而非度数，原因在于只有弧度制才能保证弧长公式 s = rθ 直接成立，其中 r 为转动半径，s 为对应弧长。

角位移的符号规定：通常以逆时针方向为正方向（右手定则：右手四指弯向旋转方向，拇指所指为角位移向量方向）。需要特别注意，1转（一整圈）= 2π rad ≈ 6.283 rad，1 rad ≈ 57.3°，这两个换算数字在解题中频繁用到。

### 角速度 ω

角速度定义为角位移对时间的导数：

$$\omega = \frac{d\theta}{dt}$$

其单位为 rad/s。对于匀速转动，ω = Δθ/Δt。角速度与线速度的关系为：

$$v = r\omega$$

这里 v 是圆周上某点的切向线速度，r 是该点到转轴的距离。常见转速用 n（转/分钟，rpm）表示，换算关系为：

$$\omega = \frac{2\pi n}{60}$$

例如，家用电风扇转速约为 1200 rpm，对应角速度 ω = 2π × 1200 / 60 = 40π ≈ 125.7 rad/s。

### 角加速度 α

角加速度定义为角速度对时间的导数：

$$\alpha = \frac{d\omega}{dt} = \frac{d^2\theta}{dt^2}$$

单位为 rad/s²。角加速度与切向线加速度 $a_t$ 的关系为：

$$a_t = r\alpha$$

需要注意这里是**切向**加速度，而圆周运动中还存在指向圆心的向心加速度（法向加速度）：

$$a_n = \omega^2 r = \frac{v^2}{r}$$

两者方向互相垂直，合加速度大小为 $a = \sqrt{a_t^2 + a_n^2}$。

### 匀角加速运动的运动学方程

当角加速度 α 为常数时，转动运动学方程与匀变速直线运动方程形式完全对应：

| 直线运动 | 转动运动 |
|---|---|
| $v = v_0 + at$ | $\omega = \omega_0 + \alpha t$ |
| $x = v_0 t + \frac{1}{2}at^2$ | $\theta = \omega_0 t + \frac{1}{2}\alpha t^2$ |
| $v^2 = v_0^2 + 2ax$ | $\omega^2 = \omega_0^2 + 2\alpha\theta$ |

以一辆摩托车发动机为例：从静止（ω₀ = 0）起在 4 秒内转速达到 3000 rpm（ω = 100π rad/s），其角加速度 α = (100π - 0)/4 = 25π ≈ 78.5 rad/s²，这4秒内转过角位移 θ = ½ × 25π × 16 = 200π rad，约转了100圈。

## 实际应用

**工业机械：齿轮传动系统**
两个啮合齿轮的接触点线速度相同，因此 $r_1\omega_1 = r_2\omega_2$，传动比等于半径之比的倒数。当主动齿轮半径 r₁ = 5 cm、从动齿轮半径 r₂ = 20 cm 时，从动轮角速度为主动轮的 1/4，这是减速器的工作原理。

**汽车制动分析**
汽车紧急刹车时，车轮从 ω₀ = 80 rad/s 减速到 0，制动距离对应的车轮总转角 θ 可用 $\omega^2 = \omega_0^2 + 2\alpha\theta$ 计算。若 α = -400 rad/s²，则 θ = 8 rad，轮胎转过约 1.27 圈完成制动，对应路面制动距离 = rθ（r 为轮胎半径）。

**天文观测：地球自转**
地球自转角速度 ω = 2π/86400 ≈ 7.27 × 10⁻⁵ rad/s。赤道处（r = 6.37 × 10⁶ m）的线速度 v = rω ≈ 463 m/s，这正是为什么航天发射场尽可能选在低纬度地区——可借助地球自转的线速度节省燃料。

## 常见误区

**误区一：混淆角速度与转速**
很多学生将"转速 n = 3000 rpm"直接当作角速度使用。实际上 n 的单位是转/分钟，必须转换为 ω = 2πn/60（单位 rad/s）才能代入运动学公式。若直接用 n = 3000 代入 v = rω，会导致结果偏大约 628 倍（2π×60的因子）。

**误区二：认为角加速度与角速度同向则加速**
α 和 ω 同号时物体确实在角速度增大方向加速，但需注意符号参考方向的统一性。若 ω = −20 rad/s（顺时针），α = −5 rad/s²（同为顺时针），则物体顺时针加速，而非减速——这与直线运动中"负加速度一定减速"的错误类比相同。

**误区三：切向加速度代表全部加速度**
匀角加速转动中 α 为常数，但这并不意味着圆周上各点的总加速度不变。法向加速度 $a_n = \omega^2 r$ 随 ω 的增大而增大，因此即使 α 不变，合加速度 $\sqrt{(r\alpha)^2 + (\omega^2 r)^2}$ 也在持续增大，方向也在持续偏转。

## 知识关联

**与圆周运动的联系**
圆周运动课程中已建立线速度 v、向心加速度 $a_n = v^2/r$ 的概念，转动运动学将这些线量与角量通过 v = rω、$a_n = ω²r$ 精确对应，补充了切向加速度 $a_t = rα$ 的描述，从而构成完整的圆周运动动力学准备。

**向转动动力学的过渡**
转动运动学仅描述"如何运动"，不追问"为何如此运动"。下一步的转动动力学引入转动惯量 $I = \sum m_i r_i^2$ 和力矩 τ，通过牛顿第二定律的转动形式 τ = Iα，将本章的角加速度 α 与外力矩直接关联，形成完整的转动力学体系。理解 α 的运动学定义，是正确使用 τ = Iα 的前提。
