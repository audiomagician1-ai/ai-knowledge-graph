# 量子角动量

## 概述

量子角动量是描述微观粒子旋转运动状态的物理量，但它与经典力学中的角动量 **L = r × p** 存在根本性差异：量子角动量的大小和分量只能取离散值，这一量子化特性由薛定谔方程对球坐标系中旋转对称问题的本征值求解直接导出。量子角动量分为两类：轨道角动量（源于粒子的空间运动）和自旋角动量（粒子内禀属性，无经典对应）。

量子角动量的理论框架由玻尔在1913年提出角动量量子化假设（**L = nħ**）后逐步发展，海森堡和狄拉克在1920年代用代数对易关系取代了直观图像，给出了现代量子力学的角动量算符理论。轨道角动量量子化不仅解释了氢原子光谱的分立谱线，更为理解化学键、原子结构和核物理提供了基本框架。

## 核心原理

### 角动量算符与对易关系

量子力学中，轨道角动量三个分量算符 $\hat{L}_x$、$\hat{L}_y$、$\hat{L}_z$ 满足以下对易关系：

$$[\hat{L}_x, \hat{L}_y] = i\hbar\hat{L}_z, \quad [\hat{L}_y, \hat{L}_z] = i\hbar\hat{L}_x, \quad [\hat{L}_z, \hat{L}_x] = i\hbar\hat{L}_y$$

这三个关系可以统一写成 $\hat{\mathbf{L}} \times \hat{\mathbf{L}} = i\hbar\hat{\mathbf{L}}$。由于三个分量两两不对易，它们不能同时具有确定值，这是量子角动量区别于经典角动量的关键点。然而，总角动量平方算符 $\hat{L}^2 = \hat{L}_x^2 + \hat{L}_y^2 + \hat{L}_z^2$ 与任意分量算符对易：$[\hat{L}^2, \hat{L}_z] = 0$，因此可以同时精确测量 $L^2$ 和 $L_z$。

### 量子数与本征值

对 $\hat{L}^2$ 和 $\hat{L}_z$ 联立求解本征方程，得到两套量子数：

- **角量子数** $l = 0, 1, 2, 3, \ldots$（对应 s, p, d, f 轨道）
- **磁量子数** $m_l = -l, -l+1, \ldots, 0, \ldots, l-1, l$，共 $2l+1$ 个取值

对应本征值为：

$$\hat{L}^2 \psi = l(l+1)\hbar^2 \psi, \quad \hat{L}_z \psi = m_l \hbar \psi$$

注意总角动量大小为 $\sqrt{l(l+1)}\hbar$，而**不是** $l\hbar$。以 $l=1$ 的 p 轨道为例，角动量大小为 $\sqrt{2}\hbar \approx 1.414\hbar$，而 $L_z$ 只能取 $-\hbar, 0, +\hbar$ 三个值，角动量矢量不能完全对齐 z 轴，这一"空间量子化"现象在史特恩-格拉赫实验（1922年）中被直接验证。

### 角动量耦合与 Clebsch-Gordan 系数

当系统含有两个角动量 $\mathbf{J}_1$（量子数 $j_1$）和 $\mathbf{J}_2$（量子数 $j_2$）时，总角动量 $\mathbf{J} = \mathbf{J}_1 + \mathbf{J}_2$ 的量子数 $j$ 取值受**三角不等式**限制：

$$|j_1 - j_2| \leq j \leq j_1 + j_2$$

且 $j$ 以整数步长递增。例如，两个 $l=1$ 的轨道角动量耦合，可以得到 $j = 0, 1, 2$，对应的态空间维度从 $3 \times 3 = 9$ 保持不变，但分解为 $1 + 3 + 5 = 9$ 个本征态。从非耦合基 $|j_1, m_1\rangle|j_2, m_2\rangle$ 到耦合基 $|j, m\rangle$ 的变换系数称为 **Clebsch-Gordan（CG）系数**，记为 $\langle j_1 m_1 j_2 m_2 | j m \rangle$，这些系数可查标准表格或用递推关系计算，在原子物理和粒子物理的跃迁矩阵元计算中大量出现。

## 实际应用

**氢原子能级精细结构**：电子的自旋角动量 $s=1/2$ 与轨道角动量 $l$ 耦合（自旋-轨道耦合），产生总角动量 $j = l \pm 1/2$。这导致氢原子 $n=2, l=1$ 的能级分裂为 $j=3/2$（$2p_{3/2}$）和 $j=1/2$（$2p_{1/2}$）两条，能量差约为 $4.5 \times 10^{-5}$ eV，对应钠黄线的双线结构（589.0 nm 和 589.6 nm）。

**核磁共振（NMR）**：$^1$H 核的自旋角动量量子数 $I = 1/2$，在外磁场中 $m_I = \pm 1/2$ 两能级发生塞曼分裂，跃迁频率（拉莫尔频率）为 $\nu = \gamma B_0 / 2\pi$，其中 $\gamma$ 为旋磁比。MRI 技术正是利用这一量子化的核自旋角动量实现成像。

**原子多重态命名**：光谱项符号 $^{2S+1}L_J$ 直接对应角动量耦合结果，其中 $2S+1$ 为自旋多重度，$L$ 为总轨道角动量量子数（用字母 S, P, D, F 表示），$J$ 为总角动量量子数。例如碳基态的光谱项为 $^3P_0$，表示 $S=1, L=1, J=0$。

## 常见误区

**误区一：认为角动量大小等于 $l\hbar$**。许多初学者沿用玻尔模型的结论，将角动量大小写为 $l\hbar$。实际上波动力学给出的本征值是 $\sqrt{l(l+1)}\hbar$，当 $l \gg 1$ 时两者近似相等，但对 $l=1$ 误差高达 41%。玻尔的 $l\hbar$ 公式是早期量子论的近似，已被现代量子力学修正。

**误区二：认为可以同时知道 $L_x$、$L_y$、$L_z$ 的精确值**。由不对易关系 $[\hat{L}_x, \hat{L}_y] = i\hbar\hat{L}_z$，当 $\hat{L}_z$ 的期望值不为零时，$\hat{L}_x$ 和 $\hat{L}_y$ 满足不确定关系 $\Delta L_x \cdot \Delta L_y \geq \frac{\hbar}{2}|\langle \hat{L}_z \rangle|$。只有在 $m_l = 0$ 的态下，三个分量的期望值才全为零（但仍不能同时确定）。

**误区三：混淆轨道角动量和自旋角动量的量子数取值规则**。轨道角动量量子数 $l$ 只能取非负整数（0, 1, 2, …），因为它来源于球谐函数的单值性约束；而自旋角动量的量子数 $s$ 可以取半整数（如电子 $s = 1/2$，$\Delta^{++}$ 粒子 $s = 3/2$），这是粒子的内禀属性，不受波函数单值性约束。

## 知识关联

**前置概念——算符与可观测量**：量子角动量的整个讨论建立在厄米算符本征值理论之上。$\hat{L}^2$ 和 $\hat{L}_z$ 之所以可同时测量，根本原因是它们对易从而存在共同本征态；对易子 $[\hat{L}_x, \hat{L}_y] = i\hbar\hat{L}_z$ 直接给出对应的不确定性关系，这正是算符代数的具体应用。

**后续概念——选择定则**：角动量量子数的变化规则直接制约了原子跃迁的允许与禁止。电偶极跃迁的选择定则 $\Delta l = \pm 1$、$\Delta m_l = 0, \pm 1$ 正是由角动量守恒（光子携带角动量 $\hbar$）和 CG 系数非零条件共同推导出来的。没有量子角动量的量子化结构，选择定则无从建立，原子光谱的谱线强度和偏振规律也无法解释。
