# 有限深方势阱

## 概述

有限深方势阱（Finite Square Well）是量子力学中一个经典势能模型，其势能函数定义为：在阱宽区间 $-a \leq x \leq a$ 内 $V(x) = -V_0$（$V_0 > 0$），在阱外区域 $|x| > a$ 时 $V(x) = 0$。与无限深方势阱的硬壁边界不同，有限深方势阱允许粒子波函数在阱壁处以指数形式衰减渗入经典禁区，这一特性使其成为理解量子隧穿的直接前驱模型。

历史上，有限深方势阱的分析随薛定谔方程在1926年建立后迅速展开。它是理解核势阱、半导体量子阱（如GaAs/AlGaAs异质结构）以及氘核束缚态（$V_0 \approx 35\ \text{MeV}$，$a \approx 1\ \text{fm}$）的理论基础。不同于无限深势阱的无限多束缚态，有限深方势阱的束缚态数目有限，且至少存在一个束缚态——这是量子力学的重要定理之一。

有限深方势阱的核心价值在于同时展现了**束缚态**（$-V_0 < E < 0$）与**散射态**（$E > 0$）两类物理场景。束缚态对应粒子被束缚于势阱区域内，而散射态则涉及透射系数的共振增强现象。掌握这一模型，是理解真实物理系统（如核物理束缚、固体中电子带结构）中量子效应的关键一步。

## 核心原理

### 束缚态的波函数结构与量子化条件

对于束缚态（$-V_0 < E < 0$），令 $\kappa = \sqrt{-2mE}/\hbar > 0$，$k = \sqrt{2m(E+V_0)}/\hbar > 0$，薛定谔方程的解分三段：

- 阱外左侧（$x < -a$）：$\psi = A e^{\kappa x}$
- 阱内（$|x| \leq a$）：$\psi = B\cos(kx)$（偶宇称）或 $\psi = B\sin(kx)$（奇宇称）
- 阱外右侧（$x > a$）：$\psi = A e^{-\kappa x}$

利用 $x = a$ 处波函数及其导数的连续性条件，偶宇称态满足：

$$k \tan(ka) = \kappa$$

奇宇称态满足：

$$-k \cot(ka) = \kappa$$

引入无量纲变量 $z = ka$，$z_0 = \frac{a}{\hbar}\sqrt{2mV_0}$，这两个超越方程化为：

$$\tan z = \sqrt{(z_0/z)^2 - 1} \quad \text{（偶宇称）}$$

$$-\cot z = \sqrt{(z_0/z)^2 - 1} \quad \text{（奇宇称）}$$

束缚态的数目由 $z_0$ 的大小决定：每当 $z_0$ 越过 $n\pi/2$（$n = 1, 2, 3, \ldots$）时，增加一个新的束缚态。即使 $V_0$ 极小（$z_0 \to 0^+$），偶宇称方程总有一个解，保证至少一个束缚态存在。

### 波函数渗透与指数衰减

有限深方势阱最显著区别于无限深势阱的特征：粒子在阱外的存在概率非零。波函数在阱外以 $e^{-\kappa|x|}$ 形式衰减，特征穿透深度为：

$$\delta = \frac{1}{\kappa} = \frac{\hbar}{\sqrt{-2mE}}$$

当束缚能接近势阱顶部（$E \to 0^-$）时，$\kappa \to 0$，穿透深度 $\delta \to \infty$，波函数在阱外延伸极广。这与无限深势阱中波函数严格为零的硬壁条件截然不同，是量子隧穿效应的物理先兆。

### 散射态与透射系数的共振现象

对于散射态（$E > 0$），入射粒子可能被部分反射，也可能完全透射。定义阱内波矢 $k_1 = \sqrt{2m(E+V_0)}/\hbar$，阱外波矢 $k_2 = \sqrt{2mE}/\hbar$，透射系数为：

$$T = \left[1 + \frac{V_0^2 \sin^2(2k_1 a)}{4E(E+V_0)}\right]^{-1}$$

当 $\sin(2k_1 a) = 0$，即 $2k_1 a = n\pi$（$n$ 为正整数）时，$T = 1$，发生**完全透射共振（Ramsauer-Townsend共振）**。这要求阱宽恰好等于阱内德布罗意半波长的整数倍：$2a = n\lambda_1/2 = n\pi\hbar/k_1$。1921年Ramsauer和Townsend分别在实验中发现惰性气体对低能电子的异常高透射率，正是此共振效应在三维势阱中的体现。

## 实际应用

**半导体量子阱器件**：GaAs/Al$_x$Ga$_{1-x}$As异质结构中，GaAs层形成有限深方势阱（典型深度约200–300 meV，阱宽5–20 nm）。电子在阱内的量子化能级间跃迁产生激光发射，量子阱激光器的发射波长由有限深方势阱的束缚态能级公式精确控制。

**核物理中的氘核模型**：用有限深方势阱模拟核力，氘核（质子-中子体系）只有一个束缚态（$J^\pi = 1^+$，结合能 $E_B = 2.225\ \text{MeV}$）。这与氘核的 $z_0$ 值仅略大于 $\pi/2$ 相对应，说明核力恰好足以维持唯一的束缚态。

**扫描隧道显微镜（STM）的势垒分析**：STM的针尖与样品之间的真空间隙等效为有限高势垒，其物理描述与有限深方势阱的阱外指数衰减区完全对应。隧道电流 $I \propto e^{-2\kappa d}$（$d$ 为间隙宽度）直接源于束缚态阱外渗透的数学结构。

## 常见误区

**误区一：认为有限深方势阱总有无限多束缚态**  
无限深方势阱有无限多个束缚态（$E_n = n^2 \pi^2 \hbar^2 / 2ma^2$），但有限深方势阱的束缚态数目是有限的，由 $z_0 = a\sqrt{2mV_0}/\hbar$ 决定。仅当 $V_0 \to \infty$ 时才退化为无限深势阱的结论。很多初学者将两者混淆，错误地对有限深阱应用无限深阱的能级公式。

**误区二：散射态的透射系数总小于1，且在共振点外单调变化**  
实际上透射系数随入射能量非单调变化，在共振点（$2k_1 a = n\pi$）处 $T = 1$，而非仅在 $E \gg V_0$ 时才接近1。初学者常认为势垒（正值势能）才有透射振荡，而忽略势阱（负值势能）也能引发Ramsauer-Townsend型的完全透射共振。

**误区三：边界条件只要求波函数连续，不要求导数连续**  
无限深方势阱中，由于势能在阱壁处为无穷大，导数可以不连续。但在有限深方势阱中，势能处处有限，因此**波函数与其一阶导数在所有点均须连续**。遗漏导数连续性条件会导致错误的超越方程，从而算出错误的能级。

## 知识关联

**前置概念——无限深方势阱**：有限深方势阱的束缚态波函数在阱内仍为正弦/余弦形式，与无限深势阱一致，但量子化条件从 $k \cdot 2a = n\pi$（硬壁）变为需求解超越方程。能级值也因渗透效应而低于同参数无限深阱的对应能级，这是有限深阱波函数的"有效宽度"大于 $2a$ 的直接后果。

**后续概念——量子隧穿**：有限深方势阱中阱外的指数衰减波函数 $\psi \propto e^{-\kappa x}$ 是量子隧穿的数学原型。量子隧穿本质上是将阱外衰减区（阱的延伸）改为有限厚势垒：粒子的波函数在势垒中同样指数衰减，但若势垒宽度有限，波函数在对侧仍有非零振幅，从而产生透射。有限深方势阱散射态的透射系数公式与单势垒隧穿公式在数学结构上直接相关。
