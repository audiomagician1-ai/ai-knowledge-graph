# 薛定谔方程

## 概述

薛定谔方程（Schrödinger Equation）是量子力学的基本运动方程，由埃尔温·薛定谔于1926年提出。它在量子力学中的地位，相当于牛顿第二定律在经典力学中的地位——给定初始条件和势能函数，薛定谔方程完全确定了量子系统的演化。薛定谔因此获得1933年诺贝尔物理学奖。

经典力学用确定的轨迹（位置和速度）描述粒子，而量子力学用**波函数** $\Psi(\vec{r}, t)$ 描述粒子的状态。波函数本身不是可直接测量的物理量，但它的模方 $|\Psi|^2$ 给出粒子在各位置出现的概率密度。薛定谔方程就是决定波函数如何随时间和空间变化的方程。

学习薛定谔方程前需要掌握波粒二象性、德布罗意关系（$\lambda = h/p$）和玻尔模型的局限性。掌握薛定谔方程后，可以进入量子力学的核心应用：氢原子精确解、量子隧穿、谐振子、角动量量子化等。

## 核心知识点

### 含时薛定谔方程

量子力学的基本方程（一维形式）：

$$i\hbar\frac{\partial}{\partial t}\Psi(x,t) = \left[-\frac{\hbar^2}{2m}\frac{\partial^2}{\partial x^2} + V(x,t)\right]\Psi(x,t)$$

三维推广：

$$i\hbar\frac{\partial}{\partial t}\Psi(\vec{r},t) = \hat{H}\Psi(\vec{r},t)$$

其中哈密顿算符 $\hat{H} = -\frac{\hbar^2}{2m}\nabla^2 + V(\vec{r},t)$，$\hbar = h/(2\pi) = 1.055 \times 10^{-34}$ J·s 是约化普朗克常数。

关键特征：
- 方程是**线性的**：若 $\Psi_1$ 和 $\Psi_2$ 是解，则 $c_1\Psi_1 + c_2\Psi_2$ 也是解（叠加原理）
- 方程是**一阶时间偏微分方程**：给定初始波函数 $\Psi(\vec{r}, 0)$，未来所有时刻的波函数完全确定
- 方程中出现虚数 $i$：波函数一般是复数值的

### 定态薛定谔方程

当势能不显含时间 $V = V(\vec{r})$ 时，可以分离变量 $\Psi(\vec{r},t) = \psi(\vec{r})e^{-iEt/\hbar}$，得到定态方程：

$$\hat{H}\psi(\vec{r}) = E\psi(\vec{r})$$

这是一个**本征值方程**：能量 $E$ 是哈密顿算符的本征值，$\psi$ 是本征函数。

经典应用——无限深势阱（一维，宽度 $L$）：
- 本征能量：$E_n = \frac{n^2\pi^2\hbar^2}{2mL^2}$，$n = 1, 2, 3, \ldots$
- 能量是量子化的（只取离散值），基态能量 $E_1 > 0$（零点能）
- 本征函数：$\psi_n(x) = \sqrt{2/L}\sin(n\pi x/L)$

### 波函数的物理意义

玻恩的统计诠释：$|\Psi(\vec{r},t)|^2 d^3r$ 是在时刻 $t$ 于 $\vec{r}$ 处的体积元 $d^3r$ 中发现粒子的概率。

由此产生的约束：
- **归一化条件**：$\int_{-\infty}^{+\infty}|\Psi|^2 d^3r = 1$（粒子必须在某处）
- **连续性**：波函数及其一阶导数必须连续（除了势能无穷大处）
- **有限性**：波函数在无穷远处趋于零（束缚态）

### 概率流守恒

薛定谔方程保证概率守恒。定义概率流密度：

$$\vec{j} = \frac{\hbar}{2mi}\left(\Psi^*\nabla\Psi - \Psi\nabla\Psi^*\right)$$

满足连续性方程 $\frac{\partial\rho}{\partial t} + \nabla\cdot\vec{j} = 0$，其中 $\rho = |\Psi|^2$。

## 关键要点

1. **薛定谔方程是公设**：它不能从更基本的原理推导出来，其正确性由实验验证。类似于牛顿第二定律在经典力学中的地位
2. **方程是确定性的，不确定性来自测量**：给定初始波函数，薛定谔方程唯一确定未来所有时刻的波函数。"量子不确定性"体现在测量结果是概率性的，而非方程本身的演化
3. **能量量子化是边界条件的结果**：并非人为假设能量是离散的，而是可接受的波函数（归一化、连续、有限）只在特定能量值下存在
4. **非相对论近似**：薛定谔方程不满足洛伦兹协变性，不适用于接近光速的粒子。相对论修正版本是狄拉克方程（自旋-1/2 粒子）和克莱因-戈尔登方程（自旋-0 粒子）

## 常见误区

1. **"波函数就是粒子的轨迹"**：波函数描述的是概率分布，不是粒子的运动路径。粒子没有经典意义上的轨迹。$|\Psi|^2$ 告诉我们"粒子可能在哪里被发现"
2. **"薛定谔方程只能解氢原子"**：它是通用的量子运动方程，适用于所有非相对论量子系统——从自由粒子、势阱、谐振子到多电子原子和分子。只是大多数复杂系统无法精确求解，需要近似方法
3. **"零点能违反能量守恒"**：基态能量 $E_1 > 0$ 是不确定性原理的直接后果。将粒子完全限制在有限空间中，位置不确定度有限，因此动量不确定度非零，对应非零动能

## 知识衔接

**先修概念**：波粒二象性、德布罗意关系（$p = h/\lambda$）、光电效应、玻尔模型、不确定性原理

**后续概念**：
- **量子隧穿**：粒子穿越经典禁区的概率（定态方程在势垒中的指数衰减解）
- **氢原子**：三维球坐标下的精确解，引出主量子数/角量子数/磁量子数
- **量子谐振子**：$V = \frac{1}{2}kx^2$，等间距能级 $E_n = (n+\frac{1}{2})\hbar\omega$
- **自旋**：薛定谔方程不包含自旋，需要泡利方程或狄拉克方程
- **多粒子系统**：全同粒子、泡利不相容原理、量子统计
