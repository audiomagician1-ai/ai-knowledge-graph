# 交流电路

## 概述

交流电路（Alternating Current Circuit）是指电流方向和大小随时间周期性变化的电路系统。与直流电路中电流恒定不变不同，交流电路中的电压和电流遵循正弦函数规律变化，标准表达式为 $u = U_m \sin(\omega t + \varphi)$，其中 $U_m$ 为峰值电压，$\omega$ 为角频率，$\varphi$ 为初相位。中国大陆民用电网采用频率 50 Hz、有效值 220 V 的正弦交流电，这意味着峰值电压约为 311 V。

交流电的大规模应用始于19世纪末。1888年，尼古拉·特斯拉（Nikola Tesla）完善了交流发电机和变压器系统，由此引发了著名的"电流之战"——直流拥护者爱迪生与交流拥护者威斯汀豪斯之间的商业竞争。交流电最终胜出，核心原因在于变压器可以将交流电升压至数十万伏特进行远距离输电，从而大幅降低线路能量损耗（损耗 $P = I^2 R$，升压减小电流 $I$，损耗随之骤降）。

在高中物理和大学电磁学课程中，交流电路的分析涉及阻抗、相位差、谐振等概念，这些概念无法用纯直流电路的欧姆定律直接处理，需要引入复数或相量图方法。

## 核心原理

### 交流电的基本量

描述交流电需要四个关键物理量：

- **峰值（最大值）**：$U_m$ 或 $I_m$，是正弦波的振幅。
- **有效值（均方根值，RMS）**：对纯正弦波，$U = \dfrac{U_m}{\sqrt{2}} \approx 0.707 U_m$，$I = \dfrac{I_m}{\sqrt{2}}$。有效值的物理意义是：与直流电路产生相同热效应所对应的等效电压/电流值。220 V 市电即为有效值。
- **频率与角频率**：频率 $f$（单位 Hz）与角频率 $\omega = 2\pi f$（单位 rad/s）的关系。50 Hz 对应 $\omega \approx 314\ \text{rad/s}$。
- **相位与初相位**：描述正弦波在时间轴上的"起点"，决定了不同元件电压与电流之间的超前/滞后关系。

### 纯电阻、纯电感、纯电容的相位关系与阻抗

交流电路中，三种基本元件对电流的"阻碍"性质各不相同：

| 元件 | 阻抗（模） | 电压与电流的相位关系 |
|------|-----------|---------------------|
| 纯电阻 $R$ | $Z_R = R$ | 同相位（$\varphi = 0$） |
| 纯电感 $L$ | 感抗 $X_L = \omega L$ | 电压超前电流 90° |
| 纯电容 $C$ | 容抗 $X_C = \dfrac{1}{\omega C}$ | 电流超前电压 90°（电压滞后电流 90°） |

感抗 $X_L$ 随频率升高而增大，因此电感对高频信号有强烈阻碍作用（高频截止特性）；容抗 $X_C$ 随频率升高而减小，电容则阻碍低频信号（低频截止特性）。这是滤波器设计的物理基础。

### RLC 串联电路与谐振

将电阻 $R$、电感 $L$、电容 $C$ 依次串联后接交流电源，总阻抗为：

$$Z = \sqrt{R^2 + (X_L - X_C)^2} = \sqrt{R^2 + \left(\omega L - \frac{1}{\omega C}\right)^2}$$

电路中电流有效值：$I = \dfrac{U}{Z}$

电源电压与电流的相位差：$\tan\varphi = \dfrac{X_L - X_C}{R}$

**谐振条件**：当 $X_L = X_C$，即 $\omega L = \dfrac{1}{\omega C}$ 时，$(X_L - X_C) = 0$，阻抗达到最小值 $Z_{\min} = R$，电流达到最大值 $I_{\max} = \dfrac{U}{R}$。此时的角频率称为谐振角频率：

$$\omega_0 = \frac{1}{\sqrt{LC}}$$

对应的谐振频率：$f_0 = \dfrac{1}{2\pi\sqrt{LC}}$

谐振时，电感两端电压 $U_L = I \cdot X_L$ 和电容两端电压 $U_C = I \cdot X_C$ 可以**远大于**电源电压 $U$，这种现象称为**电压谐振**（串联谐振）。品质因数 $Q = \dfrac{\omega_0 L}{R} = \dfrac{1}{\omega_0 CR} = \dfrac{1}{R}\sqrt{\dfrac{L}{C}}$ 描述谐振的"锐利程度"，$Q$ 越大，谐振峰越尖锐，选频能力越强。

## 实际应用

**收音机调谐电路**：收音机内置 RLC 串联回路，通过旋转旋钮改变电容 $C$ 的容值，使 $f_0 = \dfrac{1}{2\pi\sqrt{LC}}$ 等于目标广播频率（如调至 990 kHz），该频率的信号在谐振时电流最大，其余频率被大幅抑制，实现选台。

**电力系统功率因数校正**：工厂中大量感性负载（电动机）导致电压超前电流，功率因数 $\cos\varphi < 1$，使电网输送的视在功率远大于实际有功功率，造成浪费。电力公司要求用户并联电容器以补偿感抗，提高 $\cos\varphi$（通常要求不低于 0.9），减少电流，降低线路损耗。

**医学中的电外科设备**：高频手术刀利用 400 kHz 至 3 MHz 的高频交流电通过人体组织时产生的热效应切割和凝血，而非引起神经肌肉电刺激（神经刺激发生在 50 Hz 附近），这正是利用了容抗在高频时极小、感抗在高频时极大的频率特性。

## 常见误区

**误区一：交流电路中各元件电压可以直接相加**

在 RLC 串联电路中，$R$、$L$、$C$ 各自两端的有效值电压**不能**直接代数求和。例如，若 $U_R = 3\ \text{V}$，$U_L = 8\ \text{V}$，$U_C = 4\ \text{V}$，则电源电压并非 $3 + 8 + 4 = 15\ \text{V}$，而应用相量合成：$U = \sqrt{U_R^2 + (U_L - U_C)^2} = \sqrt{9 + 16} = 5\ \text{V}$。直接相加是把有效值当成瞬时值的错误，忽略了相位差。

**误区二：谐振时电路中"没有电抗"**

串联谐振时总阻抗确实等于纯电阻 $R$，电路呈纯阻性，但这并不意味着电感和电容消失。此时 $X_L = X_C$ 两者相互抵消，$U_L$ 和 $U_C$ 大小相等、方向相反，但各自的电压绝对值可能比电源电压大数倍甚至数十倍（取决于 $Q$ 值）。忽视这一点会导致在高 $Q$ 电路设计中选用耐压不足的电容，引发击穿事故。

**误区三：有效值公式适用于所有交流电波形**

$U = \dfrac{U_m}{\sqrt{2}}$ 仅对**正弦波**成立。对方波，有效值等于峰值；对三角波，有效值为 $\dfrac{U_m}{\sqrt{3}}$。工频市电为标准正弦波，但变频电源、逆变器输出的非正弦波若误用此公式会产生计算错误。

## 知识关联

**与直流电路的联系与区别**：直流电路中欧姆定律 $I = U/R$ 仅涉及电阻，而交流电路中将其推广为 $I = U/Z$，阻抗 $Z$ 综合了电阻、感抗和容抗三部分。直流电路中电感相当于短路（稳态时 $X_L = 0$），电容相当于断路（稳态时不通直流），这是判断直流稳态的基础。

**与电感和电容单元的连接**：电感的自感系数 $L$（单位 H）决定感抗大小 $X_L = \omega L$；电容量 $C$（单位 F）决定容抗大小 $X_C = 1/(\omega C)$。单独学习电感、电容时仅需掌握其储能特性，进入交流电路后必须同时考虑它们的频率响应特性，两者共同构成谐振的物理基础。
