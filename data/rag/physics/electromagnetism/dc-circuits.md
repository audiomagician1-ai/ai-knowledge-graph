# 直流电路

## 概述

直流电路（Direct Current Circuit）是指电流方向不随时间变化的电路系统，电荷沿固定方向在导体中持续流动。与交流电路不同，直流电路中的电动势源（如电池、直流发电机）维持恒定的电压极性，电子始终从负极流向正极（物理学定义的电流方向从正极流向负极）。最早的系统性直流电路研究由乔治·欧姆（Georg Simon Ohm）于1827年完成，他通过实验建立了电压、电流与电阻之间的定量关系。

直流电路是电磁学中最基础的电路模型，托马斯·爱迪生的第一套商业供电系统（1882年纽约珍珠街电站）正是直流电路的大规模实用化案例，输出电压为110V直流电。理解直流电路的工作规律，是分析更复杂电路系统（如RC暂态电路、非线性电路）的必要前提，同时也是理解欧姆定律、基尔霍夫定律等核心定律的应用场景。

## 核心原理

### 欧姆定律

欧姆定律是直流电路的基本定量规律，表达式为：

**U = IR**

其中 U 为导体两端电压（单位：伏特 V），I 为通过导体的电流（单位：安培 A），R 为导体的电阻（单位：欧姆 Ω）。该定律只适用于纯电阻元件（线性元件），对于二极管、灯泡灯丝等非线性元件，U-I 关系不成立。

对于一段完整含电源的电路，欧姆定律扩展为**全电路欧姆定律**：

**I = ε / (R + r)**

其中 ε 为电源电动势（V），R 为外电路总电阻（Ω），r 为电源内阻（Ω）。内阻的存在导致路端电压 U = ε - Ir 始终小于电动势，这是实际电源与理想电源的核心区别。

### 串联与并联电路规律

**串联电路**中，各电阻首尾相连，电流处处相等（I₁ = I₂ = I₃），各电阻分担电压，总电阻为各分电阻之和：

**R_串 = R₁ + R₂ + R₃ + …**

各电阻两端电压与其阻值成正比（分压原理）：U₁/U₂ = R₁/R₂。

**并联电路**中，各电阻两端电压相等（U₁ = U₂ = U₃），总电流等于各支路电流之和，总电阻倒数等于各分电阻倒数之和：

**1/R_并 = 1/R₁ + 1/R₂ + 1/R₃ + …**

两个电阻并联时，总电阻 R = R₁R₂/(R₁+R₂)，该值永远小于其中任意一个分电阻。各支路电流与其阻值成反比（分流原理）：I₁/I₂ = R₂/R₁。

### 基尔霍夫定律在直流电路中的应用

对于包含多个电源和多条支路的复杂直流电路，仅用欧姆定律无法直接求解，需引入**基尔霍夫两大定律**：

**基尔霍夫电流定律（KCL）**：在任意节点处，流入节点的电流之和等于流出节点的电流之和，即 ΣI_入 = ΣI_出。这反映了电荷在节点处不积累的物理事实。

**基尔霍夫电压定律（KVL）**：沿任意闭合回路一周，各段电压的代数和为零，即 ΣU = 0（或等价表述：回路中所有电动势之和等于所有电阻压降之和）。这是能量守恒定律在直流电路中的体现。

对于一个具有 n 个节点、b 条支路的直流电路，独立的KCL方程数为 (n-1) 个，独立的KVL方程数为 b-(n-1) 个，两类方程合计恰好等于 b，足以求解所有支路电流。

### 电功率与焦耳定律

直流电路中，电阻消耗的电功率为：

**P = UI = I²R = U²/R**

电阻在时间 t 内产生的热量由**焦耳定律**给出：

**Q = I²Rt**

注意 P = UI 适用于任意电路元件（包括电机等），而 Q = I²Rt 仅适用于纯电阻元件，两者在含非纯电阻元件的电路中并不等价。

## 实际应用

**分压器与分流器电路**：利用串联分压原理，可将一个12V电源分成多个电压输出，例如阻值为3Ω和9Ω的电阻串联时，9Ω电阻两端获得9V电压，用于为不同器件供电。万用表内部即通过串联大阻值电阻（扩程电阻）来测量高电压，通过并联小阻值电阻（分流电阻）来测量大电流。

**汽车直流电路系统**：现代汽车采用12V（或商用车24V）直流电路系统，蓄电池内阻约为0.02~0.05Ω。起动电机工作时电流可达200A以上，此时内阻压降达到4~10V，路端电压明显下降，这正是冬季低温时（内阻增大）汽车难以启动的电路原因。

**手电筒串联故障分析**：三节1.5V电池串联提供4.5V电压，若其中一节电池因内阻增大到5Ω（新电池约0.5Ω），而灯泡电阻约为10Ω，则工作电流从原来的约0.29A降至约0.23A，亮度明显减弱，这是直流串联电路中"一损俱损"特性的典型体现。

## 常见误区

**误区一：混淆电动势与路端电压**。很多学生认为电池标注的1.5V或9V就是电路中的实际电压，实际上这是电动势 ε，而实际路端电压 U = ε - Ir 会随外电路电流增大而降低。当外电路短路（R→0）时，路端电压趋近于零，但电流达到最大值 I = ε/r，此时全部电能在内阻上耗散为热量。

**误区二：并联电路"分电流"等于"分电压"**。并联电路中电压相等，电流按阻值反比分配；串联电路中电流相等，电压按阻值正比分配。两者恰好相反，常被混淆。具体地，两支路阻值分别为2Ω和6Ω并联时，2Ω支路通过的电流是6Ω支路的3倍，而非1/3倍。

**误区三：直流电路分析中忽略导线电阻**。在中学题目中，导线被视为理想导体（R=0），但在实际工程电路中，截面积为1mm²、长度10m的铜导线约有0.17Ω电阻。大电流传输中，导线电阻造成的压降和功率损耗不可忽略，这也是高压输电降低损耗（P损 = I²R，升压减小电流）的物理依据。

## 知识关联

**前置概念**：直流电路的分析建立在**电荷**概念之上——电流 I = ΔQ/Δt 本质上是单位时间内通过导体截面的电荷量，电位差（电压）则是驱动电荷定向移动的原因。没有对电荷和电场力的理解，欧姆定律中的 U 和 I 便只是抽象符号。

**后续发展**：掌握直流电路后，引入电容元件即进入**RC电路**的分析——电容充放电过程中电压和电流随时间按指数规律变化（时间常数 τ = RC），这是直流稳态之外的暂态行为，是直流分析向动态分析的第一步延伸。复杂网络电路的系统性求解则需要专门学习**基尔霍夫定律**的矩阵化方法（节点电压法、网孔电流法）。而将电源从直流替换为正弦交变电源，则进入**交流电路**领域，此时电阻、电感、电容的组合行为远比直流电路复杂，需要引入阻抗和相位的概念。
