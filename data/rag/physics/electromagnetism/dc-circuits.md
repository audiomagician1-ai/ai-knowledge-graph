# 直流电路

## 概述

直流电路（DC Circuit）是指电流方向和大小不随时间变化的电路系统。与交流电不同，直流电中电荷始终沿同一方向流动——从高电位流向低电位，形成稳定的单向电流。电池、干电池、光伏电池等都是典型的直流电源，其正负极之间维持着固定的电位差，驱动电路中的自由电子定向移动。

直流电路的理论基础由多位科学家奠定。1826年，乔治·西蒙·欧姆通过大量实验提出了欧姆定律，建立了电压、电流与电阻之间的定量关系。1845年，古斯塔夫·基尔霍夫在欧姆定律的基础上提出了两条回路定律，使复杂直流网络的分析成为可能。这两套定律共同构成了直流电路分析的完整数学工具。

直流电路在现代技术中无处不在：手机电池、汽车电气系统、太阳能发电装置的储能环节以及几乎所有数字电子设备，都依赖稳定的直流电供电。理解直流电路的分析方法，是学习后续RC电路瞬态响应和交流电路分析的必要前提。

## 核心原理

### 欧姆定律

欧姆定律是直流电路最基本的定量规律，表述为：

$$U = IR$$

其中，$U$ 为导体两端的电压（单位：伏特 V），$I$ 为流过导体的电流（单位：安培 A），$R$ 为导体的电阻（单位：欧姆 Ω）。这一关系仅适用于**纯电阻元件**（即欧姆元件），其 $U$-$I$ 图像是过原点的直线，斜率即为电阻值 $R$。

电阻本身由导体材料的物理属性决定，其计算公式为：

$$R = \rho \frac{L}{A}$$

其中 $\rho$ 为电阻率（单位：Ω·m），$L$ 为导体长度，$A$ 为横截面积。铜的电阻率约为 $1.7 \times 10^{-8}\ \Omega\cdot\text{m}$，远低于铁（约 $1.0 \times 10^{-7}\ \Omega\cdot\text{m}$），这也是导线普遍使用铜材料的原因。

### 串联与并联电路

**串联电路**中，各电阻首尾相接，电流无分支，通过每个电阻的电流相同。串联总电阻为：

$$R_{\text{串}} = R_1 + R_2 + \cdots + R_n$$

各电阻两端的电压之和等于总电压，即 $U = U_1 + U_2 + \cdots + U_n$，电压按电阻大小成正比分配（**分压原理**）。

**并联电路**中，各电阻两端电压相同，总电流等于各支路电流之和。并联总电阻满足：

$$\frac{1}{R_{\text{并}}} = \frac{1}{R_1} + \frac{1}{R_2} + \cdots + \frac{1}{R_n}$$

两电阻并联时，等效电阻为 $R_{\text{并}} = \frac{R_1 R_2}{R_1 + R_2}$，结果始终小于任一单个电阻值。家庭用电中各电器并联接在 220V 电路上，正是利用了并联各支路电压相同的特性。

### 基尔霍夫定律

对于含有多个电源和多条支路的**复杂直流网络**，单靠欧姆定律无法直接求解，必须使用基尔霍夫定律。

**基尔霍夫电流定律（KCL）**：在任意节点处，流入节点的电流之和等于流出节点的电流之和：

$$\sum I_{\text{流入}} = \sum I_{\text{流出}}$$

该定律本质上是电荷守恒定律在节点处的体现——稳定直流状态下节点不积累电荷。

**基尔霍夫电压定律（KVL）**：沿任意闭合回路一周，所有元件上电压降的代数和等于零：

$$\sum U = 0$$

实际应用时，需先标注各支路电流方向（可假设），然后对节点列 KCL 方程，对独立回路列 KVL 方程，联立求解。对于含 $b$ 条支路、$n$ 个节点的电路，独立的 KCL 方程有 $n-1$ 个，独立的 KVL 方程有 $b-(n-1)$ 个，两类方程恰好足以求解所有支路电流。

## 实际应用

**手电筒电路分析**：一节 1.5V 干电池接一个 15Ω 灯泡，由欧姆定律得电流 $I = 1.5/15 = 0.1\ \text{A}$，灯泡功率 $P = UI = 1.5 \times 0.1 = 0.15\ \text{W}$。若串联一节额外电阻来限流保护 LED，也可用串联分压原理精确计算限流电阻值。

**汽车起动电路**：汽车电瓶电压通常为 12V，起动电机内阻约 0.1Ω，启动瞬间电流可达 $I = 12/0.1 = 120\ \text{A}$，因此导线必须使用截面积极大的铜缆。这一情形也说明了电源内阻对实际端电压的影响：真实电源的端电压为 $U = \varepsilon - Ir$，其中 $\varepsilon$ 为电动势，$r$ 为内阻，大电流时内阻压降不可忽略。

**电阻分压器设计**：两个电阻串联后接在 9V 电源两端，若需从中间节点取出 3V，则两电阻之比应为 $R_1 : R_2 = 2:1$（$R_1$ 在上，$R_2$ 在下）。这一分压器原理广泛用于模拟电路的基准电压产生。

## 常见误区

**误区一：混淆"电流相同"与"串联"的充分必要关系**

学生常认为只要电流处处相同就一定是串联，但这忽略了测量位置的问题。实际上，只有在同一条无分支导线上，电流才处处相同；两段电路串联时，这两段中的电流确实相等，但若整个回路中存在隐藏的并联支路（如寄生电阻），电流就会分流。分析时必须画出完整电路图，明确节点位置。

**误区二：认为并联中"每个电阻消耗的功率相同"**

并联时各支路电压相同（均等于总电压），但由于各支路电阻不同，电流不同，功率 $P = U^2/R$ 也不同。电阻**越小**的支路，消耗功率越大，流过的电流也越大。这与串联电路中电阻越大分得的**电压**越大恰好相反，两者在"哪个元件承担更多"这一问题上逻辑完全相反，是高频失分点。

**误区三：忽略电源内阻，将路端电压等同于电动势**

理想电源模型中 $U = \varepsilon$，但实际电源有内阻 $r$。当外接负载减小（电流增大）时，内阻上的压降 $Ir$ 增大，路端电压 $U = \varepsilon - Ir$ 下降。测量空载电压（即几乎无电流时）才接近电动势，带负载后路端电压会低于电动势，负载越重，差值越大。

## 知识关联

直流电路的分析建立在**电荷**和**电场力做功**（电位差）的概念之上：电荷在电场中的定向移动构成电流，而正是两点间的电位差（电压）提供了驱动力。没有对电荷与库仑力的理解，欧姆定律中的 $U$、$I$、$R$ 三量的物理意义将无从建立。

学完直流电路后，下一步自然延伸到**RC电路**：在直流电路中引入电容器，电路不再处于稳态，而会出现充放电过程，电流和电压随时间按指数规律 $I(t) = I_0 e^{-t/RC}$ 变化，时间常数 $\tau = RC$ 正是用到了本章的电阻和将要学习的电容两个量。此外，**基尔霍夫定律**在本章中已初步使用，后续将作为独立专题深入讨论，并推广到含复数阻抗的**交流电路**分析中。掌握直流电路中对节点和回路的系统化方程列写方法，直接决定了能否在交流电路中顺利运用相量法。
