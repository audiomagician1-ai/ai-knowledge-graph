# 电荷

## 概述

电荷是物质的一种基本属性，描述粒子与电磁场相互作用的能力。自然界中只存在两种电荷：正电荷和负电荷。同种电荷相互排斥，异种电荷相互吸引。这一规律早在18世纪由本杰明·富兰克林（Benjamin Franklin）命名并系统化——他将玻璃棒摩擦丝绸后产生的电荷定义为"正电荷"，将橡胶棒摩擦毛皮后产生的电荷定义为"负电荷"，这套命名沿用至今。

电荷的国际单位是库仑（C），以法国物理学家查尔斯·库仑（Charles-Augustin de Coulomb）命名。一个电子携带的电荷量为 −1.602×10⁻¹⁹ C，这个数值被称为基本电荷，用符号 *e* 表示。之所以研究电荷具有重要意义，是因为电荷是理解所有电现象（电场、电流、磁场）的逻辑起点——没有电荷，电磁学的整个体系将无从建立。

## 核心原理

### 电荷守恒定律

电荷守恒定律指出：在一个孤立系统中，电荷的总量始终保持不变。电荷既不能凭空创造，也不能凭空消失，只能从一个物体转移到另一个物体。当玻璃棒与丝绸摩擦时，玻璃棒失去电子变为正电，丝绸获得电子变为负电，但二者的总电荷量之和仍为零。在粒子物理学中，正电子与电子发生湮灭时，两个粒子消失，但它们的电荷（+e 与 −e）之和仍为零，电荷守恒定律得以严格验证。

电荷守恒是已知物理学中最精确的守恒定律之一，目前实验限制表明质子的寿命超过 10³⁴ 年，与电荷守恒的预测高度吻合。

### 库仑定律

库仑定律定量描述两个点电荷之间的相互作用力。其公式为：

$$F = k \frac{q_1 q_2}{r^2}$$

其中：
- *F* 为两电荷之间的静电力（单位：牛顿 N）
- *k* 为库仑常数，等于 8.988×10⁹ N·m²/C²
- *q₁*、*q₂* 分别为两个点电荷的电荷量（单位：C）
- *r* 为两点电荷之间的距离（单位：m）

当 *F* 为正值，说明两电荷同号，力为斥力；当 *F* 为负值，说明两电荷异号，力为引力。库仑定律与万有引力定律在数学形式上完全相同（都是平方反比定律），但库仑力可以是排斥力，而引力只能是吸引力。值得注意的是，库仑定律严格成立于真空中的静止点电荷之间；若介质存在，则需将 *k* 替换为 $\frac{1}{4\pi\varepsilon}$，其中 ε 为介质的电容率。

### 电荷量子化

电荷量子化是指自然界中任何物体所携带的电荷量都是基本电荷 *e* = 1.602×10⁻¹⁹ C 的整数倍，即：

$$q = ne \quad (n = 0, \pm1, \pm2, \pm3, \ldots)$$

这一性质由美国物理学家罗伯特·密立根（Robert Millikan）在1909年的油滴实验中得到首次精确测量。密立根通过测量带电油滴在重力和电场力平衡时的状态，精确得出了 *e* 的数值，并因此获得1923年诺贝尔物理学奖。

需要注意的是，夸克粒子携带的电荷为 +2/3 *e* 或 −1/3 *e*，似乎违反了量子化规则，但夸克无法单独存在（色禁闭），因此所有可以被直接观测的自由粒子电荷量仍然是整数倍的 *e*。

## 实际应用

**静电复印（激光打印机）**：激光打印机的硒鼓表面均匀分布正电荷，激光照射使特定区域放电，带负电的碳粉颗粒受异种电荷吸引附着于放电区域，最终转印到纸张上形成文字或图像。整个过程依赖精确控制电荷分布。

**避雷针**：避雷针利用尖端放电原理工作。尖端处电荷密度极高，导致附近空气电离，形成导电通道，将雷云与大地之间的电荷中和，从而保护建筑物。富兰克林于1752年发明了避雷针，这正是他对正负电荷研究的直接应用成果。

**心电图（ECG）**：心肌细胞在收缩和舒张过程中发生去极化，导致细胞内外电荷分布变化，产生微弱电流。心电图仪通过测量体表不同位置之间的电位差（由电荷分布不均引起），记录心脏的电活动规律，帮助诊断心律不齐、心肌梗塞等疾病。

## 常见误区

**误区一：摩擦生电是"创造"了电荷**。实际上，摩擦只是使电荷从一个物体转移到另一个物体，并未产生新的电荷。玻璃棒摩擦后带正电，是因为玻璃棒上的电子被转移到丝绸上，玻璃棒失去电子而非"获得"了正电荷。两者电荷量的绝对值相等、符号相反，总量为零。

**误区二：电荷量可以取任意连续值**。在宏观电路中，由于涉及大量电子（1 C 约等于 6.24×10¹⁸ 个基本电荷），电荷看起来是连续可分的，但微观层面上电荷严格量子化。在纳米器件和单电子晶体管的设计中，必须考虑电荷量子化效应，否则器件分析将出现原则性错误。

**误区三：库仑定律适用于所有带电体**。库仑定律的严格形式只适用于真空中的静止点电荷。当带电体的几何尺寸与二者距离相比不可忽略时（如两个相距较近的带电球体），需要对电荷分布进行积分计算，不能直接使用简单的 $F = kq_1q_2/r^2$ 公式。

## 知识关联

**通往电场**：电荷是电场的来源。任何静止的点电荷 *q* 都会在周围空间产生电场，电场强度 $E = kq/r^2$。学习完电荷的基本性质（尤其是库仑定律）之后，电场可以被理解为"单位正电荷在该点受到的库仑力"，是电荷概念的直接延伸。

**通往直流电路**：电流的本质是电荷的定向移动，电流强度 $I = \Delta q / \Delta t$，即单位时间内通过导体截面的电荷量。电荷守恒定律在电路中直接表现为基尔霍夫电流定律（KCL）：任意节点处流入电荷量等于流出电荷量。

**通往磁场**：运动的电荷不仅产生电场，还会产生磁场。一个速度为 *v* 的点电荷 *q* 在磁场 *B* 中受到洛伦兹力 $\vec{F} = q\vec{v} \times \vec{B}$，这一公式将电荷、运动与磁场三者直接联系起来，是理解电动机和发电机工作原理的基础。
