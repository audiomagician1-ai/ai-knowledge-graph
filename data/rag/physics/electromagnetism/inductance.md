# 电感

## 概述

电感（Inductance）是描述导体回路抵抗电流变化能力的物理量，符号为 **L**，单位是**亨利（H）**，以物理学家约瑟夫·亨利命名。一亨利定义为：当电流以每秒1安培的速率变化时，回路中产生1伏特感应电动势所对应的电感量。实际电路中常用毫亨（mH）和微亨（μH）。

1831年法拉第发现电磁感应后，亨利在同年独立发现了自感现象，并于1832年正式提出电感概念。电感的物理本质是：变化电流在导体中产生变化磁通量，而变化磁通量反过来在导体自身（自感）或邻近导体（互感）中诱导出反抗原变化的电动势，即楞次定律的具体体现。

电感在电力系统、无线通信和电子滤波器中无处不在。螺线管电感量的计算公式为 $L = \mu_0 \mu_r \frac{N^2 A}{l}$，其中 $\mu_0 = 4\pi \times 10^{-7}\ \text{H/m}$ 为真空磁导率，$N$ 为匝数，$A$ 为截面积，$l$ 为线圈长度，$\mu_r$ 为相对磁导率。这意味着增加匝数对电感量的影响是平方关系，翻倍匝数使电感量增大四倍。

## 核心原理

### 自感（Self-Inductance）

自感是线圈对自身电流变化的"惯性"。当线圈中电流 $i$ 随时间变化时，线圈产生的磁通量也随之变化，导致线圈自身产生感应电动势：

$$\varepsilon_L = -L \frac{di}{dt}$$

负号表示自感电动势方向始终反对电流的变化——电流增大时，自感电动势阻碍增大；电流减小时，自感电动势阻碍减小。这一"阻尼"特性使电感成为电路中的"电流记忆元件"：流过电感的电流不能突变。具体地，若在 $t=0$ 时刻强制电感电流从1A瞬间变为0，理论上需要无穷大的电压，实际电路中会产生高压放电火花。

### 互感（Mutual Inductance）

两个相邻线圈之间存在互感，当线圈1中电流 $i_1$ 变化时，其产生的变化磁通量穿越线圈2，在线圈2中感应电动势：

$$\varepsilon_2 = -M \frac{di_1}{dt}$$

互感系数 $M$ 的单位同样是亨利，$M = k\sqrt{L_1 L_2}$，其中 $k$（$0 \leq k \leq 1$）是耦合系数，衡量两线圈磁通量的重叠程度。当两线圈完全耦合（共轴紧绕）时 $k=1$，磁能可100%传递，变压器正是基于此原理设计的。互感是单向的磁相互作用：线圈2中电流的变化同样会在线圈1中感应出 $\varepsilon_1 = -M \frac{di_2}{dt}$，且两者的互感系数 $M$ 相同，这一互易性（Reciprocity）由麦克斯韦方程组保证。

### RL电路的暂态响应

将电阻 $R$ 和电感 $L$ 串联后接直流电源 $V_0$，电路方程为：

$$V_0 = L\frac{di}{dt} + Ri$$

其解为 $i(t) = \frac{V_0}{R}\left(1 - e^{-\frac{R}{L}t}\right)$，电流从0按指数规律增大至稳定值 $V_0/R$。时间常数 $\tau = L/R$ 是RL电路的特征参数：经过一个 $\tau$，电流达到最终值的63.2%；经过 $5\tau$，电流基本达到稳定（99.3%）。若 $L = 10\ \text{mH}$，$R = 100\ \Omega$，则 $\tau = 0.1\ \text{ms}$，这正是高频开关电路中关注电感暂态时长的典型量级。

### 磁场中储存的能量

电感储存能量于磁场中，而非像电容那样储存于电场。流过电感电流为 $I$ 时，储存的磁场能量为：

$$W_L = \frac{1}{2}LI^2$$

这与弹簧弹性势能 $\frac{1}{2}kx^2$ 在形式上完全类比：$L$ 对应弹簧劲度系数 $k$，电流 $I$ 对应位移 $x$。磁场的能量密度为 $w = \frac{B^2}{2\mu_0\mu_r}$（单位 J/m³），这一公式说明磁场越强，单位体积储存的能量越多，铁芯电感通过提高 $\mu_r$ 来大幅增加单位体积储能能力。

## 实际应用

**电源滤波器**：开关电源中，电感与电容组成LC低通滤波器，利用电感阻抗 $X_L = \omega L = 2\pi f L$ 随频率升高而增大的特性，阻止高频纹波通过，让直流分量顺利输出。典型值如 $L = 100\ \mu\text{H}$，在50Hz时阻抗约0.03Ω，在100kHz时阻抗约63Ω，滤波效果相差约2000倍。

**变压器原理**：两线圈通过互感耦合，初级线圈匝数 $N_1$、次级匝数 $N_2$，变压比为 $\frac{V_2}{V_1} = \frac{N_2}{N_1}$。理想变压器中磁通量完全耦合（$k=1$），且能量守恒，故 $V_1 I_1 = V_2 I_2$，升压必降流，降压必升流。

**无线充电（Qi标准）**：基于互感原理，发射线圈工作频率为110–148 kHz，通过谐振耦合将能量传输至接收线圈，传输距离通常小于1cm时效率可超80%。系统通过调节 $M$ 和工作频率 $f$ 来优化传输效率。

**继电器保护电路**：继电器线圈断电瞬间，由于 $\varepsilon = -L\frac{di}{dt}$ 产生极高的反向电压（可达数百伏），会损坏并联的半导体器件。为此，必须在线圈两端并联续流二极管（飞轮二极管），提供电流释放通路，将反向峰值电压限制在约0.7V。

## 常见误区

**误区1：电感阻碍所有电流，而非变化的电流**
许多初学者认为电感像电阻一样始终阻碍电流。实际上，稳定直流通过电感时，$\frac{di}{dt} = 0$，自感电动势为零，理想电感对直流的阻抗为零。电感阻碍的是**电流的变化率**，频率越高，$X_L = 2\pi fL$ 越大，对高频交流的阻碍才越显著。直流电路稳态下电感相当于一根导线。

**误区2：电感断路瞬间电流为零**
切断含有电感的电路时，电流并不立即变为零。由于 $W_L = \frac{1}{2}LI^2$ 的磁场能量必须释放，电流会通过电弧（空气击穿）继续流动，这就是拔插头时产生火花的物理原因。电感"强迫"电流维持，这与电容"强迫"电压维持形成对偶关系。

**误区3：互感系数 $M$ 越大越好**
在变压器设计中，高互感系数（$k \to 1$）确实代表高效率传输，但在多线圈电路板（PCB）布局中，意外的寄生互感会导致电路间串扰，引发信号失真。因此，高速数字电路布线时刻意降低平行走线长度，正是为了减小不希望出现的互感耦合。

## 知识关联

**前置基础——法拉第电磁感应**：电感的定义直接建立在法拉第定律 $\varepsilon = -\frac{d\Phi}{dt}$ 上。自感电动势 $\varepsilon_L = -L\frac{di}{dt}$ 是将法拉第定律应用于线圈自身磁通量的直接推论：$\Phi = Li$，对时间求导即得。没有法拉第定律，电感量 $L$ 的物理定义无法建立。

**延伸方向——交流电路**：电感在交流电路中表现为感抗 $X_L = \omega L$，使电压相位超前电流90°（$\phi = +90°$），与电容的容抗 $X_C = \frac{1}{\omega C}$（电压滞后电流90°）相互补偿，形成LC谐振条件 $\omega_0 = \frac{1}{\sqrt{LC}}$。交流电路中的RLC串联谐振、阻抗匹配和滤波器设计全部依赖对电感频率特性的深刻理解，是学习交流电路的核心物理图像。
