# 自由电子模型

## 概述

自由电子模型是描述金属中导电电子行为的理论框架，其核心假设是将金属内的导电电子视为可在固体中自由运动的粒子，忽略电子与离子实之间的相互作用以及电子间的库仑排斥。该模型分为两个历史阶段：1900年Paul Drude提出经典自由电子模型，将电子视为服从麦克斯韦-玻尔兹曼分布的经典气体；1928年Arnold Sommerfeld将量子力学引入该框架，用费米-狄拉克分布取代经典分布，建立了量子自由电子模型（Sommerfeld模型）。

Drude模型成功解释了欧姆定律和维德曼-弗朗兹定律（Wiedemann-Franz Law），但对电子热容量的预测偏差高达100倍——经典模型预测每个电子贡献3k_B/2的热容，而实验测量值远小于此。Sommerfeld模型通过引入泡利不相容原理解决了这一问题，揭示了只有费米面附近约k_BT/E_F比例的电子才能被热激发，从而正确给出金属电子热容与温度的线性关系。

## 核心原理

### Drude经典模型：碰撞弛豫时间

Drude模型引入弛豫时间τ（典型值约为10⁻¹⁴秒）描述电子碰撞频率。在外加电场**E**下，电子受到平均散射前的漂移运动驱动，电导率公式为：

**σ = ne²τ/m**

其中n为自由电子数密度，e为电子电荷，m为电子质量。该公式正确给出了欧姆定律的微观基础。维德曼-弗朗兹定律表述为热导率κ与电导率σ之比满足κ/σ = LT，其中L为洛伦兹数，理论值为2.44×10⁻⁸ W·Ω·K⁻²，与多数金属实验值吻合。Drude模型的致命缺陷在于将电子视为经典粒子，无法解释为何电子在低温下热容趋近于零。

### Sommerfeld模型：费米能与费米球

Sommerfeld模型将电子视为填充在三维势箱中的量子费米气体。在绝对零度T=0时，电子按能量从低到高依次填充，直到最高占据能级，即费米能E_F。费米能的计算公式为：

**E_F = (ℏ²/2m)(3π²n)^(2/3)**

其中ℏ为约化普朗克常数，n为电子数密度。对于铜（Cu），n = 8.49×10²⁸ m⁻³，代入计算得E_F ≈ 7.04 eV，对应费米温度T_F = E_F/k_B ≈ 81,600 K，远高于室温300 K，这说明金属中电子的量子效应极为显著，经典统计完全失效。

在动量空间中，所有被占据的量子态构成半径为**k_F = (3π²n)^(1/3)**的球体，称为费米球，其表面即为费米面。铜的费米波矢k_F ≈ 1.36×10¹⁰ m⁻¹，费米速度v_F = ℏk_F/m ≈ 1.57×10⁶ m/s，约为光速的0.5%。

### 电子态密度与热容修正

单位体积单位能量间隔内的量子态数目称为态密度（Density of States），在自由电子模型中：

**g(E) = (1/2π²)(2m/ℏ²)^(3/2) · E^(1/2)**

态密度随能量呈抛物线形上升。在有限温度T下，费米-狄拉克分布函数f(E) = 1/[exp((E-μ)/k_BT)+1]决定各态的占据概率。电子热容的精确结果为：

**C_el = (π²/2) · (k_BT/E_F) · Nk_B = γT**

其中γ = π²Nk_B²/(2E_F)称为Sommerfeld系数。铜的实验测量γ ≈ 0.695 mJ/(mol·K²)，与自由电子模型理论值0.505 mJ/(mol·K²)接近但有偏差，该偏差由有效质量修正来解释（见能带理论）。

## 实际应用

**金属电阻率的温度依赖性**：在低温区，Sommerfeld模型预测电子-声子散射贡献的电阻率正比于T⁵（Bloch-Grüneisen定律的量子极限），在高温区正比于T，与金属铂、铜等的实验数据定量吻合。

**X射线发射光谱**：金属中电子从导带跃迁至内层空穴时发射的X射线强度分布直接反映态密度g(E)的形状。铝（Al）的软X射线发射谱实验结果与自由电子模型预测的抛物线型态密度高度吻合，证实了Sommerfeld模型对简单金属的有效性。

**热电子发射（Richardson-Dushman方程）**：金属加热时电子克服功函数逸出的热发射电流密度为J = AT²exp(-φ/k_BT)，其中A = 4πmek_B²/h³ = 1.20×10⁶ A/(m²·K²)，该公式直接来源于自由电子费米气体的热力学分析，用于真空管和电子枪设计。

## 常见误区

**误区一：费米能随温度升高而显著降低**。实际上，化学势μ（T）相对于E_F的偏移量正比于(k_BT/E_F)²，对铜在室温下该修正仅约为E_F的0.003%，可近似视为恒定。费米能是近似与温度无关的材料常数，不应与经典气体的平均能量混淆。

**误区二：Drude模型中的碰撞对象是离子实**。Drude最初假设电子与静止离子碰撞，但Sommerfeld之后的量子力学分析表明，电子在完美周期晶格中不发生散射，真实散射来源是晶格振动（声子）和杂质缺陷。弛豫时间τ ∝ 1/(声子密度)，故电阻率在高温下随声子数增多而线性上升。

**误区三：自由电子模型适用于所有金属**。该模型对碱金属（Li、Na、K）和贵金属（Cu、Ag、Au）效果良好，但对过渡金属（如Fe、Ni）严重失效，因为3d电子的强关联效应和非抛物线色散关系要求能带理论才能处理。Sommerfeld系数γ的实验值与自由电子预测值之比γ_exp/γ_free定义有效质量比m*/m，铁的m*/m ≈ 10，远偏离自由电子极限。

## 知识关联

**与声子的关联**：Drude模型中的弛豫时间τ在Sommerfeld框架下被重新诠释为费米面电子被声子散射的平均自由时间。低温下声子数目正比于T³（德拜模型），导致电子-声子散射率正比于T，这是金属电阻率低温T⁵行为（Bloch-Grüneisen）向高温T行为过渡的物理根源。

**与泡利不相容原理和量子统计的关联**：Sommerfeld模型的成功完全依赖于对电子施加费米-狄拉克统计而非玻尔兹曼统计。泡利不相容原理保证每个量子态至多一个电子占据，直接决定了费米球的存在和电子热容比经典预测小T/T_F倍这一量子压缩效应。

**通向能带理论**：自由电子模型忽略了离子实周期势对电子的调制作用。当引入弱周期势（近自由电子近似）时，费米球在布里渊区边界处发生能量劈裂，形成禁带，自然过渡到能带理论。Sommerfeld模型的费米能、态密度和有效质量概念在能带理论中全部保留，只是色散关系E(k)从抛物线变为更复杂的形式。
