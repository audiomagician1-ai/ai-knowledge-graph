# 自由电子模型

## 概述

自由电子模型是描述金属中导电电子行为的理论框架，其核心假设是：金属中的价电子可以脱离具体离子实的束缚，在整个晶体中自由运动，就像理想气体中的分子一样。这一模型分为两个历史阶段：1900年Paul Drude提出的经典版本（Drude模型），以及1928年Arnold Sommerfeld引入量子力学修正后的量子版本（Sommerfeld模型）。两者的根本差异在于电子统计方式的选取：Drude使用Maxwell-Boltzmann分布，而Sommerfeld使用Fermi-Dirac分布，后者正确反映了电子作为费米子必须服从泡利不相容原理的物理事实。

自由电子模型的重要性体现在它能以最少的参数解释金属的大量宏观电学性质。Drude模型成功解释了欧姆定律和Wiedemann-Franz定律（热导率与电导率之比正比于温度），而Sommerfeld模型进一步解决了Drude模型对电子热容量的严重高估问题——经典模型预测每个电子贡献 3/2 k_B 的热容，而实验测量值比这低约100倍，Sommerfeld模型通过费米-狄拉克统计将这一差异完美修正。

## 核心原理

### Drude模型的碰撞机制

Drude模型将金属中的传导电子视为自由粒子，它们周期性地与离子实发生碰撞，平均碰撞时间（弛豫时间）τ 约为 10⁻¹⁴ 秒。在外加电场 **E** 下，电子在相邻碰撞间被加速，碰撞后速度方向随机化。由此推导出电导率公式：

**σ = ne²τ / m_e**

其中 n 是导电电子数密度，e 是电子电荷，m_e 是电子质量。Drude模型的严重缺陷在于把电子当作经典粒子，导致对室温电子平均自由程的估算约为0.1 nm（原子间距量级），而实验测量铜的实际自由程约为40 nm，两者相差400倍，这一矛盾直到Sommerfeld模型才被化解。

### 费米能与费米-狄拉克分布

Sommerfeld模型的关键在于引入**费米能（Fermi energy）E_F**，即绝对零度下电子填充能级的最高能量。对于三维自由电子气，费米能的表达式为：

**E_F = (ℏ²/2m_e)(3π²n)^(2/3)**

以铜为例，其电子数密度 n ≈ 8.49×10²⁸ m⁻³，代入计算得 E_F ≈ 7.04 eV，对应费米温度 T_F = E_F/k_B ≈ 81700 K。室温 T ≈ 300 K 远低于 T_F，这意味着室温下铜中绝大多数电子仍处于量子简并状态，只有费米面附近约 k_BT 能量范围内（约0.026 eV）的电子才能被热激发，参与热容贡献的有效电子比例约为 T/T_F ≈ 1/272。

电子占据能级的概率由费米-狄拉克分布函数描述：

**f(E) = 1 / [exp((E - E_F)/k_BT) + 1]**

在 T=0 时该函数在 E_F 处呈阶跃：低于 E_F 的能级完全占满（f=1），高于 E_F 的能级完全空（f=0）。有限温度下阶跃被软化为宽度约 k_BT 的过渡区。

### 电子热容的量子修正

Sommerfeld模型给出金属自由电子对热容的贡献为：

**C_el = (π²/2)(T/T_F)nk_B = γT**

其中 γ = π²nk_B²/(2E_F) 称为Sommerfeld系数，单位为 J/(mol·K²)。铜的实验测量值 γ ≈ 0.695 mJ/(mol·K²)，与Sommerfeld模型的理论预测吻合良好。注意热容与温度 T 呈线性关系，而非Drude经典模型预测的常数值。在极低温度下，电子热容 C_el = γT 与声子热容 C_ph ∝ T³（德拜模型）可通过测量总热容 C = γT + AT³ 来分离，这是实验测定费米能的标准方法之一。

### 态密度与费米球

在k空间中，自由电子在绝对零度下占据半径为费米波矢 k_F = (3π²n)^(1/3) 的球形区域，称为**费米球**。铜的费米波矢 k_F ≈ 1.36×10¹⁰ m⁻¹，对应费米速度 v_F = ℏk_F/m_e ≈ 1.57×10⁶ m/s，约为光速的0.5%。态密度（单位能量范围内的量子态数）在费米能处的值 g(E_F) = 3n/(2E_F)，这个量直接决定了体系的磁化率（泡利顺磁性）和电子热容系数 γ。

## 实际应用

自由电子模型对金属的**霍尔效应**给出直接预测：霍尔系数 R_H = -1/(ne)，仅由载流子密度决定。对于钠（Na），n ≈ 2.65×10²⁸ m⁻³，理论预测 R_H ≈ -2.36×10⁻¹⁰ m³/C，与实验值 -2.50×10⁻¹⁰ m³/C 吻合在5%以内，验证了自由电子模型的有效性。

在**热电子发射**（Richardson-Dushman方程）中，金属加热后电子从费米海越过表面势垒逃逸，发射电流密度 J = AT²exp(-φ/k_BT)，其中 φ 是逸出功，A 是Richardson常数（约为120 A·cm⁻²·K⁻²），这一公式直接来源于Sommerfeld模型对费米面附近电子热激发的描述，是电子管和热阴极X射线管的物理基础。

自由电子模型也用于解释金属的**光学性质**中的等离子体振荡，等离子体频率 ω_p = (ne²/ε₀m_e)^(1/2)，对于铝 ω_p 对应波长约80 nm（紫外区），这预测铝在可见光频率下应表现为高反射率而在紫外区变为透明，与实验完全一致。

## 常见误区

**误区一：认为Drude模型和Sommerfeld模型的区别仅在于精度细节。** 两个模型的本质差别在于物理假设：Drude模型中电子的平均热动能为 (3/2)k_BT（经典能均分），而Sommerfeld模型中即使在 T=0 时电子仍具有巨大的平均动能 (3/5)E_F（对铜约4.2 eV），远超热能。这不是精度问题，而是量子统计与经典统计的根本对立。对电子热容、磁化率等热力学量的预测，两个模型的结果相差可达两个数量级。

**误区二：自由电子模型意味着电子受到的总势能为零。** 事实上，模型假设的是晶格离子产生的势场在整个晶体中**均匀分布**，对每个电子贡献相同的常数势能，因此可以选取该常数为势能零点，但这并不等于忽略了离子-电子相互作用的整体效果。这种"均匀背景正电荷"近似被称为"胶冻模型"（jellium model）。真正被忽略的是晶格的**周期性势场调制**，而正是这一调制导致了能隙的形成，这是自由电子模型无法解释的，也是过渡到能带理论的动机。

**误区三：费米能随温度升高而显著降低。** 在 T << T_F 条件下，费米能随温度的修正为 E_F(T) ≈ E_F(0)[1 - (π²/12)(T/T_F)²]。对铜在室温下，这一修正仅约为 (π²/12)(300/81700)² ≈ 4.4×10⁻⁵，即室温使铜的费米能只降低不到0.01%。因此在绝大多数金属物性讨论中，费米能可视为温度无关的常数。

## 知识关联

自由电子模型直接依赖**泡利不相容原理**——正是由于电子是自旋1/2的费米子，每个量子态最多容纳一个电子，才使得费米海（费米球内的充满态）成为可能，并导致费米能的存在。**量子统计**中的费米-狄拉克分布是Sommerfeld模型的数学核心，没有这一分布就无法正确计算有限温度下的电子占据数和热容。**声子**作为晶格振动的量子，与自由电子模型形成互补：在极低温金属热容 C = γT + AT³ 的测量中，γT 项来自自由电子（Sommerfeld），AT³ 项来自声子（德拜），两者的分离实验是固体物理实验的经典范例。
