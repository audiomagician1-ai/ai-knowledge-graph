# 晶体结构

## 概述

晶体结构是指原子、离子或分子在三维空间中以严格周期性规律排列所形成的有序状态。这种长程有序性使晶体区别于液体和非晶态固体——不仅原子间距固定，而且整个宏观样品中相同的原子排列单元无限重复延伸。1912年，劳厄（Max von Laue）首次通过X射线衍射实验证明了晶体内部的周期性原子排列，奠定了晶体学的实验基础。

晶体结构的描述依赖于两个独立概念的组合：**格子（Lattice）**与**基元（Basis）**。格子是空间中数学点的周期性阵列，本身不含物理内容；基元是附加在每个格点上的实际原子集合。完整晶体结构 = 布拉菲格子 + 基元。正因为如此，即使两种晶体具有相同的格子类型，基元不同也会导致完全不同的物理性质，例如金刚石与闪锌矿同属面心立方格子，但金刚石基元为两个碳原子，闪锌矿基元为一个锌和一个硫原子。

理解晶体结构对于预测材料的力学强度、导电性、光学性质以及热膨胀系数至关重要。硅单晶的金刚石立方结构决定了其各向同性的半导体特性，而石墨的六方层状结构则导致其面内导电性远强于层间方向，这两者的差异直接来源于晶体结构的不同。

## 核心原理

### 布拉菲格子的14种类型

1848年，法国晶体学家奥古斯特·布拉菲（Auguste Bravais）严格证明：三维空间中满足周期性平移对称的点阵仅有 **14种**，分属7个晶系。布拉菲格子由三个基矢 $\mathbf{a}_1, \mathbf{a}_2, \mathbf{a}_3$ 完全确定，格点位置为：

$$\mathbf{R} = n_1\mathbf{a}_1 + n_2\mathbf{a}_2 + n_3\mathbf{a}_3, \quad n_1, n_2, n_3 \in \mathbb{Z}$$

14种格子按晶系分类如下（括号内为该晶系包含的布拉菲格子数）：立方（3）、四方（2）、正交（4）、单斜（2）、三斜（1）、三方（1）、六方（1）。以立方晶系为例，其三种格子分别为简单立方（SC）、体心立方（BCC）和面心立方（FCC）。BCC的配位数为8，FCC的配位数为12，FCC的堆积效率为 $\pi/(3\sqrt{2}) \approx 74.05\%$，是等球密堆积效率最高的结构之一。

### 晶胞参数与7个晶系

每种晶系由特定的晶胞参数约束来定义。三斜晶系最一般，6个参数 $a, b, c, \alpha, \beta, \gamma$ 全部独立无约束；立方晶系约束最严格，要求 $a = b = c$ 且 $\alpha = \beta = \gamma = 90°$，仅需1个独立参数。六方晶系的特征约束为 $a = b \neq c$，$\alpha = \beta = 90°$，$\gamma = 120°$。石墨的六方晶胞参数为 $a = 2.461$ Å、$c = 6.708$ Å，层内碳碳间距（1.42 Å）远小于层间距（3.354 Å），直接反映了其各向异性键合特征。

### 密勒指数（Miller Indices）

密勒指数 $(hkl)$ 是描述晶体中一族平行晶面方向的整数标记方法。取某晶面在三个基矢方向上的截距（以晶格常数为单位）分别为 $p, q, r$，取倒数后化为最小整数比，即得到密勒指数 $(hkl) = (1/p : 1/q : 1/r)$。若晶面平行于某轴，截距为无穷大，对应指数为0。

相邻晶面族 $(hkl)$ 之间的面间距 $d_{hkl}$ 对于立方晶系有精确公式：

$$d_{hkl} = \frac{a}{\sqrt{h^2 + k^2 + l^2}}$$

其中 $a$ 为晶格常数。面心立方结构中，$(111)$ 面的面间距 $d_{111} = a/\sqrt{3}$，而由于FCC的消光规则，$(100)$ 面对应的衍射峰在X射线实验中不会出现，这是识别FCC晶体的重要特征。方向指数用 $[hkl]$ 表示（如 $[110]$ 为面对角线方向），一族等价方向用 $\langle hkl \rangle$ 表示。

### 倒格子

倒格子（Reciprocal Lattice）是与实空间格子对偶的数学构造，其基矢定义为：

$$\mathbf{b}_1 = 2\pi \frac{\mathbf{a}_2 \times \mathbf{a}_3}{\mathbf{a}_1 \cdot (\mathbf{a}_2 \times \mathbf{a}_3)}$$

（$\mathbf{b}_2, \mathbf{b}_3$ 类推）。倒格子的每个格点 $\mathbf{G} = h\mathbf{b}_1 + k\mathbf{b}_2 + l\mathbf{b}_3$ 恰好对应实空间中一族 $(hkl)$ 晶面，且 $|\mathbf{G}_{hkl}| = 2\pi/d_{hkl}$。FCC的倒格子是BCC，BCC的倒格子是FCC，这一互为对偶的关系在分析衍射图样时极为关键。

## 实际应用

**半导体工业中的晶向控制**：硅晶圆的切割方向直接影响器件性能。(100)面硅片的表面态密度低于(111)面，因此MOSFET器件通常采用(100)面硅片以降低界面陷阱。硅的金刚石立方结构（空间群 $Fd\bar{3}m$，晶格常数 5.431 Å）中，共价键沿 $\langle 111 \rangle$ 方向分布，这决定了(111)解理面是硅的自然解理平面。

**合金相变中的晶体结构转变**：纯铁在912°C以下为BCC结构（α-铁），在912°C至1394°C之间转变为FCC结构（γ-铁，奥氏体）。FCC结构中碳原子的最大固溶度为2.14 wt%，而BCC结构中仅为0.022 wt%，这一差异直接来源于FCC八面体间隙（半径0.414r）大于BCC八面体间隙（半径0.155r）。钢铁热处理的所有工艺均建立在这两种晶体结构转换的基础上。

**NaCl结构的离子晶体**：氯化钠晶体由两个FCC格子沿体对角线位移 $a/2$ 相互穿插而成，每个Na⁺被6个Cl⁻以八面体形式包围，晶格常数 $a = 5.640$ Å，马德隆常数（Madelung Constant）为1.7476，定量描述了离子静电相互作用对结合能的贡献。

## 常见误区

**误区1：布拉菲格子就是晶体结构本身**
许多初学者将14种布拉菲格子等同于14种晶体结构，实际上布拉菲格子只是空间点阵，还需要加上基元才能构成真实晶体。NaCl和CsCl虽然都是"立方"结构，但NaCl属于FCC格子配双原子基元，CsCl属于简单立方格子配双原子基元，二者的格子类型不同。将格子与结构混淆会导致错误计算衍射峰位置和强度。

**误区2：密勒指数越大，晶面越重要**
实际上低指数晶面（如(100)、(110)、(111)）往往具有最高的原子面密度，因此表面能最低，在晶体生长和腐蚀中优先暴露。高指数晶面如(531)在理想晶体表面上面密度低，通常不稳定，容易分解为低指数晶面的台阶结构。

**误区3：所有固体都可用布拉菲格子描述**
准晶体（Quasicrystal）由谢赫特曼（Dan Shechtman）于1982年发现，其衍射图样具有五重对称性（二十面体对称），但这在经典布拉菲格子理论中是被严格禁止的——周期性格子不允许五重旋转对称轴。准晶体拥有长程有序但无平移周期性，是对"晶体必须符合14种布拉菲格子"这一误区的最重要实验反例。谢赫特曼因此获得2011年诺贝尔化学奖。
