# 原子光谱

## 概述

原子光谱是原子在能级跃迁时发射或吸收特定波长电磁辐射所形成的谱线图样。每种元素的原子光谱具有唯一性，如同"原子指纹"——氢原子在可见光范围内仅有四条谱线（656.3 nm 红色、486.1 nm 青色、434.0 nm 蓝紫色、410.2 nm 紫色），这一组合在宇宙中任何地方都保持不变。

原子光谱的系统研究始于19世纪。1885年，巴尔末（Johann Balmer）发现氢原子可见光谱满足经验公式 $\lambda = B\cdot\frac{n^2}{n^2-4}$（B = 364.56 nm，n = 3, 4, 5...），但当时无法解释其物理根源。直到1913年玻尔模型提出，才首次从量子化能级的角度解释了氢原子谱线的成因。1925年之后，量子力学进一步引入主量子数、角量子数、磁量子数和自旋量子数四个量子数，完整描述了原子光谱的精细结构。

原子光谱在天文学、化学分析和物理研究中具有不可替代的地位。天文学家通过分析星光的吸收谱线来测定恒星的元素组成和径向速度（多普勒位移）；法医和材料科学家利用原子发射光谱仪在微克量级上鉴定元素成分。

## 核心原理

### 发射谱与吸收谱的形成机制

**发射谱**由处于激发态的原子向低能级跃迁时产生。当原子从能级 $E_n$ 跃迁到 $E_m$（$E_n > E_m$）时，辐射光子的频率严格满足 **玻尔频率条件**：

$$h\nu = E_n - E_m$$

其中 $h = 6.626 \times 10^{-34}\ \text{J·s}$ 为普朗克常数。发射谱在连续暗背景上呈现亮线，每条亮线对应一个特定的能级跃迁。

**吸收谱**的形成机制相反：当连续谱光通过基态或低激发态原子气体时，光子能量恰好等于某一跃迁能级差的波长被原子吸收，形成连续背景上的暗线（夫琅禾费线）。太阳光谱中存在超过600条夫琅禾费线，其中589 nm处的两条暗线（D线）对应钠原子的3p→3s跃迁，这是19世纪就被精确测量的经典数据。吸收谱与发射谱的谱线位置完全对应，但谱线形态不同。

### 四个量子数与原子态描述

氢原子及类氢原子的完整量子态需要四个量子数来描述：

- **主量子数 $n$**：$n = 1, 2, 3, ...$，决定电子能量（氢原子能量 $E_n = -13.6/n^2\ \text{eV}$）和轨道大小。
- **角量子数 $l$**：$l = 0, 1, ..., n-1$，决定轨道角动量大小 $|\vec{L}| = \sqrt{l(l+1)}\,\hbar$，对应 s、p、d、f 轨道类型。
- **磁量子数 $m_l$**：$m_l = -l, ..., 0, ..., +l$，共 $2l+1$ 个取值，决定轨道角动量在外磁场方向上的分量 $L_z = m_l\hbar$。
- **自旋量子数 $m_s$**：仅取 $+\frac{1}{2}$ 或 $-\frac{1}{2}$，对应电子自旋向上或向下。

这四个量子数共同决定原子的能量本征态，多电子原子还需考虑泡利不相容原理（每个量子态最多容纳一个电子）。

### 选择定则——并非所有跃迁都被允许

选择定则是量子力学对电偶极辐射跃迁的严格限制，由跃迁矩阵元 $\langle f|\hat{r}|i\rangle$ 是否为零决定。对于电偶极跃迁，允许的跃迁必须同时满足：

$$\Delta l = \pm 1, \quad \Delta m_l = 0, \pm 1, \quad \Delta m_s = 0$$

主量子数 $\Delta n$ 无限制，但 $\Delta l = 0$ 或 $|\Delta l| > 1$ 的跃迁是**禁戒跃迁**，其跃迁概率比允许跃迁低 $10^3$ 到 $10^6$ 倍。例如，氢原子 2s 态（$l=0$）无法通过电偶极辐射直接跃迁到 1s 态（$l=0$，违反 $\Delta l = \pm 1$），导致 2s 态的寿命长达约 $\frac{1}{7}$ 秒，远超典型激发态寿命 $\sim 10^{-8}\ \text{s}$。禁戒跃迁并非完全不发生，而是通过磁偶极或电四极机制以极低概率进行。

### 谱线系与精细结构

氢原子谱线依据终态主量子数分为不同谱线系：终态 $n=1$ 为莱曼系（紫外区）、$n=2$ 为巴尔末系（可见及近紫外）、$n=3$ 为帕邢系（红外区）。将相对论修正和自旋-轨道耦合纳入考虑后，原本单一的谱线分裂为多条靠近的谱线，即**精细结构**。钠 D 线实为 589.0 nm 和 589.6 nm 两条谱线，间距 0.6 nm，正是3p态 $j=3/2$ 与 $j=1/2$ 之间自旋-轨道耦合能量差所致。

## 实际应用

**天文光谱学**：哈勃1929年利用星系光谱中氢原子谱线的红移，发现了宇宙膨胀规律。现代天文学通过分析类星体光谱中莱曼系谱线的红移，可以测定其距离达数十亿光年。

**原子钟与时间标准**：铯-133原子的基态超精细跃迁频率被定义为 9,192,631,770 Hz，这是国际单位制中"秒"的定义基础。铯原子钟的授时精度优于 $10^{-14}$，即3000万年误差不超过1秒。

**焰色反应与工业检测**：钾盐燃烧发出紫色（767 nm 和 770 nm），钙盐发橙红色（423 nm），锂盐发深红色（671 nm），均对应各元素特征发射谱线。工业上利用原子吸收光谱仪可在 $\text{ng/mL}$ 量级检测重金属污染。

## 常见误区

**误区一：吸收谱与发射谱谱线完全相同**。两者谱线位置对应，但形态不同：发射谱是暗背景上的亮线，吸收谱是连续背景上的暗线。更重要的是，实际吸收谱通常只显示从**基态**出发的跃迁谱线，因为室温下绝大多数原子处于基态；而发射谱则可能包含激发态之间的跃迁谱线。因此同一元素的吸收谱通常比发射谱的谱线条数少。

**误区二：选择定则意味着禁戒跃迁永远不发生**。选择定则专指电偶极辐射，禁戒跃迁可通过磁偶极辐射（要求 $\Delta l = 0, \Delta j = 0, \pm 1$）或电四极辐射等高阶过程发生，只是跃迁速率极低。星际介质中氢原子 21 cm 谱线（1420 MHz）就是 1s 态超精细结构之间的磁偶极禁戒跃迁，虽然单次跃迁概率约为每 $10^7$ 年一次，但由于宇宙中氢的数量极其巨大，21 cm 线成为射电天文学的重要观测手段。

**误区三：主量子数相同的原子态能量必然相同**。这仅在纯氢原子（忽略精细结构）时成立。引入自旋-轨道耦合后，同一 $n$ 下不同 $l$ 和 $j$ 的能级会发生分裂；在多电子原子中，屏蔽效应使得相同 $n$ 但不同 $l$ 的轨道能量有明显差异（如 3d 轨道能量高于 4s 轨道，这正是过渡元素电子填充顺序反常的原因）。

## 知识关联

**与玻尔模型的关系**：玻尔模型通过量子化条件 $L = n\hbar$ 正确预言了氢原子的能级公式和巴尔末系谱线波长，但无法解释谱线的相对强度、精细结构或多电子原子光谱。原子光谱的量子力学处理是对玻尔模型的根本性升级，四个量子数和选择定则是玻尔模型中完全缺失的内容。
