# 玻尔模型

## 概述

1913年，26岁的尼尔斯·玻尔发表了三篇论文，提出了一个半经典的氢原子模型。在卢瑟福发现原子核仅两年后，经典电磁理论预言绕核运动的电子会持续辐射能量并螺旋坠入原子核——原子不应稳定存在。玻尔通过引入量子化假设大胆解决了这个危机，不仅解释了氢原子光谱的精确波长，还为量子力学的建立铺平了道路。

## 核心知识点

### 1. 玻尔的三个基本假设

**假设一：定态轨道**
电子只能在特定的圆形轨道上运动，轨道上的电子不辐射电磁波。这些允许的轨道由量子数 n = 1, 2, 3, ... 标记。这直接违背了经典电磁学的预言（加速电荷必须辐射），但是解释了原子的稳定性。

**假设二：角动量量子化**
电子轨道角动量只能取 $\hbar$ 的整数倍：$L = n\hbar$，其中 $\hbar = h/(2\pi) = 1.055 \times 10^{-34}$ J*s。这是玻尔最核心的创新——将普朗克的量子概念从辐射场推广到物质粒子。

**假设三：频率条件（跃迁规则）**
电子在两个定态之间跃迁时，发射或吸收的光子频率满足：h*nu = |E_n - E_m|。这直接将爱因斯坦的光量子假说与原子结构联系起来。

### 2. 氢原子能级

从角动量量子化条件出发，结合库仑力提供向心力的经典方程，可以推导出氢原子的能级公式：

**E_n = -13.6 eV / n^2**

其中 n = 1, 2, 3, ... 为主量子数。

关键数值：
- **基态** (n=1): E_1 = -13.6 eV，这就是氢原子的电离能
- **第一激发态** (n=2): E_2 = -3.4 eV
- **第二激发态** (n=3): E_3 = -1.51 eV
- **电离极限** (n -> infinity): E = 0

推导中的关键常数：13.6 eV = m_e * e^4 / (2 * h-bar^2)，其中 m_e 是电子质量，e 是电子电荷。这个值与实验测量的氢原子电离能精确吻合，是玻尔模型最大的定量成功。

### 3. 光谱线系

当电子从高能级 n_i 跃迁到低能级 n_f 时，发射光子的波数为：

**1/lambda = R_H * (1/n_f^2 - 1/n_i^2)**

其中 R_H = 1.097 x 10^7 m^(-1) 为里德伯常数。

氢原子光谱的各线系：

| 线系名称 | n_f | n_i | 波段 | 发现年份 |
|----------|-----|-----|------|---------|
| 莱曼系 (Lyman) | 1 | 2,3,4... | 紫外 | 1906 |
| 巴尔末系 (Balmer) | 2 | 3,4,5... | 可见光 | 1885 |
| 帕邢系 (Paschen) | 3 | 4,5,6... | 近红外 | 1908 |
| 布拉凯特系 (Brackett) | 4 | 5,6,7... | 红外 | 1922 |
| 普芳德系 (Pfund) | 5 | 6,7,8... | 远红外 | 1924 |

巴尔末系的可见光线：H-alpha (656.3 nm, 红), H-beta (486.1 nm, 蓝绿), H-gamma (434.0 nm, 蓝紫), H-delta (410.2 nm, 紫)。例如，H-alpha 线对应电子从 n=3 跃迁到 n=2，代入公式 1/lambda = 1.097e7 * (1/4 - 1/9) 可精确计算出 656.3 nm。玻尔模型计算值与这些实验波长的一致精度达到 0.02%。

> 思考题：为什么巴尔末系落在可见光范围而莱曼系在紫外？这与 $1/n_f^2$ 项的大小有什么关系？

### 4. 玻尔半径

基态电子轨道半径（玻尔半径）是原子物理中的基本长度单位：

**a_0 = h-bar^2 / (m_e * e^2 * k_e) = 0.529 x 10^(-10) m = 0.529 A**

第 n 个轨道的半径为 r_n = n^2 * a_0。

玻尔半径的物理意义：
- 定义了原子的特征尺度（约 0.5 埃）
- 在量子力学的精确解中，a_0 正好是氢原子基态径向概率密度最大值处的距离
- 作为原子单位制(a.u.)中的长度单位被广泛使用

### 5. 玻尔模型的局限性

虽然玻尔模型对氢原子取得了辉煌成功，但它存在根本性的局限：

- **多电子原子**：无法正确预测氦原子及更重原子的光谱。电子-电子相互作用使问题无法用简单的轨道图像处理。
- **谱线精细结构**：高分辨率光谱显示每条谱线实际上是多条紧密线的组合（精细结构），玻尔模型无法解释。需要索末菲的椭圆轨道修正和后来的自旋-轨道耦合理论。
- **谱线强度**：模型无法预测各跃迁的概率（即谱线的相对亮度），这需要量子力学的矩阵元计算。
- **概念局限**：电子并不真的在确定轨道上运行。量子力学表明电子以概率云（波函数|psi|^2）的形式存在。玻尔轨道对应的是径向概率密度的峰值位置。

正是这些局限推动了1925-1926年海森堡矩阵力学和薛定谔波动力学的建立。

## 关键要点

1. 玻尔模型通过**角动量量子化** L = n*h-bar 解决了原子稳定性危机
2. 氢原子能级 **E_n = -13.6/n^2 eV** 与实验电离能精确吻合
3. 光谱公式 1/lambda = R_H(1/n_f^2 - 1/n_i^2) 以 0.02% 的精度复现了氢光谱全部线系
4. 玻尔半径 a_0 = 0.529 A 定义了原子的特征尺度
5. 模型在氢原子上成功但无法推广到多电子体系，这一困难直接催生了量子力学

## 常见误区

1. **"电子真的在圆形轨道上运行"**：玻尔模型的轨道是一个有用的近似图像，但量子力学表明电子没有确定的轨迹。氢原子基态的电子概率密度是球对称的云，在 r = a_0 处概率密度最大。
2. **"玻尔模型已经完全过时"**：虽然被量子力学取代，但它给出的能级公式 E_n = -13.6/n^2 eV 对氢原子仍然精确。而且玻尔半径、里德伯常数等概念在现代物理中仍被广泛使用。
3. **混淆"轨道"与"轨域"**：玻尔的"orbit"（轨道，确定路径）与量子力学的"orbital"（轨域，概率分布）是根本不同的概念。
4. **认为玻尔模型只是猜测**：虽然假设看似ad hoc，但角动量量子化条件可以从德布罗意驻波条件 2*pi*r = n*lambda 自然导出（1924年），赋予了它更深的物理基础。

## 知识衔接

### 先修知识
- **光电效应** — 爱因斯坦光量子假说 E = h*nu 是玻尔频率条件的基础
- **库仑定律** — 电子-质子间的库仑引力提供向心力，是能级推导的起点

### 后续学习
- **原子光谱** — 多电子原子的光谱分析与选择定则
- **量子力学基础** — 薛定谔方程对氢原子的精确解，给出完整的量子数 (n, l, m_l, m_s)
- **塞曼效应** — 磁场中谱线分裂，揭示角动量的空间量子化
