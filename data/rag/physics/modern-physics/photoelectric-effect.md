# 光电效应

## 概述

光电效应（Photoelectric Effect）是指光照射到金属表面时，电子从金属中被释放出来的现象。这个效应由赫兹在1887年首先观察到，但经典电磁理论无法解释其关键实验特征。1905年，爱因斯坦提出**光量子假说**——光不是连续的波，而是由一个个能量包（光子）组成——成功解释了光电效应的所有实验事实，并因此获得1921年诺贝尔物理学奖。

光电效应是量子力学的开端之一，它与黑体辐射（普朗克1900年）一起，打开了"量子"的大门。学习光电效应需要了解经典电磁波理论的基本概念（频率、强度、能量），掌握后可以进入玻尔模型、波粒二象性、德布罗意关系等量子力学核心概念。

## 核心知识点

### 实验事实

光电效应的四个关键实验发现：

1. **存在截止频率**：只有当入射光频率 $\nu$ 大于某个阈值频率 $\nu_0$ 时才能产生光电效应，与光的强度无关
2. **瞬时发射**：光电子几乎在光照到达的瞬间（$< 10^{-9}$ s）就被发射，不存在经典理论预测的"能量积累时间"
3. **最大动能与频率线性相关**：光电子的最大动能随入射光频率增加而线性增大，与光强无关
4. **光电流与光强成正比**：增大光强只增加每秒发射的电子数，不改变单个电子的能量

### 经典理论的困难

经典电磁理论认为光是连续的波，能量与振幅（强度）成正比：
- 应该是任何频率的光只要足够强都能打出电子 → 与实验1矛盾
- 弱光下应该需要时间积累足够能量 → 与实验2矛盾
- 电子能量应该与光强相关 → 与实验3矛盾

### 爱因斯坦光量子解释

爱因斯坦假设光由**光子**（photon）组成，每个光子能量为：

$$E = h\nu = \frac{hc}{\lambda}$$

其中 $h = 6.626 \times 10^{-34}$ J·s 是普朗克常数。

光电效应方程：

$$h\nu = W + E_k^{\max} = W + \frac{1}{2}mv_{\max}^2$$

其中 $W = h\nu_0$ 是金属的**逸出功**（Work Function）——将电子从金属表面拉出所需的最小能量。

整理得：

$$E_k^{\max} = h\nu - W = h(\nu - \nu_0)$$

这完美解释了所有实验事实：
- 截止频率：$\nu < \nu_0$ 时光子能量不足以克服逸出功
- 瞬时发射：单个光子在一次碰撞中就能提供全部能量
- 线性关系：$E_k^{\max}$ 对 $\nu$ 作图是斜率为 $h$ 的直线
- 光强效应：光强增大 = 光子数增多 = 更多电子被打出

### 典型金属的逸出功

| 金属 | 逸出功 $W$ (eV) | 截止波长 $\lambda_0$ (nm) |
|:---|:---:|:---:|
| 铯 (Cs) | 2.1 | 590 (可见光) |
| 钠 (Na) | 2.3 | 540 |
| 铝 (Al) | 4.3 | 289 (紫外) |
| 金 (Au) | 5.1 | 243 (紫外) |

## 关键要点

1. **光电效应证明光具有粒子性**：这与光的干涉/衍射实验（波动性）并不矛盾，而是波粒二象性的体现
2. **逸出功是材料属性**：不同金属的逸出功不同，决定了截止频率不同
3. **遏止电压可以测量最大动能**：在实验中施加反向电压 $U_0$ 使光电流为零，则 $eU_0 = E_k^{\max} = h\nu - W$
4. **普朗克常数 $h$ 是从光电效应实验中精确测定的**：密立根1916年的精密实验验证了爱因斯坦方程，测得的 $h$ 值与普朗克从黑体辐射得到的一致

## 常见误区

1. **"增大光强就能打出电子"**：如果频率低于截止频率，即使用再强的光也打不出电子。因为每个光子的能量只由频率决定，强度只决定光子数量
2. **"光电效应推翻了光的波动理论"**：没有推翻，而是证明了光同时具有波动性和粒子性。干涉实验仍然有效，光电效应补充了粒子性的证据
3. **"所有电子都以最大动能飞出"**：$E_k^{\max}$ 是表面层电子吸收一个光子后的最大可能动能。内层电子需要额外能量逃逸，因此动能更小。实验测到的是动能的分布

## 知识衔接

**先修概念**：电磁波（频率、波长、能量）、经典波动理论

**后续概念**：
- **玻尔模型**：将量子化思想应用于原子结构
- **波粒二象性**：光电效应（粒子性）+ 双缝实验（波动性）的统一
- **德布罗意关系**：物质也具有波动性，$\lambda = h/p$
- **康普顿散射**：光子与电子碰撞，进一步证实光子具有动量 $p = h/\lambda$
- **光伏效应**：光电效应在半导体中的应用，太阳能电池的工作原理
