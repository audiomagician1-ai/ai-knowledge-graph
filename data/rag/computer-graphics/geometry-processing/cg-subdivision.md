# 细分曲面

## 概述

细分曲面（Subdivision Surfaces）是一种通过反复对粗糙多边形网格施加细分规则，逐步生成光滑极限曲面的几何处理方法。每次细分操作将原始网格的每条边和每个面分裂为更多的子元素，并按照特定的加权平均公式重新计算顶点位置，使网格在无限次细分后趋向于连续光滑的极限曲面。

细分曲面的理论基础来自于样条曲线的推广。1978年，Edwin Catmull与Jim Clark在论文中提出了针对任意拓扑四边形网格的 Catmull-Clark 细分方案；同年，Charles Loop 在其硕士论文中提出了针对三角形网格的 Loop 细分方案。这两种方案分别对应不同的网格类型，至今仍是图形学中最广泛使用的细分算法。

细分曲面在工业界与影视特效中扮演重要角色。Pixar 动画工作室的 OpenSubdiv 库基于 Catmull-Clark 细分，《玩具总动员》等早期三维动画中的角色曲面均采用此技术建模。相比于传统 NURBS，细分曲面能够自然处理任意拓扑的网格（包括亏格不为零的封闭曲面），而无需手动拼接多个曲面片。

## 核心原理

### Catmull-Clark 细分规则

Catmull-Clark 细分作用于四边形（或混合多边形）网格，每步将每个 $n$ 边面细分为 $n$ 个四边形。细分后网格中顶点分为三类，计算公式如下：

**面点（Face Point）**：每个面生成一个新顶点，位置为该面所有原始顶点的算术平均：
$$F = \frac{1}{n}\sum_{i=1}^{n} V_i$$

**边点（Edge Point）**：每条边生成一个新顶点，位置为该边两端点与相邻两面的面点的平均：
$$E = \frac{V_1 + V_2 + F_1 + F_2}{4}$$

**顶点点（Vertex Point）**：原始顶点更新公式（$n$ 为顶点的度，即相邻边数）：
$$V' = \frac{n-2}{n}V + \frac{1}{n^2}\sum F_i + \frac{1}{n^2}\sum M_i$$
其中 $\sum F_i$ 为相邻面点之和，$\sum M_i$ 为相邻各边中点之和（各取 $n$ 项）。

经过足够多次细分后，Catmull-Clark 极限面在正则顶点（度为4）处达到 $C^2$ 连续，而在非正则顶点（度 $\neq 4$）处仅达到 $C^1$ 连续。

### Loop 细分规则

Loop 细分作用于三角形网格，每步将每个三角形细分为4个更小的三角形（即 1-to-4 分裂）。边点插入每条边的中点附近，顶点更新公式依赖于顶点度 $n$：

$$\beta = \frac{1}{n}\left(\frac{5}{8} - \left(\frac{3}{8} + \frac{1}{4}\cos\frac{2\pi}{n}\right)^2\right)$$

原始顶点的新位置为：
$$V' = (1 - n\beta)V + \beta\sum_{i=1}^{n} V_i$$

其中 $\sum V_i$ 为所有邻接顶点之和。Loop 细分的极限面在正则顶点（度为6的三角形网格顶点）处达到 $C^2$ 连续，非正则顶点处为 $C^1$ 连续。

### 极限位置与极限法向量

细分曲面存在解析的极限位置公式，无需真正迭代无限次。以 Catmull-Clark 为例，度为 $n$ 的顶点的极限位置为：
$$V_{\infty} = \frac{n^2 V + 4\sum F_i + \sum M_i}{n(n+5)}$$

利用极限位置公式，可以在第一次细分后直接将顶点移动到极限位置，大幅减少渲染所需的迭代次数。极限法向量则通过计算极限切向量的叉积获得，用于光照计算。

### 边界与折痕处理

对于开放网格的边界边，以及艺术家希望保留锐利折痕的边，细分规则需要特殊处理。Catmull-Clark 的折痕（Crease）规则将折痕边的边点计算简化为仅使用两个端点的均值 $E = (V_1 + V_2)/2$，而不考虑相邻面点，使折痕在极限面上保持 $C^0$ 而非光滑连接。折痕权重可以设置为 $[0, 1]$ 的连续值，实现可控的半锐折痕效果。

## 实际应用

**角色动画建模**：在三维动画制作流程中，艺术家使用低多边形控制网格（通常只有数百至数千个面）来描述角色形状，然后在渲染时自动执行2到4级 Catmull-Clark 细分，生成光滑的高精度曲面。控制网格修改直觉直接，不需要手动调整每个高精度顶点。

**GPU 硬件细分**：DirectX 11 与 OpenGL 4.0 引入了曲面细分着色器（Tessellation Shader）管线阶段，包括 Hull Shader、固定硬件 Tessellator 和 Domain Shader。虽然这一套管线主要实现的是 PN 三角形或 Phong 细分等近似方案，但也可以配合计算着色器实现 Catmull-Clark 细分的 GPU 加速，Pixar 的 OpenSubdiv 3.0 即支持此特性。

**自适应细分**：实时应用中不需要对整个网格均匀细分。根据曲率、屏幕空间投影面积或到摄像机距离动态调整细分级别（LOD），平坦区域保持低细分层数，弯曲明显的区域提升层数，从而在帧率与质量之间取得平衡。

## 常见误区

**误区一：细分级别越高，极限面就越准确**。实际上 Catmull-Clark 存在解析极限位置公式，在第1级细分后直接将顶点移到极限位置，效果等价于无限次细分，而无需真正执行高级别的迭代。继续迭代细分只会增加面数，收益迅速递减，3至4级细分在实践中已经足够接近极限面。

**误区二：Catmull-Clark 和 Loop 细分可以互换使用于任意网格**。Catmull-Clark 的设计目标是四边形（Quad）主导的网格，每次细分后所有面都变成四边形；Loop 细分则专为三角形网格设计，每次将三角形做 1-to-4 分裂。将 Loop 细分应用于含有四边形的混合网格，或将 Catmull-Clark 应用于纯三角形网格，会导致拓扑结构混乱或无法得到正确的极限面连续性保证。

**误区三：非正则顶点处细分面不光滑，因此该方法存在本质缺陷**。事实上，非正则顶点处 $C^1$ 连续（切平面连续但曲率不连续）对于大多数视觉应用已经完全足够。在均匀光照下，$C^1$ 间断处产生的曲率不连续在视觉上几乎不可察觉。只有在曲率敏感的工业设计（如汽车车身反射）场景下，才需要额外采用 $\sqrt{3}$ 细分或 T-样条等 $C^2$ 方案。

## 知识关联

**依赖的前置知识——网格表示**：细分操作直接在半边数据结构（Half-Edge Data Structure）或翼边结构上进行，需要快速访问每个顶点的邻接面和邻接边集合。Catmull-Clark 顶点点更新公式中的 $n$（顶点度）正是通过遍历半边邻接环来获取的，若底层网格表示不支持高效邻域查询，细分运算的实现代价将大幅增加。

**与样条曲面的关系**：Catmull-Clark 细分在正则网格（全部由度4顶点组成的四边形网格）上的极限面与双三次 B 样条曲面完全等价，因此可将 Catmull-Clark 理解为双三次 B 样条向任意拓扑网格的推广。Loop 细分在正则三角形网格上的极限面则等价于 Box 样条（Box Spline）曲面。

**与几何处理后续方向的关系**：掌握细分曲面的极限位置与极限法向量公式，是理解 T-样条（T-Splines）和非均匀细分方案的必要基础，后者允许局部加密而无需在整个网格上传播拓扑变化，是 IGA（等几何分析，Isogeometric Analysis）中的核心几何工具。
