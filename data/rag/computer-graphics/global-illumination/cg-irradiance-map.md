# 辐照度贴图

## 概述

辐照度贴图（Irradiance Map）是一种将半球面积分结果预计算并存储的光照缓存技术，专门用于表示场景中某一点从环境接收到的漫反射辐照度（Irradiance）。其数学定义为：对法线方向 **n** 处的辐照度 E(**n**) = ∫_Ω L(ω) cos(θ) dω，其中 L(ω) 是入射光的辐亮度，θ 是入射方向 ω 与法线 **n** 的夹角。由于漫反射表面的 BRDF 是常数（1/π），辐照度贴图实际上提前完成了这个半球积分，使得运行时漫反射光照计算退化为一次简单的贴图采样。

辐照度贴图的概念在1998年 Ramamoorthi 与 Hanrahan 的论文《An Efficient Representation for Irradiance Environment Maps》中被系统化阐述，核心发现是：漫反射辐照度在球面方向上是一个极其平滑的函数，其高频成分可被安全截断，仅用 **9个三阶球谐系数**（每个颜色通道各9个，合计27个浮点数）即可捕捉超过99%的漫反射光照信息。这一发现使辐照度贴图得以以极低的存储成本在实时渲染中广泛应用。

辐照度贴图的实际意义在于将昂贵的半球积分从每帧运行时计算转移到离线预计算阶段。对于静态场景，一张辐照度贴图只需生成一次，之后的每次漫反射查询只需传入27个浮点数并执行一次多项式求值，延迟几乎可以忽略不计，这使它成为游戏和实时渲染引擎中环境漫反射光照的标准方案。

## 核心原理

### 从环境贴图到辐照度贴图

原始的环境贴图（如 HDR 立方体贴图）存储的是来自各方向的辐亮度 L(ω)。要获得某表面点的漫反射响应，需要对该点法线方向的整个上半球进行积分，蒙特卡洛采样通常需要数百次甚至上千次贴图查询才能收敛。辐照度贴图通过**预先对每个可能的法线方向 n 完成这一积分**，将结果存储为以法线方向为索引的新贴图。典型的立方体辐照度贴图每面分辨率仅需 32×32 或 64×64 像素，因为其内容极度模糊（高频信息已被积分平滑），与原始 512×512 的环境贴图相比，存储量降低约200倍。

### 球谐函数压缩存储

辐照度贴图最常见的存储格式是低阶球谐（Spherical Harmonics, SH）系数。L 阶球谐展开需要 (L+1)² 个基函数，三阶（L=2）展开共9个系数。Ramamoorthi 证明，漫反射核函数 cos(θ) 在球谐域中衰减极快：l=0 的直流分量携带约87%的能量，l=1 分量携带约12%，l=2 分量携带约1%，l≥3 的高频成分合计贡献不超过0.1%。因此，辐照度的球谐重建公式为：

**E(n) ≈ Σ_{l=0}^{2} Σ_{m=-l}^{l} L̂_lm · Â_l · Y_lm(n)**

其中 L̂_lm 是环境光辐亮度的球谐系数，Â_l 是漫反射核的球谐投影系数（已知常数，分别为 Â_0=π, Â_1=2π/3, Â_2=π/4），Y_lm(n) 是实数球谐基函数。运行时只需将 L̂_lm 与 Â_l 合并为 27 个颜色通道系数传入 Shader，即可用一个多项式完成辐照度重建。

### 预计算流程

辐照度贴图的离线生成分为三步：首先，从场景中某位置（或将环境视为无限远）捕获一张全向 HDR 环境贴图；其次，对该贴图执行球谐投影——即将 L(ω) 与每个球谐基函数 Y_lm(ω) 的乘积在球面上数值积分，通常采用均匀球面采样（如 Hammersley 序列）以 1024 至 4096 个样本点保证精度；最后，将得到的9组 RGB 系数打包写入结构体或传入 GPU 常量缓冲区。对于需要空间变化辐照度的大型场景，则在世界空间中以光探针网格的形式，在每个探针位置分别执行上述流程。

## 实际应用

**Unreal Engine 的天空光照（Sky Light）** 默认使用三阶球谐辐照度贴图表示来自天空的漫反射贡献，整个天空的漫反射光照仅用27个浮点数描述，每帧更新成本极低。当场景启用"实时捕获"模式时，引擎会以约每秒一帧的频率重新投影球谐系数，以响应动态天空变化。

**Unity 的 Ambient Probe** 同样以9个 SH 系数存储全局环境辐照度，通过 `SHEvalLinearL0L1` 和 `SHEvalLinearL2` 两个 HLSL 函数在顶点着色器或像素着色器中重建辐照度，整个计算仅需约27次乘加运算。

在**角色漫反射光照**场景中，辐照度贴图与光探针结合使用：预先在场景关键位置放置光探针，每个探针存储一组27个系数，运行时根据角色世界坐标在相邻探针之间插值，以极低的运行时开销实现空间连续的漫反射环境光照。

## 常见误区

**误区一：辐照度贴图也能表示高光反射**。辐照度贴图专门服务于漫反射（Lambertian）表面，其预计算已将 cos(θ) 核卷积进去，输出是与视角无关的辐照度值。对于镜面高光，需要使用预滤波环境贴图（Pre-filtered Environment Map）配合 BRDF LUT，不能混用辐照度贴图，否则会导致高光计算错误。

**误区二：三阶球谐足以精确表示所有辐照度情形**。在绝大多数户外和室内漫反射场景下 L=2 的9个系数已经足够，但当环境光包含极强的方向性局部光源（如近处的聚光灯）时，低阶球谐会产生可见的低频震荡（振铃效应，Ringing Artifact），此时需要升阶至 L=4（25个系数）或改用其他基函数（如球面高斯）。

**误区三：辐照度贴图等同于 Irradiance Volume**。辐照度贴图通常指单一位置或无限远环境的辐照度预计算结果；而辐照度体积（Irradiance Volume）是将辐照度贴图扩展到三维空间网格，每个网格节点存储一套球谐系数，用于表示空间变化的辐照度分布，两者规模和用途有本质差异。

## 知识关联

辐照度贴图的理解以**全局光照概述**中的辐亮度传输方程为前提，需要清楚辐照度 E 与辐亮度 L 的积分关系，以及为何漫反射表面可以将 BRDF 提出积分号外。

向前延伸，**球谐函数**是辐照度贴图压缩存储的数学核心，理解球谐基函数的正交性、旋转不变性以及各阶频率特性，能够解释为什么三阶截断对漫反射误差如此之小（低于0.1%）。

**光探针**技术将辐照度贴图从单点扩展到空间网格，每个探针本质上就是一套辐照度贴图（SH系数组），学习光探针需要已经掌握单点辐照度贴图的生成与查询方式，才能理解探针插值与混合的正确做法。
