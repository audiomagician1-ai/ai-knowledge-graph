# 菲涅尔效应

## 概述

菲涅尔效应描述了光线在两种介质交界面处，反射率随观察角度变化而变化的物理现象。当视线方向与表面法线夹角（掠射角）接近90°时，几乎所有材质都会呈现接近100%的镜面反射；而当视线垂直于表面时，反射率降至该材质的基础反射率F₀。这一现象由法国物理学家奥古斯丁-让·菲涅尔（Augustin-Jean Fresnel）于19世纪初通过波动光学理论推导得出。

在PBR（基于物理的渲染）材质体系中，菲涅尔效应是决定材质边缘高光强度的关键物理机制。非金属（电介质）材质如塑料、木材在垂直观察时F₀仅约为0.04（即4%反射率），但在掠射角下同样会产生强烈高光。金属材质的F₀则普遍高于0.5，且反射颜色受材质本身颜色调制。正确模拟菲涅尔效应，是区分金属与非金属材质视觉差异的核心物理依据。

## 核心原理

### 完整菲涅尔方程

完整菲涅尔方程将光分解为s偏振和p偏振两个分量，分别计算反射率：

- **s偏振反射率**：
  Rs = [(n₁cosθᵢ - n₂cosθₜ) / (n₁cosθᵢ + n₂cosθₜ)]²

- **p偏振反射率**：
  Rp = [(n₁cosθₜ - n₂cosθᵢ) / (n₁cosθₜ + n₂cosθᵢ)]²

其中 n₁、n₂ 分别为入射介质和折射介质的折射率，θᵢ 为入射角，θₜ 为折射角（由斯涅尔定律 n₁sinθᵢ = n₂sinθₜ 确定）。最终菲涅尔反射率取两分量的算术平均：F = (Rs + Rp) / 2。

对于导体（金属），折射率为复数形式 ñ = n + ik，其中 k 为消光系数，完整计算涉及复数运算，实时渲染中计算成本极高。

### Schlick近似公式

1994年，Christophe Schlick 在论文 *An Inexpensive BRDF Model for Physically-based Rendering* 中提出了一种低成本近似公式，至今仍是实时PBR渲染的标准方案：

**F(θ) = F₀ + (1 - F₀)(1 - cosθ)⁵**

其中：
- **F₀**：法线入射时的基础反射率，由两种介质的折射率决定，计算为 F₀ = ((n₁ - n₂)/(n₁ + n₂))²
- **cosθ**：半程向量（Halfway Vector）H 与视线方向 V 的点积，即 cosθ = dot(H, V)
- **指数5**：经过实验验证，幂次取5时对大多数材质的拟合误差在1%以内

该公式仅需一次幂运算和若干加减乘操作，在GPU着色器中执行成本极低，与完整菲涅尔方程的视觉差异在实际渲染中几乎不可分辨。

### F₀与材质类型的对应关系

不同材质的F₀取值差异显著，直接决定了菲涅尔曲线的起点：

- **常见电介质**：水约0.02，玻璃约0.04~0.05，皮肤约0.028，宝石约0.05~0.08
- **金属**：铁约0.56，铜约(0.95, 0.64, 0.54)（RGB三通道各不相同），金约(1.0, 0.71, 0.29)
- **半导体与特殊材质**：硅约0.35，介于金属与电介质之间

金属的F₀是带颜色的三维向量，这正是金属高光呈现材质固有颜色（金子呈金色高光）的物理原因；而非金属的F₀通常为灰度标量，高光呈现光源颜色。

## 实际应用

**水面与玻璃渲染**：在游戏引擎中渲染水面时，当摄像机以低角度观察水面，水面会呈现接近镜面的强烈反射；垂直俯视时则可透过水面看到水底，反射率显著降低。这正是Schlick公式中 (1-cosθ)⁵ 项在掠射角急剧趋近于1的直接体现。Unreal Engine的水体材质即通过Schlick近似实现该效果。

**车漆与清漆层材质（Clearcoat）**：汽车漆面由有色底漆和透明清漆层叠加而成。清漆层（折射率约1.5，F₀≈0.04）独立参与菲涅尔计算，在边缘产生白色高光，压住底层金属漆的彩色高光，这一多层菲涅尔叠加模型在Disney BRDF及UE4的Clearcoat着色模型中有完整实现。

**布料材质的逆向菲涅尔**：天鹅绒等纤维材质呈现出与光滑表面相反的特征——正面观察时最亮，边缘反而变暗。这是因为纤维微结构导致的前向散射效应，渲染时需要用修改过的、中心强而边缘弱的菲涅尔曲线来模拟，是对标准菲涅尔效应的刻意违背。

## 常见误区

**误区一：非金属材质边缘高光不需要菲涅尔计算**。实际上，哑光黑色塑料（F₀≈0.04）在90°掠射角时反射率同样趋近于100%。忽略非金属菲涅尔效应会导致材质边缘缺乏高光，视觉上显得"发闷"，与真实物理不符。

**误区二：Schlick公式中的cosθ应使用法线N与视线V的夹角**。在Cook-Torrance微表面模型中，菲涅尔项使用的是**半程向量H与视线V的夹角**（cosθ = dot(H, V)），而非宏观法线与视线的夹角。使用宏观法线会导致掠射角时高光不够强烈，破坏能量守恒。

**误区三：F₀可以任意设为0来表示"完全无反射"材质**。物理上不存在F₀=0的材质（即便最暗的煤炭F₀也约为0.02）。在PBR工作流中，将非金属F₀设为0会导致边缘完全没有菲涅尔高光，是非物理的错误设置，美术规范通常要求非金属F₀最低不低于0.02。

## 知识关联

菲涅尔效应在Cook-Torrance BRDF方程中作为独立的F项出现：BRDF = (D × F × G) / (4 × dot(N,L) × dot(N,V))，与法线分布函数D项（控制高光形状）和几何遮蔽函数G项（控制自遮挡）并列，三者共同构成微表面镜面反射模型。理解菲涅尔效应中折射率 n 对F₀的影响，将直接引出**薄膜干涉**（Thin-Film Interference）的学习——薄膜干涉正是利用菲涅尔反射在纳米级厚度薄膜上下表面的叠加干涉，产生肥皂泡、CD光盘等彩虹色泽，是菲涅尔效应在多层介质中的复杂延伸。
