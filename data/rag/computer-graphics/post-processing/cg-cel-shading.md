# 赛璐璐着色

## 概述

赛璐璐着色（Cel Shading，又称 Toon Shading）是一种将三维模型渲染成二维手绘动画风格的着色技术，其名称来源于传统动画制作中使用的赛璐璐胶片（Celluloid）——动画师将角色绘制在透明胶片上，再逐帧拍摄合成画面。赛璐璐着色的核心特征是用离散的、阶梯化的色带代替连续渐变的光照，配合明显的黑色轮廓线，从视觉上模拟手绘着色工艺的平涂分区效果。

这项技术在游戏领域的重大突破发生于 2000 年，Sega 发布的 PlayStation 2 游戏《噗哟噗哟》最早尝试了类似风格，而 2002 年的《塞尔达传说：风之杖》将赛璐璐着色推向主流，其着色模型至今仍是业界教学的经典参考案例。在实时渲染的后处理管线中，赛璐璐着色通常拆分为两个独立Pass来实现：第一个 Pass 计算阶梯化光照，第二个 Pass 绘制轮廓线描边。

赛璐璐着色的重要性在于它以极低的计算成本（相比物理渲染）实现了强烈的艺术风格标识。一个标准的三段式赛璐璐模型仅需对漫反射强度执行一次阈值分割，计算量远低于 PBR（基于物理渲染）管线，因此在移动平台和风格化游戏中被大量采用。

## 核心原理

### 阶梯化光照的数学实现

赛璐璐着色的光照阶梯化通过对朗伯漫反射值 `NdotL`（法线与光源方向的点积）执行离散化操作实现。标准公式如下：

```
diffuse = dot(N, L)            // 范围 [-1, 1]，clamp 到 [0, 1]
celColor = floor(diffuse * steps) / steps
```

其中 `steps` 是色阶数量，通常取 2（暗/亮两段）或 3（暗/中/亮三段）。`floor()` 函数将连续的光照值截断到离散区间，产生明显的色带边界。另一种常见实现是使用一张一维渐变纹理（Ramp Texture）作为查找表，将 `NdotL` 作为 UV 坐标采样，艺术家可以自由调整每段色带的宽度和颜色，比纯数学公式更具可控性。

### 轮廓线的生成方式

赛璐璐风格的黑色轮廓线有三种主流生成方式，它们在后处理管线中扮演着不同角色：

**背面扩张法（Back-Face Inflation）**：在第一个 Pass 正常渲染前向面，第二个 Pass 只渲染背向面，并在顶点着色器中将每个顶点沿法线方向向外偏移固定距离（通常为 0.02 到 0.05 单位），配合纯黑色材质，从摄像机角度观察时背面边缘恰好露出一圈黑边。这是性能最优的方案，《风之杖》采用的正是此方法。

**屏幕空间边缘检测法**：在后处理阶段对深度缓冲或法线缓冲执行 Sobel 算子或 Roberts Cross 算子卷积，检测相邻像素间的深度差或法线夹角突变点，将其标记为轮廓并用黑色覆盖。Sobel 算子使用 3×3 卷积核，能同时检测深度不连续与法线不连续两种边缘，描边质量更稳定。

**几何着色器法**：在几何着色器阶段检测相邻三角面的法线夹角，若夹角超过设定阈值（一般为 45°）则判定为折痕边，沿该边生成四边形色带。此方法生成的描边严格贴合模型几何形状，但不适用于不支持几何着色器的移动设备。

### 高光的赛璐璐化

传统 Phong 高光产生平滑衰减的光斑，赛璐璐着色要求将高光也离散化：对 Blinn-Phong 中的 `pow(NdotH, shininess)` 结果执行 `step(threshold, specular)` 函数，将高光截断为一个均匀的白色圆斑。典型参数中 `shininess` 取 128，`threshold` 取 0.8，高光从柔和渐变变为一个硬边圆形光斑，符合手绘动画中高光的表达习惯。

## 实际应用

**《罪恶装备 Xrd》的三重Pass方案**：Arc System Works 在 2014 年发布的《罪恶装备 Xrd》中将赛璐璐着色推向了近乎手绘的境界。他们的方案不仅使用 Ramp Texture 控制光照分段，还手工绘制了专属的高光贴图和阴影遮罩贴图叠加在模型表面，轮廓线宽度根据摄像机距离动态缩放（近处线细、远处线粗），使描边在任何距离下都保持视觉一致性。

**Unity 中的 URP 赛璐璐实现**：在 Unity Universal Render Pipeline 中，赛璐璐着色通常以自定义 Unlit Shader 实现：在 Fragment Shader 中对 `_MainLightDirection` 与顶点法线做点积，用 `smoothstep(0.48, 0.52, NdotL)` 代替 `step` 使色带边界带有约 0.04 宽度的柔化过渡，避免完全硬边在抗锯齿不足时出现的锯齿闪烁。

**移动端的 Ramp Texture 优化**：在移动平台上，将 4 个不同材质的光照 Ramp 合并进一张 RGBA 四通道纹理，一次采样可同时获取皮肤、布料、金属、头发四种材质的着色结果，将采样次数从 4 次压缩为 1 次，显著降低带宽压力。

## 常见误区

**误区一：认为色阶越多越好**
初学者常认为增加色阶数（如从 3 段增加到 8 段）能让画面更精细。然而当 `steps` 超过 5 时，阶梯边界变得密集，视觉上已接近连续渐变，丧失了赛璐璐着色扁平化的标志特征，反而变成了劣质的 Phong 着色。大多数成熟风格化作品坚持使用 2 至 3 段色阶。

**误区二：将描边粗细设为固定屏幕像素宽度**
用背面扩张法时若偏移量设为世界空间固定值（如 0.05 单位），远处模型描边会变得极细甚至消失，近处则过粗。正确做法是将偏移量乘以顶点到摄像机的距离（`distance(worldPos, cameraPos)`），或在后处理描边中将 Sobel 检测阈值与深度值联动，使描边宽度在屏幕空间保持视觉恒定。

**误区三：赛璐璐着色不需要法线贴图**
由于阶梯化光照对微小的法线变化不敏感，部分开发者认为法线贴图在赛璐璐风格中无效。实际上高分辨率法线贴图仍然影响阴影分界线的形状——法线贴图会改变 `NdotL` 的逐像素分布，从而改变色带边界的轮廓走向，为布料褶皱、肌肉轮廓等细节增加表现力。

## 知识关联

赛璐璐着色以**描边效果**为前置知识，描边效果中介绍的背面扩张法和屏幕空间边缘检测均直接服务于赛璐璐的轮廓线生成，赛璐璐着色则将这两者与阶梯化光照整合为一套完整的风格化渲染方案。

在实现路径上，赛璐璐着色属于后处理管线的组合应用：轮廓线生成依赖后处理阶段对深度缓冲和法线缓冲的读取，而 Ramp Texture 的光照离散化则发生在几何渲染阶段，两者共同构成最终的赛璐璐画面。若要进一步拓展，可以探索基于机器学习的自动 Ramp 生成，或将赛璐璐着色与屏幕空间环境光遮蔽（SSAO）结合，为手绘风格增加体积感阴影，但这些已超出标准赛璐璐技术的范畴，进入了风格化渲染的前沿研究领域。
