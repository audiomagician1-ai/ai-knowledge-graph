# 赛璐璐着色

## 概述

赛璐璐着色（Cel Shading，又称 Toon Shading）是一种模拟手绘动画赛璐璐胶片风格的非真实感渲染技术，其核心特征是将连续的光照强度量化为有限数量的离散色阶，配合硬边轮廓线，产生类似日本动画的平涂效果。"赛璐璐"得名于20世纪早期动画工业中使用的透明醋酸纤维素（Cellulose Acetate）薄片，画师在其上涂抹平滑均匀的颜色区块，不存在渐变过渡。现代游戏中的赛璐璐着色正是对这一手工工艺的数字还原。

该技术在2000年由Cel Damage和《塞尔达传说：风之杖》（2002）等游戏带入主流视野，尤其是《风之杖》将赛璐璐着色应用于完整3D场景，证明了该风格在商业项目中的可行性。赛璐璐着色的重要性在于它是一种刻意"欺骗"人眼的技术：通过强制压缩光照信息，它将三维模型在视觉上"压平"为二维插画，服务于特定的美术叙事而非追求物理真实。

## 核心原理

### 阶梯化漫反射（Diffuse Quantization）

标准 Lambert 漫反射公式为 `diffuse = max(0, dot(N, L))`，输出值域为 [0, 1] 的连续浮点数。赛璐璐着色的关键变换是将该连续值映射至离散色阶：

```
quantized = floor(diffuse * levels) / levels
```

其中 `levels` 决定色阶数量，典型值为 2～4。当 `levels = 2` 时，表面只有"亮"与"暗"两种状态，视觉上产生极为硬朗的明暗分界，与动画手稿的效果高度吻合。更实用的工业做法是使用一张**Ramp Texture（渐变贴图）**：以漫反射点积结果为 UV 横坐标采样一维贴图，由美术师手动绘制阶梯形灰度，可精确控制每段阶梯的宽度与颜色倾向，比数学取整更灵活。

### 轮廓线生成

赛璐璐风格的轮廓线与描边效果紧密相关，但在后处理管线中有其特定实现路径。最常见的后处理方案是**边缘检测描边**：对深度缓冲或法线缓冲执行 Sobel 或 Roberts Cross 卷积，检测相邻像素间的法线/深度突变，突变超过阈值处绘制黑色描边。以Sobel算子为例，水平和垂直方向各用一个3×3卷积核，最终边缘强度 `E = sqrt(Gx² + Gy²)`，E 大于预设阈值时输出轮廓像素。

另一种不依赖后处理的方案是**背面扩张法（Back-Face Inflation）**：在顶点着色器中将模型背面顶点沿法线方向外扩固定距离（通常0.02～0.05单位），并以纯黑色渲染该扩张壳体，正面渲染时背面壳体恰好在轮廓处露出一圈黑边。该方法不需要额外的渲染Pass读取，但对非封闭网格或复杂拓扑效果较差。

### 高光的硬边处理

Blinn-Phong 模型的镜面高光 `specular = pow(dot(N, H), shininess)` 同样是连续值。赛璐璐着色通过阈值截断将其二值化：

```
cel_specular = step(threshold, specular)
```

`step(a, x)` 在 x < a 时返回 0，x ≥ a 时返回 1，产生一个边缘锋利的亮斑，而非传统的柔和光晕。阈值通常取 0.8～0.95，高 shininess 指数（如 128～256）配合二值化可产生小而饱满的卡通高光点，是角色面部质感的重要组成部分。

## 实际应用

在 Unity 的 URP 管线中，赛璐璐着色常通过自定义 Lit Shader 实现：在 Fragment Shader 中将 `_MainLightDirection` 与顶点法线的点积结果传入一张 `_RampTex`（1×256 像素的渐变贴图），该贴图由美术师在 Photoshop 中手绘3段色阶，中间段带有偏暖的环境光颜色，高亮段为纯白，阴影段为深蓝偏紫，整体效果比数学量化更具手绘感。

《罪恶装备 Xrd》系列（2014）对赛璐璐着色做了极致的工程化应用：角色使用多张不同的 Ramp 贴图分别控制皮肤、头发、服装的阶梯响应，并在高速运动帧中将轮廓线宽度动态加粗至1.5倍，模拟手绘动画中"动感线"的视觉冲击。这一做法已成为3A级赛璐璐游戏的行业标准参考。

在移动端开发中，后处理边缘检测描边因带宽开销较大（需要额外读取法线和深度RT），通常被替换为背面扩张法，并将扩张距离根据屏幕分辨率动态缩放，以保证在不同屏幕密度下轮廓线视觉宽度一致。

## 常见误区

**误区一：色阶越多越好**  
初学者常认为增加色阶数（如设为8层）会让渲染结果更"细腻"，但实际上超过4个色阶后，赛璐璐风格特有的平涂感会显著减弱，退化为普通低质量的Phong着色。真正的手绘动画（如吉卜力工作室的制作规范）大量使用2色阶（光影二分），3色阶仅用于强调材质厚重感，选择色阶数量应服务于美术风格而非追求"更多信息"。

**误区二：轮廓线只需要一种宽度**  
将全模型轮廓线设为统一宽度（如恒定2像素）会使角色看起来僵硬，缺乏手绘感。真实的动画制作中，轮廓线宽度随曲面曲率和视角动态变化——弯曲度高的区域（如指节、颌骨）线条更粗，平坦区域更细。在着色器中可通过计算表面法线与视线方向夹角（菲涅耳项 `1 - dot(N, V)`）来动态调节背面扩张距离，模拟这种变宽效果。

**误区三：赛璐璐着色不需要考虑法线方向**  
由于色阶的硬边特性，模型法线的微小偏差在赛璐璐渲染下会直接暴露为光影边界的锯齿跳变，远比真实感渲染更显眼。因此赛璐璐风格的模型通常需要专门烘焙"卡通法线"——将高频法线细节平滑化处理，并人工调整光影分界区域的法线朝向，确保阴影边界沿角色的设计轮廓落点，而非沿几何拓扑的随机走向分布。

## 知识关联

赛璐璐着色以**描边效果**为前置知识：后处理边缘检测的 Sobel 卷积、背面扩张的渲染顺序控制，均建立在描边效果的技术基础上。掌握描边效果的两种主要后处理管线（基于深度和基于法线的边缘检测）后，赛璐璐着色的轮廓部分可直接复用，只需叠加光照量化的 Fragment Shader 逻辑。

在更宏观的图形学后处理体系中，赛璐璐着色是**非真实感渲染（NPR）**的典型入门案例：它对光照管线的介入（量化漫反射、二值化高光）是理解油画滤镜、素描渲染、水彩模拟等更复杂NPR技术的直观参照。学习者通过赛璐璐着色理解"如何有意破坏物理正确性以达到艺术目的"，这一思维方式在整个后处理领域均有广泛迁移价值。
