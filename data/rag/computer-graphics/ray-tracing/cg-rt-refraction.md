# 光追折射

## 概述

光追折射是光线追踪渲染系统中模拟透明或半透明物体（如玻璃、水、钻石）视觉效果的核心技术。当光线击中折射介质表面时，渲染器需要计算光线在介质内部的弯曲路径，这一弯曲行为由折射率（Index of Refraction，简称IOR）决定。折射率描述了光在真空中的速度与在该介质中速度之比，例如玻璃的IOR约为1.5，水为1.333，钻石高达2.417。

折射现象的数学描述来源于17世纪荷兰数学家威里布里德·斯涅尔（Willebrord Snell），他于1621年发现了描述折射角关系的定律。在早期光线追踪系统中，Turner Whitted于1980年发表的论文《An Improved Illumination Model for Shaded Display》首次将折射光线纳入递归光追框架，这篇论文标志着真实感折射渲染的开端。

折射渲染的精确性直接影响玻璃器皿、水下场景、宝石切割面等效果的真实度。错误的折射实现会导致物体背面看起来没有放大或扭曲效果，或者玻璃球内部的焦散图案完全缺失，使场景失去物理可信度。

## 核心原理

### 斯涅尔定律与折射光线计算

斯涅尔定律的数学表达式为：

**n₁ · sin(θ₁) = n₂ · sin(θ₂)**

其中 n₁ 是入射介质的折射率，n₂ 是折射介质的折射率，θ₁ 是入射角（光线与法线的夹角），θ₂ 是折射角。在光追实现中，通常使用向量形式直接计算折射方向向量，避免显式计算三角函数。给定归一化入射方向向量 **d**、表面法线 **n** 及折射率比值 η = n₁/n₂，折射向量 **t** 的计算公式为：

**t = η·d + (η·cosθ₁ - cosθ₂)·n**

其中 cosθ₁ = -(**d**·**n**)，cosθ₂ = √(1 - η²·(1 - cosθ₁²))。GLSL和HLSL等着色语言提供了内置的`refract()`函数直接实现此计算。

### 全内反射（Total Internal Reflection）

当光线从光密介质（高IOR）射向光疏介质（低IOR）时，存在临界角 θ_c，满足：

**θ_c = arcsin(n₂ / n₁)**

当入射角超过 θ_c 时，折射光线无法存在，所有能量均被反射回入射介质，即发生全内反射。以玻璃（IOR=1.5）射向空气（IOR=1.0）为例，临界角约为41.8°。在代码实现中，当计算折射向量时 cosθ₂² < 0（即根号内为负数），即表明发生全内反射，此时应生成一条纯反射光线而非折射光线。钻石IOR高达2.417，其临界角仅约24.4°，这正是钻石切割设计能产生大量内部反射闪光的物理原因。

### 菲涅耳方程与能量分配

实际折射物体的表面同时发生反射和折射，两者之间的能量比例由菲涅耳方程决定。在实时渲染中常用Schlick近似公式计算反射率 F：

**F(θ) = F₀ + (1 - F₀)·(1 - cosθ)⁵**

其中 F₀ = ((n₁ - n₂)/(n₁ + n₂))² 是垂直入射时的基础反射率。对于玻璃（n₁=1.0, n₂=1.5），F₀ ≈ 0.04，意味着正对视角时玻璃仅反射约4%的光能量，而以掠射角观察时反射率接近100%。光追折射实现必须用随机采样或加权分配来同时生成反射和折射两条子光线，以正确模拟能量守恒。

### 色散模拟

色散是折射率随光线波长变化而产生的现象，导致白光通过棱镜后分解为彩虹色谱。阿贝数（Abbe number，V）量化了材质的色散程度：

**V = (n_d - 1) / (n_F - n_C)**

其中 n_d、n_F、n_C 分别是材质对黄光（589nm）、蓝光（486nm）、红光（656nm）的折射率。普通玻璃的阿贝数约为64，高色散玻璃可低至25。在光追中模拟色散，需要将光线分为多个波长通道分别计算不同的折射角，通常采样红（700nm）、绿（546nm）、蓝（436nm）三个代表波长即可产生明显的彩色边缘效果。

## 实际应用

**玻璃球与焦散渲染**：光线追踪折射的经典测试场景是玻璃球。一条光线射入玻璃球后，经历入射折射、内壁反射和出射折射多次弯曲，最终在球后方地面形成焦散（Caustic）亮斑。正确的IOR=1.5实现会在球后产生向前偏移约0.5个球半径的焦散集中区域，这一现象可作为折射实现正确性的定量验证。

**水下场景与全内反射混合**：模拟从水下向上看时，IOR从1.333跳变到1.0，临界角约48.6°。超出此角度的区域发生全内反射，显示水下环境；小于此角度的区域显示水面以上的折射图像，形成著名的"斯涅尔窗口（Snell's Window）"效果，直径约束在97.2°的圆锥范围内。

**钻石切割模拟**：钻石高IOR（2.417）使得光线在多个刻面之间经历大量全内反射后才从冠部射出，产生高色散彩光。渲染圆形明亮式切割钻石的57个刻面时，需要支持至少8-10次递归折射/反射弹射深度，才能捕捉完整的光路。

## 常见误区

**误区一：折射光线不需要处理全内反射分支**
许多初学者只实现了正常折射路径，未检测 cosθ₂² < 0 的情况。这导致从玻璃内部以大角度观察时，本应看到全内反射效果的区域出现错误的"黑色"或"空洞"，因为程序将虚数cosθ₂当作有效折射计算，产生NaN或非物理向量。

**误区二：忽略光线方向与折射率比值的内外判断**
折射计算中的 η = n₁/n₂ 值取决于光线是从外部进入介质还是从内部射出，必须通过点积 **d**·**n** 的正负号判断当前光线处于介质哪一侧。若始终使用 η = 1.0/1.5 而不做内外翻转，则光线离开玻璃时折射方向会完全错误，玻璃后方物体的位移扭曲会呈相反方向。

**误区三：色散模拟等于简单的RGB通道分离**
有些实现试图用不同IOR直接分别渲染红、绿、蓝通道再合并，忽略了路径的连贯性问题。正确的色散模拟需要在每一次折射弹射时都对同一波长通道使用对应的IOR，若光线经过两块折射体，第二块折射体同样需要用同一波长的IOR，而不能在不同弹射层混用不同通道的路径。

## 知识关联

**与光追反射的关系**：反射计算使用的是入射角等于反射角的镜面反射定律，仅需一个入射角参数；折射则在此基础上引入了第二介质的折射率，是反射计算的延伸扩展。全内反射将二者统一：当折射不可能发生时，系统退化为纯反射行为，因此折射实现必须内嵌完整的反射逻辑作为回退路径。

**与光谱渲染的衔接**：本章介绍的色散计算以三通道（RGB）近似为主，仅使用阿贝数模拟有限的波长依赖性。光谱渲染则将波长采样范围扩展至380nm–780nm的连续可见光谱，每条光路携带完整的光谱功率分布（SPD），能正确模拟荧光、薄膜干涉等RGB模型无法表达的现象。折射的斯涅尔定律形式在光谱渲染中保持不变，差异仅在于折射率查表的精度和波长采样密度。
