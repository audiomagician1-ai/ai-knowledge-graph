# 亲缘选择

## 概述

亲缘选择（Kin Selection）是进化生物学中解释利他行为起源的核心理论，指个体通过帮助携带相同基因的亲属繁殖，从而间接提升自身基因在后代中传播频率的自然选择过程。这一概念由英国生物学家W.D. Hamilton于1964年在论文《社会行为的遗传进化》（*The Genetical Evolution of Social Behaviour*）中正式提出，彻底改变了人们对动物利他行为的理解。

在Hamilton之前，达尔文式的自然选择理论面临一个严重的悖论：如果自然选择只偏爱提高个体存活与繁殖的特征，那么工蜂放弃自身繁殖去照顾蜂王的行为，或者松鼠发现天敌时发出警告叫声而暴露自身的行为，究竟如何能通过自然选择保留下来？亲缘选择理论通过引入"广义适合度"（inclusive fitness）这一概念，将帮助亲属繁殖的间接遗传收益纳入个体的适合度计算，完美解答了这一悖论。

亲缘选择理论的意义不仅限于解释工蜂，它还为真社会性昆虫的进化、鸟类的合作繁殖、人类的亲族利他行为等大量现象提供了统一的遗传学解释框架。

## 核心原理

### Hamilton法则

亲缘选择的核心是 **Hamilton法则**，其数学表达式为：

**rB > C**

其中：
- **r**（relatedness）= 施助者与受助者之间的亲缘系数，即两者共享同一祖先遗传下来的某一特定等位基因的概率
- **B**（benefit）= 受助者因利他行为获得的繁殖收益（以后代数量计）
- **C**（cost）= 施助者因利他行为付出的繁殖代价（以后代数量计）

当 rB > C 时，利他基因在种群中的频率将会增加，利他行为得以进化出来。举例而言，亲兄弟姐妹之间的亲缘系数 r = 0.5，半同胞兄弟 r = 0.25，亲表兄妹 r = 0.125。若一只松鼠发出警戒声音代价 C = 0.1（以减少的存活率换算），而使一个亲兄弟避免被捕食的收益 B = 0.5，则 rB = 0.5 × 0.5 = 0.25 > 0.1，警戒行为符合Hamilton法则，可以进化出来。

### 广义适合度

传统达尔文适合度（classical fitness）仅计算个体自身的直接繁殖成功率，而Hamilton提出的**广义适合度**（inclusive fitness）= 直接适合度 + 间接适合度。间接适合度指通过帮助亲属繁殖而传播到下一代的基因拷贝数，按亲缘系数 r 加权计算。这意味着一个基因"视角"下，它不在乎是通过"自己的身体"还是通过"携带同一基因的亲属的身体"进入下一代——重要的是总拷贝数增加。这一思想后来被Richard Dawkins在《自私的基因》（1976年）中以"基因的眼光"通俗化表达。

### 亲缘系数的计算

亲缘系数 r 由共同祖先的数量和传递路径决定。在二倍体有性生殖物种中，亲子之间 r = 0.5，同父同母兄弟姐妹之间 r = 0.5，祖孙之间 r = 0.25。但在膜翅目昆虫（蜂、蚂蚁、黄蜂）中，雄性由未受精卵发育（单倍体），雌性由受精卵发育（二倍体），导致姐妹之间的亲缘系数 r = 0.75，高于其与自己后代的亲缘系数 r = 0.5。这一**单倍二倍性**（haplodiploidy）特征被Hamilton用来解释为何真社会性（eusociality）在膜翅目中独立演化了至少11次之多。工蜂"宁愿"帮助母亲抚育妹妹（r = 0.75），而非自己繁殖（后代 r = 0.5），在遗传上是"合算的"。

## 实际应用

**膜翅目真社会性的进化**：蜜蜂工蜂不育并终生服务于蜂群，这一极端利他行为因单倍二倍性导致的 r = 0.75 而符合Hamilton法则。这是亲缘选择理论最经典的实证案例之一。

**贝尔丁地松鼠的警戒行为**：保罗·谢尔曼（Paul Sherman）1977年的野外研究发现，雌性贝尔丁地松鼠在亲属在场时发出警戒叫声的频率显著高于无亲属在场时，且雌性（定居型，亲属聚集）比雄性（扩散型，孤立生活）更频繁发出警报，精确地符合亲缘选择的预测。

**人类的亲族利他行为**：跨文化研究表明，人类在财产继承、育儿援助、危机救助等行为中普遍偏向近亲，且偏好程度与亲缘系数正相关。母系祖母（r = 0.25，确定血缘）比父系祖父（r = 0.25，存在亲子不确定性，实际遗传贡献平均偏低）投入更多育儿时间的现象，被演化心理学家用来支持亲缘选择框架。

**合作繁殖鸟类**：长尾山雀（*Aegithalos caudatus*）的研究显示，失去自身巢的个体更倾向于帮助亲属（r > 0）的巢，而非帮助非亲属的巢，直接证明了亲缘关系在帮助行为决策中的作用。

## 常见误区

**误区一：亲缘选择等同于"亲属偏好"的有意识决策**。亲缘选择是基因层面的自然选择过程，不要求个体有意识地识别亲缘系数或计算Hamilton法则。动物通过嗅觉标记、共同成长环境、外貌相似性等近似性线索（proximate cues）识别亲属，利他行为的进化发生在基因频率层面，而非认知决策层面。

**误区二：亲缘系数越高，利他行为一定越强**。Hamilton法则 rB > C 表明，高 r 值只是促进利他行为的条件之一，受助者的收益 B 和施助者的代价 C 同样关键。对一个 r = 0.5 的兄弟给予代价极高的帮助（如牺牲自己所有后代），不一定符合Hamilton法则；而对 r = 0.125 的表亲给予代价极低、收益极高的帮助则完全可能符合。

**误区三：亲缘选择是群体选择的一种形式**。亲缘选择与群体选择是两种不同的机制。亲缘选择在个体或基因层面运作——个体通过帮助亲属传播自身基因，这与群体整体利益无关。数学上可以证明，亲缘选择模型（Hamilton 1964）与某些群体选择模型在特定条件下结果等价，但二者的逻辑基础和假设条件存在根本差异，不可混淆。

## 知识关联

亲缘选择理论直接建立在自然选择的基础上：自然选择以基因为单位筛选适合度，而亲缘选择扩展了"适合度"的定义，将基因通过亲属传播的间接途径纳入计算。理解孟德尔遗传中的基因分离定律是计算亲缘系数 r 的前提——正是基因的50%概率传递规律决定了亲子间 r = 0.5 的数值。

亲缘选择理论与互惠利他主义（reciprocal altruism，由Trivers于1971年提出）共同构成解释生物利他行为的两大支柱：前者解释针对遗传相关个体的利他行为，后者解释针对非亲属个体的利他行为。两种机制在自然界中可以同时作用，如灵长类动物的社群行为往往同时涉及亲缘帮助和互惠交换。亲缘选择理论还直接推动了社会生物学（sociobiology）和演化心理学（evolutionary psychology）这两个交叉学科在20世纪70年代后的兴起，Wilson的《社会生物学》（1975年）正是大量援引Hamilton的亲缘选择框架来分析动物乃至人类的社会行为。
