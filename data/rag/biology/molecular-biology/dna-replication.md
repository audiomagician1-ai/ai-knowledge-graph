# DNA复制

## 概述

DNA复制是细胞在分裂前将遗传信息精确加倍的核心过程：以亲代双链DNA的两条链各自为模板，按碱基互补配对规则（A配T，G配C）各合成一条新链，最终产生两个序列与亲代完全相同的子代DNA双链分子。该过程发生于细胞周期的**S期（DNA合成期）**，在真核生物中完成于细胞核内，在原核生物中发生于拟核的类膜结合区域。

历史节点值得精确记录：1953年Watson与Crick在《Nature》发表双螺旋模型论文时，即在文末写下"我们注意到所假设的特定配对方式直接暗示了遗传物质可能的复制机制"。1958年，Matthew Meselson与Franklin Stahl完成了被Francis Crick誉为"生物学史上最优雅实验"的密度梯度离心实验，以¹⁵N（重氮）标记亲代大肠杆菌DNA，转入¹⁴N培养基繁殖一代后，离心带出一条位于轻链与重链之间的中间密度条带，繁殖两代后出现中间密度与轻密度各占一半的两条带——与半保留模型预测完全吻合，彻底排除了全保留复制（子代应出现纯重带与纯轻带）和弥散复制（每代均为均匀中间密度）两种备选模型（Meselson & Stahl, 1958）。

## 核心原理

### 半保留复制的分子基础

半保留复制并非一种假设上的可能，而是由碱基互补配对的物理化学本质决定的必然结果。亲代双链在解旋酶作用下局部分离，暴露出两条反向平行的单链模板：一条为3'→5'方向，另一条为5'→3'方向。新链上每个碱基的掺入均由氢键能量驱动：A-T之间形成2个氢键，G-C之间形成3个氢键，G-C碱基对因此更稳定，这也是GC含量较高的基因组复制保真度相对更高的原因之一。

Meselson-Stahl实验的关键定量数据：¹⁵N标记的DNA（重链）在CsCl密度梯度离心中沉降位置对应密度约1.724 g/cm³，¹⁴N标记的DNA（轻链）约1.710 g/cm³，杂合链（含一条重链一条轻链）恰好处于1.717 g/cm³处。实验结果的精确性使其成为验证分子遗传学理论的经典范本。

### 复制起点、复制叉与复制子

DNA复制不从随机位置启动，而从专一的**复制起点（Origin of Replication，ori）**开始。大肠杆菌染色体全长约4.6 Mb，只有一个复制起点**oriC**（长约245 bp），其中含有9-mer保守序列（TTATCCACA）重复出现13次，供起始蛋白**DnaA**识别结合。DnaA与oriC结合后招募解旋酶DnaB，形成**复制前起始复合体（pre-RC）**，从起点向两侧各延伸一个复制叉，形成双向θ型复制结构，两叉以约1000 bp/s的速度向相反方向移动，最终在终止区域（ter序列）相遇并终止。

人类细胞基因组约为3×10⁹ bp，若仅靠一个复制叉以哺乳动物复制叉速度（约50–100 bp/s）复制，完成全部基因组需逾一年时间。实际上，人类基因组有约3万–5万个复制起点同步激活，每个复制起点与其相邻复制起点之间的复制单元称为**复制子（replicon）**，平均长度约100 kb，S期仅需约6–8小时即可完成全基因组复制。复制起点的激活受到严格的时序调控：常染色质区域通常在S期早期激活，异染色质区域（如着丝粒周围）在S期晚期激活。

复制叉前进时，解旋产生的扭转张力以正超螺旋形式积累于复制叉前方。**拓扑异构酶Ⅱ（topoisomerase II，又称DNA解旋酶）**通过切断并重接双链来释放这种张力，每次催化需消耗2个ATP分子。已解开的单链区域由**单链结合蛋白（SSB，原核）**或**复制蛋白A（RPA，真核）**稳定包覆，防止单链回折形成发卡结构或被核酸酶降解。

### DNA聚合酶的作用机制与保真性

DNA聚合酶催化的核心化学反应为：新的脱氧核苷三磷酸（dNTP）的α-磷酸基与引物/新链3'-OH发生亲核攻击，形成磷酸二酯键，同时释放焦磷酸（PPi），后者被焦磷酸酶迅速水解为2个无机磷酸分子（Pi），使反应热力学上不可逆：

$$\text{(dNMP)}_n + \text{dNTP} \xrightarrow{\text{DNA Pol}} \text{(dNMP)}_{n+1} + \text{PP}_i$$

$$\text{PP}_i + \text{H}_2\text{O} \xrightarrow{\text{焦磷酸酶}} 2\,\text{P}_i \quad (\Delta G \approx -31\,\text{kJ/mol})$$

新链只能沿**5'→3'**方向延伸，原因在于聚合酶的活性位点结构只能容纳3'-OH作为亲核基团，无法利用5'-OH。这一方向性造成两条模板链的合成方式截然不同：

- **前导链（leading strand）**：模板方向为3'→5'，新链沿复制叉移动方向连续合成，一旦引物建立后几乎不需要重新引发。
- **滞后链（lagging strand）**：模板方向为5'→3'，新链只能间歇合成一系列反向于复制叉运动方向的**冈崎片段（Okazaki fragments）**——真核生物中每段约100–200 nt，原核生物约1000–2000 nt。1968年，日本生化学家冈崎令治（Reiji Okazaki）利用放射性³H标记脉冲实验，率先在大肠杆菌中检测到这些短片段，随后这些片段以其姓氏命名。

滞后链合成后还需完成：①**引物切除**——大肠杆菌中由DNA聚合酶Ⅰ（Pol Ⅰ）的5'→3'外切酶活性切除RNA引物，同时以已有冈崎片段为引物填补缺口；②**片段连接**——DNA连接酶利用NAD⁺（原核）或ATP（真核）作为能量来源，催化相邻片段之间缺口处磷酸二酯键的形成。

大肠杆菌**DNA聚合酶Ⅲ（Pol Ⅲ）**是主要复制酶，其核心由α（聚合）、ε（3'→5'校对外切酶）、θ三个亚基组成，组装为不对称二聚体后同时合成前导链和滞后链，复制速度约1000 nt/s。ε亚基的3'→5'外切酶活性在错误碱基掺入后迅速检测错误并切除，将内在错误率从约10⁻⁵降至10⁻⁷。加上后续的**错配修复系统（MMR）**，最终综合错误率约为10⁻⁹–10⁻¹⁰，即每复制约10亿–100亿个碱基才出现一个错误。

## 关键公式与定量分析

计算复制产物数量时，若起始DNA为1个双链分子，经过$n$轮半保留复制后，得到子代DNA分子总数为$2^n$，其中含有亲代旧链的分子永远只有**2个**（各含一条亲代链），其余$2^n - 2$个分子均为两条全新链构成。

$$\text{含亲代链的分子数} = 2 \quad (\text{任意} n \geq 1)$$

$$\text{全新链构成的分子数} = 2^n - 2$$

例如，经过3轮复制后共产生$2^3 = 8$个子代DNA分子，其中含亲代旧链的仍只有2个，全新合成的有6个。这一规律是高考及竞赛中常见的定量推断题型。

## 实际应用

### PCR技术的分子设计依据

**聚合酶链式反应（PCR）**由Kary Mullis于1983年构想、1985年正式发表，其工作原理完全模拟体内DNA复制的三个核心步骤：94–98°C变性（模拟解旋酶解链）→ 50–65°C退火（引物与模板结合，模拟引发酶引物合成）→ 72°C延伸（Taq聚合酶以4'→3'方向合成新链）。每个循环使目标片段数量翻倍，30个循环理论上可将单拷贝DNA扩增至约$2^{30} \approx 10^9$倍。Taq聚合酶来源于嗜热菌*Thermus aquaticus*，最适温度75–80°C、耐受94°C，使PCR无需在每个循环中重新加入酶成为可能。

```python
# 模拟PCR循环中DNA分子数量增长（理论最大值）
def pcr_amplification(initial_copies: int, cycles: int) -> int:
    """
    计算n个PCR循环后目标片段的拷贝数（理论值）
    假设每个循环效率为100%（实际约85-95%）
    """
    return initial_copies * (2 ** cycles)

# 示例：从1个拷贝出发，经35个循环
result = pcr_amplification(1, 35)
print(f"35个循环后理论拷贝数: {result:,}")  # 输出: 34,359,738,368
```

### 端粒酶与真核染色体末端问题

线性染色体的末端复制存在**末端复制问题**：滞后链最末端的RNA引物被切除后留下的缺口无法被填补（因为没有更上游的3'-OH可供延伸），导致染色体每复制一次便缩短约50–200 bp的端粒序列。**端粒酶（telomerase）**是一种携带RNA模板（人类端粒酶RNA含5'-AAUCCC-3'重复单元）的逆转录酶，能以自身RNA为模板延伸染色体3'端的端粒序列（人类端粒重复单元为TTAGGG），弥补缩短损失。胚胎干细胞和肿瘤细胞中端粒酶高度活跃，体细胞中端粒酶活性极低，端粒缩短被视为细胞衰老的分子时钟之一（Blackburn, Greider & Szostak因此获2009年诺贝尔生理学或医学奖）。

### DNA复制与抗肿瘤药物靶点

多种抗癌药物以DNA复制相关酶为靶点。例如，**喜树碱（camptothecin）**及其衍生物（伊立替康、拓扑替康）特异性抑制**拓扑异构酶Ⅰ**，使其切断单链后无法重新连接，导致复制叉遭遇切口时发生双链断裂，从而杀伤快速分裂的肿瘤细胞。**依托泊苷（etoposide）**则靶向拓扑异构酶Ⅱ，通过相同的"毒化"机制发挥抗肿瘤效果。

## 常见误区

**误区1：两条链合成速度相同**
前导链合成连续、快速；滞后链需要反复引发冈崎片段、切除引物、填补缺口、连接，整体效率低于前导链。这正是为什么真核细胞中需要大量复制叉同步工作以维持S期时长在合理范围内。

**误区2：DNA聚合酶可以从头合成新链**
DNA聚合酶Ⅲ（及真核生物的Pol δ、Pol ε）只能将dNTP连接到已有链的3'-OH末端，**不能从零开始合成**。每条新链的起始都依赖引发酶（primase）合成5–10 nt的RNA引物，这是滞后链上冈崎片段需要大量引物的根本原因。
