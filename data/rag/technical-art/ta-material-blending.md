# 材质混合技术

## 概述

材质混合技术是指在单一材质或着色器中，将两种或多种不同的表面属性（如颜色、粗糙度、法线方向）按照特定权重进行插值合并的工艺方法。其核心目标是在不增加Draw Call的前提下，让同一个网格表面呈现出多种材质的视觉过渡效果。例如，一块岩石表面的苔藓与裸露岩石之间的自然渐变，或者土路上压出的车辙中泥土与砾石的交界区域。

该技术体系在实时渲染领域的成熟应用可追溯到2000年代中期的地形渲染需求。彼时开发者发现，使用顶点颜色作为混合权重来合并多张地表贴图，是在硬件限制条件下实现丰富地面细节的最经济方案。Unreal Engine 3在2006年前后将这套流程标准化，形成了"地形层混合"的基础范式，随后逐步演化出高度混合、法线混合等更精细的变体方案。

理解材质混合技术对技术美术至关重要，原因在于它直接决定了材质过渡区域是否看起来"真实"。一个只使用线性插值（Lerp）的过渡区域往往显得生硬，而引入高度图信息后，岩石压过泥土的边缘会呈现出符合物理直觉的侵入感，这种差异在近景或特写镜头下极为明显。

## 核心原理

### 顶点颜色混合（Vertex Color Blending）

顶点颜色混合使用网格本身的顶点颜色数据（RGBA四个通道）作为混合权重，每个通道独立控制一种材质层的影响范围。其基本公式为：

```
最终属性 = Lerp(材质A属性, 材质B属性, VertexColor.R)
```

其中 `VertexColor.R` 的值域为 [0, 1]，0代表完全使用材质A，1代表完全使用材质B。由于一个顶点颜色具有RGBA四个通道，理论上可以同时控制最多4层材质的混合，但实际使用中第四个通道（Alpha）常被保留用于其他特殊用途（如植被的透明遮罩）。

顶点颜色混合的精度受网格顶点密度限制，顶点稀疏的低模网格会导致混合边界模糊不可控。因此在Houdini或Maya中制作资产时，艺术家需要在过渡区域额外增加顶点循环（Edge Loop）来提高权重控制精度。

### 高度混合（Height-Based Blending）

高度混合引入高度贴图（Height Map）信息来替代或修正线性权重，使材质的过渡边界沿着表面的凹凸细节走。其核心计算公式为：

```
调制权重 = saturate((VertexWeight + HeightMap - 1.0 + Sharpness) / Sharpness)
```

其中 `Sharpness` 参数（通常取值范围 0.01 ~ 0.5）控制过渡边界的硬度，值越小边界越锋利。该公式的效果是：在顶点权重为0.5的过渡中点，高度图值高的区域（凸起部分）倾向于保留上层材质，低洼处则被下层材质"填充"，从而模拟出"材质积聚在凹陷处"的自然现象。

以雪地和岩石的混合为例：高度混合会让雪优先覆盖岩石的凸出棱角和顶部平台，而裂缝和凹槽中雪的积聚会明显少于简单Lerp的结果，这与现实中雪的物理行为高度一致。

### 法线混合（Normal Blending）

在合并两种材质的法线贴图时，简单地对法线向量进行线性插值会导致法线长度偏移（归一化后方向不准确）以及细节互相抵消。技术美术领域常用的正确方法是**重定向法线混合（Reoriented Normal Blending，RNB）**，其计算步骤如下：

1. 将基础法线 `n1` 的Z分量与XY分量分开处理
2. 将细节法线 `n2` 视为对 `n1` 坐标系的扰动向量
3. 最终法线 = `normalize(n1.xyz * dot(n1, n2) - n2 * n1.z)`

相比另一种常见方法"叠加混合（Additive Blending）"，RNB在两张法线的方向差异较大时（超过45度）仍能产生正确结果，不会出现法线平坦化或翻转的伪影。Unreal Engine的 `BlendAngleCorrectedNormals` 节点即实现了等效于RNB的算法。

## 实际应用

**地形多层混合**：在UE5的Landscape系统中，每个地形层使用一张 `LayerBlend` 权重图叠加高度混合，可以让草地和砾石的过渡沿着地形凹凸起伏分布，而非生硬的矩形笔刷边缘。

**建筑破损效果**：在城市场景资产中，墙壁材质（砖块+灰泥）可使用同一网格的顶点颜色R通道存储破损区域，高度混合确保砖缝凸出部分的灰泥先剥落，而砖块内部保留更多灰泥，这比手绘遮罩贴图节省大量贴图内存（通常节省一张512×512的Mask贴图）。

**角色污迹与磨损**：武器或盔甲的磨损材质常将两层材质（光洁金属 + 磨损金属）通过高度混合叠加，顶部棱角（高度图值高）更快露出磨损底层，贴近真实物理磨损规律。法线混合同时将磨损划痕的法线细节与基础铁甲法线合并，避免细节丢失。

## 常见误区

**误区一：认为高度混合可以完全替代顶点颜色权重**
高度混合只是对权重进行"锐化修饰"，它依赖于一个初始顶点权重作为偏置量。如果顶点权重为0（完全无混合），高度图再丰富也无法让第二层材质出现。两者是相乘关系而非替代关系，去掉顶点权重直接用高度图驱动混合会导致材质在整个网格上随机"闪烁"。

**误区二：法线混合直接使用 `Lerp(Normal_A, Normal_B, Alpha)`**
对法线贴图做Lerp的本质是对[-1,1]范围内的RGB数值做线性插值，中间值的向量长度会小于1，归一化后方向偏差可达10~15度（在掠射角光照下极为明显）。此外，两个细节相反的法线（如凸起和凹陷）相互Lerp会使细节完全抵消，而RNB则能保留两者的相对扰动。

**误区三：顶点颜色混合精度足够用于所有过渡情况**
顶点颜色的插值精度被每个顶点之间的距离所限制。当网格面片尺寸为1米×1米而顶点间距为0.5米时，混合边界的最小可控宽度约等于0.5米。要实现厘米级的精细过渡（如皮肤毛孔边缘的材质变化），必须改用贴图通道（如专用Mask贴图）而非顶点颜色。

## 知识关联

**前置概念——材质分层**：材质混合技术的前提是已经定义了若干独立的材质层（Layer），每层拥有完整的Base Color、Roughness、Normal等属性集合。混合技术负责的是如何在这些层之间进行权重计算，而非如何构建单一层内部的属性。若材质分层的参数定义不规范（如两层之间纹理坐标UV比例不统一），混合边界区域会出现贴图密度突变的视觉断层。

**后续概念——天气效果材质**：天气效果材质（如积雪、积水、淋湿表面）是材质混合技术的直接应用场景延伸。雪覆盖效果本质上是一个由世界空间法线Y分量（朝上的面）驱动权重、结合高度混合的动态材质混合过程；湿润效果则是在干燥材质与湿润材质（降低粗糙度约0.3~0.5，提升SSS次表面散射）之间进行基于遮罩的混合。掌握本节的三种混合方法后，天气效果材质中的参数调节逻辑将变得直接可理解。
