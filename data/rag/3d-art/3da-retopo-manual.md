# 手动拓扑重构

## 概述

手动拓扑重构（Manual Retopology）是指3D美术师通过专用工具，在已有的高模或扫描网格表面逐面片手动绘制新的低多边形网格的过程。其核心目标是将高面数的雕刻模型（通常数百万面）转化为结构清晰、面数合理、布线规律的生产用模型，同时完整保留原始形体轮廓。与自动拓扑算法不同，手动拓扑对每一个顶点的位置和每一条边的走向都拥有人工控制权，这正是其不可替代性所在。

手动拓扑作为一种系统性工作流在游戏工业管线中兴起于2000年代中后期。随着ZBrush（Pixologic，2009年发布ZBrush 3.5）等雕刻软件的普及，高模→拓扑→烘焙法线贴图的三段式工作流逐渐成为行业标准。早期美术师需要在3ds Max或Maya中贴合参考网格手动放置顶点，效率极低，每个角色头部的手动拓扑往往耗时一整天。2011年前后Blender引入PolyBuild原型功能，2016年RetopoFlow 2.0作为Blender插件正式发布，将笔触式工作模式引入手动拓扑流程，操作效率提升了40%至60%（据CG Cookie官方评测数据）。

手动拓扑的核心价值在于对边循环（Edge Loop）布局的精确控制。一个优质的游戏角色头部拓扑通常由800至1500个四边面构成，且所有面都遵循面部肌肉走向排列——这种精度是任何自动重拓扑算法在2024年以前都无法稳定实现的，因此手动拓扑至今仍是高质量角色资产的首选生产方法。

参考资料：《Digital Sculpting with Mudbox》(Nossov, 2011, Focal Press)；RetopoFlow官方文档 (Farley et al., 2022, CGCookie)。

## 核心原理

### Poly Build 工具的表面吸附工作机制

Blender的Poly Build工具依赖**表面投影吸附（Surface Snapping）**原理运作。启用"Snap to Face"模式后，新建顶点会自动投影到参考高模的最近面法线方向，确保新网格紧贴高模表面而不浮空或穿模。具体来说，Blender在每次顶点落点时执行一次射线检测（Ray Cast），将鼠标所在屏幕坐标反投影为三维射线，与高模网格求交后取最近交点作为新顶点的世界坐标。

操作时，左键点击空白区域可直接生成新的三角/四边面；左键拖拽已有顶点可在吸附状态下调整其位置；Ctrl+左键点击边可进行挤出操作——这三个动作构成Poly Build日常操作的约90%内容。值得注意的是，在拓扑曲率变化剧烈的区域（如鼻翼沟、眼睑边缘），吸附精度会因高模面的法线方向变化加剧而产生轻微偏移，此时需手动开启"Project Individual Elements"选项以逐顶点重新投影。

### RetopoFlow 的笔触绘制系统

RetopoFlow采用完全不同于点击操作的**笔触绘制逻辑**，其底层基于曲面参数化（Surface Parameterization）思想，将三维曲面展开为局部二维坐标系后再铺设四边面网格。

核心工具**Contours**允许美术师沿模型截面方向绘制一条引导线，软件自动在该截面生成均匀分布的顶点环，顶点数量可通过鼠标滚轮实时调整（默认生成12个顶点，范围3至64）。工具**Strokes**允许沿任意曲线路径铺设条状四边面带，最适合处理手臂、腿部等管状结构；**Patches**工具则专用于在已存在的四条边界边循环之间自动填充四边面网格，适合封闭耳窝、鼻孔等凹陷区域。RetopoFlow 3.0（2022年发布）支持在操作过程中实时预览UV拉伸着色（Stretch Map），帮助美术师直观评估面的比例是否均匀，UV拉伸超过20%时着色区域变为红色警示。

### 四边面优先原则与极点控制

手动拓扑的质量标准首要原则是**全四边面（All-Quad）网格**。三角面（Triangle）在蒙皮动画中会产生不可预期的形变尖锐点，N-gon（超过4条边的面）在细分曲面时会出现曲率计算错误，因此必须规划"极点（Pole）"的位置。

**极点**指连接奇数条边的顶点：连接3条边称为"3极点"（凹陷区域常见），连接5条边称为"5极点"（布线汇流或分叉时使用）。标准做法是将5极点放置在非关节区域，例如脸颊侧面（颧骨外侧约1至2个面的位置）或肩胛骨背面中央，绝对避免将其放置在膝盖、肘关节等需要弯曲变形的关节中心——极点处的顶点在骨骼权重绘制时难以获得平滑插值，会导致关节弯曲时出现明显褶皱。

控制极点的数量和位置，是衡量手动拓扑技术水平最直观的指标之一。一个标准人体头部拓扑中，极点数量应控制在8至14个，且每个极点都应对应明确的布线转向逻辑。

## 关键公式与质量指标

拓扑质量可通过**各向异性比（Aspect Ratio，AR）**量化评估，每个四边面的AR定义如下：

$$AR = \frac{L_{max}}{L_{min}}$$

其中 $L_{max}$ 为四边面最长边长度，$L_{min}$ 为最短边长度。理想四边面的 $AR = 1.0$（正方形），生产标准允许 $AR \leq 2.5$；当 $AR > 4.0$ 时，该面在UV展开时会产生严重拉伸，且在细分时引入曲率误差，必须重新布线。

在Blender中可通过以下Python脚本批量检测所有四边面的AR值，快速定位问题区域：

```python
import bpy
import bmesh

obj = bpy.context.active_object
bm = bmesh.from_edit_mesh(obj.data)

threshold = 2.5  # 设定AR警戒值
bad_faces = []

for face in bm.faces:
    if len(face.verts) == 4:
        edges = face.edges
        lengths = sorted([e.calc_length() for e in edges])
        ar = lengths[-1] / lengths[0] if lengths[0] > 0 else 999
        if ar > threshold:
            bad_faces.append((face.index, round(ar, 3)))

for idx, ar in bad_faces:
    print(f"Face {idx}: AR = {ar}  ← 超标，需重新布线")
```

运行上述脚本后，控制台将列出所有AR超标的面片索引及其具体比值，美术师可直接对照面片编号在视窗中定位并修正。

## 实际应用

### 游戏角色面部拓扑标准流程

制作用于实时渲染的游戏NPC头部时，标准操作步骤如下：

**第一步：锚定关键边循环。** 首先在眼眶和嘴部分别绘制两个完整的同心边循环环，眼眶内圈约8至10个顶点，口轮匝肌区域约10至12个顶点。这两个开口区域的边循环密度直接决定眨眼（Eye Blink）和口型动画（Viseme）的变形质量。

**第二步：放射状向外延伸布线。** 以眼眶和嘴部为起点，沿面部肌肉走向（颧大肌、笑肌方向）向外延伸边循环，与耳前区域的纵向边循环自然汇合，汇合处放置5极点进行布线转向。

**第三步：封闭额头与下颌区域。** 额头区域的横向边循环应与头顶边循环平行排列，间距均匀（AR控制在1.5以内）；下颌骨至颈部的过渡通常需要插入一至两个5极点以实现面数减少（Loop Cut Reduction）。

完成上述流程后，整个头部面数通常落在900至1200个四边面之间，完整操作时间使用Poly Build约需2至4小时，使用RetopoFlow约需1至2.5小时。

### 四足动物躯干拓扑

对犬类角色躯干使用RetopoFlow的Contours工具时，沿脊柱方向每隔约1.5至2个解剖单位绘制一条横截面引导线，每圈默认生成16至20个顶点以保证躯干曲线的还原精度。四肢根部（肩关节、髋关节处）需增加截面密度至每2cm一圈，确保蒙皮后肢体抬起时不出现网格压缩变形。整个犬类躯干（不含头部、四肢末端）的生产标准约为2000至3500个四边面。

### 硬表面与有机体混合拓扑

对机甲类角色（如科幻题材游戏中的动力装甲），硬表面部件允许局部使用N-gon，但前提是这些N-gon所在面不参与骨骼蒙皮，仅作为静态装甲板呈现。此类情况下，可在关节连接处手动插入边循环将N-gon拆分为四边面，保留表面硬边的锐利轮廓同时满足动画兼容性要求。

## 常见误区

**误区一：极点位置随意放置。** 许多初学者习惯将5极点放在视觉上"顺手"的位置，而不考虑其与关节的空间关系。正确做法是在开始拓扑前，先在参考高模上用不同颜色标注出所有关节弯曲轴线，确保所有极点距离弯曲轴线至少2个面的间距。

**误区二：边循环密度全局均匀。** 整张脸使用相同密度的面网格是资源浪费，且会导致嘴角、眼角等高变形区域面数不足，眼皮、脸颊等低变形区域面数冗余。正确做法是在关节和高变形区域集中布线（局部密度提高30%至50%），在静态平坦区域合并边循环（减少面数）。

**误区三：忽视法线方向一致性。** 使用Poly Build快速铺面时，部分面片的法线方向可能因操作顺序不同而翻转。法线翻转的面在烘焙法线贴图时会产生黑色阴影污染，必须在拓扑完成后执行"Recalculate Outside Normals"（Blender快捷键：Shift+N）统一法线方向，并逐一检查凹陷区域（鼻孔内壁、耳道）是否仍有反转面残留。

**误区四：将手动拓扑与高模直接合并。** 手动拓扑完成的低模与高模是两个独立网格，低模不应在任何情况下与高模执行布尔运算或合并操作，两者的关联仅通过烘焙流程（法线贴图、AO贴图）建立。混淆二者关系会导致烘焙时的投影距离（Extrusion/Ray Distance）设置混乱，产生法线失真。

## 知识关联

手动拓扑重构在完整的3D角色生产管线中处于承接雕刻与烘焙的关键节点，其质量直接影响后续所有环节的结果：

- **前置概念——拓扑目标**：在开始手动拓扑之前，美术师必须明确目标平台的面数预算（移动端角色头部通常≤500三角面，主机/PC端NPC头部约1500至3000三角面），以及模型是否需要支持面部动画（BlendShape/Morph Target）。这些目标参数决定边循环的密度分布策略。

- **后续概念——面部拓扑环**：本文介绍的眼眶和口轮匝肌边循环布局是面部拓扑环专题的直接前提，面部拓扑环进一步细化鼻翼软骨区域的放射状布线与额肌横向布线的衔接规则。
