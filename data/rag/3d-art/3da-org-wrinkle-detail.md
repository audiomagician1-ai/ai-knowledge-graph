# 皱纹与细节

## 概述

皱纹与细节是有机建模中专门处理皮肤表面微观特征的技术领域，涵盖法令纹、眼角鱼尾纹、额头横纹、鼻翼两侧沟纹以及毛孔等0.01mm至数毫米尺度的表面信息。这类细节通常不通过增加基础网格多边形来实现，而是依赖法线贴图（Normal Map）、置换贴图（Displacement Map）以及高度贴图（Height Map）等纹理技术将微观几何信息烘焙存储。

皱纹细节制作技术的现代体系在2000年代初期随着ZBrush软件的普及而成熟。ZBrush 2.0（2004年发布）引入了Alpha笔刷系统，允许艺术家使用皮肤扫描照片作为Alpha贴图直接在高模上绘制毛孔和褶皱纹理，这一工作流程至今仍是行业标准。人类面部皮肤包含约1500万到2000万个毛孔，真实感角色的毛孔密度与这一生物数据保持一致是现代影视级写实角色的基本要求。

在影视与游戏制作中，皮肤细节的精度直接影响观众对角色真实感的判断，因为人脑对人脸的识别极为敏感。缺少法令纹或额头皱纹的面部会触发"恐怖谷"效应（Uncanny Valley），该概念由日本机器人学家森政弘（Masahiro Mori）于1970年提出，并在三维角色领域被广泛引用（Mori, 1970）。游戏角色通常在法线贴图中存储皮肤细节，而影视CG角色则会在渲染时激活数千万面的置换细分来呈现真实毛孔几何。《数字人体建模与渲染》（Pixologic 官方培训教材，2019年版）将皮肤分层细节雕刻列为影视级角色的入门门槛技能。

## 核心原理

### 皱纹的形成规律与拓扑方向

皮肤皱纹总是沿垂直于皮肤拉伸方向生成，这一规律由皮肤弹性纤维的走向决定。额头横纹垂直于眉毛上抬的方向（水平走向），眼角鱼尾纹以放射状向颧骨扩散，法令纹从鼻翼延伸至嘴角呈弧形下坠，颈部横纹则垂直于颈部伸缩轴线分布。在ZBrush雕刻中，使用Standard笔刷或Dam_Standard笔刷绘制皱纹时，笔触方向必须遵循这些解剖规律，否则即便皱纹形状正确也会显得僵硬不自然。

面部动态皱纹（如微笑时产生的鱼尾纹）应当以比静态皱纹更浅的深度雕刻，约为静态皱纹深度的60%-70%，为后续绑定变形留出形变余量。静态法令纹通常雕刻深度在ZBrush单位0.008至0.015之间（对应ZBrush默认球体尺寸），而鱼尾纹的动态版本则控制在0.004至0.009之间。

### 毛孔的分层雕刻方法

真实皮肤的微观结构可以分为三个层次：

- **宏观层**：皱纹与面部肌肉凸起，深度0.5mm以上，在ZBrush Subdivision Level 3（约15万面）处理；
- **中观层**：皮肤纹理流向与毛孔群分布，深度0.05mm–0.5mm，在Subdivision Level 5（约600万面）处理；
- **微观层**：单个毛孔与微小皮肤凸起，深度0.01mm–0.05mm，在Level 6或Level 7（1200万面至2400万面）处理。

直接跳至高级别细分处理毛孔而忽略中观纹理，会导致毛孔漂浮在光滑皮肤上的视觉问题，这是缺乏分层意识的典型失误。使用Alpha笔刷制作毛孔时，推荐将笔刷 Focal Shift 调整到 +50 至 +70 之间，使笔触边缘过渡更柔和，避免单个毛孔边缘过于锐利。面部不同区域毛孔尺寸差异显著：鼻翼区域毛孔直径约为0.2mm–0.5mm，远大于面颊和额头区域的0.05mm–0.2mm，贴图中必须以不同密度的Alpha图层分区表达这一差异。

### 法线贴图与置换贴图的信息存储机制

完成高模雕刻后，皱纹与毛孔信息需从高模烘焙到低模可用的贴图中。法线贴图通过RGB三通道分别存储表面法线在切线空间中的 $X$、$Y$、$Z$ 偏转分量，使低模在渲染时模拟高模光照反应，但不改变实际顶点位置，无法产生真正的轮廓遮挡与自阴影。置换贴图则存储真实的几何偏移量，渲染时根据贴图数值实际推拉顶点，能产生轮廓变化和自阴影，但要求渲染器支持微多边形细分，如 Arnold 的 Adaptive Subdivision 或 RenderMan 的 Reyes 渲染模式。

影视管线通常同时使用两种贴图：用16位灰度置换贴图处理皱纹深度（精度远高于8位，可分辨 $2^{16}=65536$ 个深度级别），用切线空间法线贴图处理毛孔方向细节。烘焙置换贴图时，ZBrush 的 Map Size 建议设置为4096×4096（面部特写级别）或8192×8192（影视级别），并将 Displacement Intensity 设置为1.0以保持数值一致性，导出时选择32位EXR格式以避免精度损失。

## 关键公式与技术参数

置换贴图在渲染器中的实际顶点偏移量由以下公式决定：

$$
P_{displaced} = P_{original} + N \cdot D(u,v) \cdot S
$$

其中：
- $P_{original}$ 为原始顶点位置（三维向量）
- $N$ 为该顶点处的法线方向（单位向量）
- $D(u,v)$ 为置换贴图在 UV 坐标 $(u,v)$ 处的灰度值，范围 $[0,1]$，中性值为0.5（不偏移）
- $S$ 为置换强度缩放系数，影视项目通常设置在0.02至0.08之间（以场景单位计）

以下是在 Houdini 中通过 VEX 代码应用皮肤置换的简化写法：

```vex
// Houdini VEX - 皮肤置换着色器片段
surface skin_displacement(
    string disp_map = "";
    float disp_scale = 0.05;
)
{
    // 采样置换贴图（16位灰度EXR）
    float disp_value = texture(disp_map, s, t);
    // 中性值0.5归零处理
    float offset = (disp_value - 0.5) * 2.0 * disp_scale;
    // 沿法线方向推拉顶点
    P += normalize(N) * offset;
    // 重新计算法线（微细分细节还原）
    N = computenormal(P);
}
```

此代码中将贴图灰度值先做 `(value - 0.5) × 2.0` 的重映射，将 $[0,1]$ 范围转换为 $[-1,1]$，确保灰度0.5对应零偏移，避免模型整体膨胀或收缩。

## 实际应用

### 影视级角色流程（以《阿丽塔：战斗天使》为参考）

《阿丽塔：战斗天使》（2019年，Weta Digital制作）中的人脸皮肤细节采用了多源数据融合方案：演员罗莎·萨拉萨尔（Rosa Salazar）的面部通过多相机阵列扫描获取，扫描仪使用超过160颗相机在0.04秒内同步拍摄，获得包含毛孔几何的超高精度面部网格，随后在ZBrush中进行手工清理与细节强化，最终置换贴图分辨率达到16384×16384像素。

### 游戏级角色流程（以《最后生还者 Part II》为参考）

顽皮狗（Naughty Dog）在《最后生还者 Part II》（2020年）中的角色皮肤细节使用4096×4096法线贴图存储，法线贴图中的皱纹信息通过专属的"皱纹遮罩贴图"（Wrinkle Mask）与面部绑定联动——当角色表情变化时，遮罩贴图动态混合静息状态与最大形变状态的两张法线贴图，实现动态皱纹效果，而无需额外几何计算。

### 案例：ZBrush 中鼻翼毛孔的标准操作步骤

例如，制作鼻翼区域毛孔时：
1. 确保模型处于 Subdivision Level 7（约2400万面），导入鼻翼区域皮肤扫描Alpha（推荐使用 Texturing.xyz 的 Multichannel Face Scan 数据）；
2. 设置笔刷为 DragRect，Alpha Intensity 为18–22，Focal Shift 为 +60；
3. 在鼻翼区域以不规则间距拖拽印盖，避免重复图案（Tiling）；
4. 降低至 Subdivision Level 5，用 Smooth 笔刷轻刷一遍，强度设为5–8，消除毛孔间的过渡生硬感；
5. 在 Level 6 补充手动雕刻少量单独放大的毛孔，模拟鼻翼中央最粗大的毛孔群。

## 常见误区

### 误区一：将所有皱纹雕刻为相同深度

面部皱纹的深度受年龄、区域和动态状态三重因素影响。法令纹在60岁以上人物中深度约为鱼尾纹的1.5–2倍，而额头横纹在年轻角色中应极浅甚至仅凭纹理表现。将全脸皱纹统一雕刻至相同ZBrush深度是导致角色"雕像感"而非"皮肤感"的首要原因。

### 误区二：忽略皮肤高光对细节感知的关键作用

毛孔和细纹在皮肤表面的可见程度，70%以上依赖渲染时的高光表现，而非几何深度本身。在哑光皮肤区域（如面颊T区以外部分），即使毛孔几何完整，无正确SSS（次表面散射）和镜面反射贴图，也会导致细节消失。细节雕刻完成后必须配合 Specular Roughness 贴图（毛孔中心粗糙度约0.7，毛孔间皮肤脊粗糙度约0.4）才能在渲染中充分显现。

### 误区三：法线贴图烘焙时未对齐UV接缝

法线贴图的切线空间计算依赖UV展开的连续性。若面部UV在鼻梁或耳廓处存在尖锐接缝，且烘焙软件（如 Marmoset Toolbag 或 Substance Painter）未正确处理接缝法线平均，会在渲染时产生明显的接缝高光错误。解决方案是在烘焙前确认低模所有UV接缝处的硬边（Hard Edge）与UV边界完全对应，并在 Marmoset 烘焙选项中勾选"Match by UV Island"而非默认的"Match by Ray"。

## 知识关联

皱纹与细节制作位于头部建模完成之后的精细化阶段，与以下技术领域直接关联：

- **次表面散射（SSS）渲染**：皮肤的半透明特性使皱纹凹陷处呈现轻微血色，这需要在 Arnold 或 Cycles 的 SSS 着色器中将散射半径设置为 $[0.8, 0.4, 0.3]$（RGB，对应皮肤红/绿/蓝散射深度，单位为厘米），皮肤细节与散射贴图必须在相同UV空间对齐。
