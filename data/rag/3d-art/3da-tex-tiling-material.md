# 可平铺材质

## 概述

可平铺材质（Tileable Texture）是一种经过特殊设计的纹理贴图，其四条边缘完全匹配，当多张贴图以阵列方式拼接时不会出现肉眼可见的接缝。这类材质通常以512×512、1024×1024、2048×2048像素等2的幂次方分辨率制作，在游戏、影视和建筑可视化中被大量应用于地面、墙面、布料等重复性大面积表面。

可平铺材质的制作理念最早在20世纪90年代随实时3D游戏引擎的普及而形成规范。彼时GPU显存极为有限，开发者必须用一张128×128的砖墙贴图反复平铺来覆盖整个关卡，因此"无缝"成为纹理资产的基本质量标准。现代PBR工作流中，可平铺材质往往包含Albedo（基础色）、Normal（法线）、Roughness（粗糙度）、Metallic（金属度）等多张通道贴图，每张都必须独立满足无缝条件。

可平铺材质的核心价值在于极高的显存复用率——一张2048×2048的混凝土贴图仅占用约16MB显存（RGBA各8位），却能以任意密度覆盖数千平方米的虚拟地面，而无需为不同区域制作独立贴图。这与Trim Sheet（修边片）等非平铺方案构成互补关系。

## 核心原理

### 边缘连续性：偏移测试的数学基础

可平铺材质的无缝特性依赖于**周期性边界条件**：贴图左边缘的像素颜色与右边缘完全对称，上边缘与下边缘同理。检验这一条件最直接的方法是**偏移测试（Offset Test）**：将贴图在UV空间内向X轴和Y轴各偏移50%（即0.5个UV单位），若偏移后的图像中央出现明显十字接缝，则材质未达到无缝标准。在Photoshop中执行此操作的快捷方式是滤镜→其他→位移，分别输入图像宽度和高度的一半像素值。

Substance Designer中内置的节点`Make It Tile Patch`和`Make It Tile Photo`正是自动化处理这一周期性问题的工具：前者通过在边缘区域进行内容感知混合（Content-Aware Blending）消除接缝；后者专为照片扫描材质设计，会在边界处生成带权重的羽化叠加区，混合宽度默认为图像尺寸的20%。

### 频率分离与宏观-微观细节分层

高质量可平铺材质通常采用**频率分离**策略，将纹理内容分为低频（整体光照趋势、大面积色调变化）和高频（微细表面细节、噪点纹理）两个层级。低频信息必须刻意弱化——若一块石材贴图的左侧整体偏暗而右侧偏亮，即便边缘像素完全匹配，平铺后人眼仍会识别出重复的明暗块状图案，这种现象称为**重复感（Tiling Artifact）**。

消除低频重复感的常见方法是在Substance Designer中使用`Histogram Scan`节点将贴图的亮度分布强制压缩至接近均匀状态，或使用`Blur HQ`节点提取低频层后将其减除。实验表明，当贴图的低频分量（半径32像素以上的模糊版本）的标准差低于整体标准差的15%时，平铺重复感会显著降低。

### 法线贴图的无缝制作特殊性

法线贴图的无缝处理与颜色贴图存在本质差异。法线贴图的RGB通道分别编码表面法线在X、Y、Z方向的分量，其中R和G通道（对应切线空间的X轴和Y轴偏转）在边界处必须满足**梯度连续性**而非仅仅值连续性。若仅做像素级混合而不重新计算法线向量，边界处会出现虚假的高光凸起或凹陷条带。

在Substance Designer中正确的做法是在高度图层级完成无缝处理，再通过`Normal`节点从无缝高度图中一次性生成法线贴图，而非对两张有接缝的法线贴图直接做混合。高度图转法线的标准公式为：

```
Normal.R = -(∂H/∂x) × Intensity
Normal.G = -(∂H/∂y) × Intensity
Normal.B = sqrt(1 - R² - G²)
```

其中`Intensity`参数控制表面凹凸的视觉强度，Substance Designer中默认值为3.0。

## 实际应用

**游戏地面材质制作流程**：在制作泥土地面的可平铺材质时，通常先在Substance Designer中用`Tile Generator`节点配合`Warp`节点生成随机化的卵石或裂缝高度图，确保节点图本身在数学上即满足周期性，无需后期修补。随后叠加`Grunge Map`系列噪点节点增加表面杂质，每层Grunge节点均需开启`Tile Safe`模式以保持边缘一致性。最终输出的PBR材质套装（Albedo + Normal + Roughness + AO）通常以2048×2048分辨率导出，在Unreal Engine 5中将`Tiling`参数设为4.0意味着该贴图在每4个单位长度内重复一次。

**检验无缝质量的三步标准流程**：①在贴图查看器（如Substance 3D Painter的Viewer或Marmoset Toolbag）中将UV平铺值设为3×3或4×4，目视检查边界；②单独检查法线贴图的平铺效果，在掠射角（Grazing Angle，视线与表面夹角约5°-15°）下最容易暴露法线接缝；③将材质应用至一个简单的平面网格并在运动中观察，因人眼对动态画面中的重复图案更为敏感。

**Substance Designer中的自动化无缝验证**：可在节点图输出端连接一个`Tile Sampler`节点，将`Pattern`输入设为当前材质输出，`X/Y Amount`均设为3，即可在节点图内部实时预览3×3平铺效果，无需导出后再检验。

## 常见误区

**误区一：边缘像素完全一致就等于无缝**。许多初学者只检查贴图四条边的像素是否吻合，却忽视了低频色调差异导致的视觉重复感。一张边缘完美匹配但整体左暗右亮的木纹贴图，在4×4平铺后会呈现出明显的棋盘格状明暗分布，在技术上"无缝"但在视觉上完全失败。正确做法是在完成边缘匹配后，额外执行低频均匀化处理。

**误区二：对所有通道使用相同的无缝处理强度**。Roughness（粗糙度）和AO（环境光遮蔽）贴图的高频细节丰富程度远低于Albedo贴图，若对其使用与Albedo相同半径的混合区域（例如图像宽度的20%），会导致接缝区域的粗糙度或AO数值被过度平滑，表现为平铺时接缝位置出现异常光滑或异常明亮的条带。一般建议Roughness和AO的混合宽度不超过图像宽度的10%。

**误区三：1×1预览合格即代表材质质量过关**。在Substance Designer或Photoshop中仅以单张贴图查看时，所有内容看起来都很自然，但人眼识别重复图案的能力在多张拼接时会被激活。行业标准要求可平铺材质必须在至少3×3（即9张）的平铺条件下通过目视检查，部分高端项目（如AAA级写实游戏）要求在6×6条件下仍看不出明显重复感，这通常需要结合顶点绘制或Virtual Texture等运行时随机化技术共同实现。

## 知识关联

**与Substance Designer的关联**：可平铺材质制作高度依赖Substance Designer的程序化节点系统。理解Substance Designer中的`Tile Generator`、`Make It Tile`系列节点和`Uniform Color`等基础节点是制作高质量可平铺材质的前提。Substance Designer的图（Graph）本质上是一个函数，其输入坐标天然支持周期性采样，这使得程序化生成的材质比基于照片扫描的材质更容易实现无缝化。

**与UV展开的关联**：可平铺材质的使用前提是目标网格的UV坐标处于0-1范围之外或在引擎中设置了UV平铺值，否则贴图不会重复。地面、墙面等大面积网格通常使用**世界空间UV投射（World Space UV Projection）**，确保不同网格片段之间的贴图能够无缝对齐，这是可平铺材质在实际场景中正确应用的关键技术配合点。
