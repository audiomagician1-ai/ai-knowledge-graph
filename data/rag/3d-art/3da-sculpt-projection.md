# 细节投射

## 概述

细节投射（Detail Projection / Project Detail）是数字雕刻中一种将高精度网格上的雕刻细节"烘印"到另一个不同拓扑结构网格上的技术。其本质是通过射线检测（Ray Casting）从目标网格表面向外发射垂直法线射线，采样源网格对应位置的几何偏移量，再将该偏移写回目标网格的顶点位置。这使得雕刻师可以在两个完全不同的多边形布线结构之间迁移纹理级别的表面细节，而无需手工重新雕刻。

这一技术在 ZBrush 软件的 Project All 功能中得到最广泛的实现，该功能最早在 ZBrush 3.5（2009年发布）中以较完整的形式出现，后续版本持续改进了投射精度与边界处理算法。Blender 的多分辨率工作流中也包含类似的 Reshape 和 Project 操作，原理相同但参数命名有差异。细节投射对于重拓扑（Retopology）工作流至关重要：当雕刻师在高面数草模上完成所有细节后，需要构建干净的低面数拓扑网格供绑定和动画使用，此时就必须将原有细节从旧网格迁移到新网格，这个步骤如果用手工雕刻来完成，可能需要数十小时，而细节投射可将其压缩至数分钟。

## 核心原理

### 法线射线采样机制

细节投射的核心公式可以简化表述为：

**P'_target = P_target + d · N_target**

其中 P_target 是目标网格顶点的原始位置，N_target 是该顶点的法线方向，d 是沿法线方向与源网格表面交点的有符号距离偏移量。程序沿每个顶点法线向内和向外分别发射射线，取与源网格最近的交点，计算距离 d 后写入目标顶点。如果射线在设定的搜索半径（Search Distance）内没有命中源网格，则该顶点保持原位不动。

### 细节层级与多分辨率的关系

细节投射只能传递**等于或低于**目标网格当前细分级别所能表达的细节密度。若目标网格处于多分辨率级别2（约 4 万面），而源网格的皮肤毛孔细节需要级别6（约 400 万面）才能表达，则投射后毛孔信息会丢失。因此标准流程是：先将目标网格逐步提升细分级别，在每个级别分别执行一次 Project All，从粗到细逐层捕获对应频率的形体信息——宏观轮廓在低级别投射，微观纹理在高级别投射。ZBrush 中该流程的典型配置是从 Subdivision Level 1 开始，每提升一级执行一次投射，直至最高级别。

### 搜索半径与遮挡问题

投射搜索半径直接影响结果质量。在 ZBrush Project All 面板中，**Distance** 参数（范围 0–1，对应模型包围盒的百分比）控制射线探测的最大距离。若两网格形态差异过大（例如新拓扑与旧雕刻姿势略有不同），将 Distance 设为 0.05–0.1 通常能覆盖大多数细节偏移而不引入噪点。过大的 Distance 值会导致"穿透投射"：射线穿过近端表面后命中身体内侧的几何体，在手指缝隙、鼻孔内壁等薄壁区域产生明显凸起或凹陷错误。解决方案是结合遮罩（Masking）分区域执行投射，或启用 **Projection Strength**（投射强度）参数逐步混合。

## 实际应用

**重拓扑后细节迁移**是最典型的场景。角色雕刻完成后，美术师在原始动态网格上方手动绘制干净的四边形拓扑线（通常使用 ZBrush 的 ZRemesher 或外部软件如 Maya 的 Quad Draw 工具），得到一张约 8,000–20,000 面的干净网格。将此网格导入 ZBrush 后作为 SubTool 叠加在雕刻原件上，再将其细分至与原件相当的级别，对齐后执行 Project All，即可将皱纹、肌肉压痕、皮肤质感等所有细节从乱序的旧拓扑完整迁移过来。

**不同姿势间的细节复用**也依赖细节投射。T-Pose 基础上雕刻的皮肤纹理需要应用到摆好动作的角色网格上时，两者姿势不同但形态相似，通过投射可以快速将纹理从 T-Pose 版本迁移到 A-Pose 版本，节省重复雕刻时间。

**Alpha 置换投射的对比**：细节投射是纯几何操作，直接修改顶点位置，不产生任何贴图文件；而使用 Alpha 贴图叠加纹理则是在雕刻笔刷中采样灰度图。前者保留了原始雕刻的所有有机不规则性，后者引入的是可重复的图案纹理，两者目的不同，不能互相替代。

## 常见误区

**误区一：认为细节投射等同于法线贴图烘焙**。法线贴图烘焙同样使用射线采样源网格，但输出是 RGB 格式的法线偏转贴图，最终效果依赖渲染时的光照计算，几何体本身并无改变。细节投射则真实地修改目标网格的顶点坐标，产生的是真实几何形状，可以被进一步雕刻、再次投射或用于生成新的烘焙。混淆两者会导致流程错误：投射之后的网格才能去做法线烘焙，顺序不可颠倒。

**误区二：一次 Project All 就能完整传递所有细节**。如前所述，每次投射只能捕获当前细分级别对应频率的细节。许多初学者在级别6一次性执行投射，结果低频形体（颧骨轮廓、鼻梁走势）因为级别1–3的逐步累积缺失而出现形变或凸起噪点，最终高频细节反而不准确。正确做法是严格按照"从低级别到高级别逐层投射"的顺序操作，每次投射前可降低 Projection Strength 到 50–70% 做多次叠加，减少单次过冲。

**误区三：两个网格必须完全对齐才能投射**。两个网格的轴心和世界坐标确实需要尽量接近，但不要求顶点数量、分布或拓扑完全相同——这正是该技术存在的意义。只要整体形态相似，搜索射线就能找到对应表面。真正需要警惕的是法线方向的翻转：如果目标网格有反转法线的面，射线会向内发射而非向外，导致局部细节反相（凹变凸）。

## 知识关联

细节投射直接依赖**多分辨率雕刻**（Multi-resolution Sculpting）的细分层级体系——没有对目标网格进行足够级别的细分，投射所获得的细节就会因顶点密度不足而被强制平滑掉，细节投射的价值也随之消失。理解多分辨率的频率分层逻辑（低级别管形体，高级别管纹理）是正确执行逐层投射流程的前提条件。

在整个角色制作管线中，细节投射是连接雕刻阶段与技术美术阶段的关键操作节点：雕刻师完成细节投射后，下游的法线烘焙、置换贴图导出、次表面散射贴图绘制才有了准确的几何基础。掌握该技术的判断标准是：能够针对不同形态差异程度的网格对，正确设定 Distance 参数和逐层投射顺序，并能识别投射错误的类型（穿透、频率缺失、法线反转），通过参数调整和局部遮罩修复。
