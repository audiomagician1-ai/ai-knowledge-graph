# 躯干建模

## 概述

躯干建模是指在三维软件中为人形角色创建胸腔、骨盆、腰腹区域以及四肢根部几何体块，并将其有机连接为单一连贯网格的过程。成年男性标准比例中，胸腔宽度约为骨盆宽度的1.3倍（女性约0.95倍），两者之间的腰部收缩区域仅由2–4排循环边（Edge Loop）支撑过渡，布线密度的分配直接决定后续骨骼绑定时的蒙皮质量。

躯干建模作为独立工序的标准化流程约在2000年代中期随Pixar《超人特攻队》（2004）的角色动画制作和游戏引擎骨骼动画支持的普及而成熟。此前，角色体形多依赖置换贴图（Displacement Map）模拟体积，而非精确的多边形体块分区。如今的躯干模型不仅需要满足静态美观，还必须保证角色从T-Pose过渡到运动极限姿态时，胸肩关节与髋关节不产生拓扑撕裂（Topology Tearing）。

以次世代游戏角色为例，完整角色约10,000–20,000个多边形面中，躯干（含四肢根部）通常占据40%–55%的面数预算，因为肩袖、腋窝和腹股沟三个高活动度关节区域各自至少需要一圈专用支撑边保证形变。学习躯干建模的核心参考文献之一是 Ami Chopine 所著《3D Art Essentials》（Focal Press, 2011），其中第7章详细讲解了人体体块比例与布线逻辑的对应关系。

## 核心原理

### 胸腔的盒型起形与肋骨弧度

胸腔建模通常从一个 **8列 × 6行** 分段的立方体（Box Primitive）开始，沿胸骨正中线切割后折叠，获得对称基形。胸腔并非简单圆柱——正视图中呈上宽下窄的倒梯形，侧视图中有约 **15°的后倾斜面**，对应现实解剖中第1肋至第12肋的斜向走势。建模时需在胸骨剑突位置（约位于锁骨到脐部全长的 **60%** 处）收缩横向循环边，以模拟肋弓下缘的内凹形态。

忽略此收缩会导致角色穿衣后腰部呈现"桶形"，即便绑定权重正确也无法展现服装的自然悬垂感。胸腔顶部锁骨凹槽（Suprasternal Notch）同样需要单独一排循环边标记，该循环边是脖子模型与躯干焊接的参考环，焊接后两侧顶点法线方向须保持连续，避免渲染时出现高光断层。

### 骨盆体块与腰腹过渡区

骨盆在三维中被抽象为上口宽、下部收紧的碗形体，髂前上棘（ASIS）是建模时必须用额外顶点标记的地形特征点，该点在人体表面是可触摸的骨性突起，出现在腰线下方约 **3–4 cm** 处，对角色腰带类道具的卡位至关重要。

腰腹过渡区的布线规则：在胸腔底部与骨盆顶部之间保持 **4–6排** 横向循环边，其中第2、3排应对齐脐孔位置（肚脐位于耻骨联合上方约 **9–10 cm**），用于支撑腹直肌分段的体积感。该区域若减少至2排以下，角色做弯腰 45° 以上动作时腹部面片将出现明显的体积塌陷（Volume Collapse），在蒙皮权重层面再如何调整也无法修复这一拓扑缺陷。

### 肩部与髋部的关节循环边系统

肩关节是躯干建模中拓扑复杂度最高的区域，需构建"双循环"结构：**外层循环边**环绕三角肌附着处，**内层循环边**紧贴肩关节窝位置。两圈循环边之间的四边形面片在绑定后承担三角肌与斜方肌蒙皮权重的渐变过渡，要求相邻面片的长宽比（Aspect Ratio）**不超过 4:1**，否则变形时出现锯齿状褶皱。

髋关节同理，需在股骨大转子（Greater Trochanter）位置保留一圈单独循环边。该循环边是腿部筒状模型与骨盆焊接的接缝所在，焊接点的顶点法线方向必须与腿筒轴向一致，偏差超过 **15°** 时渲染时将产生可见的法线缝隙（Shading Seam）。

### 四肢根部的连接拓扑与N-Gon分割法

腋窝和腹股沟的连接采用"N-Gon分割法"处理布线汇聚问题：腋窝处躯干约 **12根** 横向边与手臂约 **8根** 横向边汇聚时，需插入 1–2 个三角形或五边形作为拓扑过渡，并将这些非四边形面安置于运动幅度最小的皮肤区域（腋窝前侧而非后侧的皮肤滑动区）。

在 Autodesk Maya 中，完成此操作的标准工具链为 **Multi-Cut → Target Weld**；在 Blender 中对应 **Loop Cut + Slide** 配合手动顶点合并（Merge by Distance，阈值设为 0.001 m）；在 ZBrush 中则通过 **ZRemesher** 后手动在 GoZ 桥接的 Maya 场景中修正极点（Pole Point）位置。极点（5条或更多边汇聚的顶点）若落在高曲率形变区，蒙皮后的视觉瑕疵无法通过贴图遮盖，必须在建模阶段解决。

## 关键公式与布线计算

在规划躯干模型的面数分配时，可使用以下经验公式估算各区域的循环边需求数量：

$$
L_{required} = \left\lceil \frac{R_{joint} \times A_{max}}{K_{poly}} \right\rceil
$$

其中：
- $L_{required}$：关节区域所需的最少横向循环边数量
- $R_{joint}$：关节的标准化活动半径（肩关节取 1.0，髋关节取 0.85，脊柱取 0.6）
- $A_{max}$：该关节在动画中的最大弯曲角度（单位：度）
- $K_{poly}$：单条循环边可承载的最大弯曲角度（经验值约为 30°–35°）

**案例**：肩关节最大外展角度为 180°，$R_{joint}=1.0$，$K_{poly}=30$，则：

$$
L_{required} = \left\lceil \frac{1.0 \times 180}{30} \right\rceil = 6 \text{ 条循环边}
$$

这与业界实践中肩袖区域标准配置 **6–8 条循环边** 高度吻合。对于次世代游戏的 LOD0（最高精度层级），$K_{poly}$ 可降低至 22–25，从而将肩关节循环边提升至 8 条；对于移动端 LOD2，$K_{poly}$ 可放宽至 45，循环边数降为 4 条。

以下是一段用于在 Blender Python API 中批量检测躯干模型极点位置的脚本片段：

```python
import bpy
import bmesh

def find_poles(obj, pole_valence=5):
    """
    检测网格中的极点（5条以上边汇聚的顶点），
    返回极点在局部坐标系中的位置列表。
    pole_valence: 定义极点的最低边数，默认5
    """
    bm = bmesh.new()
    bm.from_mesh(obj.data)
    poles = []
    for vert in bm.verts:
        if len(vert.link_edges) >= pole_valence:
            poles.append(vert.co.copy())
    bm.free()
    return poles

obj = bpy.context.active_object
pole_positions = find_poles(obj, pole_valence=5)
print(f"发现 {len(pole_positions)} 个极点，坐标如下：")
for pos in pole_positions:
    print(f"  ({pos.x:.3f}, {pos.y:.3f}, {pos.z:.3f})")
```

运行此脚本后，建模师可将极点坐标与解剖参考图对照，确认所有5-极点均落在腋窝前侧褶皱或腹股沟内侧等低形变区域。

## 实际应用

### 次世代游戏角色躯干制作流程

以《黑暗之魂3》风格重甲骑士为例，裸体基础躯干（Base Mesh）面数控制在 **3,500–4,500** 个面（Poly）。制作流程分四步：

1. **体块比例阶段**：在 Maya 中用 Box Primitive 快速占位胸腔（8×6段）和骨盆（6×4段），两个体块保持独立，暂不焊接，以便通过 Move/Scale 工具反复调整比例，参照达·芬奇《维特鲁威人》图示中8头身的肩宽（约2头宽）与腰宽（约1.5头宽）。

2. **焊接与过渡区精雕阶段**：在 Maya 中执行 **Mesh → Combine → Edit Mesh → Merge Vertices**（阈值 0.01 cm），将两个体块合并为单网格，随后在腰腹区插入4排 Loop Cut，在脐孔处收缩两端顶点以形成腹直肌外侧缘的微凸形态。

3. **ZBrush 高模雕刻阶段**：将 Maya 基础网格导入 ZBrush，使用 **DynaMesh（分辨率 256）** 重拓扑后，以 **ClayBuildup + DamStandard** 笔刷雕刻胸大肌、腹外斜肌和髂嵴处的肌肉与骨性隆起，高模面数可达 200万–500万。

4. **低模重拓扑与法线烘焙**：使用 ZBrush 的 **ZRemesher**（目标面数 4,000）或 Maya 的 **Retopology 工具**生成干净四边形低模，再通过 Marmoset Toolbag 或 Substance Painter 将高模法线烘焙至低模，实现以 4,000 面呈现 500 万面细节的视觉效果。

### 影视与VFX角色躯干

影视级角色（如《复仇者联盟》系列 ILM 制作的灭霸）的躯干基础网格面数不设严格上限，Weta Digital 在制作《猩球崛起》（2011）时披露其猩猩角色单个躯干的绑定用网格约为 **15,000–18,000** 个面，重点在于腋窝与肩胛骨区域的滑动拓扑（Sliding Topology），允许肩胛骨在皮下"游走"而不产生网格穿插。

## 常见误区

**误区一：将腰部循环边数量压缩至2排以节省面数。**
后果是角色做前弯或侧弯时腹部产生体积塌陷，蒙皮权重无法修复拓扑缺陷。正确做法是在腰腹区保留 4 排循环边（约增加 40–60 个面），代价极小但收益显著。

**误区二：肩关节区域只设一圈循环边。**
单循环边结构在手臂上抬 90° 时肩部轮廓会出现明显的"棱角化"（Faceting），需要至少 **双循环（Inner + Outer）** 结构，两圈循环边间距约为 **1–1.5 cm**（以1:1比例模型计）。

**误区三：将骨盆髂嵴完全省略，以平滑面代替。**
髂嵴是服装（腰带、盔甲裙摆）的物理碰撞锚点，若无对应的几何隆起，布料模拟时腰带会出现穿模或滑落。需在髂嵴最高点处额外插入一圈循环边，并将该处顶点沿法线方向外推 **3–5 mm**。

**误区四：在 T-Pose 下只检查静态外观，未测试极端姿态变形。**
