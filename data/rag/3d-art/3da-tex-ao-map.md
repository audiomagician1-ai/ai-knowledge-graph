# AO贴图

## 概述

AO贴图（Ambient Occlusion Map，环境光遮蔽贴图）是一种灰度贴图，用于记录模型表面各点被周围几何体遮挡环境光的程度。其像素值范围从纯黑（0，完全遮蔽）到纯白（1，完全暴露），黑色区域出现在缝隙、凹槽、转角等光线难以到达的位置，如螺栓根部、布料折叠处、盔甲接缝内侧。

AO贴图的理论基础来自1998年前后渲染研究领域提出的环境光遮蔽算法，最初用于离线渲染的全局光照模拟。游戏引擎因为实时计算全局光照代价极高，所以美术师会预先将遮蔽信息烘焙成静态贴图，在运行时以极低的成本叠加到材质上，模拟出体积感和深度感。现代PBR（基于物理的渲染）工作流中，AO贴图几乎是每个资产的标配输出之一。

AO贴图的价值在于它能让平坦的着色表面产生真实的明暗层次感。一个没有AO贴图的金属头盔，其接缝处会显得飘浮而缺乏质感；加入AO后，接缝深处的暗部立即传达出材料的厚度与物理接触关系。

## 核心原理

### 遮蔽值的计算逻辑

AO的核心计算方式是：对模型表面某一点P，向其法线半球方向发射N条射线（常见取值64至512条），统计其中被周围几何体阻挡的射线比例。若64条射线中有16条被遮挡，则该点的遮蔽值为 `AO = 1 - (16/64) = 0.75`，映射到贴图上为较亮的灰色，仍有较多光线可达。公式可简写为：

```
AO(P) = 1 - (被遮蔽射线数 / 总射线数)
```

射线的最大追踪距离（Max Distance）参数直接影响遮蔽范围的大小。距离设置过大会让整个模型都变暗，过小则凹陷处无法被识别，通常以模型最小细节尺寸的2至5倍为起点进行调整。

### 在Substance Painter中烘焙AO贴图

在Substance Painter里，AO烘焙位于`Texture Set Settings → Bake Mesh Maps`面板中，具体为**Ambient Occlusion**选项。关键参数包括：

- **Secondary Rays（次级射线数）**：控制采样质量，默认值为64，高质量输出可设为256或512，数值翻倍时烘焙时间大致也翻倍。
- **Max Frontal Distance / Max Rear Distance**：分别控制法线正面和背面的遮蔽追踪深度，调整这两个值可以解决模型不同厚度区域遮蔽不准确的问题。
- **Self Occlusion**：设置为"Always"时模型自身几何体参与遮蔽计算，设置为"By Mesh Name"时可以让分开命名的子网格互相不遮蔽，常用于处理角色装备件之间的关系。
- **Distribution**：Cosine模式令靠近法线中心方向的射线密度更高，更接近物理真实；Uniform模式则均匀分布，各有适用场景。

烘焙完成后，AO贴图出现在`Shelf → Baked Meshes`中，Substance Painter会自动将其连接到材质通道供后续混合使用。

### AO贴图在材质中的混合方式

AO贴图不是独立的颜色信息，它必须与其他通道相乘（Multiply）才能发挥作用。标准做法有两种：

1. **乘以Base Color（基础色）**：将AO贴图与Albedo贴图在乘法混合模式下叠加，凹陷处颜色变暗，产生积尘和阴影感。在Substance Painter中，`Channel → AO`会通过内置的AO混合层自动以Multiply模式叠加在Base Color通道上，强度由`Color Effect → AO Intensity`滑块控制，默认为1.0。

2. **直接输入引擎材质AO通道**：在Unreal Engine 5的M_Standard材质中，AO贴图连接到`Ambient Occlusion`插槽，引擎将其与实时光照的间接光分量相乘，只压暗间接照明而不影响直接光，效果更接近物理真实。

两种方式并不互斥，高端资产常同时使用：AO乘进Albedo增加细节层次，AO输入引擎AO槽影响间接光。

## 实际应用

**武器道具制作**：制作一把步枪时，枪管与枪身连接的螺纹区域、弹匣插槽内壁、导轨凹槽底部，这些位置在AO贴图上均呈现明显黑色。美术师可以在Substance Painter中用AO贴图驱动污垢（Dirt）智能蒙版：`Smart Mask → Dirt`内部即使用AO值作为主要输入，使积垢自动聚集在遮蔽值低的区域，而无需手动绘制每一处细节。

**角色皮肤和布料**：布料折叠产生的AO暗部在视觉上传达织物的厚度和重量感。若烘焙AO时未启用高多边形（High Poly）到低多边形（Low Poly）的法线转移，皮肤或布料的细微折痕可能无法被低模几何体捕捉，此时需要将高模参与AO烘焙（即"From Mesh"模式），让高模的几何细节投影到低模AO贴图上。

**建筑与场景资产**：墙角、窗框嵌入部分、地板与墙面交界处，这些区域的AO值往往在0.2至0.5之间，形成柔和的接触暗部，是区分独立模型与集成场景质感的关键差异来源。

## 常见误区

**误区一：AO贴图可以替代光影烘焙**
AO贴图仅记录几何遮蔽关系，不包含方向性光源投射的阴影信息。一个处于阳光直射下的场景，AO只能暗化缝隙，无法表现物体在地面上投下的完整阴影。光影烘焙（Lightmap）才承担方向性阴影的存储任务，两者共同使用才能完整再现光照效果。

**误区二：AO贴图强度越高越好**
将AO强度设置到最大（即直接在Albedo上大力相乘）会导致整个模型的暗部过于死黑，细节丢失，同时在强光环境下暗部会显得不自然。通常Albedo上的AO混合强度建议控制在0.3至0.7之间，具体依照资产的材质类型和光照环境调整。金属材质比非金属对过重AO更为敏感，容易显得脏腻。

**误区三：AO烘焙出现漏光黑斑是法线问题**
初学者常将AO贴图上出现的异常黑斑归因于法线贴图错误。实际上，AO黑斑多由以下原因造成：低模与高模之间的Cage（笼体）偏移量设置不当、模型UV存在重叠导致遮蔽信息错误叠加、或Max Distance过大使远处几何体也被纳入计算。排查时应先检查UV是否有重叠（Substance Painter在烘焙日志中会报告UV重叠警告），再逐步调整Cage偏移值。

## 知识关联

学习AO贴图需要已经熟悉Substance Painter的烘焙工作流，尤其是高模/低模的对烘焙设置（By Mesh Name命名规范）以及UV展开的基本质量要求——UV接缝位置和孤岛排列会直接影响AO烘焙的边缘质量。

AO贴图与同样在Substance Painter中烘焙的**曲率贴图（Curvature Map）**形成互补：曲率贴图高亮凸起边缘、压暗凹陷边缘，侧重形状轮廓；AO贴图则关注光线可达性，侧重遮蔽深度。两者在智能材质（Smart Material）中经常同时作为蒙版输入使用，共同决定磨损、积垢等表面效果的分布位置。掌握AO贴图的烘焙与混合之后，即可进入完整的PBR多通道贴图综合运用阶段。
