# 投影展开

## 概述

投影展开（Projection Unwrap）是UV展开技术中最基础的自动化方式，其原理是将一个虚拟的几何形状（平面、圆柱、球体或立方体）从特定方向"投射"到3D模型表面，将三维坐标直接映射为二维UV坐标。与手动缝合展开不同，投影展开不需要美术师手动标记接缝，而是由软件按照几何形状的数学关系自动计算UV位置。

投影展开的概念来源于地图制图学中的地图投影理论。墨卡托投影（1569年）将球形地球投影为平面地图的方法，与今天3D软件中球形UV投影的数学基础基本相同。3D图形软件在1990年代将这一思想引入到UV坐标计算中，形成了现代DCC（数字内容创作）软件中沿用至今的四种标准投影类型。Blender、Maya、3ds Max等主流软件均内置这四种投影方式。

投影展开对于初学者尤为重要，因为它能快速为简单几何体生成可用的UV布局，且在制作地形、建筑外墙、机械零件等具有明确主方向的物体时，投影展开可以实现零拉伸或极低拉伸的贴图效果，大幅节省手动展开时间。

## 核心原理

### 平面投影（Planar Projection）

平面投影沿单一轴向（X、Y或Z轴）将模型表面"压扁"到一张平面上，其数学本质是正交投影：UV坐标直接等于世界空间坐标中对应轴向的两个分量。例如，沿Z轴投影时，顶点的U坐标 = 顶点X坐标，V坐标 = 顶点Y坐标，Z轴深度信息被完全丢弃。

平面投影的致命缺陷是**拉伸问题**：当模型表面法线与投影方向夹角超过约60°时，UV会产生严重拉伸（拉伸比例与法线夹角的余弦成反比）。因此平面投影最适用于单面平坦物体，如地面、墙面、旗帜，以及卡通风格的树叶面片等。在Maya中，平面投影默认沿Y轴（垂直向下）投射，这与游戏引擎中地形贴图的常见设置一致。

### 圆柱投影（Cylindrical Projection）

圆柱投影将模型包裹在一个虚拟圆柱体内，U坐标由顶点围绕中心轴的角度（azimuth angle）决定，V坐标由顶点沿轴方向的高度决定。其公式为：

- **U = arctan(X / Z) / (2π)**
- **V = Y / Height**

圆柱投影在接缝处（U=0与U=1的交界处）会产生一条垂直的硬边接缝，因此在使用时需要将接缝放置在模型的背面或不显眼位置。圆柱投影最适合应用于：树干、柱子、角色手臂与腿部、瓶子身体等具有明显中心轴且截面接近圆形的物体。在制作角色腿部时，美术师通常将接缝设置在大腿内侧。

### 球形投影（Spherical Projection）

球形投影使用经纬度坐标系，U坐标对应经度，V坐标对应纬度。其公式为：

- **U = (π + arctan2(Z, X)) / (2π)**
- **V = arcsin(Y / r) / π + 0.5**

球形投影在南北两极（V=0和V=1附近）会产生明显的集中收缩（极点拉伸），这与世界地图中南极洲被严重放大的原因完全相同。球形投影适用于星球、眼球、球形灯笼等接近标准球体的模型，但几乎不适用于角色头部（因为头部形态偏离球体较多）。

### 盒形/立方体投影（Box/Cubic Projection）

盒形投影将6个平面投影方向（+X、-X、+Y、-Y、+Z、-Z）同时应用于模型，每个面的多边形根据其表面法线最接近的轴向，自动归入对应的平面投影区。这使得盒形投影能够以最低拉伸处理所有朝向的表面，是上述四种方法中通用性最强的一种。

盒形投影在6个区域的边界处会出现UV接缝，因此在转角处贴图会有轻微错位。它非常适合建筑方块、箱子、石头、机械外壳等以直角面为主的物体。Blender中的"智能UV投影（Smart UV Project）"本质上是盒形投影的自动化增强版本，它在盒形投影的基础上额外处理了接缝合并与孤岛间距。

## 实际应用

**建筑场景制作**：游戏场景中的墙面通常直接使用平面投影+平铺贴图（Tiling Texture），UV不需要精确展开，仅需设置正确的投影轴向和贴图缩放比例，一块10×5米的墙面只需几秒钟即可完成UV设置。

**道具模型快速贴图**：制作一个木桶时，桶身使用圆柱投影处理侧面的木纹纹路，桶顶和桶底分别使用平面投影处理端面，合计3次投影操作即可覆盖整个木桶表面，无需手动标记任何接缝。

**地形与岩石**：开放世界游戏中的地形通常使用三平面投影（Triplanar Projection，即盒形投影在Shader中的实时版本），在Unreal Engine的地形材质编辑器中，WorldAlignedTexture节点实现的正是这一功能，能使贴图在任意坡度地形上无缝过渡。

**HDRI天空球制作**：将360°全景照片贴到天空球模型上时，球形投影能确保图片的经纬度与球体表面精确对应，这是球形投影在影视渲染场景中最标准的用途。

## 常见误区

**误区一：盒形投影适用于所有模型**。部分初学者认为盒形投影通用性强，可以直接用于所有物体。但对于有机体（角色脸部、四肢、生物皮肤），盒形投影会在6个面交界处（鼻梁、耳朵侧面等曲率变化剧烈的区域）产生明显的UV接缝与贴图错位，此类模型必须使用手动展开方式。

**误区二：投影方向不重要，软件会自动处理**。投影方向是投影展开中最关键的参数。以圆柱投影为例，若中心轴方向设置错误（如将竖直圆柱的轴设为X轴而非Y轴），UV会发生90°的严重扭曲，贴图完全无法使用。正确做法是展开前先观察模型的主要延伸轴向，手动指定对应的投影轴。

**误区三：投影展开后UV不需要检查**。许多新手在完成投影后直接进入贴图绘制阶段。实际上，投影展开后应立即在UV编辑器中检查：①UV孤岛是否有重叠（重叠会导致同一贴图区域被多个面共享，产生镜像错误）；②UV是否超出0~1范围（超出范围的UV在大多数引擎中会产生贴图平铺，在不需要平铺时会出错）。

## 知识关联

**前置知识**：学习投影展开需要先理解UV坐标系的基本概念，即UV空间是一个0到1的二维正方形，对应贴图的水平与垂直方向，以及为何3D模型需要UV才能正确显示贴图。

**后续工具**：掌握投影展开的四种类型后，可以进入展开工具的学习，包括Blender的"展开（Unwrap）"命令（基于角度加权最小二乘法的LSCM算法）以及Maya的"最优平面展开（Unfold3D）"，这些工具能处理投影展开无法应对的复杂有机体模型。同时，理解投影展开的拉伸成因，是后续学习UV拉伸检查（通过棋盘格贴图可视化拉伸区域）的必要基础。
