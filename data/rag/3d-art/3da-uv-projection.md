# 投影展开

## 概述

投影展开（Projection Unwrap）是UV展开的一种基础方法，其核心思想是将一个虚拟"投影源"从特定方向照射到三维网格上，把三维表面坐标直接"压平"成二维UV坐标。与手动缝合展开不同，投影展开不需要用户逐一标记接缝线，而是由软件根据选定的投影几何体（平面、圆柱、球形或盒形）自动计算每个顶点的UV位置。

投影展开的方法最早出现在上世纪80年代的计算机图形学研究中，随着Maya、3ds Max、Blender等DCC软件的普及，它逐渐成为美术工作流中最快速的UV生成手段。Blender的"智能UV投影（Smart UV Project）"本质上就是多个平面投影的自动化组合版本，而Maya的"平面映射（Planar Mapping）"和"圆柱映射（Cylindrical Mapping）"至今仍是角色管线中的高频操作。

投影展开的价值在于速度与可预测性：对于形状规则的物体（建筑构件、机械零件、植被面片），一次投影操作即可得到几乎无扭曲的UV，节省大量手动展开时间。但对于复杂有机体，单次投影必然产生拉伸，因此理解四种投影方式各自的适用边界，是合理选择展开策略的前提。

## 核心原理

### 平面投影（Planar Projection）

平面投影将三维坐标沿单一轴向"正射影"到一个虚拟平面上。以沿Z轴投影为例，每个顶点的UV值直接取其XY世界坐标（归一化后映射到0–1范围）：

> **U = (X − X_min) / (X_max − X_min)**
> **V = (Y − Y_min) / (Y_max − Y_min)**

这意味着顶点的Z坐标被完全丢弃。正面朝向投影方向的多边形UV间距均匀，而侧面和背面的多边形会出现严重拉伸甚至重叠（UV重叠在烘焙时会导致错误）。平面投影最适合地面贴花、墙面砖块、人物面部（配合后续手动修正侧面）等单一朝向的表面。在3ds Max中，平面映射Gizmo可以旋转45°来匹配斜面，确保投影方向与面法线尽量平行。

### 圆柱投影（Cylindrical Projection）

圆柱投影将网格包裹在一个虚拟圆柱体内侧，把三维坐标转换为角度（θ）和高度（h）两个参数：

> **U = θ / 2π = arctan(X, Z) / 2π**
> **V = (Y − Y_min) / (Y_max − Y_min)**

当物体轮廓接近圆柱——例如树干、手臂、瓶身、柱子——圆柱投影可以做到几乎零拉伸。关键限制是圆柱有一条"背缝（back seam）"，位于θ = 0/2π处，此处的面会出现UV断裂，美术人员需要手动把接缝移到视觉不重要的位置（如树干背面）。圆柱投影的帽盖（顶面与底面）必须额外用平面投影处理，这是树干或瓶子UV制作的标准两步流程。

### 球形投影（Spherical Projection）

球形投影将坐标转换为经纬度：

> **U = (π + arctan(Z, X)) / 2π**
> **V = arccos(Y / r) / π**

其中r为顶点到球心的距离。球形投影在赤道区域拉伸最小，但在两极附近（V接近0或1）会产生严重的"极点聚集"现象，导致极点纹理密度极高而中部密度不足。球形投影的典型用途是星球表面、眼球、低多边形石块等近似球形的物体。在游戏资产制作中，人眼虹膜通常用球形投影配合平铺贴图完成，因为只使用赤道±30°范围内失真最小的区域。

### 盒形投影（Box / Cubic Projection）

盒形投影本质上是6个平面投影的自动组合，按面法线方向将网格分组：法线最接近+X的面用X轴方向平面投影，最接近-Y的面用-Y轴方向平面投影，以此类推。每个面内部拉伸极小，但6个"盒面"之间的接缝处会出现UV断裂，在该区域的连续笔刷绘制会被打断。盒形投影最适合建筑外墙、箱子、硬表面机械等六面体主导的模型，Substance Painter的"Triplanar"投影贴图本质上是盒形投影的实时着色器变体，允许在不展UV的情况下直接使用世界坐标平铺纹理。

## 实际应用

**建筑场景中的墙面砖块**：一栋建筑的正面墙体直接使用平面投影（沿Z轴），UV均匀铺展后在Substance Painter中平铺砖块贴图，全程无需手动缝合，制作时间可控制在5分钟以内。侧面另建UV岛，同样用平面投影处理，最终在UV编辑器中手动拼合岛的位置。

**角色手臂**：在Maya中选中手臂圆柱段（不含手掌），执行"Cylindrical Mapping"，调整Gizmo高度使其与手臂长度对齐，将背缝移至肘内侧，接缝在正常姿态下被遮挡。手掌部分单独用平面投影或手动展开处理。

**低多边形石头**：使用Blender的"Smart UV Project"（角度阈值默认66°），该工具内部把法线角度差超过66°的面划分为独立组，分别执行平面投影再合并UV岛，适合快速生成视觉效果可接受的石块UV，但岛间接缝较多，不适合需要手绘连续细节的情况。

## 常见误区

**误区一：圆柱投影适合所有细长物体**。投影类型与物体的"截面形状"而非"长宽比"相关。一根方形截面的柱子用圆柱投影会在四个侧面产生45°方向的拉伸，正确做法是用盒形投影或对四个侧面分别执行平面投影。只有截面接近圆形时，圆柱投影才是最优选择。

**误区二：球形投影可以处理整个球体而无需担心极点**。球形投影在球体两极的UV密度是赤道的π/2（约1.57倍）以上，极点区域纹理采样率远高于中部，在最终渲染中极点区域纹理会看起来比中部"更密更小"。实际管线中，球形物体的两极通常需要独立的UV岛或手动修正极点顶点的UV位置。

**误区三：盒形投影可以完全替代手动展开**。盒形投影在6个面的交界接缝处一定存在UV断裂，烘焙法线贴图时该区域会出现明显的黑边或错误法线方向，因为烘焙器无法在断裂UV处正确插值。角色脸部、武器边缘高光区域等视觉敏感部位不能依赖盒形投影，必须使用无断裂接缝的手动展开。

## 知识关联

**前置概念：UV展开概述**。理解UV坐标系（U轴=水平0–1，V轴=垂直0–1）和UV岛的基本定义，是区分四种投影产生的UV岛形态差异的必要基础；没有这些知识，学习者无法判断投影结果的拉伸程度和接缝位置是否合理。

**后续概念：展开工具**。Blender的"展开（Unwrap）"、Maya的"Unfold3D"等基于角度或面积保持算法的展开工具，通常用于平面/圆柱投影无法覆盖的复杂有机曲面。学习展开工具时，会频繁对比"投影初始化→展开优化"的两步工作流与纯投影方法的效果差异，投影展开的适用边界将成为理解何时必须切换到算法展开的判断标准。
