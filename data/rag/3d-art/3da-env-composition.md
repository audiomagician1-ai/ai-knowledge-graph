# 场景构图

## 概述

场景构图（Scene Composition）是3D环境美术中通过摆放物体、安排视觉元素来引导玩家视线、建立空间层次感并控制叙事节奏的设计方法。区别于2D绘画构图，3D场景构图必须同时考虑玩家可移动的多个视角，而不仅仅是一个固定画面，因此设计师需要在三维空间内创造出无论从哪个主要入口进入都能成立的视觉秩序。

场景构图理论最早从电影摄影学和舞台设计借鉴而来。20世纪90年代随着3D游戏的兴起，《古墓丽影》（1996）等早期作品开始将摄影机构图理论系统化地应用于关卡设计，设计师预判玩家会在哪些"关键观察点"（Key Viewpoints）驻足，并针对这些视角专门优化场景布局。Tyke Nunez在2014年GDC演讲《Environment Art in Naughty Dog》中明确指出，《最后生还者》每个关键场景都设置了至少3个预设观察视角，美术团队针对每个视角独立验证构图质量。如今场景构图已成为环境美术中的核心基础技能，直接影响玩家是否能直觉性地理解空间走向与叙事意图。

场景构图不佳的直接后果数据上可测量：Bungie内部测试数据显示，《命运》早期原型关卡中没有引导线设计的区域，玩家平均迷路时长比有引导线设计的区域长420%。掌握场景构图能让环境美术师用最少的物件数量传达最清晰的空间信息，在多边形预算严格的移动端项目中，精准的构图设计甚至可以减少30%以上的场景填充物数量而不影响视觉丰富度。

## 核心原理

### 视觉引导线（Leading Lines）

视觉引导线是场景中能够牵引玩家视线朝特定方向移动的线性元素，包括道路边缘、围栏、河岸、建筑屋檐轮廓或光影边界。在3D场景中，引导线通常以"汇聚透视"形式呈现：多条平行线在画面深处汇聚于一个灭点（Vanishing Point），从而制造强烈的纵深感并将视线引向目标物体，如BOSS房入口、任务道具或景观地标。

有效的引导线设计遵循"三线汇聚原则"：单独一条边缘线往往不足以驱动视线，但当道路边缘、左侧建筑檐口和右侧灯柱排列同时指向同一消失点时，视线被强制聚焦的效果将显著增强。《黑暗之魂3》（2016，FromSoftware）的洛斯里克城堡入口场景是教科书级案例：石板路两侧的柱廊、城墙顶部轮廓线和前景旗帜的对角线方向，三组引导线同时汇聚于画面正中偏上约1/3处的城门，玩家几乎无法忽视这个目标。

反方向的遮挡元素（如前景的残墙断壁）能通过"局部遮挡—露出目标"的张力进一步强化引导效果。设计师将此手法称为"Peek-a-Boo Framing"（捉迷藏构图），即用前景物件遮住目标的60%-70%，仅露出足以激发好奇心的关键部分，驱使玩家主动前进探索。

### 层次感（Depth Layering）

3D场景的层次感通常划分为前景（Foreground，距摄影机0-5米）、中景（Midground，5-30米）和远景（Background，30米以上）三个标准层次。每个层次在视觉密度、细节精度和色彩饱和度上都应有明显区分：前景物件细节最丰富、饱和度最高；远景则通过大气散射（Atmospheric Scattering）呈现低饱和度的蓝灰色调，形成"空气透视"（Aerial Perspective）效果，这一规律最早由列奥纳多·达·芬奇在15世纪的绘画手记中系统描述，并被现代游戏引擎的指数雾（Exponential Height Fog）功能所模拟。

层次感的可量化标准为**明度对比差值**：

$$\Delta L = L_{foreground} - L_{background} \geq 30$$

其中 $L$ 为0-100标准化明度值（对应HSL色彩空间中的Lightness通道）。当 $\Delta L < 30$ 时，场景前后景融为一体，玩家感知到的空间深度将显著下降。《塞尔达传说：荒野之息》（2017，Nintendo EPD）的环境美术团队在GDC 2017分享中透露，该作每个开放区域的前景暗色植被与天际线亮度差值严格控制在40-55之间，这一数值区间在玩家焦点组测试中获得了最高的"空间感知清晰度"评分。

《游戏场景设计》（李涛，电子工业出版社，2020）将层次划分进一步细化为"五层模型"：极近景（0-2米）、近景（2-8米）、中景（8-25米）、远景（25-80米）、天际线（80米以上），每相邻两层之间的饱和度降低约15%-20%，可作为实际制作时调色的量化参考依据。

### 空间节奏（Spatial Rhythm）

空间节奏指玩家在场景中移动时经历的"开阔—收缩—开阔"交替变化序列。宽敞区域（如广场、山顶平台）令玩家释放紧张感并俯瞰全局；狭窄通道（如巷道、洞穴入口）则制造压迫感并聚焦注意力。这种节奏设计对应关卡设计理论中的**"三段式节奏"**：紧张（Tension）→ 释放（Release）→ 高潮（Climax），与音乐的强弱节拍在心理体验上具有相似规律（参见 Schell, 2008, *The Art of Game Design*）。

具体空间比例数据参考：相邻"开阔区域"与"收缩通道"的宽度比例建议至少达到**3:1**。若开阔区域宽度为15米，连接通道宽度应不超过5米，玩家才能产生可感知的空间收缩感。《战神》（2018，Santa Monica Studio）在每个主要战斗竞技场前均设置了宽度约3-4米、长度约8-12米的线性通道，与战斗区域的25-40米直径形成约8:1至10:1的极端宽度比，显著放大了进入战斗空间时的戏剧张力。

## 关键公式与检验方法

### 黄金分割与三分法网格

三分法（Rule of Thirds）将画面水平和垂直各三等分，形成4个交叉点，称为"趣味点"（Power Points）。场景中最重要的视觉焦点（如关键NPC、任务目标、地标建筑）应放置在距离画面边缘约1/3处，而非正中心。数学表达为：

$$x_{focal} = \frac{W}{3} \quad \text{或} \quad x_{focal} = \frac{2W}{3}$$

$$y_{focal} = \frac{H}{3} \quad \text{或} \quad y_{focal} = \frac{2H}{3}$$

其中 $W$ 为画面宽度，$H$ 为画面高度。在Unreal Engine 5中可通过 `Show → Advanced → Rule of Thirds` 叠加层直接在视口中显示三分网格，辅助场景摆放决策。

### 剪影检查代码流程（UE5 Python自动化）

在Unreal Engine 5中，可通过以下Python脚本批量将材质替换为纯色，快速进行剪影检查（Silhouette Check）：

```python
import unreal

# 获取当前关卡中所有StaticMeshActor
actor_subsystem = unreal.get_editor_subsystem(unreal.EditorActorSubsystem)
all_actors = actor_subsystem.get_all_level_actors()

# 创建纯黑剪影材质引用
silhouette_mat = unreal.load_asset("/Game/Debug/M_Silhouette_Black")

static_mesh_actors = [a for a in all_actors 
                      if isinstance(a, unreal.StaticMeshActor)]

# 批量覆盖材质
for actor in static_mesh_actors:
    mesh_comp = actor.static_mesh_component
    num_mats = mesh_comp.get_num_materials()
    for i in range(num_mats):
        mesh_comp.set_material(i, silhouette_mat)

print(f"已替换 {len(static_mesh_actors)} 个静态网格体的材质为剪影材质")
```

剪影检查的判断标准：前景、中景、远景三个层次的剪影轮廓线之间应至少保留**10%以上的画面高度**作为可见间隙。若三层剪影完全重叠，需通过调整物件高度或摄影机仰角来重新建立层次间距。

## 实际应用案例

### 案例一：《最后生还者》博物馆关卡

Naughty Dog在《最后生还者》（2013）的博物馆关卡中运用了"框景构图"（Framing Composition）技巧：玩家进入每个展厅前，都会经过一段走廊，走廊尽头的门框恰好将下一个展厅的核心展品或叙事焦点框在画面中央偏上1/3处。这种设计使门框本身成为天然的画框，提前预告下一空间的主题内容，同时引导玩家持续向前移动。据环境美术总监 Erick Pangilinan 透露，博物馆关卡的每一个门洞角度都经过至少5次迭代调整，以确保框内可见内容具有明确的叙事信息量。

### 案例二：《原神》璃月港的远景层次设计

《原神》（2020，miHoYo）璃月港的场景设计将中国山水画的"三远法"（平远、深远、高远）转化为3D构图语言：玩家站在港口码头时，前景是停靠的帆船桅杆（高饱和度暖色木材质），中景是鳞次栉比的白墙灰瓦建筑群，远景是低饱和度蓝紫色调的岩山轮廓。三个层次的明度差值约为45，色温从前景的暖黄色（约5500K）到远景的冷蓝色（约8000K）形成渐变，符合上述 $\Delta L \geq 30$ 的层次感标准，并额外利用色温变化加强了空间深度感知。

## 常见误区

**误区一：引导线越多越好。**
引导线数量超过5条且方向分散时，玩家视线会陷入"构图噪声"，失去明确焦点。正确做法是将所有引导线的方向归并为1-2个主要汇聚方向，其余线条作为从属元素保持与主引导线的夹角小于15°。

**误区二：层次感等同于物件密度。**
增加前景的模型数量并不等于建立了层次感。若前景物件与中景物件使用相同饱和度和明度的材质，无论摆放多少物件，画面依然会显得扁平。层次感的本质是**视觉属性差异**（明度、饱和度、细节密度），而非几何体数量。

**误区三：空间节奏仅由通道宽度决定。**
天花板高度（或开放天空面积）同样构成空间节奏的重要维度。封闭区域天花板高度3米与开放广场的无限天空之间的对比，在心理感受上等效于水平方向的3:1宽度比。《生化危机7》（2017，Capcom）的庄园关卡精确地利用了"低矮地下室（2.1米净高）→宽敞餐厅（4.5米净高）"的垂直节奏变化制造心理张力，而非单纯依赖平面面积的变化。

**误区四：固定玩家视角来验证构图。**
3D场景的构图验证必须在玩家实际移动路径上的多个位置动态检查，而不能仅截一张宣传截图来判定构图是否成功。建议在Unreal Engine的Sequencer中录制玩家路径摄影机动画，逐帧检查关键位置的三分法、引导线和层次感是否同时成立。
