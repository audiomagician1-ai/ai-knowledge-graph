# 对称拓扑

## 概述

对称拓扑是一种在三维角色与道具重拓扑过程中，利用模型的左右对称性，仅对半侧网格进行手动拓扑布线，再通过镜像操作生成完整低模的工作方法。这种方法将重拓扑的实际工作量直接减半，对于人体、兽形生物、机械零件等具有明显双侧对称结构的模型尤为适用。

该方法在游戏与影视行业的角色管线中被广泛采用。早期三维制作中，艺术家需对整个模型逐面手动布线；随着 ZBrush 的 ZRemesher 镜像选项、Maya 的多边形镜像命令以及 Blender 的 Mirror 修改器逐渐成熟，对称拓扑的工作流程得以标准化，成为角色美术师日常的基础操作习惯。

对称拓扑之所以重要，不仅因为它节省时间，更因为它保证了两侧布线的绝对一致性。手动对称布线时，人工误差会导致左右边数不一致或循环边走向偏差，而镜像操作从数学上保证左右顶点数、边循环结构与面数完全相同，这对后续蒙皮权重镜像（Weight Mirror）和形态键（Shape Key / Blend Shape）的正确工作至关重要。

## 核心原理

### 半模布线与对称轴设定

对称拓扑的第一步是确定对称轴，绝大多数人形角色以 X 轴为镜像轴，即 YZ 平面作为对称面。在开始布线前，需将雕刻高模沿对称轴精确对齐到世界坐标原点，确保中线顶点（如鼻尖、脊椎、胸骨）严格位于 X=0 的平面上。若高模本身存在轻微不对称，可在 ZBrush 中执行 **Geometry > Mirror And Weld**，以强制对称后再进行拓扑。

只对模型的一半（通常取 +X 侧，即右侧）进行手动布线。中线处的边必须沿 X=0 平面贴合，中线顶点的 X 坐标必须手动归零，否则镜像后会在中缝产生间隙或重叠顶点，导致法线断裂。

### 镜像操作与焊接公差

布线完成后，执行镜像命令生成 -X 侧的对称半模。镜像后需立即执行**焊接（Merge / Weld）**操作，将中线处的对应顶点合并为单一顶点。Blender 的 Mirror 修改器提供 **Merge Distance** 参数（默认值 0.001 单位），该阈值决定了哪些中线顶点会被自动焊接；若模型尺寸较大，需相应调高此值，通常设为模型总高度的 0.01% 左右。Maya 中则在 **Mesh > Mirror** 对话框中设置 **Merge Threshold**，典型值为 0.001~0.01 厘米，根据场景单位灵活调整。

焊接后应检查中缝的顶点法线是否连续，可通过显示法线方向（法线长度通常设为 0.1~0.5 单位可视化）来快速排查断裂点。

### 拓扑规律与循环边的对称保持

半模布线时，所有循环边（Edge Loop）必须以中线为终点或穿越中线。例如嘴唇的环形循环边从 +X 绕过中线后，在 -X 侧应形成完整闭合环；鼻翼处的循环边则终止于 X=0 中缝。这类设计直接决定镜像后网格是否能正确形成完整的肌肉走向循环。面部标准布线中，眼眶通常使用 6~8 个四边面构成的循环，嘴唇使用独立的 2~3 层同心循环，这些循环只需在半模上规划，镜像后自动补全。

## 实际应用

**游戏角色头部重拓扑**：以一个包含约 500 万多边形的 ZBrush 头部雕刻为例，美术师仅需对右侧半脸完成约 800~1200 个四边面的布线（含耳朵、眼睑、鼻孔等细节区域），镜像焊接后整个头部低模约为 1600~2400 个面，单侧工时通常为 3~6 小时，相比整模布线节省约 40%~50% 的时间（因中缝处理额外消耗约 10% 工时）。

**机械硬表面对称拓扑**：头盔、护甲、载具等硬表面模型同样适用。硬表面的对称拓扑中，中线处常需要保留**锐边支撑循环（Support Loop）**，镜像前需确保支撑循环的末端顶点 X 坐标归零，否则镜像后中缝处倒角宽度会出现不对称的视觉瑕疵。

**蒙皮权重镜像的前提**：在 Maya 中执行 **Skin > Mirror Skin Weights** 时，系统依赖左右两侧顶点在 X 轴上的精确对称位置进行映射。若拓扑本身通过对称拓扑方法生成，顶点的 X 对称关系由镜像操作保证，权重镜像的成功率接近 100%；若拓扑为手工双侧布线，即使视觉上对称，细微的顶点位置偏差也会导致权重映射出现错误绑定。

## 常见误区

**误区一：中线顶点未归零导致焊接失败**
许多初学者在半模布线阶段忽略将中缝顶点的 X 坐标手动归零（Snap to X=0），导致镜像后中缝两侧顶点距离超过焊接阈值，Merge 操作无法自动合并，最终模型在中线处出现可见的裂缝或法线断裂。正确做法是在布线过程中始终开启**轴向吸附（Axis Snap）**或定期使用"将选中顶点 X 坐标归零"命令（Blender 快捷键：S → X → 0 → Enter）。

**误区二：对不对称细节强行使用对称拓扑**
面部伤疤、不对称发型、单侧机械配件等细节不适合用对称拓扑处理。正确方式是先完成对称基础网格的镜像焊接，在应用（Apply）镜像修改器后，再针对不对称细节手动修改对应区域的顶点与面，而不是试图在半模阶段预留不对称变化。

**误区三：镜像后未检验拓扑流向的完整性**
镜像操作本身不会检测循环边是否正确闭合。如果半模布线时某条循环边在中线处断开而非终止于中线顶点，镜像后该循环将无法形成闭环，导致蒙皮变形时出现局部褶皱。检验方法是在完整模型上执行 **Select Edge Loop**，依次选取所有主要循环边，确认每条环形循环都能完整绕行一圈或在中线对称点处正确终止。

## 知识关联

对称拓扑以**手动拓扑重构**的基本布线技能为前提，要求操作者已熟练掌握四边面布线原则、循环边规划以及面向蒙皮与法线的布线逻辑。在具体工具层面，Blender 的 Mirror 修改器、Maya 的 Instance + Mirror 工作流以及 3ds Max 的 Symmetry 修改器均是实现对称拓扑的主要技术手段，各有微小的工作流差异但原理相同。

对称拓扑生成的网格直接服务于后续的**UV 展开**（对称 UV 可共用同一 UV 空间，贴图利用率提升约 50%）、**蒙皮绑定**（权重镜像依赖拓扑对称）以及**形态键制作**（Blend Shape 的镜像校正也依赖顶点对称性）。掌握对称拓扑后，进一步可学习利用 ZRemesher 的 **Use Polypaint** 引导线功能结合镜像，实现半自动化的对称拓扑辅助流程，大幅缩短高精度角色的重拓扑周期。
