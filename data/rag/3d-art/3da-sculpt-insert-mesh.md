# Insert Mesh笔刷

## 概述

Insert Mesh笔刷（插入网格笔刷）是ZBrush中一类特殊的雕刻工具，其核心功能是将预先制作好的三维网格模型直接"印入"当前雕刻对象的表面，而不是通过拉伸或压缩多边形来塑造形状。与普通的Clay笔刷或Standard笔刷不同，Insert Mesh笔刷携带的是一个完整的独立网格，点击表面即可将该网格融合嵌入，同时保持原始网格的拓扑结构。

这类笔刷的前身可以追溯到ZBrush 4R2版本（2011年发布）引入的IMM（Insert Multi Mesh）系统，该系统允许单个笔刷内存储多个不同的预制网格，通过滚动选择不同的部件。这一设计将重复性的细节添加工作从数分钟的手工雕刻缩短至数秒内完成。在游戏角色、概念怪物设计和硬表面道具制作中，螺钉、鳞片、皮革绳结、铆钉等细节往往需要重复出现数十乃至数百次，Insert Mesh笔刷正是解决这类需求的专用工具。

理解Insert Mesh笔刷的价值，在于认识到它改变了"每个细节都从零开始雕刻"的工作流程。当一名角色艺术家需要在盔甲上放置80颗铆钉时，使用预制铆钉Insert Mesh笔刷可以确保每颗铆钉的体积、比例完全一致，同时通过笔刷的方向控制和缩放快捷键（Alt+拖拽可缩放插入网格）灵活调整单个部件的大小。

## 核心原理

### 插入机制与布尔运算

Insert Mesh笔刷在插入网格时依赖ZBrush的DynaMesh布尔合并功能。当笔刷将预制网格放置到表面后，再次执行DynaMesh重新拓扑（快捷键Ctrl+拖拽空白区域），系统会将插入的网格与原始模型进行并集（Union）运算，使两者融为一体，接缝处自动生成过渡多边形。如果在插入前按住Alt键，则插入操作会变为差集（Subtraction）模式，即在模型表面"挖去"预制网格的形状，常用于制作孔洞、凹槽和装饰压花。

### IMM笔刷的多网格存储结构

一个IMM笔刷文件（.ZBP格式）内部可以存储最多数百个独立子网格，每个子网格在笔刷面板的M按钮菜单中以缩略图列表展示。艺术家按下M键并在预览窗口中点击目标缩略图即可切换当前激活的插入网格。Pixologic官方随ZBrush附带的IMM库包含IMM Primitives（基础几何体）、IMM Rope（绳索模块）、IMM Curve Tube（曲线管道）等多套预设库，其中IMM Rope库包含超过40种不同编织方式的绳索截面网格。

### Curve模式与路径插入

Insert Mesh笔刷可以与Stroke菜单中的Curve描边模式结合使用，激活后笔刷不再是单点插入，而是沿用户绘制的曲线路径连续重复排列预制网格。此模式下，Curve Res（曲线分辨率）参数控制单位长度内网格的重复密度，Size参数控制每个实例的缩放比例。这一组合特别适合制作链条、缝合线、脊椎骨链等有规律重复的有机或机械细节。曲线路径绘制完成后，可通过拖拽曲线控制点实时调整路径形态，所有已排列的网格实例会实时跟随更新位置。

### 插入方向控制

Insert Mesh笔刷的插入方向默认垂直于笔触点所在表面的法线方向，即预制网格会"竖立"在表面上。通过Brush菜单中的Orientation选项，可以切换为固定世界坐标轴方向插入，适用于需要所有部件保持统一朝向的硬表面场景（例如垂直向下的螺栓）。旋转插入网格的方法是在点击前按住空格键然后拖拽，可以在插入前预先旋转网格的角度。

## 实际应用

**游戏角色盔甲制作**：在制作中世纪骑士盔甲时，艺术家将提前雕刻好的六角铆钉保存为IMM笔刷。在胸甲边缘需要放置约60颗铆钉的场景中，使用Insert Mesh笔刷逐一点击定位，全程不超过3分钟，且每颗铆钉的高度和体积完全统一，随后一次DynaMesh操作将所有铆钉融合进胸甲模型。

**怪物鳞片与皮肤纹理**：ZBrush内置的Scale IMM笔刷（鳞片系列）包含鱼鳞、爬行动物鳞甲、骨质突起等数十种预制形状。结合Curve模式沿脊背绘制曲线，可以在约30秒内铺排出一整列脊椎骨质突起，这是纯手工雕刻无法达到的效率。

**硬表面道具设计**：制作科幻风格枪械时，螺丝刀槽、散热孔阵列、管道接头等标准化机械细节均可制作为IMM笔刷库。通过Alt键的减法插入模式，在金属面板上打出规则的散热孔排列，每个孔洞的深度和形状由预制网格精确控制，相比ZBrush的2.5D Alpha遮罩方式产生更真实的三维深度感。

**自定义IMM笔刷创建**：艺术家可以将任意SubTool子模型转换为Insert Mesh笔刷。操作路径为：选中目标SubTool → Brush菜单 → Create InsertMesh，系统会自动将当前模型封装进新笔刷。将多个网格合并进同一IMM笔刷时，需借助Tool → Append功能将各子模型组合后再一次性转换。

## 常见误区

**误区一：认为Insert Mesh笔刷插入后可以直接雕刻细节**。刚插入的预制网格在DynaMesh合并前是以独立网格叠加在表面的状态存在的，此时对该区域进行普通雕刻操作可能产生穿插错误或顶点乱序。正确流程是插入完毕所有部件后，统一执行一次DynaMesh（建议分辨率设置为512或以上）完成融合，再进行后续细节雕刻。

**误区二：混淆Insert Mesh笔刷与Alpha遮罩笔刷的功能边界**。Alpha遮罩笔刷（如DamStandard配合Alpha纹理）产生的是表面高度图压印效果，本质是对现有多边形的位移操作，不增加真实三维体积。Insert Mesh笔刷插入的是具有完整X/Y/Z体积的真实几何体，从任意角度观察都具有三维轮廓。因此铆钉、突起物等需要真实体积感的部件必须使用Insert Mesh而非Alpha笔刷。

**误区三：忽视插入网格的多边形密度匹配**。如果预制网格的多边形密度（如50,000面的高精度铆钉）与当前雕刻主体的密度（如5,000面的低精度躯干）相差悬殊，DynaMesh融合后接缝区域会出现明显的密度跳变，导致边缘过渡粗糙。建议预制IMM网格的边缘部分多边形密度与目标模型接近，或在融合后提升目标模型的DynaMesh分辨率至与插入网格匹配的级别。

## 知识关联

Insert Mesh笔刷建立在**雕刻笔刷**基础操作的熟练掌握之上，尤其需要理解笔刷强度（Z Intensity）、笔刷尺寸（Draw Size）的控制方式，因为这两个参数同样影响Insert Mesh笔刷插入时的网格缩放比例与嵌入深度。此外，对ZBrush SubTool系统的理解有助于制作和管理自定义IMM笔刷库，因为自定义IMM笔刷的来源正是现有的SubTool模型。

在工作流的延伸方向上，掌握Insert Mesh笔刷后，艺术家通常会进一步学习**ZModeler笔刷**（用于硬表面多边形建模）和**Nanomesh系统**（ZBrush 4R7引入，可在多边形面上密集实例化分布网格，是Insert Mesh Curve模式的进阶替代方案）。对于需要将雕刻结果导出到游戏引擎的艺术家，Insert Mesh笔刷产生的高精度融合网格通常还需要配合**ZRemesher**工具进行重拓扑，将动辄数百万面的DynaMesh结果优化为游戏可用的低多边形模型。
