# 形体阴影

## 概述

形体阴影（Form Shadow）是物体表面因自身几何结构遮挡光源而产生的暗部区域，与投射阴影（Cast Shadow）的本质区别在于：形体阴影始终附着在物体本身的曲面或平面上，而非投射到其他物体或地面。理解形体阴影的关键是找到**明暗交界线**（Terminator Line）——这条线标志着受光面与背光面的分界，是形体阴影中明度最低的区域，而非阴影的边缘轮廓。

形体阴影的系统研究可追溯到文艺复兴时期。列奥纳多·达·芬奇在《绘画论》（约1490年代）中明确区分了"本影"与"投影"，并指出曲面物体的明暗交界线是渐变的，而不是一条硬边。这一观察成为后续几百年学院派素描训练的核心依据。在概念设计领域，正确理解形体阴影让设计师能够仅凭几何基本体（球体、圆柱、盒体）快速建立三维体积感，在进入精细细节之前先确认形体读取是否清晰。

形体阴影的重要性在于：它由物体的**几何形状**决定，而非由纹理或材质决定。这意味着同一个球体在平行光下，无论表面是金属还是橡胶，其明暗交界线的位置完全相同——材质只影响阴影内外的反射强度与衔接方式，不改变交界线本身的位置。

## 核心原理

### 明暗交界线的几何定义

明暗交界线是光线方向向量与物体表面法线方向**垂直**的点的集合，即表面法线与入射光向量的夹角恰好等于90°的位置。用公式表达：

> **N · L = 0**
>
> 其中 N 为表面法线单位向量，L 为指向光源的单位向量

当 N · L > 0 时，该点处于受光面；当 N · L < 0 时，该点处于形体阴影区。明暗交界线正是 N · L = 0 的等值线。对于平行光源（如太阳光），这条线在球体上是一个**大圆**，在圆柱体上是两条**平行的母线**，在盒体上则退化为**棱边**本身。

### 三种基本体上的形体阴影特征

**球体**：明暗交界线是一个完整的椭圆形大圆，其倾斜角度直接对应光源的入射角度。光源从正侧方（90°侧光）照射时，交界线恰好过球心将球一分为二；光源从正前方（45°前侧光）照射时，交界线向背光侧偏移，受光面占据约75%的可见面积。交界线两侧的过渡宽度体现了球面曲率——球体上没有硬边，亮部到阴影是一段渐变灰阶，但交界线本身的明度最低，略低于阴影区中间部位（因为阴影区会接收环境光反弹）。

**圆柱体**：圆柱的形体阴影在侧面上表现为两条沿轴向延伸的明暗交界线，将侧面分为受光的前半圆弧和背光的后半圆弧。重要的是，圆柱体的顶面（平面圆）遵循平面受光规律，其明暗由顶面法线与光源夹角决定，与侧面的形体阴影逻辑完全独立。当光源从侧上方45°照射时，圆柱侧面受光面约占侧面周长的50%，而顶面同时处于受光状态。

**盒体（长方体）**：盒体的形体阴影逻辑最为直接。由于每个面都是平面，整张面要么全部受光，要么全部处于形体阴影中，不存在渐变过渡。明暗交界"线"退化为棱边，判断方式是计算每个面的法线方向与光源方向的点积——三个可见面（顶面、正面、侧面）中，点积为正的面受光，点积为负的面处于形体阴影，点积最接近0的面明度最低（侧锋光时最暗）。盒体因此是训练初学者辨别"哪个面朝向光源"最高效的工具。

### 形体阴影内部的反射光（Reflected Light）

形体阴影区域并非纯黑，而是接收来自周围环境的间接光，称为**反射光**或**填充光**。在标准的三球练习（Three-Sphere Exercise）中，可观察到阴影区内靠近轮廓边缘处往往比阴影区中间更亮——这是因为地面或周边浅色物体将光线反弹回来，照亮了背光侧的边缘。反射光的明度规则是：**它必须暗于物体受光面上任何区域的明度**，否则立体感将崩溃。违反这一规则是初学者在绘制形体阴影时最常见的明度失控问题。

## 实际应用

**角色头部体积分析**：人类头部可以简化为一个改良的球体加盒体组合。在概念设计的快速草图阶段，设计师先确定光源方向，然后在头部球体上画出明暗交界线位置，再根据颧骨、鼻梁等面的朝向判断局部的形体阴影归属，而不是逐笔描绘皱纹或毛孔。这种"先整体形体阴影、后局部细节"的工作流程，确保了整体光照逻辑一致。

**机械硬表面设计**：飞船或机甲的概念草图中，每一块装甲板都可视为盒体的变形。设计师通过刻意控制各面板的朝向角度，来决定哪些面在标准3/4侧光下处于形体阴影中。将某块装甲板倾斜5°~15°，就能让它从受光面跳进阴影区，产生强烈的面与面之间的明度对比，从而在不增加细节的情况下增强硬表面的结构感。

**数字绘图中的明暗交界线辅助**：在Procreate或Photoshop中绘制球体时，常用方法是先用单色铺出受光面与形体阴影的大色块，只区分"亮部"和"暗部"两个区域，再在明暗交界线处加深0.5~1个明度档，最后在阴影区靠近边缘处提亮反射光。这个三步工作流直接对应形体阴影的三段式结构：受光区 → 交界线（最暗）→ 阴影区（含反射光的中暗）。

## 常见误区

**误区一：把明暗交界线画在物体轮廓边缘处**
许多初学者习惯在球体轮廓的边缘画出最深的暗部，认为轮廓线=明暗分界。实际上，球体的轮廓线是视线切线位置，与光线方向无关。在45°侧光下，球体的明暗交界线位于球心偏后方，而轮廓边缘（视觉上的最边缘）往往因接收环境反射光而比交界线更亮。将最暗处错放在轮廓会让球体看起来像一个平面圆圈加了阴影，而非立体球。

**误区二：认为形体阴影在所有光源类型下形状相同**
平行光（如太阳光）在球体上产生的明暗交界线是标准大圆；但点光源（Point Light）距离越近，产生的明暗交界线越向背光侧收缩，受光面占比越大（可超过75%）。面光源（Area Light）则会使交界线产生模糊宽度，形成一段既不属于完整受光也不属于完整阴影的"半影过渡带"，宽度可达球体半径的20%~30%。概念设计中若不区分光源类型便套用相同的形体阴影画法，会导致光源逻辑矛盾。

**误区三：阴影区内加高光**
部分设计师为了增加细节感，在形体阴影区内添加亮色笔触或高光点，认为这能表达材质的细腻。但在物理上，镜面高光（Specular Highlight）只出现在受光面内，且位置由入射角与视角的中间向量（Half Vector）决定。在形体阴影区内添加高光会打破 N · L = 0 的几何约束，使观者无法准确读取形体的空间朝向。正确做法是将所有高光严格限制在受光面一侧，阴影区内只允许存在反射光（且明度低于受光区最暗处）。

## 知识关联

形体阴影的前置知识是**光源类型**：必须知道当前场景用的是平行光、点光源还是面光源，才能准确判断明暗交界线的位置与清晰度。不同光源类型直接决定了 N · L = 0 这条等值线的锐利程度。
