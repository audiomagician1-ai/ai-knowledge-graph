# 大气透视

## 概述

大气透视（Atmospheric Perspective），又称空气透视（Aerial Perspective）或色彩透视，是描述物体随观察距离增大而呈现色彩冷移、饱和度下降、明度压缩、边缘模糊这一系列视觉衰减现象的绘画与设计原理。其物理机制源于大气中的水蒸气微粒（直径约0.1–10μm）、尘埃和气溶胶颗粒对光线的**米氏散射（Mie Scattering）**与**瑞利散射（Rayleigh Scattering）**：远处物体反射的光线需穿越更厚的空气柱才能抵达观察者视网膜，途中蓝色短波长光（约450nm）被大气优先散射并叠加进入视线，造成远景整体向冷蓝方向偏移，而红橙等长波长信息则因散射损耗较小而在近景保留更多。

这一现象最早由达·芬奇（Leonardo da Vinci）在15世纪末系统性文字化。他在《绘画论》（*Trattato della Pittura*，成书约1490–1510年）中写道："距离越远，物体的颜色就应越向天空的蓝色靠拢，轮廓线应逐渐消融，细节应逐步消失。"达·芬奇在其《岩间圣母》（*Virgin of the Rocks*，1483–1486，现存巴黎卢浮宫）的背景山峦中直接实践了这一原理，使画面在仅约60cm的画布进深上产生了视觉上数公里的空间深度。

在概念设计工作流中，大气透视是以**纯色彩手段**建立空间层次的最高效路径。游戏美术、电影概念图和插画中，正确运用大气透视可以仅凭色调变化传递近景、中景、远景的明确层级，在粗略草图阶段即能建立可信的三维纵深感，无需依赖精确的线性透视网格。

## 核心原理

### 色彩偏移：固有色向天空色线性插值

随距离增加，物体固有色（Local Color）发生两种叠加变化：**色相向蓝紫偏移**（因瑞利散射以450nm蓝光为峰值），以及**饱和度持续下降**（散射光为中性漫反射，混入后稀释原色纯度）。一棵近处饱和度约85%的深绿色树木（HSB约130°, 85%, 35%），在3公里外会呈现蓝绿偏灰的中间色（约195°, 30%, 55%），在10公里以上则几乎与天空色融为一体（约210°, 10%, 75%）。

数字绘画中最常用的插值公式为：

$$C_{远景} = C_{固有色} \times (1 - t) + C_{天空色} \times t$$

其中 $t \in [0, 1]$ 为归一化距离系数，$t=0$ 表示近景，$t=1$ 表示与天际线齐平的极远景。实践中 $t$ 值并非线性分布，而是随距离呈幂函数增长（指数约0.5–0.7），即近景变化慢、远景变化快。

在晴天正午条件下，天空采样色通常约为 RGB(160, 185, 215)（偏冷的蓝灰）；黄昏时则偏移至 RGB(220, 185, 160)（暖橙灰），导致大气透视方向反转为**暖色方向**——这是初学者常忽略的时段差异。

### 明度压缩：远景明度向天际亮度收敛

大气透视对明度（Value）的影响遵循**中灰收敛**规律：无论近景物体本身是极深色还是极浅色，随距离增大，其明度均向画面中偏亮的天际灰（通常明度值约65%–80%）靠拢。具体表现为：远处的**暗部被提亮**，远处的**亮部被略微压暗**，整体明度对比范围被压缩。

近景物体的明度对比可跨越80–90个灰阶单位（0–255体系），而远景物体的明度对比往往被压缩至15–25个单位以内。数字绘画中，这一效果通过叠加半透明的浅冷灰雾层实现——该图层填充天空采样色，以10%–60%不等的不透明度依距离梯度叠加于各中远景层之上。

### 轮廓模糊与细节消融梯度

大气透视的第三个维度是**边缘清晰度（Edge Quality）**的距离衰减。近景物体具有硬边（Hard Edge）和可辨识纹理（如树皮纹路、岩石裂缝），中景开始转为软边（Soft Edge），远景则仅剩色块轮廓，纹理信息几乎归零。此梯度变化在概念设计实务中提供了一种**时间分配的正当依据**：远景只需处理色调关系，无需细化表面信息，人眼对这种处理的接受度极高，因为真实视觉系统的空间频率分辨率本就随距离衰减。

## 关键公式与数字参数

以下 Python 伪代码演示了如何在程序化生成场景中批量计算各距离层的大气透视混合色：

```python
import numpy as np

def atmospheric_blend(local_color, sky_color, distance, max_distance, gamma=0.6):
    """
    local_color: np.array([R, G, B])，近景固有色，范围 0–255
    sky_color:   np.array([R, G, B])，天空采样色
    distance:    当前物体距观察者距离（单位：米）
    max_distance: 场景最大可见距离（单位：米）
    gamma:       幂函数指数，控制衰减曲线形状（0.5–0.7 为典型晴天值）
    """
    t = (distance / max_distance) ** gamma
    t = np.clip(t, 0.0, 1.0)
    result = local_color * (1 - t) + sky_color * t
    return result.astype(int)

# 示例：近景深绿树（RGB 40, 80, 35），天空色（RGB 160, 185, 215），5km处（最大可见20km）
local  = np.array([40,  80,  35])
sky    = np.array([160, 185, 215])
blended = atmospheric_blend(local, sky, distance=5000, max_distance=20000)
# 输出约 [94, 126, 107]，蓝绿偏灰的中间色
```

此处 `gamma=0.6` 对应典型晴天大气散射曲线；薄雾天气可将 `gamma` 降低至 0.3–0.4，使衰减更激进；极清澈高原空气可提升至 0.8–0.9。

## 实际应用

**游戏场景概念图**：《地平线：零之曙光》（*Horizon Zero Dawn*，2017）的环境概念图采用明显的三段式大气透视：近景植被为饱和暖绿（HSB约125°, 80%, 40%），中景山丘转为中性蓝绿（约190°, 35%, 55%），远山退为淡蓝灰（约205°, 12%, 72%）并与天空几乎同值。这种处理使玩家在二维屏幕上能直觉感知到场景纵深超过数公里，而整个大气透视关系在草图阶段只需三种颜色即可确立。

**电影概念设计**：为山地或海洋场景创作概念图时，工业流程中通常新建专用"Atmosphere"图层，填充天空采样色（晴天约 RGB 180, 200, 220），以20%–60%不等的不透明度梯度叠加于各中远景层之上。Feng Zhu（朱峰）在其 FZD School of Design 公开教程（2012年系列）中将这一图层称为"大气预算层"，强调其不透明度应与场景设定的空气湿度/污染度直接挂钩：沙漠晴天约15%，热带海岸约40%，工业城市雾霾场景可达65%以上。

**水彩手绘训练**：传统水彩画家训练大气透视时常用"三景饱和度测试"：前景色彩饱和度不低于70%，中景不超过50%，远景不超过25%。这一量化标准源自约翰·辛格·萨金特（John Singer Sargent）的写生记录，他在19世纪90年代的威尼斯水彩系列中系统性地使用了这三段阈值，使景深在快速写生中依然可辨（参见《Sargent Watercolors》，Elaine Kilmurray，2013，MFA Publications）。

**例如**：绘制一幅城市鸟瞰概念图时，前景建筑群以固有色直接涂布（砖红 RGB 180, 80, 50），中景建筑群同色系叠加20%天空灰（结果约 RGB 168, 113, 103），远景建筑群叠加50%天空灰（约 RGB 170, 140, 133），最远处城市轮廓叠加75%天空灰（约 RGB 175, 156, 156），天际线建筑几乎与天空融合。整个过程不需要绘制任何透视线，仅靠色调分层即可建立真实可信的城市纵深。

## 常见误区

**误区一：无论何种光照条件，远景一律偏蓝**。这是最普遍的错误。大气透视的色偏方向取决于天空色，而天空色随时段剧烈变化。黄昏时天空主导色为暖橙，此时大气透视方向为**暖色偏移**，远山呈粉橙灰而非蓝灰；正午高原晴天大气透视衰减极弱，远山仍保留较多固有色饱和度，色偏量远小于平原阴天。

**误区二：大气透视仅影响色彩，不影响明度**。实际上对许多暗色近景物体而言，距离增加会使其明度**显著提亮**（暗部收敛至天际明度）。例如一片明度仅15%的深色岩壁，在5公里外其明度可能已被提升至45%–55%，这一变化远比色相偏移更显眼，但绘画者常常漏掉它。

**误区三：大气透视效果应均匀分布于整幅画面**。真实大气透视是从观察者位置向外辐射的，同等距离的物体受到相同程度的衰减，与画面的上下左右位置无关。初学者有时误将画面底部近景、顶部远景作为唯一的纵深方向，而忽略了侧面延伸物体（如从左近处延伸到右远处的道路）同样应呈现大气透视梯度。

**误区四：大气透视与景深模糊（摄影虚化）是同一回事**。景深虚化（Depth of Field Blur）是镜头光学现象，模糊方式为各向同性的高斯模糊，且受焦点位置控制；大气透视是大气光学现象，主要影响色彩和对比度，边缘模糊仅为次生效果，且在有雾天气中极远处的对比度损失远比边缘模糊更为关键。两者在概念设计中应分别独立处理。

## 关键公式小结

大气透视三要素的量化衰减关系可归纳如下：

- **色相偏移量**：$\Delta H \approx (H_{sky} - H_{local}) \times t^{0.6}$，单位为色相角度（°）
- **饱和度衰减**：$S_{结果} = S_{固有} \times (1 - t)$，$t$ 越大饱和度趋近于 $S_{sky}$
- **明度收敛**：$V_{结果} = V_{固有} + (V_{sky} - V_{固有}) \times t$，线性插值
- **对比度压缩比**：远景局部对比度 ≈ 近景对比度 $\times (1-t)^{1.2}$

其中 $t = \left(\dfrac{d}{d_{max}}\right)^{\gamma}$，$d$ 为观察距离，$d_{max}$ 为场景最大能见度，$\gamma$ 为大气散射指数（晴天约0.6，薄雾约0.35）。
