# 色彩混合

## 概述

色彩混合是指两种或两种以上颜色相互叠加或融合，产生新颜色的物理与视觉过程。根据混合介质的不同，色彩混合分为**加法混色**（Additive Color Mixing）和**减法混色**（Subtractive Color Mixing）两大类，二者的原色体系、混合结果甚至光学本质都截然相反。

加法混色的概念由物理学家托马斯·杨（Thomas Young）于1802年提出雏形，后经赫尔曼·冯·亥姆霍兹（Hermann von Helmholtz）完善为三色视觉理论。减法混色则与印刷工业的发展密切相关，19世纪末彩色印刷的商业化使CMY体系逐步成为标准。理解这两套体系的区别，是数字绘画、平面设计乃至摄影后期工作中做出正确色彩决策的前提。

在数字绘画领域，画布的显示介质（屏幕发光）决定了底层运算是加法混色逻辑，但绘画软件中的"混合模式"和颜料笔刷模拟的是减法混色效果。这一矛盾造成了大量初学者在数字媒介上调色失准的问题，因此掌握两种混色机制的本质差异具有直接的实践意义。

## 核心原理

### 加法混色：光的叠加

加法混色发生在**光线直接叠加**的场景中，典型设备是RGB显示器、舞台照明和彩色LED。其三原色为红（Red, 约700nm）、绿（Green, 约546nm）、蓝（Blue, 约435nm）。

混合规律如下：
- 红 + 绿 = 黄
- 红 + 蓝 = 品红（Magenta）
- 绿 + 蓝 = 青（Cyan）
- 红 + 绿 + 蓝 = **白色**（三色等量叠加，亮度达到最大值255）
- 全部缺失 = **黑色**（亮度为0）

在RGB数值体系中，加法混色的计算公式为逐通道线性叠加：`R_out = R1 + R2`（结果钳制在0–255范围内）。等量的红光（255, 0, 0）与绿光（0, 255, 0）混合，精确产生黄色（255, 255, 0），这与颜料混合的直觉完全不同。

### 减法混色：颜料对光的吸收

减法混色发生在**颜料、油墨、滤光片**等吸收特定波长光线的场景中。颜料的颜色来自它所**反射**的波长，而混合颜料等于叠加吸收，光线被逐步"减去"，因此越混越暗。

印刷工业标准三原色为青（Cyan）、品红（Magenta）、黄（Yellow），即CMY体系：
- 青 + 品红 = 蓝
- 青 + 黄 = 绿
- 品红 + 黄 = 红
- 青 + 品红 + 黄 = **黑色**（理论上，实际印刷因油墨纯度不足呈暗褐色，故引入单独的K黑版，形成CMYK四色印刷标准）

传统美术教育中常以红、黄、蓝作为颜料三原色（RYB体系），这是历史上的近似方案，其色域覆盖不如CMY全面，例如RYB体系中混合出的绿色往往偏灰暗，而CMY的青加黄可以产生更鲜艳的绿色。

### 数字绘画中的混色逻辑

Photoshop、Procreate等软件的图层混合模式直接映射加法或减法混色的数学逻辑：

| 混合模式 | 对应机制 | 公式 |
|---------|---------|------|
| 滤色（Screen） | 加法混色 | `1 - (1-A)(1-B)` |
| 正片叠底（Multiply） | 减法混色 | `A × B`（归一化到0–1） |
| 叠加（Overlay） | 混合型 | 亮部Screen，暗部Multiply |

在颜料笔刷（Wet Mix模式）中，软件通过采样周边颜色并运行类减法算法模拟真实颜料混合。若直接在图层上叠加半透明色块，则底层是加法透明度合成（Alpha Compositing），二者行为不同，会产生截然不同的视觉结果。

## 实际应用

**数字插画调色**：在Procreate中，将一个橙色笔触直接叠在蓝色底色上，因软件默认正常混合模式走透明度合成，结果是两色的平均过渡而非泥巴色。但若开启"颜料混合"笔刷，软件模拟减法混色，橙与蓝互为补色，混合结果会趋向中性灰褐色，这才是真实颜料混合的视觉感受。

**印刷与屏幕色彩校准**：一张在RGB屏幕上设计的海报（色值如R:255 G:100 B:0的橙色）转换为CMYK印刷时，需将加法色域映射到减法色域。由于CMYK色域比sRGB窄约30%–40%，高饱和度的橙、翠绿等颜色在印刷品上会显著变暗变灰，这是由两种混色体系色域差异直接导致的。

**舞台灯光设计**：三台分别装有红、绿、蓝滤镜的追光灯同时照射同一区域，在舞台地板上产生白色光斑，而遮挡其中两台时得到单一原色——这是加法混色在物理空间中的直观演示，与颜料混合的逻辑完全相反。

## 常见误区

**误区一：认为数字屏幕上的颜色混合等同于颜料混合**
许多初学者将RGB数值直接"相加"来预测颜料混合效果。实际上，RGB (255, 0, 0) 与 RGB (0, 0, 255) 的加法叠加产生品红色 (255, 0, 255)，但同等红色与蓝色颜料混合产生的是暗紫色，因为颜料在减去黄色波段后留下的反射光集中在紫色频段，与加法结果在色相和明度上均不同。

**误区二：三原色只有一套**
"红黄蓝是三原色"是美术基础教育中流传的不精确说法，专指传统颜料体系的RYB原色。加法混色的原色是RGB，减法混色的标准原色是CMY，三套体系服务于不同介质，不可混用。在数字绘画软件中设置调色板时，若以RYB逻辑操作RGB滑块，调出的颜色会与预期偏差明显。

**误区三：正片叠底（Multiply）等于现实颜料混合**
正片叠底的公式 `A × B` 在数学上模拟的是两片彩色玻璃叠放的减法效果（透射光减法），而非颜料表面反射的减法混色。二者结果接近但并不相同：颜料混合涉及散射和漫反射，在混合比例、边缘过渡上比正片叠底的硬性乘法更复杂，这也是绘画软件专门开发颜料混合算法（如Procreate的流量混合引擎）的原因。

## 知识关联

**前置知识：色轮与色彩模型**——色轮本质上是减法混色（RYB或CMY）的视觉化工具，色轮上的互补色关系（如橙与蓝180°对位）直接预测了减法混色趋向中性灰的现象；HSB、RGB、CMYK等色彩模型则分别对应加法或减法混色的数学描述框架。理解色轮所属的体系（RYB颜料色轮 vs. RGB光色轮），是正确运用混色规律的前提。

在色彩理论的学习路径上，色彩混合为后续的**色彩协调**（如互补配色、分裂互补配色为何有效）和**色彩校正**（RGB-CMYK转换、色域映射）提供了机制性的解释——知道补色在减法混色下相消，才能理解为何互补色搭配视觉上最为强烈，以及印刷时为何需要刻意保留颜色的饱和度余量。
