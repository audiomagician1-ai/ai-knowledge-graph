# 自编码器（Autoencoder）

## 核心概念

自编码器是一种无监督神经网络，它学习将输入数据压缩为低维表示（编码），然后从这个压缩表示重建原始输入（解码）。训练目标是最小化输入和重建输出之间的差异。

## 基本结构

```
输入 x → [编码器 Encoder] → 潜在表示 z → [解码器 Decoder] → 重建 x̂
                               (bottleneck)
```

- **编码器**: 将高维输入映射到低维潜在空间
- **潜在空间**: 数据的压缩表示（通常维度远小于输入）
- **解码器**: 从低维表示重建高维输出
- **损失函数**: L = ||x - x̂||² （重建误差）

## 主要类型

### 1. 标准自编码器（Undercomplete）
瓶颈层维度小于输入维度，强制学习数据的压缩表示。

### 2. 变分自编码器（VAE）
在潜在空间引入概率分布：
- 编码器输出μ和σ（均值和方差）
- 潜在表示从N(μ, σ²)中采样
- 损失 = 重建误差 + KL散度（正则化）
- 可以从潜在空间采样生成新数据

### 3. 去噪自编码器（Denoising）
输入加入噪声，模型学习恢复干净数据：
- 提高对噪声的鲁棒性
- 学习更有意义的特征

### 4. 稀疏自编码器（Sparse）
在损失函数中加入稀疏性约束：
- 大部分潜在单元为零
- 学习类似于稀疏编码的特征
- 最近在LLM可解释性中受到关注

## 应用场景

| 应用 | 说明 |
|------|------|
| 数据降维 | 类似PCA但非线性 |
| 异常检测 | 异常数据重建误差大 |
| 图像去噪 | 去噪自编码器 |
| 数据生成 | VAE从潜在空间采样 |
| 特征学习 | 作为预训练的特征提取器 |
| LLM可解释性 | 稀疏自编码器分析神经元激活 |

## VAE与GAN的区别

| 特征 | VAE | GAN |
|------|-----|-----|
| 训练方式 | 最大化似然下界 | 对抗训练 |
| 训练稳定性 | 稳定 | 不稳定 |
| 生成质量 | 可能模糊 | 通常更清晰 |
| 潜在空间 | 结构化，可插值 | 无结构保证 |
| 数学基础 | 变分推断 | 博弈论 |

## 与神经网络和无监督学习的关系

自编码器是无监督学习的经典方法，使用神经网络实现。它不需要标签数据，通过重建任务自动学习数据的内在结构和特征。
