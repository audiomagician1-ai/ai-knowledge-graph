# 无监督学习

## 概述

无监督学习（Unsupervised Learning）是指在训练数据**不包含人工标注标签**的条件下，让算法自动发现数据内在结构、分布规律或潜在表示的一类机器学习方法。与监督学习依赖$(x_i, y_i)$配对样本不同，无监督学习仅接收输入向量集合$\{x_1, x_2, ..., x_n\}$，目标是从中提取有意义的信息，例如聚类结构、数据密度分布或低维表示。

无监督学习的理论根源可追溯至1950年代的信息论与统计学习理论。Geoffrey Hinton等人在1986年提出的受限玻尔兹曼机（RBM）是早期深度无监督学习的重要里程碑。2006年，Hinton发表的《A Fast Learning Algorithm for Deep Belief Nets》标志着深度无监督预训练方法的成熟，该论文证明逐层无监督预训练可有效解决深层网络训练困难问题。

无监督学习的重要性体现在数据现实——现实世界中有标注的数据仅占全部数据的极小比例（估计不足1%），而互联网每天产生的海量文本、图像、传感器数据绝大多数没有标签。掌握无监督学习意味着能够利用这些廉价的未标注数据提取特征、压缩表示，为下游有监督任务提供更好的初始化或直接产出可用结果。

## 核心原理

### 聚类：发现数据内在分组

聚类是无监督学习最直接的任务形式，目标是将相似样本归入同一组。**K-Means算法**是最经典的聚类方法，其目标函数为：

$$J = \sum_{k=1}^{K} \sum_{x_i \in C_k} \|x_i - \mu_k\|^2$$

其中$K$为簇数，$\mu_k$为第$k$个簇的质心，$C_k$为属于该簇的样本集合。K-Means通过交替执行"分配步骤"和"更新步骤"迭代优化$J$，但其结果对初始质心敏感，且只能发现凸形簇。**DBSCAN**（Density-Based Spatial Clustering of Applications with Noise）则通过两个参数——邻域半径$\epsilon$和最小样本数$MinPts$——定义核心点、边界点与噪声点，能识别任意形状的簇并自动处理噪声，无需预设$K$值。

### 降维：压缩与提取潜在结构

降维方法寻求将高维数据映射至低维空间，同时最大限度保留信息。**主成分分析（PCA）**通过计算数据协方差矩阵$\Sigma = \frac{1}{n}X^TX$的特征向量，找到数据方差最大的方向，是线性降维的基准方法。PCA的第一主成分方向$w_1 = \arg\max_{\|w\|=1} w^T\Sigma w$，即$\Sigma$的最大特征值对应的特征向量。

非线性降维方法中，**t-SNE**（t-distributed Stochastic Neighbor Embedding，2008年由van der Maaten和Hinton提出）使用Student t分布衡量低维空间中点对的相似性，极擅长可视化高维数据的局部聚集结构。其困惑度（perplexity）参数通常设置在5到50之间，控制对邻域规模的假设。**UMAP**（2018年提出）在保留局部与全局结构方面优于t-SNE，且计算速度更快，已成为单细胞RNA测序分析的标准工具。

### 生成模型：学习数据概率分布

生成式无监督学习方法直接对数据的概率分布$p(x)$进行建模。**变分自编码器（VAE）**将编码器建模为后验分布$q_\phi(z|x)$，解码器建模为似然函数$p_\theta(x|z)$，优化证据下界（ELBO）：

$$\mathcal{L} = \mathbb{E}_{q_\phi(z|x)}[\log p_\theta(x|z)] - D_{KL}(q_\phi(z|x) \| p(z))$$

**生成对抗网络（GAN，Goodfellow等人2014年提出）**则通过生成器$G$与判别器$D$的零和博弈学习数据分布，训练目标为：

$$\min_G \max_D \mathbb{E}_{x \sim p_{data}}[\log D(x)] + \mathbb{E}_{z \sim p_z}[\log(1 - D(G(z)))]$$

GAN不显式建模$p(x)$，而是通过对抗过程隐式学习分布，生成样本质量往往高于VAE，但训练稳定性是其长期痛点。

## 实际应用

**客户细分与市场分析**：电商平台利用K-Means或层次聚类对用户购买行为向量进行聚类，典型应用如将数千万用户压缩为8-20个具有差异化消费特征的群体，用于精准营销策略制定，无需任何预先定义的用户类别标签。

**异常检测**：工业设备传感器数据通常没有标注的故障样本。无监督方法如**Isolation Forest**和基于自编码器的重构误差可识别离群点——若某时间点的传感器读数重构误差超过训练集均值3个标准差，则标记为潜在故障，无需事先定义"正常"与"异常"。

**大语言模型的预训练**：GPT系列模型的核心训练阶段使用的是**下一词预测**任务，本质上是无监督学习——训练数据仅为海量文本，没有任何人工标注，模型通过最大化$\sum_i \log P(x_i | x_{i-k}, ..., x_{i-1})$自动学习语言的统计规律和语义结构。GPT-3使用了3000亿个token的无标注文本进行训练。

**医学图像特征提取**：在医疗领域标注数据极度稀缺且昂贵，无监督预训练（如对比学习SimCLR）可在无标注的大规模医学图像上学习图像特征表示，之后仅需少量标注数据微调即可达到接近全监督的病灶检测精度。

## 常见误区

**误区一：认为无监督学习"没有目标函数"因此是完全自由的**。事实上，无监督学习的每种方法都有明确的数学目标：K-Means最小化簇内平方和，PCA最大化投影方差，VAE最大化ELBO。"无监督"指的是缺少外部标签监督，而非缺少优化目标。混淆这一点会导致对算法调参和失败分析束手无策。

**误区二：聚类结果就是数据的"真实分组"**。聚类结果强烈依赖于所选算法和距离度量。同一批用户数据，使用欧氏距离K-Means与余弦相似度谱聚类可能产生完全不同的分组。没有外部标签，无法客观判断哪种聚类"更正确"，只能通过轮廓系数（Silhouette Score）、Davies-Bouldin指数等内部评估指标或业务含义来评价分组质量。

**误区三：无监督学习不需要数据预处理**。K-Means对特征尺度极其敏感——若一个特征的数值范围是0-1000，另一个是0-1，前者将主导距离计算。PCA同样要求在计算协方差矩阵前对各特征进行标准化（零均值、单位方差），否则方差大的特征会虚假地占据主要主成分。忽略这一步会导致严重错误的结果。

## 知识关联

**来自监督学习的知识迁移**：监督学习中的梯度下降、正则化、批归一化等优化技术在训练VAE、GAN等深度无监督模型时同样适用。同时，理解监督学习中特征工程的局限性——手工特征需要领域专家且无法扩展到海量数据——正是无监督表示学习被需要的根本动机。

**通向自编码器**：普通自编码器是无监督学习降维思想的直接深度化：编码器将输入压缩为瓶颈表示，解码器尝试重构原始输入，以重构误差$\|x - \hat{x}\|^2$为损失函数。掌握PCA的降维原理和无监督目标函数设计思路，是理解自编码器的瓶颈结构为何能自动提取有意义特征的前提。

**通向自监督学习**：自监督学习可以看作无监督学习的进化形态——它通过从数据自身构造代理任务（如掩码预测、旋转预测、对比正负样本对）生成伪标签，将无监督问题转化为"有标签"的形式，从而能使用更强的监督学习优化框架。BERT的Masked Language Modeling和SimCLR的对比损失均属此类，它们的出发点正是无监督学习的核心问题：如何在无外部标注的情况下从数据中提取通用表示。
