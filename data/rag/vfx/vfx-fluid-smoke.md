# 烟雾模拟

## 概述

烟雾模拟是流体特效中专门处理低密度气态物质运动的技术，其核心在于维护一个随时间演变的**体积密度场（Volume Density Field）**，用标量值表示三维空间中每个体素（Voxel）内烟雾粒子的聚集程度。与火焰模拟不同，烟雾本身不发光，其视觉效果完全依赖外部光源在密度场中的散射与吸收，因此渲染管线与火焰有本质差别。

烟雾模拟的数学基础可追溯至1999年前后Jos Stam在SIGGRAPH发表的《Stable Fluids》论文，该论文将Navier-Stokes方程的求解改为无条件稳定的隐式方法，使实时或接近实时的烟雾模拟成为可能。此前的烟雾效果主要依靠粒子系统与贴图欺骗，难以表现烟柱翻滚、涡旋卷曲等体积感细节。

烟雾模拟的重要性在于：电影视觉特效中大量场景——爆炸残烟、工厂烟囱、魔法云雾——均需要精确的体积散射效果。即便是游戏中的低精度烟雾，也需要理解密度场衰减规律，才能在有限预算内设计出可信的近似方案。

## 核心原理

### 密度场的传输方程

烟雾模拟的核心是对密度标量场 `ρ(x, t)` 的求解，遵循对流-扩散方程：

```
∂ρ/∂t + (u · ∇)ρ = κ∇²ρ - d·ρ
```

其中：
- `u` 为速度场（由Navier-Stokes方程独立求解）
- `κ` 为扩散系数，控制烟雾向周围空气"渗透"的速率
- `d` 为**耗散系数（Dissipation Rate）**，每帧将密度乘以衰减因子（典型值为0.97~0.995），使烟雾在无外力驱动时自然消散

耗散项是烟雾区别于不可压缩纯液体模拟的关键设计：液体不耗散，而烟雾在扩散至足够低密度后应从视觉上消失，耗散项以最低计算代价实现这一效果。

### 浮力驱动与温度耦合

真实烟雾的上升行为来自**热浮力（Thermal Buoyancy）**。在模拟中通常引入温度标量场 `T(x, t)`，将浮力作为速度场的外力项叠加：

```
F_buoyancy = α·ρ·ĝ - β·(T - T_ambient)·ĝ
```

其中 `α` 为烟雾密度对浮力的贡献系数（密度高则下沉），`β` 为温度浮力系数（温度高则上升），`T_ambient` 约为293K（室温），`ĝ` 为向上单位向量。两项的拮抗关系决定烟雾是缓慢飘散还是猛烈上冲。在与火焰模拟衔接时，火焰的温度场直接作为烟雾模拟的热源输入，浓烟随火舌上方气柱快速向上卷动。

### 涡旋增强（Vorticity Confinement）

由于网格分辨率限制，数值耗散会"吃掉"速度场中的细小涡旋，导致烟雾看起来软塌、缺乏翻滚细节。**涡旋增强（Vorticity Confinement）**技术由Fedkiw等人于2001年提出，通过计算涡量场 `ω = ∇×u` 并反向添加一个恢复力：

```
F_vc = ε · h · (N × ω)
```

其中 `h` 为网格间距，`N` 为涡量梯度归一化方向，`ε` 为可调强度参数（通常设为0.1~2.0）。该力专门放大涡旋边缘的旋转，使烟柱边界呈现卷曲、翻卷的高频细节，而不会破坏整体流场的质量守恒。

### 体积渲染与单次/多次散射

烟雾的渲染采用**体积光线步进（Ray Marching）**方法。光线从摄像机出发，每步进 `Δt`（通常为体素尺寸的0.5~1倍）采样密度，累积消光和散射：

- **消光系数** `σ_t = σ_a + σ_s`（吸收+散射之和），典型烟雾的散射反照率（σ_s/σ_t）在0.7~0.9之间
- **单次散射**：仅计算直接光源经一次散射到达摄像机的贡献，计算量为O(N·L)（N为步进次数，L为光源数）
- **多次散射近似**：真实浓烟内部光线被多次反弹，常用**Deep Shadow Map**或**环境光遮蔽（AO）**预计算来近似模拟多次散射的"内透亮"效果，将性能开销从指数级降至线性级

## 实际应用

**电影级浓烟：** 在Houdini的PyroFX系统中，典型的爆炸浓烟模拟使用128³至512³体素网格，耗散系数设为0.98，涡旋增强ε为0.5，结合高分辨率Noise扰动（频率约为基础网格的4~8倍）来补偿网格分辨率不足。渲染时使用Mantra的体积着色器，开启路径追踪多次散射，一帧渲染时间可达数分钟。

**游戏实时烟雾：** 受帧率限制，实时烟雾通常使用64³以下的低分辨率网格，关闭多次散射，仅用预计算的"烟雾光照纹理"近似散射结果。耗散系数可提高至0.9以加快消散，减少持续更新的帧数开销。Unreal Engine的Niagara系统支持低分辨率Sparse Voxel烟雾，利用稀疏数据结构只计算有密度的体素，节省60%以上内存。

**魔法/奇幻风格烟雾：** 通过将耗散系数设为1.0（零耗散）并叠加自定义外力场（如螺旋形向心力），可使烟雾绕轴盘旋聚拢而不消散，产生魔法漩涡效果。此时浮力系数β归零，烟雾不再受热力驱动，完全由设计师定义的力场控制形态。

## 常见误区

**误区一：耗散系数越大烟雾"消失越快"。**
耗散系数 `d` 是每帧密度保留比例，值为0.99表示每帧保留99%密度（消散慢），值为0.90表示每帧保留90%（消散快）。许多初学者混淆"耗散系数"与"耗散速率"，将0.99当作"高速耗散"反向设置，导致烟雾瞬间消失。正确理解是：耗散系数越接近1.0，烟雾越持久。

**误区二：提高网格分辨率即可替代涡旋增强。**
虽然更高分辨率确实能保留更多涡旋细节，但数值耗散与网格间距的关系是线性的：分辨率翻倍只能减少约50%的涡旋耗散，而计算量增加8倍（三维体积）。涡旋增强以极低的额外计算（仅多一次散度运算）恢复等效于2~3倍分辨率提升的细节量，两者定位完全不同，生产流程中通常同时使用。

**误区三：烟雾渲染的散射反照率越高越真实。**
散射反照率过高（接近1.0）会使烟雾看起来像白色固体云团，失去半透明感。真实黑烟（如柴油燃烧）的散射反照率仅约0.3~0.5，因为碳颗粒强烈吸收可见光。将炫酷黑烟的散射反照率错误设为0.9，会得到灰白色绒毛状外观，完全背离目标效果。

## 知识关联

**前置概念——火焰模拟：** 烟雾模拟的温度场初始条件通常直接来自火焰模拟输出。火焰燃烧区域产生的高温气体（1000K以上）作为烟雾的热源，驱动上方冷却烟柱的浮力运动。两者共用同一速度场求解器，但密度场一个代表燃料/温度，一个代表碳烟颗粒，物理意义不同。

**后续概念——实时流体策略：** 理解了体积密度场的传输方程和耗散机制后，实时流体策略的核心问题变为"如何在保持可信密度场演变的前提下最大化降采样"。稀疏体素、时间复用（Temporal Reprojection）、异步模拟等实时策略均是针对烟雾密度场更新代价高昂这一瓶颈而设计的具体解决方案。
