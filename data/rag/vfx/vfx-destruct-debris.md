# 碎片与残骸

## 概述

碎片与残骸特效是破碎销毁系统的附属视觉层，专门处理主体物件破碎后飞散的次级粒子现象，涵盖石块碎片、木质刨花、混凝土粉尘、玻璃渣粒以及爆炸烟雾云团。与负责主体网格分裂的 Voronoi 破碎不同，碎片与残骸系统的职责是填充破碎事件在时间轴和三维空间上留下的"视觉空白"——让一次爆炸或撞击从接触瞬间到余波消散全程具有物理可信度。

该技术分支在1990年代后期随粒子系统在电影视效中的大规模应用而成形。《黑客帝国》（1999）的子弹时间场景首次将高密度混凝土粉尘与慢动作摄影机捕捉结合，使业界确立了"主碎片 + 微粒尘云"双层架构的事实标准。在游戏引擎领域，Unreal Engine 3（2006年发布）内置的 Cascade 粒子系统正式将碎片粒子与物理网格的联动写入工具链，之后被 Niagara 系统取代，后者支持 GPU 粒子数量达数百万级别，单帧最大发射量可超过 500 万个粒子（Epic Games, 2019, *Niagara Technical Overview*）。

碎片与残骸之所以独立于主破碎系统存在，原因在于其对运行时性能与视觉密度的权衡逻辑完全不同：主碎片块需要精确的凸包碰撞体（Convex Hull），而粉尘和烟雾粒子通常使用廉价的 Billboard 公告板或 Signed Distance Field 体积渲染，二者的 GPU 指令流互不竞争，拆分管理才能各自达到最优帧率效果。

## 核心原理

### 粒子发射与破碎事件绑定

碎片粒子的生命周期始于破碎事件触发的那一帧。系统从伤害模型传入的冲击能量值（Impact Energy，单位通常为引擎内的归一化 0~1 浮点数，或物理仿真中的焦耳值）决定粒子的初始喷射速度、数量和扩散锥角。

粒子初速度计算公式为：

$$V_0 = K_m \times \sqrt{E_{\text{impact}}} \times \left(1 + \alpha \cdot \sin\theta\right)$$

其中：
- $V_0$：粒子初速度（m/s，引擎坐标系）
- $K_m$：材质系数，混凝土约为 **0.40**，玻璃约为 **0.82**，木材约为 **0.25**，金属约为 **0.60**
- $E_{\text{impact}}$：碰撞能量归一化值（0~1）
- $\alpha$：法线偏移系数，默认取 **0.15**
- $\theta$：入射角（碰撞法线与表面法线的夹角）

这意味着玻璃碎片在相同撞击力下（$E_{\text{impact}} = 1.0$，$\theta = 0$）飞散速度约为 0.82 m/s 单位量，是混凝土粉尘（0.40）的 **2.05 倍**，符合材质脆性特征——玻璃杨氏模量约 70 GPa，混凝土仅约 30 GPa，脆而不韧，单次冲击释放动能更集中（参见《材料力学》，刘鸿文，高等教育出版社，2017）。

粒子发射点并非单一质心，而是沿破碎面的 Voronoi 边缘离散分布：每条断裂缝都是一条独立的粒子发射带，系统按缝隙长度每 2~5 cm 采样一个发射锚点，使粉尘看起来从裂缝中"挤出"而非从物体中心爆发，避免视觉上的集中喷射感。

### 多层粒子分类与寿命管理

碎片残骸在实现上严格分为三个独立粒子层，每层参数互不共享，由图层优先级（Layer Priority）决定渲染排序：

**粗碎片层（Macro Debris）**：单片尺寸 5 cm ~ 30 cm，使用真实 3D 网格（低多边形碎石预制体，面数通常限制在 50~150 个三角面），携带刚体物理与简化凸包碰撞，寿命 2~8 秒，数量通常限制在 **20~60 个**以节约碰撞计算资源。落地摩擦系数按材质设定：混凝土碎块 μ = 0.65，木板 μ = 0.40。

**中粒子层（Meso Particles）**：单片尺寸 0.5 cm ~ 5 cm，使用旋转的 Billboard 平面模拟小石子和木屑，不携带精确物理碰撞，改用简化抛物线运动叠加 Perlin 噪声扰动（频率 2~8 Hz），寿命 1~4 秒，数量 **200~800 个**。角速度范围设为 90°/s ~ 720°/s，使碎片在空中自然翻滚。

**微尘/烟雾层（Micro Dust & Smoke）**：视觉尺寸 10 cm ~ 1 m，使用 Flipbook 序列帧纹理（标准规格为 8×8 帧，总 64 帧，纹理分辨率 2048×2048）或实时体积烟雾，受环境风场影响显著（风速每增加 1 m/s，粒子横向漂移约增加 0.3 m/s），寿命 **3~12 秒**，GPU 粒子数量可达 **5000~50000 个**。烟雾颜色根据材质动态选取：混凝土为灰白色（RGB 约 210, 200, 190），木材燃烧为黄褐色（RGB 约 160, 120, 60），爆炸黑烟为深灰色（RGB 约 40, 35, 30）。

### 物理衰减与地面沉积

粒子落地后不会简单消失，而是进入"沉积阶段"，分两步执行：

**第一步：静态贴花印记**。粗碎片停止滚动后（速度 < 0.05 m/s 持续 0.5 秒），系统在落点生成一张静态 Decal 贴花，分辨率 512×512，UV 旋转随机化，记录碎石存在的持久视觉信息，保留时长通常为 30~120 秒后淡出。

**第二步：尘云贴地扩散**。微尘粒子在距地面 0.3 m 以内时触发"展开"行为，横向扩散速度提升 **3~5 倍**，同时 Alpha 值从 0.8 线性衰减至 0，模拟尘云贴地扩散的黏性流体行为。这一阶段的流场驱动通常使用预烘焙向量场（Vector Field，体积分辨率典型为 16×16×8），而非实时 Navier-Stokes 流体模拟，可节约约 **60%** 的 GPU 计算开销。

## 关键公式与算法

### 粒子数量按能量分级公式

实际项目中，三个粒子层的数量通常按冲击能量分级动态决定，以下为一段 Unreal Engine Niagara 系统中使用的伪代码逻辑：

```cpp
// 根据冲击能量决定三层粒子生成数量
// ImpactEnergy: 归一化冲击能量 [0.0, 1.0]
// MaterialType: 0=混凝土, 1=玻璃, 2=木材, 3=金属

void SpawnDebrisLayers(float ImpactEnergy, int MaterialType)
{
    // 粗碎片层：上限60个，按能量线性缩放
    int MacroCount = (int)(ImpactEnergy * 60.0f);
    MacroCount = clamp(MacroCount, 3, 60);

    // 中粒子层：玻璃额外乘以1.8倍系数
    float MesoMult = (MaterialType == 1) ? 1.8f : 1.0f;
    int MesoCount = (int)(ImpactEnergy * 800.0f * MesoMult);
    MesoCount = clamp(MesoCount, 20, 800);

    // 微尘层：低能量撞击不生成烟雾（阈值 0.2）
    int DustCount = 0;
    if (ImpactEnergy > 0.2f)
    {
        DustCount = (int)((ImpactEnergy - 0.2f) / 0.8f * 50000.0f);
        DustCount = clamp(DustCount, 0, 50000);
    }

    EmitMacroDebris(MacroCount, ImpactEnergy);
    EmitMesoParticles(MesoCount, ImpactEnergy);
    EmitDustSmoke(DustCount, ImpactEnergy);
}
```

当 `ImpactEnergy = 0.5`，玻璃材质时，三层分别生成 **30 个粗碎片、720 个中粒子、18750 个微尘粒子**，总发射量约 19500 个，在现代 GPU（RTX 3070 级别）上每帧耗时约 **0.8 ms**。

### 烟雾透明度衰减曲线

烟雾粒子的 Alpha 值随时间按指数衰减而非线性衰减，以模拟真实尘雾的消散特性：

$$\alpha(t) = \alpha_0 \times e^{-\lambda t} \times \left(1 - \frac{d}{d_{\max}}\right)$$

其中 $\alpha_0$ 为初始不透明度（通常取 0.7~0.9），$\lambda$ 为衰减速率常数（静止空气中 $\lambda \approx 0.3$，强风时 $\lambda \approx 0.8$），$d$ 为粒子当前离摄像机距离，$d_{\max}$ 为远裁平面距离。距离因子确保远处烟雾自动透明，节省 overdraw 开销。

## 实际应用

### 建筑爆破场景

混凝土柱被炸药破坏时，系统按以下时序执行：

- **0~0.05 秒**：伤害模型广播 `ImpactEnergy = 0.95`，Voronoi 主破碎触发，同时在 12 条主裂缝上各开启一条粒子发射带，共发射 57 个粗碎片（$20 \times 0.95 \approx 57$，取整后受上限约束为 57）
- **0.05~0.3 秒**：中粒子层 760 个碎石飞溅，沿抛物线散射，最远水平飞行距离约 4.5 m（初速 V₀ ≈ 0.95×0.40 × √0.95 ≈ 0.37，换算实际场景乘以比例因子后为约 9.3 m/s，飞行 0.4 s 落地）
- **0.3~5 秒**：35000 个微尘粒子形成直径约 6 m、高约 3 m 的尘雾云团，Alpha 从 0.85 按 λ=0.35 指数衰减
- **5~30 秒**：Decal 贴花在地面保留碎石痕迹，随后在 30 秒时启动 2 秒淡出动画

### 玻璃窗破碎场景

玻璃材质 $K_m = 0.82$ 导致碎片飞散速度显著更高，单块玻璃渣初速可达 **12~18 m/s**（对比混凝土粉尘约 5~7 m/s）。中粒子层数量乘以 1.8 系数，达到 **约 900 个**，且每片粒子角速度提升至 **360°/s ~ 1080°/s**，营造玻璃渣高速旋转的视觉特征。微尘层不产生灰尘，改为生成微量透明反光粒子（Alpha 仅 0.15~0.25）模拟玻璃粉末的光学折射效果。
