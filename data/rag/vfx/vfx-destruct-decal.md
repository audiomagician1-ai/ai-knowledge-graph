# 破坏痕迹

## 概述

破坏痕迹（Damage Decals）是指通过贴花（Decal）系统将弹痕、裂缝、焦痕、血迹等视觉标记投影到场景几何体表面的技术手段。与变形破坏不同，破坏痕迹不改变原始网格的拓扑结构，而是在表面叠加一层额外的渲染信息，以极低的性能开销模拟真实的破坏效果。

破坏贴花系统最早在2000年代初的第一人称射击游戏中得到广泛运用。《半衰期2》（2004年）率先在当时的游戏引擎中实现了持久化弹孔贴花，每个弹孔以独立的四边形面片附着在墙面上，配合法线贴图呈现出凹陷感。现代引擎如Unreal Engine 5采用Mesh Decal与DBuffer Decal两种管线，彻底解决了早期贴花在延迟渲染中的透明度排序问题。

破坏痕迹在视觉叙事层面具有不可替代的功能：一堵墙上密集的弹孔告诉玩家这里曾经发生过激烈战斗，地面上的焦痕暗示爆炸的中心位置。这些痕迹是场景可读性的重要组成部分，帮助玩家在没有文字提示的情况下理解空间历史。

## 核心原理

### 贴花投影机制

破坏贴花本质上是一个定向投影盒（Projection Box）。系统将一个有限体积的长方体放置于碰撞点，把贴花纹理沿投影轴方向"照射"到该体积内所有相交的几何表面。投影的UV坐标由顶点在投影盒内的局部坐标直接推导，公式为：

```
U = (LocalPos.x / BoxWidth) + 0.5
V = (LocalPos.y / BoxHeight) + 0.5
```

其中 `LocalPos` 是顶点在投影盒局部空间中的坐标，`BoxWidth` 和 `BoxHeight` 是投影盒在对应轴向的半尺寸。这一机制允许贴花自动跨越复杂曲面，无需手动UV展开。

### DBuffer与前向渲染贴花的区别

Unreal Engine 的 DBuffer Decal 在几何通道（GBuffer Pass）完成后、光照计算之前写入专用的 DBuffer，可以修改基础色（Base Color）、法线（Normal）和粗糙度（Roughness）三个通道。焦痕贴花通常会将粗糙度提升至 0.9 以上，模拟炭化表面的无光泽质感；弹痕贴花则在法线通道写入向内凹陷的半球法线，制造出金属变形感。前向渲染贴花只影响颜色，无法修改光照参数，因此在PBR工作流中优先选用DBuffer方案。

### 贴花池与生命周期管理

破坏痕迹系统必须对贴花数量设置上限，否则大量贴花会导致Draw Call激增。常见的做法是维护一个固定容量的**循环贴花池（Ring Buffer Decal Pool）**，新弹孔在最旧的贴花位置上复用对象。典型配置为：每种痕迹类型保留32到64个实例，超出后按FIFO（先入先出）顺序覆盖。部分游戏对距离玩家超过15米的贴花进行淡出处理，通过alpha渐变在约0.5秒内消除，避免突然消失产生的跳变感。

### 法线与深度偏移

贴花面片与底层几何体共面会产生Z-fighting闪烁问题。解决方法是给贴花的深度值添加一个微小的负偏移（Depth Bias），通常设置为 `-0.0001` 到 `-0.001` 的归一化深度单位，确保贴花始终渲染在底层表面之上，同时不影响阴影投射。

## 实际应用

**弹痕系统**：在FPS游戏中，子弹命中墙壁时系统执行线段检测（Line Trace），获取碰撞点坐标和表面法线，随后沿法线方向生成一个直径约3至8厘米的投影盒。弹痕贴花包含四张纹理：颜色贴图（带碎裂纹路）、法线贴图（凹陷边缘）、粗糙度贴图和蒙版贴图。《使命召唤》系列针对不同材质（混凝土、金属、木材、玻璃）分别维护独立的贴花集合，共约12种表面类型。

**裂缝贴花**：建筑物受到冲击后产生的蛛网状裂缝通常使用程序化生成的裂缝图案，以碰撞点为中心向外辐射，裂缝贴花的宽度随距离按平方衰减。为增强立体感，裂缝贴花的法线通道会在裂缝两侧写入相反方向的倾斜法线，模拟混凝土断裂翘起的微小高度差。

**焦痕与熏黑**：爆炸或火焰产生的焦痕半径通常是爆炸伤害半径的1.2到1.5倍，贴花边缘使用径向渐变蒙版实现自然衰减。焦痕贴花的颜色通道从中心的纯黑（0, 0, 0）向外过渡到深棕（0.15, 0.08, 0.03），同时粗糙度通道设置为均匀的高粗糙值，消除表面高光。

## 常见误区

**误区一：贴花数量越多，视觉效果越真实**
大量堆叠的贴花会导致严重的渲染开销，每个DBuffer贴花都需要对其投影体积内的所有像素执行额外的渲染通道。实测在Unreal Engine 5中，同一区域200个以上活跃贴花会使该区域的帧时间上升30%至50%。真实感来自于**合理的密度和多样的角度变化**，而非单纯叠加数量。

**误区二：弹痕应当精确吸附在表面法线方向**
直接沿碰撞法线放置贴花在曲面上会导致投影拉伸变形。正确做法是将投影盒的朝向限制在与法线夹角不超过45度的范围内，对于曲率过大的表面（如圆柱），应将贴花投影轴对齐到世界空间的最近正交方向，以减少UV拉伸。

**误区三：破坏痕迹与变形破坏可以互相替代**
破坏痕迹无法表现几何体碎裂、形变或体积丢失，它只处理表面视觉信息。墙壁上的弹孔如果不配合实际的网格穿孔或视差贴图（Parallax Occlusion Mapping，深度通常设置为0.05到0.15个单位），在近距离观察时会暴露出"纸片感"。两者在流程上是互补关系，并非替代关系。

## 知识关联

**前置概念——变形破坏**：变形破坏处理网格层面的几何变化，破坏痕迹则在变形破坏发生后的表面上叠加视觉细节。例如，一堵墙被炮弹击中后，变形破坏负责将墙体切割成碎块，而破坏痕迹负责在弹坑周围的残余墙体表面附上烧焦和碎裂纹路，两个系统协同触发才能实现完整的击中反馈。

**后续概念——破碎音效**：破坏痕迹系统在生成贴花的同时会携带表面材质信息（混凝土、金属、玻璃等），这份材质信息直接传递给音效系统，用于选择对应的碰撞音效资产。破碎音效与贴花类型的绑定通常在材质数据表（Material Data Table）中统一配置，确保弹痕的视觉类型和声音反馈保持一致。
