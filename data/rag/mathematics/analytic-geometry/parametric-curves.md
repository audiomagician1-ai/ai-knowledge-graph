# 参数曲线

## 概述

参数曲线是通过引入第三个变量（参数）来同时表示平面或空间中点坐标的曲线表示方法。其标准形式为 $x = f(t)$，$y = g(t)$，其中 $t$ 在某区间 $[a, b]$ 上变化，曲线上的每一点对应参数 $t$ 的一个取值。与直接用 $y = f(x)$ 表示的显函数曲线不同，参数曲线能够自然地描述不满足函数关系的图形，例如闭合曲线和自交曲线。

参数曲线的思想可以追溯到17世纪。笛卡尔在研究摆线时最早系统地使用了参数描述：他以圆的滚动角 $\theta$ 为参数，写出 $x = r(\theta - \sin\theta)$，$y = r(1 - \cos\theta)$，完美刻画了这条无法用 $y = f(x)$ 形式写出的著名曲线。此后，莱布尼茨和伯努利兄弟在研究悬链线、星形线等问题时进一步发展了这套语言。

在解析几何中引入参数曲线的根本原因是**表达能力的扩展**：圆、椭圆、阿基米德螺线等大量几何对象无法用单一的显函数描述，却可以用参数方程优雅地统一表示。此外，参数化形式天然携带了曲线的**走向信息**，即参数 $t$ 增大时点沿曲线运动的方向，这在物理建模和计算机图形学中具有直接意义。

## 核心原理

### 参数方程的建立与几何意义

建立参数曲线的关键在于选取恰当的几何量或物理量作为参数。以单位圆为例，选取圆心角 $\theta$ 为参数，得到 $x = \cos\theta$，$y = \sin\theta$，$\theta \in [0, 2\pi)$。这里参数 $\theta$ 不是坐标，而是描述点在圆上位置的"索引"。参数方程所定义的点集 $\{(f(t), g(t)) \mid t \in I\}$ 就是参数曲线的轨迹。

同一条轨迹可以有无穷多种参数化方式，例如圆还可以参数化为 $x = \frac{1-t^2}{1+t^2}$，$y = \frac{2t}{1+t^2}$（有理参数化），但后者会遗漏点 $(-1, 0)$，因此不同参数化在覆盖范围上可能有差异，这是使用时必须注意的细节。

### 消参方法：从参数形式到普通方程

消参是指从参数方程中消去参数 $t$，得到曲线满足的直角坐标方程 $F(x, y) = 0$。常用的消参技术有以下几种：

**代数消元法**：当 $f(t)$ 和 $g(t)$ 都是 $t$ 的有理函数时，直接用一个方程解出 $t$，代入另一个方程。例如 $x = t + 1$，$y = t^2 - 2t$，由第一式得 $t = x - 1$，代入得 $y = (x-1)^2 - 2(x-1) = x^2 - 4x + 3$。

**三角恒等式法**：当参数方程含有三角函数时，利用 $\sin^2 t + \cos^2 t = 1$ 等恒等式消参。椭圆 $x = a\cos t$，$y = b\sin t$ 消参：由两式得 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = \cos^2 t + \sin^2 t = 1$，即得标准椭圆方程。

**双曲型消参**：对含有 $\sec t$、$\tan t$ 的参数方程，利用 $\sec^2 t - \tan^2 t = 1$。如 $x = a\sec t$，$y = b\tan t$，消参得 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$，即双曲线。

消参时必须同步处理参数的**定义域限制**，因为消参后得到的方程可能包含更多点（即超出原参数范围的部分）。例如 $x = \cos^2 t$，$y = \sin^2 t$ 消参得 $x + y = 1$，但由于 $\cos^2 t \geq 0$ 且 $\sin^2 t \geq 0$，实际轨迹只是线段 $x + y = 1$（$0 \leq x \leq 1$），而非整条直线。

### 参数曲线的微分性质

参数曲线上的切线斜率由链式法则给出：若 $f'(t) \neq 0$，则

$$\frac{dy}{dx} = \frac{dy/dt}{dx/dt} = \frac{g'(t)}{f'(t)}$$

二阶导数（用于判断曲线弯曲方向）为

$$\frac{d^2y}{dx^2} = \frac{\frac{d}{dt}\left(\frac{g'(t)}{f'(t)}\right)}{f'(t)} = \frac{g''(t)f'(t) - g'(t)f''(t)}{[f'(t)]^3}$$

当 $f'(t_0) = g'(t_0) = 0$ 时，点 $(f(t_0), g(t_0))$ 称为**奇点**，该点处切线方向需用极限方法进一步分析。摆线在 $\theta = 0, 2\pi, 4\pi, \ldots$ 处就是这样的尖点（cusp）。

## 实际应用

**星形线**（astroid）是参数曲线的经典案例：$x = a\cos^3 t$，$y = a\sin^3 t$，$t \in [0, 2\pi]$。消参得 $x^{2/3} + y^{2/3} = a^{2/3}$。星形线描述的是一个半径为 $a/4$ 的小圆在半径为 $a$ 的大圆内部滚动时，小圆上一个定点的轨迹，在机械工程中用于设计内摆线齿轮。

**摆线**在最速降线问题（1696年约翰·伯努利提出）中是最优解：一质点从 $A$ 滑到 $B$ 无摩擦最快的路径恰好是摆线弧，其参数方程为 $x = r(\theta - \sin\theta)$，$y = r(1 - \cos\theta)$。此结论无法用普通显函数方程推导，必须借助参数形式进行变分计算。

**计算机辅助设计**中，贝塞尔曲线是参数曲线的工程实现。三次贝塞尔曲线由4个控制点 $P_0, P_1, P_2, P_3$ 定义：$\mathbf{B}(t) = (1-t)^3P_0 + 3(1-t)^2t P_1 + 3(1-t)t^2 P_2 + t^3P_3$，$t \in [0,1]$。PostScript、SVG等格式中的曲线路径均采用这一参数形式存储。

## 常见误区

**误区一：消参后的方程就完整代表了原曲线。** 许多学生认为消参成功即任务完成，却忽略了参数范围对轨迹的限制。例如 $x = e^t$，$y = e^{2t}$ 消参得 $y = x^2$，但由于 $e^t > 0$，实际轨迹只是抛物线 $x > 0$ 的右半部分，不包含 $y$ 轴及左半部分。必须在消参后重新用原参数约束范围写出 $x$ 的定义域。

**误区二：参数 $t$ 必须有具体的物理或几何意义。** 参数是纯粹的数学工具，不需要对应时间、角度等实体。对于同一条曲线，可以选用任意单调函数对参数进行**重参数化**：若 $t = \varphi(s)$ 单调，则 $x = f(\varphi(s))$，$y = g(\varphi(s))$ 描述完全相同的轨迹。学生在建立参数方程时不必拘泥于寻找"有意义"的参数，选取计算方便的参数即可。

**误区三：参数曲线一定比普通方程复杂。** 实际上，许多曲线的参数形式反而比直角方程简洁。笛卡尔叶形线的直角方程为 $x^3 + y^3 = 3axy$，而其参数方程为 $x = \frac{3at}{1+t^3}$，$y = \frac{3at^2}{1+t^3}$，后者在求面积、弧长等积分计算时更易操作。
