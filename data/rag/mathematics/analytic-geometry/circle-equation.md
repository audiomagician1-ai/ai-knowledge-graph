# 圆的方程

## 概述

圆的方程是用代数方法描述平面上圆这一几何图形的核心工具。平面上到定点距离等于定长的所有点的集合构成圆，将这一几何定义翻译为坐标语言，就得到圆的方程。圆的标准方程形式为 $(x - a)^2 + (y - b)^2 = r^2$，其中 $(a, b)$ 是圆心坐标，$r > 0$ 是半径。

解析几何由法国数学家笛卡尔在1637年出版的《几何学》中奠基，圆的方程是其中最早被系统讨论的曲线方程之一。将几何图形转化为代数方程，使得求交点、判断位置关系等几何问题可以转化为方程组的求解，极大拓展了处理几何问题的手段。

圆的方程在解析几何课程中处于特殊位置：它是学生第一次系统接触"曲线↔方程"对应关系的场合。椭圆、双曲线、抛物线的方程推导都沿用了圆的方程推导思路——先写出点到焦点（或定点）距离满足的条件，再化简为代数式。因此，透彻理解圆的方程的推导逻辑，是后续学习所有二次曲线的基础。

## 核心原理

### 标准方程的推导

设圆心为 $C(a, b)$，半径为 $r$。对于圆上任意一点 $P(x, y)$，由两点距离公式 $|CP| = \sqrt{(x-a)^2 + (y-b)^2}$，令此距离等于 $r$ 并两边平方，得：

$$
(x - a)^2 + (y - b)^2 = r^2
$$

特别地，当圆心在原点时，$a = b = 0$，方程化为 $x^2 + y^2 = r^2$。标准方程中，$r^2$ 必须严格大于零；若等式右边为零，图形退化为单点；若为负数，方程无实数解，图形不存在。

### 一般方程及其识别

将标准方程 $(x - a)^2 + (y - b)^2 = r^2$ 展开，整理后得到圆的**一般方程**：

$$
x^2 + y^2 + Dx + Ey + F = 0
$$

其中 $D = -2a$，$E = -2b$，$F = a^2 + b^2 - r^2$。

反之，给定一个含 $x^2 + y^2$ 项的二元二次方程，通过**配方**可以还原出圆心和半径：
$$
\left(x + \frac{D}{2}\right)^2 + \left(y + \frac{E}{2}\right)^2 = \frac{D^2 + E^2 - 4F}{4}
$$

因此，一般方程表示圆的充要条件是：$x^2$ 与 $y^2$ 的系数相等且不为零（通常为1），**且** $D^2 + E^2 - 4F > 0$。当 $D^2 + E^2 - 4F = 0$ 时退化为点圆，小于零时无图形。

从一般方程快速读出圆心坐标：$\left(-\dfrac{D}{2},\ -\dfrac{E}{2}\right)$；半径：$r = \dfrac{\sqrt{D^2 + E^2 - 4F}}{2}$。

### 圆与直线的位置关系

设直线 $l: y = kx + m$（或一般式 $Ax + By + C = 0$），圆 $O: (x-a)^2 + (y-b)^2 = r^2$，**圆心到直线的距离**决定了位置关系：

$$
d = \frac{|Aa + Bb + C|}{\sqrt{A^2 + B^2}}
$$

- $d > r$：直线与圆**相离**，无交点；
- $d = r$：直线与圆**相切**，恰有1个公共点，切点处半径⊥切线；
- $d < r$：直线与圆**相交**，有2个公共点，弦长为 $2\sqrt{r^2 - d^2}$。

弦长公式 $l = 2\sqrt{r^2 - d^2}$ 中，$d$ 是弦的垂直平分线（即圆心到弦所在直线）的距离，这条垂直平分线必过圆心。

## 实际应用

**例1：由三点确定圆的方程。** 已知三点 $A(1, 1)$，$B(1, -1)$，$C(3, 0)$，求过三点的圆。将三点代入一般方程 $x^2 + y^2 + Dx + Ey + F = 0$，建立含 $D, E, F$ 的三元线性方程组，解方程组得 $D = -4, E = 0, F = 2$，即圆方程为 $x^2 + y^2 - 4x + 2 = 0$，配方得 $(x-2)^2 + y^2 = 2$，圆心 $(2, 0)$，半径 $\sqrt{2}$。

**例2：求过圆上一点的切线方程。** 圆 $x^2 + y^2 = 25$ 上点 $P(3, 4)$ 处的切线，利用"切线⊥半径"：圆心 $(0,0)$ 到 $P$ 的斜率为 $\frac{4}{3}$，切线斜率为 $-\frac{3}{4}$，切线方程为 $3x + 4y - 25 = 0$。注意切线方程中系数 $3, 4$ 恰为点 $P$ 的坐标，这正是公式 $x_0 x + y_0 y = r^2$ 的体现。

**例3：弦长问题。** 直线 $y = x + 1$ 被圆 $(x-1)^2 + (y-1)^2 = 9$ 截得的弦长：圆心 $(1,1)$ 到直线 $x - y + 1 = 0$ 的距离 $d = \frac{|1-1+1|}{\sqrt{2}} = \frac{1}{\sqrt{2}}$，弦长 $= 2\sqrt{9 - \frac{1}{2}} = 2\sqrt{\frac{17}{2}} = \sqrt{34}$。

## 常见误区

**误区1：将标准方程右边误写为 $r$ 而非 $r^2$。** 学生看到"半径为3"，有时直接写 $(x-1)^2 + (y-2)^2 = 3$，这实际表示半径为 $\sqrt{3}$ 的圆，而非半径为3的圆。圆的标准方程等号右边是半径的**平方**，务必区分 $r$ 与 $r^2$。

**误区2：一般方程配方后忘记判断 $D^2 + E^2 - 4F > 0$。** 例如 $x^2 + y^2 - 2x + 4y + 10 = 0$，配方后右边为 $1 + 4 - 10 = -5 < 0$，方程无实数解，不表示任何圆，但学生常忽略此判断，直接当作圆处理。

**误区3：直线与圆的切线概念混淆。** 有些学生认为过圆外一点只能作一条切线，实际上过圆外一点可作**两条**切线。过圆上一点才恰好只有一条切线（即该点处的切线）。在解切线方程时，设斜率为 $k$ 再令判别式 $\Delta = 0$ 是通用方法，但当切线斜率不存在（竖直切线）时，必须单独验证，否则会漏解。

## 知识关联

**前置知识：距离与中点公式。** 圆的标准方程的推导直接使用了两点距离公式 $d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$；弦的垂直平分线过圆心这一性质，需要用到中点公式来求弦中点坐标。若对距离公式不熟悉，标准方程的建立过程就会出现障碍。

**后续拓展：椭圆与双曲线。** 椭圆的定义"到两定点距离之和为常数"可视为圆的推广——圆是两焦点重合时椭圆的特例（离心率 $e = 0$）。椭圆标准方程 $\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1$（$a > b > 0$）的推导，和圆的方程推导一样，都是对距离条件进行配方化简。双曲线中直线与曲线位置关系的讨论（相离、相切、相交、渐近），也直接沿用了圆与直线位置关系中"圆心到直线距离 $d$ 与 $r$ 比较"的分析框架。掌握圆中"代入方程组，用判别式 $\Delta$ 判断交点个数"和"几何法用 $d$ 与 $r$ 比较"这两套方法，对椭圆、双曲线同类问题同样适用。
