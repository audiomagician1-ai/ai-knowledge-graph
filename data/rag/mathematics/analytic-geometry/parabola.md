# 抛物线

## 概述

抛物线是平面内到一个定点（焦点）的距离等于到一条定直线（准线）的距离的点的集合。这个"等距"条件是抛物线区别于椭圆和双曲线的根本特征——椭圆的焦点距离之和为常数，双曲线的焦点距离之差为常数，而抛物线只涉及一个焦点和一条准线，且比值恰好等于1（即离心率 $e=1$）。

抛物线的研究可追溯至古希腊数学家阿波罗尼斯（约公元前262—190年），他在《圆锥曲线》八卷中系统研究了抛物线，并命名为"parabola"（意为"并列/相等"），暗指焦点距离与准线距离相等的性质。17世纪，伽利略证明了抛体在忽略空气阻力时的运动轨迹正是抛物线，从而使这条曲线在物理学中获得了重要地位。

抛物线的实用价值体现在其反射性质上：从焦点发出的光线经抛物面反射后平行于轴射出，卫星天线、探照灯、汽车车头灯均利用这一原理。掌握抛物线的标准方程是分析焦点坐标、准线位置和弦长等几何量的基础工具。

## 核心原理

### 标准方程与参数含义

抛物线共有四种标准方程，取决于开口方向：

| 标准方程 | 开口方向 | 焦点坐标 | 准线方程 |
|---|---|---|---|
| $y^2 = 2px$（$p>0$） | 向右 | $\left(\dfrac{p}{2}, 0\right)$ | $x = -\dfrac{p}{2}$ |
| $y^2 = -2px$（$p>0$） | 向左 | $\left(-\dfrac{p}{2}, 0\right)$ | $x = \dfrac{p}{2}$ |
| $x^2 = 2py$（$p>0$） | 向上 | $\left(0, \dfrac{p}{2}\right)$ | $y = -\dfrac{p}{2}$ |
| $x^2 = -2py$（$p>0$） | 向下 | $\left(0, -\dfrac{p}{2}\right)$ | $y = \dfrac{p}{2}$ |

以 $y^2 = 2px$ 为例，参数 $p$ 是焦点到准线的距离，也称**焦准距**。焦点到顶点的距离为 $\dfrac{p}{2}$，而不是 $p$，这是初学者最常混淆之处。顶点始终位于原点，这使得标准方程的计算比非标准位置简洁得多。

### 焦点弦与通径

过焦点作垂直于对称轴的弦，称为**通径**（latus rectum），其长度是 $2p$。这是抛物线上过焦点的最短弦。

对于过焦点的任意弦（焦点弦），设两端点 $A(x_1, y_1)$ 和 $B(x_2, y_2)$ 在抛物线 $y^2 = 2px$ 上，则有以下关系：

$$x_1 x_2 = \frac{p^2}{4}, \quad y_1 y_2 = -p^2$$

焦点弦长公式为：

$$|AB| = x_1 + x_2 + p = \frac{2p}{\sin^2\theta}$$

其中 $\theta$ 是弦与对称轴的夹角。当 $\theta = 90°$ 时弦长最小，等于通径 $2p$；当 $\theta \to 0°$ 时弦长趋于无穷大，体现了抛物线无限延伸的特性。

### 点到焦点的距离公式

对于抛物线 $y^2 = 2px$ 上的点 $P(x_0, y_0)$，其到焦点 $F\!\left(\dfrac{p}{2}, 0\right)$ 的距离为：

$$|PF| = x_0 + \frac{p}{2}$$

这由抛物线定义直接得出：点到焦点的距离等于点到准线 $x = -\dfrac{p}{2}$ 的距离，即 $x_0 - \left(-\dfrac{p}{2}\right) = x_0 + \dfrac{p}{2}$。由于 $x_0 \geq 0$，所以 $|PF| \geq \dfrac{p}{2}$，等号当 $x_0 = 0$（顶点处）成立。

## 实际应用

**例题：已知抛物线 $y^2 = 8x$，过焦点作倾斜角为 $60°$ 的弦，求弦长。**

由 $y^2 = 8x$ 知 $2p = 8$，即 $p = 4$，焦点为 $(2, 0)$。用焦点弦长公式：

$$|AB| = \frac{2p}{\sin^2 60°} = \frac{8}{\left(\frac{\sqrt{3}}{2}\right)^2} = \frac{8}{\frac{3}{4}} = \frac{32}{3}$$

**卫星天线设计**：某卫星天线抛物面的截面方程为 $x^2 = 4y$（单位：米），则 $2p = 4$，$p = 2$，焦点在 $(0, 1)$ 处，即距顶点（天线中心最低点）1米处放置信号接收器，可将所有平行于轴的信号汇聚于该点。

**抛体运动**：以初速度 $v_0$ 和仰角 $45°$ 抛出的物体，其轨迹方程为 $y = x - \dfrac{g}{v_0^2}x^2$，整理后为 $\left(x - \dfrac{v_0^2}{2g}\right)^2 = -\dfrac{v_0^2}{g}\left(y - \dfrac{v_0^2}{4g}\right)$，这是一条开口向下的抛物线，顶点即为最高点。

## 常见误区

**误区一：将 $y^2 = 2px$ 中的 $p$ 当作焦点横坐标。**
焦点横坐标是 $\dfrac{p}{2}$，不是 $p$。例如 $y^2 = 6x$ 中，$2p = 6$，所以 $p = 3$，焦点为 $\left(\dfrac{3}{2}, 0\right)$，准线为 $x = -\dfrac{3}{2}$。许多学生误写焦点为 $(3, 0)$，实为将 $2p$ 整体当成 $p$ 代入。

**误区二：认为抛物线有两条准线或两个焦点。**
抛物线只有一个焦点和一条准线，这与椭圆（两焦点）和双曲线（两焦点两准线）形成对比。混淆来源于在学习圆锥曲线时未清晰区分离心率 $e = 1$ 的特殊性：椭圆 $e < 1$，双曲线 $e > 1$，正是 $e = 1$ 使抛物线退化为只需一侧约束的开放曲线。

**误区三：把抛物线 $y^2 = 2px$ 与二次函数 $y = ax^2$ 等同处理。**
$y^2 = 2px$ 是关于 $x$ 轴对称的隐函数，它不是函数（同一个 $x$ 对应两个 $y$ 值）；而 $y = ax^2$ 是关于 $y$ 轴对称的显函数。两者都是抛物线，但标准方程、对称轴方向完全不同。在解析几何中遇到 $y^2 = 2px$ 时，不能套用二次函数的顶点式、对称轴公式。

## 知识关联

**前置知识**：二次函数 $y = ax^2 + bx + c$ 的图象就是抛物线，其顶点式 $y = a(x-h)^2 + k$ 揭示了顶点坐标，这与解析几何中抛物线的顶点概念相通。但二次函数阶段未引入焦点和准线，解析几何正是在此基础上用距离定义重新审视这条曲线，使其从"函数图象"升格为"轨迹方程"。

**后续概念**：抛物线是圆锥曲线统一定义（焦点-准线定义，离心率 $e$）的特殊情形（$e=1$）。学完抛物线后，圆锥曲线的统一定义将用焦半径公式 $r = \dfrac{ep}{1 - e\cos\theta}$（极坐标形式）同时描述椭圆、双曲线和抛物线，抛物线的焦点弦公式 $|AB| = \dfrac{2p}{\sin^2\theta}$ 也将在统一框架下得到推广。
