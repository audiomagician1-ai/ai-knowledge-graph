# 距离与中点公式

## 概述

距离与中点公式是解析几何中将代数计算与几何度量相结合的两组基本公式。**两点距离公式**将平面上任意两点之间的直线距离表达为坐标差的函数：对于点 $A(x_1, y_1)$ 和点 $B(x_2, y_2)$，其距离为 $d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$。**中点公式**则给出线段 $AB$ 的中点坐标 $M\left(\dfrac{x_1+x_2}{2},\ \dfrac{y_1+y_2}{2}\right)$。

两点距离公式的理论根源是古希腊毕达哥拉斯定理（约公元前 500 年），但将其纳入坐标系框架是 17 世纪笛卡尔建立解析几何体系后的成果。在笛卡尔 1637 年的著作《几何学》中，坐标方法首次系统地允许用代数式描述几何对象的位置与距离，从而使这两个公式得以精确表达。

这两个公式在实际数学学习中承担着非常具体的任务：距离公式是推导圆的标准方程的直接依据（圆定义为到定点距离等于定长的点的集合），而中点公式则是证明四边形各类性质（如平行四边形对角线互相平分）的计算工具。

## 核心原理

### 两点距离公式的推导

设平面直角坐标系中两点 $A(x_1, y_1)$、$B(x_2, y_2)$，过 $A$ 作平行于 $x$ 轴的水平线，过 $B$ 作平行于 $y$ 轴的竖直线，两线交于点 $C(x_2, y_1)$。由此构成直角三角形 $ABC$，其中两直角边长度分别为：

$$AC = |x_2 - x_1|, \quad BC = |y_2 - y_1|$$

由勾股定理得：

$$|AB|^2 = AC^2 + BC^2 = (x_2 - x_1)^2 + (y_2 - y_1)^2$$

因此 $d = |AB| = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$。

注意公式中对坐标差取平方，所以无论 $x_2 - x_1$ 正负均不影响结果，不必使用绝对值符号。当两点在同一水平线上（$y_1 = y_2$）时，公式退化为 $d = |x_2 - x_1|$；当在同一竖直线上（$x_1 = x_2$）时，退化为 $d = |y_2 - y_1|$。

### 中点公式的推导

中点 $M$ 的横坐标等于 $A$、$B$ 两点横坐标的算术平均值，纵坐标同理。严格推导基于向量等分：$M$ 是 $AB$ 中点当且仅当 $\overrightarrow{AM} = \overrightarrow{MB}$，即：

$$x_M - x_1 = x_2 - x_M \implies x_M = \frac{x_1 + x_2}{2}$$

$$y_M - y_1 = y_2 - y_M \implies y_M = \frac{y_1 + y_2}{2}$$

中点公式也可推广：若点 $P$ 将线段 $AB$ 按比例 $\lambda : 1$ 内分，则 $x_P = \dfrac{x_1 + \lambda x_2}{1 + \lambda}$，这是高中阶段"分点公式"的基础，中点是 $\lambda = 1$ 的特殊情况。

### 公式的数值感知

在实际计算中，$A(1, 2)$ 与 $B(4, 6)$ 之间的距离为 $\sqrt{(4-1)^2+(6-2)^2} = \sqrt{9+16} = \sqrt{25} = 5$，恰好是整数，这是经典的 3-4-5 勾股数组的坐标化体现。两点中点为 $\left(\dfrac{1+4}{2}, \dfrac{2+6}{2}\right) = (2.5,\ 4)$。

## 实际应用

**判断三角形形状**：已知三点 $A(0,0)$、$B(4,0)$、$C(2,2\sqrt{3})$，分别计算三边长：$AB = 4$，$AC = \sqrt{4+12} = 4$，$BC = \sqrt{4+12} = 4$，三边相等，因此 $\triangle ABC$ 是等边三角形。这一判断完全依赖两点距离公式，无需画图。

**求圆的方程**：圆的标准方程 $(x-a)^2 + (y-b)^2 = r^2$ 中，半径 $r$ 定义为圆上任意一点 $(x,y)$ 到圆心 $(a,b)$ 的距离，其表达式正是两点距离公式的平方形式。若已知直径两端点为 $A(2,-1)$ 和 $B(6,3)$，则先用中点公式求圆心 $(4, 1)$，再用距离公式求半径 $r = \sqrt{(6-4)^2+(3-1)^2} = \sqrt{8} = 2\sqrt{2}$，写出方程 $(x-4)^2+(y-1)^2=8$。

**验证平行四边形**：四点 $A(1,1)$、$B(4,1)$、$C(5,3)$、$D(2,3)$，$AC$ 中点 $= \left(3, 2\right)$，$BD$ 中点 $= \left(3, 2\right)$，两对角线中点重合，由中点公式直接证明 $ABCD$ 是平行四边形。

## 常见误区

**误区一：忘记对坐标差取平方，而是取绝对值后相加**。错误写法 $d = |x_2-x_1| + |y_2-y_1|$ 实际上是"曼哈顿距离"，不是欧氏直线距离。以 $A(0,0)$、$B(1,1)$ 为例，正确答案是 $\sqrt{2} \approx 1.414$，而错误公式给出 $2$，相差超过 41%。直线距离公式必须先平方再相加再开根。

**误区二：中点公式误用为"坐标差的一半加上其中一点"**。部分学生写成 $x_M = x_1 + \dfrac{x_2 - x_1}{2}$，虽然化简后等价于正确公式，但在数值计算时容易因运算顺序错误出错，建议始终使用 $x_M = \dfrac{x_1 + x_2}{2}$ 的标准形式。

**误区三：将距离公式用于三维空间时漏掉 $z$ 轴分量**。在高中平面几何阶段，距离公式只含两个坐标差，但进入立体坐标系后，空间两点距离为 $d = \sqrt{(x_2-x_1)^2+(y_2-y_1)^2+(z_2-z_1)^2}$，遗漏 $z$ 项会导致计算结果偏小。

## 知识关联

**前置知识**：两点距离公式的推导完全依赖直角坐标系中"坐标表示位移"的概念，以及勾股定理 $a^2+b^2=c^2$。若不理解坐标差 $x_2-x_1$ 代表水平位移，则无法理解为何可以构造辅助直角三角形。

**直接后续**：圆的标准方程 $(x-a)^2+(y-b)^2=r^2$ 是两点距离公式对"到定点距离等于定值"这一条件的直接代数翻译，学生在推导圆方程时每一步都在使用距离公式。中点公式在后续学习椭圆中点弦问题时也会反复出现——联立椭圆方程与直线方程后，用韦达定理求出两根之和，再除以 2 即得中点横坐标，与中点公式思路完全一致。
