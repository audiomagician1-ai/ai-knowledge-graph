# 距离与中点公式

## 概述

距离与中点公式是解析几何中用坐标代数方法度量平面上两点之间几何关系的基本工具。距离公式将两点间的线段长度转化为坐标差的代数运算，中点公式则给出线段中点的精确坐标表达式，两者共同构成平面几何度量计算的代数基础。

距离公式的本质来源于勾股定理。设平面上两点 $P_1(x_1, y_1)$ 和 $P_2(x_2, y_2)$，以这两点为端点作直角三角形，横向直角边长为 $|x_2 - x_1|$，纵向直角边长为 $|y_2 - y_1|$，斜边即为所求距离。这一几何对应关系最早由笛卡尔在1637年发表的《几何学》中系统化，奠定了用代数方程描述几何图形的方法论。

中点公式源于线段中点将两端点坐标"平均"的几何直觉：中点横坐标恰好是两端点横坐标的算术平均值，纵坐标同理。这两个公式在圆的方程推导、直线方程建立、向量计算等后续内容中被反复直接调用，因此在解析几何学习链条中属于不可跳过的度量基础。

## 核心原理

### 两点距离公式

设平面直角坐标系中两点 $P_1(x_1, y_1)$，$P_2(x_2, y_2)$，则两点之间的距离为：

$$|P_1P_2| = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$

其中 $(x_2 - x_1)$ 是横坐标之差，$(y_2 - y_1)$ 是纵坐标之差。由于两差值均被平方，所以公式中不需要绝对值符号——负数平方后仍为正数，计算结果天然非负。注意当 $x_1 = x_2$ 时，公式退化为 $|y_2 - y_1|$，即两点在同一竖直线上；当 $y_1 = y_2$ 时，退化为 $|x_2 - x_1|$，即两点在同一水平线上。这两种退化情形均与直接观察一致，是验证公式正确性的直观检验。

推导过程明确：过 $P_1$ 作平行于 $x$ 轴的射线，过 $P_2$ 作平行于 $y$ 轴的射线，两线交于点 $Q(x_2, y_1)$。则 $P_1Q = |x_2 - x_1|$，$P_2Q = |y_2 - y_1|$，由勾股定理得 $|P_1P_2|^2 = |P_1Q|^2 + |P_2Q|^2$，开方即得距离公式。

### 中点坐标公式

设线段 $P_1P_2$ 的中点为 $M(x, y)$，则：

$$x = \frac{x_1 + x_2}{2}, \quad y = \frac{y_1 + y_2}{2}$$

简写为 $M\!\left(\dfrac{x_1+x_2}{2},\, \dfrac{y_1+y_2}{2}\right)$。

中点公式的严格推导依赖相似三角形或向量的等分条件：$M$ 是 $P_1P_2$ 的中点，当且仅当 $\overrightarrow{P_1M} = \overrightarrow{MP_2}$，即 $(x - x_1, y - y_1) = (x_2 - x, y_2 - y)$，解方程组立即得到上述结果。这一推导同时揭示了中点公式是向量"等分"条件的坐标表达。

### 与原点距离的特例

当一端点为原点 $O(0, 0)$ 时，距离公式化简为 $|OP| = \sqrt{x^2 + y^2}$，这正是点 $(x, y)$ 到原点的距离，也是圆方程 $x^2 + y^2 = r^2$ 的几何含义基础——以原点为圆心、半径为 $r$ 的圆上各点，恰好满足到原点距离等于 $r$。

## 实际应用

**判断三角形形状**：已知三点 $A(0,0)$、$B(3,0)$、$C\!\left(\frac{3}{2}, \frac{3\sqrt{3}}{2}\right)$，计算三边长：$|AB|=3$，$|BC|=\sqrt{\left(\frac{3}{2}\right)^2+\left(\frac{3\sqrt{3}}{2}\right)^2}=\sqrt{\frac{9}{4}+\frac{27}{4}}=3$，$|AC|=3$，三边相等，判定为等边三角形。若仅用几何法，在坐标给定后无法快速判定；距离公式将判断转化为纯数值计算。

**求线段中点再验证距离**：线段端点为 $P_1(-2, 4)$，$P_2(6, -2)$，中点 $M = \left(\dfrac{-2+6}{2}, \dfrac{4+(-2)}{2}\right) = (2, 1)$。验证：$|P_1M| = \sqrt{(2-(-2))^2+(1-4)^2}=\sqrt{16+9}=5$，$|P_2M|=\sqrt{(6-2)^2+(-2-1)^2}=\sqrt{16+9}=5$，两段相等，中点公式结果正确。

**建立圆的方程的前置步骤**：已知圆的一条直径端点为 $A(1,3)$ 和 $B(5,7)$，圆心即直径中点 $C = (3, 5)$，半径 $r = |CA| = \sqrt{(3-1)^2+(5-3)^2}=\sqrt{8}=2\sqrt{2}$。圆的方程即 $(x-3)^2+(y-5)^2=8$，全程依赖中点公式和距离公式。

## 常见误区

**误区一：距离公式中遗漏平方根**。初学者常写成 $|P_1P_2| = (x_2-x_1)^2+(y_2-y_1)^2$，忘记开方。正确结果是对坐标差的平方和取算术平方根。例如 $P_1(1,1)$、$P_2(4,5)$：错误写法得 $9+16=25$，正确结果为 $\sqrt{25}=5$。数值上恰好等于整数容易掩盖此错误，应注意区分。

**误区二：中点公式与坐标差混淆**。有学生将中点坐标写作 $\left(\dfrac{x_2-x_1}{2}, \dfrac{y_2-y_1}{2}\right)$，这实际上是向量 $\overrightarrow{P_1P_2}$ 的一半，而非中点坐标。中点坐标是两点坐标之"和"的一半，写成 $\dfrac{x_1+x_2}{2}$，"和"与"差"的混淆会导致中点偏离正确位置。

**误区三：认为公式中 $P_1$、$P_2$ 的顺序影响距离结果**。由于 $(x_2-x_1)^2=(x_1-x_2)^2$，距离 $|P_1P_2|=|P_2P_1|$，顺序交换不影响距离值。但对于有向线段或向量模，方向有意义，此处仅限两点间距离为无方向的正实数。

## 知识关联

**前置知识**：距离公式直接依赖直角坐标系中点的坐标表示 $(x, y)$ 和勾股定理 $a^2+b^2=c^2$。没有坐标系的建立就无法定义 $x_1, y_1, x_2, y_2$ 这四个量，也无法将几何距离转化为代数运算。

**后续应用**：圆的方程 $(x-a)^2+(y-b)^2=r^2$ 的定义正是"平面上到定点 $(a,b)$ 距离等于 $r$ 的点的集合"，其中"距离"直接调用距离公式展开。求圆心需要用中点公式处理直径端点。此外，在椭圆、双曲线的焦点距离条件中，距离公式同样是建立方程的基本操作。掌握这两个公式，是后续所有圆锥曲线方程推导的代数出发点。
