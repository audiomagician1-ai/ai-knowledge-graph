# 双曲线

## 概述

双曲线是平面内到两个定点（焦点）距离之差的绝对值等于常数的点的集合。设两焦点为 $F_1$、$F_2$，点 $P$ 在双曲线上，则定义式为 $||PF_1| - |PF_2|| = 2a$，其中要求 $2a < |F_1F_2|$（即 $2a < 2c$，也就是 $a < c$）。这一条件与椭圆的 $2a > 2c$ 恰好相反，导致双曲线在形状和方程上都与椭圆有本质区别。

双曲线的系统研究可追溯至古希腊数学家阿波罗尼斯（约公元前262—190年），他在著作《圆锥曲线论》中将圆锥被平面截得的三种曲线统一命名，双曲线由此得名。17世纪笛卡尔建立坐标系后，双曲线从几何描述转变为代数方程，其方程形式才得以精确表达。

双曲线在物理和工程中有不可替代的应用价值：双曲线导航系统（LORAN）利用两个发射站信号的时间差确定船舶位置，正是基于双曲线的焦点距离差定义；卫星轨道中，速度超过逃逸速度的天体沿双曲线轨迹运动；双曲型冷却塔的外形截面正是双曲线的旋转体。

## 核心原理

### 标准方程与参数关系

以焦点在 $x$ 轴上的双曲线为例，其标准方程为：

$$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1 \quad (a > 0,\ b > 0)$$

其中三个基本参数满足关系：

$$c^2 = a^2 + b^2$$

注意与椭圆的关键区别：椭圆是 $c^2 = a^2 - b^2$，而双曲线是 $c^2 = a^2 + b^2$，$b$ 不再受 $a$ 约束，可以大于、等于或小于 $a$。双曲线的实轴长为 $2a$（顶点 $(\pm a, 0)$），虚轴长为 $2b$（虚顶点 $(0, \pm b)$），虚轴上的点并不在双曲线上，但决定了渐近线的斜率。

若焦点在 $y$ 轴上，方程变为 $\dfrac{y^2}{a^2} - \dfrac{x^2}{b^2} = 1$，两支开口朝上和朝下。判断焦点位置的口诀是：**正项分母对应的轴方向即为焦点所在轴**。

### 渐近线

双曲线 $\dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} = 1$ 的两条渐近线方程为：

$$y = \pm \frac{b}{a} x$$

渐近线不是双曲线本身的一部分，而是双曲线无限延伸时趋近但永不到达的直线。焦点到渐近线的距离恰好等于 $b$：设渐近线 $bx - ay = 0$，焦点 $(c, 0)$ 到该直线的距离为 $\dfrac{|bc|}{\sqrt{a^2+b^2}} = \dfrac{bc}{c} = b$，这一结论在解题中频繁使用。

等轴双曲线是 $a = b$ 的特殊情形，此时渐近线斜率为 $\pm 1$，渐近线互相垂直，方程可写成 $x^2 - y^2 = a^2$。

### 离心率

双曲线的离心率定义为 $e = \dfrac{c}{a}$，由于 $c > a$，所以双曲线的离心率 $e > 1$。这是与椭圆（$0 < e < 1$）和抛物线（$e = 1$）的本质区别。

离心率决定了双曲线两支的"张开程度"：$e$ 越接近 $1$，两支越窄越"陡峭"；$e$ 越大，两支越宽越"平坦"，渐近线与 $x$ 轴的夹角越大。具体来说，渐近线斜率 $k = \pm\dfrac{b}{a} = \pm\sqrt{e^2 - 1}$，因此 $e = 2$ 时渐近线斜率为 $\pm\sqrt{3}$，即渐近线与实轴成 $60°$ 角。

## 实际应用

**求双曲线方程的典型步骤**：已知离心率 $e = \sqrt{2}$，焦点在 $x$ 轴，则 $c = \sqrt{2}\,a$，代入 $b^2 = c^2 - a^2 = 2a^2 - a^2 = a^2$，即 $a = b$，这正是等轴双曲线。若再知顶点坐标 $(\sqrt{3}, 0)$，则 $a = \sqrt{3}$，方程为 $\dfrac{x^2}{3} - \dfrac{y^2}{3} = 1$。

**焦点三角形问题**：设双曲线上一点 $P$（在右支），$|PF_1| - |PF_2| = 2a$，若已知 $|PF_1| = 5$，$a = 2$，则 $|PF_2| = 3$；若已知 $\angle F_1PF_2 = 60°$，可由余弦定理 $|F_1F_2|^2 = |PF_1|^2 + |PF_2|^2 - 2|PF_1||PF_2|\cos 60°$ 联立 $|PF_1| - |PF_2| = 4$ 求出具体长度，进而求出 $b^2 = c^2 - a^2$。

**直线与双曲线的交点**：将直线方程代入双曲线方程得到关于 $x$ 的二次方程，需注意双曲线有两支，判别式 $\Delta > 0$ 只是有实数解的必要条件，还需验证交点的 $x$ 坐标是否满足各支的范围限制（右支要求 $x \geq a$，左支要求 $x \leq -a$）。

## 常见误区

**误区一：将 $b^2 = a^2 - c^2$ 用于双曲线**。很多学生在椭圆和双曲线之间混用参数关系。椭圆因为 $a > c$ 才有 $b^2 = a^2 - c^2 > 0$；双曲线中 $c > a$，必须使用 $b^2 = c^2 - a^2$。两者符号相反，混淆后计算结果为负数，应立即警觉。

**误区二：认为双曲线的 $a > b$ 恒成立**。椭圆长轴对应的参数一定大于短轴，但双曲线没有这一约束。双曲线 $\dfrac{x^2}{4} - \dfrac{y^2}{9} = 1$ 中 $a = 2 < b = 3$ 完全合法，此时 $c = \sqrt{13}$，渐近线斜率 $\pm\dfrac{3}{2}$，两支开口比等轴双曲线更宽。

**误区三：将渐近线方程 $y = \pm\dfrac{b}{a}x$ 用于焦点在 $y$ 轴的情形**。当方程为 $\dfrac{y^2}{a^2} - \dfrac{x^2}{b^2} = 1$ 时，渐近线应为 $y = \pm\dfrac{a}{b}x$，分子分母与 $x$ 轴焦点情形恰好互换，不能套用同一公式。

## 知识关联

学习双曲线前，必须掌握**圆的方程**中建立坐标系的方法与距离公式推导思路——双曲线标准方程的推导过程（将 $||PF_1| - |PF_2|| = 2a$ 代入坐标表达式化简）与圆的方程推导结构相似，都依赖两点间距离公式的代数化简，但双曲线需要对绝对值分情况处理，代数化简步骤更复杂。

掌握双曲线后，可进入**圆锥曲线统一定义**的学习。统一定义将椭圆（$e < 1$）、抛物线（$e = 1$）和双曲线（$e > 1$）用焦点-准线比值 $e = \dfrac{r}{d}$ 统一描述，双曲线中 $b^2 = a^2(e^2-1)$ 以及准线方程 $x = \pm\dfrac{a}{e} = \pm\dfrac{a^2}{c}$ 都将在统一框架中得到统一推导和几何解释。
