# 直线与圆锥曲线

## 概述

直线与圆锥曲线的交点问题是解析几何的核心计算主题，其本质是将直线方程与圆锥曲线方程联立，通过消元得到一元二次方程，再借助韦达定理（根与系数关系）来研究交点的坐标性质、弦的中点、弦长等几何量。这一方法将几何问题完全代数化，使得"判断交点个数""求弦中点轨迹""计算弦长"等问题都可以统一在同一套代数框架下处理。

该方法的核心思想在17世纪由笛卡尔和费马分别建立解析几何体系时奠定基础。对于抛物线 $y^2 = 2px$、椭圆 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$、双曲线 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ 这三类方程，联立直线后得到的二次方程的判别式 $\Delta$ 直接决定了直线与曲线的位置关系（相交、相切、相离），而韦达定理给出 $x_1 + x_2$ 和 $x_1 x_2$ 的精确表达式，无需求解方程本身便能获得大量几何信息。

在高考和竞赛中，直线与圆锥曲线的题型涵盖弦中点轨迹、定点共线、焦点弦性质等，难度集中在7分以上。掌握这一主题意味着能处理绝大多数解析几何的压轴计算题。

## 核心原理

### 联立方程与判别式

设直线方程为 $y = kx + m$（斜率存在时），将其代入椭圆方程 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$，展开化简后得到：

$$
(b^2 + a^2k^2)x^2 + 2a^2kmx + a^2m^2 - a^2b^2 = 0
$$

此方程的判别式为：
$$
\Delta = 4a^4k^2m^2 - 4(b^2 + a^2k^2)(a^2m^2 - a^2b^2)
$$

化简得 $\Delta = 4a^2b^2(b^2 + a^2k^2 - m^2)$。当 $\Delta > 0$ 时直线与椭圆有两个交点，$\Delta = 0$ 时相切，$\Delta < 0$ 时相离。**注意**：若直线斜率不存在（即 $x = c$ 形式），需单独代入计算，不能用上述判别式。

对于抛物线 $y^2 = 2px$，若将直线写成 $x = ty + s$（以 $y$ 为主变量联立），往往能避免出现斜率不存在的分类讨论，是处理抛物线问题的常用技巧。

### 韦达定理的应用

在联立后得到的二次方程 $Ax^2 + Bx + C = 0$ 中，设两根为 $x_1, x_2$，则：
$$
x_1 + x_2 = -\frac{B}{A}, \quad x_1 x_2 = \frac{C}{A}
$$

以椭圆联立结果为例，$x_1 + x_2 = \frac{-2a^2km}{b^2 + a^2k^2}$，$x_1 x_2 = \frac{a^2m^2 - a^2b^2}{b^2 + a^2k^2}$。

**弦中点坐标**：$x_0 = \frac{x_1 + x_2}{2} = \frac{-a^2km}{b^2 + a^2k^2}$，对应 $y_0 = kx_0 + m$，由此可建立弦中点满足的轨迹方程。

**弦长公式**：$|AB| = \sqrt{1+k^2} \cdot |x_1 - x_2| = \sqrt{1+k^2} \cdot \sqrt{(x_1+x_2)^2 - 4x_1x_2}$，全部用韦达定理的结果代入即可，不需要求出 $x_1, x_2$ 的具体值。

### 中点弦与斜率关系（中点条件法）

若已知弦中点为 $(x_0, y_0)$，则两端点 $(x_1, y_1)$、$(x_2, y_2)$ 均在曲线上，以椭圆为例：
$$
\frac{x_1^2}{a^2} + \frac{y_1^2}{b^2} = 1, \quad \frac{x_2^2}{a^2} + \frac{y_2^2}{b^2} = 1
$$

两式相减得：
$$
\frac{x_1^2 - x_2^2}{a^2} + \frac{y_1^2 - y_2^2}{b^2} = 0
$$

因式分解后化简，弦的斜率满足：
$$
k = \frac{y_1 - y_2}{x_1 - x_2} = -\frac{b^2 x_0}{a^2 y_0}
$$

这一结论称为**椭圆的中点弦斜率公式**，可直接记忆使用，双曲线类似地有 $k = \frac{b^2 x_0}{a^2 y_0}$（注意符号相反）。

## 实际应用

**例1：弦中点轨迹问题**  
已知椭圆 $\frac{x^2}{4} + \frac{y^2}{3} = 1$，过点 $P(1, 0)$ 的弦 $AB$ 的中点轨迹。  
设中点为 $(x_0, y_0)$，由中点弦斜率公式得 $k = -\frac{3x_0}{4y_0}$，又直线过 $P(1,0)$ 故 $k = \frac{y_0 - 0}{x_0 - 1}$。联立消去 $k$：$-\frac{3x_0}{4y_0} = \frac{y_0}{x_0 - 1}$，整理得 $3x_0(x_0-1) + 4y_0^2 = 0$，即 $3x_0^2 - 3x_0 + 4y_0^2 = 0$，此为弦中点的轨迹方程（一个椭圆弧段）。

**例2：焦点弦的调和性质**  
对于抛物线 $y^2 = 4x$（即 $p = 2$），过焦点 $F(1,0)$ 的弦两端点参数满足 $x_1 x_2 = 1$（由韦达定理直接得出），进一步可证明 $\frac{1}{|AF|} + \frac{1}{|BF|} = 1$（焦点弦的调和性质），其中弦长最小值在直线 $x=1$ 时取得，最小弦长为 $4p = 4 \times 2 = 4$（即通径长度）。

## 常见误区

**误区1：忽略斜率不存在的情形**  
在设直线为 $y = kx + m$ 时，垂直于 $x$ 轴的直线 $x = c$ 被排除在外。学生常在结论中忘记验证这一特殊情形。例如讨论直线与抛物线的位置关系时，$x = 0$（即 $y$ 轴）与抛物线 $y^2 = 2x$ 只有一个公共点 $(0,0)$，但用斜率存在的联立方法无法发现，需要单独验证。

**误区2：$\Delta > 0$ 不等于联立有两个不同实交点**  
联立方程得到二次方程 $Ax^2 + Bx + C = 0$，必须保证 $A \neq 0$（即曲线不退化且直线非特殊方向）。若 $A = 0$，方程变为一次方程，只有一个 $x$ 值，不能套用判别式和韦达定理。此外，对于双曲线，即使 $\Delta > 0$，也要确认两个 $x$ 根所对应的点确实在双曲线的同一支或不同支上。

**误区3：韦达定理结果直接等于坐标**  
联立直线 $y = kx + m$ 与圆锥曲线后，韦达定理给出的是关于 $x$ 的方程的两根 $x_1, x_2$，对应的 $y$ 坐标需回代 $y_i = kx_i + m$ 才能得到。学生有时误以为韦达定理直接给出点的完整坐标，跳过回代步骤，导致后续弦中点的 $y_0$ 计算出错。

## 知识关联

本主题直接依赖**一元二次方程**的判别式 $\Delta = b^2 - 4ac$ 和韦达定理 $x_1 + x_2 = -b/a, x_1 x_2 = c/a$；若不熟练运用这两个工具，联立后的代数运算将无法推进。同时需要熟悉**圆锥曲线统一定义**中三类曲线的标准方程形式，因为椭圆、抛物线、双曲线联立后得到的二次方程系数结构各不相同，抛物线联立只产生一次项和二次项（无常数项干扰），而双曲线的 $x^2$ 系数可能为负，必须分别掌握。
