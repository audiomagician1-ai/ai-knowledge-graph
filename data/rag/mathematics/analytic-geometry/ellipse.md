# 椭圆

## 概述

椭圆是平面内到两个定点（焦点）距离之和为常数的点的集合。设两焦点为 $F_1$、$F_2$，椭圆上任意一点 $P$ 满足 $|PF_1| + |PF_2| = 2a$，其中常数 $2a$ 必须严格大于两焦点间距 $2c$，即 $2a > 2c$，否则轨迹退化为线段或空集。

椭圆的研究可追溯至古希腊数学家阿波罗尼斯（约公元前262—前190年），他在《圆锥曲线论》中系统研究了用平面截圆锥所得的曲线，"椭圆"一词（elleipsis）源自希腊语，意为"不足"，描述截面角度小于母线角时所得的曲线形状。开普勒在1609年发现行星轨道是以太阳为焦点的椭圆，使椭圆从纯粹几何问题跃升为描述天体运动的核心工具。

椭圆在工程与物理中具有独特意义：椭圆形拱顶具有声学聚焦性质（一个焦点发出的声波会汇聚到另一焦点），椭圆齿轮被用于需要不等速传动的机械装置。掌握椭圆方程是理解行星轨道、光学反射镜设计及参数化曲线建模的必要前提。

## 核心原理

### 标准方程与参数关系

当椭圆的两个焦点位于 $x$ 轴上时，以中心为原点建立坐标系，标准方程为：

$$\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 \quad (a > b > 0)$$

其中三个参数满足基本关系 $a^2 = b^2 + c^2$，$a$ 为半长轴，$b$ 为半短轴，$c$ 为半焦距。顶点坐标为 $(\pm a, 0)$ 和 $(0, \pm b)$，焦点坐标为 $(\pm c, 0)$。

若焦点位于 $y$ 轴上，则方程变为 $\dfrac{x^2}{b^2} + \dfrac{y^2}{a^2} = 1$，此时分母较大的项对应长轴方向——这是判断椭圆焦点位置的直接依据。注意：椭圆标准方程要求两分母均为正数且不相等，若 $a = b$ 则方程退化为圆 $x^2 + y^2 = a^2$。

### 焦点与焦半径

椭圆上点 $P(x_0, y_0)$ 到两焦点 $F_1(-c, 0)$、$F_2(c, 0)$ 的距离称为**焦半径**，计算公式为：

$$|PF_1| = a + ex_0, \quad |PF_2| = a - ex_0$$

其中 $e = c/a$ 为离心率。这两个公式可由椭圆定义直接推导，利用 $|PF_1| + |PF_2| = 2a$ 以及距离公式联立得出。焦半径公式在求椭圆上特定点到焦点距离时远比直接用距离公式计算高效。

当 $P$ 在右顶点 $（a, 0）$ 时，$|PF_1| = a + c$，$|PF_2| = a - c$，两者均为正数，验证了 $a > c$ 的必要性。椭圆的**通径**（过焦点且垂直于长轴的弦）长度为 $\dfrac{2b^2}{a}$，是描述焦点附近曲线"宽度"的特征量。

### 离心率与形状

离心率 $e = \dfrac{c}{a}$ 严格满足 $0 < e < 1$，这是椭圆区别于其他圆锥曲线的数值特征。$e$ 越接近 $0$，椭圆越趋近圆形（当 $e=0$ 时即为圆）；$e$ 越接近 $1$，椭圆越"扁平"，两焦点越靠近端点。

地球绕太阳公转轨道的离心率约为 $0.0167$，接近正圆；而哈雷彗星轨道的离心率约为 $0.967$，极度扁长。由 $e = c/a$ 及 $b^2 = a^2 - c^2$，可得 $b^2/a^2 = 1 - e^2$，因此离心率完全决定了椭圆的形状（即长短轴之比），而 $a$ 的值决定椭圆的大小。

## 实际应用

**行星轨道计算**：已知地球近日点距离约 $1.471 \times 10^8$ km，远日点距离约 $1.521 \times 10^8$ km，由焦半径公式 $a - c = 1.471 \times 10^8$，$a + c = 1.521 \times 10^8$，解得 $a = 1.496 \times 10^8$ km，$c = 0.025 \times 10^8$ km，从而 $e \approx 0.0167$。

**椭圆反射镜设计**：在医学碎石机中，冲击波从一个焦点发出，经椭圆形反射腔汇聚到另一个焦点（即结石位置）。设计时需要精确确定 $a$、$b$ 使得焦点间距 $2c$ 与患者体内目标位置相符。

**考题典型情形**：给定椭圆 $\dfrac{x^2}{25} + \dfrac{y^2}{9} = 1$，则 $a=5$，$b=3$，$c=4$，$e=0.8$，通径长 $= \dfrac{2 \times 9}{5} = 3.6$。设点 $P$ 在椭圆上且横坐标 $x_0 = 3$，则 $|PF_2| = 5 - 0.8 \times 3 = 2.6$，$|PF_1| = 5 + 0.8 \times 3 = 7.4$。

## 常见误区

**误区一：混淆标准方程中 $a$、$b$ 的位置**。椭圆方程 $\dfrac{x^2}{m} + \dfrac{y^2}{n} = 1$ 中，较大的分母对应长轴，而非一律认为 $x$ 项分母是 $a^2$。若 $m < n$，则 $a^2 = n$，焦点在 $y$ 轴上，此时 $c^2 = n - m$ 而非 $m - n$。直接套用 $c^2 = a^2 - b^2$ 而不先判断长轴方向是高频错误来源。

**误区二：将焦半径公式中 $x_0$ 的符号搞错**。公式 $|PF_1| = a + ex_0$（其中 $F_1$ 为左焦点）在 $x_0 < 0$ 时仍然成立，此时 $|PF_1| = a + e \cdot(\text{负数}) < a$，物理意义是点 $P$ 在左侧时离左焦点更近。学生常错误地认为距离公式结果应该"加绝对值"，从而丢失符号信息。

**误区三：认为 $e$ 越大椭圆"越大"**。$e$ 描述的是形状（扁平程度），与椭圆的实际尺寸（由 $a$ 决定）无关。两个椭圆可以有相同的 $e=0.8$，但一个 $a=5$、另一个 $a=100$，面积相差400倍。面积公式 $S = \pi ab = \pi a^2\sqrt{1-e^2}$ 同时包含 $a$ 和 $e$ 两个量，缺一不可。

## 知识关联

**与圆的方程的联系**：圆是椭圆的特殊情形（$a = b$，$e = 0$），圆的方程 $(x-h)^2 + (y-k)^2 = r^2$ 中焦点退化为圆心。椭圆中心平移的方程为 $\dfrac{(x-h)^2}{a^2} + \dfrac{(y-k)^2}{b^2} = 1$，处理方法与圆的平移完全类比，但需额外追踪长轴方向。

**通向圆锥曲线统一定义**：椭圆的离心率 $0 < e < 1$ 是将其纳入圆锥曲线统一定义（焦点-准线定义）的关键参数。椭圆上点到焦点距离与到对应准线距离之比恒等于 $e$，准线方程为 $x = \pm\dfrac{a}{e} = \pm\dfrac{a^2}{c}$。当 $e=1$ 时统一定义给出抛物线，$e>1$ 时给出双曲线，三种曲线由同一定义框架统一。

**通向三角参数化**：椭圆 $\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1$ 可参数化为 $x = a\cos t$，$y = b\sin t$，$t \in [0, 2\pi)$。这一参数化将椭圆上点的坐标计算转化为三角函数运算，是处理椭圆面积（$S = \pi ab$）和椭圆弧长积分的基础形式。
