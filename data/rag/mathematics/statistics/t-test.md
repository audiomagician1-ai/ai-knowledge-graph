# t检验

## 概述

t检验（t-test）是一种基于t分布的参数假设检验方法，专门用于在总体方差未知的条件下，对均值进行统计推断。其核心统计量服从自由度为 $n-1$（单样本）或 $n_1+n_2-2$（等方差双样本）的t分布，而非正态分布，这一特性使其在小样本场景下优于z检验。

t检验由英国统计学家威廉·戈塞特（William Sealy Gosset）于1908年提出。戈塞特当时在爱尔兰吉尼斯啤酒厂工作，由于公司保密政策，他以笔名"Student"发表论文，因此t检验也被称为"Student's t-test"。这一背景说明，t检验最初的动机是解决工业品质控制中小样本均值估计的问题。

在实际统计分析中，总体方差 $\sigma^2$ 往往无法得知，必须用样本方差 $S^2$ 代替。但直接用 $S^2$ 替代 $\sigma^2$ 后，标准化统计量不再服从标准正态分布，而是服从t分布。t分布的尾部比正态分布更厚，自由度越小尾部越厚，这直接影响临界值的选取，使小样本推断更为保守和严谨。

## 核心原理

### 单样本t检验

单样本t检验用于判断样本均值是否与某一已知的总体均值 $\mu_0$ 存在显著差异。其检验统计量为：

$$t = \frac{\bar{X} - \mu_0}{S / \sqrt{n}}$$

其中：
- $\bar{X}$ 为样本均值
- $\mu_0$ 为假设的总体均值（零假设值）
- $S$ 为样本标准差（除以 $n-1$，即无偏估计）
- $n$ 为样本量

该统计量在零假设 $H_0: \mu = \mu_0$ 成立时，服从自由度为 $n-1$ 的t分布。若 $|t| > t_{\alpha/2}(n-1)$（双侧检验），则在显著性水平 $\alpha$ 下拒绝 $H_0$。

### 双样本独立t检验（等方差假设）

当比较两个独立正态总体均值 $\mu_1$ 与 $\mu_2$ 时，若可假定两总体方差相等（$\sigma_1^2 = \sigma_2^2 = \sigma^2$），使用合并方差（pooled variance）估计 $\sigma^2$：

$$S_p^2 = \frac{(n_1-1)S_1^2 + (n_2-1)S_2^2}{n_1 + n_2 - 2}$$

检验统计量为：

$$t = \frac{\bar{X}_1 - \bar{X}_2}{S_p\sqrt{\frac{1}{n_1}+\frac{1}{n_2}}}$$

此统计量在 $H_0: \mu_1 = \mu_2$ 下服从自由度为 $n_1 + n_2 - 2$ 的t分布。

### Welch's t检验（不等方差）

当两总体方差不相等（$\sigma_1^2 \neq \sigma_2^2$）时，需使用Welch校正。统计量形式不变，但自由度通过Welch-Satterthwaite公式近似计算：

$$\nu \approx \frac{\left(\frac{S_1^2}{n_1}+\frac{S_2^2}{n_2}\right)^2}{\frac{(S_1^2/n_1)^2}{n_1-1}+\frac{(S_2^2/n_2)^2}{n_2-1}}$$

由于自由度通常不为整数，Welch's t检验的自由度一般小于等方差版本的 $n_1+n_2-2$，临界值更大，检验更为保守。

### 配对样本t检验

配对t检验用于同一受试对象在两种条件下测量值的比较（例如治疗前后的血压）。将每对差值 $d_i = X_{1i} - X_{2i}$ 视为一组新数据，对差值均值是否为零作单样本t检验：

$$t = \frac{\bar{d}}{S_d / \sqrt{n}}$$

其中 $\bar{d}$ 为差值均值，$S_d$ 为差值的样本标准差，自由度为 $n-1$（$n$ 为配对数）。配对设计通过消除个体间差异，可显著提高检验功效。

## 实际应用

**药物临床试验**：某新型降压药试验中，研究者对12名受试者测量服药前后血压差值，得 $\bar{d} = 8.3$ mmHg，$S_d = 4.1$ mmHg。配对t统计量 $t = 8.3/(4.1/\sqrt{12}) \approx 7.01$，查t分布表 $t_{0.025}(11) = 2.201$，由于 $7.01 \gg 2.201$，拒绝"服药无效"的零假设，结论具有高度统计显著性（$p < 0.001$）。

**工业质量控制**：生产线需检验某零件平均直径是否为规格值 $\mu_0 = 50$ mm，抽取16个样品，得 $\bar{X}=50.3$ mm，$S=0.4$ mm，计算 $t = (50.3-50)/(0.4/\sqrt{16}) = 3.0$，与 $t_{0.025}(15)=2.131$ 比较，拒绝 $H_0$，判定该批次直径偏大，触发质量预警。

**教育研究中的双样本应用**：比较A、B两种教学方法的考试成绩，两组样本量分别为25和30，先使用F检验（方差齐性检验）判断是否满足等方差假设，再决定使用合并方差t检验还是Welch's t检验。

## 常见误区

**误区一：忽略前提条件，将t检验应用于非正态大偏态数据**  
t检验要求数据来自正态总体（或样本量足够大以满足中心极限定理）。对于严重偏态的小样本数据（如 $n < 30$ 且存在明显非对称性），直接使用t检验会导致I类错误率失控。此时应考虑非参数替代方法，如Wilcoxon符号秩检验（对应单样本/配对t检验）或Mann-Whitney U检验（对应独立双样本t检验）。

**误区二：混淆配对t检验与独立双样本t检验**  
将配对数据当作独立样本处理，会损失统计功效，因为独立样本t检验的误差项包含了本可通过配对消除的个体间变异。反之，将独立数据错误地用配对t检验分析，则会夸大统计显著性。判断标准明确：同一对象重复测量，或刻意匹配的两组对象，才使用配对设计。

**误区三：等方差假设未经检验直接使用合并方差公式**  
在双样本场景中，许多初学者直接套用合并方差t检验公式，而忽略了先用Levene检验或Bartlett检验验证方差齐性。若两组方差之比超过4:1（即标准差之比超过2:1），强行使用合并方差会导致检验结果失真，应改用Welch's t检验。

## 知识关联

**前置概念**：抽样分布为t检验提供了理论基础——当 $X_i \sim N(\mu, \sigma^2)$ 时，$\frac{\bar{X}-\mu}{S/\sqrt{n}}$ 服从 $t(n-1)$ 分布，这一结论是推导t统计量的根本依据。假设检验的框架（零假设与备择假设的设立、I类II类错误的权衡、显著性水平 $\alpha$ 的选取）直接规定了t检验的决策规则，没有假设检验体系，t统计量的计算结果将无法做出"拒绝/不拒绝"的判断。

**后续概念**：方差分析（ANOVA）是t检验在多组均值比较场景下的推广。当比较3组及以上均值时，若反复使用t检验进行两两比较，会因多重比较问题导致总体I类错误率远超设定的 $\alpha$ 值（例如对5组数据做10次t检验，即使每次 $\alpha=0.05$，总体犯错概率约为 $1-0.95^{10} \approx 40\%$）。方差分析通过F统计量（组间方差/组内方差）在单次检验中评估所有组均值，从根本上解决了这一问题，其中组内方差的估计思想与t检验的合并方差 $S_p^2$ 一脉相承。
