# 贝叶斯统计

## 概述

贝叶斯统计是一套以贝叶斯定理为基础的推断框架，其核心思想是将参数 θ 视为随机变量，通过观测数据持续更新对参数的信念。与频率派统计将参数视为固定未知常数不同，贝叶斯统计允许研究者将领域知识以**先验分布 π(θ)** 的形式注入模型，再结合似然函数 L(x|θ) 得到**后验分布 π(θ|x)**。这一范式由托马斯·贝叶斯（Thomas Bayes）在1763年的遗作中首次提出，经拉普拉斯（Laplace）在19世纪系统化发展，最终在20世纪后半叶随计算机算力的提升而广泛普及。

贝叶斯统计的重要性体现在它能够自然地处理小样本问题、将先验知识量化，以及提供参数的完整概率分布而非单一点估计。在频率派方法中，一个参数的95%置信区间意味着"若重复实验无穷次，95%的区间会覆盖真值"；而贝叶斯统计的95%**可信区间（Credible Interval）**则直接表达"参数以95%的概率落在该区间内"，语义更直观。

## 核心原理

### 贝叶斯更新公式

贝叶斯统计的运算核心是后验分布的计算公式：

$$\pi(\theta \mid x) = \frac{L(x \mid \theta)\, \pi(\theta)}{\int L(x \mid \theta)\, \pi(\theta)\, d\theta}$$

其中：
- **π(θ)** 是先验分布，反映观测数据之前对 θ 的信念；
- **L(x|θ)** 是似然函数，即在参数为 θ 时观测到数据 x 的概率；
- **∫ L(x|θ)π(θ)dθ** 是边际似然（证据），作为归一化常数；
- **π(θ|x)** 是后验分布，融合了先验与数据信息。

由于分母仅为归一化常数，实践中常写成正比关系：**π(θ|x) ∝ L(x|θ)·π(θ)**，这一简化形式是推导共轭先验的出发点。

### 先验分布的选择

先验分布的选择直接影响后验结果，分为三类：

**信息先验（Informative Prior）**：基于已有研究或专家知识构建，例如已知某药物副作用概率约为10%，可设先验为 Beta(2, 18)，其均值恰好为 2/(2+18) = 0.1。

**无信息先验（Non-informative Prior）**：当先验知识匮乏时使用，最典型的是**杰弗里斯先验（Jeffreys Prior）**，定义为 π(θ) ∝ √(I(θ))，其中 I(θ) 是费雪信息量。对于伯努利参数 p，杰弗里斯先验为 Beta(1/2, 1/2)，而非直觉上的均匀分布 Beta(1,1)，原因在于均匀分布在参数变换下不具备不变性。

**共轭先验（Conjugate Prior）**：若先验与后验属于同一分布族，则称先验为似然函数的共轭先验。共轭先验的最大优势是使后验具有解析解，避免数值积分。

### 共轭先验的经典配对

共轭先验是贝叶斯统计中最具计算价值的工具。以下三对是最重要的共轭关系：

**1. 二项似然 + Beta先验 → Beta后验**  
若 x ~ Binomial(n, p)，先验 p ~ Beta(α, β)，则观测到 k 次成功后：  
$$p \mid x \sim \text{Beta}(\alpha + k,\; \beta + n - k)$$  
参数 α 和 β 可解释为"虚拟观测"中的成功次数和失败次数，后验均值为 (α+k)/(α+β+n)。

**2. 泊松似然 + Gamma先验 → Gamma后验**  
若 x₁,...,xₙ ~ Poisson(λ)，先验 λ ~ Gamma(α, β)，则：  
$$\lambda \mid x \sim \text{Gamma}\!\left(\alpha + \sum x_i,\; \beta + n\right)$$

**3. 正态似然（已知方差σ²）+ 正态先验 → 正态后验**  
若 xᵢ ~ N(μ, σ²)（σ² 已知），先验 μ ~ N(μ₀, τ²)，则后验均值是数据均值与先验均值的**精度加权平均**：  
$$\mu_{\text{post}} = \frac{\frac{\mu_0}{\tau^2} + \frac{n\bar{x}}{\sigma^2}}{\frac{1}{\tau^2} + \frac{n}{\sigma^2}}$$  
当样本量 n → ∞ 时，后验均值趋近于样本均值 x̄，体现贝叶斯统计"数据主导"的渐进性质。

### 后验推断的点估计与区间估计

后验分布本身包含完整的参数信息，可从中提取：
- **后验均值（Posterior Mean）**：最小化平方损失时的最优估计；
- **后验众数（MAP，最大后验估计）**：最小化0-1损失，等价于频率派中的正则化极大似然估计；
- **后验中位数**：最小化绝对值损失；
- **最高后验密度区间（HPD Interval）**：区间内密度均高于区间外，是最短的可信区间。

## 实际应用

**医学诊断中的贝叶斯更新**：假设某疾病患病率（先验）为 1%，检测灵敏度为 99%，特异度为 95%。检测阳性后，后验患病概率为 (0.99×0.01)/(0.99×0.01 + 0.05×0.99) ≈ 16.7%，远低于大众直觉的"99%阳性准确率"。这一计算直接说明了低基率疾病筛查中假阳性的普遍性。

**A/B测试中的贝叶斯方法**：互联网公司对两个版本（版本A转化率 pA、版本B转化率 pB）各设 Beta(1,1) 无信息先验，随着流量积累不断更新后验，可直接计算 P(pB > pA | 数据) 并随时终止实验，而无需频率派方法中固定样本量的限制。

**自然语言处理中的朴素贝叶斯分类器**：垃圾邮件过滤器利用贝叶斯统计，以词汇出现频率作为似然，以历史垃圾邮件比例作为先验，计算邮件属于垃圾邮件的后验概率。实践中常用拉普拉斯平滑（加 1 的伪计数）等价于设置均匀先验，防止零似然问题。

## 常见误区

**误区一：先验选择是主观的，因此贝叶斯统计不客观**  
这一批评混淆了"主观"与"任意"。贝叶斯框架要求先验必须在观测数据之前确定，并且不同研究者的先验在充分数据下会收敛到相同的后验——正式的说法是**贝尔斯坦-冯·米泽斯定理（Bernstein–von Mises Theorem）**：在正则条件下，后验分布以真实参数为中心的正态分布为极限，先验的影响随 n 增大以 1/n 速率消失。

**误区二：MAP估计等同于贝叶斯推断**  
MAP（最大后验估计）只取后验分布的众数，丢弃了参数不确定性的全部信息。例如，当后验为双峰分布时，MAP仅取其中一个峰，而完整的后验分析会揭示参数存在两个可信区域。在贝叶斯统计中，完整后验分布才是推断的对象，MAP不过是一种从后验中提取点估计的策略，与频率派的MLE在形式上类似（加了先验的惩罚项），不代表贝叶斯推断的全貌。

**误区三：共轭先验在计算上总是首选**  
共轭先验只在少数标准似然函数下存在。对于复杂层次模型或非标准似然，共轭先验往往不存在或导致过强的分布假设。现代贝叶斯统计更多依赖**马尔可夫链蒙特卡洛（MCMC）**方法（如Metropolis-Hastings算法、No-U-Turn Sampler）从任意后验中采样，Stan和PyMC3等工具已将这类方法标准化，使共轭先验的计算优势在许多实际场景中不再关键。
