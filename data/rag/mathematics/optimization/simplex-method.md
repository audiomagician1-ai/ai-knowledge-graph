# 单纯形法

## 概述

单纯形法（Simplex Method）是求解线性规划问题的系统性迭代算法，由乔治·丹齐格（George Dantzig）于1947年在美国空军后勤规划研究中提出。该算法通过在可行域的顶点（极点）之间有序移动，逐步改善目标函数值，直至找到最优解或判定问题无界。

单纯形法的命名来源于"单纯形"这一几何对象——n维空间中由n+1个顶点构成的最简凸多面体（如三角形是二维单纯形，四面体是三维单纯形）。但实际算法操作的是线性规划可行域的顶点集合，而非单纯形本身。理解这一命名背景有助于把握算法的几何本质。

单纯形法在实践中的效率惊人：尽管理论上最坏情况下需要遍历指数级数量的顶点（Klee-Minty问题，1972年证明），但对于工业界的典型问题，迭代次数通常仅在约束数量的1到3倍之间。这种实用高效性使其在运筹学中被广泛采用达七十余年。

## 核心原理

### 标准型与基本可行解

单纯形法要求线性规划问题化为标准型：

$$\min \mathbf{c}^T\mathbf{x}, \quad \text{s.t. } A\mathbf{x} = \mathbf{b},\ \mathbf{x} \geq \mathbf{0}$$

其中 $A$ 为 $m \times n$ 矩阵（$m < n$），$\mathbf{b} \geq \mathbf{0}$。不等式约束通过引入**松弛变量**（slack variables）转化为等式。例如约束 $2x_1 + 3x_2 \leq 12$ 加入松弛变量 $x_3$ 后变为 $2x_1 + 3x_2 + x_3 = 12$，$x_3 \geq 0$。

从 $A$ 的 $n$ 列中选取 $m$ 个线性无关列构成**基矩阵** $B$（$m \times m$ 可逆），其余列构成非基矩阵 $N$。令非基变量 $\mathbf{x}_N = \mathbf{0}$，则基变量 $\mathbf{x}_B = B^{-1}\mathbf{b}$。若 $B^{-1}\mathbf{b} \geq \mathbf{0}$，则称此解为**基本可行解**，对应可行域的一个顶点。

### 最优性检验与旋转操作

对于当前基本可行解，定义**检验数**（reduced cost）为：

$$\bar{c}_j = c_j - \mathbf{c}_B^T B^{-1} \mathbf{a}_j$$

其中 $\mathbf{a}_j$ 是变量 $x_j$ 对应的 $A$ 的列向量，$\mathbf{c}_B$ 是基变量的目标系数向量。**最优性条件**：若所有非基变量的检验数 $\bar{c}_j \geq 0$（最小化问题），则当前解为最优解。

若存在某非基变量 $x_s$ 使得 $\bar{c}_s < 0$，则 $x_s$ 有资格**进基**（entering variable）。选取进基变量后，通过**最小比值检验**（minimum ratio test）确定离基变量 $x_r$：

$$\theta^* = \min_{i: \bar{a}_{is} > 0} \frac{\bar{b}_i}{\bar{a}_{is}}$$

其中 $\bar{b}_i = (B^{-1}\mathbf{b})_i$，$\bar{a}_{is} = (B^{-1}\mathbf{a}_s)_i$。该步骤防止解离开可行域。以 $(r, s)$ 元素为**主元**（pivot）进行行变换，完成一次**旋转**操作，基变换后目标函数值改善量为 $|\bar{c}_s| \cdot \theta^*$。

### 几何解释

从几何视角看，每次旋转操作对应在可行域多面体的**相邻顶点**之间移动——两个顶点共享 $m-1$ 个相同的基变量（即共享一条"棱"）。以二维问题为例：可行域为凸多边形，单纯形法从某顶点出发，沿使目标函数值严格下降的边移动到相邻顶点，直至到达最优顶点（该顶点处所有相邻边的方向均不能改善目标函数）。在三维问题中，顶点沿多面体的棱移动，每次移动恰好改变一个基变量。

## 实际应用

**生产计划问题**：设工厂生产两种产品，约束为 $x_1 + 2x_2 \leq 8$（机时），$3x_1 + x_2 \leq 9$（原料），目标为最大化利润 $5x_1 + 4x_2$。引入松弛变量后初始基为 $\{x_3, x_4\}$，检验数 $\bar{c}_1 = -5 < \bar{c}_2 = -4$，选 $x_1$ 进基，最小比值检验得 $\theta^* = \min(8/1, 9/3) = 3$，$x_4$ 离基。经两次旋转后在顶点 $(2, 3)$ 处得到最优值 22。

**大规模运输问题**：物流行业中含有数千个出发地和目的地的运输问题，利用单纯形法的网络单纯形变体（network simplex）求解，每次旋转对应在运输网络中增加或删除一条运输路径。IBM的CPLEX求解器将此算法作为核心组件，可在数秒内求解含百万变量的线性规划问题。

**退化与循环**：当最小比值检验出现平局（$\theta^* = 0$）时，产生**退化**（degeneracy），目标函数值不改变，理论上可能无限循环。布兰德规则（Bland's Rule，1977年提出）通过始终选取下标最小的合格进基/离基变量来保证有限步内终止，代价是收敛速度可能略有下降。

## 常见误区

**误区一：认为单纯形法搜索所有顶点**。实际上算法只访问可行域中目标函数值单调改善的顶点路径，对于一个含50个约束、100个变量的问题，理论顶点数可达 $\binom{100}{50} \approx 10^{29}$ 个，但单纯形法通常只需约50-100次旋转即可找到最优解。

**误区二：将检验数与目标函数偏导数混淆**。检验数 $\bar{c}_j$ 是相对于当前基的**缩减成本**，它已经扣除了将 $x_j$ 引入基后对其他基变量的间接影响（通过 $\mathbf{c}_B^T B^{-1} \mathbf{a}_j$ 项体现）。若直接使用原始目标系数 $c_j$ 判断最优性，当 $B^{-1}$ 不是单位矩阵时将得出错误结论。

**误区三：认为单纯形法不适用于大规模问题**。修正单纯形法（Revised Simplex Method）通过维护 $B^{-1}$ 的LU分解，每次旋转只更新一列，将计算复杂度从 $O(mn^2)$ 降低至 $O(m^2)$ 级别，使其能处理含数万至数十万约束的实际问题。

## 知识关联

单纯形法建立在线性规划的可行域理论之上：极点定理保证了线性规划若有最优解则必在顶点取到，这正是单纯形法只搜索顶点的理论依据。若不熟悉线性规划的标准型转化和基本可行解概念，无法理解基矩阵 $B$ 的构造与旋转操作的合法性。

单纯形法是推导**对偶理论**的直接入口：旋转操作中产生的向量 $\mathbf{y}^* = \mathbf{c}_B^T B^{-1}$（即影子价格向量）恰好是对偶问题的最优解。当单纯形法在原问题上终止时，该向量同时满足对偶可行性，这一现象直接揭示了强对偶定理的构造性证明。互补松弛条件 $x_j \bar{c}_j = 0$ 在单纯形法的每一步中都自然成立（非基变量取零值，检验数非负），是原始—对偶关系的直接体现。
