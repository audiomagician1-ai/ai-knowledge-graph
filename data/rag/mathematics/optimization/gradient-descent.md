# 梯度下降

## 概述

梯度下降（Gradient Descent）是一种通过沿负梯度方向迭代更新参数来最小化目标函数的一阶优化算法。其核心思想源于这样一个数学事实：函数在某点处下降最快的方向恰好是该点梯度的反方向。给定可微函数 $f(\theta)$，梯度下降的参数更新公式为：

$$\theta_{t+1} = \theta_t - \eta \cdot \nabla f(\theta_t)$$

其中 $\theta_t$ 是第 $t$ 步的参数值，$\eta > 0$ 是学习率，$\nabla f(\theta_t)$ 是损失函数在 $\theta_t$ 处的梯度向量。

梯度下降法最早由法国数学家奥古斯丁-路易·柯西（Augustin-Louis Cauchy）于1847年提出，用于求解线性方程组。在20世纪80年代，随着反向传播算法的普及，梯度下降成为训练多层神经网络的标准方法。它之所以重要，在于它能将高维参数空间中复杂的最小化问题转化为可以逐步计算的迭代过程，使得参数量高达数十亿的深度学习模型训练成为可能。

## 核心原理

### 学习率的作用与选取

学习率 $\eta$ 决定了每次迭代沿负梯度方向移动的步长大小，是梯度下降中最关键的超参数。$\eta$ 过大时，参数更新幅度过大，可能在极小值附近来回震荡甚至发散；$\eta$ 过小时，收敛速度极慢，训练效率低下。一个经典的经验法则是：对于凸二次函数 $f(\theta) = \frac{1}{2}\theta^T A \theta$，最优学习率为 $\eta^* = \frac{2}{\lambda_{\max} + \lambda_{\min}}$，其中 $\lambda_{\max}$ 和 $\lambda_{\min}$ 分别是 Hessian 矩阵 $A$ 的最大和最小特征值。实践中常用的学习率范围为 $10^{-4}$ 到 $10^{-1}$，并结合学习率衰减策略动态调整。

### 收敛条件与收敛速率

梯度下降的收敛性取决于目标函数的性质。对于 $L$-光滑（Lipschitz 连续梯度）的凸函数，若学习率满足 $\eta \leq \frac{1}{L}$，则梯度下降以 $O(1/t)$ 的速率收敛，即经过 $t$ 次迭代后目标函数误差不超过 $O(1/t)$。对于 $\mu$-强凸函数（条件数为 $\kappa = L/\mu$），收敛速率提升为线性收敛，误差以 $\left(1 - \frac{\mu}{L}\right)^t$ 的指数速度衰减。常见的停止准则有两种：一是梯度范数小于阈值 $\|\nabla f(\theta_t)\|_2 < \epsilon$（如 $\epsilon = 10^{-6}$）；二是相邻两次迭代参数变化量 $\|\theta_{t+1} - \theta_t\|_2 < \delta$ 足够小。

### 三种变体：批量、随机与小批量

**批量梯度下降（Batch GD）**每次使用全部 $n$ 个训练样本计算梯度 $\nabla f(\theta) = \frac{1}{n}\sum_{i=1}^n \nabla f_i(\theta)$，更新方向准确但计算代价为 $O(n)$，在大数据集上不可行。**随机梯度下降（SGD）**每次随机选取单个样本 $i$ 计算梯度 $\nabla f_i(\theta)$，计算快但梯度估计噪声大，参数更新轨迹呈锯齿形震荡。**小批量梯度下降（Mini-batch GD）**每次使用 $B$ 个样本（典型值为 32、64 或 128），在计算效率与梯度估计方差之间取得平衡，是深度学习中最常用的形式。三种变体的梯度方差分别为 $0$、$\sigma^2$、$\sigma^2/B$，批量大小 $B$ 直接控制方差与计算成本的权衡。

## 实际应用

**线性回归中的梯度下降**：对于均方误差损失 $f(\theta) = \frac{1}{2n}\|X\theta - y\|^2$，其梯度为 $\nabla f(\theta) = \frac{1}{n}X^T(X\theta - y)$。对于 $n=1000$、特征维度 $d=100$ 的数据集，批量梯度下降每步需计算 $1000 \times 100$ 的矩阵乘法，而 SGD 每步只需计算单个样本的梯度，计算量降低 1000 倍。

**逻辑回归训练**：在二分类问题中，交叉熵损失的梯度形式为 $\nabla f(\theta) = \frac{1}{n}X^T(\hat{y} - y)$，其中 $\hat{y} = \sigma(X\theta)$ 是 Sigmoid 输出。梯度下降从随机初始化出发，通常在几百次迭代内即可收敛到满意解，这比牛顿法每步需要计算并求逆 $d \times d$ Hessian 矩阵的代价低得多。

**鞍点问题**：在高维非凸优化（如神经网络训练）中，梯度为零的点不一定是极小值点，也可能是鞍点。研究表明，随机梯度下降的噪声实际上有助于逃离鞍点，这是 SGD 在神经网络训练中优于批量梯度下降的重要原因之一。

## 常见误区

**误区一：学习率越小，效果一定越好。** 很多初学者认为缩小学习率总能提高训练稳定性，但学习率过小会导致收敛速度呈 $O(1/\eta t)$ 级别降低，训练所需迭代次数可能增加数十倍。对于强凸函数，收缩学习率还会破坏线性收敛速率。正确做法是使用学习率调度策略，如余弦退火（Cosine Annealing）或阶梯式衰减。

**误区二：梯度下降总能找到全局最小值。** 梯度下降只能保证在凸函数上收敛到全局最小值。对于非凸函数，它只能保证收敛到梯度为零的驻点，该点可能是局部极小值、极大值或鞍点。在神经网络中，损失函数高度非凸，梯度下降找到的是局部极小值或近似驻点，而非全局最优解。

**误区三：梯度下降对所有参数应用相同学习率是最优的。** 标准梯度下降对每个参数分量乘以相同的 $\eta$，但不同参数的梯度尺度可能差异悬殊。当某些特征稀疏（对应参数梯度接近零）时，标准梯度下降更新这些参数极为缓慢。AdaGrad、RMSProp 等自适应学习率方法通过为每个参数维护独立的历史梯度信息来解决这一问题。

## 知识关联

梯度下降以**损失函数**为优化目标，损失函数的形式直接决定了梯度的解析表达式；从**梯度与方向导数**的知识中我们知道负梯度方向是函数值下降最快的方向，这正是梯度下降更新规则的数学依据。在**凸优化**框架下，梯度下降具有严格的全局收敛保证，而在非凸情形下只能保证局部收敛；**无约束优化**的一阶必要条件 $\nabla f(\theta^*) = 0$ 正是梯度下降的终止目标。

掌握梯度下降后，**反向传播**算法实质上是高效计算神经网络中复合函数梯度 $\nabla_\theta L$ 的链式法则应用，它解决了梯度下降在深度网络中"如何高效求梯度"的问题。进一步地，**神经网络基础**将梯度下降嵌入到前向传播与反向传播的完整训练循环中。**随机优化**则在梯度下降的框架上引入动量（Momentum）、Adam 等改进方法，将基础梯度下降的 $O(1/t)$ 收敛速率提升至自适应版本，解决实际大规模训练中的效率瓶颈。
