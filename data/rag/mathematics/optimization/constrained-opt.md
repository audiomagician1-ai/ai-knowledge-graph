# 约束优化

## 概述

约束优化（Constrained Optimization）是在变量满足一组约束条件的前提下，寻找目标函数最小值或最大值的数学问题。与无约束优化不同，约束优化的可行域（feasible region）是由等式约束和不等式约束共同定义的集合，最优解必须位于该可行域内，而非整个变量空间。这一根本区别导致梯度下降、牛顿法等无约束方法无法直接应用。

约束优化问题的标准形式由以下三部分构成：目标函数 $f(\mathbf{x})$、等式约束 $h_i(\mathbf{x}) = 0$（$i = 1, \ldots, m$）和不等式约束 $g_j(\mathbf{x}) \leq 0$（$j = 1, \ldots, p$）。这一标准形式由19世纪数学家对极值问题的系统化整理演变而来，库恩（Harold Kuhn）和塔克（Albert Tucker）在1951年发表的论文中正式建立了不等式约束问题的最优性理论，将约束优化推向现代形态。

约束优化的实际意义在于绝大多数工程与经济问题天然带有约束：生产计划受产能上限约束，桥梁设计受材料强度约束，机器学习中的支持向量机（SVM）问题本质上是一个带不等式约束的二次规划问题。掌握约束优化是理解这些领域核心算法的前提。

## 核心原理

### 可行域与活跃约束

可行域 $\mathcal{F} = \{\mathbf{x} \mid h_i(\mathbf{x}) = 0,\ g_j(\mathbf{x}) \leq 0\}$ 是所有满足约束的点的集合。在可行域边界上，不等式约束 $g_j(\mathbf{x}) = 0$ 称为**活跃约束**（active constraint），而满足严格不等式 $g_j(\mathbf{x}) < 0$ 的约束称为**非活跃约束**。最优解的性质由活跃约束集决定——内点处所有不等式约束均非活跃，边界点处至少有一个不等式约束活跃。识别活跃约束集是约束优化算法设计的核心任务之一，活跃集方法（Active Set Method）正是基于此思想构建的。

### KKT最优性条件

约束优化问题的局部最优性由**KKT条件**（Karush-Kuhn-Tucker Conditions）刻画，这是约束优化最重要的理论工具。KKT条件要求在最优点 $\mathbf{x}^*$ 处同时满足：

$$\nabla f(\mathbf{x}^*) + \sum_{i=1}^m \lambda_i \nabla h_i(\mathbf{x}^*) + \sum_{j=1}^p \mu_j \nabla g_j(\mathbf{x}^*) = 0$$

以及互补松弛条件 $\mu_j g_j(\mathbf{x}^*) = 0$、对偶可行性 $\mu_j \geq 0$，和原始可行性。其中 $\lambda_i$ 为等式约束的拉格朗日乘子（可取任意符号），$\mu_j$ 为不等式约束的KKT乘子（非负）。互补松弛条件的含义是：若某约束非活跃（$g_j < 0$），则对应乘子 $\mu_j = 0$；若乘子非零，则该约束必须活跃。Karush早在1939年其硕士论文中已独立推导出等价条件，但直到1951年才因KT论文而广为人知。

### 约束规格条件（CQ）

KKT条件为局部最优的**必要条件**，但需要满足**约束规格**（Constraint Qualification, CQ）才能保证其成立。最常用的是线性无关约束规格（LICQ）：在最优点 $\mathbf{x}^*$ 处，所有等式约束梯度与活跃不等式约束梯度构成的集合线性无关。若LICQ不成立（例如可行域在最优点处形成尖角），KKT条件可能失效，即存在不满足KKT条件的局部最优点。此外，凸优化问题中若满足Slater条件（存在严格可行点，即所有不等式约束严格成立），则KKT条件既是必要条件也是充分条件，此时问题具有强对偶性，原始问题与对偶问题的最优值相等（对偶间隙为零）。

### 凸约束优化与非凸约束优化

当目标函数为凸函数且可行域为凸集（等式约束为仿射函数，不等式约束为凸函数）时，问题为**凸约束优化**，其任意局部最优解即为全局最优解。这一性质使得凸约束优化在理论上具有完整性：多项式时间算法（如内点法）可在精度 $\epsilon$ 下以 $O(\log(1/\epsilon))$ 次迭代求解。非凸约束优化则可能存在多个局部最优，求解全局最优通常是NP难问题。

## 实际应用

**线性规划（LP）** 是约束优化最简单的形式：目标函数和所有约束均为线性，可行域是多面体（凸多面体）。Dantzig在1947年提出的单纯形法沿多面体顶点搜索最优解，其最优点必然位于顶点（极点）处，这是线性规划几何理论的核心结论。

**二次规划（QP）** 目标函数为二次函数、约束为线性的问题，支持向量机的训练问题可化为：最小化 $\frac{1}{2}\|\mathbf{w}\|^2$ 使得 $y_i(\mathbf{w}^T \mathbf{x}_i + b) \geq 1$（$i = 1, \ldots, n$）。这是一个凸QP问题，可用序列最小优化（SMO）算法高效求解。

**非线性规划（NLP）** 广泛出现于化工过程优化、航空轨迹规划等领域。例如，在给定燃料约束 $m_{\text{fuel}} \leq M$ 下最小化飞行时间，涉及非线性动力学约束，通常采用序列二次规划（SQP）方法，将原问题在当前迭代点线性化为QP子问题逐步求解。

## 常见误区

**误区1：认为KKT条件是最优性的充要条件。** KKT条件在一般非凸问题中仅为必要条件，满足KKT条件的点称为"KKT点"，可能是局部最优、鞍点甚至局部极大值。只有在凸优化且满足Slater条件时，KKT条件才同时是充要条件。忽视这一点会导致将非最优的KKT点误判为全局解。

**误区2：等式约束和不等式约束的乘子符号混淆。** 等式约束乘子 $\lambda_i$ 无符号限制（可正可负），因为等式约束既可从两侧施加压力；而不等式约束乘子 $\mu_j$ 必须非负（$\mu_j \geq 0$），这源于不等式约束只能单向推压最优点。将 $\mu_j$ 允许取负值会破坏KKT条件的几何意义——目标函数梯度必须位于约束梯度的非负线性组合形成的锥中。

**误区3：将可行域非空视为理所当然。** 当约束相互矛盾时（例如 $x \geq 3$ 且 $x \leq 1$），可行域为空集，问题无解。在高维工程问题中，约束冲突极为常见，求解前需进行可行性分析（Phase I问题），这是约束优化的预处理步骤，与无约束优化完全不同。

## 知识关联

约束优化直接依赖无约束优化的基础：理解梯度、海森矩阵和凸函数性质是分析KKT条件的前提，而约束优化通过引入拉格朗日函数将有约束问题转化为无约束鞍点问题，从而建立两者的桥梁。

**拉格朗日乘数法**是约束优化的核心求解框架，专门处理等式约束问题，是KKT条件在纯等式约束场景下的特例，掌握约束优化的可行域概念和活跃约束思想后，拉格朗日乘数法的几何意义（目标函数等值线与约束曲面相切）将更加清晰。

**多目标优化**将单目标约束优化推广至同时优化多个目标函数的场景，帕累托最优前沿的求解方法（如加权和法、$\epsilon$-约束法）本质上是将多目标问题转化为一系列单目标约束优化问题，因此约束优化的求解技术是多目标优化算法的直接基础。
