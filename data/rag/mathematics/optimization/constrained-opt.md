# 约束优化

## 概述

约束优化（Constrained Optimization）是在一组明确的数学约束条件下寻找目标函数最优值的方法。其标准形式为：在满足 $g_i(x) \leq 0$（$i=1,\ldots,m$）和 $h_j(x) = 0$（$j=1,\ldots,p$）的前提下，最小化目标函数 $f(x)$。这与无约束优化的根本区别在于，可行域被约束条件压缩为整个空间的一个子集，最优解可能恰好落在约束边界上而非内点。

约束优化问题的系统性理论研究可追溯到1951年，Harold Kuhn和Albert Tucker发表了著名的KKT条件（Karush-Kuhn-Tucker条件），而实际上William Karush在1939年的硕士论文中已独立给出了相同结论。这一理论体系为处理不等式约束提供了统一的一阶最优性刻画，成为现代非线性规划的理论基石。等式约束的处理则更早，可追溯至拉格朗日18世纪的工作。

约束优化在工程设计、资源调度、机器学习（如支持向量机的训练问题）中无处不在。以航空工程为例，飞机翼型设计必须在升力不低于某阈值（不等式约束）、翼展固定（等式约束）的条件下最小化阻力，单纯的无约束方法无法捕捉这类物理限制。

## 核心原理

### 可行域与约束分类

约束优化中，**可行域** $\mathcal{F} = \{x \in \mathbb{R}^n \mid g_i(x) \leq 0,\ h_j(x) = 0\}$ 是所有满足约束条件的点的集合。约束分为两类：

- **等式约束** $h_j(x) = 0$：将解限制在一个低维流形上。例如预算约束 $p_1 x_1 + p_2 x_2 = I$ 将二维平面上的可行域压缩为一条直线。
- **不等式约束** $g_i(x) \leq 0$：定义一个半空间或曲线围成的区域。当某不等式约束在最优点处取等号（$g_i(x^*) = 0$），称该约束为**积极约束（Active Constraint）**；严格不等号成立时称为**非积极约束**，非积极约束在局部分析中可暂时忽略。

### KKT条件

对于等式约束与不等式约束混合问题，KKT条件是局部最优的必要条件。在一阶正则性（如LICQ：线性无关约束品质条件）满足时，最优点 $x^*$ 处存在乘子 $\lambda_i \geq 0$（对应不等式约束）和 $\mu_j$（对应等式约束），使得：

$$\nabla f(x^*) + \sum_{i=1}^m \lambda_i \nabla g_i(x^*) + \sum_{j=1}^p \mu_j \nabla h_j(x^*) = 0$$

同时满足**互补松弛条件**：$\lambda_i g_i(x^*) = 0$，$\forall i$。互补松弛意味着若某约束非积极（$g_i(x^*) < 0$），则其对应乘子 $\lambda_i$ 必须为零；若乘子 $\lambda_i > 0$，则该约束必须积极。

### 凸约束优化的特殊性

当目标函数 $f$ 为凸函数，所有不等式约束 $g_i$ 也为凸函数，所有等式约束 $h_j$ 为仿射函数时，问题称为**凸约束优化**。此类问题的关键性质是：任意满足KKT条件的点都是**全局最优解**，局部最优即全局最优。线性规划是凸约束优化的特例，其最优解必然在可行域的顶点（极点）上取得，这一性质是单纯形法（Simplex Method，1947年由Dantzig提出）的算法基础。

对于非凸约束优化（如带整数约束的混合整数规划），KKT条件仍是局部最优的必要条件，但无法保证全局最优，通常需要分支定界（Branch and Bound）等组合搜索策略。

## 实际应用

**支持向量机（SVM）训练**：SVM的核心是一个二次约束优化问题——在约束 $y_i(w^T x_i + b) \geq 1$ 下最小化 $\frac{1}{2}\|w\|^2$。这是一个凸二次规划问题，通过引入拉格朗日乘子并分析KKT条件，可以将原始的 $n$ 维问题转化为对偶的 $N$（样本数）维问题，从而高效求解。

**投资组合优化**：Markowitz均值-方差模型（1952年提出）要求在 $\sum_i w_i = 1$（等式约束，权重之和为1）和 $w_i \geq 0$（不等式约束，禁止卖空）的限制下，最小化投资组合方差 $w^T \Sigma w$，同时保证期望收益 $\mu^T w \geq r_{\min}$。

**结构工程中的拓扑优化**：在体积约束 $V(x) \leq V_0$（不等式约束）下最小化结构柔度（即最大化刚度），每个有限元单元的材料密度 $0 \leq x_e \leq 1$ 构成盒式约束，这类问题通常有数万至数百万个设计变量。

## 常见误区

**误区一：认为约束越多，问题越容易求解**。直觉上约束缩小了搜索空间，似乎更容易找到最优解。但每增加一个非线性约束都会使可行域的几何结构更复杂，KKT条件的验证需要检查所有积极约束的组合，积极集策略的计算代价随约束数量指数增长。只有等式约束将问题降维到流形上时，才真正简化了搜索。

**误区二：将KKT条件与充分条件混淆**。KKT条件在一般情形下仅是必要条件，满足KKT条件的点称为"KKT点"，但不一定是最优解（也可能是鞍点或局部最大值）。只有在凸规划中，KKT条件才同时是充要条件。例如对于非凸目标函数 $f(x) = x^3$ 加上区间约束，KKT条件可能在非最优的边界点处满足。

**误区三：忽视约束品质条件（CQ）的必要性**。KKT条件成立需要在最优点处满足某种正则性假设（如LICQ、Slater条件等）。若约束品质条件不满足，最优点处可能不存在有限的KKT乘子。例如，约束 $g_1(x) = (x-1)^2 \leq 0$ 和 $g_2(x) = (x-1)^3 \leq 0$ 的唯一可行点 $x=1$ 处，两个梯度线性相关，LICQ不满足，无法保证KKT乘子存在。

## 知识关联

约束优化以无约束优化为前提：无约束优化中的梯度法、牛顿法等构成了约束优化内点法和序列二次规划（SQP）的子问题求解器。在无约束情形下局部最优的一阶必要条件 $\nabla f(x^*) = 0$，在约束优化中被KKT条件所推广——当可行域为全空间时，KKT条件恰好退化为无约束的稳定点条件。

约束优化向上衍生出两个重要方向：**拉格朗日乘数法**将等式约束问题转化为无约束问题，通过构造拉格朗日函数 $L(x, \mu) = f(x) + \mu^T h(x)$ 的稳定点来求解，是处理等式约束的核心算法框架；**多目标优化**则在约束优化基础上将标量目标函数推广为向量值目标，Pareto最优性取代了单一最优性的概念，约束优化的可行域概念在多目标情形下依然保留，成为帕累托前沿搜索的底层框架。
