# 多目标优化

## 概述

多目标优化（Multi-Objective Optimization）是指在同一决策问题中同时优化两个或两个以上相互冲突的目标函数，形式化地表达为：最小化向量函数 **F(x)** = (f₁(x), f₂(x), ..., fₖ(x))，满足约束条件 g(x) ≤ 0，其中 k ≥ 2。与单目标优化不同，多目标优化通常不存在使所有目标同时达到最优的解，因为改善一个目标往往必须以牺牲另一个目标为代价，这种内在张力使得最优解的概念本身需要重新定义。

多目标优化作为独立数学分支的奠基工作可追溯至1896年，意大利经济学家维尔弗雷多·帕累托（Vilfredo Pareto）在研究资源分配时引入了后来以其命名的最优性概念。1951年，库恩和塔克尔将这一思想纳入严格的数学框架。1971年，达斯古普塔和皮尔斯在效用理论中系统阐述了多目标决策理论。1980年代，遗传算法与多目标优化的结合（尤其是1994年Deb提出的NSGA算法，2000年改进为NSGA-II）极大推动了该领域的计算实践。

多目标优化在工程设计、经济政策和机器学习中无处不在——飞机设计需同时最小化重量与最大化强度，机器学习模型训练需平衡精度与泛化性能，投资组合需同时优化收益与风险。理解多目标优化迫使决策者承认目标之间的权衡是客观存在而非计算失误，这一认识本身具有重大的方法论价值。

## 核心原理

### Pareto最优与Pareto前沿

Pareto支配（Pareto Dominance）是多目标优化中判断解优劣的核心准则：称解 **x\*** 支配解 **x**（记作 **x\*** ≻ **x**），当且仅当对所有目标 i 有 fᵢ(**x\***) ≤ fᵢ(**x**)，且至少存在一个目标 j 使得 fⅉ(**x\***) < fⅉ(**x**)。如果一个解不被任何其他解支配，则称其为 **Pareto最优解**（或非支配解）。

所有Pareto最优解构成的集合称为**Pareto最优集**，其在目标空间中的像称为**Pareto前沿**（Pareto Front）。在双目标最小化问题中，Pareto前沿通常呈现从左上到右下的单调递减曲线：改善目标1必须接受目标2变差。理论上，连续凸问题的Pareto前沿是连续曲线；而非凸或整数规划问题的Pareto前沿可能是离散点集甚至不连通集合。

Pareto前沿的形状直接影响后续求解方法的选择——凸的Pareto前沿可用线性加权法完整覆盖，而非凸的Pareto前沿只有ε-约束法或进化算法才能找到前沿上的所有解。

### 权重法（加权求和法）

权重法将多目标问题转化为单目标标量优化：构造函数 f_scalar(x) = Σᵢ wᵢ fᵢ(x)，其中权重 wᵢ ≥ 0，且 Σᵢ wᵢ = 1。通过系统改变权重向量 **w** = (w₁, w₂, ..., wₖ)，可以得到Pareto前沿上的不同点——每一组权重对应前沿上至多一个点。

权重法有两个关键局限性，必须明确指出：

1. **非均匀采样问题**：即使均匀分布权重值，目标空间中得到的Pareto前沿采样点也不均匀分布，在曲率大的区域会严重欠采样。
2. **非凸盲区问题**：当Pareto前沿为非凸曲线时，目标空间中前沿的凹陷部分（concave region）无法被任何权重向量覆盖，也就是说存在真实的Pareto最优解，权重法原则上无法找到。这一结论由Deb于1999年通过反例严格证明。

在实际使用中，权重法适用于目标函数均已归一化（量纲统一）且Pareto前沿近似凸的情景，否则需要结合其他方法。

### ε-约束法

ε-约束法（ε-Constraint Method）的核心思路是：保留一个目标作为主目标进行最小化，将其余 k-1 个目标转化为不等式约束：

> 最小化 f₁(x)  
> 满足：f₂(x) ≤ ε₂，f₃(x) ≤ ε₃，...，fₖ(x) ≤ εₖ  
> 以及原有约束 g(x) ≤ 0

通过系统性地改变约束右端值向量 **(ε₂, ..., εₖ)**，可以逐步扫描出Pareto前沿的全貌。与权重法相比，ε-约束法即使对非凸Pareto前沿也能找到前沿上的所有点，这是其决定性优势。

实践中的关键步骤是确定 εᵢ 的取值范围：先分别单独最小化每个目标，得到其理想值（Ideal Value）和在最优化其他目标时的最差值（Nadir值），以此确定 εᵢ 的扫描区间。若 εᵢ 取值过小，问题可能无可行解；若取值过大，该约束不活跃，对应解不在Pareto前沿上。

## 实际应用

**工程结构优化**：设计桁架结构时，目标是同时最小化结构总质量（f₁）和最大节点位移（f₂）。这两个目标明显冲突——减轻质量通常需要减少材料截面积，而减少位移则要求增加截面积。用ε-约束法可以得到质量-位移的完整Pareto前沿，工程师可从中根据实际载荷要求选取合理折中点。

**机器学习超参数调优**：训练神经网络时，模型在训练集上的损失（f₁）与参数数量或L2正则项（f₂）构成多目标问题。使用NSGA-II算法搜索Pareto前沿，可以可视化精度-复杂度的权衡关系，从而选出满足部署硬件约束（参数量上限）的最优架构，这比单纯调节正则化系数更系统。

**投资组合选择**：马科维茨均值-方差模型（1952年）本质上是双目标优化：最大化预期收益 E(r)，同时最小化投资组合方差 Var(r)。其效率前沿（Efficient Frontier）就是该问题的Pareto前沿，可用权重法求解（因为该问题的前沿是凸的）。

## 常见误区

**误区一：认为找到Pareto前沿就完成了优化任务。** Pareto前沿只给出了所有非劣解的集合，最终仍需决策者根据业务偏好从前沿上选取一个解。多目标优化将技术问题（找到前沿）与决策问题（在前沿上选点）明确分离，但不能替代后者。许多初学者误以为求出Pareto前沿即等于找到了"最优解"，这混淆了优化与决策两个不同层次的问题。

**误区二：认为权重法总能找到所有Pareto最优解，只要选取足够多的权重组合。** 这一误解忽略了非凸Pareto前沿的存在。对于非凸目标空间，前沿的凹陷段上的点不对应任何权重向量的最优解——从几何上看，目标空间中的支撑超平面无法与前沿的凹陷部分相切。在这类问题中，必须使用ε-约束法或进化算法（如NSGA-II）。

**误区三：多目标优化中目标越多越好。** 当目标数量 k 超过4至5个时，Pareto前沿的维度快速增加，几乎所有解都相互非支配，Pareto支配关系失去区分能力（称为"维数灾难"）。此时传统的Pareto方法效果退化，需要转向专为高维目标空间设计的"多准则"方法，如2007年提出的MOEA/D（基于分解的多目标进化算法）。

## 知识关联

多目标优化以**约束优化**为直接前驱——ε-约束法将目标函数显式转化为约束，KKT最优性条件在Pareto最优性的数学证明中同样适用（Pareto最优解满足的一阶条件是KKT条件的向量推广）。约束优化中的可行域、活跃约束等概念直接被ε-约束法继承。
