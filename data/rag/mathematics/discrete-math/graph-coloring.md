# 图着色

## 概述

图着色（Graph Coloring）是图论中的经典组合优化问题，核心任务是给图的顶点（或边）分配颜色，使得相邻的顶点（或相邻的边）获得不同颜色，同时最小化所用颜色的总数。这里"相邻"指两顶点之间存在一条边。图着色问题最早源于1852年弗朗西斯·格思里（Francis Guthrie）在为英国地图着色时提出的疑问：是否只需4种颜色就能保证地图上任意两个相邻国家颜色不同？这一问题催生了长达124年的数学探索。

图着色问题的重要性体现在两方面：理论上，它与NP完全性理论密切相关，判断一个图是否可以用3种颜色着色（3-着色问题）是经典的NP完全问题；应用上，无线电频率分配、考试排课、寄存器分配等实际问题均可归约为图着色问题。色数χ(G)是刻画图结构复杂度的重要参数，深刻揭示了图的拓扑与组合性质之间的联系。

## 核心原理

### 色数与顶点着色

图G的**色数**（Chromatic Number）记作χ(G)，定义为使图G的顶点着色合法所需的最少颜色数。所谓合法着色，指对图中每一条边(u, v)，顶点u与顶点v的颜色不同。色数是图的拓扑不变量，不依赖顶点的标号方式。

几个基础结论：

- **完全图** Kₙ 的色数χ(Kₙ) = n，因为n个顶点两两相邻，每个顶点必须使用不同颜色。
- **二部图**（二分图）的色数χ = 2（前提非空），因为顶点可分为两个独立集，各自用一种颜色。
- **奇圈** C₂ₖ₊₁ 的色数为3，而**偶圈** C₂ₖ 的色数为2。
- 色数的下界：χ(G) ≥ ω(G)，其中ω(G)是图G的**团数**（最大完全子图的顶点数）。但这个下界并不总紧，存在团数为2（即无三角形）却色数任意大的图（如Mycielski构造）。

### 贪心着色与Brook定理

**贪心着色算法**按某一顶点排列顺序依次为每个顶点分配当前可用的编号最小颜色。贪心着色使用的颜色数不超过Δ(G)+1，其中Δ(G)是图的最大度。**Brooks定理**（1941年）给出更紧的上界：对连通图G，若G既不是完全图也不是奇圈，则χ(G) ≤ Δ(G)。例如，最大度为4的非完全连通图，其色数不超过4。

### 四色定理

**四色定理**（Four Color Theorem）断言：任意平面图的色数不超过4，即χ(G) ≤ 4对所有平面图G成立。

历史沿革：
- 1879年，Alfred Kempe发表了一个证明，后来于1890年被Percy Heawood发现存在漏洞，但Heawood利用Kempe的方法证明了**五色定理**（χ ≤ 5对平面图成立）。
- 1976年，Kenneth Appel和Wolfgang Haken借助计算机对1936种不可避免构型逐一验证，完成了四色定理的首次完整证明。这也是数学史上第一个依赖计算机辅助完成的重大定理证明。
- 1997年，Robertson、Sanders、Seymour和Thomas给出了简化版的计算机辅助证明，将不可避免构型数减少至633种。

四色定理与**Kuratowski定理**（平面图判定）密切相关：图是平面图当且仅当它不含K₅或K₃,₃作为细分子图。

### 边着色与Vizing定理

**边着色**是对图的边分配颜色，使共享顶点的两条边颜色不同。边着色所需最少颜色数称为**色指数**（Chromatic Index），记作χ'(G)。**Vizing定理**（1964年）给出精确范围：对简单图G，Δ(G) ≤ χ'(G) ≤ Δ(G)+1。色指数等于Δ(G)的图称为**第一类图**，等于Δ(G)+1的图称为**第二类图**。完全图K₂ₙ是第一类图（χ' = 2n-1），而K₂ₙ₊₁是第二类图（χ' = 2n+2）。

## 实际应用

**频率分配**：将无线基站建模为顶点，通信范围重叠的两基站之间连边，则为每个基站分配不干扰的频段等价于图着色问题，色数即所需最少频段数。

**考试排课**：将每门考试建模为顶点，有共同学生的两门考试之间连边，则合法着色方案对应不冲突的考试时间段分配，色数即所需最少时间段数。

**编译器寄存器分配**：程序中生命周期重叠的变量不能共用同一寄存器，将变量建为顶点，生命周期重叠则连边，寄存器数量即色数约束，这是编译器后端优化的经典方法。

**数独**：9×9数独可以建模为81个顶点的图，同行、同列、同宫格的格子之间连边，解数独等价于在色数约束为9的图中寻找一种合法9-着色方案。

## 常见误区

**误区一：团数等于色数**。许多学生认为χ(G) = ω(G)，但这是错误的。1955年，Mycielski构造了一族图，其中每张图没有三角形（ω = 2），但色数可以任意大。例如，Mycielski第k步构造的图色数为k+1，却不含K₃。因此ω(G)只是色数的下界，而非精确值。

**误区二：四色定理对非平面图同样成立**。四色定理仅适用于平面图（即可以画在平面上使边不交叉的图）。完全图K₅本身需要5种颜色（χ(K₅) = 5），而K₅不是平面图，这并不违反四色定理。非平面图的色数上界需要用其他定理（如Hadwiger猜想）来刻画。

**误区三：贪心算法总能给出最优着色**。贪心着色的结果依赖顶点的处理顺序，不同的顺序可能产生差异很大的着色数。对同一个图，某种顶点排列可以得到χ(G)种颜色的最优解，而另一种排列可能需要多得多的颜色。确定图的精确色数是NP难问题，不存在已知的多项式时间精确算法。

## 知识关联

图着色以**图论基础**中的顶点、边、度、路径、连通性、平面图等概念为前提。特别是：理解二部图的等价定义（无奇圈⟺可2-着色）需要掌握图的路径与圈的概念；理解四色定理需要平面图的Euler公式（V - E + F = 2）与平面图的边数上界3V - 6；理解Vizing定理的证明需要熟悉图的匹配理论。

在更高级的图论与组合数学中，图着色连接到**色多项式**（Chromatic Polynomial）P(G, k)，它计算用k种颜色对图G进行合法着色的方案总数，例如路径Pₙ的色多项式为k(k-1)ⁿ⁻¹。此外，图着色与**Ramsey理论**相互呼应：Ramsey数R(s,t)的研究同样涉及对完全图进行2-着色时的结构必然性。
