# 自然数与整数

## 概述

自然数与整数是算术基础领域中的重要概念，难度等级为 1/9。

自然数、整数的概念及基本运算

## 核心内容

自然数是从 $0$ 开始的非负整数集合 $\mathbb{N} = \{0, 1, 2, 3, \ldots\}$。整数集 $\mathbb{Z}$ 则在自然数基础上加入负数：$\mathbb{Z} = \{\ldots, -2, -1, 0, 1, 2, \ldots\}$。

**Peano公理** 是自然数的形式化基础：
1. $0$ 是自然数
2. 每个自然数 $n$ 都有后继 $S(n)$
3. 不同自然数有不同后继
4. $0$ 不是任何自然数的后继
5. 数学归纳法原理成立

## 例题与练习

**例1**: $|{-5}| = 5$，$|3| = 3$

**例2**: 整数的绝对值定义：$|n| = \begin{cases} n & n \geq 0 \\ -n & n < 0 \end{cases}$
