# 四则运算

## 概述

四则运算是算术基础领域中的重要概念，难度等级为 1/9。

加减乘除及其运算律

## 核心内容

四则运算(加、减、乘、除)是算术的基本操作。

**运算律**：
- 交换律：$a + b = b + a$，$a \times b = b \times a$
- 结合律：$(a + b) + c = a + (b + c)$
- 分配律：$a \times (b + c) = a \times b + a \times c$

减法和除法不满足交换律和结合律。除法中除数不能为零：$a \div 0$ 无意义。

## 例题与练习

**例1**: $17 \times 23 = 17 \times 20 + 17 \times 3 = 340 + 51 = 391$

**例2**: 验证分配律 $5 \times (8 + 3) = 5 \times 8 + 5 \times 3 = 40 + 15 = 55$
