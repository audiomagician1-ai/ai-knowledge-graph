# 泰勒展开

## 概述

泰勒展开是将光滑函数在某点邻域内用多项式逼近的核心技术，由英国数学家布鲁克·泰勒（Brook Taylor）于1715年在《正与反增量方法》中正式发表。其基本思想是：若函数 $f(x)$ 在点 $x_0$ 处存在 $n$ 阶导数，则可以用一个 $n$ 次多项式来近似替代 $f(x)$，且多项式的每一项系数由 $f$ 在 $x_0$ 处的各阶导数值唯一确定。

麦克劳林（Colin Maclaurin）在1742年将展开点限定为 $x_0=0$ 的特殊情形系统化，形成了麦克劳林级数，虽然名称上与泰勒分离，但本质上是泰勒展开的子情形。泰勒展开之所以重要，不仅在于它为 $e^x$、$\sin x$、$\ln(1+x)$ 等超越函数提供了可计算的多项式表示，更在于它是微积分中极限计算、误差分析、微分方程近似求解的基础工具。

## 核心原理

### 泰勒公式与余项

函数 $f(x)$ 在 $x_0$ 处的 $n$ 阶泰勒展开为：

$$f(x) = \sum_{k=0}^{n} \frac{f^{(k)}(x_0)}{k!}(x - x_0)^k + R_n(x)$$

其中 $f^{(k)}(x_0)$ 表示 $f$ 在 $x_0$ 处的 $k$ 阶导数，$k!$ 是阶乘，$R_n(x)$ 是余项。余项有两种常用形式：

**拉格朗日余项**：要求 $f$ 在包含 $x_0$ 和 $x$ 的开区间上 $n+1$ 阶可导，此时

$$R_n(x) = \frac{f^{(n+1)}(\xi)}{(n+1)!}(x - x_0)^{n+1}$$

其中 $\xi$ 是介于 $x_0$ 与 $x$ 之间的某个值（不可精确指定）。这一形式与拉格朗日中值定理结构一致，可用于误差上界估计。

**皮亚诺余项**：仅要求 $f$ 在 $x_0$ 处 $n$ 阶可导，余项表示为

$$R_n(x) = o\left((x - x_0)^n\right), \quad x \to x_0$$

皮亚诺余项不给出余项的显式表达式，但足够用于极限计算，是处理 $\frac{0}{0}$ 型不定式的高效工具。

### 常用函数的麦克劳林展开

以下六个展开式是解题的核心工具，必须熟记：

$$e^x = \sum_{n=0}^{\infty} \frac{x^n}{n!} = 1 + x + \frac{x^2}{2!} + \frac{x^3}{3!} + \cdots, \quad x \in (-\infty, +\infty)$$

$$\sin x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n+1}}{(2n+1)!} = x - \frac{x^3}{6} + \frac{x^5}{120} - \cdots, \quad x \in (-\infty, +\infty)$$

$$\cos x = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n}}{(2n)!} = 1 - \frac{x^2}{2} + \frac{x^4}{24} - \cdots$$

$$\ln(1+x) = \sum_{n=1}^{\infty} \frac{(-1)^{n-1} x^n}{n} = x - \frac{x^2}{2} + \frac{x^3}{3} - \cdots, \quad x \in (-1, 1]$$

$$(1+x)^\alpha = 1 + \alpha x + \frac{\alpha(\alpha-1)}{2!}x^2 + \cdots, \quad x \in (-1, 1)$$

$$\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n = 1 + x + x^2 + \cdots, \quad |x| < 1$$

注意 $\ln(1+x)$ 在 $x=1$ 处收敛（给出 $\ln 2 = 1 - \frac{1}{2} + \frac{1}{3} - \cdots$），在 $x=-1$ 处发散。

### 余项的误差估计

使用拉格朗日余项可以定量控制逼近误差。例如，用 $e^x \approx 1 + x + \frac{x^2}{2}$ 在 $x=0.1$ 处的误差满足：

$$|R_2(0.1)| = \frac{e^\xi}{3!}(0.1)^3 \leq \frac{e^{0.1}}{6} \times 0.001 \approx 1.84 \times 10^{-4}$$

这种估计在数值分析中用于确定需要保留多少项才能达到指定精度。

## 实际应用

**极限计算**：求 $\lim_{x \to 0} \frac{e^x - 1 - x}{x^2}$，将 $e^x = 1 + x + \frac{x^2}{2} + o(x^2)$ 代入，分子变为 $\frac{x^2}{2} + o(x^2)$，极限为 $\frac{1}{2}$。若用洛必达法则需要求导两次，而泰勒展开一步给出结果。

**物理中的小角度近似**：在单摆周期公式推导中，将 $\sin\theta$ 用 $\theta - \frac{\theta^3}{6}$ 替代，当摆角小于 $5°$（约0.087弧度）时，三次项贡献不足 $0.1\%$，这就是"单摆小角近似"的数学依据。

**数值计算**：计算 $\sqrt{e}$ 时，取 $e^{0.5}$ 的5项麦克劳林展开：$1 + 0.5 + \frac{0.25}{2} + \frac{0.125}{6} + \frac{0.0625}{24} \approx 1.6484$，与真实值 $1.6487...$ 误差仅约 $0.02\%$。

**微分方程近似解**：对于 $y'' + e^x y = 0$ 类方程，将 $e^x$ 展开至所需阶次，可将方程化为常系数形式进行近似求解。

## 常见误区

**误区一：混淆收敛域与展开的有效范围**。$\frac{1}{1-x}$ 的泰勒级数在 $|x| \geq 1$ 处完全无效，但函数本身在这些点（除 $x=1$）仍有定义。学生常误以为"展开式存在"等价于"展开式收敛到原函数"——实际上即使在 $|x|<1$ 内，也需要验证余项趋于零（此处成立），而对于某些无穷可微但非解析的函数（如 $f(x) = e^{-1/x^2}$），泰勒级数在 $x_0=0$ 处各阶导数均为零，级数收敛到零而非原函数。

**误区二：滥用有限项展开做乘法时丢失高阶项**。在计算 $\lim_{x\to 0} \frac{\sin x - x + x^3/6}{x^5}$ 时，若 $\sin x$ 只展开到 $x^3$ 项，分子恰好为零，会错误得出极限为0。必须展开至 $x^5$：$\sin x = x - \frac{x^3}{6} + \frac{x^5}{120} + o(x^5)$，分子为 $\frac{x^5}{120}$，极限为 $\frac{1}{120}$。展开阶数至少要与分母的幂次匹配。

**误区三：拉格朗日余项中 $\xi$ 的误用**。$\xi$ 的具体位置依赖于 $x$，不能在不同展开式中共用同一个 $\xi$。例如在比较两个函数时，各自的余项中的 $\xi_1$、$\xi_2$ 是不同的中间点，强行令 $\xi_1 = \xi_2$ 会导致错误结论。

## 知识关联

泰勒展开的推导依赖**高阶导数**：公式中系数 $\frac{f^{(k)}(x_0)}{k!}$ 直接由 $k$ 阶导数定义，若 $f$ 在 $x_0$ 处仅有有限阶导数，则展开截止于相应阶次。从**级数与求和**的视角看，泰勒展开将级数收敛条件具体化——余项趋于零等价于泰勒级数在该点收敛到函数值。
