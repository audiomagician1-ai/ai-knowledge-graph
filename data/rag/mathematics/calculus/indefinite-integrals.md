# 不定积分

## 概述

不定积分是求导运算的逆运算，其核心任务是：已知函数 $f(x)$，寻找所有满足 $F'(x) = f(x)$ 的函数 $F(x)$。若 $F'(x) = f(x)$，则称 $F(x)$ 为 $f(x)$ 的一个**原函数**。不定积分的记号为 $\int f(x)\,dx$，其中 $\int$ 是莱布尼茨于17世纪引入的拉长S形符号（取自拉丁语 *summa*），$f(x)$ 称为被积函数，$dx$ 指明积分变量。

从历史上看，牛顿与莱布尼茨在17世纪70年代分别独立建立微积分体系时，求原函数的方法已作为反微分（antidifferentiation）的概念出现。莱布尼茨在1675年的手稿中首次使用了 $\int$ 符号，而现代教材中"不定积分"这一术语则是将原函数族的全体统一表示为带任意常数 $C$ 的形式。

不定积分之所以重要，在于它是计算定积分、求解微分方程的基础工具。没有不定积分，就无法通过牛顿-莱布尼茨公式 $\int_a^b f(x)\,dx = F(b) - F(a)$ 将定积分化为具体数值计算。此外，物理学中从加速度还原速度、从速度还原位移，本质上都是求不定积分的过程。

## 核心原理

### 原函数存在性与不唯一性

若 $f(x)$ 在区间 $I$ 上连续，则 $f(x)$ 在 $I$ 上必有原函数——这是原函数存在定理的确切表述。但原函数并不唯一：若 $F(x)$ 是 $f(x)$ 的一个原函数，则 $F(x) + C$（$C$ 为任意实数常数）也是原函数，且 **$f(x)$ 的所有原函数都可以写成 $F(x) + C$ 的形式**。这一结论的依据是：若 $G'(x) = F'(x) = f(x)$，则 $[G(x) - F(x)]' = 0$，由拉格朗日中值定理推出 $G(x) - F(x)$ 在 $I$ 上为常数。因此，不定积分的完整定义为：

$$\int f(x)\,dx = F(x) + C$$

其中 $C$ 称为**积分常数**，缺少 $C$ 则表达不完整，会丢失无穷多个原函数。

### 基本积分公式表

不定积分的计算依赖若干基础公式，这些公式均通过对右侧求导加以验证：

| 被积函数 $f(x)$ | 不定积分结果 |
|---|---|
| $x^n$（$n \neq -1$） | $\dfrac{x^{n+1}}{n+1} + C$ |
| $\dfrac{1}{x}$ | $\ln|x| + C$ |
| $e^x$ | $e^x + C$ |
| $\sin x$ | $-\cos x + C$ |
| $\cos x$ | $\sin x + C$ |
| $\sec^2 x$ | $\tan x + C$ |
| $\dfrac{1}{\sqrt{1-x^2}}$ | $\arcsin x + C$ |
| $\dfrac{1}{1+x^2}$ | $\arctan x + C$ |

注意 $\int x^{-1}\,dx = \ln|x| + C$ 而非 $\ln x + C$，加绝对值是因为 $x$ 可以为负。

### 线性性质

不定积分满足两条线性运算法则：

$$\int [f(x) + g(x)]\,dx = \int f(x)\,dx + \int g(x)\,dx$$

$$\int k f(x)\,dx = k\int f(x)\,dx \quad (k \neq 0 \text{ 为常数})$$

利用这两条法则，可以将多项式等复合表达式拆分逐项积分。例如 $\int (3x^2 - 2\cos x)\,dx = x^3 - 2\sin x + C$，其中两项共用同一个积分常数 $C$，而不是写两个 $C_1$、$C_2$（两个任意常数之和仍为任意常数）。

## 实际应用

**运动学还原：** 已知质点加速度 $a(t) = 6t - 4$，求速度函数。对 $a(t)$ 求不定积分得 $v(t) = \int (6t-4)\,dt = 3t^2 - 4t + C$，积分常数 $C$ 由初始条件（如 $v(0) = 5$）确定，得 $C = 5$，即 $v(t) = 3t^2 - 4t + 5$。

**电路分析：** 电容器中电荷量与电流满足 $i(t) = \frac{dq}{dt}$，已知电流 $i(t)$ 时，通过不定积分 $q(t) = \int i(t)\,dt$ 还原电荷量，积分常数对应初始电荷。

**微分方程求解：** 最简单的一阶微分方程 $y' = f(x)$ 的通解直接就是 $y = \int f(x)\,dx$。例如 $y' = e^{2x}$，求解得 $y = \frac{1}{2}e^{2x} + C$（此处需用换元，$\int e^{2x}dx = \frac{1}{2}e^{2x}+C$）。

## 常见误区

**误区一：遗漏积分常数 $C$**
写 $\int 2x\,dx = x^2$ 是不完整的。原函数有无穷多个（$x^2,\; x^2+1,\; x^2-\pi$ 等），必须写 $x^2 + C$。在解微分方程时，漏写 $C$ 会导致只得到特解而非通解，丢失无穷多个解。

**误区二：对 $\int \frac{1}{x}dx$ 的处理**
许多学生写成 $\ln x + C$，忽略了 $x < 0$ 时 $(\ln(-x))' = \frac{1}{x}$ 同样成立，所以正确结果为 $\ln|x| + C$。若定义域限定为 $x > 0$ 或 $x < 0$ 的某一区间，则分支上可以去掉绝对值，但在未指定时必须保留。

**误区三：不定积分与定积分混淆**
不定积分 $\int f(x)\,dx$ 的结果是一个**函数族**（带 $C$），没有数值；定积分 $\int_a^b f(x)\,dx$ 的结果是一个**具体数值**，与变量 $x$ 无关。两者通过牛顿-莱布尼茨公式相连，但概念本质不同。把不定积分结果代入上下限、忘记减去 $F(a)$ 等错误，均源于对这两个概念的混淆。

## 知识关联

**与求导法则的关系：** 不定积分的每一个基本公式都是一条求导法则的"逆读"。例如，$(\sin x)' = \cos x$ 直接给出 $\int \cos x\,dx = \sin x + C$。因此，熟练掌握导数公式是验证和推导积分公式的直接途径——对结果求导等于被积函数，是检验积分计算正确性的唯一标准。

**与微积分基本定理的关系：** 微积分基本定理的第一部分指出，若 $\Phi(x) = \int_a^x f(t)\,dt$，则 $\Phi'(x) = f(x)$，这说明变上限积分是 $f(x)$ 的一个原函数，从而将定积分与不定积分联结起来。第二部分（牛顿-莱布尼茨公式）则直接利用不定积分 $F(x)$ 来计算 $\int_a^b f(x)\,dx$。

**通向后续主题：** 掌握不定积分基本公式和线性性质后，将进入**积分技巧**的学习，包括换元积分法（令 $u = g(x)$，利用 $du = g'(x)dx$ 化简被积式）和分部积分法（公式 $\int u\,dv = uv - \int v\,du$）。这两种方法能处理 $\int \sin(3x)\,dx$、$\int x e^x\,dx$ 等无法直接查表的复合情形，是不定积分计算能力的核心扩展。
