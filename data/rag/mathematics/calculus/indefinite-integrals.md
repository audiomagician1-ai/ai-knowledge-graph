# 不定积分

## 概述

不定积分是求导运算的逆运算。给定函数 $f(x)$，若存在函数 $F(x)$ 使得 $F'(x) = f(x)$ 在某区间上成立，则称 $F(x)$ 为 $f(x)$ 的一个**原函数**，而 $f(x)$ 的全体原函数构成的集合称为 $f(x)$ 的不定积分，记作 $\int f(x)\,dx$。

不定积分符号 $\int$ 由莱布尼茨（Gottfried Wilhelm Leibniz）于1675年引入，源自拉丁文"summa"（求和）的首字母"S"的拉长写法。符号中的 $dx$ 不仅表示积分变量为 $x$，更在换元法等技巧中具有独立的运算意义。不定积分的完整表达式必须包含任意常数 $C$，写作 $\int f(x)\,dx = F(x) + C$，这个 $C$ 代表所有原函数之间仅相差一个常数的本质差异。

不定积分的重要性在于：它是计算定积分的主要途径。根据微积分基本定理，$\int_a^b f(x)\,dx = F(b) - F(a)$，因此求出不定积分（即找到原函数）是计算区间上面积、路程、做功等实际量的前提步骤。

## 核心原理

### 原函数存在定理

并非所有函数都有原函数。**连续函数一定有原函数**，这是原函数存在的充分条件。具体而言，若 $f(x)$ 在区间 $[a,b]$ 上连续，则变上限积分 $F(x) = \int_a^x f(t)\,dt$ 就是 $f(x)$ 的一个原函数，且 $F'(x) = f(x)$。但含有跳跃间断点的函数（如符号函数在 $x=0$ 处）在包含该点的区间上没有原函数。注意：有界但含有震荡间断点的函数可能存在原函数，因此连续性是充分而非必要条件。

### 不定积分的基本公式

不定积分的基础是一张由求导公式"反推"得到的基本积分表，常用公式包括：

| 被积函数 | 不定积分 |
|---|---|
| $x^n$（$n \neq -1$） | $\dfrac{x^{n+1}}{n+1} + C$ |
| $\dfrac{1}{x}$ | $\ln\|x\| + C$ |
| $e^x$ | $e^x + C$ |
| $\sin x$ | $-\cos x + C$ |
| $\cos x$ | $\sin x + C$ |
| $\dfrac{1}{1+x^2}$ | $\arctan x + C$ |
| $\dfrac{1}{\sqrt{1-x^2}}$ | $\arcsin x + C$ |

其中 $\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C$ 中的限制条件 $n \neq -1$ 至关重要：当 $n = -1$ 时结果为对数函数，这是幂函数积分唯一的例外情形。

### 线性性质

不定积分满足线性性质：

$$\int [\alpha f(x) + \beta g(x)]\,dx = \alpha \int f(x)\,dx + \beta \int g(x)\,dx$$

其中 $\alpha, \beta$ 为任意常数。这一性质允许将复杂被积函数拆分为若干简单项分别积分，例如 $\int (3x^2 + 2\cos x)\,dx = 3 \cdot \dfrac{x^3}{3} + 2\sin x + C = x^3 + 2\sin x + C$。注意：多个不定积分合并时，所有任意常数合并为一个 $C$，而不是写成 $C_1 + C_2$。

### 微分与不定积分的互逆关系

不定积分与微分互为逆运算，但顺序不同结果稍有差异：

- $\dfrac{d}{dx}\left[\int f(x)\,dx\right] = f(x)$（先积后导，恢复原函数）
- $\int F'(x)\,dx = F(x) + C$（先导后积，需加任意常数 $C$）

第一个式子无需加 $C$，因为微分运算会消去常数项；第二个必须加 $C$，因为积分结果不唯一。

## 实际应用

**运动学中的速度与位移**：已知质点速度 $v(t) = 3t^2 - 4t + 1$（单位：m/s），求位移函数。对 $v(t)$ 求不定积分得 $s(t) = \int (3t^2 - 4t + 1)\,dt = t^3 - 2t^2 + t + C$。初始条件 $s(0) = 0$ 代入，确定 $C = 0$，从而得到完整的位移函数 $s(t) = t^3 - 2t^2 + t$。这个例子说明不定积分给出通解，而具体问题需要初始条件来确定 $C$ 的值。

**电路分析**：在RC电路中，已知电容电流 $i(t) = Ce^{-t/\tau}$（$\tau$ 为时间常数），电容电压 $u_C = \dfrac{1}{C}\int i(t)\,dt$。对指数函数积分得 $u_C = \dfrac{1}{C} \cdot (-\tau C e^{-t/\tau}) + C_0 = -\tau e^{-t/\tau} + C_0$，再由初始电压确定 $C_0$。

**微分方程求解**：形如 $y' = f(x)$ 的最简微分方程的通解直接由不定积分给出：$y = \int f(x)\,dx + C$。例如 $y' = \sin x$ 的通解为 $y = -\cos x + C$，其中 $C$ 对应无穷多条平行曲线构成的曲线族。

## 常见误区

**误区一：遗漏任意常数 $C$**
写 $\int 2x\,dx = x^2$ 是错误的，正确结果是 $x^2 + C$。不写 $C$ 意味着只给出了无穷多个原函数中的一个，而"不定积分"本质上是所有原函数的集合。在后续利用初始条件求特解时，若丢失 $C$ 则无法完成该步骤。

**误区二：对 $\dfrac{1}{x}$ 的积分漏掉绝对值**
$\int \dfrac{1}{x}\,dx = \ln|x| + C$，而非 $\ln x + C$。后者仅在 $x > 0$ 时成立；当 $x < 0$ 时，$(\ln(-x))' = \dfrac{1}{-x} \cdot (-1) = \dfrac{1}{x}$，因此绝对值符号将两段情况统一处理，定义域为 $x \neq 0$ 的全体实数。

**误区三：误认为不定积分与原函数是同一概念**
原函数是一个具体的函数（如 $x^2$），而不定积分 $\int 2x\,dx$ 表示所有原函数构成的函数族 $\{x^2 + C \mid C \in \mathbb{R}\}$。几何上，这对应平面上一族纵向平移的抛物线，而非单条曲线。两者区别在于：原函数可以有无穷多个，不定积分将它们统一表示。

## 知识关联

**与求导法则的关系**：每一条求导公式都对应一条积分公式。熟练掌握 $(\sin x)' = \cos x$、$(e^x)' = e^x$、$(x^n)' = nx^{n-1}$ 等公式，是直接验证积分结果正确性的标准方法——对积分结果求导，若等于被积函数则积分正确。

**通向积分技巧**：基本积分表只能处理简单函数。对于 $\int \sin^2 x\,dx$、$\int xe^x\,dx$、$\int \sqrt{1-x^2}\,dx$ 等复杂情形，需要分别引入**三角恒等式化简**、**分部积分法**（公式：$\int u\,dv = uv - \int v\,du$）和**三角换元法**。不定积分的基本概念是学习这三类技巧的出发点。

**与定积分的关联**：微积分基本定理（Newton-Leibniz公式）$\int_a^b f(x)\,dx = F(b) - F(a)$ 将不定积分（求原函数）与定积分（计算面积）桥接起来。求定积分的标准流程是：先求不定积分得 $F(x)$，再代入上下限做差。这也是为什么在学习黎曼和定义的定积分之后，通常通过不定积分来实际计算具体数值。
