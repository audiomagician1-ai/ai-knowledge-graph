# 曲线围面积

## 概述

曲线围面积是利用定积分计算由曲线所围成的平面区域面积的方法。其核心思想来源于黎曼积分的几何意义：将曲线下方的区域分割成无数个宽度趋近于零的矩形条，每个矩形的面积为 $f(x_i) \cdot \Delta x$，将这些矩形面积求和并取极限，即得定积分 $\int_a^b f(x)\,dx$，这个值在几何上恰好等于曲线 $y = f(x)$ 与 $x$ 轴之间、$x = a$ 到 $x = b$ 范围内的有向面积。

这一思想最早由阿基米德用"穷竭法"计算抛物线弓形面积，他证明了抛物线弓形面积恰好等于同底同高三角形面积的 $\dfrac{4}{3}$。牛顿和莱布尼茨在17世纪建立微积分基本定理后，将求面积问题统一化为求原函数的问题，使复杂曲线围成的面积计算变得系统可行。

曲线围面积的重要性在于：它是定积分最直观的几何应用，也是学习体积计算（旋转体、截面法）、弧长、曲面积等更高维度积分应用的直接前置。掌握不同情形下正确建立积分表达式的方法，是这一主题的关键能力。

## 核心原理

### 一、单条曲线与坐标轴围成的面积

当曲线 $y = f(x)$ 在 $[a, b]$ 上连续且 $f(x) \geq 0$ 时，面积公式直接为：

$$S = \int_a^b f(x)\,dx$$

若 $f(x)$ 在某子区间上取负值，则定积分计算结果会减去负值部分（有向面积相消）。为求**实际几何面积**，必须取绝对值：

$$S = \int_a^b |f(x)|\,dx$$

实际操作中，需先找出 $f(x) = 0$ 的零点，将 $[a, b]$ 分割成若干子区间，在每个子区间上确定符号后分段积分再求和。例如，$y = \sin x$ 在 $[0, 2\pi]$ 上的实际面积为：

$$S = \int_0^{\pi} \sin x\,dx + \left|\int_{\pi}^{2\pi} \sin x\,dx\right| = 2 + 2 = 4$$

而非直接计算 $\int_0^{2\pi} \sin x\,dx = 0$。

### 二、两条曲线围成的面积

设 $f(x) \geq g(x)$ 在 $[a, b]$ 上成立，则两曲线 $y = f(x)$ 与 $y = g(x)$ 之间的面积为：

$$S = \int_a^b \left[f(x) - g(x)\right]\,dx$$

积分限 $a$、$b$ 通常由两曲线的交点决定，即先联立方程组 $f(x) = g(x)$ 求解横坐标。若在 $[a, b]$ 内两曲线大小关系发生交换（存在交叉点 $c$），则需拆分：

$$S = \int_a^c \left[f(x) - g(x)\right]\,dx + \int_c^b \left[g(x) - f(x)\right]\,dx$$

**典型例题**：求抛物线 $y = x^2$ 与直线 $y = x$ 围成的面积。  
交点：$x^2 = x \Rightarrow x = 0,\, x = 1$。在 $[0,1]$ 上 $x \geq x^2$，故：

$$S = \int_0^1 (x - x^2)\,dx = \left[\frac{x^2}{2} - \frac{x^3}{3}\right]_0^1 = \frac{1}{2} - \frac{1}{3} = \frac{1}{6}$$

### 三、以 $y$ 为积分变量

当曲线更适合用 $x = \varphi(y)$ 的形式表示时，改用 $y$ 为积分变量可大幅简化计算。两曲线 $x = \varphi_1(y)$ 与 $x = \varphi_2(y)$（$\varphi_1 \geq \varphi_2$）在 $[c, d]$ 上围成的面积为：

$$S = \int_c^d \left[\varphi_1(y) - \varphi_2(y)\right]\,dy$$

例如，抛物线 $x = y^2$ 与直线 $x = y + 2$ 围成的区域，若用 $x$ 作变量则需拆开区间，改用 $y$ 积分则交点为 $y^2 = y + 2 \Rightarrow y = -1,\, y = 2$，直接得：

$$S = \int_{-1}^{2} (y + 2 - y^2)\,dy = \left[\frac{y^2}{2} + 2y - \frac{y^3}{3}\right]_{-1}^{2} = \frac{9}{2}$$

### 四、参数方程与极坐标下的面积

**参数方程**情形：若曲线由 $x = x(t),\, y = y(t)$，$t \in [\alpha, \beta]$ 给出，面积公式转化为：

$$S = \int_\alpha^\beta y(t)\, x'(t)\,dt \quad \text{（注意符号随参数方向变化）}$$

**极坐标**情形：曲线 $r = r(\theta)$ 在角度 $[\theta_1, \theta_2]$ 上围成的扇形面积公式为：

$$S = \frac{1}{2}\int_{\theta_1}^{\theta_2} r^2(\theta)\,d\theta$$

这来自将区域分割为无数个顶角为 $d\theta$、半径为 $r(\theta)$ 的小扇形，每个小扇形面积近似为 $\dfrac{1}{2}r^2\,d\theta$。

## 实际应用

**工程截面设计**：计算工字钢截面中空部分面积，即用大矩形面积减去腹板两侧矩形面积，本质上是两组直线围成的定积分计算。

**概率密度函数**：在概率论中，随机变量落在区间 $[a, b]$ 内的概率 $P(a \leq X \leq b) = \int_a^b f(x)\,dx$，即概率密度曲线与 $x$ 轴围成的面积，正态分布曲线下 $\pm 1\sigma$ 范围内面积约为 $0.6827$。

**心形线围成面积**：极坐标曲线 $r = a(1 - \cos\theta)$ 围成的心形区域面积为：

$$S = \frac{1}{2}\int_0^{2\pi} a^2(1-\cos\theta)^2\,d\theta = \frac{3\pi a^2}{2}$$

此结果综合运用了极坐标面积公式与二倍角公式 $\cos^2\theta = \dfrac{1+\cos 2\theta}{2}$。

## 常见误区

**误区一：忽视函数正负号，直接套用定积分结果作为面积。**  
定积分 $\int_a^b f(x)\,dx$ 是有向面积，当 $f(x) < 0$ 的区间存在时，该段积分为负数，会与正值区间相抵消。求实际几何面积必须先分段确定正负区间，再分别计算后取绝对值相加。上文 $\sin x$ 在 $[0, 2\pi]$ 上的例子是此类错误的典型场景。

**误区二：交点求解不完整，导致积分区间错误。**  
两曲线可能有两个以上的交点。例如 $y = \sin x$ 与 $y = \cos x$ 在 $[0, 2\pi]$ 上有两个交点 $x = \dfrac{\pi}{4}$ 和 $x = \dfrac{5\pi}{4}$，围成两个独立区域，必须分两段积分。若仅找到一个交点则面积计算完全错误。

**误区三：不考虑积分变量的选取方向，强行使用 $x$ 变量。**  
面积与选用 $x$ 还是 $y$ 作积分变量无关，结果相同，但选择不当会大幅增加计算难度，甚至导致表达式需要分多段才能完成。对于开口向左右的抛物线（如 $x = y^2$），优先换用 $y$ 作积分变量，可将多段积分化为单段，避免中间步骤的繁琐错误。

**前置知识**：曲线围面积直接依赖**定积分的计算技术**，包括牛顿-莱布尼茨公式 $\int_a^b f(x)\,dx = F(b) - F(a)$、换元积分法（参数方程转化时必用）和分部积分法。极坐标面积公式还需要熟悉 $\cos^2\theta$ 的半角化简处理。
