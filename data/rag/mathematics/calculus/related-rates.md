# 相关变化率

## 概述

相关变化率（Related Rates）是微积分中利用链式法则，求两个或多个随时间变化的相关量之间变化速率关系的方法。当两个量 x 和 y 都是时间 t 的函数，且满足某个几何或物理约束关系 F(x, y) = 0 时，对该约束方程两端同时对 t 求导，即可建立 dx/dt 与 dy/dt 之间的等式——这就是相关变化率的核心操作。

这类问题的历史可追溯至17世纪牛顿和莱布尼茨创立微积分之初。牛顿在研究行星轨道面积扫过速率与径向速率的关系时，已隐含了相关变化率的思想。现代微积分教材中，相关变化率通常作为链式法则的直接应用，出现在导数章节的末尾，典型例题包括梯子滑落、水箱排水、气球膨胀等经典场景。

相关变化率问题的实际意义在于：自然界中许多量无法直接测量其变化速率，但可以通过测量另一个相关量的变化速率间接推算。例如，雷达无法直接测量飞机的飞行速度，但可以测量飞机与雷达站之间距离的变化速率，再结合几何关系，反推飞机的水平速度。

## 核心原理

### 链式法则的时间求导形式

相关变化率的数学基础是链式法则。设 y = f(x)，而 x = x(t)，则：

$$\frac{dy}{dt} = \frac{dy}{dx} \cdot \frac{dx}{dt}$$

更一般地，若两变量满足隐函数关系 F(x, y) = 0，对 t 求全导数得：

$$\frac{\partial F}{\partial x} \cdot \frac{dx}{dt} + \frac{\partial F}{\partial y} \cdot \frac{dy}{dt} = 0$$

**关键点**：求导必须在 F(x, y) = 0 两端同时对 t 进行，而不是先从方程解出一个变量再求导。后者有时可行，但相关变化率的方法更具普遍性，尤其适用于无法显式解出某变量的情形。

### 标准解题步骤

解相关变化率问题有固定的五步流程，缺一不可：

1. **画图并命名变量**：用字母标注所有随时间变化的量，例如梯子问题中用 x 表示梯子底端到墙的水平距离，y 表示梯子顶端距地面的高度。
2. **写出约束方程**：根据题目的几何或物理条件建立 x、y（及其他量）之间的静态关系，如梯子问题中的 x² + y² = L²（L 为梯长，常数）。
3. **两端对 t 求导**：得到含有 dx/dt 和 dy/dt 的方程，梯子例子中为 2x(dx/dt) + 2y(dy/dt) = 0。
4. **代入已知瞬时值**：将题目给出的某一特定时刻的 x、y 数值以及已知的某个速率代入。
5. **解出目标速率**：用代数方法求出未知的变化率，并判断正负号的物理意义（负号通常表示减小）。

### 符号正负与方向约定

变化率的符号直接反映物理含义。若规定 y 为水箱中水的高度，则 dy/dt < 0 表示水位下降，dy/dt > 0 表示水位上升。解题者在设定变量时必须明确正方向。一个常见错误是将"底端以 2 m/s 的速度向外滑动"写作 dx/dt = −2，这与正方向的设定矛盾，会导致最终答案符号错误。正确做法是：若 x 定义为底端与墙的距离且向外为正，则 dx/dt = +2。

## 实际应用

### 经典例题：滑梯问题

一架长 5 m 的梯子靠在墙上，底端以 0.5 m/s 的速率沿地面向外滑动。当底端距墙 3 m 时，顶端下滑速率是多少？

约束方程为勾股定理：x² + y² = 25。  
对 t 求导：2x(dx/dt) + 2y(dy/dt) = 0。  
当 x = 3 时，y = √(25 − 9) = 4。  
代入 dx/dt = 0.5：2(3)(0.5) + 2(4)(dy/dt) = 0，解得 dy/dt = −3/8 = −0.375 m/s。  
负号表明顶端以 0.375 m/s 的速率向下滑动。

### 气球膨胀问题

球形气球以 100 cm³/s 的速率充气（即 dV/dt = 100）。当半径 r = 5 cm 时，半径增大速率 dr/dt 为多少？

球体体积公式 V = (4/3)πr³，对 t 求导：dV/dt = 4πr²(dr/dt)。  
代入 r = 5 和 dV/dt = 100：100 = 4π(25)(dr/dt)，解得 dr/dt = 1/π ≈ 0.318 cm/s。

### 雷达测速

飞机以恒定高度 3 km 飞行，地面雷达站测得飞机与雷达的距离 s 正以 400 km/h 的速率增大。当 s = 5 km 时，飞机的水平速度 dx/dt 为多少？

约束：x² + 9 = s²（高度 3 km 为常数）。  
对 t 求导：2x(dx/dt) = 2s(ds/dt)。  
当 s = 5 时，x = √(25 − 9) = 4 km。  
代入：2(4)(dx/dt) = 2(5)(400)，解得 dx/dt = 500 km/h。

## 常见误区

### 误区一：在特定时刻代入值之前就代入

最典型的错误是先将 x = 3、y = 4 代入约束方程 x² + y² = 25，得到 9 + 16 = 25（一个常数等式），然后再对 t 求导，结果得到 0 = 0，毫无信息。**约束方程必须在保持变量形式（x(t)、y(t)）的状态下先对 t 求导，得到速率方程后，才能代入特定时刻的数值。**

### 误区二：混淆常量与变量

在建立约束方程时，必须区分哪些量是常数、哪些量随时间变化。梯子问题中梯长 L 是常数，对 t 求导后其导数为零；但梯子底端位置 x 和顶端位置 y 都是时间的函数，求导后产生 dx/dt 和 dy/dt 项。若将 y 误当常数处理，求导后 dy/dt = 0，则无法建立有效的速率关系。

### 误区三：忽视单位一致性

相关变化率问题中各量的单位必须统一。若距离用米、时间用秒，则速率单位为 m/s；若将 dx/dt = 0.5 m/s 与 x = 300 cm 直接混用，最终计算结果的量纲将是错误的。在代入数值前，务必统一所有量的单位制。

## 知识关联

**前置概念**：相关变化率直接依赖**链式法则**（Chain Rule）——没有链式法则，就无法对隐函数形式的约束方程关于 t 求导。此外，常见约束方程来自**勾股定理**、**相似三角形**、**球体/圆锥体积公式**等几何知识，需要学生熟练运用这些公式。**隐函数求导**（Implicit Differentiation）是相关变化率的直接技术前身：隐函数求导对 x 求导，相关变化率则对 t 求导，思路完全一致。

**横向联系**：相关变化率与**最优化问题**（Optimization）同属导数应用的核心板块，两者都需要建立约束方程并对其求导，但最优化关注的是函数极值，而相关变化率关注的是瞬时变化速率。在物理学中，相关变化率直接对应**运动学**中速度与位置的关系，以及**热力学**中压强、体积、温度三者的同时变化分析（如绝热过程中的 PV^γ = 常数）。
