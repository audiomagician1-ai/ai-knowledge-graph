# 求导法则

## 概述

求导法则是一套将复杂函数的导数计算转化为基本运算的系统性规则，核心包括：和差法则、积法则（乘积法则）、商法则以及链式法则。这些规则使得任何由初等函数通过四则运算和复合构成的函数，都能在不回归极限定义的情况下直接求导。

求导法则的历史可追溯至17世纪，莱布尼茨（Leibniz）在1684年发表的论文《一种求极大极小值和切线的新方法》中首次系统给出了和法则与积法则的符号表达，牛顿则用流数法给出了等价结论。乘积法则在莱布尼茨记号下写作 $d(uv) = u\,dv + v\,du$，这一形式至今仍被教科书沿用。

这套法则的实际意义在于：若每次求导都要回到 $\lim_{h\to 0}\frac{f(x+h)-f(x)}{h}$ 的定义，则 $\sin(x^2+3x)$ 这样的函数需要极其繁琐的展开。掌握链式法则后，此题只需不超过三步即可完成，这正是微积分实用化的关键一步。

## 核心原理

### 和差法则与常数倍法则

设 $f(x)$ 与 $g(x)$ 均可导，则：

$$[f(x) \pm g(x)]' = f'(x) \pm g'(x)$$
$$[c \cdot f(x)]' = c \cdot f'(x), \quad c \text{ 为常数}$$

这两条法则合称**线性法则**，其本质是导数运算符 $\frac{d}{dx}$ 是线性算子。利用线性法则可将多项式逐项求导：$(3x^4 - 5x^2 + 7)' = 12x^3 - 10x$，每一步只需用幂函数导数公式 $(x^n)' = nx^{n-1}$。

### 乘积法则（积法则）

$$[f(x) \cdot g(x)]' = f'(x)g(x) + f(x)g'(x)$$

一个常见的直观理解：将乘积看作矩形面积，$f$ 与 $g$ 分别是两边长，面积的微小变化 $d(fg)$ 由"新边长 × 另一边的增量"两项叠加构成，高阶小量 $df \cdot dg$ 可忽略。例如，$(x^2 \sin x)' = 2x \sin x + x^2 \cos x$，其中第一项来自 $x^2$ 求导，第二项来自 $\sin x$ 求导。

乘积法则可推广到三个函数之积：$(fgh)' = f'gh + fg'h + fgh'$，每次对其中一个因子求导、其余保持不变，最后求和，形式高度对称。

### 商法则

$$\left[\frac{f(x)}{g(x)}\right]' = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2}, \quad g(x) \neq 0$$

商法则可由乘积法则推导：令 $h = f/g$，则 $f = hg$，两边用积法则求导后解出 $h'$ 即得。注意分子中两项顺序是"**先导后减**"（$f'g - fg'$），颠倒顺序将导致符号错误。例如 $(\tan x)' = \left(\frac{\sin x}{\cos x}\right)' = \frac{\cos^2 x + \sin^2 x}{\cos^2 x} = \frac{1}{\cos^2 x} = \sec^2 x$。

### 链式法则（复合函数求导）

设 $y = f(u)$，$u = g(x)$，且 $g$ 在 $x$ 处可导，$f$ 在 $u = g(x)$ 处可导，则：

$$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx} = f'(g(x)) \cdot g'(x)$$

链式法则是四条法则中最具威力的一条，专门处理**复合函数**（函数套函数）。求导时需识别"外层函数"与"内层函数"：对 $y = \sin(x^2+3x)$，外层是 $\sin(\cdot)$，内层是 $x^2+3x$，则：

$$y' = \cos(x^2+3x) \cdot (2x+3)$$

多层嵌套时链式法则可连续应用，如 $y = e^{\sin(x^2)}$：

$$y' = e^{\sin(x^2)} \cdot \cos(x^2) \cdot 2x$$

从最外层依次剥离，每剥一层乘上该层的导数。

## 实际应用

**求三角函数的衍生导数公式**：利用商法则，$(\cot x)' = \left(\frac{\cos x}{\sin x}\right)' = \frac{-\sin^2 x - \cos^2 x}{\sin^2 x} = -\csc^2 x$。类似地可得 $(\sec x)' = \sec x \tan x$，$(\csc x)' = -\csc x \cot x$，这些公式均由商法则一次性推出，无需记忆额外来源。

**物理中的速度与加速度**：若位置函数为 $s(t) = A\sin(\omega t + \phi)$（简谐振动），则速度为 $v(t) = s'(t) = A\omega\cos(\omega t + \phi)$，此处链式法则应用于内层 $\omega t + \phi$。加速度 $a(t) = v'(t) = -A\omega^2\sin(\omega t + \phi) = -\omega^2 s(t)$，直接体现了简谐振动的特征方程。

**经济学中的边际分析**：若总利润 $P(q) = R(q) - C(q)$，由和差法则得 $P'(q) = R'(q) - C'(q)$，即边际利润等于边际收益减边际成本。若 $R(q) = p(q) \cdot q$ 且价格 $p$ 是 $q$ 的函数，则求 $R'(q)$ 需用乘积法则：$R'(q) = p'(q)q + p(q)$。

## 常见误区

**误区一：乘积法则误用为"各自求导相乘"**
许多初学者将 $(fg)' = f'g'$，这是严重错误。以 $f(x) = x$，$g(x) = x^2$ 为例：正确结果 $(x^3)' = 3x^2$，而 $f'g' = 1 \cdot 2x = 2x \neq 3x^2$。乘积法则的正确形式要求**两个交叉项**的叠加，不可简化。

**误区二：链式法则中只对外层求导，忘记乘以内层导数**
对 $y = (3x+1)^{10}$ 直接写 $y' = 10(3x+1)^9$ 是不完整的。完整结果应为 $y' = 10(3x+1)^9 \cdot 3 = 30(3x+1)^9$。内层 $3x+1$ 的导数 $3$ 是必须乘上的因子，遗漏将导致系数错误。

**误区三：商法则分子顺序颠倒**
商法则分子是 $f'g - fg'$，而非 $fg' - f'g$。如果记忆时将两项顺序颠倒，整个结果符号相反。可以用助记口诀"**低求高减高求低，低低作分母**"（低指分母 $g$，高指分子 $f$）来固定顺序。

## 知识关联

求导法则直接建立在**导数概念**（极限定义）之上：四条法则均由 $f'(x) = \lim_{h\to 0}\frac{f(x+h)-f(x)}{h}$ 严格推导，是极限运算性质在导数语境下的集中体现。

掌握求导法则后，**高阶导数**的计算（如 $f''(x)$、$f^{(n)}(x)$）只需对求导结果反复应用同一套法则，链式法则在此过程中会频繁嵌套出现。**隐函数求导**在等式两边对 $x$ 求导时，对含 $y$ 的项必须通过链式法则得出 $\frac{dy}{dx}$ 因子，否则无法正确处理。在**函数图像分析**中，求极值点、拐点都依赖于对 $f'(x)$ 和 $f''(x)$ 的准确计算，这些计算的基础正是本节的四条法则。在**不定积分**中，换元积分法（$u$-替换）本质上是链式法则的逆向运算；在**常微分方程**中，建立和求解方程同样频繁调用乘积法则与链式法则。
