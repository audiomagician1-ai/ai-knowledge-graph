# 最值问题

## 概述

最值问题是指在给定区间或约束条件下，求函数的最大值（maximum）或最小值（minimum）的数学问题。利用导数可以系统地定位函数的极值点和端点，从而确定整体最值。这一方法的理论基础来自费马定理（Fermat's Theorem）：若函数 $f(x)$ 在内点 $x_0$ 处取得极值，且 $f'(x_0)$ 存在，则必有 $f'(x_0) = 0$。

最值问题的求解方法在17世纪随微积分的创立而成熟。约翰·伯努利（Johann Bernoulli）和莱布尼茨等人已将导数工具用于光学和力学中的最优路径问题。19世纪，魏尔斯特拉斯极值定理（Weierstrass Extreme Value Theorem）从理论层面保证了：定义在闭区间 $[a, b]$ 上的连续函数**必然**能取到最大值与最小值，为求最值提供了存在性依据。

最值问题在工程设计（最小材料用量）、经济学（最大利润）和物理学（最短时间路径）中无处不在。掌握导数求最值的完整流程，是分析和解决实际优化问题的基本数学工具。

## 核心原理

### 闭区间上的最值求法

对于闭区间 $[a, b]$ 上的连续函数 $f(x)$，最值必在以下三类点中产生：
1. **驻点**（稳定点）：满足 $f'(x) = 0$ 的点；
2. **不可导点**：$f'(x)$ 不存在的点（如尖点、折点）；
3. **端点**：$x = a$ 或 $x = b$。

具体步骤为：求出区间内所有驻点和不可导点，计算这些点以及两端点处的函数值，逐一比较，最大者为最大值，最小者为最小值。例如，$f(x) = x^3 - 3x$ 在 $[-2, 3]$ 上，令 $f'(x) = 3x^2 - 3 = 0$，解得 $x = \pm 1$，再计算 $f(-2) = -2$，$f(-1) = 2$，$f(1) = -2$，$f(3) = 18$，故最大值为 $18$，最小值为 $-2$。

### 极值的判断：一阶导数法与二阶导数法

**一阶导数符号变化法**：若 $f'(x)$ 在 $x_0$ 两侧从正变负，则 $x_0$ 是极大值点；从负变正，则是极小值点；符号不变则不是极值点。

**二阶导数判别法**：若 $f'(x_0) = 0$ 且 $f''(x_0) \neq 0$，则：
$$f''(x_0) < 0 \Rightarrow x_0 \text{ 为极大值点}; \quad f''(x_0) > 0 \Rightarrow x_0 \text{ 为极小值点}$$
当 $f''(x_0) = 0$ 时，此法失效，必须回到一阶导数符号法。例如 $f(x) = x^4$ 在 $x=0$ 处，$f'(0) = 0$，$f''(0) = 0$，但 $x=0$ 是极小值点，需用一阶法判断。

### 实际应用中的建模与约束化简

许多最值问题并非直接给出函数表达式，而是需要先**建模**。关键步骤是：设定自变量，写出目标函数，并利用约束条件将多元表达式化为**单变量函数**，再限定其自然定义域（往往是开区间或半开区间）。

经典模型：用长度为 $L$ 的铁丝围成一个矩形，求面积最大时的边长。设一边为 $x$，则另一边为 $\frac{L}{2} - x$，目标函数为：
$$S(x) = x\left(\frac{L}{2} - x\right) = \frac{L}{2}x - x^2, \quad x \in \left(0, \frac{L}{4}\right)$$
令 $S'(x) = \frac{L}{2} - 2x = 0$，解得 $x = \frac{L}{4}$，此时为正方形，面积最大为 $\frac{L^2}{16}$。

## 实际应用

**圆柱罐最省材料问题**：给定体积 $V$，求表面积最小的圆柱尺寸。设底面半径为 $r$，高为 $h$，约束为 $\pi r^2 h = V$，代入表面积公式 $S = 2\pi r^2 + 2\pi r h$，化简为 $S(r) = 2\pi r^2 + \frac{2V}{r}$。令 $S'(r) = 4\pi r - \frac{2V}{r^2} = 0$，解得 $r = \sqrt[3]{\frac{V}{2\pi}}$，对应高 $h = 2r$，即**高等于直径**时材料用量最省。这一结论在饮料罐设计中有直接应用。

**经济学中的最大利润**：设某商品的需求函数为 $p = 100 - 2q$，总成本为 $C(q) = 20q + 500$，则利润 $\pi(q) = (100 - 2q)q - (20q + 500) = -2q^2 + 80q - 500$。令 $\pi'(q) = -4q + 80 = 0$，得 $q = 20$，最大利润为 $\pi(20) = 300$。

## 常见误区

**误区一：仅找驻点，忽略端点和不可导点。** 在闭区间问题中，最值不一定出现在驻点处。函数 $f(x) = |x|$ 在 $[-1, 2]$ 上，最大值在端点 $x=2$ 处取得，值为 $2$；最小值在不可导点 $x=0$ 处取得，值为 $0$。若只令导数为零，完全找不到这两个最值。

**误区二：混淆极值与最值。** 极值是局部概念，最值是全局概念。一个函数可以有多个极大值，但极大值未必大于极小值——例如 $f(x) = x^3 - 6x^2 + 9x + 1$ 的极大值为 $f(1)=5$，极小值为 $f(3)=1$，但若区间扩展到 $[0, 5]$，端点 $f(5)=11$ 才是最大值，比极大值 $5$ 大得多。

**误区三：开区间问题套用闭区间步骤。** 在开区间或无界区间上，连续函数不一定取到最值。若实际问题的物理约束保证唯一驻点，且函数在该点的导数符号从正变负（或验证 $f'' < 0$），则该驻点就是最大值，**不需要与端点比较**，因为端点处函数值趋向边界极限（通常为零或无穷）。

## 知识关联

**前置概念**：求解最值问题必须熟练掌握导数的计算规则（链式法则、乘积法则等），能够对多项式、三角函数、指数对数函数正确求导，才能顺利解出 $f'(x) = 0$。同时，极限的概念用于分析无界区间上函数趋势，判断最值是否存在。

**延伸方向——动态规划（数学）**：导数求最值适用于函数连续可微的情形，是"连续优化"的核心工具。而动态规划（Dynamic Programming）处理的是离散阶段的最优决策问题，将原问题分解为子问题递推求解，本质上是对**离散状态集合**取最值，与本文方法形成互补。例如，最短路径问题无法对连续变量求导，但可以用动态规划的 Bellman 方程 $V(s) = \min_{a} [c(s,a) + V(s')]$ 逐步求解最优值。理解连续最值问题的结构有助于类比理解动态规划中"局部最优推出全局最优"的思想。
