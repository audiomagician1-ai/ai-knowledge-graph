# 勾股数组

## 概述

勾股数组是指满足方程 $a^2 + b^2 = c^2$ 的正整数三元组 $(a, b, c)$，其中 $a, b, c$ 均为正整数。最小的勾股数组是 $(3, 4, 5)$，因为 $9 + 16 = 25$。若三元组中 $\gcd(a, b, c) = 1$，则称其为**本原勾股数组**（primitive Pythagorean triple），例如 $(3, 4, 5)$、$(5, 12, 13)$、$(8, 15, 17)$ 均是本原勾股数组，而 $(6, 8, 10)$ 是 $(3, 4, 5)$ 的整数倍，不是本原的。

勾股数组的系统研究可追溯至古巴比伦时期。公元前约1800年的楔形文字泥板 **Plimpton 322** 记录了15组勾股数组，表明古代数学家已掌握生成这些数组的方法。古希腊数学家欧几里得在《几何原本》第十卷中给出了本原勾股数组的完整参数化描述，这是数论史上最早的参数化定理之一。

勾股数组在数论中的重要性在于：它是研究整数方程 $x^2 + y^2 = z^2$ 解的结构的入口，并且其参数化方法直接启发了费马大定理的研究路径——费马正是通过研究 $n = 4$ 的情形，发展出"无穷递降法"证明了 $x^4 + y^4 = z^4$ 无正整数解。

## 核心原理

### 本原勾股数组的参数化定理

**定理**：所有本原勾股数组 $(a, b, c)$（其中 $a$ 为奇数，$b$ 为偶数）可以由以下公式唯一表示：

$$a = m^2 - n^2, \quad b = 2mn, \quad c = m^2 + n^2$$

其中参数 $m, n$ 满足：
- $m > n > 0$（正整数）
- $\gcd(m, n) = 1$（互质）
- $m$ 与 $n$ 奇偶性不同（一奇一偶）

例如，取 $m = 2, n = 1$，得 $a = 3, b = 4, c = 5$；取 $m = 3, n = 2$，得 $a = 5, b = 12, c = 13$；取 $m = 4, n = 1$，得 $a = 15, b = 8, c = 17$（此时 $a > b$，但仍满足条件）。

### 定理的证明思路

证明分两步。**第一步**验证公式给出的三元组确实是本原勾股数组：直接计算 $a^2 + b^2 = (m^2-n^2)^2 + (2mn)^2 = m^4 - 2m^2n^2 + n^4 + 4m^2n^2 = (m^2+n^2)^2 = c^2$，验证勾股关系成立。再利用 $\gcd(m,n)=1$ 且奇偶性不同，可以证明 $\gcd(a,b,c)=1$。

**第二步**证明所有本原勾股数组都形如此式，是证明的核心难点。若 $(a, b, c)$ 是本原勾股数组且 $b$ 为偶数，则由 $b^2 = c^2 - a^2 = (c-a)(c+a)$ 出发，利用 $\gcd(c-a, c+a) = 2$（因为 $a, c$ 均为奇数）以及本原性条件，可以证明 $c-a$ 和 $c+a$ 必须分别是某两个整数平方的两倍，令 $c - a = 2n^2, c + a = 2m^2$，则 $m, n$ 满足所有条件。

### 奇偶性约束的必要性

在本原勾股数组中，$a$ 和 $b$ 不能同时为奇数，也不能同时为偶数。若 $a, b$ 均为偶数，则 $c$ 也为偶数，$\gcd(a,b,c) \geq 2$，违反本原性。若 $a, b$ 均为奇数，则 $a^2 + b^2 \equiv 1 + 1 = 2 \pmod{4}$，但完全平方数模4只能余0或1，而2不是完全平方数模4的可能值，产生矛盾。因此本原勾股数组中必有**恰好一个偶数**，且偶数总是 $b = 2mn$ 的形式。

## 实际应用

### 生成所有本原勾股数组

利用参数化公式，可以系统枚举本原勾股数组。按斜边 $c = m^2 + n^2$ 排序，前几组为：

| $m$ | $n$ | $a$ | $b$ | $c$ |
|-----|-----|-----|-----|-----|
| 2 | 1 | 3 | 4 | 5 |
| 3 | 2 | 5 | 12 | 13 |
| 4 | 1 | 15 | 8 | 17 |
| 4 | 3 | 7 | 24 | 25 |
| 5 | 2 | 21 | 20 | 29 |

注意 $c = 25$ 对应的是 $(7, 24, 25)$ 而非 $(20, 21, 29)$，因为同一斜边长可能对应多组本原勾股数组。

### 判断整数能否作为直角三角形的斜边

奇素数 $p$ 能作为本原勾股数组的斜边，当且仅当 $p \equiv 1 \pmod{4}$。例如 $p = 5$ 是斜边 $(3,4,5)$，$p = 13$ 是斜边 $(5,12,13)$，而 $p = 7, 11, 19$ 不能成为任何本原勾股数组的斜边。这个结论与高斯整数中的素数分解理论直接相关，为 $n^2 + 1$ 类型的数论问题提供了几何直觉。

### 在密码学和竞赛中的应用

在数学竞赛中，勾股数组参数化常用于证明形如"满足某条件的直角三角形面积不能是完全平方数"的命题。面积公式 $S = \frac{1}{2}ab = mn(m^2-n^2) = mn(m-n)(m+n)$ 的因子分析是此类证明的标准手段。

## 常见误区

**误区一：认为"三边成等比数列"可以是勾股数组**。设 $a, b, c$ 成等比数列且 $a^2 + b^2 = c^2$，则 $b^2 = ac$，代入得 $a^2 + ac = c^2$，即 $(c/a)^2 - (c/a) - 1 = 0$，解得 $c/a = \phi = \frac{1+\sqrt{5}}{2}$（黄金比例），这是无理数，因此不存在整数勾股数组使三边成等比数列。

**误区二：将参数化公式中 $m, n$ 奇偶相同的情形忘记排除**。若取 $m = 3, n = 1$（同为奇数），公式给出 $a = 8, b = 6, c = 10$，虽然 $8^2 + 6^2 = 10^2$ 成立，但 $\gcd(8, 6, 10) = 2$，这不是本原勾股数组，而是 $(3, 4, 5)$ 的两倍。奇偶性不同的条件正是为了保证 $\gcd(m^2-n^2, 2mn, m^2+n^2) = 1$。

**误区三：认为勾股数组有无穷多组但本原勾股数组只有有限组**。事实恰好相反：本原勾股数组有无穷多组，因为参数对 $(m, n) = (k, k-1)$（$k \geq 2$，$k$ 与 $k-1$ 自动互质且奇偶不同）生成无穷多个本原勾股数组，斜边 $c = (2k-1)^2/2 + 1/2$ 随 $k$ 增大而趋于无穷。

## 知识关联

**与勾股定理的关系**：勾股定理给出了 $a^2 + b^2 = c^2$ 这一关系，但它本身只处理几何关系，不追问何时有整数解。勾股数组的参数化定理正是在勾股定理的方程框架下，引入数论工具回答"整数解的完整结构是什么"这一问题。
