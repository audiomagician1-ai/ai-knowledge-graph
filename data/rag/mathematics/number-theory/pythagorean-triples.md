# 勾股数组

## 概述

勾股数组（Pythagorean Triple）是满足方程 $a^2 + b^2 = c^2$ 的正整数三元组 $(a, b, c)$，其中 $c$ 为最大值，代表直角三角形的斜边长度。最著名的例子是 $(3, 4, 5)$，它在古埃及建筑中已被实际应用于构造直角，古巴比伦泥板 Plimpton 322（约公元前 1800 年）更记录了超过 15 组勾股数，表明人类对这一结构的探索已有近四千年历史。

在古希腊，毕达哥拉斯学派给出了生成部分勾股数组的公式：取奇数 $m$，则 $(m,\ \frac{m^2-1}{2},\ \frac{m^2+1}{2})$ 是一组勾股数。然而这一公式只能生成 $a$ 为奇数且 $b, c$ 相差为 1 的情形，无法覆盖所有解。直到欧几里得在《几何原本》第十卷中给出完整的参数化描述，才真正解决了所有本原勾股数组（Primitive Pythagorean Triple）的分类问题。

勾股数组的理论是数论中整数方程（丢番图方程）求解的经典范例，其参数化方法直接启发了费马大定理的研究路径，并与有理点在代数曲线上的分布理论紧密相连。

## 核心原理

### 本原勾股数组的定义与基本性质

若三元组 $(a, b, c)$ 满足 $a^2 + b^2 = c^2$ 且 $\gcd(a, b, c) = 1$（即三者互素），则称之为**本原勾股数组**。所有勾股数组都是某个本原勾股数组 $(a_0, b_0, c_0)$ 的正整数倍 $(ka_0, kb_0, kc_0)$，因此研究本原情形即可掌握全部结构。

本原勾股数组具有以下精确的奇偶性质，可通过模 2 的同余分析严格证明：

- $a$ 与 $b$ 中恰好有一个为偶数，另一个为奇数；
- $c$ 必为奇数；
- 偶数边必须是 4 的倍数。

这意味着 $(2, 2, 2\sqrt{2})$ 这类情形从一开始就被排除在整数解之外，而 $(6, 8, 10)$ 虽满足勾股方程，但因 $\gcd(6,8,10)=2$ 而非本原组。

### 欧几里得参数化公式

**欧几里得公式**是本原勾股数组的完整分类定理：所有本原勾股数组恰好由以下参数给出（其中 $b$ 为偶数边）：

$$a = m^2 - n^2,\quad b = 2mn,\quad c = m^2 + n^2$$

其中参数 $m, n$ 满足以下三个条件：
1. $m > n > 0$（正整数）
2. $\gcd(m, n) = 1$（互素）
3. $m \not\equiv n \pmod{2}$（奇偶性不同）

**验证**：$a^2 + b^2 = (m^2-n^2)^2 + (2mn)^2 = m^4 - 2m^2n^2 + n^4 + 4m^2n^2 = (m^2+n^2)^2 = c^2$，公式成立。

例如，取 $m=2, n=1$，得 $(3, 4, 5)$；取 $m=3, n=2$，得 $(5, 12, 13)$；取 $m=4, n=1$，得 $(15, 8, 17)$，即 $(8, 15, 17)$。这三个结果覆盖了最小的三组本原勾股数组，彼此不重复。

### 公式的唯一性与完备性证明思路

欧几里得公式不仅能生成本原勾股数组，而且"恰好"覆盖全部本原解——这两个方向都需要证明。

**完备性**的核心步骤：设 $(a, b, c)$ 为本原勾股数组，$b$ 为偶数。则 $\gcd(c-a, c+a) = 2$（利用 $c, a$ 均奇数），将 $b^2 = (c-a)(c+a)$ 写为两个互素因子之积，再利用唯一分解定理，令 $c - a = 2n^2$，$c + a = 2m^2$，解出 $a = m^2 - n^2$，$c = m^2 + n^2$，$b = 2mn$，并验证 $m, n$ 必须满足上述三个条件。这一推导过程将同余理论（奇偶性分析）与算术基本定理（唯一分解）紧密结合。

## 实际应用

**生成具体数组**：利用欧几里得公式系统列举本原勾股数组时，按 $m$ 从小到大排列，$m=2$ 只有 $n=1$ 一种情形，给出 $(3,4,5)$；$m=3$ 给出 $n=2$ 的 $(5,12,13)$；$m=4$ 给出 $n=1$ 的 $(8,15,17)$ 和 $n=3$ 的 $(7,24,25)$。可以验证，斜边不超过 100 的本原勾股数组共有 **16** 组。

**竞赛题型中的应用**：在数学竞赛中，常见题型是"证明方程 $x^2 + y^2 = z^2$ 的所有正整数解"，解法标准流程即为先设 $\gcd(x,y,z)=d$，约去后还原为本原情形，再套用欧几里得公式。例如求满足斜边为质数 $p$ 的所有本原勾股数组：由 $c = m^2 + n^2 = p$，需将质数 $p$ 表示为两个正整数的平方和，由费马二平方定理，$p$ 能被如此表示当且仅当 $p = 2$ 或 $p \equiv 1 \pmod{4}$，且表示方式唯一（不计顺序），因而对应唯一的本原勾股数组。

**密码学与椭圆曲线**：勾股数组对应单位圆 $x^2 + y^2 = 1$ 上的有理点 $\left(\frac{a}{c}, \frac{b}{c}\right)$，参数化方法等价于从点 $(-1, 0)$ 出发的有理斜率直线与单位圆的第二交点，斜率 $t = \frac{n}{m}$ 与欧几里得参数直接对应。这一几何视角是椭圆曲线有理点分类方法的最简单原型。

## 常见误区

**误区一：认为 $(a, b, c)$ 中 $a < b < c$ 且 $a$ 必须最小**

在欧几里得公式的标准形式中，$b = 2mn$ 是偶数边，但 $a = m^2 - n^2$ 与 $b$ 的大小关系并不固定。例如 $m=4, n=1$ 时 $a=15$，$b=8$，$a > b$。教材有时将 $(8, 15, 17)$ 和 $(15, 8, 17)$ 视为同一组，但套用公式时需注意哪条边对应公式中的 $a$，哪条对应 $b$，否则会在验证互素条件时出错。

**误区二：放宽条件后公式仍能生成本原组**

若忽略条件 $\gcd(m, n) = 1$ 或 $m \not\equiv n \pmod{2}$ 中的任何一个，公式仍能生成满足勾股方程的三元组，但结果不再是本原组。例如 $m=2, n=2$ 时（两者同奇偶），得 $(0, 8, 8)$，不是正整数组；$m=3, n=1$ 时（均奇），$\gcd(3,1)=1$ 满足，但 $3 \equiv 1 \pmod 2$，得 $(8, 6, 10) = 2 \times (4,3,5)$，并非本原组。两个条件缺一不可。

**误区三：本原勾股数组中的奇数边可以任意分配**

有些学生认为本原勾股数组中两条直角边可以同时为奇数。但由 $a^2 + b^2 = c^2$ 在模 4 意义下分析：若 $a, b$ 均奇，则 $a^2 \equiv b^2 \equiv 1 \pmod{4}$，从而 $c^2 \equiv 2 \pmod{4}$，而完全平方数模 4 只能余 0 或 1，矛盾。因此本原组中直角边的奇偶组合只有"一奇一偶"这唯一情形。

## 知识关联

**与勾股定理的关系**：勾股定理 $a^2 + b^2 = c^2$ 是几何命题，勾股数组则是在整数范围内寻找其全部解，是从几何到数论的具体化。欧几里得参数化所解决的问题，本质上是给出该方程在正整数上的完整解集。

**与同余理论的关系**：本原勾股数组奇偶性的证明（偶数边整除 4、斜边必须为奇数）完全依赖模 2 和模 4 的同余计算，是同余理
