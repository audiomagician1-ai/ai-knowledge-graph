# 勾股定理

## 概述

勾股定理描述直角三角形三边之间的定量关系：若直角三角形的两条直角边长为 a 和 b，斜边长为 c，则 **a² + b² = c²**。这一等式精确刻画了"面积守恒"的几何事实——以两条直角边为边长所作的正方形，面积之和恰好等于以斜边为边长所作正方形的面积。

此定理在中国古代称为"勾股定理"，因为古人将直角三角形较短的直角边称为"勾"，较长的直角边称为"股"，斜边称为"弦"。《周髀算经》中记载了商高与周公的对话，提到"勾三股四弦五"，时间约在公元前 1000 年前后。同一定理在西方以古希腊数学家毕达哥拉斯（Pythagoras，约公元前 570—495 年）命名，称为"Pythagorean theorem"，但历史证据表明巴比伦人早在公元前 1800 年已知其数值关系。

该定理的重要性在于它是联结代数与几何的基础桥梁之一：用坐标计算两点之间的距离、判断三角形是否为直角三角形、在测量学中计算不可直达的距离，这些计算均直接依赖 a² + b² = c² 这一等式。

## 核心原理

### 面积拼接证明（赵爽弦图）

中国三国时期数学家赵爽在注释《周髀算经》时，给出了一种以面积为核心的几何证明。构造一个边长为 c 的大正方形，内部用四个全等的直角三角形（直角边为 a、b）拼成一圈，中心余下一个小正方形。

- 大正方形面积：c²  
- 四个直角三角形总面积：4 × (½ab) = 2ab  
- 中心小正方形边长为 (b − a)，面积：(b − a)² = b² − 2ab + a²

由 c² = 2ab + (b − a)² 展开得：

> c² = 2ab + b² − 2ab + a² = **a² + b²**

这一证明无需任何代数预设，仅靠面积的分割与重组即可完成，被认为是最直观的几何证明之一。

### 相似三角形证明

对直角三角形 ABC（∠C = 90°），过 C 作斜边 AB 的高，垂足为 D。此时三角形 ABC 被分割为两个较小的直角三角形 ACD 与 CBD，且三者相互相似（AA 相似条件）。

由 △ACD ∽ △ABC，得：
> AC / AB = AD / AC，即 **AC² = AB · AD**，即 **b² = c · AD**

由 △CBD ∽ △ABC，得：
> BC / AB = BD / BC，即 **BC² = AB · BD**，即 **a² = c · BD**

两式相加：a² + b² = c(AD + BD) = c · c = **c²**。

此证明直接利用了相似三角形对应边成比例的性质，是相似三角形知识的直接延伸应用。

### 逆定理与等价性

勾股定理的逆命题同样成立：若三角形三边满足 a² + b² = c²，则该三角形是直角三角形，且 c 对应的角为直角。判断步骤为：将三边中最长边的平方与另外两边平方之和比较——相等则为直角，小于则为钝角，大于则为锐角三角形。逆定理广泛用于施工放线（"3-4-5 法则"可验证墙角是否为直角）。

## 实际应用

**求对角线长度**：矩形长 8 cm、宽 6 cm，其对角线 d 满足 d² = 8² + 6² = 64 + 36 = 100，故 d = 10 cm。

**梯子问题**：一架 5 m 长的梯子斜靠墙，梯脚距墙 3 m，则梯顶距地面高度 h = √(5² − 3²) = √(25 − 9) = √16 = **4 m**。这类计算是土木工程中验算支撑结构的基本操作。

**坐标平面内两点距离**：平面内点 P(x₁, y₁) 与 Q(x₂, y₂) 之间的距离公式直接由勾股定理导出：

> PQ = √[(x₂ − x₁)² + (y₂ − y₁)²]

水平差 (x₂ − x₁) 与竖直差 (y₂ − y₁) 分别为直角三角形的两条直角边，PQ 为斜边。

**判断直角三角形**：已知三边为 5、12、13，验算：5² + 12² = 25 + 144 = 169 = 13²，故该三角形是直角三角形。三元组 (5, 12, 13) 是满足勾股定理的整数解，称为勾股数组。

## 常见误区

**误区一：认为 a² + b² = c² 对所有三角形成立**
该等式仅在直角三角形中成立，且 c 必须是斜边（最长边）。若将任意边代入 c 的位置，结论将完全错误。余弦定理 c² = a² + b² − 2ab·cosC 是一般三角形的对应公式，当 C = 90° 时才退化为 a² + b² = c²。

**误区二：混淆"勾股定理"与"勾股数组"**
a² + b² = c² 是一个对所有正实数 a、b、c 均可验证的等式关系，而勾股数组特指满足该等式的正整数三元组，如 (3, 4, 5)、(5, 12, 13)、(8, 15, 17)。无理数边长（如 a = 1, b = 1, c = √2）同样满足勾股定理，但不属于勾股数组。

**误区三：斜边平方等于两直角边之和（忘记平方）**
初学者常将 a² + b² = c² 误写为 a + b = c，但三角形不等式明确指出任意两边之和大于第三边，因此 a + b > c 恒成立，两者含义截然不同。正确表述必须保持各项的平方形式。

## 知识关联

**前置知识衔接**：勾股定理的相似三角形证明要求学生已掌握"两角相等则两三角形相似（AA条件）"以及相似比与面积比的关系。若相似三角形的基础不扎实，高上的投影证明将难以理解。

**通向坐标几何**：勾股定理直接导出平面直角坐标系中的距离公式 √[(Δx)² + (Δy)²]，这是坐标几何初步的出发点。没有这一等式，圆的方程 (x − h)² + (y − k)² = r² 也无法建立。

**通向三角比与余弦定理**：在直角三角形中，sinθ = a/c、cosθ = b/c 的定义依赖勾股定理推导出恒等式 sin²θ + cos²θ = 1。当三角形不含直角时，余弦定理 c² = a² + b² − 2ab·cosC 将勾股定理推广到一般情形，C = 90° 时 cosC = 0，方程自然还原为 a² + b² = c²。

**勾股数组的数论延伸**：整数解 (a, b, c) 的完整分类公式为 a = m² − n²、b = 2mn、c = m² + n²（m > n > 0，互质且奇偶性不同），这属于初等数论的研究范畴，是后续学习整数分解与费马大定理背景知识的起点。
