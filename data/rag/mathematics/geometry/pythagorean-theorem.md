# 勾股定理

## 概述

勾股定理描述直角三角形三边之间的定量关系：若直角三角形的两条直角边长为 $a$ 和 $b$，斜边长为 $c$，则：

$$a^2 + b^2 = c^2$$

这一等式精确刻画了一个几何事实——以两条直角边为边长分别作正方形，两个正方形的面积之和恰好等于以斜边为边长所作正方形的面积。

在中国古代，直角三角形中较短的直角边称为"勾"，较长的直角边称为"股"，斜边称为"弦"。《周髀算经》（约成书于公元前 100 年，记录内容可追溯至公元前 1000 年前后）中记载了商高与周公的对话，明确提到"勾三股四弦五"，这是目前有文献可考的最早勾股数记录之一。同一定理在西方以古希腊数学家毕达哥拉斯（Pythagoras，约公元前 570—495 年）命名，称为"Pythagorean theorem"。然而考古证据表明，巴比伦泥板文书 Plimpton 322（约公元前 1800 年）中已系统列出了多组勾股数对，如 $(119, 120, 169)$、$(3367, 3456, 4825)$ 等，说明其数值规律的发现远早于毕达哥拉斯。

该定理是连接代数与平面几何的核心桥梁：坐标平面上两点间的距离公式、判断三角形类型、在测量学中计算不可直达的距离，以及后续三角比和余弦定理的推导，均直接以 $a^2 + b^2 = c^2$ 为出发点。

## 核心原理

### 面积拼接证明（赵爽弦图）

中国三国时期数学家赵爽（约公元 200—260 年）在注释《周髀算经》时，给出了一种纯几何的面积证明，后人称之为"赵爽弦图"。构造方法如下：

取边长为 $c$ 的正方形，在其内部拼入四个全等的直角三角形（两直角边分别为 $a$ 和 $b$，$a < b$），每个三角形的斜边恰好构成大正方形的一条边。四个三角形拼合后，中心余下一个边长为 $(b - a)$ 的小正方形。

列出面积等式：

$$c^2 = 4 \times \frac{1}{2}ab + (b-a)^2 = 2ab + b^2 - 2ab + a^2 = a^2 + b^2$$

整个证明不依赖任何代数预设，仅靠面积的分割与重组完成，被《数学：确定性的丧失》（Morris Kline, 1980）列为世界上最直观的几何证明之一。

### 相似三角形证明（欧几里得方法）

欧几里得在《几何原本》第一卷命题 47（约公元前 300 年）中给出了另一种证明，其核心依赖相似三角形的比例关系。

对直角三角形 $ABC$（$\angle C = 90°$，斜边 $AB = c$，直角边 $BC = a$，$AC = b$），过 $C$ 作斜边 $AB$ 的垂线，垂足为 $D$，设 $AD = m$，$BD = n$，则 $m + n = c$。

此时产生三对相似三角形：

- $\triangle ACD \sim \triangle ABC$（AA 条件：共用 $\angle A$，均含直角）

$$\frac{b}{c} = \frac{m}{b} \Rightarrow b^2 = cm$$

- $\triangle CBD \sim \triangle ABC$（AA 条件：共用 $\angle B$，均含直角）

$$\frac{a}{c} = \frac{n}{a} \Rightarrow a^2 = cn$$

两式相加：

$$a^2 + b^2 = cn + cm = c(n + m) = c \cdot c = c^2$$

这一路径将勾股定理的证明转化为相似三角形比例性质的直接推论，与前置知识"相似三角形"形成自然衔接。

### 逆定理与三角形类型判断

勾股定理的逆命题同样成立，且可进一步推广为三角形类型的判别准则。设三角形三边满足 $a \leq b \leq c$，则：

| 条件 | 三角形类型 |
|---|---|
| $a^2 + b^2 = c^2$ | 直角三角形（$c$ 对应直角） |
| $a^2 + b^2 > c^2$ | 锐角三角形（三角均为锐角） |
| $a^2 + b^2 < c^2$ | 钝角三角形（$c$ 对应钝角） |

逆定理在建筑放线中有直接应用：施工时用"3-4-5 法则"验证墙角是否为直角——取绳子在两墙方向上分别量出 3 m 和 4 m，若对角线恰好为 5 m（因为 $3^2 + 4^2 = 9 + 16 = 25 = 5^2$），则该墙角确为直角。这一操作在埃及古王国时期（约公元前 2600 年）的建筑记录中已有体现，当时从事此工作的人被称为"绳索伸展者"（rope stretchers）。

## 关键公式与勾股数

### 基本公式与变形

由 $a^2 + b^2 = c^2$ 出发，可直接推导三种"已知两边求第三边"的计算公式：

$$c = \sqrt{a^2 + b^2}, \quad a = \sqrt{c^2 - b^2}, \quad b = \sqrt{c^2 - a^2}$$

在坐标几何中，平面上两点 $P_1(x_1, y_1)$ 和 $P_2(x_2, y_2)$ 的距离公式由勾股定理直接导出：

$$d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$

这一公式将勾股定理从几何图形推广到了数值坐标系，是解析几何的起点。

### 勾股数的通项公式

满足 $a^2 + b^2 = c^2$ 的正整数三元组 $(a, b, c)$ 称为勾股数组（Pythagorean triple）。所有本原勾股数组（即 $a, b, c$ 互质）均可由以下参数公式生成（参见 《初等数论》，华罗庚，科学出版社，1979）：

$$a = m^2 - n^2, \quad b = 2mn, \quad c = m^2 + n^2$$

其中 $m > n > 0$，$m$ 与 $n$ 互质，且 $m - n$ 为奇数。

例如取 $m = 2, n = 1$：$a = 3, b = 4, c = 5$；取 $m = 3, n = 2$：$a = 5, b = 12, c = 13$；取 $m = 4, n = 1$：$a = 15, b = 8, c = 17$。将本原勾股数乘以任意正整数 $k$，所得 $(ka, kb, kc)$ 也是勾股数组，如 $(6, 8, 10)$、$(9, 12, 15)$ 等。

### 用代码验证勾股数

```python
# 枚举 100 以内所有勾股数组
triples = []
for a in range(1, 101):
    for b in range(a, 101):
        c_sq = a**2 + b**2
        c = int(c_sq**0.5)
        if c * c == c_sq and c <= 100:
            triples.append((a, b, c))

for triple in triples[:10]:
    a, b, c = triple
    print(f"{a}² + {b}² = {c}²  →  {a**2} + {b**2} = {c**2}")
```

输出的前三行为：
```
3² + 4² = 5²  →  9 + 16 = 25
5² + 12² = 13²  →  25 + 144 = 169
6² + 8² = 10²  →  36 + 64 = 100
```

## 实际应用

**例1：矩形对角线**
矩形长 $8$ cm、宽 $6$ cm，其对角线 $d$ 满足：
$$d = \sqrt{8^2 + 6^2} = \sqrt{64 + 36} = \sqrt{100} = 10 \text{ cm}$$

**例2：梯子问题**
一架 $5$ m 长的梯子斜靠墙，梯脚距墙底 $3$ m，则梯顶距地面高度：
$$h = \sqrt{5^2 - 3^2} = \sqrt{25 - 9} = \sqrt{16} = 4 \text{ m}$$

**例3：屏幕尺寸**
电视机屏幕标称"55 英寸"指的是对角线长度为 55 英寸（约 139.7 cm）。若屏幕宽高比为 16:9，设宽为 $16k$、高为 $9k$，则：
$$\sqrt{(16k)^2 + (9k)^2} = 55 \Rightarrow k\sqrt{256 + 81} = 55 \Rightarrow k = \frac{55}{\sqrt{337}} \approx 2.997$$
实际宽约 $47.95$ cm，高约 $26.97$ cm。这一计算是消费电子产品标注尺寸的标准方法。

**例4：导航与测量**
一艘船从港口出发，先向正东航行 $9$ 海里，再向正北航行 $40$ 海里，此时船与港口的直线距离为：
$$d = \sqrt{9^2 + 40^2} = \sqrt{81 + 1600} = \sqrt{1681} = 41 \text{ 海里}$$

## 常见误区

**误区1：将定理应用于非直角三角形**
$a^2 + b^2 = c^2$ 仅在直角三角形中成立。若三角形三内角为 $50°、60°、70°$，则任意两边的平方和均不等于第三边的平方，此时应使用余弦定理：$c^2 = a^2 + b^2 - 2ab\cos C$。

**误区2：混淆斜边与直角边**
公式中 $c$ 必须是斜边（直角的对边，也是最长边）。若错误地将某条直角边当作 $c$ 代入，会得到不等式而非等式，计算出负数的平方根，从而出现无意义结果。判断方法：$c$ 永远是三边中最长的一条。

**误区3：忽略逆定理的条件**
逆定理要求三边均为正实数且构成三角形。三边 $(1, 1, \sqrt{2})$ 满足 $1^2 + 1^2 = (\sqrt{2})^2$，是直角三角形；但 $(0, 3, 3)$ 中 $0^2 + 3^2 = 3^2$ 成立，却因边长为 0 不构成三角形，不能套用逆定理。

**误区4：勾股定理不适用于三维空间的误解**
实际上，长方体中对角线长度可用三维推广公式：
$$d = \sqrt{l^2 + w^2 + h^2}$$
这是对勾股定理两次应用的结果：先由底面对角线 $\sqrt{l^2 + w^2}$，再与高 $h$ 构成直角三角形，即可得到体对角线。

## 知识关联

**前置知识**：相似三角形的 AA 判定准则和对应边比例性质是"相似三角形证明"路径的直接依赖；理解三角形面积公式 $S = \frac{1}{2}ab$ 是赵爽弦图证明的基础。

**直接延伸——坐标几何**：将 $a^2 + b^2 = c^2$ 中的 $a, b$ 替换为坐标差 $(x_2-x_1)$ 和 $(y_2-y_1)$，即得两点距离公式，这是坐标几何初步的第一个核心公式。

**直接延伸——三角比**：在直角三角形中定义 $\sin\theta = \frac{a}{c}$，$\cos\theta = \frac{b}{c}$，则由 $a^2 + b^2 = c^2$ 两边除以 $c^2$ 立刻得到：

$$\sin^2\theta + \cos^2\theta = 1$$

这是三角恒等式体系的根基公式。
