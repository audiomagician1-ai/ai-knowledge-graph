# 角与角度

## 概述

角（angle）是由两条射线从同一端点出发所形成的几何图形。这个共同的端点称为**顶点**（vertex），两条射线称为角的**两边**（sides）。从历史上看，古巴比伦人约在公元前2000年就将圆分为360等份，建立了沿用至今的"度"制度量体系——选择360这个数字，是因为它有24个因数，非常便于整除计算。

角度（degree，符号"°"）是度量角大小的标准单位。一个完整圆周等于360°，由此推出直角为90°、平角为180°、周角为360°。这套体系与日常生活密切相关：时钟每小时时针转过30°（360°÷12），分针每分钟转过6°（360°÷60），建筑工人用45°斜角制作标准拼接，这些都是角度在现实中的直接体现。

理解角的分类与度量，是进入三角形面积计算、尺规作图以及后续弧度制学习的必要前提。弧度制中1弧度约等于57.3°，这一换算本身就需要对"角度"有精确认识才能理解。

## 核心原理

### 角的分类

按照度数大小，角可以精确分为以下类型：

| 类型 | 度数范围 | 特征 |
|------|----------|------|
| 零角 | 0° | 两边重合 |
| 锐角（acute angle） | 0° < α < 90° | 小于直角 |
| 直角（right angle） | α = 90° | 两边互相垂直，用"□"标注 |
| 钝角（obtuse angle） | 90° < α < 180° | 大于直角小于平角 |
| 平角（straight angle） | α = 180° | 两边成一条直线 |
| 优角（reflex angle） | 180° < α < 360° | 大于平角 |
| 周角（full angle） | α = 360° | 两边重合一圈 |

直角是最特殊的基准角——不大不小，正好是平角的一半。判断两条直线是否垂直，就看它们所成的角是否为90°。

### 互余与互补

**互余（complementary）**：两角之和恰好等于90°，则互称余角。例如35°与55°互余，因为35°+55°=90°。余角关系常出现在直角三角形的两个锐角之间——直角三角形中两锐角之和必为90°，这是其必然性质。

**互补（supplementary）**：两角之和恰好等于180°，则互称补角。例如120°与60°互补，因为120°+60°=180°。两条直线相交时，相邻两角互为补角；而对顶角相等（均为"互补的补角"），这是由互补关系直接推导的结论。

一个重要性质：**同一角的余角唯一，补角唯一**。若∠A的余角为∠B，∠A的余角也为∠C，则∠B=∠C，均等于(90°−∠A)。这一性质在几何证明中频繁用于角度的传递推导。

### 角的度量工具与计算

量角器（protractor）是度量角的标准工具，其刻度从0°到180°排列，分内外两圈（方便从左或从右量角）。使用时，量角器中心对准顶点，基准线对齐一条边，另一条边所指的刻度即为角的度数。

角度的加减运算遵循60进制换算规则：
- 1° = 60′（角分），1′ = 60″（角秒）
- 例：37°45′ + 28°30′ = 66°15′（45+30=75′=1°15′，进位后37+28+1=66°）

这种60进制源自古巴比伦的六十进位法，与时间的小时—分钟—秒计量系统同源。

## 实际应用

**建筑与工程**：坡屋顶的坡度角通常在20°~45°之间，低于15°排水不畅，高于60°施工困难。工程图纸标注"45°倒角"，指的是切削出一个两边均为45°的锐角斜面，这利用了45°是直角的精确等分。

**钟表问题**：经典数学题"几点整时时针与分针所成的角度是多少"，需要精确使用角度计算。以3点整为例，时针指向3，分针指向12，时针走了圆周的3/12=1/4，即90°，两针夹角恰好是直角。

**折纸与平铺**：正六边形的内角均为120°，是大于90°的钝角，因此三个正六边形可以拼合在一起（120°×3=360°），实现无缝铺地。而正五边形内角为108°，三个拼合为324°<360°，留有缝隙，无法单独密铺。

**视角问题**：人眼的水平视野约为180°（双眼合计），单眼舒适视区约60°~90°，这直接影响显示器的摆放角度要求。

## 常见误区

**误区1：认为"平角就是一条直线，不算角"**
平角确实看起来像一条直线，但它是一个完整的角，度数为180°，有明确的顶点和两条反向射线。混淆这一点会导致在计算"一条直线上的角度关系"时出错。直线上某点两侧各形成的角，相加必须等于180°（平角），而非360°。

**误区2：互余互补是两个角的性质，不能说"一个角自己与自己互补"**
互余和互补必须是**两个不同角**之间的关系。45°的余角是另一个45°，而不能说"45°自己互余"。在解题中，常见错误是把"∠A的补角"误写成"180°−∠A自身"而忘记它指的是另一个角的度数。

**误区3：钝角三角形中两个锐角之和等于180°**
这是错误的。任意三角形的三个内角之和等于180°（这是三角形内角和定理，与角的互余互补不同）。钝角三角形中一个角大于90°，其余两角均为锐角，且两锐角之和小于90°，而非互余的90°。

## 知识关联

**前置概念——点线面基础**：射线（ray）的概念是定义角的基础。角由两条射线构成，射线有方向和端点，这与线段（有两端点）、直线（无端点）严格区分。没有射线概念，"角的两边从顶点出发"这一定义就无法精确表述。

**后续概念——三角形**：三角形的三个内角之和等于180°，直接运用了平角的度数定义。等腰三角形的两底角相等、直角三角形两锐角互余，这些性质都以角的互余互补关系为工具展开推导。

**后续概念——尺规作图**：用圆规和直尺平分一个角（角平分线作法）、作60°角、作直角，都要求对角的度数有精确认识，并利用互余互补判断所作角是否正确。

**后续概念——弧度制**：弧度制以弧长与半径之比定义角度，1 rad = 180°/π ≈ 57.2958°。从度制切换到弧度制，需要熟练掌握90°、180°、360°等关键角度，才能正确换算π/2、π、2π等常用弧度值。
