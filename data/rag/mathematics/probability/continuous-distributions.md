# 连续分布

## 概述

连续分布是指随机变量 $X$ 可以取某一区间（甚至全体实数）上任意值的概率分布。与离散分布不同，连续随机变量取任意单一精确值的概率为零，即 $P(X = c) = 0$，概率只能通过区间来度量：$P(a \leq X \leq b) = \int_a^b f(x)\,dx$，其中 $f(x)$ 称为**概率密度函数（PDF）**。这是连续分布与离散分布最根本的区别。

连续分布的数学基础在17至19世纪逐步建立。正态分布由棣莫弗（Abraham de Moivre）于1733年首次描述，高斯（Gauss）在误差分析中系统发展了它，指数分布则随泊松过程理论在20世纪初得到精确刻画。连续分布的严格定义依赖勒贝格测度理论，但工程和应用概率中主要使用黎曼积分框架。

连续分布在实际中描述身高、温度、等待时间、测量误差等自然量，是统计推断、工程可靠性分析和机器学习中概率模型的基础工具。三类最重要的连续分布——均匀分布、正态分布、指数分布——覆盖了从"完全无偏"到"对称钟形"再到"无记忆衰减"的典型概率形态。

## 核心原理

### 概率密度函数的基本条件

一个函数 $f(x)$ 成为合法概率密度函数，必须同时满足两个条件：

1. **非负性**：$f(x) \geq 0$，对所有 $x$ 成立
2. **归一性**：$\int_{-\infty}^{+\infty} f(x)\,dx = 1$

任何区间 $[a,b]$ 上的概率就是密度函数在该区间的定积分。**累积分布函数（CDF）**定义为 $F(x) = P(X \leq x) = \int_{-\infty}^{x} f(t)\,dt$，由微积分基本定理可得 $f(x) = F'(x)$，这正是连续分布需要定积分作为先修知识的原因。

### 均匀分布 $U(a, b)$

均匀分布描述随机变量在区间 $[a,b]$ 上等可能分布的情形，其密度函数为：

$$f(x) = \frac{1}{b-a}, \quad a \leq x \leq b$$

区间外 $f(x) = 0$。均匀分布是所有连续分布中形式最简单的，其 CDF 为 $F(x) = \frac{x-a}{b-a}$（在 $[a,b]$ 内）。一个典型实例：若公共汽车每10分钟一班，乘客到站时间随机，等待时间服从 $U(0,10)$，等待超过7分钟的概率为 $\frac{10-7}{10} = 0.3$，无需积分直接由几何意义得出。

### 正态分布 $N(\mu, \sigma^2)$

正态分布是连续分布中最重要的一种，其密度函数为：

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} \exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right), \quad -\infty < x < +\infty$$

参数 $\mu$ 控制分布中心（均值），$\sigma^2$ 控制分布宽度（方差）。当 $\mu=0, \sigma=1$ 时称为**标准正态分布** $N(0,1)$，其 CDF 记为 $\Phi(x)$，无封闭解析式，需查表或数值计算。

正态分布的**68-95-99.7法则**是重要工具：约68%的概率落在 $[\mu-\sigma, \mu+\sigma]$，约95%落在 $[\mu-2\sigma, \mu+2\sigma]$，约99.7%落在 $[\mu-3\sigma, \mu+3\sigma]$。任意正态分布可通过**标准化变换** $Z = \frac{X-\mu}{\sigma}$ 转化为 $N(0,1)$，从而利用标准正态分布表计算概率。

### 指数分布 $\text{Exp}(\lambda)$

指数分布描述泊松过程中事件间隔时间，参数 $\lambda > 0$ 称为速率（rate）。其密度函数为：

$$f(x) = \lambda e^{-\lambda x}, \quad x \geq 0$$

其 CDF 为 $F(x) = 1 - e^{-\lambda x}$，具有封闭解析式，计算十分方便。指数分布最重要的特性是**无记忆性**（Memoryless Property）：

$$P(X > s+t \mid X > s) = P(X > t)$$

这表示"已经等待了 $s$ 时间"不会改变未来等待时间的分布，指数分布是连续分布中唯一具有无记忆性的分布。实例：若某电子元件寿命服从 $\text{Exp}(0.01)$（即平均寿命100小时），则元件在已运行50小时后，再运行超过100小时的概率与新元件完全相同。

## 实际应用

**正态分布**在质量控制中用于描述产品尺寸误差：若某零件标称长度10cm，加工误差服从 $N(10, 0.01)$（即 $\sigma=0.1$cm），则超出公差 $\pm 0.2$cm（即 $2\sigma$）的废品率约为 $1 - 0.9545 = 4.55\%$。

**指数分布**在网络工程中建模数据包到达间隔时间和服务器响应时间。若某服务器平均每分钟处理60个请求（$\lambda = 60$），则两次请求间隔超过0.1分钟的概率为 $e^{-60 \times 0.1} = e^{-6} \approx 0.0025$。

**均匀分布**在随机数生成中是基础：计算机生成的伪随机数本质上服从 $U(0,1)$，再通过**逆变换法**（$X = F^{-1}(U)$）可模拟任意其他连续分布，这是蒙特卡洛模拟的核心技术。

## 常见误区

**误区1：认为 $f(x)$ 直接表示概率**  
概率密度函数 $f(x)$ 的值可以超过1（例如 $U(0, 0.5)$ 的密度值为2），它本身不是概率。概率必须通过对 $f(x)$ 积分得到。混淆密度与概率是初学者最常犯的错误。

**误区2：认为正态分布在所有对称情形下都适用**  
正态分布虽然对称，但具有特定的"轻尾"特征（峰度为3）。实际数据如金融收益率通常呈现**厚尾（Fat Tail）**，用正态分布会严重低估极端事件概率。例如2008年金融危机中，若假设收益率服从正态分布，"6σ事件"的理论概率约为 $10^{-9}$，实际却频繁发生。

**误区3：将指数分布的无记忆性误解为"独立性"**  
无记忆性是关于同一随机变量条件概率的性质，并非说不同时间段的等待相互独立。例如放射性衰变的等待时间具有无记忆性，但这不意味着可以通过历史数据预测下次衰变——正是无记忆性保证了这一不可预测性。

## 知识关联

学习连续分布需要**随机变量**的概念作为基础，才能理解"随机变量的分布"指什么；需要**定积分**的计算技能，因为所有连续概率计算本质上都是积分运算，$P(a \leq X \leq b) = \int_a^b f(x)\,dx$。

掌握连续分布后，下一步是计算**期望与方差**：连续随机变量的期望定义为 $E[X] = \int_{-\infty}^{+\infty} x f(x)\,dx$，这与离散情形的求和完全类比但必须用积分实现。三种分布的期望有简洁结论：$U(a,b)$ 的期望为 $\frac{a+b}{2}$，$N(\mu, \sigma^2)$ 的期望为 $\mu$，$\text{Exp}(\lambda)$ 的期望为 $\frac{1}{\lambda}$。

**正态分布**作为后续的独立专题，将深入讨论中心极限定理为何使正态分布在统计中具有普遍性——这是连续分布体系中最深刻的理论结果。
