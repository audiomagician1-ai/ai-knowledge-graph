# 连续分布

## 概述

连续分布是描述连续型随机变量概率行为的数学工具。与离散分布不同，连续型随机变量可以取某个区间内的任意实数值，因此无法用"某点的概率"来描述，而必须用**概率密度函数（PDF, Probability Density Function）** $f(x)$ 来刻画其分布规律。对连续分布而言，单个点的概率恒为零，即 $P(X = c) = 0$，只有区间上的概率才有意义：$P(a \leq X \leq b) = \int_a^b f(x)\,dx$。

连续分布的概念随着17至18世纪积分理论的发展而逐步成形。正态分布的钟形曲线由棣莫弗（Abraham de Moivre）于1733年首次描述，后经高斯（Carl Friedrich Gauss）系统化，用于描述测量误差。指数分布则在20世纪随排队论和可靠性工程的兴起而得到广泛应用。均匀分布是最简单的连续分布，其历史可追溯到几何概型。

连续分布在统计推断、物理学、工程学和金融学中均有核心地位。例如，人的身高、产品寿命、等待时间、测量误差等现实量，分别对应正态、指数、均匀等不同的连续分布模型。

## 核心原理

### 概率密度函数的基本性质

概率密度函数 $f(x)$ 必须满足两个条件：
1. **非负性**：$f(x) \geq 0$，对所有 $x$ 成立；
2. **归一性**：$\int_{-\infty}^{+\infty} f(x)\,dx = 1$。

由此定义**累积分布函数（CDF）**：$F(x) = P(X \leq x) = \int_{-\infty}^{x} f(t)\,dt$，它是 $f(x)$ 的原函数，且 $F'(x) = f(x)$（在 $f$ 连续处成立）。这正是连续分布与定积分知识深度结合的体现——求概率本质上就是计算定积分。

### 均匀分布 $U(a, b)$

区间 $[a, b]$ 上的均匀分布，其概率密度函数为：

$$f(x) = \begin{cases} \dfrac{1}{b-a}, & a \leq x \leq b \\ 0, & \text{其他} \end{cases}$$

这意味着变量落在等长子区间的概率相等。均匀分布的CDF为 $F(x) = \dfrac{x-a}{b-a}$（$a \leq x \leq b$）。例如，$U(0, 1)$ 的密度函数为常数 $1$，其期望为 $0.5$，方差为 $\dfrac{1}{12}$。均匀分布在随机数生成和蒙特卡洛仿真中是基础工具，所有其他连续分布均可通过 $U(0,1)$ 用逆变换法生成。

### 正态分布 $N(\mu, \sigma^2)$

正态分布是连续分布中最重要的模型，其概率密度函数为：

$$f(x) = \frac{1}{\sqrt{2\pi}\,\sigma} \exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right), \quad -\infty < x < +\infty$$

其中 $\mu$ 为均值，$\sigma^2$ 为方差。当 $\mu=0, \sigma=1$ 时称为**标准正态分布** $N(0,1)$，其CDF记为 $\Phi(x)$，满足 $\Phi(-x) = 1 - \Phi(x)$。正态分布具有对称性和"68-95-99.7法则"：$\mu \pm \sigma$ 内含约68.27%的概率，$\mu \pm 2\sigma$ 内含约95.45%，$\mu \pm 3\sigma$ 内含约99.73%。任意正态变量可通过标准化变换 $Z = \dfrac{X - \mu}{\sigma}$ 转化为标准正态变量，从而查表计算概率。

### 指数分布 $\text{Exp}(\lambda)$

指数分布用于描述泊松过程中两次事件之间的等待时间，其概率密度函数为：

$$f(x) = \begin{cases} \lambda e^{-\lambda x}, & x \geq 0 \\ 0, & x < 0 \end{cases}$$

其中 $\lambda > 0$ 称为速率参数（单位时间内的平均事件数），均值为 $\dfrac{1}{\lambda}$，方差为 $\dfrac{1}{\lambda^2}$。指数分布具有独特的**无记忆性**：$P(X > s+t \mid X > s) = P(X > t)$，即已经等待了 $s$ 时间后，再等 $t$ 时间的概率与刚开始时相同。在连续分布中，只有指数分布具备这一性质（离散版对应几何分布）。

## 实际应用

**正态分布应用——质量控制**：某工厂生产螺栓，直径服从 $N(10\text{mm},\, 0.04\text{mm}^2)$（即 $\sigma = 0.2\text{mm}$），规格要求直径在 $9.6\text{mm}$ 到 $10.4\text{mm}$ 之间。将界限标准化：$Z = \dfrac{10.4-10}{0.2} = 2$，合格率为 $P(-2 \leq Z \leq 2) = 2\Phi(2)-1 \approx 95.44\%$。

**指数分布应用——设备寿命**：某电子元件的寿命 $T \sim \text{Exp}(0.01)$，平均寿命为 $100$ 小时。求该元件能正常工作超过 $200$ 小时的概率：$P(T > 200) = e^{-0.01 \times 200} = e^{-2} \approx 0.1353$，约 $13.53\%$。

**均匀分布应用——公交等待**：某公交车每10分钟一班，乘客随机到达，等待时间 $W \sim U(0, 10)$。等待超过6分钟的概率为 $P(W > 6) = \dfrac{10-6}{10} = 0.4$，即40%。

## 常见误区

**误区1：认为 $f(x)$ 直接表示概率，且 $f(x) \leq 1$**
$f(x)$ 是概率密度，不是概率本身，它可以大于1。例如，$U(0, 0.5)$ 的密度函数值为2，这完全合法，因为密度值乘以区间长度才得到概率。区间长度趋近于零，概率也趋近于零，所以 $P(X = c) = 0$ 对所有连续分布成立。

**误区2：正态分布与指数分布混用于等待时间**
正态分布要求数据关于均值对称，而等待时间、寿命等非负且右偏的数据更适合指数分布（或其推广形式Gamma分布）。若将服从指数分布的寿命数据强行用正态分布建模，会出现负值区域被赋予正概率的荒谬结果。

**误区3：均匀分布是"没有规律"的分布**
均匀分布描述的是"等可能性"，是一种高度结构化的分布，而非随机或无规律。真正的"无规律"意味着完全不确定，而均匀分布给出了明确的、在有限区间上等权重的概率结构。蒙特卡洛模拟中，$U(0,1)$ 正是因其结构的规则性才被用作其他分布的生成基础。

## 知识关联

**先修知识——随机变量**：理解连续分布必须先掌握连续型随机变量的定义，即取值为不可数集合（通常是实数区间）的变量。连续型与离散型随机变量的根本区别在于求和与积分的替换：离散分布用 $\sum$ 求概率，连续分布用 $\int$ 求概率。

**先修知识——定积分**：连续分布中所有概率计算、期望计算均依赖定积分。$P(a \leq X \leq b) = \int_a^b f(x)\,dx$ 直接对应黎曼积分的几何意义——曲线下面积。没有定积分的计算能力，连续分布将止步于形式理解。

**后续概念——期望与方差**：有了连续分布的密度函数，期望定义为 $E(X) = \int_{-\infty}^{+\infty} x f(x)\,dx$，方差为 $D(X) = \int_{-\infty}^{+\infty} (x-\mu)^2 f(x)\,dx$。这两个数字特征将三种分布的"中心"与"散布"程度定量化，是连续分布参数估计和假设检验的基础。

**后续概念——正态分布的深化**：正态分布在三种基础连续分布中拥有最丰富的理论延伸，包括多元正态分布、中心极限定理（说明大量独立随机变量之和趋近于正态分布）以及线性变换的封闭性（正态变量的线性组合仍为正态分布）。这些性质使正态分布在统计学中占据极其特殊的地位。
