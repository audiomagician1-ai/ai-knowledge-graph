# 条件概率

## 概述

条件概率描述的是在**已知某事件发生的前提下**，另一事件发生的概率。给定事件 B 已发生，事件 A 的条件概率记作 P(A|B)，读作"在 B 发生的条件下 A 的概率"。这一概念的本质是将样本空间从完整的 Ω **压缩**到 B 所对应的子空间，再在这个新空间中重新计算 A 的可能性。

条件概率的严格定义由 18 世纪的概率理论逐渐形成，直到 1933 年，苏联数学家柯尔莫哥洛夫（Kolmogorov）在其公理化体系中给出了现代标准定义。他将条件概率作为一个派生量，而非基本公理，明确规定只要 P(B) > 0，便可通过如下公式计算：

$$P(A|B) = \frac{P(A \cap B)}{P(B)}$$

条件概率之所以重要，是因为现实世界中的随机事件很少在真空中发生——天气预报、医学诊断、金融风控都需要在掌握部分信息的前提下更新概率判断。掌握条件概率是理解一切"信息更新"类推理的前提。

## 核心原理

### 定义公式与几何直觉

条件概率的计算公式为：

$$P(A|B) = \frac{P(A \cap B)}{P(B)}, \quad \text{其中 } P(B) > 0$$

其中 P(A ∩ B) 是 A 与 B 同时发生的概率，P(B) 是条件所依赖的已知事件的概率。从韦恩图的角度理解：整个样本空间 Ω 缩小为 B，而我们关注的是在 B 的"面积"中，A ∩ B 所占的比例。这一"样本空间收缩"的直觉是条件概率区别于无条件概率的根本特征。

注意 P(A|B) 与 P(B|A) **一般不相等**。例如设 A = "骰子点数为偶数"，B = "骰子点数大于3"，则 P(A|B) = 1/3（{4,6}在{4,5,6}中），而 P(B|A) = 2/3（{4,6}在{2,4,6}中）。混淆两者是典型错误。

### 乘法公式

由条件概率定义直接变形，可得**乘法公式**（也称乘法法则）：

$$P(A \cap B) = P(B) \cdot P(A|B) = P(A) \cdot P(B|A)$$

这一公式将联合概率分解为"条件"与"边缘"的乘积，便于逐步计算多步随机过程。乘法公式可推广至多个事件的链式形式：

$$P(A_1 \cap A_2 \cap \cdots \cap A_n) = P(A_1) \cdot P(A_2|A_1) \cdot P(A_3|A_1 \cap A_2) \cdots P(A_n|A_1 \cap \cdots \cap A_{n-1})$$

例如，从一副52张扑克牌中不放回地连抽两张黑桃，第一张抽到黑桃的概率为 13/52，第二张抽到黑桃的条件概率为 12/51，因此两张都是黑桃的概率为 (13/52) × (12/51) = 156/2652 ≈ 5.88%。

### 条件概率本身构成概率测度

P(·|B) 满足概率的三条柯尔莫哥洛夫公理：非负性、归一性（P(Ω|B) = 1）、以及可列可加性。这意味着所有针对无条件概率成立的定理（如全概率公式的扩展形式）在条件概率框架下同样适用。这一特性使得条件概率不仅仅是一个计算工具，而是一套完整的概率体系，可以在其上继续定义条件期望、条件分布等更高层次的概念。

## 实际应用

**医学诊断中的阳性预测值**：设某疾病患病率 P(D) = 0.01（1%），某检测方法的灵敏度（真阳性率）P(+|D) = 0.95，特异度（真阴性率）P(−|¬D) = 0.90。问检测阳性者真正患病的概率 P(D|+) 是多少？利用乘法公式先计算 P(D ∩ +) = 0.01 × 0.95 = 0.0095，再计算 P(+) = 0.0095 + 0.09 × 0.99 ≈ 0.0986，最终得到 P(D|+) ≈ 9.6%——远低于直觉上的"95%"，这正是条件概率与乘法公式联合分析的典型场景。

**条件概率树形图**：在质量控制中，假设生产线 A 生产了 40% 的产品，次品率 2%；生产线 B 生产了 60% 的产品，次品率 5%。树形图中每条路径代表一个联合概率，P(A ∩ 次品) = 0.4 × 0.02 = 0.008，P(B ∩ 次品) = 0.6 × 0.05 = 0.030。这种分解方式是全概率公式的直接来源，也展示了乘法公式在多分支情形下的自然延伸。

## 常见误区

**误区一：将 P(A|B) 误读为 P(B|A)**  
这是所谓的"检察官谬误"（Prosecutor's Fallacy）：法庭上有人错误地将"无辜者留下该证据的概率极低"等同于"留下该证据者无辜的概率极低"。即混淆 P(证据|无辜) 与 P(无辜|证据)。两者由贝叶斯定理联系，但在先验概率不为1/2时必然不相等。

**误区二：忽略 P(B) > 0 的前提条件**  
条件概率公式在 P(B) = 0 时无定义——分母为零。在连续概率分布中，单点事件的概率为零（如"恰好等于某值"），此时定义条件概率需要用到更高级的测度论工具（正则条件概率），不能直接套用基本公式。初学者常在此处犯下将零概率事件作为条件的错误。

**误区三：认为条件概率总是小于原概率**  
P(A|B) 与 P(A) 的大小关系取决于 A 与 B 的关联方向。若 B 的发生对 A 有利，则 P(A|B) > P(A)；若 B 与 A 独立，则 P(A|B) = P(A)；若 B 的发生排除了 A 的部分可能性，才有 P(A|B) < P(A)。例如，已知"掷骰子结果为偶数"，则"结果为6"的概率从1/6**提升**到1/3。

## 知识关联

**前置知识**：条件概率以"概率基础"中的样本空间 Ω、事件的集合运算（交、并、补）和基本概率公理为基础。若不理解 P(A ∩ B) 的含义，则无法理解条件概率分子的来源。

**直接延伸——贝叶斯定理**：将乘法公式 P(A ∩ B) = P(B|A)·P(A) 代入条件概率定义，得到贝叶斯定理 P(A|B) = P(B|A)·P(A)/P(B)。贝叶斯定理本质上是条件概率公式的"翻转"应用，专门用于在观察到结果 B 后反推原因 A 的概率。

**事件的独立性**：独立性被精确定义为 P(A ∩ B) = P(A)·P(B)，等价于 P(A|B) = P(A)（当 P(B) > 0 时）。这说明独立性是条件概率等于无条件概率的特殊情形，而非凭直觉判断的"互不影响"。

**条件期望**：在掌握条件概率 P(X = x | Y = y) 的离散分布后，可自然引入条件期望 E[X|Y]，它是对随机变量 X 在已知 Y 值后的平均预测，是条件概率概念向期望计算领域的推广。
