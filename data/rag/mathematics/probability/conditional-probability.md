# 条件概率

## 概述

条件概率描述的是在**已知某一事件发生的前提下**，另一事件发生的概率。与无条件概率不同，条件概率将样本空间从全集 Ω 压缩到了已知事件所定义的子集上，从而改变了各事件的相对可能性。形式上，事件 A 在事件 B 已发生条件下的条件概率记作 P(A|B)，读作"在 B 发生的条件下 A 的概率"。

条件概率的概念由英国数学家托马斯·贝叶斯（Thomas Bayes，1701—1761）在其遗作《论机会学说中一个问题的解》（1763年发表）中隐含地使用，后经皮埃尔-西蒙·拉普拉斯在19世纪初系统整理，形成了现代概率论的重要基石。柯尔莫哥洛夫（Kolmogorov）1933年发表的《概率论基础》给出了严格的测度论定义，将条件概率纳入公理化体系。

条件概率的重要性在于：现实世界中的概率判断几乎总是在"已知部分信息"的情况下进行的。医学诊断中，医生拿到检验结果后才评估患病概率；气象预报中，当前气压和湿度数据是给出降雨概率的前提条件。忽视已知信息、直接使用无条件概率，会导致严重的决策偏差。

## 核心原理

### 定义公式

设 P(B) > 0，则事件 A 在事件 B 条件下的条件概率定义为：

$$P(A \mid B) = \frac{P(A \cap B)}{P(B)}$$

其中：
- P(A ∩ B) 是 A 与 B 同时发生的概率（联合概率）；
- P(B) 是已知条件事件的概率，作为"新的归一化因子"；
- 要求 P(B) > 0，否则条件概率无意义（不能以概率为零的事件为条件）。

几何直觉上，该公式将整个样本空间"缩放"到 B 的范围内，A 在新空间中的占比即为 P(A|B)。例如，掷一颗均匀骰子，A = {出现偶数} = {2,4,6}，B = {出现大于3的点} = {4,5,6}，则 P(A ∩ B) = P({4,6}) = 2/6，P(B) = 3/6，所以 P(A|B) = (2/6)/(3/6) = 2/3。

### 乘法公式

由条件概率的定义可直接推导出**乘法公式（积事件概率公式）**：

$$P(A \cap B) = P(B) \cdot P(A \mid B) = P(A) \cdot P(B \mid A)$$

乘法公式可推广到 n 个事件的情形（链式法则）：

$$P(A_1 \cap A_2 \cap \cdots \cap A_n) = P(A_1) \cdot P(A_2 \mid A_1) \cdot P(A_3 \mid A_1 \cap A_2) \cdots P(A_n \mid A_1 \cap \cdots \cap A_{n-1})$$

这一链式展开在马尔可夫链、语言模型（如 n-gram 模型）中被广泛使用——一段文字的概率被分解为每个词在前序词语条件下的条件概率之积。

### 条件概率满足概率公理

固定条件事件 B 后，P(·|B) 本身是一个合法的概率测度，满足柯尔莫哥洛夫三条公理：
1. **非负性**：P(A|B) ≥ 0；
2. **归一性**：P(Ω|B) = P(Ω ∩ B)/P(B) = P(B)/P(B) = 1；
3. **可列可加性**：若 A₁, A₂, … 互不相容，则 P(⋃Aᵢ|B) = ΣP(Aᵢ|B)。

这意味着概率论中对于普通概率成立的定理（如全概率公式）在条件概率体系下依然成立，只需将背景从 Ω 替换为 B 即可。

## 实际应用

**医学检验的假阳性问题**：某疾病患病率（先验概率）为 0.1%，检测试剂灵敏度 P(阳性|患病) = 99%，特异度 P(阴性|未患病) = 95%。利用条件概率定义和全概率公式可计算 P(阳性) ≈ 0.001×0.99 + 0.999×0.05 ≈ 0.05094。再由乘法公式反推，P(患病|阳性) = P(患病∩阳性)/P(阳性) ≈ 0.000990/0.05094 ≈ 1.94%。直觉上大多数人以为阳性意味着接近99%的患病率，但条件概率的严格计算揭示了真实概率不足2%，这是低发病率背景下假阳性主导结果的典型案例。

**摸球不放回问题**：袋中有3个红球、2个蓝球，连续无放回抽取两球。第一次取红球的概率 P(R₁) = 3/5，在此条件下第二次仍取红球的概率 P(R₂|R₁) = 2/4 = 1/2，因为第一次取走后袋中只剩4球、2个红球。由乘法公式，P(R₁ ∩ R₂) = (3/5)×(1/2) = 3/10。若放回则两次独立，P(R₁ ∩ R₂) = (3/5)² = 9/25，两者结果不同，体现了条件概率在"历史信息改变未来状态"场景中的核心作用。

**自然语言处理中的二元语言模型**：P("机器学习") = P("机器") × P("学习"|"机器")，通过对大规模语料统计 P("学习"|"机器") = 出现"机器学习"的次数 / 出现"机器"的次数，即可用乘法公式估算词序列的联合概率，这是经典 n-gram 语言模型的直接应用。

## 常见误区

**误区一：混淆 P(A|B) 与 P(B|A)**

P(A|B) 与 P(B|A) 不仅不相等，甚至可以相差悬殊。以检验问题为例，P(阳性|患病) = 99% 与 P(患病|阳性) ≈ 1.94% 之间差了50倍。这种混淆被称为"条件概率倒置谬误"（Prosecutor's Fallacy），曾在法庭案件中导致冤假错案。正确转换 P(A|B) 与 P(B|A) 需要用到贝叶斯公式：P(B|A) = P(A|B)·P(B) / P(A)，不可凭直觉互换。

**误区二：将条件概率等同于因果关系**

P(A|B) > P(A) 仅表示事件 B 的发生对 A 的概率有"相关性"提示，不代表 B 导致了 A。例如，P(溺亡|购买冰淇淋) > P(溺亡) 在夏季数据中成立，原因是共同的混杂变量（高温天气）同时促进了游泳和购买冰淇淋，条件概率无法直接推断因果机制。

**误区三：忽略条件 P(B) > 0 的限制**

当 P(B) = 0 时，定义式 P(A∩B)/P(B) 出现除以零的问题，条件概率在经典定义下无法成立。连续型随机变量中，P(X = x) = 0 对任意单点成立，此时"给定 X = x 的条件概率"需要用条件密度函数来重新定义，不能直接套用离散情形的公式。初学者常忽略这一点，在连续变量场景下错误地使用离散定义。

## 知识关联

**前置概念——概率基础**：条件概率的计算依赖对 P(A∩B) 和 P(B) 的正确求解，因此需要熟练掌握古典概型、几何概型及集合运算（并、交、补）下的概率计算规则。

**后续概念——贝叶斯定理**：贝叶斯公式 P(B|A) = P(A|B)·P(B)/P(A) 直接由条件概率定义和乘法公式推导而来，是条件概率"反向使用"的核心工具，用于由观测结果反推原因的概率（后验概率）。

**后续概念——事件的独立性**：若 P(A|B) = P(A)，即已知 B 发生不改变 A 的概率，则称 A 与 B 独立。独立性的定义等价于 P(A∩B) = P(A)·P(B)，这一等价关系直接来源于条件概率乘法公式的特殊情形。

**后续概念——条件期望**：将条件概率的思想从事件推广到随机变量，得到条件期望 E(X|Y=y)，即在已知随机变量 Y 取某值的条件下 X 的期望值。条件期望是条件概率在连续情形下的自然延伸，也是鞅论和随机过程的基础工具。
