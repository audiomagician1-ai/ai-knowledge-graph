# 概率基础

## 概述

概率论研究随机现象的数量规律，其核心对象是**样本空间**、**事件**与**概率**这三个层层递进的概念。样本空间 $\Omega$ 是一次随机试验所有可能结果的集合；事件是样本空间的子集；概率则是将事件映射到 $[0,1]$ 区间的一个函数。这三者共同构成了一次完整的概率模型描述框架。

概率的公理化体系由苏联数学家柯尔莫哥洛夫（Kolmogorov）于1933年在《概率论基础》中正式建立。在此之前，概率仅凭直觉与频率估计使用了近三个世纪。柯尔莫哥洛夫给出了三条公理：非负性（$P(A) \geq 0$）、规范性（$P(\Omega) = 1$）、可列可加性（互斥事件之并的概率等于各概率之和），从此概率论成为严格的数学分支。

**古典概型**（classical probability model）是最基础的概率计算模型，其成立需要两个前提：样本空间有限，且每个基本事件等可能发生。这一模型直接将概率计算转化为**计数问题**，从而与排列组合知识深度衔接。

## 核心原理

### 样本空间与事件的集合结构

样本空间 $\Omega$ 的元素称为**样本点**（基本事件）。例如，掷两枚骰子的样本空间共有 $6 \times 6 = 36$ 个样本点，形如 $(i, j)$，其中 $i, j \in \{1,2,3,4,5,6\}$。

事件之间的运算完全对应集合运算：
- **事件A与B同时发生** → 交集 $A \cap B$
- **事件A或B至少一个发生** → 并集 $A \cup B$
- **事件A不发生** → 补集 $\bar{A}$（或 $A^c$）
- **互斥事件**：$A \cap B = \emptyset$，即两事件不能同时发生

必然事件对应全集 $\Omega$，不可能事件对应空集 $\emptyset$，这两个极端事件的概率分别固定为1和0。

### 古典概型的概率公式

在古典概型条件下，事件 $A$ 的概率定义为：

$$P(A) = \frac{|A|}{|\Omega|} = \frac{\text{A 包含的基本事件数}}{\text{样本空间中基本事件总数}}$$

**例：** 从1到20的整数中随机取一个数，求该数既是偶数又是3的倍数的概率。  
- $|\Omega| = 20$  
- 既是偶数又是3的倍数，即6的倍数：$\{6, 12, 18\}$，共3个  
- $P(A) = \dfrac{3}{20}$

此类计算的核心难点在于分子的计数，需要灵活运用加法原理、乘法原理、组合数 $C_n^k = \dfrac{n!}{k!(n-k)!}$ 等工具。

### 概率的加法公式与容斥原理

对于任意两事件，加法公式为：

$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

当 $A$ 与 $B$ 互斥时，$P(A \cap B) = 0$，公式退化为 $P(A \cup B) = P(A) + P(B)$。

利用对立事件（complementary event）可以简化许多计算。若直接计算事件 $A$ 的概率复杂，可转而计算 $P(\bar{A})$，再利用：

$$P(A) = 1 - P(\bar{A})$$

**经典场景：** "至少有一个"类问题几乎总应使用对立事件方法。例如，10件产品中有3件次品，随机抽取4件，至少有1件次品的概率：$P = 1 - \dfrac{C_7^4}{C_{10}^4} = 1 - \dfrac{35}{210} = \dfrac{5}{6}$。

### 等可能性的判断

古典概型成立的关键前提——**等可能性**——不能想当然。例如，"某场比赛队A可能赢、可能输、可能平，共三种结果，所以各占1/3概率"是错误推理，因为三种结果并不等可能。等可能性需要物理或对称性依据（如均匀硬币、均匀骰子、不放回随机抽样等）才能成立。

## 实际应用

**抽签公平性：** 5张签中有2张中签，5人依次抽取，每人抽到中签的概率均为 $\dfrac{2}{5}$。这一结论可用古典概型严格证明：将5人的抽取顺序视为全排列，每种排列等可能，计算第 $k$ 个人抽到中签的比例恒为 $\dfrac{2}{5}$，与顺序无关。

**质检问题：** 一批100件产品中有5件次品，随机抽取10件，求恰好含2件次品的概率。样本空间大小为 $C_{100}^{10}$，事件中次品恰好2件的组合数为 $C_5^2 \cdot C_{95}^8$，故：
$$P = \frac{C_5^2 \cdot C_{95}^8}{C_{100}^{10}}$$

此类问题是古典概型与组合数计算的典型结合点，也是后续**超几何分布**（抽样方法中的无放回抽样模型）的直接前驱。

**密码破解概率：** 4位数字密码（每位0-9）被随机猜测，猜中的概率为 $\dfrac{1}{10^4} = 0.01\%$，若已知首位为非零数字，则样本空间缩减为 $9 \times 10^3 = 9000$，概率变为 $\dfrac{1}{9000}$。

## 常见误区

**误区一：将"等可能"与"有限"混淆**  
古典概型要求同时满足有限性和等可能性，缺一不可。掷一枚不均匀硬币，虽然结果有限（正/反两种），但概率不等，不能套用 $P = \dfrac{m}{n}$ 公式。

**误区二：计数时忽略元素的"区分性"**  
抽取时是否有序、是否放回，直接影响样本空间大小。从4人中选2人组成小组（无序），样本空间大小为 $C_4^2 = 6$；若是选正、副组长（有序），则为 $A_4^2 = 12$。分子、分母必须采用同一计数口径（同为有序或同为无序），否则结果错误。

**误区三：对立事件与互斥事件的混淆**  
互斥仅要求 $A \cap B = \emptyset$，并不要求 $A \cup B = \Omega$；而对立事件额外要求 $A \cup B = \Omega$，即二者恰好覆盖全部情况。事件"至少2人生日相同"与"至多1人生日相同"是对立事件；但"恰好2人生日相同"与"恰好3人生日相同"只是互斥事件，不是对立事件。

## 知识关联

**前置知识——组合数学：** 古典概型的计算直接依赖 $C_n^k$ 和 $A_n^k$ 的计算能力。若组合计数出错，概率结果必然错误，因此本模块是组合知识的直接应用场景。

**后续——条件概率：** 古典概型中"已知某信息后重新计算概率"的场景，会自然引出条件概率 $P(A|B) = \dfrac{P(A \cap B)}{P(B)}$。上述密码问题中"已知首位非零"正是条件概率的雏形。

**后续——随机变量：** 当我们不只满足于"某事件是否发生"，而是想用一个数值描述试验结果（如"掷骰子的点数"），就需要引入随机变量的概念，概率基础中样本空间与事件的结构是随机变量定义的载体。

**后续——抽样方法：** 质检问题中的不放回抽样模型，在统计学中进化为超几何分布；放回抽样则对应二项分布，两者均以古典概型的等可能抽样假设为前提。
