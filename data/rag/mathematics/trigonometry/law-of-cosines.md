# 余弦定理

## 概述

余弦定理是描述三角形三边长度与其中一个角的余弦值之间关系的基本公式。对于任意三角形 ABC，设三边长分别为 $a$、$b$、$c$（分别对应角 $A$、$B$、$C$ 的对边），则余弦定理表达为：$c^2 = a^2 + b^2 - 2ab\cos C$。该公式对锐角三角形、钝角三角形和直角三角形均成立，是处理"已知两边及其夹角"或"已知三边求角"类问题的核心工具。

余弦定理的历史可追溯至古希腊数学家欧几里得。在其著作《几何原本》（约公元前 300 年）的第二卷命题 12 和 13 中，欧几里得以几何语言描述了与余弦定理等价的命题，但尚未使用三角比形式。现代代数形式的余弦定理由法国数学家韦达（François Viète）等人在 16 世纪逐步整理成型，并与三角函数体系结合，形成今天教科书中的标准公式。

余弦定理的重要性在于它将勾股定理推广到一般三角形。当 $C = 90°$ 时，$\cos C = 0$，公式退化为 $c^2 = a^2 + b^2$，即勾股定理。因此勾股定理是余弦定理的特殊情形，而余弦定理填补了非直角三角形中边角关系的计算空白。

## 核心原理

### 公式推导：向量法

设三角形 ABC 的三顶点为位置向量，令 $\vec{CA} = \vec{b}$，$\vec{CB} = \vec{a}$，则 $\vec{AB} = \vec{a} - \vec{b}$，其模 $|\vec{AB}| = c$。

$$c^2 = |\vec{a} - \vec{b}|^2 = |\vec{a}|^2 - 2\vec{a}\cdot\vec{b} + |\vec{b}|^2 = a^2 + b^2 - 2ab\cos C$$

其中 $\vec{a}\cdot\vec{b} = ab\cos C$ 是两向量的点积定义。这一推导说明余弦定理本质上是向量模长公式在几何三角形中的直接体现。

### 三种对称形式

余弦定理对三角形三个角各有一个形式，三者完全对称：

$$a^2 = b^2 + c^2 - 2bc\cos A$$
$$b^2 = a^2 + c^2 - 2ac\cos B$$
$$c^2 = a^2 + b^2 - 2ab\cos C$$

使用时需注意：公式左侧的边 $c$ 对应的是**夹角** $C$ 所在的那一项，即 $2ab\cos C$ 中 $a$ 和 $b$ 是夹住角 $C$ 的两条边。混淆边与对应角是初学者最常见的错误。

### 反用公式：由三边求角

余弦定理可以变形，用于由已知三边长反推角度：

$$\cos C = \frac{a^2 + b^2 - c^2}{2ab}$$

这里 $a$、$b$、$c$ 必须满足三角不等式（任意两边之和大于第三边），否则分子绝对值大于 $2ab$，导致 $|\cos C| > 1$，无解。通过此式得到 $\cos C$ 后，因 $C \in (0°, 180°)$，余弦值在 $(-1, 1)$ 范围内单调递减，可以唯一确定角 $C$，不存在正弦定理中"大边对大角"判断的歧义问题。

## 实际应用

**测量不可达距离**：已知两点 $A$、$B$ 之间的距离无法直接测量，可选定可达点 $C$，分别测量 $CA = b = 350\,\text{m}$，$CB = a = 420\,\text{m}$，以及夹角 $\angle ACB = 60°$。代入公式：

$$AB^2 = 350^2 + 420^2 - 2\times350\times420\times\cos 60° = 122500 + 176400 - 147000 = 151900$$

得 $AB \approx 389.7\,\text{m}$。

**判断三角形形状**：对于三边分别为 $5$、$7$、$9$ 的三角形，计算最大角（最长边 $c=9$ 的对角 $C$）：

$$\cos C = \frac{25 + 49 - 81}{2\times5\times7} = \frac{-7}{70} = -0.1 < 0$$

故 $C > 90°$，该三角形为钝角三角形。

**导航与航空**：在航空导航中，飞机从 $A$ 点飞往 $B$ 点，中途偏航至 $C$ 点，已知 $AC = 200\,\text{km}$，$BC = 150\,\text{km}$，夹角为 $110°$，用余弦定理即可算出原航线距离 $AB$，辅助飞行员规划返航路径。

## 常见误区

**误区一：将夹角与对角混淆**

公式 $c^2 = a^2 + b^2 - 2ab\cos C$ 中，$C$ 是边 $a$ 与边 $b$ 的**夹角**，而不是边 $c$ 的某个端点角。学生常误写成 $c^2 = a^2 + b^2 - 2bc\cos A$（把 $A$ 代入 $C$ 的位置却仍用 $bc$），导致公式错误。正确记忆方式：$\cos$ 后面的角字母 $=$ 等号左侧边字母，而等号右侧相减项的两个边字母恰好是其余两边。

**误区二：认为余弦定理只适用于锐角三角形**

部分学生看到 $\cos C$ 后，误认为 $C$ 必须是锐角才能使用公式。实际上，余弦定理对任意角度 $C \in (0°, 180°)$ 均成立。当 $C$ 为钝角时，$\cos C < 0$，$-2ab\cos C > 0$，计算结果使 $c^2 > a^2 + b^2$，符合钝角三角形中对边最长的几何直觉。

**误区三：在"两边一角"情况下盲目使用正弦定理**

当已知两边及其**夹角**（SAS条件）时，必须使用余弦定理，而不能用正弦定理。若误用正弦定理，会因缺少对角信息而无法直接求解。只有已知两角及任意一边（AAS）或两边及其中一边的对角（SSA）时，正弦定理才更便捷。

## 知识关联

**前置知识**：余弦定理直接依赖**三角比**中余弦函数的定义（$\cos\theta = \text{邻边}/\text{斜边}$）及其在钝角时的扩展（$\cos\theta < 0$）；同时，**勾股定理**是余弦定理 $C=90°$ 时的特例，学生理解勾股定理后，余弦定理可视为其"带修正项"的推广，修正项 $-2ab\cos C$ 量化了角 $C$ 偏离直角的程度。

**后续延伸**：掌握余弦定理后，可进入**三角函数应用**，解决测量、工程中更复杂的实际问题，包括与正弦定理联合使用的综合解三角形问题。在更高层次的**球面三角**中，余弦定理推广为球面余弦公式：$\cos a = \cos b \cos c + \sin b \sin c \cos A$，其中 $a$、$b$、$c$ 表示球面三角形的三条弧边（以角度量度），该公式在天文学和大地测量学中用于计算球面上两点间的角距离。
