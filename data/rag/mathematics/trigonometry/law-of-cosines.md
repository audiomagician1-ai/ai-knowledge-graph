# 余弦定理

## 概述

余弦定理描述了三角形三边长度与其中一个角的余弦值之间的定量关系，核心公式为 **c² = a² + b² − 2ab·cosC**，其中 a、b、c 是三角形三边长，C 是边 c 所对应的角。该公式可以对 a 或 b 所对的角 A、B 对称地写出：a² = b² + c² − 2bc·cosA，b² = a² + c² − 2ac·cosB。

余弦定理的历史可追溯至欧几里得的《几何原本》（约公元前 300 年），第二卷命题 12 和 13 以几何语言分别描述了钝角三角形和锐角三角形的等价结论。现代代数形式由法国数学家韦达（François Viète）等人在 16 世纪逐步整理为方程形式。值得注意的是，当角 C = 90° 时，cos90° = 0，余弦定理退化为 c² = a² + b²，即勾股定理——这说明勾股定理是余弦定理在直角三角形情形下的特例。

余弦定理的重要性在于它将正弦定理无法单独解决的"两边夹角（SAS）"和"三边已知（SSS）"两类三角形求解问题纳入统一框架。在导航、测量、物理力学等领域中，已知两段距离及其夹角的情形极为普遍，余弦定理提供了直接的计算路径。

## 核心原理

### 代数推导

在坐标系中，将角 C 置于原点，令 A = (b, 0)，B = (a·cosC, a·sinC)。则边 c = AB 的长度平方为：

c² = (b − a·cosC)² + (a·sinC)²
   = b² − 2ab·cosC + a²cos²C + a²sin²C
   = a² + b² − 2ab·cosC

最后一步利用了三角恒等式 sin²C + cos²C = 1。整个推导只依赖坐标距离公式和基本三角恒等式，无需额外假设，对锐角、钝角和直角三角形均成立。

### 由余弦定理求角

余弦定理还可以改写为求角公式：

cosC = (a² + b² − c²) / (2ab)

给定三边长 a、b、c，代入即可求得角 C。例如三边为 3、4、5 时：cos C = (9 + 16 − 25)/(2·3·4) = 0/24 = 0，得 C = 90°，验证了这是直角三角形。再如三边为 5、7、8 时：cosA = (49 + 64 − 25)/(2·7·8) = 88/112 ≈ 0.786，A ≈ 38.2°。

### 与向量内积的联系

设向量 **u** 和 **v** 分别表示三角形的两条边，夹角为 C，则：

|**u** − **v**|² = |**u**|² + |**v**|² − 2|**u**||**v**|cosC

这正是余弦定理的向量形式。向量内积定义 **u**·**v** = |**u**||**v**|cosC，使得余弦定理与高维向量空间的夹角计算直接统一——这也是机器学习中"余弦相似度"命名的来源，其数学根源即在此。

### 适用情形分类

- **SAS（两边夹角）**：已知 a、b 和 C，直接用 c² = a² + b² − 2ab·cosC 求第三边。
- **SSS（三边已知）**：用 cosC = (a² + b² − c²)/(2ab) 求任意角，再用正弦定理或再次应用余弦定理求其余角。
- 已知两角一边（AAS/ASA）或已知两边非夹角（SSA）时，通常优先用正弦定理处理，余弦定理在这两种情形不是最优选择。

## 实际应用

**三角测量与土地勘测**：设测量员在 A、B 两点，测得 AB = 200 m，A 到目标 C 的距离 AC = 150 m，∠BAC = 55°。用余弦定理：BC² = 200² + 150² − 2·200·150·cos55° ≈ 40000 + 22500 − 34415 ≈ 28085，BC ≈ 167.6 m。全程无需实地测量 BC 距离。

**航海与航空导航**：已知出发点 A、中转点 B、目的地 C，以及 AB、AC 距离和夹角，飞行员可以用余弦定理计算 BC 的直线距离，从而评估绕行与直飞的时间差异。

**物理学中的力的合成**：两个力 F₁ = 40 N 和 F₂ = 30 N，夹角为 60°，合力大小为：F² = 40² + 30² − 2·40·30·cos(180°−60°) = 1600 + 900 + 1200 = 3700，F ≈ 60.8 N。注意此处补角转换是因为平行四边形法则中相关角为外角。

## 常见误区

**误区一：将 cosC 的符号方向搞反**。部分学生在钝角情形（C > 90°）中忘记 cosC 为负值，误将 −2ab·cosC 计算为负数相减，导致 c² 偏小。实际上当 C = 120° 时，cos120° = −0.5，−2ab·(−0.5) = +ab，使得 c² > a² + b²，符合直觉（钝角对边最长）。

**误区二：误用余弦定理于 SSA 情形**。已知两边和一个非夹角时，直接列方程 c² = a² + b² − 2ab·cosC 会引入未知角 C，方程中同时出现两个未知量，无法直接求解，此时应先用正弦定理确定角度再回头计算。

**误区三：认为三边必须满足特定比例余弦定理才成立**。余弦定理对任意满足三角不等式（a + b > c 等）的三边均成立，不要求边长为整数或满足特殊比例。出现 cosC > 1 或 cosC < −1 的结果，说明的是输入的三边数据不能构成真实三角形，而非公式有误。

## 知识关联

**前置知识的衔接**：余弦定理依赖**勾股定理**作为其 C = 90° 时的特例验证，也依赖**三角比**（特别是 cos 函数的定义域与值域：cosC ∈ [−1, 1] 对应 C ∈ [0°, 180°]）以确保求角公式有唯一解。

**通向正弦定理**：解完余弦定理求出某一边后，常转而用正弦定理 a/sinA = b/sinB = c/sinC 求其余角，二者配合可完整解决所有斜三角形问题。在"SSS 解三角形"中，建议先用余弦定理求最大角（即最长边对应的角），确认其是否为钝角，再用正弦定理处理其余两角，避免反正弦函数的双值问题。

**延伸至球面三角**：在球面三角学中，余弦定理推广为球面余弦定理：cosα = cosβ·cosγ + sinβ·sinγ·cosA，其中 α、β、γ 为球面三角形的三条边（以弧度表示的圆心角），A 为 α 所对的球面角。这一公式是航空大圆航线计算和天文学坐标转换的基础，结构与平面余弦定理高度类比但本质不同。
