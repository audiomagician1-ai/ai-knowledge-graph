# 正弦定理

## 概述

正弦定理描述了三角形中边与其对角正弦值之间的比例关系：在任意三角形ABC中，各边长与其对角的正弦值之比相等，且等于外接圆直径的两倍。其完整表达式为：

$$\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$$

其中 $a, b, c$ 分别为角 $A, B, C$ 的对边长，$R$ 为三角形外接圆半径。这一等比关系将三角形的几何量（边长）与角度量（三角比）直接联系起来。

正弦定理最早可追溯至古希腊数学家托勒密（约公元100—170年）的《天文学大成》，其中包含等价于正弦定理的弦长关系。伊斯兰数学家阿布·韦法（940—998年）在其三角学著作中给出了更明确的表述。欧洲数学家则在15至16世纪将其系统化，成为解三角形的标准工具。

正弦定理之所以重要，在于它适用于**一切三角形**，不限于直角三角形。当已知两角一边（AAS或ASA）或两边一对角（SSA）时，正弦定理提供了求解其余元素的直接途径，这是单纯依赖勾股定理所无法完成的。

## 核心原理

### 几何证明：外接圆联系

正弦定理最优雅的证明依赖外接圆。设三角形ABC的外接圆圆心为O，半径为R。在圆上取直径BC'，则弦BC = a。由圆的性质，同弧BC所对的圆心角为圆周角的两倍，即 $\angle BC'C = A$（同弧BC的圆周角）。在直角三角形BC'C中（BC'为直径，故 $\angle BCC' = 90°$），有：

$$\sin A = \sin \angle BC'C = \frac{BC}{BC'} = \frac{a}{2R}$$

因此 $a = 2R \sin A$，整理即得 $\dfrac{a}{\sin A} = 2R$。对边 $b, c$ 同理可证，三比值均等于 $2R$。这一证明揭示了正弦定理中常数 $2R$ 的几何本质：它是外接圆的直径，而非任意常数。

### 面积公式的衍生

由正弦定理可推导三角形面积公式。三角形面积 $S = \dfrac{1}{2}ab\sin C$。将 $a = 2R\sin A$，$b = 2R\sin B$ 代入，得：

$$S = 2R^2 \sin A \sin B \sin C$$

这一形式直接用三角形的三个内角表达面积，常用于竞赛题中化简含 $R$ 的表达式。

### 解三角形的两类情形

**情形一（AAS/ASA）**：已知两角一边，第三角由内角和 $A+B+C=180°$ 确定，其余两边由正弦定理直接计算。例如已知 $A=30°, B=45°, a=5$，则 $C=105°$，

$$b = \frac{a \sin B}{\sin A} = \frac{5 \times \frac{\sqrt{2}}{2}}{\frac{1}{2}} = 5\sqrt{2}$$

**情形二（SSA）**：已知两边一对角，设已知 $a, b, A$，由 $\sin B = \dfrac{b \sin A}{a}$ 求角B。此情形需判断解的个数：
- 若 $\sin B > 1$，无解；
- 若 $\sin B = 1$，$B = 90°$，唯一解；
- 若 $\sin B < 1$，$B$ 可能有两个取值（$B_1$ 和 $180° - B_1$），需结合 $A + B < 180°$ 判断一解或两解，即"歧义情形"（ambiguous case）。

## 实际应用

**测量距离**：在地面测量中，观测者在A、B两点测量山顶C的仰角，已知AB距离及两角度，即可用正弦定理求AC或BC的距离。若 $AB = 100$ 米，$\angle CAB = 60°$，$\angle CBA = 75°$，则 $\angle ACB = 45°$，

$$AC = \frac{AB \cdot \sin 75°}{\sin 45°} = \frac{100 \times \frac{\sqrt{6}+\sqrt{2}}{4}}{\frac{\sqrt{2}}{2}} = 50(\sqrt{3}+1) \approx 136.6 \text{ 米}$$

**航海导航**：船只在两个已知位置观测同一灯塔，记录方位角后，利用正弦定理计算船只与灯塔的距离，是传统航海定位的基本计算方法。

**物理中的力分解**：在斜面受力分析中，合力与两个分力方向确定后，三力构成的力三角形可用正弦定理求各分力的大小，避免了繁琐的坐标分解。

## 常见误区

**误区一：将正弦定理局限于锐角三角形**。部分学生认为，只有在锐角三角形中 $\sin A = \dfrac{a}{2R}$ 才成立。实际上，钝角三角形中证明同样成立——当A为钝角时，圆周角定理需取补角处理，但最终结论不变，正弦定理对所有三角形（锐角、直角、钝角）均成立。

**误区二：SSA条件下默认唯一解**。给定两边一对角时，许多学生不验证解的个数便直接写出答案。例如，$a=5, b=7, A=30°$ 时，$\sin B = \dfrac{7 \times 0.5}{5} = 0.7$，此时 $B_1 \approx 44.4°$ 和 $B_2 \approx 135.6°$ 均满足 $A + B < 180°$，存在两个合法三角形，忽略任一解均属错误。

**误区三：混淆正弦定理与余弦定理的适用场景**。正弦定理要求已知条件中至少有一对"角与其对边"的配对（即已知 $a$ 且已知 $A$，或同等条件），若仅已知三边（SSS）或两边夹角（SAS），则无法直接使用正弦定理，应使用余弦定理 $c^2 = a^2 + b^2 - 2ab\cos C$。

## 知识关联

正弦定理以**三角比**（$\sin, \cos, \tan$ 的定义及特殊角值）为计算基础，学生需熟练掌握 $30°, 45°, 60°$ 等特殊角的精确值才能完成符号化计算。

正弦定理与**余弦定理**共同构成解任意三角形的完整工具集：正弦定理处理AAS、ASA、SSA情形，余弦定理处理SAS、SSS情形。两者组合使用可解决所有确定三角形的问题。

在**三角函数应用**阶段，正弦定理是解决测量、导航、物理建模等实际问题的直接桥梁。此外，将正弦定理写成 $a = 2R\sin A$ 的形式，是推导**积化和差**公式及某些**解析几何**中圆与三角形关系的起点，例如利用外接圆半径 $R = \dfrac{abc}{4S}$ 联系边、角与面积。
