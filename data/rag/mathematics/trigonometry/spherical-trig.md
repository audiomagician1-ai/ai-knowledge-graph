# 球面三角

## 概述

球面三角学研究的是球面上由三段大圆弧围成的**球面三角形**的性质与计算方法。所谓大圆，是指球面上圆心与球心重合的圆，其半径等于球的半径。三段大圆弧构成的封闭图形称为球面三角形，它的三个顶点、三条边（用其对应的圆心角度数表示，取值范围为 $0°$ 到 $180°$）以及三个内角共同决定了该图形的几何性质。

球面三角学的历史可追溯至公元前2世纪，希腊天文学家喜帕恰斯（Hipparchus）为解决天体位置计算问题而发展出早期成果。公元7世纪，印度数学家将正弦函数引入球面计算；到公元11世纪，波斯学者阿布·纳斯尔·曼苏尔（Abu Nasr Mansur）系统整理了球面三角的基本定理。欧洲天文学在16世纪大量采用这些工具进行航海导航，雷格蒙塔努斯（Regiomontanus）于1464年写成的《论各种三角形》是西方第一部系统性球面三角学著作。

球面三角与平面三角最本质的区别在于：球面三角形的三内角之和**大于** $180°$，其超出 $180°$ 的部分称为**球面超（spherical excess）**，记为 $\varepsilon = A + B + C - 180°$，且 $0° < \varepsilon < 360°$。这一性质使得球面三角无法直接套用平面三角公式，需要一套独立的定理体系。

## 核心原理

### 球面余弦定理

球面三角形中，设三边对应的圆心角为 $a, b, c$（单位为弧度或度），三个内角为 $A, B, C$，则**边的球面余弦定理**为：

$$\cos a = \cos b \cos c + \sin b \sin c \cos A$$

对应地，另外两边有对称形式。与平面余弦定理 $a^2 = b^2 + c^2 - 2bc\cos A$ 相比，球面版本用 $\cos a$ 替代 $a^2$，用乘积形式替代差，体现了球面曲率的影响。当 $a, b, c$ 趋近于零时，利用 $\cos x \approx 1 - x^2/2$ 和 $\sin x \approx x$ 展开，可以验证球面余弦定理退化为平面余弦定理，说明平面几何是球面几何在小尺度下的近似。

此外还有**角的球面余弦定理**（对偶公式）：

$$\cos A = -\cos B \cos C + \sin B \sin C \cos a$$

这一对偶性来自于球面三角的极三角形（polar triangle）概念，每个球面三角形都有唯一对应的极三角形，两者的边角关系互相补余。

### 球面正弦定理

球面三角形满足：

$$\frac{\sin a}{\sin A} = \frac{\sin b}{\sin B} = \frac{\sin c}{\sin C}$$

形式上与平面正弦定理完全类似，但各边采用的是**圆心角**而非长度。需要注意，球面正弦定理存在多解问题，和平面情形一样，当已知两边一角（SSA）时，可能有零个、一个或两个解。

### 球面超与球面面积

对半径为 $R$ 的球，球面三角形的面积公式为：

$$S = R^2 \cdot \varepsilon = R^2 (A + B + C - \pi)$$

其中 $A, B, C$ 以弧度计。此公式由阿尔贝特·吉拉尔（Albert Girard）于1629年首次发表，故又称**吉拉尔定理**。例如，地球半径约 $6371$ km，若某球面三角形三内角之和为 $185°$，则球面超 $\varepsilon = 5° = 5\pi/180$ rad，面积约为 $6371^2 \times 5\pi/180 \approx 3.55 \times 10^6 \, \text{km}^2$。

### 纳皮尔法则（直角球面三角）

当球面三角形有一个角 $C = 90°$ 时，称为**直角球面三角形**，此时可用纳皮尔圆形记忆法则简化计算。约翰·纳皮尔（John Napier）于1614年提出将五个元素 $\{a, b, \bar{A}, \bar{c}, \bar{B}\}$（其中横线表示余角）排成圆圈，任意一个元素的正弦等于相邻两元素的余弦之积，或等于不相邻两元素的正弦之积。例如：

$$\sin a = \sin A \sin c = \tan b \cot B$$

## 实际应用

**航海与航空大圆航线计算**：两地之间的最短路径在球面上是大圆弧。已知出发地（纬度 $\phi_1$，经度 $\lambda_1$）和目的地（纬度 $\phi_2$，经度 $\lambda_2$），利用球面余弦定理可求出两点间大圆弧对应的圆心角 $d$：

$$\cos d = \sin\phi_1 \sin\phi_2 + \cos\phi_1 \cos\phi_2 \cos(\lambda_2 - \lambda_1)$$

这正是球面余弦定理的直接应用，在GPS导航算法和飞行计划中被广泛使用。

**球面天文坐标变换**：天文学中，赤道坐标（赤经、赤纬）与地平坐标（方位角、高度角）之间的转换，本质上是球面三角形的解算。天顶、天北极、天体三点构成一个球面三角形，利用球面正弦定理和余弦定理即可完成坐标换算，这是每一位观测天文学家的基本技能。

**测地学中的大地测量**：在精密测量中，地球表面两点之间的距离和方向计算不能用平面几何近似，需要借助球面三角（或更精确的椭球面三角）进行计算，直接影响国家坐标系的建立精度。

## 常见误区

**误区一：将球面三角形的边当作长度**。球面三角形的"边"是圆心角（弧度数），不是弧长。弧长 $l = R \cdot a$，只有在给定球半径后才能换算。在代入球面余弦定理时，必须使用圆心角而非实际长度，否则公式完全失效。

**误区二：认为球面三角形的内角和是固定值**。球面三角形的内角和范围是 $(180°, 540°)$，而不是固定的某个值。内角和越大，三角形面积越大（占球面比例越高）。一个极端的例子：将球面分成8个等份的球面三角形，每个内角均为 $90°$，内角和为 $270°$，面积为球面总面积的 $1/8$。平面三角形的 $180°$ 是球面超趋近于零（三角形无限小）时的极限情形。

**误区三：球面正弦定理可以完全类比平面情形使用**。在平面三角中，已知两角一边（AAS）可以唯一确定三角形。但在球面三角中，由于边的取值范围是 $(0°, 180°)$ 且 $\sin$ 函数在此区间并非单调，需要结合余弦定理或其他条件加以判断，单独使用正弦定理仍可能出现歧义。

## 知识关联

**与余弦定理的关联**：平面余弦定理 $c^2 = a^2 + b^2 - 2ab\cos C$ 是球面余弦定理在曲率趋零时的极限情形。理解这一退化关系，有助于把握两者公式结构上"乘积取代平方"的几何意义。球面余弦定理本质上引入了曲率修正，是球面与平面几何差异的集中体现。

**与立体几何的关联**：球面三角形的三边所在的大圆平面，各自经过球心，因此球面三角形的解算等价于三个过球心平面的交线夹角计算。立体几何中两平面夹角（二面角）的概念直接对应球面三角形的内角，这使得空间向量方法可以与球面三角公式互相验证。具体地，若用单位向量 $\hat{u}, \hat{v}, \hat{w}$ 表示三个顶点，则 $\cos a = \hat{v} \cdot \hat{w}$，球面余弦定理可用向量叉积与点积推导得出。

**与天文学和测地学的衔接**：球面三角是天文球面坐标系统的数学基础，也是大地测量、卫星轨道计算、地图投影变形分析的前置工具。掌握球面余弦、正弦定理以及球面超概念，可以直接处理实球面上的角距、方位角和面积等实际问题。
