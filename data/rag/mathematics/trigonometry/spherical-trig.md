# 球面三角

## 概述

球面三角学是研究球面上由大圆弧围成的三角形（球面三角形）的几何性质与三角关系的数学分支。与平面三角形不同，球面三角形的三条"边"并非直线段，而是球面上三个大圆（即圆心与球心重合的圆）各自截取的弧段，其长度通常以所对应的圆心角（弧度或度数）来度量。

球面三角学的系统发展可追溯至古希腊天文学家门奈劳斯（Menelaus of Alexandria，约公元100年），他在《球论》（Sphaerica）中首次给出了球面三角形的严格定义及球面版本的梅内劳斯定理。中世纪伊斯兰数学家纳西尔丁·图西（Nasīr al-Dīn al-Tūsī，1201—1274年）在其著作《论完全四边形》中将球面三角学从天文学中独立出来，视为纯数学学科，并系统推导了球面正弦定理和球面余弦定理。

球面三角学在天文学（计算天体位置）、航海导航（大圆航线规划）、大地测量（地球表面两点间距离）及现代GPS定位算法中均有不可替代的作用。地球半径约为6371千米，地球表面两点间的最短路径（大圆弧）的计算正依赖球面三角公式。

## 核心原理

### 球面三角形的基本量

设球面三角形 $ABC$ 的三顶点在半径为 $R$ 的球面上，三条边 $a, b, c$ 分别是顶点 $A, B, C$ 所对弧段对应的圆心角（以弧度计），三个角 $A, B, C$ 是相应顶点处两大圆切线方向所形成的二面角。

球面三角形与平面三角形有两处根本差异：
- **内角和不等于 $\pi$**：球面三角形三内角之和满足 $\pi < A + B + C < 3\pi$，其超出 $\pi$ 的部分 $E = A + B + C - \pi$ 称为**球面超**（spherical excess）。
- **边长有范围限制**：每条边必须满足 $0 < a, b, c < \pi$，任意两边之和必须大于第三边，且任意两边之差的绝对值小于第三边。

球面超 $E$ 与球面三角形面积的关系由吉拉德定理（Girard's Theorem，1629年）给出：
$$S = R^2 \cdot E = R^2(A + B + C - \pi)$$

### 球面余弦定理

球面三角中最基本的公式是**球面余弦定理**（Spherical Law of Cosines），分为边的余弦定理和角的余弦定理两种形式。

**边的球面余弦定理**：
$$\cos a = \cos b \cos c + \sin b \sin c \cos A$$

其中 $a$ 为角 $A$ 的对边，$b, c$ 为夹角 $A$ 的两邻边，所有边均以圆心角（弧度）表示。这一公式当 $a, b, c \to 0$ 时（球面缩为平面），利用小角近似 $\cos\theta \approx 1 - \theta^2/2$，$\sin\theta \approx \theta$，可退化为平面余弦定理 $a^2 = b^2 + c^2 - 2bc\cos A$。

**角的球面余弦定理**（通过极三角形对偶得到）：
$$\cos A = -\cos B \cos C + \sin B \sin C \cos a$$

### 球面正弦定理

**球面正弦定理**为：
$$\frac{\sin a}{\sin A} = \frac{\sin b}{\sin B} = \frac{\sin c}{\sin C}$$

这与平面正弦定理形式相似，但分子中的 $\sin a$ 是边（圆心角）的正弦，而非边长本身。注意球面正弦定理同样存在**歧义情形**：已知两边一对角时，解可能不唯一，需结合球面三角形的几何约束判断。

### 半角公式与半边公式

类比平面三角的半角公式，球面三角学中有**半角公式**：
$$\sin^2\frac{A}{2} = \frac{\sin(s-b)\sin(s-c)}{\sin b \sin c}$$

其中 $s = \frac{a+b+c}{2}$ 为半周长（以弧度计）。对应地，**半边公式**（利用极三角形）为：
$$\sin^2\frac{a}{2} = \frac{-\cos S \cos(S-A)}{\sin B \sin C}$$

其中 $S = \frac{A+B+C}{2}$。这些公式在已知三边求三角或已知三角求三边时特别有用，且数值上比直接使用余弦定理更稳定。

## 实际应用

**航海大圆航线**：从上海（北纬31°，东经121°）到旧金山（北纬37°，西经122°）的最短航线并非沿纬线向东行驶，而是一段大圆弧。利用球面余弦定理，将两地经纬度转化为球面三角形的边和角，可精确计算该航线的弧长（约8630千米）及各途经点的方位角，这是现代航空和航海路径规划的标准算法。

**天体赤道坐标转换**：天文学中需要在赤道坐标（赤经、赤纬）与黄道坐标（黄经、黄纬）之间换算。两个坐标系的夹角为黄赤交角 $\varepsilon \approx 23.44°$，利用球面余弦定理和正弦定理，可建立精确的坐标变换方程组，计算任意天体在两套坐标下的位置。

**GPS中的球面距离**：Haversine公式是球面余弦定理的数值稳定变形：
$$\Delta\sigma = 2\arcsin\sqrt{\sin^2\!\left(\frac{\Delta\phi}{2}\right) + \cos\phi_1\cos\phi_2\sin^2\!\left(\frac{\Delta\lambda}{2}\right)}$$
其中 $\phi$ 为纬度，$\lambda$ 为经度，球面距离为 $d = R\Delta\sigma$。该公式在两点距离极小时避免了直接使用 $\arccos$ 的精度损失。

## 常见误区

**误区一：将球面角等同于平面角**。初学者常误认为球面三角形的角就是连接顶点的弦（直线）所成的角。实际上球面角是两个大圆平面所成的**二面角**，等于顶点处两大圆切线的夹角，而非顶点到其他顶点的弦之间的夹角。因此球面三角形三角之和可超过 $180°$，甚至一个球面三角形可以有三个直角（如以极点和赤道上两点构成的三角形，其三角均为 $90°$，内角和为 $270°$）。

**误区二：球面余弦定理与平面余弦定理可互换使用**。有人认为当球面三角形的边很短时，球面公式和平面公式结果差不多，实际工程中可以忽略差异。但在大地测量中，即使是100千米范围内的测量，若直接套用平面余弦定理，误差可达数十米量级，远超工程允许误差。正确做法是使用球面公式或更精确的椭球面公式。

**误区三：球面三角形的解的个数与平面三角形相同**。平面三角中"已知三边唯一确定三角形"的结论在球面三角中不成立。在球面上，一组三边数据实际上对应**两个**互为极三角形的球面三角形（一个与另一个关于球心对称），因此在实际应用中必须根据几何意义选取合适的解。

## 知识关联

**前置依赖**：学习球面三角必须熟练掌握平面的**余弦定理**（$c^2 = a^2 + b^2 - 2ab\cos C$），因为球面余弦定理是其球面推广，理解退化过程有助于记忆公式结构。同时需要**立体几何**中的二面角概念和空间向量知识，以理解球面角的本质定义；三维空间中两向量点积公式 $\vec{u}\cdot\vec{v} = |\vec{u}||\vec{v}|\cos\theta$ 是推导球面余弦定理最简洁的工具。

**横向联系**：球面三角学与**双曲三角学**（研究双曲面上的三角形）存在深刻的对偶性。双曲面上的余弦定理形如 $\cosh a = \cosh b \cosh c - \sinh b \sinh c \cos A$，与球面公式相比仅在符号和函数类型上不同，二者均是非欧几何在各自曲率空间中的体现——球面曲率为正（$K = 1/R^2 > 0$），双曲面曲率为负。
