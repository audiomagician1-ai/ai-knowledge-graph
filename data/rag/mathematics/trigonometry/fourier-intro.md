# 傅里叶级数初步

## 概述

傅里叶级数是将一个周期函数表示为无穷多个正弦函数与余弦函数之和的数学方法。具体地说，一个以 $2\pi$ 为周期的函数 $f(x)$，若满足狄利克雷条件（在一个周期内连续或只有有限个间断点，且有有限个极值点），则可以写成：

$$f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty}\left(a_n \cos nx + b_n \sin nx\right)$$

其中 $a_0, a_1, b_1, a_2, b_2, \ldots$ 称为傅里叶系数。

这一思想由法国数学家让-巴蒂斯特·约瑟夫·傅里叶（Jean-Baptiste Joseph Fourier）在1807年提交给法国科学院的论文《热的传播》中首次系统提出。傅里叶试图用三角级数求解热传导偏微分方程，这一做法在当时极具争议，数学家拉格朗日认为任意函数不可能用三角级数表示。然而，傅里叶的思想最终在19世纪被严格化，成为现代分析学的重要工具。

傅里叶级数的重要性在于：它揭示了任意"形状合理"的周期波形，本质上是不同频率的纯正弦/余弦振动的叠加。这不仅解释了乐器音色的差异（不同泛音分量的比例），也为信号处理中的频谱分析奠定了理论基础。

## 核心原理

### 三角函数系的正交性

傅里叶展开能够成立，根本原因在于函数系 $\{1, \cos x, \sin x, \cos 2x, \sin 2x, \ldots\}$ 在区间 $[-\pi, \pi]$ 上具有**正交性**。具体计算结果为：

$$\int_{-\pi}^{\pi} \cos mx \cos nx \, dx = \begin{cases} 0, & m \neq n \\ \pi, & m = n \neq 0 \end{cases}$$

$$\int_{-\pi}^{\pi} \sin mx \sin nx \, dx = \begin{cases} 0, & m \neq n \\ \pi, & m = n \end{cases}$$

$$\int_{-\pi}^{\pi} \sin mx \cos nx \, dx = 0 \quad (\text{对所有整数 } m, n)$$

正是因为不同频率的正弦/余弦函数之间"互不干扰"，才能将求系数的问题转化为各自独立的积分计算。这与向量正交分解的思想完全类比：正交基向量使得每个分量可以单独投影求取。

### 傅里叶系数的计算公式

利用正交性，在 $f(x)$ 的展开式两侧分别乘以 $\cos nx$ 或 $\sin nx$ 再积分，其余项因正交性全部消去，得到傅里叶系数的计算公式：

$$a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \, dx$$

$$a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \cos nx \, dx, \quad n = 1, 2, 3, \ldots$$

$$b_n = \frac{1}{\pi} \int_{-\pi}^{\pi} f(x) \sin nx \, dx, \quad n = 1, 2, 3, \ldots$$

注意 $a_0/2$ 是函数在一个周期内的平均值，物理上对应信号的直流分量。$a_1, b_1$ 对应基频（一次谐波），$a_n, b_n$ 对应 $n$ 倍频（$n$ 次谐波）。

### 奇偶函数的简化

当函数具有奇偶对称性时，傅里叶展开可以大幅简化：

- 若 $f(x)$ 是**偶函数**（$f(-x) = f(x)$），则所有 $b_n = 0$，展开式只含余弦项，称为**傅里叶余弦级数**：
  $$f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} a_n \cos nx$$

- 若 $f(x)$ 是**奇函数**（$f(-x) = -f(x)$），则所有 $a_n = 0$，展开式只含正弦项，称为**傅里叶正弦级数**：
  $$f(x) = \sum_{n=1}^{\infty} b_n \sin nx$$

### 收敛性与吉布斯现象

傅里叶级数在函数的连续点处收敛到 $f(x)$ 本身；在间断点处，级数收敛到左右极限的平均值 $\frac{f(x^-)+f(x^+)}{2}$。更值得注意的是**吉布斯现象**：在间断点附近，傅里叶级数的部分和会出现约为跳跃量 **8.9%** 的超调，无论取多少项，这个超调幅度不会消失，只是向间断点处压缩。

## 实际应用

**方波的傅里叶展开**是最经典的示例。定义方波 $f(x) = \text{sgn}(\sin x)$（即 $0 < x < \pi$ 时为 $1$，$-\pi < x < 0$ 时为 $-1$），其傅里叶级数为：

$$f(x) = \frac{4}{\pi}\left(\sin x + \frac{\sin 3x}{3} + \frac{\sin 5x}{5} + \cdots\right) = \frac{4}{\pi}\sum_{k=0}^{\infty}\frac{\sin(2k+1)x}{2k+1}$$

这说明方波只含奇次谐波，且系数以 $1/n$ 速度衰减。取 $x = \pi/2$ 代入，立刻得到著名的**莱布尼茨公式**：$\frac{\pi}{4} = 1 - \frac{1}{3} + \frac{1}{5} - \frac{1}{7} + \cdots$，这是傅里叶展开与 $\pi$ 计算的一个意外联系。

在音乐声学领域，小提琴与长笛演奏同一音高时，基频相同但各次谐波的傅里叶系数 $a_n, b_n$ 比例不同，这种差异就是音色不同的数学本质。在电子工程中，PWM（脉冲宽度调制）信号的滤波设计直接依赖对其傅里叶展开中各次谐波频率的分析。

## 常见误区

**误区一：傅里叶级数对任意函数都绝对收敛**。实际上，傅里叶级数的收敛性相当微妙。存在连续函数其傅里叶级数在某些点发散（数学家柯尔莫哥洛夫1926年甚至构造了 $L^1$ 函数使其傅里叶级数处处发散）。狄利克雷条件只是充分条件，不是必要条件，满足条件的函数级数一定收敛，但不满足条件的函数也可能收敛。

**误区二：展开的周期必须是 $2\pi$**。标准公式是针对周期 $2\pi$ 推导的，但任意周期 $T$ 的函数完全可以展开。令 $x' = \frac{2\pi x}{T}$ 换元，周期 $T$ 函数的傅里叶系数变为：
$$a_n = \frac{2}{T}\int_{0}^{T} f(x)\cos\frac{2\pi nx}{T}\,dx, \quad b_n = \frac{2}{T}\int_{0}^{T} f(x)\sin\frac{2\pi nx}{T}\,dx$$
其中 $n$ 次谐波的角频率为 $\frac{2\pi n}{T}$，而非 $n$。混淆两种公式是计算傅里叶系数时最常见的错误。

**误区三：取的项数越多，逼近效果在所有点处都更好**。吉布斯现象表明，在间断点附近增加项数只会让超调峰值更靠近间断点，峰值高度始终约为跳跃量的 $1 + \frac{1}{2}\cdot\frac{2}{\pi}\approx 1.089$ 倍，无法通过增加项数消除。
