# 三角函数图像

## 概述

三角函数图像是将正弦函数 y = sin x、余弦函数 y = cos x 等三角函数在直角坐标系中的几何表示。这类图像最显著的特征是**周期性波形**：正弦曲线从原点出发，在 x = π/2 处达到最高点 1，在 x = 3π/2 处降至最低点 −1，完成一个完整周期 2π 后重复。这种形态也称为"正弦波"，是自然界中波动现象的数学原型。

三角函数图像的研究可追溯至17世纪，欧拉（Euler）在1748年的《无穷分析引论》中首次系统地将三角函数写成解析式，并用坐标图像来刻画周期规律。在此之前，天文学家已通过托勒密弦表和印度正弦表逼近这类波形，但缺乏统一的坐标语言。

理解三角函数图像的实际意义在于：振幅、频率、相移三个参数的任何改变都会直接、可量化地改变波形的外观，而这三个参数恰好对应物理世界中声波的响度、音调和相位差，是信号处理与工程分析不可缺少的描述工具。

## 核心原理

### 基本波形：正弦与余弦的关系

标准正弦函数 y = sin x 的图像关于原点对称（奇函数），而标准余弦函数 y = cos x 关于 y 轴对称（偶函数）。两者形状完全相同，仅存在 π/2 的水平位移关系：

$$\cos x = \sin\!\left(x + \frac{\pi}{2}\right)$$

这意味着余弦图像是正弦图像向**左平移 π/2 个单位**的结果。正切函数 y = tan x 的图像则截然不同：它没有振幅限制，在 x = π/2 + kπ（k 为整数）处出现垂直渐近线，图像被分割成无数个独立的支，每支跨越区间 (−π/2 + kπ, π/2 + kπ)。

### 振幅（A）：纵向拉伸与压缩

对于一般形式 **y = A sin(Bx + C) + D**，参数 A 决定振幅，即波形距中轴线的最大距离。

- |A| > 1：图像纵向拉伸，如 y = 3 sin x 的波峰为 3，波谷为 −3
- 0 < |A| < 1：图像纵向压缩，如 y = 0.5 sin x 的最大值仅为 0.5
- A < 0：图像相对 x 轴翻转，如 y = −sin x 与 y = sin x 关于 x 轴对称

振幅的变化**不影响周期**，这是初学者容易混淆的关键点。

### 频率与周期（B）：横向拉伸与压缩

参数 B 直接决定角频率，进而确定图像的**周期 T**：

$$T = \frac{2\pi}{|B|}$$

- B = 2 时，y = sin 2x 的周期为 π，相比标准正弦图像**横向压缩为原来的一半**，单位长度内出现更密集的波形
- B = 1/2 时，y = sin(x/2) 的周期为 4π，图像**横向拉伸为两倍**
- 正弦函数的频率 f（每单位 x 的完整波数）为 f = |B| / (2π)

### 相移（C）：图像水平移动

参数 C 产生**相移**，即图像沿 x 轴的平移距离：

$$\text{相移} = -\frac{C}{B}$$

以 y = sin(2x − π/3) 为例：B = 2，C = −π/3，相移 = (π/3)/2 = **π/6**，即图像向右平移 π/6 个单位。注意相移的正负符号：C > 0 对应左移，C < 0 对应右移，许多教材直接写成 y = A sin[B(x − φ)]，此时 φ 直接表示右移量，避免符号混淆。

### 垂直位移（D）：中轴线上下移动

参数 D 将整个波形沿 y 轴平移，中轴线从 y = 0 移至 y = D。例如 y = 2 sin x + 1 的最大值为 3，最小值为 −1，中轴线为 y = 1。这在描述潮汐高度等有基准偏移的周期现象时尤为重要。

## 实际应用

**绘图题的标准步骤**：给定 y = 3 sin(2x − π/6)，应先提取参数：A = 3，B = 2，C = −π/6，D = 0；计算周期 T = 2π/2 = **π**；计算相移 = (π/6)/2 = **π/12**（向右）；确定五个关键点（起点、波峰、零点、波谷、终点）在新坐标下的位置，依次为 x = π/12, π/12 + π/4 = π/3, 7π/12, 5π/6, 13π/12，再描点连线。

**声学中的应用**：中央 A 音（440 Hz）对应的声压波形可写成 y = A sin(2π × 440 × t)，其图像周期为 1/440 秒 ≈ 0.00227 秒，振幅 A 对应音量大小。两个频率相近的音叠加时，图像出现"拍频"现象，即振幅缓慢起伏，这在调音时直接可见。

**电路分析**：交流电压 u = 220√2 · sin(100πt + π/6) 伏特中，振幅 220√2 ≈ 311 V，角频率 100π rad/s 对应频率 50 Hz，相移 π/6 表示该电压相对参考相位超前 30°，三相电系统中三条相线的图像恰好依次相差 2π/3。

## 常见误区

**误区一：混淆周期与频率中 B 的作用方向**

许多学生误认为 B 变大时图像"被拉长"，实际上 B 增大使周期 T = 2π/B **变小**，图像在水平方向被**压缩**，单位区间内出现更多完整波形。可通过比较 y = sin x（T = 2π）和 y = sin 3x（T = 2π/3）的图像直观验证。

**误区二：相移方向的符号错误**

对于 y = sin(x + π/4)，部分学生因括号内是"加π/4"而认为右移，实际上图像**向左平移 π/4**。正确逻辑是：令 x + π/4 = 0，解得 x = −π/4，说明原先 x = 0 处的值现在提前出现，即整体左移。

**误区三：将振幅等同于最大值**

当 D ≠ 0 时，振幅仍是 |A|，但函数最大值为 A + D，最小值为 −A + D。例如 y = 2 sin x + 3 的振幅为 2，最大值为 5，两者不相等。振幅描述的是波动幅度，而非波峰的绝对高度。

## 知识关联

**前置概念——单位圆**：三角函数图像本质上是将单位圆上角度与纵坐标（正弦）或横坐标（余弦）的对应关系"展开"到直角坐标系中。单位圆中角 θ 对应的点 (cos θ, sin θ) 直接决定了图像在 x = θ 处的函数值，因此单位圆四个关键角（0, π/2, π, 3π/2）对应图像的五个关键点（起点、峰、零、谷、终点）。

**后续概念——反三角函数**：三角函数图像的周期性意味着原函数不是全局单射，必须**限制定义域**才能定义反函数。正弦函数限制在 [−π/2, π/2] 上单调递增，对应反正弦 arcsin 的值域；余弦函数限制在 [0, π] 上单调递减，对应反余弦 arccos 的值域。从图像角度理解，就是截取一段"严格单调"的波形片段。

**后续概念——傅里叶级数初步**：任意以 2π 为周期的"分段光滑"函数都可以分解为若干正弦和余弦图像的叠加，即 f(x) = a₀/2 + Σ(aₙ cos nx + bₙ sin nx)。能熟练分析振幅、频率、相移的变化效果，正是理解傅里叶分解"用哪些波形、各占多大权重"的直觉基础。
