# 函数的性质

## 概述

函数的性质是对函数整体行为规律的系统性描述，主要包括单调性、奇偶性、周期性和有界性四大方面。这四种性质从不同维度刻画了一个函数的"面貌"：单调性描述函数值随自变量增减的趋势，奇偶性揭示函数图像的对称结构，周期性捕捉函数值重复出现的规律，有界性则限定函数值的范围上下限。

这套分析框架在17至18世纪随着欧拉（Leonhard Euler）对函数概念的系统化而逐步确立。欧拉在1748年出版的《无穷小量分析引论》中对函数进行了符号化定义，使得对函数行为的精确描述成为可能。现代高中数学将这四种性质作为研究函数的基本工具，是从具体函数（如二次函数、三角函数）过渡到抽象函数分析的核心方法论。

掌握函数的性质，意味着能够仅凭代数条件判断函数图像的走向与形态，而无需逐点描绘。例如，已知某函数是奇函数且在正半轴单调递增，则无需计算便能推断其在负半轴的单调性，这种推理能力在解题中极为高效。

## 核心原理

### 单调性：严格的不等式定义

函数 $f(x)$ 在区间 $I$ 上单调递增，当且仅当对 $I$ 上任意 $x_1 < x_2$，均有 $f(x_1) < f(x_2)$；单调递减则要求 $f(x_1) > f(x_2)$。这里有两个关键细节容易被忽视：第一，单调性必须指定区间，同一函数在不同区间可能有不同的单调性（如 $f(x)=x^2$ 在 $(-\infty, 0)$ 递减，在 $(0, +\infty)$ 递增）；第二，判断单调性时必须在同一单调区间内作比较，不可跨区间使用单调性进行不等式推导。

用单调性证明题的标准步骤是：设 $x_1, x_2 \in I$ 且 $x_1 < x_2$，作差 $f(x_1) - f(x_2)$，通过因式分解或配方判断差值符号，最终得出单调性结论。例如证明 $f(x) = -x + 3$ 在 $\mathbb{R}$ 上单调递减：$f(x_1) - f(x_2) = (-x_1+3)-(-x_2+3) = x_2 - x_1 > 0$，故 $f(x_1) > f(x_2)$，单调递减得证。

### 奇偶性：关于轴或原点的对称

奇函数满足 $f(-x) = -f(x)$，图像关于**原点**对称；偶函数满足 $f(-x) = f(x)$，图像关于 **$y$ 轴**对称。判断奇偶性的前提是：函数的定义域必须关于原点对称，否则既不是奇函数也不是偶函数。例如，$f(x) = x^3 + 1$ 的定义域虽然关于原点对称，但 $f(-x) = -x^3+1 \neq -f(x) = -x^3-1$，故为非奇非偶函数。

一个重要结论：若奇函数 $f(x)$ 在 $x=0$ 处有定义，则 $f(0) = 0$。这由 $f(-0)=-f(0)$ 即 $f(0)=-f(0)$ 直接推出。此结论常用于在奇偶性条件下求函数的参数值。

### 周期性：平移后图像重合

若存在非零常数 $T$，使得对定义域内所有 $x$ 均有 $f(x+T) = f(x)$，则 $f(x)$ 为周期函数，$T$ 称为一个周期。最小正周期是所有正周期中最小的那个（若存在）。标准正弦函数 $y=\sin x$ 的最小正周期为 $2\pi$；而 $y=\sin(\omega x)$ 的最小正周期为 $\dfrac{2\pi}{|\omega|}$，这是计算变换后三角函数周期的基本公式。

需注意常数函数 $f(x)=c$ 是周期函数（任意非零实数均为其周期），但它**没有最小正周期**。这是课程中少数几个"周期函数不一定存在最小正周期"的反例，需特别记忆。

### 有界性：函数值的绝对控制

函数 $f(x)$ 在区间 $D$ 上有上界，是指存在实数 $M$，使得对所有 $x \in D$ 均有 $f(x) \leq M$；有下界则类似定义。若同时有上界和下界，则称在 $D$ 上**有界**，等价于存在正数 $M$ 使得 $|f(x)| \leq M$ 对所有 $x \in D$ 成立。$y = \sin x$ 满足 $|\sin x| \leq 1$，是有界函数的典型；而 $y = \tan x$ 在 $\left(-\dfrac{\pi}{2}, \dfrac{\pi}{2}\right)$ 上无界。

## 实际应用

**利用奇偶性与单调性联合推断：** 已知 $f(x)$ 是奇函数，且在 $(0,+\infty)$ 上单调递增，可立即推断 $f(x)$ 在 $(-\infty,0)$ 上也单调递增。推理路径是：设 $x_1 < x_2 < 0$，则 $0 < -x_2 < -x_1$，由递增性得 $f(-x_2) < f(-x_1)$，再由奇函数性质 $-f(x_2) < -f(x_1)$，即 $f(x_1) < f(x_2)$，结论成立。

**周期性求函数值：** 已知 $f(x)$ 是周期为 5 的函数，且 $f(3) = 7$，则 $f(8) = f(3+5) = 7$，$f(-2) = f(-2+5) = f(3) = 7$。此类题型在高考中频繁出现，核心操作是将自变量通过加减整数个周期转化到已知区间内。

**有界性在不等式估算中的应用：** 判断 $g(x) = \dfrac{\sin x}{x^2+1}$ 是否有界，只需注意 $|\sin x| \leq 1$ 且 $x^2+1 \geq 1$，故 $|g(x)| \leq 1$，即 $g(x)$ 在 $\mathbb{R}$ 上有界，上界为 1，下界为 -1。

## 常见误区

**误区一：单调区间可以合并。** 很多学生在求出函数的多个单调递增区间后，习惯写成"$f(x)$ 在 $(-\infty,0) \cup (2,+\infty)$ 上单调递增"。这是错误的——单调区间必须是连通的区间，用并集表示单调区间会导致逻辑错误（无法在该"区间"内对任意两点使用单调不等式）。

**误区二：偶函数一定在 $y$ 轴两侧行为完全对称，奇函数在正负半轴单调性相反。** 实际上，奇函数在关于原点对称的两个区间上单调性**相同**，而非相反。$f(x)=x^3$ 在 $(-\infty,0)$ 和 $(0,+\infty)$ 上均单调递增，正是奇函数单调性一致的典型体现。

**误区三：判断奇偶性时忽略定义域检查。** $f(x) = \sqrt{x+1} + \sqrt{1-x}$ 的定义域是 $[-1,1]$，关于原点对称，验证 $f(-x) = \sqrt{1-x}+\sqrt{1+x} = f(x)$，故为偶函数。若定义域是 $[0,1]$，则因不关于原点对称，直接判定为非奇非偶，无需再计算 $f(-x)$。

## 知识关联

函数的性质以**函数概念**为基础：只有明确了定义域、值域和对应法则，才能对单调性和奇偶性进行严格定义。**二次函数** $f(x)=ax^2+bx+c$ 是练习这四种性质的理想载体——它的单调区间在顶点 $x=-\dfrac{b}{2a}$ 处分界，是偶函数当且仅当 $b=0$，不具有周期性，且在任意有限闭区间上有界。

在后续学习中，**反函数**要求原函数必须在其定义域上严格单调（单射），因此单调性是反函数存在的判定条件。**复合函数**的单调性判断遵循"同增异减"法则，即若外层函数与内层函数单调性相同则复合后单调递增，相反则单调递减，这一法则直接依赖于本章对单调性的精确定义。奇偶性在复合函数中也有传递规律：奇函数与奇函数复合得奇函数，偶函数与任何函数复合得偶函数，这些结论均由奇偶性定义出发严格推导而来。
