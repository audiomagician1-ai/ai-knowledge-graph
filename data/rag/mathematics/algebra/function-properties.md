# 函数的性质

## 概述

函数的性质是对函数行为规律的系统性描述，主要包括单调性、奇偶性、周期性和有界性四大类。这些性质从不同维度刻画了函数图像的形态特征：单调性描述函数值随自变量变化的趋势方向，奇偶性揭示函数图像关于原点或 y 轴的对称结构，周期性捕捉函数的重复规律，有界性则限定函数值的范围。

函数性质的系统化研究在 17 世纪随着微积分的兴起逐步形成。莱布尼茨和牛顿在研究曲线切线和极值时，隐含地使用了单调性的概念；而三角函数的周期性早在古希腊天文学中就已被观测到，正弦函数的周期为 $2\pi$，这一精确值源于单位圆的几何定义。

研究函数的性质具有实用意义：已知 $f(x) = x^2$ 是偶函数且在 $[0,+\infty)$ 上单调递增，可以不作图直接推断 $f(-3) = f(3) = 9$，并判断 $f(-3) > f(-2)$。这种推断能力在解方程、比较大小、讨论参数范围等问题中高效实用。

## 核心原理

### 单调性

函数单调性的严格定义基于区间内任意两点的比较：若对区间 $I$ 内任意 $x_1 < x_2$，都有 $f(x_1) < f(x_2)$，则称 $f(x)$ 在 $I$ 上单调递增。对应地，若 $f(x_1) > f(x_2)$，则单调递减。

**注意单调性是区间性质，而非整体性质。** 例如 $f(x) = x^2$ 在 $(-\infty, 0]$ 上单调递减，在 $[0, +\infty)$ 上单调递增，但在整个实数域上既不递增也不递减。判断单调性的代数方法是作差法：构造 $f(x_1) - f(x_2)$，结合 $x_1 - x_2 < 0$ 分析符号。例如，设 $f(x) = -2x + 1$，取 $x_1 < x_2$，则 $f(x_1) - f(x_2) = -2(x_1 - x_2) > 0$，故 $f$ 在 $\mathbb{R}$ 上单调递减。

### 奇偶性

奇偶性的前提是函数的定义域关于原点对称（即若 $x$ 在定义域内，则 $-x$ 也在定义域内）。满足 $f(-x) = f(x)$ 的函数称为偶函数，其图像关于 y 轴对称；满足 $f(-x) = -f(x)$ 的函数称为奇函数，其图像关于原点对称。

奇函数的一个重要推论：若奇函数 $f(x)$ 的定义域包含 $x = 0$，则 $f(0) = 0$（因为 $f(0) = -f(0)$ 推出 $f(0) = 0$）。函数 $f(x) = x^3 - x$ 是奇函数（验证：$f(-x) = (-x)^3 - (-x) = -x^3 + x = -(x^3 - x) = -f(x)$）；$f(x) = x^2 + 1$ 是偶函数；$f(x) = x^2 + x$ 则既非奇函数也非偶函数（也称"非奇非偶函数"）。

### 周期性

若存在非零常数 $T$，使得对定义域内所有 $x$ 均有 $f(x + T) = f(x)$，则称 $f(x)$ 是以 $T$ 为周期的周期函数，满足该条件的最小正 $T$ 称为最小正周期（基本周期）。

正弦函数 $\sin x$ 和余弦函数 $\cos x$ 的最小正周期均为 $2\pi$，正切函数 $\tan x$ 的最小正周期为 $\pi$。对于 $f(x) = \sin(\omega x)$（$\omega > 0$），其最小正周期为 $T = \dfrac{2\pi}{\omega}$。需要注意：并非所有周期函数都存在最小正周期，例如 Dirichlet 函数（有理数处取 1、无理数处取 0）是周期函数，但不存在最小正周期。

### 有界性

函数 $f(x)$ 在集合 $D$ 上有界，当且仅当存在正数 $M$，使得对所有 $x \in D$ 有 $|f(x)| \leq M$。$\sin x$ 和 $\cos x$ 在 $\mathbb{R}$ 上有界（取 $M = 1$ 即可）；$f(x) = \dfrac{1}{x}$ 在 $(0,1)$ 上无界（当 $x \to 0^+$ 时函数值趋向无穷）；但 $\dfrac{1}{x}$ 在 $[1, +\infty)$ 上有界（因为 $\left|\dfrac{1}{x}\right| \leq 1$）。有界性依赖于所指定的区间，同一函数在不同区间上可以有界或无界。

## 实际应用

**比较函数值大小：** 利用奇偶性和单调性联合判断。已知 $f(x)$ 是定义在 $\mathbb{R}$ 上的奇函数，且在 $(0,+\infty)$ 上单调递减，则 $f(-1)$ 与 $f(2)$ 的大小：$f(-1) = -f(1)$，而 $f(1) > f(2)$（递减），故 $-f(1) < -f(2) = f(-2)$，即 $f(-1) < f(2)$。

**方程解的个数判断：** 若 $f(x) = x^3 - 3x$ 在 $[-2, 2]$ 上求解 $f(x) = 0$，可先分析 $f(x)$ 的奇偶性（奇函数）及单调区间（在 $(-\infty, -1)$ 和 $(1,+\infty)$ 上递增，在 $(-1,1)$ 上递减），结合图像对称性，确认方程在该区间内恰好有三个实数根：$x = -\sqrt{3}, 0, \sqrt{3}$。

**信号处理中的应用：** 声音信号可以分解为不同频率的正弦函数之和（傅里叶级数），正弦函数的周期性是这一分解的数学基础。频率为 440 Hz 的 A4 音符，对应正弦函数周期 $T = \dfrac{1}{440}$ 秒。

## 常见误区

**误区一：将单调区间混淆为整体性质。** 学生常错误认为"$f(x)$ 在 $[-1,1]$ 上单调递增，在 $[1,3]$ 上也单调递增，因此在 $[-1,3]$ 上单调递增"。这个结论成立需要额外验证 $x=1$ 处的连接情况，但对于分段函数，两段都递增并不保证合并后整体递增。例如，$f(x) = \begin{cases} x & x \leq 1 \\ x - 3 & x > 1 \end{cases}$ 两段各自递增，但在整个区间上不单调。

**误区二：忽视奇偶性判断的定义域前提。** 判断 $f(x) = \sqrt{x+1} + \sqrt{1-x}$ 是否为偶函数时，必须先确认定义域 $[-1,1]$ 关于原点对称（此处满足），再验证 $f(-x) = f(x)$，才能得出偶函数的结论。若定义域为 $[0,2]$，同样的解析式就既非奇函数也非偶函数，因为定义域不关于原点对称。

**误区三：认为周期函数必有最小正周期。** 常数函数 $f(x) = c$（$c$ 为任意常数）满足对任意非零实数 $T$ 都有 $f(x+T) = f(x)$，故任意正实数都是其周期，但不存在最小正周期。这与 $\sin x$ 存在唯一最小正周期 $2\pi$ 的情况不同。

## 知识关联

函数的性质以**函数概念**和**二次函数**为基础：函数概念提供了定义域、值域、对应法则的框架，使得单调性中"对定义域内任意两点"的表述具有精确含义；二次函数 $f(x) = ax^2 + bx + c$ 是分析对称轴（体现偶函数对称性推广）和极值（体现单调性转折）的具体载体，学生已有的二次函数图像直觉有助于内化抽象定义。
