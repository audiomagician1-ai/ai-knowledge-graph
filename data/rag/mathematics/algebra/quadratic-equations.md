# 一元二次方程

## 概述

一元二次方程是含有一个未知数且未知数最高次数为2的整式方程，其标准形式为 $ax^2 + bx + c = 0$（其中 $a \neq 0$，$a$、$b$、$c$ 为实数系数）。条件 $a \neq 0$ 是严格必要的——若 $a = 0$，方程退化为一元一次方程，不再属于二次方程的范畴。

一元二次方程的系统研究可追溯至古巴比伦时期（约公元前2000年），当时的泥板文书已记录了解二次问题的算法。印度数学家婆罗摩笈多（约公元628年）首次给出接近现代形式的求根公式，阿拉伯数学家花拉子米在《代数学》中进一步系统化了这一方法。16世纪欧洲代数学家将其整理为当今通用的求根公式形式。

一元二次方程是代数方程中最基础且应用最广泛的非线性方程类型。它不仅直接描述抛物线轨迹、自由落体运动、面积优化等实际问题，更是学习二次函数、圆锥曲线、复数的直接前置知识。掌握一元二次方程的三种解法及判别式理论，是高中代数乃至后续高等数学学习的重要基石。

## 核心原理

### 求根公式（配方法推导）

对标准形式 $ax^2 + bx + c = 0$ 进行配方：

$$ax^2 + bx = -c \implies x^2 + \frac{b}{a}x = -\frac{c}{a}$$

两边加 $\left(\dfrac{b}{2a}\right)^2$，得：

$$\left(x + \frac{b}{2a}\right)^2 = \frac{b^2 - 4ac}{4a^2}$$

开方并整理，得**求根公式**：

$$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

其中分子的 $\pm$ 对应两个根 $x_1$ 和 $x_2$，分母 $2a$ 中 $a$ 的符号不影响根的存在性但影响根的符号。使用求根公式时，必须先将方程化为标准形式，再准确识别 $a$、$b$、$c$ 的值（注意带符号）。

### 判别式 $\Delta = b^2 - 4ac$

判别式 $\Delta$（读作"Delta"）决定了方程在实数范围内根的情况：

- $\Delta > 0$：方程有**两个不相等的实数根**，如 $x^2 - 5x + 4 = 0$ 中 $\Delta = 25 - 16 = 9 > 0$，两根为 $x = 4$ 或 $x = 1$。
- $\Delta = 0$：方程有**两个相等的实数根（重根）**，如 $x^2 - 6x + 9 = 0$ 中 $\Delta = 36 - 36 = 0$，重根为 $x = 3$。
- $\Delta < 0$：方程在实数范围内**无实数根**（但在复数范围内有两个共轭复数根）。

判别式还有一个几何意义：$\Delta$ 的正负决定了对应二次函数 $y = ax^2 + bx + c$ 的图像与 $x$ 轴的交点数目。

### 韦达定理（根与系数的关系）

设 $x_1$、$x_2$ 是方程 $ax^2 + bx + c = 0$ 的两个根，则由因式分解 $a(x - x_1)(x - x_2) = ax^2 + bx + c$ 展开比较系数，可得**韦达定理**：

$$x_1 + x_2 = -\frac{b}{a}, \qquad x_1 \cdot x_2 = \frac{c}{a}$$

韦达定理由法国数学家弗朗索瓦·韦达于1591年发现。它的核心价值在于：无需实际求出 $x_1$、$x_2$，即可计算含根的代数式的值。例如，已知两根之和为 $-\frac{b}{a}$，可直接计算 $x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1x_2 = \frac{b^2}{a^2} - \frac{2c}{a}$。反向应用韦达定理（已知两根，构造方程）：若两根之和为 $S$，积为 $P$，则方程为 $x^2 - Sx + P = 0$。

### 其他解法：因式分解法与换元法

因式分解法适用于 $\Delta$ 为完全平方数或方程可直接因式分解的情形，如 $x^2 - 7x + 12 = 0$ 分解为 $(x-3)(x-4) = 0$，得 $x = 3$ 或 $x = 4$。换元法用于处理形如 $(x^2 + x)^2 - 8(x^2 + x) + 12 = 0$ 的高次方程，令 $t = x^2 + x$，化为关于 $t$ 的一元二次方程后再回代。

## 实际应用

**面积与几何问题**：一块长方形土地面积为 $60 \text{ m}^2$，长比宽多 7 m，设宽为 $x$ 米，则 $x(x+7) = 60$，即 $x^2 + 7x - 60 = 0$。用求根公式得 $x = \dfrac{-7 \pm \sqrt{49 + 240}}{2} = \dfrac{-7 \pm 17}{2}$，取正值 $x = 5$，长为 12 m。

**抛体运动**：物体以初速度 $v_0 = 20 \text{ m/s}$ 竖直上抛，高度方程为 $h = 20t - 5t^2$。令 $h = 15$，得 $5t^2 - 20t + 15 = 0$，即 $t^2 - 4t + 3 = 0$，解得 $t = 1$ s 或 $t = 3$ s，分别对应上升和下降时经过该高度的时刻。

**韦达定理的竞赛应用**：已知 $x_1, x_2$ 是 $2x^2 - 3x - 1 = 0$ 的两根，求 $\dfrac{1}{x_1} + \dfrac{1}{x_2}$。直接用韦达定理：$\dfrac{x_1 + x_2}{x_1 x_2} = \dfrac{3/2}{-1/2} = -3$，无需解方程即可得到结果。

## 常见误区

**误区一：忽略 $a \neq 0$ 的条件**
在含参数的方程（如 $(m-1)x^2 + 2x - 3 = 0$）中，必须讨论 $m = 1$ 的情形。若不排除 $m = 1$，方程变为一元一次方程，将其当作一元二次方程处理会得出错误的根的数目结论。

**误区二：判别式只能判断实数根**
许多学生认为 $\Delta < 0$ 意味着"无解"，实际上此时方程在复数域内有两个共轭复数根 $x = \dfrac{-b \pm i\sqrt{4ac - b^2}}{2a}$。说"无实数根"是准确的，说"无解"是错误的，这一区分在学习复数初步后尤为重要。

**误区三：韦达定理适用于任意方程**
韦达定理的前提是方程确实有两个根（即 $\Delta \geq 0$ 且 $a \neq 0$）。若 $\Delta < 0$，在实数范围内不能直接套用 $x_1 + x_2 = -\frac{b}{a}$，因为 $x_1$、$x_2$ 此时为复数，中学阶段的韦达定理讨论通常限于实数根的情形。

## 知识关联

**与先修知识的联系**：求根公式的配方推导直接依赖完全平方公式 $(x + k)^2 = x^2 + 2kx + k^2$；因式分解法解一元二次方程要求熟练掌握十字相乘法（如将 $6x^2 + 7x - 3$ 分解为 $(2x + 3)(3x - 1)$）；一元一次方程的等式变形规则（移项、合并同类项）是整理标准形式的基础操作。

**向后续概念的延伸**：二次函数 $y = ax^2 + bx + c$ 的图像与 $x$ 轴交点坐标恰好由一元二次方程 $ax^2 + bx + c = 0$ 决定，判别式直接对应图像与 $x$ 轴的位置关系。复数初步的学习从 $\Delta < 0$ 时引入虚数单位 $i = \sqrt{-1}$，使一元二次方程在任意情形下均有两个根（代数基本定理）。直线与圆锥曲线的交点问题，本质上归结为一元二次方程或二元二次方程组，判别式方法直接迁移应用于判断切线、相交、相离三种位置关系。
