# 二次函数

## 概述

二次函数是形如 **y = ax² + bx + c**（其中 a ≠ 0，a、b、c 为常数）的函数，其自变量 x 的最高次数为 2。这一概念最早可追溯至古希腊时期阿波罗尼斯对圆锥曲线的研究，他在公元前200年左右系统描述了抛物线的几何性质，而代数形式的二次函数在17世纪笛卡尔建立坐标系后才得以完整表达。

二次函数的图像是一条**抛物线**，其开口方向、宽窄、顶点位置完全由三个系数 a、b、c 决定。其中 a 是决定图像形状最关键的参数：当 a > 0 时，抛物线开口向上，函数有最小值；当 a < 0 时，抛物线开口向下，函数有最大值；|a| 越大，抛物线开口越窄，图形越"瘦"。

掌握二次函数不仅是理解抛物线几何特性的基础，更是求解一元二次不等式、研究函数最值问题的直接工具。物理中弹道轨迹、经济学中的利润最大化模型，均以二次函数作为数学建模的核心载体。

## 核心原理

### 顶点与顶点式

二次函数 y = ax² + bx + c 可通过**配方法**变形为顶点式：

$$y = a\left(x + \frac{b}{2a}\right)^2 + \frac{4ac - b^2}{4a}$$

顶点坐标为：

$$\left(-\frac{b}{2a},\ \frac{4ac - b^2}{4a}\right)$$

顶点是抛物线上距对称轴最近的唯一点，也是函数取得最大值或最小值的点。以 y = 2x² − 4x + 1 为例，a = 2，b = −4，顶点横坐标 = −(−4)/(2×2) = 1，纵坐标 = 2(1)² − 4(1) + 1 = −1，故顶点为 (1, −1)，函数最小值为 −1。

### 对称轴

抛物线的对称轴是一条**竖直直线**，方程为：

$$x = -\frac{b}{2a}$$

对称轴将抛物线左右对称地分成两半。对于 y = ax² + bx + c，若已知图像上关于对称轴对称的两点 x₁ 和 x₂，则 x₁ + x₂ = −b/a。这一性质常用于：当已知二次函数两个零点（即一元二次方程两根 x₁、x₂）时，对称轴方程为 x = (x₁ + x₂)/2，与韦达定理中 x₁ + x₂ = −b/a 完全吻合。

### 与 x 轴的交点（零点）

将 y = 0，得一元二次方程 ax² + bx + c = 0，其判别式 **Δ = b² − 4ac** 决定了抛物线与 x 轴的位置关系：

- Δ > 0：抛物线与 x 轴有**两个不同交点**，坐标为 $\left(\frac{-b \pm \sqrt{\Delta}}{2a},\ 0\right)$
- Δ = 0：抛物线与 x 轴**相切**（只有一个交点），切点即顶点
- Δ < 0：抛物线与 x 轴**无交点**，整条曲线在 x 轴上方（a > 0 时）或下方（a < 0 时）

### 图像的平移规律

以 y = ax²（顶点在原点）为基础图像：
- 向右平移 h 个单位：y = a(x − h)²
- 向上平移 k 个单位：y = ax² + k
- 组合平移后即为一般顶点式 y = a(x − h)² + k，顶点为 (h, k)

## 实际应用

**最值问题**：某商品原价100元，每降价 x 元销量增加 20x 件，原销量200件，则收入 y = (100 − x)(200 + 20x) = −20x² + 1800x + 20000。对称轴 x = −1800/(2×(−20)) = 45，即降价45元时收入最大，最大收入为 −20(45)² + 1800(45) + 20000 = 60500 元。

**抛体运动**：忽略空气阻力，竖直方向位移 y = v₀t − ½gt²，这正是关于时间 t 的二次函数，其中 a = −g/2 < 0，抛物线开口向下，顶点对应最高点，顶点横坐标 t = v₀/g 即为到达最高点所需时间。

**围栏问题**：用总长 L 米的围栏围成矩形，设一边长为 x，则面积 S = x·(L/2 − x) = −x² + Lx/2，顶点在 x = L/4 处，最大面积为 L²/16，即正方形时面积最大。

## 常见误区

**误区一：混淆 a 与图形宽窄的关系**
许多学生认为 a 越大图形越宽，实际上恰好相反。|a| 越大，x² 的系数绝对值越大，图形在 y 方向"拉伸"越快，视觉上反而越**窄**。例如 y = 5x² 的开口比 y = x² 窄得多，而 y = 0.1x² 的开口则宽得多。

**误区二：将对称轴与 y 轴混淆**
只有当 b = 0 时，对称轴才是 y 轴（x = 0）。若 b ≠ 0，对称轴偏离 y 轴。例如 y = x² + 4x + 3 的对称轴是 x = −2，而非 y 轴。判断对称轴必须用公式 x = −b/(2a)，不可仅凭图形直觉。

**误区三：顶点纵坐标直接用 c 代替**
c 是二次函数与 **y 轴**的交点纵坐标（令 x = 0 得 y = c），而非顶点的纵坐标。顶点纵坐标为 (4ac − b²)/(4a)，只有当 b = 0 时，顶点才在 y 轴上，此时顶点纵坐标恰好等于 c。

## 知识关联

**前置知识**：二次函数的零点问题直接调用**一元二次方程**的求根公式和判别式 Δ = b² − 4ac；配方法将一般式转化为顶点式，需要熟练的完全平方公式 (a ± b)² = a² ± 2ab + b²。

**后续拓展**：掌握二次函数后，**函数的性质**（单调性、奇偶性、值域）将以二次函数为典型例子展开讨论——例如 y = x² 在 x < 0 上单调递减、x > 0 上单调递增，对称轴 x = 0 是单调性的分界线。二次函数的图像即**抛物线**，进入解析几何后将从焦点、准线等几何角度重新定义抛物线，两者描述的是同一曲线族，但视角从代数转向纯几何。此外，二次函数是研究**二次不等式**（如 ax² + bx + c > 0 的解集）的直接依据，解集的形式由 Δ 和 a 的符号共同决定。
