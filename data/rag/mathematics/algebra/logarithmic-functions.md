# 对数函数

## 概述

对数函数是代数领域中的重要概念，难度等级为 5/9。

对数的定义、运算法则、换底公式

## 核心内容

**对数**是指数的逆运算：若 $a^x = N$（$a > 0, a \neq 1$），则 $x = \log_a N$。

**运算法则**：
- $\log_a(MN) = \log_a M + \log_a N$
- $\log_a\frac{M}{N} = \log_a M - \log_a N$
- $\log_a M^n = n\log_a M$

**换底公式**: $\log_a b = \frac{\ln b}{\ln a}$

## 例题与练习

**例**: $\log_2 8 = 3$（因为 $2^3 = 8$）

**例**: $\log_3 81 = \log_3 3^4 = 4$
