# 等差数列

## 概述

等差数列（Arithmetic Sequence）是指从第二项起，每一项与前一项的差等于同一常数的数列。这个常数称为**公差**，通常用字母 $d$ 表示。例如：1, 4, 7, 10, 13 是公差为3的等差数列；20, 15, 10, 5 是公差为 −5 的等差数列；而 5, 5, 5, 5 是公差为0的常数数列，也属于等差数列的特殊情况。

等差数列的研究可追溯到古代。公元前3世纪，阿基米德已在著作中使用等差数列求和的方法。中国古代数学著作《九章算术》（约公元1世纪）中也记录了对等差数列的计算方法。1796年，年仅9岁的高斯（Carl Friedrich Gauss）在小学课堂上快速算出1到100所有整数之和为5050，正是利用了等差数列前 $n$ 项和的对称性原理，这个故事成为等差数列历史上最著名的趣闻之一。

等差数列在初等代数中的地位体现在它是研究线性增长模式的基本工具。当一个量以固定步长均匀变化时，该量在离散时间点上的值就构成等差数列，这使得等差数列成为理解线性函数在离散领域的对应物。

## 核心原理

### 通项公式

设等差数列的首项为 $a_1$，公差为 $d$，则第 $n$ 项（通项）公式为：

$$a_n = a_1 + (n-1)d$$

其中 $n$ 为正整数，$a_1$ 为第一项，$d$ 可为任意实数（包括负数和零）。该公式的推导基于逐项累加：$a_n = a_1 + d + d + \cdots + d$（共 $n-1$ 个 $d$）。

通项公式也可以用任意已知项来改写。若已知第 $m$ 项为 $a_m$，则：

$$a_n = a_m + (n-m)d$$

这说明等差数列的任意两项之差等于公差乘以项数之差：$a_n - a_m = (n-m)d$。利用此性质，只需知道任意两项及其项数，便可完全确定整个等差数列。

### 前 $n$ 项和公式

等差数列前 $n$ 项之和 $S_n$ 有两种等价的表达形式：

$$S_n = \frac{n(a_1 + a_n)}{2} \quad \text{（首末项形式）}$$

$$S_n = na_1 + \frac{n(n-1)}{2}d \quad \text{（首项公差形式）}$$

第一个公式的几何意义十分直观：$n$ 项之和等于首项与末项的平均值乘以项数，与梯形面积公式 $S = \frac{(上底+下底) \times 高}{2}$ 在结构上完全吻合，因此等差数列的求和有时也被称为"梯形公式"。

高斯的方法正是第一个公式的体现：将1到100的等差数列首尾配对，每对之和均为101，共50对，故总和为 $101 \times 50 = 5050$。用公式验证：$S_{100} = \frac{100 \times (1+100)}{2} = 5050$，完全吻合。

### 等差中项

若三个数 $a$、$b$、$c$ 成等差数列，则 $b$ 称为 $a$ 与 $c$ 的**等差中项**，满足：

$$2b = a + c \quad \Leftrightarrow \quad b = \frac{a+c}{2}$$

推广到整个等差数列中，对称位置的两项之和相等：若数列共 $n$ 项，则 $a_k + a_{n+1-k} = a_1 + a_n$（常数）。这一性质是 $S_n$ 首末项公式的代数根基，也是处理"等差数列中各项之和"类问题的核心工具。

### $S_n$ 与 $n$ 的关系

将 $S_n = na_1 + \frac{n(n-1)}{2}d$ 整理成关于 $n$ 的函数：

$$S_n = \frac{d}{2}n^2 + \left(a_1 - \frac{d}{2}\right)n$$

可见 $S_n$ 是关于 $n$ 的**二次函数**（当 $d \neq 0$ 时）或线性函数（当 $d = 0$ 时），且该二次函数的常数项为0，即图像过原点。当 $d > 0$ 时，抛物线开口向上，$S_n$ 先减后增或单调递增；当 $d < 0$ 时，开口向下，$S_n$ 先增后减，此时存在最大值。

## 实际应用

**分期付款与储蓄计划**：若某人每月定额存款，月存款额构成公差为0的等差数列（常数列），$n$ 个月后总存款用 $S_n = na$ 计算。若每月递增固定金额，则 $S_n$ 用二次公式求解。

**工程中的等间距布局**：路灯安装时，若灯柱从起点开始每隔8米设一根，第 $n$ 根灯柱距起点的距离为 $a_n = 8(n-1)$ 米，这是首项为0、公差为8的等差数列。

**竞赛题型——已知和求最值**：已知等差数列 $\{a_n\}$ 中，$a_1 = 20$，$d = -2$，求 $S_n$ 的最大值。此时 $S_n = 20n - n(n-1) = -n^2 + 21n$，对称轴为 $n = 10.5$，故在 $n = 10$ 或 $n = 11$ 时取得最大值 $S_{10} = S_{11} = 110$。

## 常见误区

**误区一：公差 $d$ 必须为正数。** 事实上，公差可以是任意实数。当 $d < 0$ 时，数列单调递减；当 $d = 0$ 时，数列为常数列。三种情况均满足等差数列定义，学生往往忽略 $d \leq 0$ 的情形而在最值问题中出错。

**误区二：混淆通项公式中的指标。** 公式 $a_n = a_1 + (n-1)d$ 中，系数是 $(n-1)$ 而非 $n$，因为首项本身不需要加公差。将其误写为 $a_n = a_1 + nd$ 会导致每项的值偏大一个 $d$，这是初学者最常见的笔误，在计算第5项或第10项时尤其容易验算发现。

**误区三：认为 $S_n/n = a_n$。** $S_n/n$ 是前 $n$ 项的**算术平均值**，等于等差数列的中间项 $a_{(n+1)/2}$（$n$ 为奇数时），而非第 $n$ 项。将平均值与通项混淆会导致错误地用均值代替末项参与计算。

## 知识关联

**前置概念——数列基础**：等差数列是在掌握数列定义（有序数的集合）、下标符号 $a_n$ 以及递推关系的基础上引入的第一类特殊数列。等差数列的递推定义 $a_{n+1} - a_n = d$（常数）直接使用了数列的递推表达方式。

**并行概念——等比数列**：等差数列以"加法"定义（相邻项差为常数），等比数列以"乘法"定义（相邻项比为常数）。两者的通项公式与求和公式在结构上形成对照，等差对应线性增长，等比对应指数增长。学习等比数列时，$S_n = a_1\cdot\frac{1-q^n}{1-q}$ 的推导方法（错位相减法）与等差求和方法（倒序相加法）形成清晰对比。

**后续概念——级数与求和**：等差数列前 $n$ 项和 $S_n$ 是有限项求和的入门案例。在级数理论中，当公差 $d \neq 0$ 时，等差级数（无穷项求和）因为通项 $a_n \to \pm\infty$ 而必然发散，这是理解"级数收敛"概念的反例之一，帮助建立对无穷求和行为的直觉。
