# 因式分解

## 概述

因式分解是将一个多项式写成若干个整式乘积形式的代数运算，与多项式乘法互为逆运算。例如，将 $x^2 - 5x + 6$ 化为 $(x-2)(x-3)$ 的过程就是因式分解。这一运算的结果必须是**积**的形式，若结果仍含加减号连接的多个项，则尚未完成因式分解。

因式分解的思想最早可追溯至古希腊时期欧几里得对整数分解的研究，但系统的多项式因式分解理论在17至18世纪随着代数符号体系成熟而确立。数学家高斯（Carl Friedrich Gauss）在1799年的博士论文中严格证明了代数基本定理，奠定了多项式可分解性的理论基础。

因式分解是求解一元二次方程的核心工具。方程 $ax^2 + bx + c = 0$ 一旦左边能分解为 $(px+q)(rx+s)=0$，便可直接利用零积法则得出两个根，避免了复杂的配方运算。此外，分式化简、不等式求解、函数零点分析都依赖因式分解技术。

## 核心原理

### 1. 提公因式法

若多项式各项都含有公共因子，则将该公因子提到括号外。公因式包括**数字公因子**和**字母公因子**两部分，必须取各项系数的最大公约数（GCD）与各项中次数最低的字母幂次之积。

**例：** $6x^3y^2 - 9x^2y^3 + 3x^2y$

- 系数 6、9、3 的 GCD 为 3
- 字母 $x$ 的最低次数为 $x^2$，$y$ 的最低次数为 $y^1$
- 公因式为 $3x^2y$，结果：$3x^2y(2xy - 3y^2 + 1)$

提完公因式后，括号内各项系数应无公约数，且不含可继续提取的公共字母幂次。

### 2. 公式法

公式法利用已知乘法恒等式逆向分解，最常用的三个公式为：

| 公式名称 | 正向（展开） | 逆向（分解） |
|---|---|---|
| 平方差公式 | $(a+b)(a-b) = a^2 - b^2$ | $a^2 - b^2 = (a+b)(a-b)$ |
| 完全平方公式（和） | $(a+b)^2 = a^2 + 2ab + b^2$ | $a^2 + 2ab + b^2 = (a+b)^2$ |
| 完全平方公式（差） | $(a-b)^2 = a^2 - 2ab + b^2$ | $a^2 - 2ab + b^2 = (a-b)^2$ |

**识别完全平方式的关键判断条件**：首末两项为完全平方数，中间项等于首末两项平方根之积的 2 倍。例如 $4x^2 + 12x + 9$，首项 $4x^2 = (2x)^2$，末项 $9 = 3^2$，中间项 $12x = 2 \times 2x \times 3$，满足条件，故分解为 $(2x+3)^2$。

平方差公式要求两项符号相反，且均可写成某式的平方。$x^2 + y^2$ **不能**用平方差公式分解（在实数范围内无法因式分解）。

### 3. 十字相乘法

十字相乘法专门用于分解形如 $ax^2 + bx + c$（$a \neq 0$）的二次三项式，其核心思路是将 $ax^2+bx+c$ 拆解为 $(p_1x + q_1)(p_2x + q_2)$，需同时满足：

$$p_1 \cdot p_2 = a, \quad q_1 \cdot q_2 = c, \quad p_1q_2 + p_2q_1 = b$$

操作时画"十字"，左列写 $p_1, p_2$（乘积为 $a$），右列写 $q_1, q_2$（乘积为 $c$），交叉相乘之和等于 $b$ 时即为正确分组。

**例：** 分解 $2x^2 + 7x + 3$

- $a=2$，$c=3$，$b=7$
- 左列尝试 $2, 1$；右列尝试 $1, 3$
- 交叉：$2×3 + 1×1 = 7$ ✓
- 结果：$(2x+1)(x+3)$

当 $a=1$ 时，十字相乘退化为**找两数之积为 $c$、之和为 $b$** 的简单情形。

## 实际应用

**应用一：化简代数分式**

$$\frac{x^2 - 9}{x^2 + 5x + 6} = \frac{(x+3)(x-3)}{(x+2)(x+3)} = \frac{x-3}{x+2} \quad (x \neq -3)$$

分子用平方差公式，分母用十字相乘法（或观察法），约分后得到最简形式，同时须注明使分母为零的限制条件 $x \neq -3$。

**应用二：不完全分解的递进处理**

$x^4 - 16$ 不能只分解一步。先用平方差：$(x^2+4)(x^2-4)$，再对 $x^2-4$ 继续用平方差：$(x^2+4)(x+2)(x-2)$。因式分解要求分解到**每个因式在整数范围内不可再分**为止，$x^2+4$ 在实数范围无法继续分解，故最终结果为 $(x^2+4)(x+2)(x-2)$。

**应用三：求多项式的零点**

已知 $f(x) = 3x^2 - 10x + 8$，先用十字相乘分解为 $(3x-4)(x-2)$，令 $f(x)=0$，得 $x = \frac{4}{3}$ 或 $x=2$，这正是后续解一元二次方程的标准流程。

## 常见误区

**误区一：分解不彻底**

$2x^2 - 8 = 2(x^2 - 4)$ 并未结束，还需继续：$= 2(x+2)(x-2)$。学生往往在提出数字公因式后忽略括号内还能继续分解，导致答案丢分。判断标准是括号内每个多次因式是否仍可套用公式或十字相乘。

**误区二：将 $a^2 + b^2$ 误认为完全平方式**

$a^2 + 2ab + b^2 = (a+b)^2$ 成立，但 $a^2 + b^2$ 在实数系数范围内**不可因式分解**。学生常将 $x^2 + 4$ 错误写成 $(x+2)^2$，而实际上 $(x+2)^2 = x^2 + 4x + 4 \neq x^2 + 4$。

**误区三：十字相乘符号判断错误**

分解 $x^2 - 5x + 6$ 时，两数之积为正 6、之和为负 5，应为 $(-2)×(-3)=6$，$(-2)+(-3)=-5$，结果是 $(x-2)(x-3)$。若忽视负号，错写为 $(x+2)(x+3)$ 展开得 $x^2+5x+6$，符号完全相反。处理常数项为正、一次项系数为负的情形时，两个常数因子必须均取负值。

## 知识关联

**前置知识：多项式**

因式分解的操作对象是多项式，识别各项系数、次数及提取公因式均需熟练掌握多项式的结构。完全平方公式和平方差公式本质上是多项式乘法公式的逆用，若不能流畅展开 $(a\pm b)^2$ 和 $(a+b)(a-b)$，则逆向识别会产生障碍。

**后续概念：一元二次方程**

解方程 $ax^2+bx+c=0$ 的"因式分解法"直接以本章技能为基础：将左边分解为两个一次因式之积，再由零积法则分别令每个因式为零。这比求根公式 $x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$ 更直观，且当判别式 $\Delta = b^2 - 4ac$ 为完全平方数时，因式分解法能给出整数或有理数根，无需处理根号。
