# 复数初步

## 概述

复数是实数系的扩展，其核心出发点是解决实数范围内无法解决的问题：方程 $x^2 + 1 = 0$ 在实数集中无解。数学家为此引入了**虚数单位** $i$，规定 $i^2 = -1$，从而将数系从实数扩展到复数集 $\mathbb{C}$。一个标准复数的形式为 $z = a + bi$，其中 $a$ 是实部（记作 $\text{Re}(z)$），$b$ 是虚部（记作 $\text{Im}(z)$），$a, b \in \mathbb{R}$。

历史上，16世纪意大利数学家卡尔达诺（Gerolamo Cardano）在1545年出版的《大术》（*Ars Magna*）中首次触碰了负数开方的问题，但他将此视为"诡辩"而非真实存在的数。直到18世纪，欧拉（Euler）在1748年引入符号 $i$ 代表 $\sqrt{-1}$，高斯（Gauss）进一步建立了复平面的几何直觉，复数才获得严格的数学地位。

复数的意义远不止解二次方程。在一元二次方程 $ax^2 + bx + c = 0$ 中，当判别式 $\Delta = b^2 - 4ac < 0$ 时，两个复数根为 $\dfrac{-b \pm \sqrt{4ac - b^2}\,i}{2a}$，它们互为共轭复数。理解复数使学生能够完整描述所有二次方程的根，并为后续学习复数三角形式、欧拉公式乃至傅里叶变换打下基础。

## 核心原理

### 虚数单位 $i$ 的幂次循环

$i$ 的整数次幂呈现严格的**4周期循环**规律：

$$i^1 = i,\quad i^2 = -1,\quad i^3 = -i,\quad i^4 = 1,\quad i^5 = i, \ldots$$

计算 $i^n$ 时，只需将 $n$ 除以4取余数：$i^n = i^{n \bmod 4}$。例如，$i^{2024} = i^{2024 \bmod 4} = i^0 = 1$。负次幂同理：$i^{-1} = \dfrac{1}{i} = \dfrac{i}{i^2} = -i$。这一周期性是复数运算中处理高次幂的基本工具，与无理数的开方运算有本质不同。

### 复数的四则运算

设 $z_1 = a + bi$，$z_2 = c + di$，四则运算规则如下：

- **加减法**：实部与虚部分别相加减  
  $z_1 \pm z_2 = (a \pm c) + (b \pm d)i$

- **乘法**：按多项式展开，并用 $i^2 = -1$ 化简  
  $z_1 \cdot z_2 = (ac - bd) + (ad + bc)i$

- **除法**：分子分母同乘以分母的共轭复数  
  $\dfrac{z_1}{z_2} = \dfrac{(a+bi)(c-di)}{c^2 + d^2} = \dfrac{ac+bd}{c^2+d^2} + \dfrac{bc-ad}{c^2+d^2}i$

其中 $\bar{z}_2 = c - di$ 称为 $z_2$ 的**共轭复数**，满足 $z_2 \cdot \bar{z}_2 = c^2 + d^2 \in \mathbb{R}$，这一性质专门用于将除法的分母有理化（此处应称"实数化"）。

### 复平面与复数的几何表示

复数 $z = a + bi$ 可在平面直角坐标系中唯一对应点 $(a, b)$，这一平面称为**复平面**（又称高斯平面）。横轴为**实轴**，纵轴为**虚轴**（纯虚数如 $3i$ 位于虚轴上）。

复数 $z = a + bi$ 的**模**定义为：

$$|z| = \sqrt{a^2 + b^2}$$

即原点到点 $(a, b)$ 的距离。模具有性质 $|z_1 \cdot z_2| = |z_1| \cdot |z_2|$，以及 $|z| = |\bar{z}|$。两个复数相等的充要条件是实部相等**且**虚部相等，即 $a + bi = c + di \Leftrightarrow a = c$ 且 $b = d$，这与实数的"一维比较"不同——复数本身**没有大小关系**，无法比较 $z_1 > z_2$。

## 实际应用

**求解负数开方**：化简 $\sqrt{-9}$ 时，先写为 $\sqrt{9 \times (-1)} = 3i$。注意 $\sqrt{-2} \cdot \sqrt{-3} \neq \sqrt{6}$，正确做法是 $\sqrt{-2} \cdot \sqrt{-3} = (\sqrt{2}\,i)(\sqrt{3}\,i) = \sqrt{6}\,i^2 = -\sqrt{6}$，直接合并根号的公式在负数范围内失效。

**解含复数的方程**：若 $z = x + yi$（$x, y \in \mathbb{R}$）满足 $(2 + i)z = 3 - i$，则：
$$z = \frac{3-i}{2+i} = \frac{(3-i)(2-i)}{(2+i)(2-i)} = \frac{6 - 3i - 2i + i^2}{5} = \frac{5 - 5i}{5} = 1 - i$$

**在复平面中的几何操作**：将复数 $z = 1 + 2i$ 乘以 $i$，得到 $iz = i + 2i^2 = -2 + i$。在复平面上观察，点 $(1, 2)$ 变为 $(-2, 1)$，恰好是绕原点**逆时针旋转90°**的结果。这揭示了乘以 $i$ 的几何意义，是复数三角形式中旋转运算的几何直觉来源。

## 常见误区

**误区一：虚数不是"真实的数"，不能参与等式运算**  
这是历史偏见的遗留。复数 $a + bi$ 的相等、加减法均有严格定义，复数系满足加法交换律、结合律和乘法分配律等全部域公理。$i$ 不是"假想"的，而是有严格代数定义的数学对象：$i := (0, 1)$，乘法规则为 $(a, b)(c, d) = (ac - bd, ad + bc)$。

**误区二：可以对负数直接使用根号乘法公式**  
实数中成立的 $\sqrt{a} \cdot \sqrt{b} = \sqrt{ab}$（$a, b \geq 0$）在负数范围**不成立**。正确做法是先将 $\sqrt{-a}$（$a > 0$）写成 $\sqrt{a}\,i$ 的形式，再进行运算。混用该公式会导致 $1 = \sqrt{1} = \sqrt{(-1)(-1)} = \sqrt{-1}\cdot\sqrt{-1} = i^2 = -1$ 这样的荒谬结论。

**误区三：虚部为零的复数不是复数**  
$a + 0i = a$ 是实数，实数集 $\mathbb{R}$ 是复数集 $\mathbb{C}$ 的**真子集**（$\mathbb{R} \subsetneq \mathbb{C}$）。同样，$b \neq 0$ 时 $a + bi$ 称为虚数，$a = 0$ 且 $b \neq 0$ 时称为**纯虚数**（如 $3i$）。这四类（实数、纯虚数、非实的虚数、复数全集）之间的包含关系是分类题的高频考点。

## 知识关联

**与前置知识的联系**：复数直接建立在一元二次方程的基础上——正是 $\Delta < 0$ 时方程无实数解的困境催生了复数的定义。无理数的引入（如 $\sqrt{2}$ 扩充了有理数系）与复数的引入逻辑相同：遇到现有数系内无解的运算，便通过定义新元素来扩充数系，体现了数系扩展的统一模式。

**与后续知识的衔接**：本节建立的模 $|z|$ 和复平面坐标 $(a, b)$ 是**复数三角形式** $z = r(\cos\theta + i\sin\theta)$ 的直接基础，其中 $r = |z|$，$\theta$ 为辐角。三角形式将乘法转化为模相乘、辐角相加，彻底揭示乘以 $i$ 等于旋转90°的几何本质，并最终通向欧拉公式 $e^{i\theta} = \cos\theta + i\sin\theta$。
