# 绝对值函数

## 概述

绝对值函数定义为 $f(x) = |x|$，其数学含义是取数轴上点 $x$ 到原点的距离。形式化定义为分段函数：当 $x \geq 0$ 时 $f(x) = x$，当 $x < 0$ 时 $f(x) = -x$。这一定义最早可追溯至魏尔斯特拉斯（Karl Weierstrass）在19世纪对实数分析的系统化整理，他在构建严格的极限理论时将绝对值作为度量距离的核心工具 [Weierstrass, 1872]。绝对值函数的图像呈经典的"V形"，顶点位于原点 $(0, 0)$，两侧射线斜率分别为 $+1$ 和 $-1$，关于 $y$ 轴对称，是典型的偶函数。

绝对值函数在代数、分析和优化领域均有不可替代的地位。在不等式研究中，$|x - a| < \delta$ 精确描述了"$x$ 与 $a$ 的距离小于 $\delta$"这一几何直觉，使得 $\varepsilon$-$\delta$ 语言成为可能。在统计与机器学习中，绝对值损失函数（L1损失，即 $L = \sum_{i=1}^n |y_i - \hat{y}_i|$）相比平方损失对离群点更鲁棒，这是其实用价值的直接体现。

## 核心原理

### 图像变换规则

绝对值函数的图像变换遵循函数变换的一般规律，但因"折叠"特性而呈现出独特的几何效果。对于 $f(x) = a|x - h| + k$，参数 $a$ 控制两侧射线的斜率（$|a|$ 越大，V形越窄），$h$ 控制顶点的水平位置，$k$ 控制顶点的垂直位置，顶点坐标固定为 $(h, k)$。当 $a < 0$ 时，图像翻转为倒V形，开口朝下。

更复杂的变换是 $f(x) = |g(x)|$，其几何意义是将 $g(x)$ 图像中位于 $x$ 轴下方的部分（即 $g(x) < 0$ 的区间）沿 $x$ 轴翻折到上方。例如，$y = |x^2 - 4|$ 的图像由抛物线 $y = x^2 - 4$ 生成：在 $x \in (-2, 2)$ 内抛物线取负值，翻折后该段变为向上的弓形，图像在 $x = \pm 2$ 处有尖角（不可微点）。

### 绝对值不等式的解法

绝对值不等式的求解依赖于两条基本公式：

$$|x| < a \iff -a < x < a \quad (a > 0)$$

$$|x| > a \iff x < -a \text{ 或 } x > a \quad (a > 0)$$

由此衍生出含参不等式 $|x - c| < r$ 的解集为开区间 $(c - r,\ c + r)$，几何上即以 $c$ 为圆心、$r$ 为半径的"区间球"。解含绝对值的不等式时，标准做法是**分情况讨论**：令绝对值内的表达式分别大于零和小于零，各自建立线性不等式求解后取交集或并集。

对于双绝对值不等式，如 $|x + 1| + |x - 3| \geq 6$，可利用**三角不等式**：$|a + b| \leq |a| + |b|$，将左侧理解为数轴上点 $x$ 到 $-1$ 与到 $3$ 的距离之和（总距离为4），结合几何分析快速确定解集为 $x \leq -2$ 或 $x \geq 4$。

### 绝对值函数的不可微性与分析性质

在顶点 $x = h$ 处，$f(x) = |x - h|$ 的左导数为 $-1$，右导数为 $+1$，两者不相等，因此顶点处函数不可微 [Rudin, 1976]。这一性质在优化问题中至关重要：L1正则化（Lasso回归）中，绝对值项在零点不可微，导致梯度下降必须使用次梯度（subgradient）方法，次梯度集合为 $[-1, 1]$。

绝对值函数满足以下代数恒等式：
- $|ab| = |a| \cdot |b|$（乘法分配律）
- $|a + b| \leq |a| + |b|$（三角不等式，等号当且仅当 $ab \geq 0$ 时成立）
- $\bigl||a| - |b|\bigr| \leq |a - b|$（逆三角不等式）

这三条性质构成了绝对值作为实数轴上度量（metric）的完整公理体系。

## 实际应用

**误差控制与精度分析：** 在数值计算中，绝对误差定义为 $\Delta x = |x_{\text{近似}} - x_{\text{真实}}|$，相对误差为 $\delta x = \Delta x / |x_{\text{真实}}|$。中学物理实验中要求"读数误差不超过最小刻度的一半"，本质上是要求 $|\text{读数} - \text{真值}| \leq 0.5 \times \text{最小刻度}$，直接使用绝对值不等式建模。

**Lasso 回归中的稀疏性：** 统计学习理论中，Lasso 回归目标函数为 $\min_{\beta} \|y - X\beta\|_2^2 + \lambda \sum_{j=1}^p |\beta_j|$，绝对值惩罚项在零点的"尖角"使得最优解倾向于将部分系数精确压缩为零，产生稀疏模型 [Tibshirani, 1996]。这是绝对值函数不可微性在统计中的直接应用。

**距离问题的代数化：** 在数轴上求"到两个定点距离之差不超过某常数"的动点范围时，设两定点为 $A(a)$、$B(b)$，条件写作 $\bigl||x-a| - |x-b|\bigr| \leq c$，利用上述不等式性质即可转化为含参线性不等式组求解，典型题型如"求满足 $||x-1| - |x+3|| \leq 2$ 的 $x$ 的范围"。

## 常见误区

**误区一：认为 $|x^2| = x^2$ 总成立而忽视复合结构。** 对实数而言 $x^2 \geq 0$ 确实使 $|x^2| = x^2$，但对复数或含参表达式，如 $|\sin^2\theta - 1| \neq \sin^2\theta - 1$（因为 $\sin^2\theta - 1 \leq 0$），需要判断内部表达式的符号后再去绝对值。直接"去掉绝对值符号"是高频错误。

**误区二：将 $|x| = -x$ 等同于"$x$ 为负数"。** 实际上 $|x| = -x$ 的充要条件是 $x \leq 0$（包含零），因为 $|0| = 0 = -0$。在分类讨论时漏掉 $x = 0$ 这个临界点会导致解集计算错误，尤其在解 $|f(x)| = -f(x)$ 类型问题时最为常见。

**误区三：对绝对值不等式两侧平方时忽略条件。** 解 $|x - 2| < x$ 时，部分学习者直接两侧平方得 $(x-2)^2 < x^2$，展开得 $-4x + 4 < 0$，得 $x > 1$。但原不等式右侧为 $x$，当 $x \leq 0$ 时不等式无解（因为绝对值恒非负），所以必须先确定 $x > 0$ 的前提条件，最终正确解集为 $x > 1$（此例结果偶然相同，但方法缺陷在更复杂题目中会导致增根）。
