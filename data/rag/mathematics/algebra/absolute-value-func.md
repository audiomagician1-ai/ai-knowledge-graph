# 绝对值函数

## 概述

绝对值函数 $f(x) = |x|$ 是代数中将"方向信息"剥离、只保留"距离信息"的典型装置。其严格数学定义为：

$$|x| = \begin{cases} x, & x \geq 0 \\ -x, & x < 0 \end{cases}$$

定义域为 $\mathbb{R}$，值域为 $[0, +\infty)$，最小值 $f(0) = 0$ 在顶点 $(0,0)$ 取得。绝对值的几何本质是数轴上点 $x$ 到原点的距离；推广形式 $|a - b|$ 则表示实数轴上两点 $a$、$b$ 之间的距离，这一几何解释是求解一切绝对值不等式的逻辑起点。

绝对值函数是分段函数的最简原型，也是高中数学与大学分析之间的重要衔接：在高中阶段，它是图像变换与不等式求解的训练场；在实变函数理论中，它是 $L^1$ 范数 $\|f\| = \int |f(x)|\,dx$ 的雏形，并与 Lipschitz 连续性、凸函数理论深度关联（Rudin, 1976）。

## 核心原理

### 图像结构与顶点形式

$f(x) = |x|$ 的图像由两条射线组成：$x \geq 0$ 段斜率为 $+1$，$x < 0$ 段斜率为 $-1$，在原点形成"V"形拐角。图像关于 $y$ 轴对称，即 $f(-x) = f(x)$，因此 $f$ 是偶函数。

一般化的绝对值函数取**顶点式**：

$$f(x) = k|x - a| + b \quad (k \neq 0)$$

- 顶点坐标为 $(a,\, b)$，是函数的极值点；
- $k > 0$ 时图像为开口向上的"V"形，顶点为最小值点，最小值 $b$；
- $k < 0$ 时图像为开口向下的倒"V"形，顶点为最大值点，最大值 $b$；
- $|k|$ 越大，两侧射线斜率绝对值越大，图像越"陡"。

例如，$f(x) = 2|x - 3| - 1$ 的顶点为 $(3, -1)$，两侧斜率为 $\pm 2$，在 $x < 3$ 时 $f(x) = -2x + 5$，在 $x \geq 3$ 时 $f(x) = 2x - 7$。

对于含两段绝对值的函数 $f(x) = |x - a| + |x - b|$（$a < b$），其图像分三段讨论：

$$f(x) = \begin{cases} -2x + (a+b), & x < a \\ b - a, & a \leq x \leq b \\ 2x - (a+b), & x > b \end{cases}$$

在区间 $[a, b]$ 上函数取最小值 $b - a$，这一结果等价于"两点间线段最短"的几何命题。

### 绝对值不等式的标准解法

**类型一：$|f(x)| < c$（$c > 0$）**

等价于 $-c < f(x) < c$，解集为有界区间。

**类型二：$|f(x)| > c$（$c > 0$）**

等价于 $f(x) > c$ 或 $f(x) < -c$，解集为两段无穷区间的并集。

**类型三：$|f(x)| \leq g(x)$（含参数）**

需分类讨论：当 $g(x) < 0$ 时无解；当 $g(x) \geq 0$ 时化为 $-g(x) \leq f(x) \leq g(x)$。

**具体案例**：解不等式 $|2x - 3| \leq 5$。

将其化为 $-5 \leq 2x - 3 \leq 5$，即 $-2 \leq 2x \leq 8$，解得 $-1 \leq x \leq 4$，解集为 $[-1, 4]$。

另一案例：解 $|x + 2| > |x - 1|$。两边平方（两边均非负时等价）：$(x+2)^2 > (x-1)^2$，展开得 $x^2 + 4x + 4 > x^2 - 2x + 1$，化简为 $6x > -3$，即 $x > -\frac{1}{2}$。

### 三角不等式与绝对值估计

绝对值满足**三角不等式**（Triangle Inequality），这是分析学中最基础的工具之一：

$$|a + b| \leq |a| + |b|$$

等号成立的充要条件是 $ab \geq 0$（$a$、$b$ 同号或至少一方为零）。由此推导出**逆三角不等式**：

$$\bigl||a| - |b|\bigr| \leq |a - b|$$

两个不等式联立，给出绝对值表达式的双向估计框架。

**应用举例**：已知 $|x - 2| \leq 1$，估计 $|3x - 5|$ 的范围。

由条件知 $1 \leq x \leq 3$，则 $3 - 3 \leq 3x - 6 \leq 3$，即 $-3 \leq 3x - 6 \leq 3$，故 $-2 \leq 3x - 5 \leq 4$，从而 $|3x - 5| \leq 4$。

对含多个绝对值之和的最优化问题，核心结论是：$|x - a_1| + |x - a_2| + \cdots + |x - a_n|$ 在 $x$ 等于所有 $a_i$ 的中位数时取最小值（Courant & Robbins, 1941）。当 $n = 2$ 时最小值为 $|a_1 - a_2|$，当 $n$ 为奇数时最小值为 $0$（$x$ 取中间那个 $a_i$）。

### 可微性分析：不可导点的经典案例

从微积分角度看，$f(x) = |x|$ 在 $x = 0$ 处**连续但不可微**。用导数定义验证：

$$f'(0^-) = \lim_{h \to 0^-} \frac{|h| - 0}{h} = \lim_{h \to 0^-} \frac{-h}{h} = -1$$

$$f'(0^+) = \lim_{h \to 0^+} \frac{|h| - 0}{h} = \lim_{h \to 0^+} \frac{h}{h} = +1$$

左右导数不相等，故 $f$ 在 $x = 0$ 处不可导。对 $x \neq 0$，有

$$f'(x) = \frac{x}{|x|} = \text{sgn}(x) = \begin{cases} 1, & x > 0 \\ -1, & x < 0 \end{cases}$$

即导函数恰好是**符号函数** $\text{sgn}(x)$。这是高等数学中"处处连续但存在不可导点"的最简单示范，也说明了为什么在优化问题中绝对值函数需要用次梯度（subgradient）而非普通梯度来处理（Boyd & Vandenberghe, 2004）。

## 关键公式总结

| 场景 | 公式/结论 |
|---|---|
| 基本定义 | $|x| = \sqrt{x^2}$ |
| 距离解释 | $|a - b|$ 为 $a$、$b$ 在数轴上的距离 |
| 不等式（小于型） | $|x - c| < r \Leftrightarrow c - r < x < c + r$ |
| 不等式（大于型） | $|x - c| > r \Leftrightarrow x > c + r$ 或 $x < c - r$ |
| 三角不等式 | $|a + b| \leq |a| + |b|$ |
| 逆三角不等式 | $\bigl||a| - |b|\bigr| \leq |a - b|$ |
| 双绝对值之和最小值 | $\min_x (|x - a| + |x - b|) = |a - b|$ |
| 导数（$x \neq 0$） | $\frac{d}{dx}|x| = \text{sgn}(x)$ |

值得注意的是，$|x| = \sqrt{x^2}$ 这一等式将绝对值与平方根联系起来，是去绝对值符号时常用的代数技巧，尤其在含绝对值的方程两边平方求解时尤为关键。

## 实际应用

### 误差分析与精度控制

在工程测量中，若某仪器的测量值为 $\hat{x}$，真实值为 $x_0$，则**绝对误差**定义为 $|\hat{x} - x_0|$。规定精度标准 $|\hat{x} - x_0| \leq \delta$ 正是一个绝对值不等式，其解集给出了真实值的置信区间 $[\hat{x} - \delta,\, \hat{x} + \delta]$。

### 最优选址问题（1-中位数问题）

设数轴上有 $n$ 个工厂位于 $a_1, a_2, \ldots, a_n$，需选一仓库位置 $x$ 使总运输距离 $\sum_{i=1}^n |x - a_i|$ 最小。答案是**中位数**：将 $a_i$ 排序后，若 $n$ 为奇数，取第 $\lceil n/2 \rceil$ 个值；若 $n$ 为偶数，取中间两个值之间的任意点均可。

例如，工厂位于 $1, 3, 7, 8, 10$（$n=5$），中位数为 $7$，最小总距离为 $|7-1|+|7-3|+|7-7|+|7-8|+|7-10| = 6+4+0+1+3 = 14$。

### 机器学习中的 L1 损失函数

在回归问题中，**MAE（Mean Absolute Error）**损失函数定义为：

$$L = \frac{1}{n}\sum_{i=1}^n |y_i - \hat{y}_i|$$

与均方误差（MSE）相比，MAE 对离群点（outliers）更鲁棒，因为它线性惩罚误差而非平方惩罚。其梯度（次梯度）在 $y_i \neq \hat{y}_i$ 时为常数 $\pm 1$，不会因误差大而产生梯度爆炸。LASSO 回归（Tibshirani, 1996）正是在损失函数中加入 $L^1$ 正则项 $\lambda \sum |w_j|$，利用绝对值的不可微性在 $0$ 处产生稀疏解，自动实现特征选择。

### 物理中的绝对值

弹簧振动系统中，位移 $x(t)$ 的绝对值 $|x(t)|$ 给出质点到平衡位置的距离（振幅范围）。交流电的有效值（RMS）虽然用平方根定义，但其背后逻辑与绝对值函数处理"有正有负量"的思路一脉相承。

## 常见误区

**误区一：将 $|x^2 - 4| = 0$ 与 $|x - 2| = 0$ 混淆**

$|x^2 - 4| = 0$ 要求 $x^2 = 4$，解为 $x = \pm 2$；而 $|x - 2| = 0$ 仅有 $x = 2$。去掉绝对值号之前必须先确认等式内部的结构。

**误区二：两边平方去绝对值的条件**

$|f(x)| = |g(x)|$ 等价于 $f(x)^2 = g(x)^2$，即 $f(x) = g(x)$ 或 $f(x) = -g(x)$，此步骤不要求 $f$、$g$ 非负。但对**不等式** $|f(x)| < |g(x)|$，两边平方等价要求保号，需单独验证。

**误区三：忽视顶点形式中 $k < 0$ 的情形**

$f(x) = -2|x - 1| + 3$ 的顶点 $(1, 3)$ 是**最大值点**，最大值为 $3$；函数无下界（值域为 $(-\infty, 3]$）。部分学生惯性认为"V形函数有最小值"，对倒V形处理出错。
