# 绝对值函数

## 概述

绝对值函数 $f(x) = |x|$ 的严格数学定义为：

$$|x| = \begin{cases} x, & x \geq 0 \\ -x, & x < 0 \end{cases}$$

定义域为 $\mathbb{R}$，值域为 $[0, +\infty)$，唯一最小值 $f(0) = 0$ 在顶点 $(0, 0)$ 取得。绝对值的几何本质是数轴上点 $x$ 到原点的有向距离去向后的无向量值；推广形式 $|a - b|$ 精确表示实数轴上两点 $a$、$b$ 之间的距离，这一几何解释是求解一切绝对值不等式的逻辑基础。

绝对值函数在历史上可追溯至 Weierstrass 在19世纪70年代对连续性的精密化处理——他在给出 $\varepsilon$-$\delta$ 语言时大量使用 $|f(x) - L|$ 的形式来度量函数值与极限的偏差。在实变函数论框架下，$|x|$ 是 $L^1$ 范数 $\|f\|_1 = \int |f(x)|\,dx$ 的雏形，也是Lipschitz条件 $|f(x) - f(y)| \leq L|x - y|$ 的核心构件（Rudin, 1976）。在高中数学层面，绝对值函数是分段函数的最简原型，图像变换与不等式求解构成其两条主线。

绝对值函数还具有若干精确的分析性质：在 $x \neq 0$ 处可微，导数为符号函数 $\text{sgn}(x)$；在 $x = 0$ 处左导数为 $-1$、右导数为 $+1$，故 $x=0$ 处不可微但连续；$f(x) = |x|$ 是凸函数，满足 $f(\lambda x + (1-\lambda)y) \leq \lambda f(x) + (1-\lambda)f(y)$ 对一切 $\lambda \in [0,1]$ 成立（Boyd & Vandenberghe, 2004）。

## 核心原理

### 图像结构与顶点式

$f(x) = |x|$ 的图像由两条射线拼接而成：在 $x \geq 0$ 侧斜率为 $+1$，在 $x < 0$ 侧斜率为 $-1$，两段在原点以"V"形拐角相接。图像关于 $y$ 轴对称，即 $f(-x) = f(x)$，因此是偶函数。

一般化形式取**顶点式**：

$$f(x) = k|x - a| + b \quad (k \neq 0)$$

其中顶点坐标为 $(a,\, b)$：
- $k > 0$ 时，顶点为最小值点，最小值为 $b$，图像为开口向上的"V"形，两侧射线斜率分别为 $+k$ 与 $-k$；
- $k < 0$ 时，顶点为最大值点，最大值为 $b$，图像为倒"V"形；
- $|k|$ 决定图像的"陡峭程度"：$|k| = 1$ 时两侧夹角为 $90°$，$|k| > 1$ 时夹角更小（更陡）。

**例如**，$f(x) = 2|x - 3| - 1$ 的顶点为 $(3, -1)$，两侧斜率为 $\pm 2$。用去绝对值展开：

$$f(x) = \begin{cases} -2x + 5, & x < 3 \\ 2x - 7, & x \geq 3 \end{cases}$$

在 $x = 0$ 时 $f(0) = 5$，在 $x = 3$ 时 $f(3) = -1$（最小值），在 $x = 6$ 时 $f(6) = 5$。

### 双绝对值函数的三段结构

对 $f(x) = |x - a| + |x - b|$（$a < b$），需以 $a$ 和 $b$ 为分界点分三段讨论：

$$f(x) = \begin{cases} -2x + (a+b), & x < a \\ b - a, & a \leq x \leq b \\ 2x - (a+b), & x > b \end{cases}$$

在区间 $[a, b]$ 上函数取常数最小值 $b - a$，这等价于几何命题"两点之间线段最短"——$x$ 可视为数轴上的动点，$|x-a| + |x-b|$ 是它到两定点 $a$、$b$ 的距离之和，当 $x$ 位于两点之间时距离之和最小，恰为 $b - a$。

进一步，对三绝对值函数 $f(x) = |x - a| + |x - b| + |x - c|$（$a < b < c$），最小值在中间值 $x = b$ 处取得，最小值为 $(c - a)$。这一结论推广至 $n$ 个绝对值之和时，最小值在所有分界点的**中位数**处取得，是统计学中中位数最小化 $L^1$ 损失的数学来源。

### 绝对值不等式的四种标准类型

**类型一：$|f(x)| < c$（$c > 0$）**

等价于 $-c < f(x) < c$，解集为有界区间。若 $c \leq 0$，则无解。

**类型二：$|f(x)| > c$（$c > 0$）**

等价于 $f(x) > c$ 或 $f(x) < -c$，解集为两段无穷区间的并集。

**类型三：$|f(x)| \leq g(x)$（$g(x)$ 含变量）**

需先讨论 $g(x)$ 的符号：当 $g(x) < 0$ 时该区域无解；当 $g(x) \geq 0$ 时等价于 $-g(x) \leq f(x) \leq g(x)$，再与 $g(x) \geq 0$ 取交集。

**类型四：$|f(x)| \geq g(x)$**

当 $g(x) \leq 0$ 时不等式对所有 $x$ 自动成立（因 $|f(x)| \geq 0$）；当 $g(x) > 0$ 时化为类型二处理。

## 关键方法与公式

### 三角不等式与逆三角不等式

绝对值满足**三角不等式**（Triangle Inequality），在分析学中居于核心地位：

$$|a + b| \leq |a| + |b|$$

等号成立的充要条件是 $ab \geq 0$（即 $a$、$b$ 同号或至少一方为零）。证明：$|a + b|^2 = (a+b)^2 = a^2 + 2ab + b^2 \leq a^2 + 2|a||b| + b^2 = (|a|+|b|)^2$，两边取算术平方根即得。

由三角不等式立刻推出**逆三角不等式**：

$$\bigl||a| - |b|\bigr| \leq |a - b|$$

证明思路：对 $a = (a - b) + b$ 用三角不等式得 $|a| \leq |a-b| + |b|$，即 $|a| - |b| \leq |a-b|$；对称地 $|b| - |a| \leq |a-b|$，两式合并即得。

两个不等式联立，形成双向估计框架：

$$\bigl||a| - |b|\bigr| \leq |a \pm b| \leq |a| + |b|$$

### 平方去绝对值技巧

对形如 $|f(x)| > |g(x)|$ 的不等式，由于两边均非负，可两边平方等价转化：

$$|f(x)| > |g(x)| \iff f(x)^2 > g(x)^2 \iff [f(x) - g(x)][f(x) + g(x)] > 0$$

后者可用零点分区间法处理。此方法在 $f(x)$、$g(x)$ 均为一次函数时尤为高效。

**案例**：解不等式 $|x + 2| > |x - 1|$。

两边平方：$(x+2)^2 > (x-1)^2$，展开后 $x^2 + 4x + 4 > x^2 - 2x + 1$，化简为 $6x > -3$，解得 $x > -\dfrac{1}{2}$，解集为 $\left(-\dfrac{1}{2}, +\infty\right)$。

几何解读：$|x+2| = |x-(-2)|$ 是 $x$ 到 $-2$ 的距离，$|x-1|$ 是 $x$ 到 $1$ 的距离，两距离之和的分界点在 $-2$ 与 $1$ 的中点 $x = -\dfrac{1}{2}$ 处——这与代数结果完全一致。

### 含参数绝对值不等式的分类讨论

对 $|x - a| < b$ 型不等式（$b$ 为参数）：
- 当 $b \leq 0$ 时，无解（绝对值非负，不可能小于非正数）；
- 当 $b > 0$ 时，解集为 $(a - b,\; a + b)$，以 $a$ 为中心、半径为 $b$ 的开区间。

这一形式在数学分析的 $\varepsilon$-$\delta$ 定义中随处可见：" $|x - a| < \delta$ "正是点 $x$ 落在以 $a$ 为中心半径为 $\delta$ 的去心邻域内的表述。

## 实际应用

### 应用一：估计含绝对值表达式的范围

**已知** $|x - 2| \leq 1$，求 $|3x - 5|$ 的取值范围。

由条件 $|x-2| \leq 1$ 得 $1 \leq x \leq 3$，故 $3 \leq 3x \leq 9$，即 $-2 \leq 3x - 5 \leq 4$。因为 $3x - 5$ 在此范围内可取负值，不能直接写成绝对值范围，需继续分析：当 $1 \leq x < \frac{5}{3}$ 时 $3x - 5 < 0$，故 $|3x-5| = 5 - 3x \in (0, 2]$；当 $x = \frac{5}{3}$ 时 $|3x-5| = 0$；当 $\frac{5}{3} < x \leq 3$ 时 $|3x-5| = 3x - 5 \in (0, 4]$。综合得 $|3x - 5| \in [0, 4]$，最大值为 $4$（在 $x = 3$ 时取得）。

### 应用二：最优化——绝对值函数的最小值问题

求 $f(x) = |x - 1| + |x + 3|$ 的最小值。

这是 $|x - 1| + |x - (-3)|$，即 $x$ 到定点 $1$ 与 $-3$ 的距离之和。由双绝对值三段结构知，最小值在 $[-3, 1]$ 上取得，最小值为 $1 - (-3) = 4$。

进一步思考：若改为求 $f(x) = 2|x - 1| + 3|x + 3|$（各段有权重），则最小值点不再是 $[-3,1]$ 上任意点，而需通过比较端点处的斜率变化确定，这属于**加权中位数**问题，在运筹学中有重要应用。

### 应用三：绝对值不等式与区间交集

解不等式组：$|x - 1| < 3$ 且 $|2x + 1| \geq 5$。

- 由 $|x-1| < 3$：$-2 < x < 4$；
- 由 $|2x+1| \geq 5$：$2x + 1 \geq 5$ 或 $2x + 1 \leq -5$，即 $x \geq 2$ 或 $x \leq -3$；
- 取交集：$(-2 < x < 4)$ 与 $(x \geq 2$ 或 $x \leq -3)$ 的交集为 $2 \leq x < 4$。

解集为 $[2, 4)$。
