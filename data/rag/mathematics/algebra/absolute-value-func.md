# 绝对值函数

## 概述

绝对值函数是形如 f(x) = |x| 的函数，其定义域为全体实数 ℝ，值域为 [0, +∞)。其核心特征是"去负取正"：当 x ≥ 0 时，|x| = x；当 x < 0 时，|x| = −x。这一分段结构使它成为代数中最直观的非线性函数之一。

绝对值的几何意义是数轴上点 x 到原点 0 的距离。更一般地，|a − b| 表示数轴上 a 与 b 两点之间的距离。这一几何解释直接决定了绝对值不等式的解法逻辑：|x − 3| < 2 描述的正是"与点 3 距离小于 2 的所有实数"，即开区间 (1, 5)。

绝对值函数在中学代数和高等数学中均有重要地位。它是学习分段函数的第一个典型案例，也是理解含参数不等式、函数图像变换的关键工具。物理中速度的大小、误差分析中的偏差量，均以绝对值形式出现。

## 核心原理

### 分段定义与图像特征

f(x) = |x| 的图像由两条射线组成：x ≥ 0 部分为斜率 +1 的射线，x < 0 部分为斜率 −1 的射线，两段在原点 (0, 0) 处形成"V"形拐点。该拐点是函数的最小值点，最小值为 0。图像关于 y 轴对称，即满足 f(−x) = f(x)，因此 f(x) = |x| 是偶函数。

对于一般形式 f(x) = |x − a| + b，图像的顶点从原点平移至 (a, b)。例如 f(x) = |x − 2| + 1 的顶点为 (2, 1)，两侧斜率仍分别为 ±1。而 f(x) = k|x − a| + b（k > 0）的图像在保持对称性的同时，两侧斜率变为 ±k，"V"形更陡或更平。当 k < 0 时，图像变为倒"V"形，顶点为最大值点。

### 绝对值不等式的解法

含绝对值的不等式有两种基本形式，对应两种截然不同的解集结构：

- **|x| < a（a > 0）**：等价于 −a < x < a，解集为有界区间 (−a, a)。
- **|x| > a（a > 0）**：等价于 x > a 或 x < −a，解集为两段无界区间的并集。

对于 |x − c| < r，解集为 (c − r, c + r)，其几何意义是以 c 为中心、半径为 r 的开区间。例如 |2x − 3| ≤ 5，先化为 |x − 3/2| ≤ 5/2，解得 −1 ≤ x ≤ 4。

含两个绝对值的不等式，如 |x + 1| + |x − 3| ≥ 4，利用三角不等式原理：|x + 1| + |x − 3| ≥ |(x+1) − (x−3)| = 4，等号当且仅当 −1 ≤ x ≤ 3 时成立，因此解集为全体实数 ℝ。

### 三角不等式

绝对值满足重要的三角不等式：

> **|a + b| ≤ |a| + |b|**

等号成立的充要条件是 ab ≥ 0（即 a、b 同号或至少一个为零）。类似地，**||a| − |b|| ≤ |a − b|**，等号成立条件为 ab ≥ 0。这两个不等式是估计绝对值表达式上下界的核心工具，在证明题和优化问题中频繁出现。

### 绝对值函数的导数与不可微点

在微积分视角下，f(x) = |x| 在 x = 0 处不可微：左导数为 −1，右导数为 +1，左右导数不相等，故原点是不可导点。而在 x ≠ 0 处，f′(x) = x/|x|（即 sgn(x)，符号函数）。这是分段函数不可微的经典案例。

## 实际应用

**误差与容差分析**：工程中规定零件尺寸误差满足 |d − 50| ≤ 0.1（单位：mm），即实际尺寸 d 落在 [49.9, 50.1] 内方为合格。这是绝对值不等式在质量控制中的直接应用。

**最优化问题**：求 f(x) = |x − 1| + |x − 5| 的最小值。几何上，f(x) 表示数轴上动点 x 到固定点 1 和 5 的距离之和。当 x ∈ [1, 5] 时，f(x) = 4，取最小值。这一思路推广到多个绝对值之和的极值问题。

**分段函数建模**：出租车计费规则"3公里内收费13元，超出部分每公里2.3元"可写为含绝对值的分段表达，其图像的"折点"结构与绝对值函数一致。

## 常见误区

**误区一：认为 |x²| = x²，因此绝对值对偶次幂无影响**。这在 x 为实数时成立（x² ≥ 0），但对于 |x²−4| 类型，不能直接去掉绝对值，因为 x²−4 在 |x| < 2 时为负值。需先判断内部表达式的正负。

**误区二：解 |x − a| < b 时忘记 b > 0 的前提**。若 b ≤ 0，则 |x − a| < b 无解（因绝对值恒 ≥ 0）；若 b = 0，则唯一解为 x = a。忽略这一条件会导致错误地写出空集为某个区间。

**误区三：将 |f(x)| = f(|x|) 视为恒等式**。反例：f(x) = x − 1，则 |f(−2)| = |−3| = 3，而 f(|−2|) = f(2) = 1，两者不等。绝对值作用在函数上与作用在自变量上，图像变换方式完全不同：|f(x)| 将 x 轴下方的图像翻折到上方，而 f(|x|) 将 x > 0 的图像关于 y 轴作对称复制。

## 知识关联

**前置概念**：绝对值函数是函数概念的具体化。理解"定义域、值域、对应法则"三要素后，绝对值函数提供了第一个典型的分段定义案例，直接强化了分段函数的读写能力。

**图像变换的延伸**：掌握 f(x) = |x − a| + b 的顶点平移规律后，可自然推广到 f(x) = a|x − h| + k 的缩放变换，以及更复杂的 y = |f(x)| 与 y = f(|x|) 两种"折叠变换"，后者在高中数学图像题中是高频考点。

**不等式体系的连接**：绝对值不等式 |x − c| < r 的区间解法，与后续学习的区间估计、误差界分析逻辑一致。三角不等式 |a + b| ≤ |a| + |b| 则直接对应向量模长不等式 |**a** + **b**| ≤ |**a**| + |**b**|，为向量内容做铺垫。
