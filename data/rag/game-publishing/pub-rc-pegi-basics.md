# PEGI分级

## 概述

PEGI（Pan European Game Information，泛欧洲游戏信息）是欧洲通行的电子游戏年龄分级系统，于2003年4月正式启用，目前覆盖英国、德国、法国、意大利、西班牙等38个欧洲国家和地区。PEGI由欧洲互动软件联合会（ISFE）管理，其设立目的是取代此前欧洲各国各自为政的分级标准，为家长和消费者提供统一的内容参考信息。

PEGI系统的运营机构是位于荷兰乌得勒支的PEGI S.A.，实际审查工作由两个专门机构负责：内容描述符审核由英国电影分级委员会（BBFC）承担，整体分级管理则由荷兰Nicam机构负责。这种双机构协作模式与美国ESRB完全由行业自律不同，引入了独立第三方参与，使PEGI的权威性和法律效力更为稳固——例如英国《2011年视频录像法案》明确赋予PEGI 12、16、18级别法律强制效力。

了解PEGI分级对于进军欧洲市场的游戏发行商至关重要。带有PEGI 18标志的游戏在多个欧洲国家禁止向未成年人销售，违规销售属于违法行为。即使是数字发行平台，Steam、PlayStation Store、Xbox等平台在欧洲地区上架游戏时均强制要求提供有效的PEGI评级证书。

## 核心原理

### 五档年龄等级划分

PEGI将游戏划分为5个年龄等级，每个等级以醒目的彩色数字标志印刷在游戏包装正面：
- **PEGI 3**：适合所有年龄段，内容温和，不含任何恐惧或暴力元素
- **PEGI 7**：可能含有轻微的恐吓内容或轻度卡通暴力
- **PEGI 12**：可能含有轻度写实暴力、轻微粗口或轻度性暗示
- **PEGI 16**：可能含有较强写实暴力、强烈语言、性行为暗示或赌博元素
- **PEGI 18**：适合成人，可能含有强烈暴力、裸露、赌博或歧视性内容

值得注意的是，PEGI不设"PEGI 10"等级，这与ESRB的"Everyone 10+"有明显差异，跨市场发行时需要特别注意两套系统之间的等级对应关系并非总是一一对应。

### 内容描述符（Content Descriptors）

除年龄等级外，PEGI还在游戏包装背面标注具体的内容描述符图标，目前共有8种标准描述符：

| 描述符 | 含义 |
|--------|------|
| Violence（暴力） | 含有暴力场景 |
| Bad Language（不良语言） | 含有粗口或侮辱性语言 |
| Fear（恐惧） | 含有可能使儿童恐惧的元素 |
| Gambling（赌博） | 教导或鼓励赌博行为 |
| Sex（性） | 含有性暗示或性行为内容 |
| Drugs（毒品） | 涉及毒品使用或美化 |
| Discrimination（歧视） | 含有种族、宗教等歧视内容 |
| In-Game Purchases（游戏内购） | 含有以真实货币购买虚拟物品功能 |

"In-Game Purchases"描述符于2018年新增，专门应对游戏内购和战利品箱争议，这是PEGI响应欧洲多国监管压力而做出的系统性更新。

### 申请与审核流程

游戏发行商提交PEGI评级的标准流程分为四个步骤：

1. **自评问卷（PEGI Online Registration System，PORS）**：发行商在PEGI官方平台填写包含约40道题目的内容问卷，涉及游戏中最强烈的暴力、性、语言等内容。系统根据回答自动生成初始分级建议。

2. **提交游戏内容**：发行商需同步提交游戏截图、视频片段或可运行版本，供审核人员比对。

3. **Nicam/BBFC复核**：审核机构对高风险内容（如PEGI 16、18）进行人工复核，确认自评结果的准确性。

4. **获得评级证书**：通过审核后，发行商获得正式授权，可在包装和宣传物料上使用PEGI标志，费用依发行规模而定，通常在数百至数千欧元之间。

整个流程通常需要2至4周，建议发行商在游戏上市前至少8周启动评级申请，以预留处理修改意见或重新审核的时间。

## 实际应用

**《堡垒之夜》（Fortnite）** 获得PEGI 12评级，并附带"暴力"和"游戏内购"两个内容描述符。Epic Games在欧洲市场的所有宣传材料均必须显示这一评级，PlayStation欧洲商店页面上的PEGI 12标志属于法定显示要求。

**《荒野大镖客：救赎2》（Red Dead Redemption 2）** 获得PEGI 18评级，标注了"暴力""毒品""歧视"三个描述符。这意味着该游戏在英国、西班牙、爱尔兰等国的实体零售渠道中，销售人员向未成年人出售属于违法行为，而非仅仅是行业自律层面的违规。

对于独立开发者，若仅在欧洲数字平台发行免费游戏，部分平台（如Google Play）允许使用IARC（国际年龄评级联盟）的统一评级代替单独申请PEGI，IARC问卷会自动生成符合PEGI标准的评级结果，大幅降低了小型开发者的合规成本。

## 常见误区

**误区一：PEGI分级与ESRB分级可以直接对应互用**
许多开发商误以为获得ESRB的"Teen（T）"就等同于PEGI 12，或"Mature（M）"等同于PEGI 18。实际上两套系统的评级标准和描述符维度存在差异，同一款游戏在两个系统中获得的级别可能不同。例如，部分在美国获评"Teen"的游戏因含有的赌博元素在欧洲被归为PEGI 16。发行商必须分别申请两套评级，不可混用。

**误区二：数字发行不需要PEGI评级**
一些开发商认为只有实体游戏才需要贴PEGI标签，数字版本无需评级。这是错误认知。Steam、Epic Games Store、PlayStation Store、Nintendo eShop等主要数字平台在欧洲地区均强制要求PEGI评级，未获评级的游戏无法在这些平台的欧洲版本上架，或会被平台自行归入"未评级"状态限制曝光。

**误区三：通过PEGI评级即可在所有欧洲国家合法销售**
PEGI覆盖38个国家，但德国是例外。德国同时使用USK（娱乐软件自我监管）系统，且受《德国青少年保护法》约束，PEGI评级在德国不具备法律效力。发行商若要在德国销售游戏，必须额外申请USK评级，这是欧洲市场合规中最常见的遗漏点之一。

## 知识关联

学习PEGI分级前，建议先掌握**ESRB分级**系统的基本框架——两套系统同样采用"年龄等级＋内容描述符"的双层结构，ESRB的"Content Descriptors"概念与PEGI的8种描述符在设计逻辑上高度相似，有助于理解PEGI的分类逻辑。两套系统最显著的结构差异在于：ESRB使用字母代码（E、T、M、AO），而PEGI直接使用数字年龄，后者对家长的直观性更强。

在PEGI之后，**CERO分级**是日本市场的对应系统，代表亚洲主要市场的评级逻辑。与PEGI相比，CERO同样使用字母分级（A、B、C、D、Z），但对性内容的严格程度远高于暴力内容，与PEGI偏重暴力管控的侧重点形成鲜明对比。掌握PEGI后再学习CERO，可以建立起对不同文化背景下内容审查标准差异的系统性认知，这对于同时进行多地区发行规划的游戏公司具有直接的实操价值。
