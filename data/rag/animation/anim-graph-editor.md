# 曲线编辑器

## 概述

曲线编辑器（Curve Editor / Graph Editor）是关键帧动画工具链中用于精确控制属性值在两个关键帧之间如何随时间变化的可视化编辑界面。它将时间轴上离散的关键帧数据转化为一条连续的贝塞尔曲线（Bézier Curve）或样条曲线（Spline），横轴代表时间（帧数），纵轴代表属性值（如位置、旋转角度、透明度等），动画师可以直接拖拽曲线形状来改变插值行为。

曲线编辑器的概念最早随着 1980 年代计算机辅助动画软件的兴起而出现。Alias/Wavefront（现 Autodesk Maya 的前身）在 1990 年代将其普及为三维动画流程的标准工具。在此之前，动画师只能依靠软件自动生成的线性或固定缓动插值，无法对运动节奏进行精细调整。今天，无论是 Maya 的 Graph Editor、After Effects 的图表编辑器，还是 Blender 的 Dope Sheet + Graph Editor 组合，都延续了这一设计范式。

曲线编辑器的重要性在于它将"何时到达某个值"与"以何种速率到达"彻底分离开来。一个物体从帧 1 的位置 0 移动到帧 24 的位置 100，仅凭两个关键帧无法决定运动是匀速、先快后慢还是弹跳式的——曲线的坡度（斜率）才是决定因素：斜率越大，该时刻速度越快；斜率为零，物体在该瞬间静止。

## 核心原理

### 曲线的数学基础：三次贝塞尔插值

曲线编辑器中最常用的插值方式是**三次贝塞尔曲线**，由四个控制点决定一段曲线：两端的关键帧点（Anchor Point）和各自附带的两个切线手柄（Tangent Handle）。其参数方程为：

**B(t) = (1−t)³P₀ + 3(1−t)²t·P₁ + 3(1−t)t²·P₂ + t³P₃**

其中 P₀、P₃ 为关键帧锚点，P₁、P₂ 为切线控制点，t ∈ [0,1] 为参数时间。切线手柄的位置直接决定曲线在关键帧附近的"进入角度"和"离开角度"，这正是动画师通过曲线编辑器手动调整的对象。

### 曲线形状与运动感的对应关系

曲线形状可以被直接读成运动节奏：
- **S 形曲线（Ease In / Ease Out）**：曲线两端坡度平缓、中间陡峭，对应先慢后快再慢的缓动效果，是模拟自然物理惯性最常用的形态。
- **直线（Linear）**：斜率恒定，对应匀速运动，常用于机械或循环性质的动作片段。
- **超出范围的曲线（Overshoot）**：曲线在目标值附近超出再回弹，纵轴数值超过终点关键帧的值，制造弹性感，常见于 UI 弹出动画。
- **平台区（Plateau）**：曲线出现水平线段，意味着属性值在这段时间内完全静止，用于表现动作的停顿或持帧（Hold）。

### 速度曲线（Velocity Curve）与值曲线（Value Curve）的区别

许多软件（如 After Effects）提供两种曲线视图切换：**值曲线**显示属性随时间的绝对数值，**速度曲线**显示该属性的变化速率（即值曲线的一阶导数）。在速度曲线视图中，曲线高于零轴表示正向加速，触碰零轴表示该瞬间速度为零——这对于调整 Ease In/Out 的精确位置极为直观。After Effects 中可通过按 **Shift+F3** 在两种视图间切换。

## 实际应用

**角色手臂跟随动作（Follow Through）调整**：动画师为手臂设置主体运动后，在曲线编辑器中将手腕旋转属性的曲线向右平移约 3～5 帧，并拉长离开关键帧的切线手柄，使手腕在身体停止后继续运动并缓慢归位，模拟软组织的延迟跟随。

**弹跳球的高度曲线**：弹跳球的 Y 轴位置曲线呈现出一系列"V"形谷底，谷底对应落地帧。专业的弹跳动画要求每次落地的切线角度接近 **90°（垂直）**，而非圆弧，以准确表现瞬间碰撞而非滑动。在曲线编辑器中可以精确地将谷底两侧切线手柄设置为"Broken（断裂）"模式，使进入和离开角度独立控制。

**UI 动效中的弹性曲线**：移动端应用动画（如 iOS Spring Animation）通常需要将元素的缩放属性曲线设置为超出 1.0 再回弹至 1.0 的波形，在曲线编辑器中表现为纵轴峰值约 **1.05～1.15**，随后下降并轻微振荡收敛，整个曲线跨度约 **24～36 帧**（按 60fps 计约 0.4～0.6 秒）。

## 常见误区

**误区一：关键帧越多，动画控制越精确**
许多初学者面对不理想的运动时，习惯增加关键帧来"修补"。但在曲线编辑器中，每增加一个关键帧就引入两组新的切线手柄，相邻曲线段互相影响，反而使曲线变得颠簸难以管理。正确做法是先用尽可能少的关键帧确定运动节拍，再通过调整切线手柄塑造曲线形态，避免"关键帧污染"。

**误区二：自动切线（Auto Tangent）总能产生自然效果**
软件默认的 Auto 或 Spline 切线会尝试计算平滑过渡，但在相邻关键帧数值差异悬殊时（例如旋转值从 0° 跳至 350°），自动切线会生成极大的中间过冲，导致属性值在帧间飙升至 400° 再跌回——这种错误在时间轴视图中完全不可见，只有打开曲线编辑器才能发现。

**误区三：曲线编辑器只用于位置属性**
曲线编辑器对**所有可设关键帧的属性**均有效：摄像机焦距（Focal Length）的呼吸感、角色面部混合形状（Blend Shape）权重的微表情节奏、灯光亮度的闪烁频率，都依赖曲线编辑器的精细控制。将其仅用于位置/旋转是对工具能力的严重低估。

## 知识关联

**前置概念——关键帧基础**：关键帧定义了曲线的锚点坐标（时间值与属性值），没有关键帧就没有曲线段可供编辑。理解关键帧的"时间打点"含义是读懂曲线横轴的前提。

**后续概念——切线类型**：曲线编辑器中每个关键帧节点的切线手柄有 Linear、Flat、Spline、Broken、Stepped 等多种类型，切线类型的选择是曲线编辑器操作的核心技能层，直接决定曲线在该节点的斜率行为与连续性。

**后续概念——行走循环**：行走循环的动画质量高度依赖曲线编辑器中对髋部上下曲线、脚踝旋转曲线进行精确的循环衔接（Pre/Post Infinity 设置为 Cycle），以及对接触帧与过渡帧之间切线斜率的手动微调，是曲线编辑器在角色动画场景下的综合应用检验。
