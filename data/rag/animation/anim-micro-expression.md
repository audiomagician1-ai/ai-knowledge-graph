# 微表情

## 概述

微表情（Micro-expression）是持续时间极短的面部肌肉运动，通常仅维持1/25秒到1/5秒（约40至200毫秒）之间，是人类在试图压抑或掩盖真实情绪时无法完全控制的不自主面部泄露。这一概念由心理学家保罗·艾克曼（Paul Ekman）在20世纪60至70年代系统研究并命名，他于1969年发表的论文《Pan-Cultural Elements in Facial Displays of Emotion》（Science, Vol. 164）首次提出跨文化普遍性情绪表达框架，随后确认全人类共享七种基本情绪的微表情模式：愤怒（Anger）、恐惧（Fear）、厌恶（Disgust）、轻蔑（Contempt）、悲伤（Sadness）、惊讶（Surprise）和快乐（Happiness）。

与完整的宏表情（Macro-expression，持续0.5至4秒）不同，微表情因其持续时间极短，往往被观察者的意识层面忽视，但大脑的潜意识层面仍会捕捉到这些信号，形成"感觉怪怪的"或"直觉上不对劲"的印象。1985年，艾克曼与莫里斯·奥沙利文（Maureen O'Sullivan）在针对测谎的实验研究中证实，普通人识别微表情的准确率仅约47%，而经过专项训练后可提升至约70%（Ekman & O'Sullivan, 1991）。

在面部动画领域，微表情的加入解决了一个经典问题：角色表情过于"干净"导致缺乏真实感。人类面孔从不是情绪的简单开关，而是充满了压抑、犹豫与掩饰的多重层次。当动画师在主体表情之外叠加恰当的微表情，角色立刻从"受控的傀儡"转变为"有内心活动的生命体"，这正是皮克斯动画角色常被评价为"比人更像人"的技术根源之一。

## 核心原理

### 时间窗口与关键帧密度

微表情在动画中的实现依赖于极高密度的关键帧设置。在标准24帧/秒的动画中，一个完整微表情仅占1到5帧；在影视级60帧/秒标准下，则对应3到12帧。动画师必须在这个时间窗口内完成面部肌肉的**激活（Onset）、峰值（Apex）与消退（Offset）**三个阶段，形成一个完整的"脉冲"曲线，而非线性过渡。

关键帧的时间分配遵循**非对称消退原则**：微表情的上升速度通常快于消退速度，上升阶段约占整体时长的30%，消退阶段占70%。这与人类面部肌肉的生理特性一致——肌肉的激活收缩比放松松弛更迅速。若将上升与消退设为等速，微表情会呈现机械感，而非生理真实性。以24帧标准、持续4帧的微表情为例：

```
帧分配示例（24fps，总时长4帧）：
  Frame 0:  表情强度 0%（中性基础表情）
  Frame 1:  表情强度 65%（快速激活，上升阶段）
  Frame 2:  表情强度 100%（峰值，Apex）
  Frame 3:  表情强度 55%（消退开始，缓慢）
  Frame 4:  表情强度 0%（完全回归中性）

上升速率：65% → 100% / 1帧 = 35%/帧
消退速率：100% → 0% / 2帧 = 50%/帧（均值）
实际消退曲线应为Ease-Out缓动，非线性匀速
```

动画软件（如Autodesk Maya或Blender）中，动画师应将消退段的切线句柄（Tangent Handle）设为Flat或Slow Out，以模拟肌肉自然松弛的减速特性。

### 面部动作编码系统（FACS）的精确应用

保罗·艾克曼与沃利斯·弗里森（Wallace Friesen）在1978年出版的《Facial Action Coding System: A Technique for the Measurement of Facial Movement》（Consulting Psychologists Press）中，将面部运动分解为44个独立的**动作单元（Action Units，简称AU）**，覆盖从额肌到颈阔肌的全部表情相关肌群。微表情动画的核心在于准确识别并激活特定AU组合，而非整张脸的模糊变形。

七种基本情绪对应的微表情AU特征组合如下：

| 情绪 | 核心AU组合 | 关键肌肉 |
|------|-----------|---------|
| 悲伤（被压抑） | AU1 + AU4 + AU15 | 内侧额肌 + 皱眉肌 + 降口角肌 |
| 恐惧 | AU1 + AU2 + AU5B + AU20 | 额肌（内+外）+ 上眼睑提肌 + 颈阔肌 |
| 愤怒 | AU4 + AU5 + AU23 + AU24 | 皱眉肌 + 上眼睑提肌 + 口轮匝肌 |
| 厌恶 | AU9 + AU15 + AU16 | 提上唇鼻翼肌 + 降口角肌 + 降下唇肌 |
| 轻蔑 | AU12R（单侧） + AU14R | 单侧颧大肌 + 单侧颊肌 |
| 惊讶 | AU1 + AU2 + AU5 + AU26 | 额肌 + 眼睑提肌 + 降颌肌 |
| 快乐（迪谢纳微笑） | AU6 + AU12 | 眼轮匝肌（眶部）+ 颧大肌 |

以压抑的悲伤微表情为例：AU1（内侧眉毛上提，形成特有的"倒U形"眉峰）+ AU4（眉毛下压并向中间聚拢）+ AU15（嘴角轻微下拉），这三个肌肉动作在1至3帧内以约15%至30%的激活幅度出现，随即迅速消退。一个完整的悲伤宏表情则会额外包含AU17（下巴颤动）和AU54（头部向前下垂），且持续时间达12帧以上。

**注意：** 轻蔑（AU12R/14R）是七种基本情绪中唯一具有明显**单侧不对称性**的微表情，双侧均等激活则判定为快乐而非轻蔑，这一区别在角色欺骗性行为设计中尤为关键。

### 微表情强度量化公式

在程序化或脚本驱动的面部动画系统（如Unreal Engine的MetaHuman或自研面部绑定系统）中，微表情的激活强度可通过以下公式计算：

$$
I(t) = I_{peak} \cdot \left[ e^{-\lambda_1 (t - t_0)} - e^{-\lambda_2 (t - t_0)} \right] \cdot H(t - t_0)
$$

其中：
- $I(t)$ 为 $t$ 时刻的微表情激活强度（0至1归一化）
- $I_{peak}$ 为峰值强度系数（典型值：被压抑情绪取0.2–0.4，未压抑情绪取0.5–0.8）
- $\lambda_1$ 为消退速率常数（典型值：0.8–1.5，值越大消退越快）
- $\lambda_2$ 为激活速率常数（典型值：3.0–6.0，必须满足 $\lambda_2 > \lambda_1$ 以确保快速上升）
- $t_0$ 为微表情触发时间
- $H(t - t_0)$ 为Heaviside阶跃函数（确保 $t < t_0$ 时强度为零）

此双指数模型（Double Exponential Model）能够自然重现"快速激活—缓慢消退"的不对称脉冲曲线，比关键帧线性插值更贴近真实肌肉动力学。

### 微表情与主体表情的"情绪冲突叠加"

微表情最强大的叙事功能在于与角色主体表情的**情绪冲突**。当角色表面呈现社交性微笑（AU6+AU12），但在微笑建立前的1至2帧出现轻蔑微表情（AU12R+AU14R，单侧），观众的潜意识会立即感知"这个笑容并不真诚"，即便意识层面完全没有注意到那1至2帧的异常。

情绪冲突叠加存在三种标准结构：

1. **前置泄露型（Pre-Masking Leak）**：真实微表情出现在主体掩盖表情建立之前（1–3帧先于宏表情），表达"角色意识到需要掩盖但稍晚了一步"。
2. **穿透型（Breakthrough）**：在长时间维持的主体表情中间，短暂出现1–2帧的真实情绪，然后立即被压回，表达"维持掩盖状态极度耗力"。
3. **残影型（Residual Trace）**：主体表情消退后，真实情绪的微弱激活仍残留1–3帧，表达"掩盖结束后真实情绪的短暂释放"。

## 关键公式与算法

除上述双指数公式外，在基于BlendShape的面部绑定中，微表情的多AU叠加强度需处理**肌肉拮抗冲突**。当AU4（皱眉，额肌收缩）与AU1（眉毛上提，额肌拉伸）同时激活时，两者物理上拮抗，叠加强度需通过以下限制函数处理：

$$
W_{combined} = W_{AU1} \cdot (1 - \alpha \cdot W_{AU4}) + W_{AU4} \cdot (1 - \alpha \cdot W_{AU1})
$$

其中 $\alpha$ 为拮抗系数（典型值0.6至0.9），防止两个相反BlendShape同时满权重激活导致几何体撕裂。悲伤微表情的AU1+AU4组合中，$\alpha$ 建议取0.7，此时AU1和AU4均以0.4激活时，实际叠加权重分别约为0.288，形成特征性的"内眉上扬且聚拢"的悲伤眉形，而不会产生几何异常。

## 实际应用

### 影视动画案例分析

**《疯狂动物城》（Zootopia，2016，Walt Disney Animation Studios）**：朱迪·霍普斯（Judy Hopps）在意识到自己的种族歧视言论被断章取义播出后，面对尼克（Nick Wilde）的质问，角色脸上出现了持续约3帧（24fps）的恐慌微表情（AU1+AU2+AU5B），随即切换为辩解的坚定表情。动画师在此处使用了前置泄露型结构，让观众在理性上接受朱迪的辩解之前，潜意识已收到"她知道自己错了"的信号，使后续的角色和解更具情感重量。

**《头脑特工队2》（Inside Out 2，2024，Pixar）**：阿莱（Riley）在冰球赛前试图强装自信时，面部出现多处叠加微表情——AU20（嘴角横向拉伸，恐惧前兆）穿插于AU12（微笑），且在说出"我没事"时的嘴唇闭合瞬间，出现约2帧的AU17（下巴轻微颤动），传达言语与内心的背离。皮克斯动画技术总监Mark Andrews在采访中指出，该片在焦虑（Anxiety）角色的设计阶段专门咨询了艾克曼研究所的顾问，以确保微表情的FACS精确性。
