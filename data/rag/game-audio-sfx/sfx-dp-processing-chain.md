# 对白处理链

## 概述

对白处理链（Dialogue Processing Chain）是游戏音频中将原始录音转化为可用对白素材的标准信号处理流程，其固定顺序为：**EQ均衡 → 压缩（Compressor）→ 去咝音（De-Esser）→ 限制器（Limiter）**。这四个环节缺一不可，且顺序具有严格的逻辑依据——每一步的输出都是下一步能正常工作的前提。

该流程的标准化始于20世纪90年代末的游戏配音后期制作行业，随着《最终幻想VII》（1997年）等早期全配音游戏的量产需求，录音棚开始将电影对白后期的处理经验系统移植到游戏工作流中，并根据游戏引擎的响度限制（通常为-6 dBFS至-3 dBFS峰值）调整链路参数。

对白处理链在游戏中的重要性体现在：游戏对白需要在动态混音环境下与音效、音乐竞争频谱空间，同时还要在耳机、电视音箱、手机喇叭等差异极大的回放设备上保持可懂度。一条配置合理的处理链可以将原始录音的动态范围从约40 dB压缩至12–18 dB，使台词在激烈的战斗音效中仍然清晰可辨。

## 核心原理

### 第一环节：EQ均衡（频谱整形）

EQ位于处理链的最前端，负责在动态处理之前清理频谱问题，原因是压缩器会将频谱缺陷同步放大。标准的对白EQ操作包括：

- **高通滤波（High-Pass Filter）**：截止频率设置在 **80–120 Hz**，以 12 dB/oct 或 18 dB/oct 的斜率滚降低频，消除话筒架振动、空调噪声和演员呼吸产生的超低频隆隆声。
- **中频提升**：在 **2–4 kHz** 范围内温和提升 2–4 dB，增强辅音清晰度（"presence boost"），这一频段对应人耳对语音可懂度最敏感的区域。
- **问题频率切除**：针对录音室房间共鸣，常见于 **300–500 Hz** 的"纸盒声"频段，使用窄带（Q值 4–8）陷波切除 3–6 dB。

EQ处理后，对白的频谱轮廓应当平坦且集中于 **200 Hz–8 kHz** 的语音核心频段。

### 第二环节：压缩（动态控制）

压缩器是处理链中参数最复杂的环节，游戏对白的典型参数范围如下：

| 参数 | 推荐范围 | 原理 |
|------|----------|------|
| 阈值（Threshold） | -24 dB 至 -18 dBFS | 捕捉中等响度以上的所有台词 |
| 比例（Ratio） | 3:1 至 5:1 | 温和压缩，避免泵浦感 |
| 启动时间（Attack） | 5–20 ms | 保留辅音瞬态 |
| 释放时间（Release） | 80–200 ms | 跟随语音自然节奏 |
| 增益补偿（Make-up Gain） | 4–8 dB | 拉高整体响度 |

压缩的核心目标是将演员不同情绪表演之间的响度差异（同一角色轻声耳语与愤怒呼喊可差达 30 dB）收束到可控范围，以便在游戏引擎中用统一的音量参数调度。**注意**：压缩必须在去咝音之前，因为增益补偿会同时抬高高频咝音，需要后续De-Esser来处理。

### 第三环节：去咝音（De-Esser）

De-Esser是一种频率选择性的动态处理器，专门针对 **5–9 kHz** 频段的"S、C、T"咝声爆发进行压缩。由于前一步的压缩和增益补偿将整体响度提升，咝音问题在此阶段会被显著暴露。

游戏对白De-Esser的检测频率通常设置为 **6–8 kHz**，因为这是中文普通话和日语配音（游戏两大主流配音语种）齿音能量最集中的区域。阈值设置以咝音不再刺耳为准，过度压缩会导致人声出现"嘶哑感"（lisp effect），使角色显得不自然。宽频模式（Wideband）与分频模式（Split-band）的选择取决于话筒型号：电容话筒录音倾向使用分频模式，仅压缩问题频段。

### 第四环节：限制器（Limiter）

限制器是处理链的最后一道防线，本质上是压缩比设为无穷大（∞:1）的极端压缩器，用于硬性截断超过阈值的所有瞬态峰值。游戏对白的限制器阈值通常设在 **-3 dBFS 至 -1 dBFS**，启动时间设为 **0.1–1 ms** 以捕捉所有瞬态。

限制器的存在是游戏音频区别于电影混音的关键原因之一：游戏引擎的数字音频路径若接收到 0 dBFS 以上的信号将产生数字削波失真（digital clipping），而游戏中玩家行为触发多个声音同时叠加的情况极为普遍，对白素材必须在导入引擎前预留足够顶空间（headroom）。

## 实际应用

**案例：开放世界RPG的NPC对白制作**

以《巫师3：狂猎》规模的游戏为例，制作团队需要处理数万行NPC对白。处理链在批处理脚本中以固定预设运行，EQ预设针对不同录音环境（主棚/远程录音/居家录音）分为三套，而压缩和限制器预设全团统一，确保所有角色导入游戏引擎后的响度一致性。

**案例：动作游戏战斗对白的响度设计**

战斗中的呼喊台词（"Battle Cry"类）在录音时本身响度就高，原始峰值可达 -6 dBFS。此类素材进入处理链时，EQ高通滤波器截止频率需上移至 **150 Hz**（演员用力发声时低频躯体共鸣更强），压缩比可适当降低至 2:1（动态范围本身已较小），但限制器阈值需严格维持在 -3 dBFS，防止与武器音效叠加时产生削波。

## 常见误区

**误区一：认为De-Esser可以放在压缩之前**

部分初学者将De-Esser提前到压缩之前，认为"先去掉问题再压缩"更合理。这一做法的问题在于：压缩器的增益补偿（Make-up Gain）在处理后会重新提升高频咝音的相对响度，导致De-Esser的工作成果被抵消。正确的逻辑是，De-Esser处理的是**压缩后**被放大的咝音状态，其阈值设置必须基于压缩器输出后的真实信号电平。

**误区二：对所有角色使用相同的EQ设置**

同一款游戏中，不同年龄、性别和录音环境的角色对白需要独立调整EQ参数。例如，老年男性角色的基频（F0）通常在 **85–120 Hz**，高通滤波器截止频率应相应下调至 **60–80 Hz**；而童声角色的基频可达 **250–300 Hz**，过低的高通设置反而会引入不必要的低频噪声。使用统一EQ预设会导致部分角色的音色特征被削平。

**误区三：限制器增益削减越多对白越"稳定"**

将限制器阈值过低（如设在 -12 dBFS）虽然能消除几乎所有峰值变化，但会导致对白的语音节律感消失——游戏中的对白需要保留一定的瞬态起伏才能听起来自然有力。过度限制后的对白在混音中往往会被音效"掩盖"，因为缺少瞬态的语音频谱密度与持续性噪声相似，人耳会优先忽略它。
