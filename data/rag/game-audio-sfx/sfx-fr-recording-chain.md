# 录音信号链

## 概述

录音信号链（Recording Signal Chain）是指音频信号从声源出发，经过一系列模拟与数字处理设备，最终被捕获并存储的完整路径。在Foley录制场景中，信号链通常由以下固定顺序构成：麦克风 → 前置放大器（Preamp）→ 模数转换器（AD Converter）→ 数字音频工作站（DAW）→ 监听系统。这条路径上的每一个环节都会对最终录制的游戏音效质量产生不可逆的影响。

信号链的概念在20世纪50至60年代随着多轨磁带录音技术的普及而逐渐成型，早期的录音工程师发现信号在传输过程中会累积噪声和失真，因此开始系统性地研究每个设备节点的电气特性。在游戏音效Foley录制中，信号链的重要性体现在其短板效应：脚步声、布料摩擦声等Foley音效的动态范围往往仅有20至40dB，任何链路上的增益设置失误都会导致底噪过高或信号削波（Clipping），直接影响后期编辑的可用性。

与现场收音相比，Foley录音室的信号链通常更为稳定和可控，但这也意味着工程师需要对每个环节进行精确的增益配置，而不能依赖动态话筒的宽容度来掩盖链路问题。

## 核心原理

### 前置放大器（Preamp）的增益结构

麦克风输出的信号电平极低，通常在 -60dBu 至 -40dBu 之间，而标准线路电平（Line Level）为 +4dBu（专业设备）或 -10dBV（消费级设备）。前置放大器的职责是将麦克风电平提升至线路电平，增益幅度通常为40至70dB。

前置放大器的关键技术指标是等效输入噪声（EIN，Equivalent Input Noise），高质量录音室前放的EIN通常低于 -128dBu（A计权），这意味着在录制细腻的Foley音效（如轻柔的纸张翻动声）时，底噪不会掩盖信号的高频细节。在实际Foley录制中，前放增益过高会引入电子底噪，过低则会在AD转换时丢失低电平细节，因此通常将录制峰值控制在 -12dBFS 至 -6dBFS 之间。

### 模数转换器（AD Converter）的量化精度

AD转换器负责将模拟电压信号转换为数字比特流，其核心参数是采样率（Sample Rate）和位深（Bit Depth）。游戏Foley录制的行业标准通常采用48kHz/24bit，其中24bit提供约144dB的理论动态范围（计算公式：动态范围 ≈ 6.02 × 位深 + 1.76 dB），远超人耳的可感知范围，为后期降噪和增益调整留出充足的操作空间。

量化误差（Quantization Error）是AD转换固有的失真来源，表现为低电平信号的颗粒感。专业的AD转换器会加入抖动（Dither）处理，将量化误差转化为随机白噪声，使其听感更为自然。在Foley录制中，录制衣物摩擦声或细小道具声时，抖动的有无对最终音效的干净程度有显著影响。

### 监听系统的信号反馈

监听系统包括监听音箱（Monitor Speakers）和监听耳机（Monitor Headphones），其在信号链中承担实时反馈的角色，但本身并不参与信号的录制路径。然而，监听系统的频率响应准确性直接影响录音师对信号链其他环节的判断。

Foley录制常用的近场监听音箱频率响应范围通常为80Hz至20kHz（±3dB），用于判断脚步声的低频质感和布料声的高频齿音。监听耳机因其封闭空间特性，在录制棚内可以更清晰地捕捉信号链中的底噪和轻微失真，特别适合在A/D转换阶段进行电平校准。

## 实际应用

在游戏Foley录制项目中，以脚步声录制为例，典型信号链配置如下：选用AKG C414（小振膜电容话筒，频率响应20Hz–20kHz）→ API 512c前置放大器（增益设置+48dB）→ Prism Sound ADA-8XR（AD转换器，48kHz/24bit）→ Pro Tools DAW（数字录制软件）→ Yamaha HS8近场监听。

录音师在录制混凝土地面上皮鞋脚步声时，需要将前放增益设置在使峰值电平落在 -10dBFS 附近，既保留了皮鞋落地瞬间的冲击瞬态，又避免了AD转换器的输入过载（0dBFS为硬限制，一旦超过即产生不可修复的数字削波）。如果后续游戏引擎要求音效文件格式为44.1kHz/16bit（如部分移动游戏平台），则需在DAW内进行采样率转换，但原始录制仍应保留48kHz/24bit的源文件。

## 常见误区

**误区一：前放增益越大，录音信噪比越好**
事实上，前放增益需要与信号电平匹配。当麦克风靠近道具录制强烈声源时，若前放增益仍设置过高（如+60dB以上），进入AD转换器的信号会超过0dBFS，产生数字削波。数字削波（Digital Clipping）与模拟过载不同，它不产生偶次谐波的"温暖"饱和感，而是生成刺耳的奇次谐波失真，且无法通过后期处理修复。

**误区二：AD转换器的采样率越高，音质越好**
192kHz采样率相较于48kHz并不会在20kHz以下的可听范围内带来音质提升，反而会使文件体积增大4倍，在游戏项目多轨Foley录制中造成存储和处理负担。目前游戏音效工业标准是48kHz，192kHz录制仅在需要对音效进行极端时间拉伸（Time-Stretching）处理时（如将脚步声降速至1/8播放时）才具有实际价值，因为更高的采样率在降速处理后仍能保留20kHz以内的完整频率内容。

**误区三：监听音箱品质不影响录制结果**
虽然监听系统不在信号录制路径中，但频率响应过度染色的监听音箱（如消费级多媒体音箱往往在200Hz和10kHz附近有明显的增益隆起）会导致录音师误判前放增益设置，例如将低频过重的音箱误认为是录音中存在低频堆积而错误地减小Foley道具的录音距离。

## 知识关联

**与麦克风选择的衔接**：不同类型麦克风的输出阻抗和电平特性直接决定了前置放大器的输入阻抗匹配需求。电容话筒（如Neumann KM184）需要48V幻象电源（Phantom Power），这由前放通过XLR线缆提供；而动圈话筒（如Shure SM7B）则需要更高的前放增益（通常需额外+20至+30dB），因此对前放的EIN指标要求更高。

**与外景录音经验的对比**：外景Foley或现场录音通常使用Sound Devices 702/788T等便携式录音机，其内置前放和AD转换器集成在同一设备中，信号链更短但灵活性较低；录音室Foley则通过分离式设备组合获得更高的动态范围和更低的底噪，典型录音室底噪可控制在 -85dBFS（A计权）以下。

**为音频编辑工作流奠定基础**：信号链的录制参数（位深、采样率、峰值电平设置）直接决定了进入音频编辑工作流时的素材质量。在Pro Tools或Reaper等DAW中，-18dBFS的标准录制电平（对应0VU模拟电平）为后期的均衡、压缩和采样率转换处理提供了充足的增益余量（Headroom），是Foley素材从录制进入编辑流程的关键质量门控点。
