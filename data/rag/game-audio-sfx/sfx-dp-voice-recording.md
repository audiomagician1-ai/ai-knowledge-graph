# 语音录制

## 概述

语音录制是游戏音频对白制作的起点，指在专业录音环境中捕获配音演员声音表演的完整技术流程。与影视配音录制不同，游戏语音录制通常需要为同一角色录制数百甚至数千条独立台词，每条台词都可能在不同游戏情境下单独触发播放，因此每句话的首尾处理、呼吸位置和情绪一致性都有独立而严格的要求。

这一实践的技术规范在2000年代随着《光晕》《质量效应》等叙事驱动型游戏的兴起而快速标准化。当时开发商发现，沿用影视录音习惯（如允许句间连续呼吸音、依赖后期剪辑拼接）会导致游戏运行时的对白触发出现明显的拼接感与不一致性，促使行业形成专门针对游戏交互特性的录音规范。

游戏语音录制之所以关键，在于它直接决定了后续所有处理环节的质量上限。一旦原始录音中存在本底噪声过高（通常要求信噪比不低于60dB）、演员嘴部杂音未被处理、或动态范围压缩过度等问题，对白编辑和音频引擎阶段均难以完全弥补。

## 核心原理

### 录音棚声学环境要求

游戏语音录制通常在混响时间（RT60）控制在80毫秒以内的干声棚（Dry Vocal Booth）中进行，而非电影配音常用的稍带空间感的录音室。极短的RT60确保录制的声音是"干净的干声"，方便音频引擎在运行时根据游戏场景动态叠加混响（如洞穴回声、室内反射）。若原始录音本身带有过多房间感，便无法在引擎中被置于开放旷野等场景而不显得突兀。

本底噪声（Background Noise Floor）应控制在 **-65dBFS以下**，通风系统是最常见的噪声污染源，专业棚通常在录音期间关闭空调并依赖隔热密封结构维持温度。

### 麦克风放置与信号链

游戏语音录制的标准麦克风摆位为距演员嘴部 **15至20厘米**，配合防喷罩（Pop Filter）置于麦克风与演员之间约5厘米处，以消除爆破音（B、P等辅音产生的气流冲击）。信号链通常为：大振膜电容话筒 → 前置放大器 → 模数转换器，采样率设置为 **48kHz / 24bit**（这是游戏音频行业标准，区别于音乐制作常用的44.1kHz）。

增益设置上，演员正常说话时峰值电平应落在 **-12dBFS至-6dBFS** 之间，为喊叫或极端情绪表演留出至少6dB的动态余量，避免削波（Clipping）。

### 配音会话结构与台词管理

专业游戏语音录制通常以"会话脚本"（Session Script）为核心文件，该文档列出所有台词的唯一ID（如 `PC_COMBAT_HURT_01` 到 `PC_COMBAT_HURT_24`）、台词文本、情绪标注（如"愤怒，中等强度"）和时长预估。录音导演（Voice Director）根据这份脚本引导演员，而非让演员自由发挥。

每条台词录制完成后，录音师需立即在数字音频工作站（DAW，通常为 Pro Tools 或 Reaper）中标记Take编号与状态（OK / NG / 备用），确保每个台词ID至少有一条确认可用的Take，以及通常2至3条备选Take，为后续对白编辑提供选择空间。

## 实际应用

**大型RPG游戏的批量录制流程**

以一款中等规模RPG为例，主角可能需要录制1500至3000条对白（包括战斗喊叫、环境反应、剧情对话三类）。录音团队通常按情绪类型分组安排录制顺序，先录情绪平稳的剧情对话，再录激烈的战斗台词，以保护演员嗓音状态。每个录制日的实际有效工作时长约为4小时，超过后演员嗓音疲劳会导致音色明显漂移，破坏角色声音的一致性。

**战斗语音的特殊要求**

战斗用的短促语音（如"Watch out!"类台词，通常在0.5至1.5秒之间）要求演员在句子开头立刻进入情绪状态，且句尾不能有拖尾呼气音，因为这类音频在引擎中会被频繁、快速地随机触发，拖尾部分会与背景音效产生叠加污染。录音师在监听时需专门注意句尾的无声处理是否干净。

## 常见误区

**误区一：将影视配音标准直接套用于游戏录制**

影视配音允许演员跟随画面节奏做连续表演，剪辑时再分割。但游戏语音录制要求每条台词都是独立完整的音频单元：句前必须有至少0.1秒的无声前置（Pre-roll silence），句后有至少0.2秒的无声尾部（Post-roll silence），并且中间不能夹杂前后台词的呼吸声。若沿用影视习惯做连续表演，几乎每条台词都需要大量手动修剪，效率极低且容易出错。

**误区二：认为高采样率一定更好**

部分新手录音师会坚持使用96kHz或192kHz录制"保留更多细节"，但游戏引擎最终播放格式通常为48kHz，过高的采样率不仅不会提升最终质量，反而造成存储空间浪费和转换时可能引入的重采样误差。游戏行业的48kHz / 24bit标准已经过充分验证，是质量与效率的合理平衡点。

**误区三：忽视监听耳机与监听音箱的差异**

在录音棚内使用开放式监听耳机（如 Sennheiser HD 650）会让漏出的声音被麦克风拾取，污染演员录音。游戏语音录制中演员应使用封闭式耳机（Closed-back Headphones）监听参考音频，且音量不宜过大，以免在安静句间通过麦克风拾取到微弱的耳机漏音。

## 知识关联

语音录制是整个游戏对白制作链条的源头，其产出的原始音频文件（Raw Takes）直接进入**对白编辑**环节，后者负责从多个Take中选择最佳表演、修剪句前句后的杂音、消除嘴部噪声并做音量标准化。录制质量的高低直接决定对白编辑的工作量：一段干净、增益合适的录音可能只需要5分钟处理，而一段信噪比不足、动态失控的录音可能需要超过30分钟的精细修复。

在设备层面，语音录制直接引出**麦克风选择**这一议题。大振膜电容话筒（如 Neumann U87、AKG C414）是游戏对白录制最常见的选择，其宽频响和低自噪声特性与上述48kHz/24bit的信号链需求高度匹配；而对于外景临时录制或特殊角色（如机械合成音效角色），不同指向性与动态特性的话筒选择会产生截然不同的原始音色基础。
