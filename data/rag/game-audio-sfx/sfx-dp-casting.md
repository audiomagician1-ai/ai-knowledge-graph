# 演员选角

## 概述

演员选角（Casting）是游戏配音制作流程中，通过系统化的试音（Audition）和多轮评估程序，为每个角色匹配最契合其声音风格与戏剧特质的配音演员的过程。游戏配音选角与影视配音选角存在本质性差异：影视配音演员通常录制100至300行对白，而大型开放世界游戏（如《上古卷轴5》）的主要配角对白量可超过3000行，次要NPC则可能共享同一演员录制多达5000行重复性台词，因此演员必须具备在长时段录制（单项目累计录音时间常达40至60小时）中维持音色一致性的生理与心理耐力。

游戏配音选角作为独立专业流程的确立，大致始于2000年代中期。《光晕2》（Halo 2，2004）聘用基弗·萨瑟兰（Kiefer Sutherland）配音引发行业关注，随后《侠盗猎车手：圣安地列斯》（2004）和《神秘海域》系列（Uncharted，2007起）相继建立起以角色深度为核心的选角标准，游戏行业才逐渐形成区别于广播剧和动画的专属选角体系。配音导演（Voice Director）通常主导整个流程，负责制定试音材料、评估候选录音，并最终向制作团队提交带有详细说明的推荐名单，而非仅仅给出一个名字。

游戏角色的声音一旦确定，通常贯穿整个系列作品。选角失误的代价极为高昂：《光晕5》（Halo 5，2015）中将士官长（Master Chief）的配音演员史蒂夫·唐尼（Steve Downes）替换为其他演员一事（虽最终未实施完整替换），已引发玩家社群的强烈争议，证明选角阶段的评估质量直接决定了整个IP的长期声音完整性。

参考文献：《The Sound of Play: A Guide to Game Audio》(Karen Collins, 2013, MIT Press) 对游戏配音产业的选角惯例演变有系统性描述，本文部分框架参照该书第7章。

## 核心原理

### 角色档案与选角简介（Casting Brief）

选角工作的起点是配音导演与游戏叙事总监（Narrative Director）共同撰写的角色档案（Character Profile），通常包含以下要素：角色年龄范围（精确到±5岁区间）、声音质感描述词（例如"低沉、疲惫、带有苏格兰高地口音、偶有颤音"）、情绪幅度说明、以及参考声音（Reference Voices）列表。参考声音可能是其他游戏角色、电影角色或真实人物，用作演员试音方向的锚点，但优秀的配音导演会明确要求演员"向参考声音靠近而非模仿"，以保留原创性。

角色档案的精细程度直接影响试音录音的风格离散度：描述词越模糊，收到的录音方向越分散，筛选成本越高。顽皮狗（Naughty Dog）工作室的标准是，主要角色的选角简介字数不少于500字，并包含至少3个具体的"角色在极端情境下会如何反应"的场景描述，例如"该角色在目睹同伴死亡时会第一时间压抑悲痛以维持领导形象，而非即时崩溃"。这类行为层面的描述为演员提供了表演内核，而不仅仅是音色方向。

### 试音材料设计（Audition Sides）

游戏选角的试音材料（Audition Sides）通常包含3至5段对白，每段时长控制在15至30秒之间，总时长一般不超过3分钟。与影视选角不同，游戏试音材料必须覆盖多种情绪状态，至少包含三类场景：

1. **常态对话**：测试演员的基准音色与语言节奏，通常选取角色向他人解释复杂情况的段落
2. **高压战斗状态**：测试演员在身体紧绷、气息急促条件下能否保持台词清晰度与情感真实性
3. **情绪极端状态**：如极度悲痛、恐惧或狂喜，用于测量演员的情绪幅度上限

部分工作室（如CD Projekt Red在制作《赛博朋克2077》时）还会在试音材料中加入1至2行完全无上下文的孤立台词，专门测试演员在缺乏语境时的自我诠释能力与创造性判断力。

### 试音评估维度与评分体系

配音导演在评估试音录音时，通常使用结构化评分表，主要维度包括：

| 评估维度 | 权重 | 说明 |
|---|---|---|
| 音色匹配度 | 30% | 自然音色与角色档案的契合程度，1-10分制 |
| 情绪真实性 | 25% | 是否传递可信的内在情感，而非仅"读出"台词 |
| 可导性（Directability） | 25% | 在A/B两版本间能否呈现有意义的方向差异 |
| 非语义发声能力 | 20% | 呼喊、受伤音效等非台词发声的自然处理质量 |

非语义发声（Non-Verbal Vocalization）在动作类游戏中尤为关键：战斗呼喊、受伤喘息、攀爬用力声等占游戏配音录制总量的比例在某些项目（如《鬼泣5》、《怪物猎人》系列）中可高达35%至50%，演员若无法自然处理这类发声，将在实际录制阶段造成严重瓶颈。

## 关键公式与评估模型

在实际选角决策中，当有多名候选人进入最终评估阶段时，部分配音导演使用加权综合评分模型（Weighted Composite Score）辅助决策。以四维评估为例，最终评分计算如式：

$$S_{final} = 0.30 \times S_{tone} + 0.25 \times S_{emotion} + 0.25 \times S_{direct} + 0.20 \times S_{nonverbal}$$

其中：
- $S_{tone}$ = 音色匹配度得分（1-10）
- $S_{emotion}$ = 情绪真实性得分（1-10）
- $S_{direct}$ = 可导性得分（1-10，仅适用于提供A/B版本试音的候选人）
- $S_{nonverbal}$ = 非语义发声得分（1-10，仅适用于动作类游戏角色）

当两名候选人综合分差在0.5分以内时，配音导演通常优先安排召回（Callback）而非直接拍板，因为0.5分以内的差距已超出试音录音评估的误差范围，必须通过现场实时导演来区分。

以下是一个简化的Python评分计算示例，用于对多名候选人进行批量排序：

```python
def calculate_casting_score(candidates: list[dict]) -> list[dict]:
    """
    计算配音演员选角加权综合评分。
    
    参数说明:
    candidates: 每个候选人为一个字典，包含四个评分维度（1-10分）
    返回: 按综合评分降序排列的候选人列表
    """
    weights = {
        'tone': 0.30,        # 音色匹配度
        'emotion': 0.25,     # 情绪真实性
        'directability': 0.25,  # 可导性
        'nonverbal': 0.20    # 非语义发声能力
    }
    
    for candidate in candidates:
        score = sum(candidate[dim] * w for dim, w in weights.items())
        candidate['final_score'] = round(score, 2)
    
    return sorted(candidates, key=lambda x: x['final_score'], reverse=True)

# 示例：三名候选人评估结果
candidates = [
    {'name': '候选人A', 'tone': 9, 'emotion': 7, 'directability': 8, 'nonverbal': 6},
    {'name': '候选人B', 'tone': 7, 'emotion': 9, 'directability': 9, 'nonverbal': 8},
    {'name': '候选人C', 'tone': 8, 'emotion': 8, 'directability': 7, 'nonverbal': 9},
]

results = calculate_casting_score(candidates)
for r in results:
    print(f"{r['name']}: {r['final_score']}/10")
# 输出:
# 候选人B: 8.15/10
# 候选人C: 8.05/10  ← 差距0.10分，建议安排召回
# 候选人A: 7.90/10
```

## 实际应用

### 《最后生还者》的选角案例

在《最后生还者》（The Last of Us，2013，顽皮狗开发）的制作过程中，创意总监尼尔·德鲁克曼（Neil Druckmann）与配音导演共同为主角乔尔（Joel）面试了超过30名候选演员，最终选定特洛伊·贝克（Troy Baker）。选定的关键理由并非贝克的音色最符合"中年德克萨斯男性"的初始档案描述，而是在召回阶段的即兴场景测试中，贝克能够在导演给出"更疲惫"、"更愤怒"等单一方向指令时，每次都呈现出可区分的、有内在逻辑的变化——这正是"可导性"评估维度的核心价值所在。

艾莉（Ellie）的选角则更为复杂：阿什利·约翰逊（Ashley Johnson）最初在第一轮试音中因"声音过于成熟"被排在候选名单第4位，但在召回阶段演绎一段乔尔发现艾莉被咬后的对峙场景时，约翰逊的表现使评估团队重新修订了对"成熟感"的定义——艾莉的角色弧度需要该成熟感。这一案例说明，召回阶段的场景测试有时会反向修正角色档案本身。

### 大规模NPC选角的工业化流程

对于拥有数百名NPC的开放世界游戏，例如《荒野大镖客：救赎2》（Red Dead Redemption 2，2018，Rockstar Games），选角流程必须工业化处理。Rockstar为该项目录制了超过500,000行对白，涉及超过700名配音演员（其中包括主线主角、支线角色和环境NPC三个层级）。NPC层级通常不进行个体选角，而是通过"声音类型库"（Voice Type Pool）机制：先对约50至80名演员进行通用型试音，再按照声线分类（如"中年男性·西部口音·粗犷型"）建立档案，后期制作时直接从库中匹配并组合，不再单独为某个NPC挑选特定演员。

## 常见误区

### 误区一：最像角色参考声音的演员是最佳选择

这是最常见的选角陷阱。参考声音（Reference Voice）的功能是确立方向，而非设定终点。若演员的试音录音与参考声音高度相似，通常意味着两种情况之一：演员正在进行刻意模仿（Impression），而非真实表演；或者角色设计本身缺乏原创性。优秀的配音导演会主动警惕"最像参考的演员"，并在评分表中设置"原创性减分项"——当某候选人明显在模仿某知名配音演员的风格时，可导性得分通常会被下调2至3分，因为模仿者在后续录制中难以脱离模板自由发挥。

### 误区二：战斗声效可以在后期用音频处理弥补

部分制作团队认为，若演员不擅长非语义发声，可以通过音高变换、失真处理等后期手段补救。这一判断在技术层面存在根本性缺陷：游戏配音与影视配音的关键差异之一，是游戏引擎需要将对白录音与战斗音效绑定到同一角色实体（Character Entity），当两类录音来自不同演员或经过大幅后期处理时，玩家在实时游戏环境中听到的声音会产生可察觉的音色断裂感（Timbre Discontinuity），尤其在无损音频输出设备（如高端耳机）上尤为明显。正确做法是在选角阶段即将非语义发声能力纳入硬性评估，而非寄望于后期处理。
