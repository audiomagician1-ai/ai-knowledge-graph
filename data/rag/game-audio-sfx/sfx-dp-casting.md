# 演员选角

## 概述

演员选角（Voice Casting）是游戏音频制作中对白制作流程的第一个实质性创作环节，指配音导演依据角色设定书（Character Bible）筛选、评估并最终确定配音演员的完整决策过程。与影视配音不同，游戏配音演员需要在同一录音棚 Session 中连续录制数百至数千条台词，因此声音的耐久性（Vocal Endurance）与情绪一致性（Emotional Consistency）是选角标准中权重最高的两项指标。

演员选角作为一个规范化流程，在北美游戏业界于2000年代初期随着大型 RPG 游戏（如《星球大战：旧共和国武士》2003年版）的商业化而逐步成形。彼时业界开始引入影视界的 ADR（Automated Dialogue Replacement）导演经验，将"角色匹配度评分表"等工具移植到游戏配音流程中，形成了沿用至今的试音稿（Audition Slate）制度。

演员选角的决策质量直接影响后续整条对白处理链的效率：一旦选角失误，演员无法稳定输出符合角色要求的声线，将导致大量补录（Pickup Session）和后期 ADR 处理成本激增。据业内估计，一次失败的选角在 AAA 游戏项目中可额外产生 15,000 至 50,000 美元的补救费用，因此配音导演通常将 30% 以上的前期时间投入选角环节。

## 核心原理

### 试音稿的设计与投递

试音稿是选角流程的核心工具。一份标准游戏试音稿通常包含三段内容：**中性对话段**（Neutral Dialogue，约5句，测试演员的基础音色与音量控制）、**情绪高峰段**（Emotional Peak，约3句，测试战斗喊叫、哭泣或愤怒等极端情绪）和**循环台词段**（Loop Lines，约8条"啊""哼""痛苦"等非语言声效）。第三段对许多演员而言是隐性淘汰关卡——能否在重复录制数十次同类非语言音节时保持音色统一，是区分专业配音演员与普通声优的关键指标。

试音稿的台词来源应优先选取游戏**实际剧本中的台词**，而非专门为试音编写的替代文本。这一原则由 Kris Zimmermann（《光晕》系列前配音导演）在其2009年 GDC 演讲中系统阐述：使用真实台词可以让导演直接评估演员与角色文本的化学反应，避免试音阶段表现优秀、正式录音时却与剧情风格脱节的"虚假选角"现象。

### 评估维度与评分方法

标准选角评估通常围绕以下五个维度打分（各维度满分10分）：

| 评估维度 | 说明 |
|------|------|
| 音色匹配度 | 声音基频（Fundamental Frequency）与角色年龄、种族、体型设定的吻合程度 |
| 情绪动态范围 | 从低语到呐喊的响度跨越，专业演员通常需覆盖 ≥30dB 的动态范围 |
| 导演可塑性 | 演员在导演给出"更疲惫"/"更轻挑"等单词指令后的响应速度与准确度 |
| 非语言音效能力 | 战斗音效（Combat Grunts）、努力音（Exertion）等游戏特有声效的表现质量 |
| 台词吞吐耐久性 | 连续录制 4 小时后音色是否出现塌陷（Vocal Fatigue） |

配音导演通常会在三名以内的候选人中组织**对比试音（Comparative Audition）**，将同一段试音稿由不同演员录制的版本进行并排回放，由导演、音频总监和创意总监共同投票，权重通常为5:3:2。

### 本地化选角的特殊考量

国际游戏的本地化版本（如日语版、德语版）在选角时需要额外考虑**音节节奏匹配（Phonetic Timing）**问题。日语对白的音节密度（Mora Density）普遍高于英语，导致日配演员需要在原版配音的时间限制内塞入更多音节，这要求选角时优先考虑语速控制能力强的演员。以《最终幻想 XV》日语版为例，主角诺克蒂斯的英日双版本配音在选角阶段分别独立进行，以避免两版角色声线互相"传染"而失去各自语言文化的表现特性。

## 实际应用

**大型 RPG 游戏的群演选角**：以《巫师 3：狂猎》为例，该游戏英语版拥有超过 950 个有台词角色，其中约 800 个为次要 NPC。针对这类群演（Day Player）的选角通常采用**声线数据库匹配**方法：音频团队维护一个内部演员数据库，每位演员标注其可覆盖的年龄段、方言和情绪区间，选角时按角色需求在数据库中过滤而非重新发起公开试音，可将群演选角时间从每角色 2 天压缩至 2 小时。

**战斗音效导向的选角场景**：在射击游戏（如《使命召唤》系列）的配音选角中，非语言战斗音效（Combat Vocalizations）的占比可高达全部录音内容的 40%。因此选角导演会专门设计"战斗音效压力测试"，要求演员在 20 分钟内连续录制不同强度的"被击中反应音（Hit Reaction）"，以测试其喉部肌肉耐力，筛除可能在实际录音 Session 中过早出现沙声（Vocal Strain）的候选人。

## 常见误区

**误区一：声音好听等于选角正确。** 许多项目新人误以为只需选择音色最悦耳的演员即可。实际上游戏选角的首要标准是"角色真实性（Character Authenticity）"而非声音审美。一个听感"粗糙"的演员，若其声线与角色背景（如来自战场的中年雇佣兵）高度吻合，其优先级应高于声音条件更优秀但与角色违和的演员。Naughty Dog 音频总监 Neil Druckmann 曾公开指出，《最后生还者》中乔尔一角正是以"不完美的质感声线"作为首要选角标准。

**误区二：试音通过即代表正式录音表现稳定。** 试音通常只录制 3–5 分钟，而正式 Session 长达 4–8 小时。部分演员试音阶段表现突出，但缺乏专业配音训练的体能基础，在 Session 后半段音色会明显下滑甚至出现沙哑。解决方法是在试音稿最后加入一段"疲劳模拟测试"：要求演员先大声朗读 10 分钟无关文本以消耗嗓子，再录制核心台词段，以此模拟长录音 Session 的真实状态。

**误区三：本地化版本直接沿用原版选角标准。** 不同语言文化对同一角色的声线预期存在显著差异。例如英语配音中的"英雄男主角"惯用低沉厚重的"蝙蝠侠嗓（Batman Voice）"，但在日语文化语境中同类角色普遍采用更年轻清亮的音色，若本地化选角机械照搬英语版的音色参数指标，将导致日语版角色声线与本地玩家预期严重脱节。

## 知识关联

演员选角流程的执行前提是完整的**配音导演**（Voice Casting Director）角色到位——导演需在选角阶段输出角色设定书和试音稿，并主导对比评估投票。没有配音导演的参与，选角决策往往退化为纯粹的主观音色偏好判断，失去系统性评估框架。

完成演员选角并锁定演员名单后，流程直接进入**对白处理链**（Dialogue Processing Chain）环节。对白处理链的第一步——录音棚搭建与信号链配置——需要依据已选定演员的声线特征进行针对性调整：例如某演员的声线在 3–5 kHz 频段存在刺耳峰值，则录音工程师需要在前置 EQ 环节预设对应的衰减点，避免在后期处理时被迫进行大幅度修正。演员选角的声线档案（Voice Profile）因此成为对白处理链配置的直接输入数据。
