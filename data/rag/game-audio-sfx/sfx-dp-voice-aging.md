# 声音老化处理

## 概述

声音老化处理是通过声学参数操控来模拟人类随年龄增长产生语音特征变化的音频处理技术，在游戏对白制作中专门用于让同一配音演员的录音听起来像不同年龄段的角色。这项技术的核心挑战在于：真实的声音老化并非简单的音调降低，而是涉及声门闭合不全（glottal incompetence）、共鸣腔萎缩、呼吸控制减弱等生理变化的综合声学表现。

从技术发展来看，早期游戏制作（2000年代之前）通常为同一角色的老年版本重新招募不同演员。声音老化处理作为后期制作手段在2010年代随开放世界叙事游戏兴起而广泛普及——《最后生还者》（Naughty Dog, 2013年）和《荒野大镖客：救赎2》（Rockstar Games, 2018年）均大量运用此类处理来呈现角色跨越数十年的时间叙事。《荒野大镖客：救赎2》的音频总监Rob Nelson在GDC 2019演讲中明确提及，该项目中约30%的老年版角色对白依赖后期老化处理而非重新招募演员，由此节省了超过600小时的录音棚时间。

声音老化处理保留原始音色"DNA"的同时，可让声音在20岁与70岁之间实现可信转变——这对于玩家对角色的情感认同至关重要。一位演员通过数百行对白建立的特定声音识别度，是任何后期聘用替代演员都无法复制的叙事资产。

## 核心原理

### 声门闭合不全的声学模拟

65岁以上人群的声带因弹性组织退化无法完全闭合，导致声门处产生持续性气流泄漏。这在声学上表现为：基频（F0）稳定性下降，谐波噪声比（HNR, Harmonics-to-Noise Ratio）通常从年轻人的约20 dB降至老年人的12–15 dB（Ferrand, 2002）。

处理手段是在干净录音上叠加受控白噪声层，与原始信号的混合比约在 $-18\,\text{dB}$ 至 $-12\,\text{dB}$ 之间。同时施加轻微调制：
- **Jitter（频率不规则性）**：约 0.5%–2.0%，破坏基频的周期稳定性
- **Shimmer（振幅不规则性）**：约 1.0%–4.0%，使谐波峰值产生逐周期幅度抖动

两者共同作用使声音失去年轻人特有的"谐波纯净感"，是老化质感最直接的感知来源。iZotope RX 11的 **Voice De-noise** 模块可在分离噪声层后精确控制回混比例，实现上述参数的数字化注入。

### 声道共鸣特性的频谱塑形

老年人口腔肌肉张力降低与咽喉组织松弛，使第一共振峰（F1）和第二共振峰（F2）的频率及带宽发生可测量偏移。具体而言，老年男性的 F1 频率平均提高约 5–8%（因口腔有效容积等效缩小），而高频共振峰的带宽则加宽，导致辅音清晰度下降（Linville, 2001，《Vocal Aging》, Singular Publishing Group）。

在制作中，这通过参数均衡实现：
- **2.5 kHz–5 kHz**：施加 1–3 dB 宽带衰减，模糊齿擦音（/s/、/f/、/θ/）的高频摩擦能量
- **200–400 Hz**：轻微提升 1–2 dB，增加胸腔共鸣的"厚重感"，针对的是泛音列中低段而非基频本身
- **800 Hz–1.2 kHz**：适度衰减 1 dB，减少口腔前向辐射能量，模拟舌体运动幅度缩小的效应

注意这三段均衡须使用较宽的 Q 值（Q ≈ 0.8–1.2），避免产生梳状滤波感。

### 基频与语速的老化特征

语音基频（F0）在老化过程中呈现**性别差异化变化**，这是最易被忽略的反直觉特征：

| 性别 | 年轻人平均 F0 | 老年人平均 F0 | 变化方向 |
|------|------------|------------|--------|
| 男性 | 约 120 Hz   | 约 100 Hz   | **降低** 10–30 Hz（声带质量增加）|
| 女性 | 约 210 Hz   | 约 230–260 Hz | **升高** 20–50 Hz（雌激素水平下降导致声带变薄）|

女性声音随年龄升高这一特征，若不加以处理，直接对女性角色进行降调会产生截然相反的听感错误。许多制作中的老年女性角色因此出现"声音反而更年轻"的感知失真。

语速方面，老年人的语音速率（speech rate）平均比年轻人慢 15%–25%（Ramig, 1983），停顿频率与时长同步增加。在 Wwise 或 FMOD 的对白触发设计中，需相应调整字幕显示的时间偏移量，通常为原始时长的 1.15–1.25 倍。

### 呼吸整合与老化质感

在完成呼吸音处理（本技术先修技能）的基础上，老化处理需将气声（breathiness）深度整合进音节结构，而非仅作为外部叠加。具体做法是对每个音节的声门冲击（glottal pulse）波形做开放商（open quotient, OQ）调整：

$$OQ = \frac{T_{\text{open}}}{T_0}$$

其中 $T_{\text{open}}$ 为声门开放时间，$T_0$ 为基频周期。典型年轻人的 $OQ$ 约为 0.40–0.50，老年人则提升至 0.55–0.65。实践中常借助 iZotope RX 的 **Spectral Repair** 或 Antares AVOX 插件套件中的 **Breath Control** 模块来逼近这一比例变化，而非直接编辑波形。

## 关键处理算法与工具链

### 基于相位声码器的时间-音高解耦

声音老化处理的核心工程挑战是：音高（F0）变化、语速变化和谐波结构变化必须**独立可控**，而非简单地通过变速播放联动。相位声码器（Phase Vocoder）通过将信号分解为短时傅里叶变换（STFT）帧并分别操控频率-幅度包络，实现了时间轴与音高轴的解耦。

以 Reaper 的 ReaPitch 插件为例，对男性角色进行老化处理的参考参数设置如下：

```python
# 声音老化处理参考参数（男性角色，20岁 → 65岁）
aging_params = {
    "pitch_shift_semitones": -1.5,        # F0 降低约 10 Hz（等效约 1.5 半音）
    "time_stretch_ratio": 1.18,           # 语速减慢 18%
    "formant_shift": +0.03,               # 共振峰轻微上移 3%（F1 变化补偿）
    "noise_mix_dB": -15,                  # 气噪混合比 -15 dB
    "jitter_percent": 1.2,                # 频率不规则性 1.2%
    "shimmer_percent": 2.5,               # 振幅不规则性 2.5%
    "high_freq_cut_dB": -2.0,             # 2.5k-5kHz 衰减量
    "low_mid_boost_dB": +1.5,             # 200-400Hz 提升量
}

# 女性角色老化（20岁 → 65岁）——注意音高方向相反
aging_params_female = {
    "pitch_shift_semitones": +1.0,        # F0 升高约 20 Hz（等效约 1 半音）
    "time_stretch_ratio": 1.20,
    "formant_shift": +0.04,
    "noise_mix_dB": -14,
    "jitter_percent": 1.5,
    "shimmer_percent": 3.0,
}
```

### 多段处理顺序的工程规范

处理链顺序直接影响最终质感。推荐的信号流为：

1. **降噪预处理**（iZotope RX Voice De-noise）→ 移除录音底噪，避免噪声被后续处理放大
2. **时间拉伸**（élastique Pro 算法，Reaper 内置）→ 优先于音高变换执行，减少相位伪影
3. **音高与共振峰分离调整**（ReaPitch 或 Antares Auto-Tune Pro）→ 共振峰偏移量须与音高偏移量**反向补偿**约 30%
4. **谐波噪声注入**（Waves Aphex Twin Aural Exciter + 自定义噪声层）
5. **参数均衡塑形**（FabFilter Pro-Q 3，三段均衡见上节）
6. **动态压缩放宽**（加宽 attack/release，模拟老年人呼吸控制减弱的动态范围扩大）

## 实际应用案例

### 案例：《最后生还者》乔尔的时间线处理

《最后生还者》（2013）开场序列需要乔尔（配音：Troy Baker）在短短20分钟内从情感健全的父亲转变为失去女儿后的中年幸存者，而续作《最后生还者 第二部》（2020）则展示了更年老的版本。音频团队（音频总监：Neil Druckmann 与音频设计师 Robert Krekel）采用了将同一录音素材处理的方式处理部分闪回场景：在 Troy Baker 30多岁的"年轻态"录音基础上，对特定闪回的"老年态"版本施加约 -2 个半音的音高下移、1.15 倍时间拉伸，以及 -14 dB 气噪混合，使两个时间线上的乔尔始终共享相同的情感底色。

### 案例：程序化老化参数的 LOD 设计

在大型开放世界游戏中，NPC 可能因随机生成系统出现从儿童到老人的年龄分布。育碧在《刺客信条：奥德赛》（2018）中实现了一套基于年龄参数的程序化声音老化系统：每个 NPC 的年龄标签（integer, 15–80）映射至一组预计算的均衡曲线和动态参数，在运行时通过 Wwise RTPC（Real-Time Parameter Control）实时调制。具体实现中，年龄参数每增加 10 年，高频衰减量递增 0.4 dB，气噪混合比递增 1 dB，语速触发器的播放速率递减 3%。

## 常见误区

### 误区一：对女性声音也施加降调处理

如前文数据所示，65岁以上女性的 F0 平均**高于**年轻女性，对女性角色做降调处理会产生听感上"反老化"的错误效果。正确做法是在音高上做轻微上移（+0.5至+1.5半音）的同时，通过气声与谐波劣化来传达老化质感。

### 误区二：将老化处理等同于单纯降低音量与高频

仅做"音量降低 + 高频滚降"会产生"距离感增加"而非"年龄感增加"的错误听感——前者是空间处理的语言，后者依赖声门层面的谐波噪声比变化。区分这两者的感知依据是：年龄处理后的声音应在近距离监听下仍保持存在感，而非听起来像是从远处传来。

### 误区三：忽视语言辅音清晰度的年龄特征

老年人对高频辅音（/s/、/ʃ/、/f/）的发音精度因舌体灵活性下降而降低，齿擦音能量减弱约 3–6 dB（在 5–8 kHz 频段）。若只处理元音区域的共振峰而忽视辅音能量，老化声音会产生"有气无力但咬字清晰"的不协调感，暴露出人工处理的痕迹。

### 误区四：对所有年龄段使用线性参数内插
