# 多元回归

## 概述

多元回归（Multiple Regression）是将一个被解释变量（因变量）对**两个或两个以上**解释变量（自变量）进行线性拟合的计量方法。其基本模型形式为：

$$Y_i = \beta_0 + \beta_1 X_{1i} + \beta_2 X_{2i} + \cdots + \beta_k X_{ki} + \varepsilon_i$$

其中 $Y_i$ 为第 $i$ 个观测的因变量，$X_{ji}$ 为第 $j$ 个解释变量，$\beta_j$ 为对应的偏回归系数（partial regression coefficient），$\varepsilon_i$ 为随机误差项。与简单线性回归（一元回归）不同，多元回归中每个 $\beta_j$ 衡量的是**在其他所有解释变量保持不变的条件下**，$X_j$ 每变动一个单位对 $Y$ 的影响，这一"控制其他变量"的特性是多元回归最重要的统计意义。

多元回归的理论基础在20世纪初由卡尔·皮尔逊（Karl Pearson）和弗朗西斯·高尔顿（Francis Galton）奠定，随后尤尔（G. Udny Yule）于1897年将其应用于济贫法研究，成为早期计量经济学的经典案例。多元回归之所以在经济学中不可或缺，根本原因在于经济现象几乎永远是**多因素决定的**：仅用一个变量回归收入往往产生严重偏误，因为教育年限、工作经验、地区等因素同时影响工资，忽略任何一个都会污染其他系数的估计。

## 核心原理

### 普通最小二乘法（OLS）在多元情形下的推广

多元回归依然使用OLS准则，即最小化残差平方和 $\sum_{i=1}^n \hat{\varepsilon}_i^2$。用矩阵表示为：

$$\hat{\boldsymbol{\beta}} = (\mathbf{X}^\top \mathbf{X})^{-1} \mathbf{X}^\top \mathbf{Y}$$

其中 $\mathbf{X}$ 为 $n \times (k+1)$ 的设计矩阵（含常数列）。OLS估计量在高斯-马尔科夫假设（GM假设）满足时具有BLUE性质（Best Linear Unbiased Estimator）。多元情形下，GM假设额外要求**不存在完全多重共线性**，即设计矩阵 $\mathbf{X}$ 满秩，$\text{rank}(\mathbf{X}) = k+1$。

### 多重共线性（Multicollinearity）

多重共线性是指模型中两个或多个解释变量之间存在高度线性相关。**完全多重共线性**（如某一变量完全等于另一变量的线性组合）会导致 $(\mathbf{X}^\top \mathbf{X})$ 不可逆，OLS系数无法计算。更常见的是**近似多重共线性**，其典型后果包括：

- **系数标准误膨胀**：方差膨胀因子（VIF）定义为 $\text{VIF}_j = \frac{1}{1 - R_j^2}$，其中 $R_j^2$ 是将 $X_j$ 对其余所有解释变量回归所得的 $R^2$。通常认为 $\text{VIF} > 10$ 表示严重共线性问题。
- **系数估计不稳定**：样本微小变化可能导致系数符号翻转或数值大幅波动。
- **$t$ 统计量偏小**：即便变量对 $Y$ 有真实影响，高共线性下 $t$ 检验往往无法拒绝零假设，产生伪性的"不显著"。

值得注意的是，多重共线性**不影响OLS估计量的无偏性**，仅影响其精度（方差），这是一个常被混淆的要点。

### 遗漏变量偏误（Omitted Variable Bias, OVB）

若真实模型含有变量 $X_2$，但研究者仅估计 $Y = \beta_0 + \beta_1 X_1 + \varepsilon$，则 $\hat{\beta}_1$ 的偏误公式为：

$$\text{Bias}(\hat{\beta}_1) = \beta_2 \cdot \tilde{\delta}_1$$

其中 $\tilde{\delta}_1$ 是将被遗漏变量 $X_2$ 对已纳入变量 $X_1$ 做辅助回归所得的斜率系数。这意味着OVB的方向和大小同时取决于**遗漏变量对 $Y$ 的真实效应 $\beta_2$** 以及**遗漏变量与 $X_1$ 的相关程度 $\tilde{\delta}_1$**。例如，仅用教育年限回归工资时，若遗漏"能力"变量（与教育正相关且对工资有正效应），则教育的系数会被向上偏误估计——这是Mincer工资方程研究中的经典问题。

### 拟合优度的调整：调整后 $R^2$

多元回归中纳入更多变量**必然**使普通 $R^2$ 不下降，因为 $R^2 = 1 - \frac{SSR}{SST}$，增加变量只会减少SSR。为此引入调整后 $R^2$（Adjusted $R^2$）：

$$\bar{R}^2 = 1 - \frac{SSR/(n-k-1)}{SST/(n-1)}$$

该指标对增加解释变量施加自由度惩罚。若新增变量对模型的实质贡献不足以弥补损失的自由度，$\bar{R}^2$ 会下降，从而为变量筛选提供参考。

## 实际应用

**明塞尔工资方程**是多元回归在劳动经济学中最著名的应用之一。雅各布·明塞尔（Jacob Mincer）于1974年提出的基准模型为：

$$\ln W_i = \beta_0 + \beta_1 \text{Educ}_i + \beta_2 \text{Exp}_i + \beta_3 \text{Exp}_i^2 + \varepsilon_i$$

其中同时纳入教育年限和工作经验（以及经验的平方项以捕捉非线性），对美国数据的估计通常得到教育年限的系数约为0.07–0.10，即每多受一年教育工资提高约7%–10%，该系数在控制了经验变量后比一元回归结果更为可信。

**需求弹性估计**中，研究者常以商品需求量为因变量，同时纳入商品自身价格、替代品价格、消费者收入等多个解释变量，方能得到净价格弹性。若仅做一元回归，价格系数会因遗漏收入变量而产生偏误（在价格与收入相关时尤为明显）。

## 常见误区

**误区一：纳入尽可能多的变量可以"消除"所有偏误。**
过度纳入变量（over-controlling）同样有害。若控制了某个中间变量（mediator）——即 $X_1$ 影响 $Y$ 的因果路径上的变量——则 $\beta_1$ 的系数会被"阻断"，低估 $X_1$ 的总效应。例如，若研究教育对收入的影响，同时控制职业类型，则会部分消除教育通过影响职业而带来的收入提升，导致教育系数低于其真实总效应。

**误区二：多重共线性意味着模型设定错误，应删除其中一个变量。**
共线性仅是数据的特征，不代表理论上变量不应同时出现。强行删除理论上应该纳入的变量会引入遗漏变量偏误，这通常比共线性导致的标准误膨胀危害更大。正确应对方式包括扩大样本量、使用岭回归（Ridge Regression）或主成分分析等，而非简单删除变量。

**误区三：$\bar{R}^2$ 高就说明模型正确。**
$\bar{R}^2 = 0.9$ 的模型仍可能因内生性问题（如遗漏变量与现有解释变量相关）导致系数估计有偏。$\bar{R}^2$ 仅描述拟合程度，无法验证因果识别的有效性。

## 知识关联

学习多元回归需要掌握**简单线性回归**（一元回归）的OLS原理，多元情形本质上是其矩阵化推广。理解偏回归系数的"控制其他变量"含义，是从一元推广到多元最关键的概念跨越。

在后续学习中，多元回归暴露出的**遗漏变量偏误**问题直接导向**内生性问题**的深入讨论（包括工具变量法和两阶段最小二乘法的引入）。当数据具有截面和时间双重维度时，**面板数据分析**通过个体固定效应控制不可观测的遗漏变量，是解决OVB的重要工具。若数据存在时间序列特征，则进入**时间序列分析**框架。此外，多元回归中若误差项方差随解释变量系统性变化，则涉及**异方差**问题，需要使用怀特标准误（White robust standard errors）进行修正。
