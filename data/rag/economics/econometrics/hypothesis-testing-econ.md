# 假设检验

## 概述

假设检验是计量经济学中用于判断样本数据是否支持某一经济理论命题的统计推断方法。其基本逻辑是：先提出关于总体参数的假设，再利用样本统计量计算出一个检验统计量，最后根据该统计量落入拒绝域还是接受域来决定是否推翻原假设。在经济学中，假设检验常用于判断"消费对收入的边际倾向是否显著不为零"或"某政策变量对GDP的影响是否统计显著"等具体问题。

假设检验的理论框架由耶日·内曼（Jerzy Neyman）与卡尔·皮尔逊之子埃贡·皮尔逊（Egon Pearson）在20世纪30年代共同建立，形成了著名的"内曼-皮尔逊框架"。这一框架明确区分了两类错误：第一类错误（拒真，Type I Error）和第二类错误（纳伪，Type II Error），并将第一类错误的概率上限定义为显著性水平α，经济学实证研究中最常用的α取值为0.05或0.01。

在计量经济学实践中，假设检验使研究者可以将回归系数的点估计与其统计不确定性结合起来评价，避免仅凭系数大小下结论。例如，一个回归系数估计值为0.3，但若其标准误为0.5，则该系数在统计上并不显著，无法排除真实值为零的可能性。

## 核心原理

### 原假设与备择假设的构造

假设检验从构建一对对立假设开始。**原假设（H₀）** 通常是研究者希望通过数据去反驳的保守命题，**备择假设（H₁）** 则是研究者倾向于支持的命题。在计量经济学中，最典型的原假设是 H₀: βₖ = 0，即第k个解释变量对被解释变量没有线性影响。备择假设可以是双侧的（H₁: βₖ ≠ 0）或单侧的（H₁: βₖ > 0）。研究者需要在收集数据之前确定假设方向，事后根据数据结果更改假设方向会导致显著性水平失效，这种做法在经济学实证研究中被称为"p-hacking"。

### t检验：单个系数的显著性

对线性回归模型 y = Xβ + ε 中的单个系数 βₖ，t检验统计量定义为：

$$t = \frac{\hat{\beta}_k - \beta_k^0}{SE(\hat{\beta}_k)}$$

其中 β_k^0 是原假设下的参数值（通常为0），SE(β̂ₖ) 是 β̂ₖ 的标准误。在误差项服从正态分布的假设下，该统计量服从自由度为 n-K 的t分布（n为观测数，K为估计参数数量，含截距项）。当样本量较大时，t分布近似于标准正态分布，此时常用的临界值为：α=0.05双侧检验对应 |t| > 1.96，α=0.01双侧检验对应 |t| > 2.576。例如，在一个样本量为100、含3个参数的工资方程中，若教育年限系数的t统计量为3.2，远超自由度97对应的临界值1.985，则在5%显著性水平下拒绝"教育年限对工资无影响"的原假设。

### F检验：多个系数的联合显著性

当需要同时检验多个系数的联合假设时，t检验不再适用，需使用F检验。F统计量通过比较有约束模型（施加H₀限制）与无约束模型的残差平方和来构建：

$$F = \frac{(RSS_R - RSS_{UR})/q}{RSS_{UR}/(n-K)}$$

其中 RSS_R 和 RSS_{UR} 分别为有约束和无约束模型的残差平方和，q 为约束个数，n-K 为无约束模型的自由度。该统计量在H₀成立时服从 F(q, n-K) 分布。一个典型应用是检验整个回归方程的联合显著性（即所有斜率系数同时为零），此时q = K-1，F统计量通常在标准回归输出表中直接报告。若一个包含5个解释变量的宏观经济模型的F统计量为18.7，对应p值小于0.001，则强烈拒绝"所有解释变量均无效"的原假设。

### p值的经济学解读

p值是指在原假设成立的条件下，观察到等于或比当前样本更极端检验统计量的概率。p值越小，说明在H₀为真时观察到当前数据的可能性越低，从而提供了更强的拒绝H₀的证据。值得注意的是，p值并非"H₀为真的概率"——这是一个常见的误解。在经济政策评估研究中，p=0.04意味着：若该政策真的无效，则有4%的概率仅凭随机误差就能得到当前这样或更大的估计效果，因此在5%水平下拒绝"政策无效"这一原假设。

## 实际应用

**工资方程的系数检验**：在明赛尔工资方程 ln(wage) = β₀ + β₁·education + β₂·experience + β₃·experience² + ε 中，研究者通常用t检验分别验证教育年限和工作经验的回归系数是否显著为正。若β₁的t值为8.5而β₃的t值为-6.2，两者均在1%水平下显著，则支持"教育回报率为正、经验的边际回报递减"的经济理论预测。

**需求弹性的假设检验**：在双对数需求模型 ln(Q) = α + β·ln(P) + γ·ln(I) + ε 中，可以用t检验验证 H₀: β = -1（单位价格弹性）与 H₀: γ = 1（单位收入弹性）。若β的估计值为-0.75、标准误为0.08，则t统计量 = (-0.75-(-1))/0.08 = 3.125，在5%水平下拒绝单位弹性假设，说明该商品需求价格弹性的绝对值显著小于1，属于价格缺乏弹性的商品。

**Chow检验（结构突变检验）**：这是F检验的一个重要应用，用于判断回归关系在两个子样本（如改革前后两个时期）之间是否发生结构性变化。若对1980年代和1990年代的消费函数分别估计，再通过F检验比较合并模型与分组模型的残差平方和，即可判断消费行为在两个年代之间是否存在统计显著的结构突变。

## 常见误区

**误区一：统计显著等于经济显著**。一个系数通过了t检验（统计显著），并不意味着该变量在经济层面上有实质意义。当样本量n非常大（如n=100,000），即使极小的系数（如β=0.0001）也可能因标准误极小而呈现高度统计显著，但其经济含义几乎可以忽略不计。正确做法是同时报告系数大小和置信区间，以经济意义来判断实质重要性。

**误区二：未拒绝H₀等于证明H₀为真**。当t检验未能在5%水平下拒绝原假设时，这并不意味着"证明了该变量没有影响"，而只是说明"数据提供的证据不足以在当前显著性水平和样本量下推翻H₀"。若样本量很小（如n=20），检验功效（Power = 1 - β）可能很低，无法检测出经济上显著的效应。

**误区三：多重假设检验中的显著性水平失控**。若在同一数据集上同时进行20个独立的t检验，即使所有原假设均为真，平均仍有约1个检验（5%×20=1）会错误地在5%水平下显示显著，这称为"多重比较问题"（Multiple Comparisons Problem）。经济学实证研究中应采用邦费罗尼校正（Bonferroni Correction）等方法，将每个单独检验的显著性水平调整为α/m（m为检验总数），以控制家族错误率。

## 知识关联

假设检验建立在线性回归的OLS估计结果之上：t检验直接使用OLS估计量 β̂ 及其标准误 SE(β̂)，而SE(β̂) 的计算依赖于OLS的方差-协方差矩阵 Var(β̂) = σ²(X'X)⁻¹。因此，线性回归的经典假设（尤其是误差项同方差性与无自相关性）对于t统计量和F统计量的分布推导至关重要：若存在异方差或自相关，OLS标准误将有偏，导致t检验和F检验得到错误的推断结论，此时需要使用怀特（White）稳健标准误或纽伊-韦斯特（Newey-West）标准误进行修正。假设检验的思想还延伸至工具变量估计中的弱工具变量检验（第一阶段F统计量须大于10）以及协整检验等更高级的计量方法，是贯穿整个实证经济学研究的基础性推断工具。
