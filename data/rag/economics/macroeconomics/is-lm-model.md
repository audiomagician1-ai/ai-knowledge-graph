# IS-LM模型

## 概述

IS-LM模型是由英国经济学家约翰·希克斯（John Hicks）于1937年在论文《凯恩斯先生与古典派》中首次提出，后经汉森（Alvin Hansen）推广，因此也称为希克斯-汉森模型。该模型将凯恩斯《通论》中的产品市场均衡与货币市场均衡整合在同一坐标系内，以利率（r）为纵轴、国民收入（Y）为横轴，通过IS曲线与LM曲线的交点，同时确定均衡利率和均衡收入水平。

IS代表Investment-Saving（投资-储蓄），LM代表Liquidity preference-Money supply（流动性偏好-货币供给）。这两条曲线分别描述产品市场和货币市场达到均衡时，利率与收入的所有可能组合。IS-LM模型的核心贡献在于揭示两个市场如何相互约束：产品市场的均衡收入影响货币需求，货币市场的均衡利率又反过来影响投资和产出，两个市场必须同时出清才能确定经济的短期均衡。

该模型是20世纪中叶宏观经济学教学和政策分析的主流框架，尽管后来遭到货币主义和理性预期学派的批评，但在分析财政政策与货币政策的相对效力时仍具有不可替代的直觉价值。

## 核心原理

### IS曲线的推导

IS曲线描述产品市场均衡时，所有（Y, r）组合的轨迹。产品市场均衡条件为：

**Y = C + I + G**

其中消费 C = C₀ + c·Y（c为边际消费倾向，0 < c < 1），投资 I = I₀ - b·r（b > 0，表示投资对利率的敏感度）。整理后得到IS方程：

**Y = (C₀ + I₀ + G) / (1 - c) - [b / (1 - c)] · r**

IS曲线斜率为 **−(1−c)/b**，斜率为负，即利率越高，投资越少，均衡收入越低。IS曲线越平坦（斜率绝对值越小），意味着投资对利率越敏感（b越大）或乘数 1/(1−c) 越大。政府支出G增加时，IS曲线向右平移，平移量等于ΔG乘以财政乘数 1/(1−c)。

### LM曲线的推导

LM曲线描述货币市场均衡时，所有（Y, r）组合的轨迹。货币市场均衡条件为实际货币供给等于实际货币需求：

**M/P = L(Y, r) = k·Y − h·r**

其中 k > 0 表示收入对货币需求的正效应，h > 0 表示利率对货币需求的负效应（流动性偏好）。整理得LM方程：

**r = (k/h)·Y − (1/h)·(M/P)**

LM曲线斜率为 **k/h**，斜率为正，即收入增加导致货币需求上升，利率随之升高。当货币供给M增加时，LM曲线向右下方平移。当h趋向无穷大（凯恩斯流动性陷阱），LM曲线变为水平线；当h趋向零（古典货币数量论极端），LM曲线变为垂直线。

### IS-LM联立均衡

将IS方程与LM方程联立求解，可得均衡收入 Y* 和均衡利率 r*。均衡收入的完整表达式为：

**Y* = [h·(C₀ + I₀ + G) + b·(M/P)] / [h·(1−c) + b·k]**

分母 h(1−c) + bk 称为IS-LM联立均衡的综合乘数分母，其中h(1−c)体现货币市场对财政乘数的"挤出"制约，bk体现货币政策效力受产品市场制约的程度。财政政策乘数为 h / [h(1−c) + bk]，货币政策乘数为 b / [h(1−c) + bk]，两者之和体现了政策传导效力在两个市场约束下的折减。

## 实际应用

**财政政策的挤出效应分析**：政府扩大赤字支出，IS曲线右移，但在LM曲线斜率为正的约束下，均衡利率上升，私人投资被部分挤出。挤出程度取决于LM曲线斜率：若货币需求利率弹性h极大（LM近似水平），挤出效应极小，财政政策完全有效；若h极小（LM近似垂直），财政乘数趋近于零，政府支出几乎完全挤出私人投资。

**2008年金融危机政策分析**：美联储将联邦基金利率降至接近零的水平，部分经济学家认为经济陷入"流动性陷阱"，即LM曲线接近水平，货币政策效力丧失。此时凯恩斯主义者主张依赖财政刺激（IS右移）而非进一步宽松货币政策，这一政策判断正是基于IS-LM框架中流动性陷阱情形的推论。

**开放经济延伸**：蒙代尔-弗莱明模型（1963年）将IS-LM框架扩展至开放经济，在固定汇率下财政政策完全有效而货币政策完全无效；在浮动汇率下结论恰好相反。这一结论的逻辑起点仍是IS-LM的基本框架。

## 常见误区

**误区一：认为IS曲线反映货币现象**。IS曲线完全由产品市场决定，其位置取决于自主消费、自主投资、政府支出和税收，与货币供给无关。货币供给的变化只移动LM曲线。混淆两条曲线的决定因素是初学者最常见的错误，记住"IS无M，LM无G"有助于区分。

**误区二：认为均衡点代表充分就业**。IS-LM模型描述产品市场与货币市场的同时出清，但均衡收入Y*可能低于充分就业收入Yf。模型本身不含劳动市场，需结合总供给曲线才能判断是否存在产出缺口。凯恩斯的核心论点之一正是经济可能长期停滞在低于充分就业的IS-LM均衡点。

**误区三：认为LM曲线斜率不变**。在实证研究中，货币需求的利率弹性（h参数）并非稳定常数。20世纪80年代金融创新导致美国货币需求函数发生结构性断裂，LM曲线形状变得难以估计，这是货币主义规则遭受质疑的重要原因之一。

## 知识关联

**与总需求的关系**：总需求曲线（AD曲线）实际上可以从IS-LM模型中推导得出。对于每个价格水平P，实际货币余额M/P不同，LM曲线位置随之变化，与IS曲线交出不同的均衡收入，将这些（P, Y*）点连接起来即为向右下倾斜的总需求曲线。因此IS-LM模型是AD曲线背后的微观机制。

**对财政政策与货币政策分析的支撑**：财政政策分析讨论IS曲线移动的幅度和方向；货币政策分析讨论LM曲线移动及利率传导机制，两者的政策效力对比必须在IS-LM联立框架下才能得出完整结论，孤立分析单个市场会高估政策乘数。

**向蒙代尔-弗莱明模型的拓展**：IS-LM中IS方程加入净出口项NX(e)（e为汇率），并补充国际资本流动方程（BP曲线），形成三市场同时均衡的蒙代尔-弗莱明模型，这是学习开放宏观经济学的直接前置步骤。
