# 菲利普斯曲线

## 概述

菲利普斯曲线（Phillips Curve）描述通货膨胀率与失业率之间的统计权衡关系，由新西兰经济学家阿尔班·威廉·菲利普斯（Alban William Housego Phillips）于1958年在《经济学》期刊（*Economica*）发表的论文《1861—1957年英国货币工资变动率与失业率的关系》中首次系统提出。菲利普斯爬梳了英国长达97年的历史数据，发现名义工资增长率与失业率之间存在显著的非线性负相关：当失业率约为2.5%时，工资增速约为0；当失业率降至1%时，工资年增速可达8%以上；而当失业率升至5%至6%时，工资增速趋近于零甚至为负。这一实证关系被萨缪尔森（Paul Samuelson）与索洛（Robert Solow）于1960年转化为通胀率与失业率之间的负相关关系，并在《美国经济评论》中命名为"菲利普斯曲线"，由此进入主流宏观政策讨论（Samuelson & Solow, 1960）。

菲利普斯曲线在1960年代被肯尼迪与约翰逊政府视为"政策菜单"：接受4%的通胀，换取4%以下的失业率；或忍受5%的失业率，保持不到2%的通胀。然而1970年代的"滞胀"（stagflation）——美国1974年失业率升至7.2%，同年CPI通胀率高达11%——彻底颠覆了这种简单解读，促使经济学界发展出附加预期的菲利普斯曲线与新凯恩斯菲利普斯曲线等修正框架。

## 核心原理

### 短期菲利普斯曲线：价格黏性与需求拉动

短期菲利普斯曲线（Short-Run Phillips Curve，SRPC）向右下倾斜的根本机制是**价格和工资的短期黏性**。当总需求扩张（例如政府增加财政支出或央行降息），企业订单增加，为满足产能需求开始竞相招募工人，劳动力市场趋紧，失业率下降。与此同时，企业在成本上升与需求旺盛的双重压力下提高商品价格，通胀率攀升。短期内，工人的工资谈判以既有的通胀预期为基准，若实际通胀超出预期，实际工资被低估，劳动供给短暂增加，进一步强化就业扩张。这种价格错觉（money illusion）在菲利普斯最初的97年数据中清晰可见：每当英国商业繁荣期，失业率普遍低于3%，名义工资增速超过4%。

定量描述这一关系的标准形式为：

$$\pi_t = \pi_t^e - \alpha(u_t - u_n) + \varepsilon_t$$

其中：
- $\pi_t$ 为 $t$ 期实际通胀率
- $\pi_t^e$ 为公众在 $t$ 期初形成的预期通胀率
- $u_t$ 为实际失业率，$u_n$ 为自然失业率（natural rate of unemployment）
- $\alpha > 0$ 为斜率参数，衡量失业缺口对通胀的敏感度，美国历史估计值约为 $0.5$（Ball & Mazumder, 2011）
- $\varepsilon_t$ 为供给冲击项，如石油价格骤升

当 $u_t < u_n$（劳动市场偏紧），即使预期通胀不变，实际通胀也将超过预期，体现短期权衡。

### 弗里德曼—费尔普斯修正：自然失业率与长期垂直线

1968年，弗里德曼（Milton Friedman）在美国经济学会主席演讲中（发表于《美国经济评论》，1968年3月）与费尔普斯（Edmund Phelps）分别独立提出"附加预期的菲利普斯曲线"（Expectations-Augmented Phillips Curve）。其核心论点是：**在长期，理性的工人和企业会完全修正通胀预期，使 $\pi_t^e = \pi_t$**，代入上式后括号项必然等于零，得到 $u_t = u_n$。

这意味着无论通胀率定在2%、5%还是10%，长期均衡失业率始终回归自然失业率 $u_n$，因此**长期菲利普斯曲线（Long-Run Phillips Curve，LRPC）是一条在 $u = u_n$ 处的垂直线**。弗里德曼据此预言：政府若试图将失业率维持在 $u_n$ 以下，必须不断提高通胀率以"欺骗"工人，最终导致通胀螺旋式上升——这正是1970年代美国滞胀的理论预测，1976年弗里德曼因此获得诺贝尔经济学奖，费尔普斯则于2006年同因此研究获奖。

### 预期形成机制：适应性预期与理性预期的分歧

短期曲线的动态行为取决于公众如何更新 $\pi_t^e$：

**适应性预期（Adaptive Expectations）**：工人根据过去实际通胀的加权平均推断未来通胀，如：

$$\pi_t^e = \lambda \pi_{t-1} + (1-\lambda)\pi_{t-2} + \cdots$$

当 $\lambda=1$ 时退化为"向后看一期"预期，即 $\pi_t^e = \pi_{t-1}$，这使短期曲线随通胀历史记录上下移动，解释了1970年代每次石油危机后曲线的向上跳跃。

**理性预期（Rational Expectations）**：卢卡斯（Robert Lucas）于1972年提出，若公众充分利用所有可用信息（包括央行政策意图），则预期误差 $\pi_t - \pi_t^e$ 不会系统性偏差，菲利普斯曲线权衡仅在预期外的政策冲击下短暂出现。这一视角暗示，一个**可信的**央行宣布反通胀政策时，预期立刻下调，SRPC快速下移，反通胀的失业代价大幅缩小——与依赖适应性预期的预测截然相反。

## 关键公式与牺牲率计算

反通胀政策的就业代价通常用**牺牲率（Sacrifice Ratio）**量化：

$$\text{牺牲率} = \frac{\text{累计GDP损失百分点}}{\text{通胀率下降百分点}}$$

以美联储主席沃尔克（Paul Volcker）1979—1983年紧缩为案例：

| 年份 | 通胀率（CPI，%） | 失业率（%） |
|------|----------------|------------|
| 1979 | 13.3 | 5.8 |
| 1980 | 12.5 | 7.1 |
| 1981 | 8.9 | 7.6 |
| 1982 | 3.8 | 9.7 |
| 1983 | 3.2 | 9.6 |

通胀率从13.3%降至约3%，累计GDP缺口约为16个百分点，牺牲率约为1.6至2.0（Ball, 1994）。对比而言，若公众对沃尔克政策完全可信且立刻调整预期，理性预期模型预测牺牲率可接近于零——实际结果居中，说明预期调整是渐进而非瞬时的。

用Python简单模拟适应性预期下的菲利普斯曲线动态：

```python
import numpy as np
import matplotlib.pyplot as plt

# 参数设定
alpha = 0.5       # 菲利普斯曲线斜率
u_n = 0.05        # 自然失业率 5%
lambda_ = 0.8     # 适应性预期权重

# 政策情景：第5期开始央行将失业率压低至3%，持续5期
T = 15
u = np.array([0.05]*4 + [0.03]*5 + [0.05]*6)
pi_e = np.zeros(T)
pi = np.zeros(T)
pi_e[0] = 0.02    # 初始预期通胀 2%

for t in range(T):
    pi[t] = pi_e[t] - alpha * (u[t] - u_n)
    if t + 1 < T:
        pi_e[t+1] = lambda_ * pi[t] + (1 - lambda_) * pi_e[t]

plt.plot(range(T), pi * 100, label='实际通胀率 (%)')
plt.plot(range(T), pi_e * 100, label='预期通胀率 (%)', linestyle='--')
plt.axhline(2, color='gray', linestyle=':', label='初始通胀 2%')
plt.xlabel('时期')
plt.ylabel('通胀率 (%)')
plt.title('适应性预期下的菲利普斯曲线动态')
plt.legend()
plt.show()
```

运行结果显示：在失业率被压低至3%的5期中，实际通胀从2%逐步攀升至约7%，并在失业率回归自然率后仍因通胀预期惯性而缓慢下行，直观展现"预期惯性"如何使短期权衡的代价在时间轴上延迟积累。

## 实际应用

### 货币政策框架中的通胀目标制

理解长期垂直的菲利普斯曲线，是1990年代以来各国央行普遍采用**通胀目标制**（inflation targeting）的理论基础。新西兰储备银行于1990年率先实施，将通胀目标区间定为1%—3%；英格兰银行于1997年获得独立性后，将目标锚定在2%（CPI）。其逻辑正是：长期无法用通胀换取更低失业，故不如将通胀锚定在低位，稳定预期，反而有利于经济平稳运行。

### 曲线平坦化与货币政策的"失业红利"

1990年代以来，大量实证发现SRPC斜率 $\alpha$ 显著降低。美联储研究显示，1990—2019年美国失业率从10%降至3.5%的过程中，核心PCE通胀率仅从约3%上升至约2%，斜率远低于1960—1980年代估计值（Hooper, Mishkin & Sufi, 2020）。可能原因包括：央行可信度提升使预期牢固锚定在2%附近、全球化压低进口商品价格、以及零售业的亚马逊效应（Amazon effect）抑制价格上涨能力。平坦化意味着央行可以在不显著推高通胀的情况下容忍更低的失业率，这在2018—2019年美联储的政策决策中有所体现——当失业率降至3.7%时，通胀依然低于2%目标。

### 新凯恩斯菲利普斯曲线（NKPC）

基于微观基础的**新凯恩斯菲利普斯曲线**（New Keynesian Phillips Curve）由罗伯特·卡尔沃（Guillermo Calvo）1983年的"错开定价"模型推导而来：

$$\pi_t = \beta E_t[\pi_{t+1}] + \kappa \hat{y}_t$$

其中 $\beta \in (0,1)$ 为折现因子，$\hat{y}_t$ 为产出缺口（实际产出偏离潜在产出的百分比），$\kappa > 0$ 取决于价格调整频率。与传统曲线最大区别在于：**预期项是向前看的 $E_t[\pi_{t+1}]$，而非向后看的 $\pi_{t-1}$**，这使通胀同时取决于未来预期而非仅历史惯性，为央行通过"前瞻性指引"管理通胀预期提供了理论依据（Galí & Gertler, 1999）。

## 常见误区

**误区一：菲利普斯曲线在任何时期都成立**。菲利普斯1958年的原始发现仅针对英国1861—1957年的名义工资与失业率，且基于特定的劳动市场制度背景。将其无条件套用于其他国家或其他历史时期（如日本1990年代长期低通胀高失业并存）是方法论错误。
