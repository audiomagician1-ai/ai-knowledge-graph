# 垄断市场

## 概述

垄断市场（Monopoly）是指整个行业中只有一家企业提供某种没有近替代品的商品或服务的市场结构。垄断企业是价格制定者（price maker），而非完全竞争中的价格接受者，因为它面对的是整个市场的向下倾斜需求曲线。这意味着垄断者若要多销售一单位产品，必须对所有购买者降价，这一特征直接决定了垄断企业的边际收益（MR）始终低于价格（P）。

垄断形成的来源主要有四类：资源垄断（如戴比尔斯公司控制全球约85%的钻石原矿供应）、专利和版权保护（美国专利有效期为申请日起20年）、政府特许经营（如烟草专卖局）、以及自然垄断（如自来水管网）。历史上第一批现代反垄断法律是1890年美国颁布的《谢尔曼反托拉斯法》，该法将限制贸易和垄断行为列为联邦犯罪，成为全球反垄断立法的蓝本。

垄断市场的研究之所以重要，是因为它能造成显著的社会福利损失。与完全竞争均衡相比，垄断均衡中产量偏低、价格偏高，导致消费者剩余减少，其中部分转化为垄断利润，另一部分则成为无谓损失（Deadweight Loss），永久从社会福利中消失。量化这一损失是评估反垄断政策效益的核心依据。

## 核心原理

### 垄断均衡：MR = MC 条件

垄断企业的利润最大化条件与完全竞争相同，仍是 **边际收益等于边际成本（MR = MC）**，但结果截然不同。设市场逆需求函数为 P = a - bQ，则总收益 TR = PQ = aQ - bQ²，边际收益 MR = a - 2bQ。可以看出，MR曲线的斜率是需求曲线斜率的**两倍**，MR曲线在需求曲线与价格轴交点的正下方与数量轴相交。

垄断均衡产量 Q* 由 MR = MC 确定，均衡价格 P* 则由将 Q* 代入需求曲线得出，满足 P* > MR = MC。这个"P > MC"的楔形差距正是垄断无效率的根源：消费者愿意按边际成本购买的那些单位被人为压制，形成无谓损失三角形，其面积 = ½ × (P* - MC) × (Q_竞争 - Q*)。

### 价格歧视：三种类型

价格歧视（Price Discrimination）是垄断者按不同价格向不同消费者出售同质商品的策略，前提是企业具有市场势力且能阻止套利。

- **一级价格歧视（完全价格歧视）**：对每位消费者收取其愿意支付的最高价格，垄断者攫取全部消费者剩余，无谓损失为零，产量达到社会最优，但分配结果极端不公平。
- **二级价格歧视（数量折扣）**：按购买数量分段定价，如电力公司前100度电每度0.6元、超出部分每度0.4元，不直接识别消费者类型，而通过菜单合同让消费者自我筛选。
- **三级价格歧视**：将市场分割成可识别的子市场（如学生票与全价票），在各子市场分别令 MR₁ = MR₂ = MC，弹性较低的市场定价更高。其最优定价规则可由**勒纳指数**表达：(P - MC)/P = -1/ε，弹性绝对值越小，加价幅度越大。

### 自然垄断与监管难题

自然垄断发生在某行业具有强烈的规模经济，以至于由单一企业生产整个市场需求比多家企业竞争更有效率的情况。典型特征是固定成本极高（如铺设铁路、电网、自来水管道）而边际成本极低且持续下降，使得平均总成本（ATC）始终高于边际成本（MC）。

这造成监管的两难困境：
- 若监管者要求按 **P = MC** 定价（社会最优），企业因价格低于ATC而长期亏损，需要政府补贴。
- 若允许按 **P = ATC** 定价（公平回报定价），企业可以保本，但产量仍低于社会最优，存在一定无谓损失。
- 若不监管，自然垄断者自选 MR = MC 产量并以高价销售，无谓损失最大。

中国电网公司是典型案例：2015年中国电力体制改革将输配电与发售电业务拆分，对仍具有自然垄断性质的输配电网络实施"准许成本加合理收益"监管，允许约8%左右的资产回报率。

## 实际应用

**制药行业专利垄断**：辉瑞公司的立普妥（阿托伐他汀）在专利保护期内（1996—2011年）长期以每片约3美元的价格出售，而仿制药上市后价格迅速跌至0.1美元以下，这一价差直观展示了专利垄断带来的高价效应。专利到期即构成合法的"市场进入"，彻底瓦解垄断地位。

**软件平台垄断与反垄断**：2020年美国司法部起诉谷歌，指控其通过每年向苹果支付约120亿美元的预装费，维持在搜索引擎市场的垄断地位（约占美国搜索市场90%以上份额）。这是1990年代微软反垄断案之后最重大的科技领域诉讼，法院于2024年裁定谷歌违反《谢尔曼法》第二条。该案揭示了数字经济中"默认设置"和网络效应如何构建现代垄断壁垒。

**一级价格歧视的现实近似**：航空公司动态定价系统通过大数据分析乘客的购买时间、设备类型、历史浏览行为来差异化定价，是对一级价格歧视的技术逼近，使航空公司的消费者剩余提取能力大幅增强。

## 常见误区

**误区一：垄断企业可以随意定任何高价**。这是错误的。垄断企业受市场需求曲线约束——它不能同时自行决定价格和产量，只能在需求曲线上选择一个点。若将价格定得过高，需求量会下降为零。垄断者的真正能力是选择利润最大化点，而非无限制加价。

**误区二：垄断企业一定赚取超额利润**。短期内，若市场需求不足，即使是垄断者也可能亏损（P < ATC）。与完全竞争不同的是，垄断者在长期也不会被迫退出至零利润均衡——它能维持超额利润的前提是市场进入壁垒始终存在。一旦壁垒消失（如专利到期），超额利润也随之消失。

**误区三：价格歧视总是有害的**。从效率角度，一级价格歧视实现帕累托最优产量，消除了无谓损失；三级价格歧视在某些条件下（如向低收入市场扩展服务）也可以提升总产量。价格歧视主要是分配问题而非效率问题——它将消费者剩余转移给生产者，但未必缩小总剩余规模。这正是反垄断政策评估时需要同时考量效率与公平两个维度的原因。

## 知识关联

**与完全竞争市场的对比**：掌握完全竞争（P = MR = MC，长期零利润）是理解垄断的基准起点。垄断的核心差异在于：需求曲线从企业视角由水平变为向下倾斜，导致 MR < P，使得均衡产量偏低、价格偏高，出现无谓损失——这在完全竞争中是不存在的。

**通往寡头市场**：垄断与下一个主题寡头（Oligopoly）的本质区别在于企业数量和策略互动。寡头中多家企业的存在使得每家企业的决策必须考虑竞争对手的反应，引入博弈论工具（如纳什均衡、古诺模型）。勒纳指数在寡头中同样适用，但其数值取决于市场集中度和企业间的协调程度，从垄断（最高）经寡头递减至完全竞争（归零），形成一条清晰的市场结构谱系。
