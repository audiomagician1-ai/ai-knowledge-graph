# 弹性理论

## 概述

弹性理论（Elasticity Theory）衡量一个经济变量对另一个经济变量变动的敏感程度，其核心公式为：**弹性系数 = 因变量变动百分比 ÷ 自变量变动百分比**。这一比率去除了单位的影响，使得不同商品（例如石油与苹果）之间的需求敏感度可以直接比较。

弹性概念由英国经济学家阿尔弗雷德·马歇尔（Alfred Marshall）在1890年出版的《经济学原理》（*Principles of Economics*）中系统化提出。马歇尔借用物理学中"弹性"一词来描述需求量对价格变动的反应幅度，解决了当时经济学家无法量化"需求敏感"这一直觉判断的困境。

弹性理论的实际价值在于直接指导定价决策和税收政策设计。企业需要知道：若将商品价格提高10%，总收入是增加还是减少？政府需要知道：对哪类商品征税不会引起消费量的大幅萎缩？这些问题都要通过弹性数值来回答。

## 核心原理

### 价格需求弹性（Price Elasticity of Demand, PED）

价格需求弹性的计算公式为：

$$E_d = \frac{\Delta Q / Q}{\Delta P / P} = \frac{\Delta Q}{\Delta P} \times \frac{P}{Q}$$

其中 $Q$ 为需求量，$P$ 为价格，$\Delta$ 表示变动量。由于需求曲线向右下方倾斜，价格与需求量反向变动，因此 $E_d$ 通常为**负值**，实际使用时取绝对值进行比较。

根据绝对值大小，弹性分为三类：
- **$|E_d| > 1$（富有弹性）**：价格变动1%，需求量变动超过1%。奢侈品、有替代品的商品通常属于此类。
- **$|E_d| < 1$（缺乏弹性）**：价格变动1%，需求量变动不足1%。生活必需品如食盐、胰岛素属于此类。
- **$|E_d| = 1$（单位弹性）**：价格变动百分比恰好等于需求量变动百分比。

**弹性与总收入的关系**是一个重要推论：当商品富有弹性时，降价会使总收入（$TR = P \times Q$）增加；当商品缺乏弹性时，提价反而使总收入增加。这是因为富有弹性的商品，降价带来的需求量增幅超过价格降幅。

影响价格需求弹性的四大因素包括：替代品的数量与接近程度、该商品在消费者预算中的占比、商品是否为必需品、以及考察的时间跨度（长期弹性通常大于短期）。

### 收入弹性（Income Elasticity of Demand, YED）

收入弹性测量需求量对消费者收入变动的反应：

$$E_y = \frac{\Delta Q / Q}{\Delta Y / Y}$$

其中 $Y$ 为消费者收入。$E_y$ 的符号本身具有经济学含义：
- **$E_y > 0$**：正常品，收入增加则购买量增加。
- **$E_y > 1$**：奢侈品（高档正常品），如珠宝、出境游，收入增加1%，需求量增加超过1%。
- **$0 < E_y < 1$**：必需正常品，如基础食品。
- **$E_y < 0$**：劣等品（Inferior Goods），如方便面、廉价公交，收入增加反而减少购买。

德国统计学家恩斯特·恩格尔（Ernst Engel）在1857年发现，随着家庭收入增加，食物支出占总支出的比例下降——这一规律称为"恩格尔定律"，本质上就是食品收入弹性通常小于1的体现。

### 交叉价格弹性（Cross-Price Elasticity of Demand, XED）

交叉弹性衡量商品A的需求量对商品B价格变动的反应：

$$E_{xy} = \frac{\Delta Q_A / Q_A}{\Delta P_B / P_B}$$

- **$E_{xy} > 0$**：A与B为**替代品**。B价格上涨，消费者转向A，A的需求量增加。例如，猪肉与牛肉：猪肉涨价时，牛肉需求上升。
- **$E_{xy} < 0$**：A与B为**互补品**。B价格上涨，B的需求量下降，连带A的需求量也下降。例如，汽油与汽车：汽油涨价导致购车需求下降。
- **$E_{xy} \approx 0$**：A与B为**独立品**，价格无关联。

交叉弹性的正负符号是判断商品关系的客观标准，避免了仅凭直觉判断替代关系的主观性。

## 实际应用

**定价策略**：航空公司对商务舱和经济舱采用差异化定价，原因是两类乘客的价格弹性截然不同——商务旅客因行程安排由公司决定，弹性较低（约$|E_d| = 0.4$）；休闲旅客有大量替代出行日期可选，弹性较高（约$|E_d| = 1.5$）。

**政府税收设计**：香烟和酒精饮料的需求价格弹性较低（香烟约为$|E_d| = 0.3$ 至 $0.5$），政府对其征收消费税时，需求量不会大幅下降，财政收入有保障，同时也有一定的抑制消费效果。这是"罪恶税"（Sin Tax）政策的弹性理论依据。

**农业政策**：农产品需求价格弹性通常低于1（缺乏弹性）。这意味着丰收年份产量大增会导致农产品价格大幅下跌，总收入反而可能减少——即"谷贱伤农"现象，这正是许多国家对农业进行价格保护和收购托底的经济学原因。

## 常见误区

**误区一：弹性与需求曲线斜率是同一回事。**
弹性和斜率都描述价格与需求量的关系，但并不相同。一条直线需求曲线的斜率恒定，但沿曲线不同点的弹性值是变化的——曲线高端（价格高、数量少）弹性较大，低端（价格低、数量多）弹性较小。只有在双曲线型需求曲线上，各点弹性才恒等于1。

**误区二：缺乏弹性的商品对价格完全没有反应。**
缺乏弹性（$|E_d| < 1$）意味着需求量变动的**百分比**小于价格变动的百分比，而非需求量完全不变。完全无弹性（$|E_d| = 0$，需求曲线垂直）在现实中极为罕见，即使是胰岛素依赖型糖尿病患者，在价格极高时也可能减少剂量或寻找替代治疗方案。

**误区三：替代品的交叉弹性越大越好。**
从企业角度看，与竞争对手产品的交叉弹性越高，意味着竞争对手降价对自身的威胁越大，市场份额越脆弱。企业通过品牌建设和差异化竞争，实质上是在降低自身产品与竞争产品之间的交叉弹性。

## 知识关联

弹性理论直接建立在**供给与需求**曲线的基础上：需求曲线的形状（陡峭或平缓）本身就反映了弹性的大小，但弹性将这种"形状"转化为可计算的数值。供给也有对应的价格供给弹性（$E_s = \frac{\Delta Q_s / Q_s}{\Delta P / P}$），原理与需求弹性完全对称。

在后续学习**市场结构与垄断定价**时，弹性是垄断企业制定最优价格的核心工具——勒纳指数（Lerner Index）$L = (P - MC)/P = -1/E_d$ 直接将垄断势力与价格弹性联系起来。在学习**税收归宿（Tax Incidence）**时，税收由生产者还是消费者承担，取决于供给弹性与需求弹性的相对大小：弹性越小的一方承担税负越多。
