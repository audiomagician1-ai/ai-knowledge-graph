# 前景理论

## 概述

前景理论（Prospect Theory）由心理学家丹尼尔·卡尼曼（Daniel Kahneman）和阿莫斯·特沃斯基（Amos Tversky）于1979年在《计量经济学》（*Econometrica*）杂志发表的同名论文中正式提出。该理论通过大量受控实验，系统描述了真实人类在面对风险与不确定性时的决策规律，直接挑战了冯·诺伊曼与摩根斯坦1944年建立的预期效用理论（Expected Utility Theory）框架。

前景理论的核心贡献在于提出了三个相互关联的行为现象：**参考点依赖**（reference dependence）、**损失厌恶**（loss aversion）和**概率权重扭曲**（probability weighting distortion）。与预期效用理论用财富的绝对水平衡量效用不同，前景理论认为人们以相对于某个参考点的**得失**来评价结果。卡尼曼因这一研究于2002年获得诺贝尔经济学奖，特沃斯基因早逝未能同享这一荣誉。

前景理论之所以重要，在于它能解释大量预期效用理论无法解释的实际现象，例如同一个人同时购买保险和彩票、股票投资者持亏售盈等看似矛盾的行为。它为行为经济学提供了第一个可以进行定量预测的替代性决策理论框架。

## 核心原理

### 一、参考点依赖与价值函数

前景理论用**价值函数 v(x)** 替代了传统的效用函数 u(W)。价值函数的自变量 x 不是财富绝对量，而是**相对于参考点的变化量**（得或失）。参考点通常是决策者当前的财富水平、期望水平或某个社会比较标准，并非固定不变。

价值函数呈现**S形曲线**：在收益域（x > 0）向上且凹（concave），在损失域（x < 0）向下且凸（convex），且整体在参考点处不可微。这意味着人们对收益的边际满足感递减，对损失的边际痛苦也递减——获得第一个100元比第二个100元更令人高兴，损失第一个100元比再多损失100元更令人痛苦。

### 二、损失厌恶系数

价值函数最关键的非对称性体现在损失域的斜率远大于收益域。卡尼曼和特沃斯基原始研究估计，**损失的心理权重约为同等数量收益的2至2.5倍**，即损失厌恶系数 λ ≈ 2.25。数学上可简洁表示为：

> v(x) = x^α，若 x ≥ 0
> v(x) = −λ(−x)^β，若 x < 0

其中 α 和 β 通常估计约为 0.88，λ ≈ 2.25。这意味着损失50元带来的痛苦，需要赢得约112元才能补偿。这一非对称性直接解释了"确定性效应"（certainty effect）：人们为了将损失可能性从小概率降至零，愿意支付超出期望值的溢价。

### 三、概率权重函数

前景理论还引入了**概率权重函数 π(p)**，指出人们并不线性地处理概率，而是**高估小概率、低估中高概率**。权重函数在接近0和1时斜率极陡，在中间区域相对平坦，呈现倒S形。具体表现为：人们对1%概率的彩票过度兴奋（高估），同时对95%与99%概率的区别几乎不敏感（低估高概率差异）。卡尼曼与特沃斯基在1992年"累积前景理论"修订版中，给出了参数 γ ≈ 0.65 的权重函数公式，以修正早期版本在处理多结果情境时的随机优势违反问题。

### 四、编辑阶段与评估阶段

前景理论将决策过程分为两个阶段：首先是**编辑阶段**，决策者对"前景"（即带概率的结果集合）进行编码、合并与简化，确定参考点并将结果标记为得或失；然后是**评估阶段**，对编辑后的前景套用价值函数和概率权重函数计算加权价值之和，选取最大值对应的方案。这一两阶段结构解释了为何问题的表述方式（framing）会改变人们的实际选择。

## 实际应用

**股票市场中的"处置效应"：** 投资者倾向于过早卖出盈利股票（锁定收益，因收益域价值函数凹）、长期持有亏损股票（拒绝实现损失，因损失厌恶）。Shefrin和Statman于1985年将这一现象命名为处置效应（disposition effect），并将其直接归因于前景理论的损失厌恶机制。

**保险与赌博的共存：** 同一消费者既购买小额损失保险（损失域的风险厌恶行为），又购买彩票（收益域在小概率高收益下的风险寻求行为）。预期效用理论无法在同一效用函数下解释这种组合行为，而前景理论的S形价值函数与概率权重函数共同提供了一致性解释。

**劳动供给的"目标收入效应"：** 纽约出租车司机研究（Camerer等，1997年）发现，司机在生意好的日子反而更早收车，因为他们以每日目标收入为参考点，一旦达标即产生"收益已足够"的凹形满足感，宁愿牺牲额外收入换取休闲。

**谈判与定价：** 消费者评价一款原价500元、现售400元的商品与原价400元的商品时，前者因提供100元"节省收益"而被感知价值更高，尽管实际支出相同。企业定价策略中常利用参考点操纵这一点设计促销方案。

## 常见误区

**误区一：损失厌恶等于风险厌恶。** 两者是不同概念。风险厌恶（risk aversion）源于效用函数的凹性，意味着人们偏好确定结果而非同等期望值的赌局，在收益域和损失域均可出现。损失厌恶特指损失与等量收益之间的心理权重不对称（λ > 1），是价值函数在参考点处的非对称性，而非曲率特征。在损失域，前景理论实际上预测的是**风险寻求**（凸函数），而非风险厌恶。

**误区二：参考点永远是当前财富水平。** 卡尼曼和特沃斯基明确指出参考点由情境决定，可以是期望所得、社会规范、历史价格或竞争对手表现。例如，得到年终奖8000元的员工，若同事普遍获得10000元，则以10000元为参考点，感知为损失2000元而非收益8000元。参考点的可变性是前景理论在实际应用中最需要精确界定的变量。

**误区三：前景理论是对所有决策的通用描述。** 该理论主要适用于存在明确概率信息的风险决策情境。在模糊性（ambiguity）决策、跨期选择或社会偏好场景中，前景理论需要与其他行为模型结合使用。此外，1979年原版前景理论对于涉及超过两个结果的前景存在随机优势违反问题，这一缺陷直到1992年**累积前景理论**（Cumulative Prospect Theory）才通过对概率进行排序累积加权的方式得到修正。

## 知识关联

**前置概念：** 学习前景理论需要先了解**预期效用理论**的基本框架——包括彩票的定义、期望值计算和效用函数的凹凸性含义——因为前景理论的每一个核心命题都是对预期效用理论具体假设的否定。行为经济学概述中关于"有限理性"的讨论为理解为何需要前景理论提供了动机。

**后续拓展：** 前景理论直接衍生出**框架效应**的研究——因为参考点和编辑方式可被外部问题表述改变，同一问题的不同框架导致不同选择；**禀赋效应**可理解为持有物品成为参考点后，出售被编码为"损失"，导致要价远高于买价；**心理账户**理论（Thaler提出）则将前景理论的价值函数应用于人们如何将资金划分到不同心理"账户"中分别核算得失。在**行为金融学**中，前景理论是解释股权溢价之谜、动量效应和过度交易等异象的基础理论工具之一。
