# 单调栈

## 核心概念

单调栈是一种特殊的栈数据结构，其中的元素始终保持单调递增或单调递减的顺序。通过维护这种单调性，可以在O(n)时间内解决一类"下一个更大/更小元素"问题。

## 两种类型

### 单调递增栈
栈中元素从底到顶递增。遇到更小的元素时弹出：
- 用于寻找"下一个更小元素"

### 单调递减栈
栈中元素从底到顶递减。遇到更大的元素时弹出：
- 用于寻找"下一个更大元素"

## 经典问题：下一个更大元素

给定数组，对每个元素找出右边第一个比它大的元素。

```python
# Python实现
def next_greater(nums):
    n = len(nums)
    result = [-1] * n
    stack = []  # 存储下标，维护单调递减
    
    for i in range(n):
        # 当前元素比栈顶大 → 栈顶找到了它的"下一个更大元素"
        while stack and nums[i] > nums[stack[-1]]:
            idx = stack.pop()
            result[idx] = nums[i]
        stack.append(i)
    
    return result

# 示例: [2,1,4,3] → [4,4,-1,-1]
```

⚠️ 注意：此为Python实现，其他语言（如Java、C++）的栈操作API不同但逻辑一致。

## 关键思想

每个元素最多入栈一次、出栈一次，因此总时间复杂度为O(n)，而非O(n²)的暴力枚举。

## 典型应用

1. **每日温度**: 找到下一个更暖天数的距离
2. **柱状图最大矩形**: 找左右两侧第一个更小柱子
3. **接雨水**: 结合单调栈计算凹槽积水
4. **股票价格跨度**: 连续天数中价格不高于当天的天数

## 解题模板

```
1. 初始化空栈和结果数组
2. 遍历数组元素(正向或反向):
   a. while 栈非空 且 当前元素破坏单调性:
      - 弹出栈顶
      - 用当前元素更新弹出元素的答案
   b. 将当前元素入栈
3. 栈中剩余元素无匹配
```

## 与栈的关系

单调栈是栈的一种应用模式。理解基本栈操作（push、pop、peek）是使用单调栈的前提。单调栈通过维护额外的单调性约束，将某些O(n²)问题优化到O(n)。
