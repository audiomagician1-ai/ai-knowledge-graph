# 费茨定律

## 概述

费茨定律（Fitts' Law）由美国心理学家保罗·费茨（Paul Fitts）于1954年在其论文《人类运动系统信息处理能力研究》中提出，最初用于研究飞行员操控仪表的运动精度。该定律的核心主张是：移动到目标所需的时间，由目标的大小和当前位置到目标的距离共同决定——目标越小、距离越远，操作所需时间越长。

费茨定律的数学表达式为：**T = a + b × log₂(2D/W)**，其中 T 是完成移动所需的时间，D 是起点到目标中心的距离，W 是目标沿运动方向的宽度，a 和 b 是由设备和用户决定的常数。括号内的 log₂(2D/W) 被称为"难度指数"（Index of Difficulty，ID），单位为比特（bits）。难度指数越高，意味着用户需要花费更多时间和精力才能准确点击目标。

在交互设计领域，费茨定律是按钮尺寸、菜单布局和触控目标设计的理论基础。苹果公司在制定 iOS 人机界面规范时明确规定，触控目标最小尺寸为 44×44 点，这一数值正是基于费茨定律对手指触控精度的分析得出的。

## 核心原理

### 距离与尺寸的对数关系

费茨定律揭示的是**对数关系**，而非线性关系。这意味着当目标距离从 100px 增加到 200px 时，所需时间的增加量，远小于从 10px 增加到 20px 时的相对影响。在实际设计中，将目标尺寸从 20px 扩大到 40px（翻倍），比将目标从 40px 扩大到 80px 带来的效率提升更为显著。这一对数特性提示设计师：对极小的目标进行放大，收益最大。

### 屏幕边缘的"无限尺寸"效应

操作系统中的菜单栏、任务栏通常被放置在屏幕边缘，这并非出于美观，而是费茨定律的直接应用。当目标位于屏幕边缘时，用户无需精确控制光标停止的位置——光标会被屏幕边界自然阻挡，等效于目标的宽度 W 趋近于无穷大，从而使难度指数 ID 趋近于 0。macOS 的顶部菜单栏和 Windows 的底部任务栏，都利用了这一原理。屏幕角落则同时享有两条边界的约束，是难度指数最低的位置，因此 macOS 的 Apple 菜单始终位于左上角。

### 难度指数与输入设备的关系

不同输入设备对应的参数 a 和 b 数值差异显著。鼠标的 b 值约为 0.1 秒/比特，而触控屏的 b 值通常更低（约 0.07 秒/比特），说明手指触控在低精度任务上更快，但在高难度（小目标、远距离）任务上表现更差。这解释了为什么移动端设计规范要求更大的触控目标：手指缺乏鼠标光标那样的像素级精确度，W 的容错空间必须更宽。触控笔（Stylus）的 b 值则接近鼠标，因此平板绘图软件可以采用更小的 UI 元素。

## 实际应用

**按钮与点击目标设计**：在网页和 App 的按钮设计中，增大点击区域是降低 ID 的最直接方式。即便按钮视觉尺寸较小，也可以通过扩大其不可见的热区（Hit Area）来改善可操作性。例如，一个图标按钮在视觉上可能只有 24px，但其热区应被扩展至至少 44px，以满足费茨定律对移动端触控的要求。

**下拉菜单与右键菜单的布局**：鼠标右键菜单通常在光标当前位置弹出，这样 D≈0，直接消除了距离变量的影响。饼状菜单（Pie Menu）将菜单选项围绕光标呈圆形分布，每个选项到光标的距离相等且极短，是费茨定律驱动的菜单布局创新，实验数据显示其选择速度比线性菜单快约 15%–20%。

**表单字段的标签点击区域**：在表单设计中，将 `<label>` 标签与 `<input>` 字段关联，可以使整个标签文字区域都成为激活该输入框的有效热区，而不仅仅是那个小小的复选框本身，这是费茨定律在表单交互中的直接应用。

## 常见误区

**误区一：目标越大越好，没有上限**。费茨定律表明扩大目标尺寸可以降低 ID，但当 W 超过一定阈值后（通常认为 W > D/2 时），继续扩大带来的时间收益趋近于零。更重要的是，过大的触控目标会占用屏幕空间，导致其他目标被迫缩小或距离变远，反而提高了整体界面的平均 ID。设计中存在一个目标尺寸与布局密度之间的帕累托最优点。

**误区二：费茨定律同等适用于一维移动和二维移动**。原始费茨定律描述的是沿一条直线的一维运动。Accot 和 Zhai 在1997年提出的"转向定律"（Steering Law）扩展了费茨定律，专门处理沿曲线路径移动的情形，例如穿越级联子菜单时的光标轨迹。将费茨定律直接套用在需要沿特定路径操作的场景（如导航菜单的子菜单展开），会低估实际操作难度。

**误区三：费茨定律只适用于桌面端**。部分设计师认为该定律主要针对鼠标操作，实际上已有大量研究将其扩展至触控屏、眼动追踪（Gaze-based Input）和语音交互。在眼动追踪输入中，目标的角尺寸（以视角度数衡量）替代了像素宽度 W，但对数关系依然成立。

## 知识关联

费茨定律的学习不依赖其他交互设计概念的前置知识，它是理解目标尺寸、间距和布局决策的基础工具。在掌握费茨定律之后，**表单设计**是自然的下一步应用场景：表单中包含大量输入字段、复选框、单选按钮和提交按钮，每一个元素的尺寸设定和排列间距都可以用费茨定律进行分析和优化。例如，为什么表单中的单选框热区应覆盖整行标签文字、为什么"提交"按钮应显著大于"取消"按钮，都有费茨定律的量化支撑。费茨定律还与菲特斯-霍尔定律（Hick's Law，选项数量与决策时间的关系）并列，共同构成交互设计中对用户操作效率进行定量分析的两大经典工具。
