# 品类核心循环

## 概述

品类核心循环是指不同游戏类型（如动作、RPG、策略、放置类等）在反复驱动玩家行为时所形成的特定循环结构。每一种游戏品类由于其核心玩法机制不同，玩家所反复执行的操作序列也截然不同，这种差异从根本上决定了各品类的游戏体验。识别一款游戏属于哪种品类，本质上就是识别其核心循环的基本形态。

"品类核心循环"这一分析框架在2000年代中后期伴随着休闲游戏和移动游戏市场的爆发而被业界广泛讨论。当《愤怒的小鸟》（2009年）与《植物大战僵尸》（2009年）先后证明不同品类循环可以服务于完全不同的受众群体时，游戏设计师开始系统整理各品类循环的共性模板。理解品类核心循环的意义在于：设计师可以借助已有品类的循环模板快速建立游戏原型，也可以通过有意混合两种品类的循环结构来创造全新的玩法体验。

## 核心原理

### 品类循环的三要素构成

任何品类的核心循环都由**输入动作、即时反馈、进展奖励**三个环节构成，但三者在不同品类中的权重分配完全不同。以射击游戏为例，输入动作（瞄准、射击）与即时反馈（命中判定、击杀音效）几乎同步发生，进展奖励（解锁武器、提升等级）则是次要的延迟层。而在放置类游戏中，输入动作极度弱化，循环几乎完全由数值增长驱动的进展奖励构成。这种权重分配的差异可以用"操作密度"来量化：动作游戏的操作密度可达每分钟数百次有效输入，放置游戏则可能低至每天数次。

### 主要品类的典型循环结构

**动作游戏循环**的基本序列为：感知威胁 → 执行操作（攻击/闪避）→ 获得即时反馈 → 进入下一威胁。这个循环的单次周期通常在3至10秒内完成，玩家在数分钟内可以经历数十次完整循环。《黑魂》系列将这一循环压缩到极致，使得单次攻击的出手与收招帧数成为核心体验的焦点。

**RPG游戏循环**的基本序列为：接取任务/目标 → 探索世界/战斗 → 获取数值成长（经验值、装备） → 解锁新内容/区域 → 接取更高难度目标。这个循环中"数值成长"环节是区别于其他品类的核心特征，玩家在每次循环后都能感受到角色能力的具体提升，这种提升通常以"+5%攻击力"或"解锁新技能"的形式量化呈现。

**策略游戏循环**（以4X策略为例）的基本序列为：收集资源 → 建造/研究 → 扩张领土/外交 → 应对竞争压力 → 调整策略。这个循环的单次周期远长于动作类，可能跨越游戏内的数十回合，玩家的决策质量而非操作速度决定了循环的结果。

**放置游戏循环**的基本序列为：登录 → 收取离线收益 → 分配资源升级 → 退出等待。这个循环最大的特点是"离线进展"机制，玩家不在线时循环依然在推进，真实游玩时间可能只占总循环时长的不足5%。

### 品类循环的转化效率指标

衡量品类核心循环设计质量的关键指标是**循环完整率**，即玩家开始一次循环后能够完成并进入下一次循环的比率。不同品类有各自的行业基准：休闲手游第一次循环完整率通常需要超过70%才能维持健康留存；而MMORPG由于循环周期更长，单次循环完整率的基准相对宽松，但7日留存率是更关键的参考指标，行业中位值约为15%至25%。

## 实际应用

**《Candy Crush》（2012年）**是品类循环分析的经典案例：其消除游戏循环为"选择目标 → 滑动消除 → 观看连锁动画 → 获得星级评分"，单次关卡循环约为1至3分钟，挫败后的"再试一次"提示设计将循环断裂成本降到最低，使玩家日均游戏次数超过5次。

**《文明》系列**的策略循环设计体现了"一回合综合征"（One More Turn Syndrome）的品类特性：每次循环结束时游戏总会给出一个"下一个目标即将完成"的信号（城市再过3回合建完、科技再过5回合研究完），驱动玩家立即进入下一个循环，这是策略品类循环特有的时间延伸机制。

**《原神》**的循环设计融合了动作品类与RPG品类：战斗循环（感知敌人→元素反应操作→击败）是动作层，而角色养成循环（刷材料→提升等级→开放新战斗内容）是RPG层，两套循环的嵌套使其同时覆盖了偏好操作体验和偏好成长体验的两类受众。

## 常见误区

**误区一：将"游戏时长"等同于"循环深度"**。许多设计师认为循环周期越长，游戏越有深度。实际上，放置游戏的单次循环可以跨越数天，但其循环结构极为简单；而《格斗天王》的单次对战循环不超过2分钟，但其操作与策略深度远超大多数长周期循环游戏。循环的品质取决于决策密度，而非时间长度。

**误区二：跨品类混合循环必然产生"1+1>2"的效果**。将两种品类的循环结构叠加并不必然带来更好的体验，叠加不当会导致循环信号冲突。例如，在一款强调即时反应的动作游戏中强行加入重度放置类的数值成长循环，可能使玩家对"该提升操作技巧还是该去刷数值"产生目标混乱，这种现象在2015至2018年间的众多"动作+放置"混合手游中普遍出现，是导致这批游戏快速流失的重要原因之一。

**误区三：品类循环模板可以照搬**。某个品类的成功循环结构背后有特定的受众心理预期支撑。"战斗→掉落装备→强化角色"这一RPG循环之所以有效，是因为玩家进入游戏前已接受了"数值成长是合理目标"的品类契约。将同样结构嫁接到解谜游戏中，玩家会因为品类预期不符而产生困惑，循环驱动力将大幅下降。

## 知识关联

在学习品类核心循环之前，需要掌握**核心循环概述**的基础定义，即核心循环是驱动玩家持续游玩的最小重复行为单元，品类核心循环是在此基础上研究不同游戏类型如何实例化这一结构。

品类核心循环是后续专项研究的分类框架。**动作游戏循环**将重点讨论帧级反馈设计与打击感构建；**RPG游戏循环**将深入分析数值成长曲线与任务引导机制；**策略游戏循环**将展开讨论信息不完整条件下的决策循环；**放置游戏循环**将专门解析挂机收益模型与登录频率设计；**手游F2P循环**则将品类循环与变现机制结合，分析付费节点如何嵌入循环中的特定卡点。这五个方向都以品类核心循环中建立的分类标准和三要素模型作为分析起点。
