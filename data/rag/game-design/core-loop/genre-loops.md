# 品类核心循环

## 概述

品类核心循环是指不同游戏类型（如动作、RPG、策略、放置等）在设计层面上所形成的、具有该品类辨识度的重复性玩法结构。同一品类的游戏之所以让玩家产生"这是同一类游戏"的直觉认知，正是因为它们共享相似的核心循环骨架——例如所有"Roguelike"游戏都遵循"进入关卡 → 收集随机强化 → 死亡/通关 → 解锁永久内容 → 重新进入"这一结构。

品类核心循环的系统性研究兴起于2010年前后移动游戏爆发期。随着App Store和Google Play的竞争加剧，游戏设计师意识到必须在极短时间内向玩家传递"品类信号"——玩家在下载一款塔防游戏时，已经预期会遇到"放置防御单位 → 敌人沿路径行进 → 击退敌人获得资源 → 升级防御"的循环结构。打破或偏离这一预期需要极高的设计成本。

研究品类核心循环的意义在于它为游戏设计提供了一套可校验的参照系。当一款新游戏的核心循环偏离品类标准超过30%（以设计决策节点计），玩家的认知摩擦会显著上升，导致新手留存率（Day-1 Retention）下降——这一现象在2015至2020年间的多份移动游戏市场报告中被反复验证。

## 核心原理

### 品类循环的三层结构共性

无论哪个品类，其核心循环都可以拆解为**动作层、反馈层、成长层**三个嵌套环路。动作层是玩家每隔数秒执行一次的最小操作单元（如FPS中的"瞄准-射击"、消除游戏中的"滑动-匹配"）；反馈层是每隔数分钟给予玩家的结果评估（关卡胜负、分数统计）；成长层是每隔数小时或数天推动角色、装备或解锁内容变化的宏观进度。品类差异主要体现在这三层的**时长比例**和**决策密度**上，而非是否存在这三层。

### 品类信号与玩家心理契约

每个品类的核心循环都承载着一种玩家心理契约：玩家选择该品类意味着他们愿意接受特定类型的挑战与奖励节奏。MOBA类游戏的循环周期固定在20至50分钟一局，每局内部包含"发育期 → 团战期 → 收割期"三段式节奏，这一结构已成为MOBA品类的合同条款——《英雄联盟》2009年推出时严格遵守了这一结构，而任何试图缩短至10分钟一局的早期设计原型均在内测中遭到核心玩家的强烈抵制。卡牌构筑类游戏的心理契约则是"随机性在构筑阶段，策略性在对战阶段"，循环公式为：**对战(测试牌组) → 获得新牌(扩展选项) → 修改牌组(优化策略) → 再次对战**。

### 品类循环的分叉点：资源流向

不同品类循环之间最关键的技术差异在于资源如何在循环中流动。RPG品类的资源（经验值、装备）具有**单向积累性**，玩家的努力不会消失，构成"永远向前"的心理安全感；而竞技类品类（如RTS）的资源在每局结束后**完全清零**，循环的吸引力来自技术本身而非资产积累。放置类游戏则采用**离线增殖**机制，将玩家离线时间转化为资源，其循环公式可表达为：**签到(领取离线收益N) → 分配资源(升级单元) → 设置离线产出配置 → 离开游戏**，其中N的计算方式直接决定了玩家的回访意愿。

## 实际应用

以"三消 + 关卡"品类（以《糖果传奇》为代表）为例，其核心循环由如下节点构成：**选择关卡 → 5步以内感受难度 → 30步内完成或失败 → 获得星级评分与金币 → 购买/等待生命值 → 进入下一关卡**。整个循环的心跳周期约为2至5分钟，玩家每天平均启动游戏4至6次，每次停留时间15分钟左右。这一结构之所以成为该品类模板，是因为它将"即时满足（消除反馈）"与"长期目标（关卡进度条）"以最低摩擦的方式绑定在同一循环内。

生存类游戏（如《饥荒》）的品类核心循环则呈现出完全不同的形态：**探索环境(收集资源) → 建造庇护所(消耗资源) → 应对季节威胁(测试准备) → 扩展基地或死亡(进入新循环)**。这里的循环没有固定时长，玩家自行决定何时推进，品类吸引力来自**资源管理的压力感**与**世界探索的不确定性**之间的张力。

## 常见误区

**误区一：认为优秀游戏必须创造全新品类循环**。实际上，绝大多数商业成功的游戏（《原神》、《明日方舟》、《植物大战僵尸》）的核心循环都高度贴合既有品类模板，创新主要发生在表现层（美术、叙事）或两个已知品类循环的**拼接方式**上，而非循环结构本身的彻底颠覆。《原神》将开放世界RPG循环与手游F2P抽卡循环拼接，两个独立循环各自保持品类标准形态，交叉点设计在角色收集节点上。

**误区二：将"循环时长短"等同于"循环质量高"**。不同品类对循环时长有截然不同的需求：硬核策略游戏（如《文明》系列）的单次循环可长达数小时甚至数天，其品类契约本就是"长周期决策满足感"；若将其压缩为5分钟一局，反而会破坏品类循环的完整性。循环时长的优化必须在**品类预期范围内**进行，而非跨品类横向比较。

**误区三：忽视子品类之间的循环差异**。"RPG"作为大品类下存在ARPG、JRPG、MMORPG等子品类，其循环结构差异极大。JRPG的战斗循环高度依赖"回合制指令选择"带来的策略感，典型循环节约在8至15回合内完成一场战斗；ARPG则将战斗循环压缩为实时操作，玩家决策窗口在0.2至0.5秒量级。混用这两种循环逻辑进行设计会导致目标玩家群体的严重撕裂。

## 知识关联

学习品类核心循环需要以**核心循环概述**中的基础概念（循环节点、奖励触发机制、玩家动机模型）为前提，理解"循环"本身的构成要件后，才能有效分辨不同品类在节点选择上的设计取舍。

进入该话题的后续学习路径是各品类的专项分析：**动作游戏循环**聚焦操作反馈与手感调校；**RPG游戏循环**深入角色成长曲线与数值设计；**策略游戏循环**研究信息不对称与决策树；**放置游戏循环**专注离线机制与回访设计；**手游F2P循环**分析付费节点在循环中的嵌入位置。品类核心循环提供了这五个专项方向的对比坐标系，使得学习者在研究任意单一品类时能够意识到哪些特征是品类独有的，哪些是所有游戏循环的共性规律。
