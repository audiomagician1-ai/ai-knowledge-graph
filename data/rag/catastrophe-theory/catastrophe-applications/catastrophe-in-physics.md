# 物理学中的突变

## 概述

突变论（Catastrophe Theory）由法国数学家勒内·托姆（René Thom）在其1972年出版的《结构稳定性与形态发生》（*Structural Stability and Morphogenesis*）中系统建立，而将这一理论的触角延伸至物理学各分支领域的关键人物是英国物理学家迈克尔·贝里（Michael Berry）及其合作者。贝里从1970年代中期起，系统研究了光学焦散线（caustics）与尖点突变、燕尾突变等之间的深刻对应，使突变论成为描述波动光学奇点的精确数学语言。物理学中的突变现象涵盖三个核心领域：**相变**（phase transitions）、**光学焦散线**（optical caustics）和**弹性屈曲**（elastic buckling），它们分别对应热力学势函数的非凸性、波前的几何奇点以及弹性系统势能面的多值性。这三类现象在数学结构上均可归约为托姆分类定理所列举的七种初等突变之一，这一事实揭示了自然界中不连续跃迁的普遍数学根源。

## 核心原理

### 突变论的数学框架在物理系统中的嵌入

物理学中的突变分析依赖于一个核心假设：系统的演化由一个光滑势函数 $V(x; \mathbf{c})$ 支配，其中 $x$ 是状态变量（内部变量），$\mathbf{c} = (c_1, c_2, \ldots, c_k)$ 是控制参数空间中的点。系统的平衡态满足：

$$\frac{\partial V}{\partial x} = 0$$

突变发生于该方程的解集出现分叉之处，即同时满足：

$$\frac{\partial V}{\partial x} = 0 \quad \text{且} \quad \frac{\partial^2 V}{\partial x^2} = 0$$

上述两个条件联立定义了控制参数空间中的**分歧集**（bifurcation set）或**突变集**（catastrophe set）。当控制参数轨迹穿越该集合时，系统状态发生不连续跳跃——即物理意义上的突变。

根据托姆分类定理（Thom, 1972），当状态变量数 $n \leq 2$ 且控制参数数 $k \leq 4$ 时，所有结构稳定的突变只有七种初等形式。物理学中最频繁出现的是折叠突变（fold catastrophe，余维数1）和尖点突变（cusp catastrophe，余维数2），其标准型势函数分别为：

$$V_{\text{fold}}(x; a) = \frac{x^3}{3} + ax$$

$$V_{\text{cusp}}(x; a, b) = \frac{x^4}{4} + \frac{ax^2}{2} + bx$$

物理量对应的具体识别是应用突变论的关键：必须确认哪个物理量扮演状态变量的角色，哪些参数构成控制空间。

### 相变与尖点突变的对应

液-气相变是突变论在热力学中最经典的应用案例。范德瓦尔斯（van der Waals）方程给出的状态方程在临界点附近展开后，其吉布斯自由能 $G(V; T, P)$ 相对于摩尔体积 $V$ 的结构正好对应尖点突变的标准势函数。

具体地，若以 $v = V - V_c$ 为状态变量（$V_c$ 为临界摩尔体积），以温度偏差 $\tau = T - T_c$ 和压力偏差 $p = P - P_c$ 为控制参数，则吉布斯自由能在临界点附近展开为：

$$G(v; \tau, p) \approx G_0 + A\tau v^2 + Bv^4 - pv + \mathcal{O}(v^5)$$

其中 $A$、$B$ 为正常数。令 $a = A\tau$，该式与尖点突变势函数 $V_{\text{cusp}}(v; a, -p)$ 在结构上完全等同。从控制参数空间 $(\tau, p)$ 的视角来看，液-气共存线正是尖点突变分歧集的一支，而临界点 $(\tau=0, p=0)$ 则精确对应于尖点突变的**尖点**（cusp point）本身。这一对应由扎曼（E. C. Zeeman, 1976）等人系统阐述，揭示了为什么液-气相变的共存线在临界点处终止——这在几何上是尖点处两支分歧线汇合的必然结果（Poston & Stewart, 1978）。

铁磁-顺磁相变（居里点转变）同样具有突变论的表述。朗道（Landau）二级相变理论中，自由能 $F(M; T, H)$（$M$ 为磁化强度，$H$ 为外场）在居里温度 $T_c$ 附近展开为：

$$F(M; \tau, H) = F_0 + \alpha\tau M^2 + \beta M^4 - HM, \quad \tau = T - T_c$$

这在数学上与尖点突变势完全同构：当 $H=0$ 时，$\tau < 0$ 对应双稳态（铁磁相），$\tau > 0$ 对应单稳态（顺磁相），相变边界 $(\tau < 0, H = 0)$ 构成分歧集的两支，其交点 $(\tau=0, H=0)$ 为尖点。

### 光学焦散线与波动突变

光学焦散线是光线几何在焦点附近汇聚时形成的亮度奇点，彩虹（rainbow）、海底波纹的光斑图案、游泳池底的光花纹均属此类。贝里与纽约州立大学的阿普拉尼斯（Upstill）在1980年发表于《光学进展》（*Progress in Optics*）的综述文章中证明：波前的奇点类型与托姆突变分类之间存在一一对应关系——**折叠突变对应普通焦散线，尖点突变对应焦散线的尖点（cusp caustic），燕尾突变（swallowtail）对应更高阶的焦散汇聚**。

彩虹的突变论分析尤为精彩。考虑平行光束射入圆形水滴后经内反射射出的偏转角 $D(\theta_i)$（$\theta_i$ 为入射角）：

$$D(\theta_i) = \pi + 2\theta_i - 4\arcsin\!\left(\frac{\sin\theta_i}{n}\right)$$

其中 $n$ 为水的折射率（$n \approx 1.332$）。彩虹角对应 $dD/d\theta_i = 0$ 的极值点，即折叠突变点。光强在该角度处积分发散（几何光学意义下），形成观测到的亮弧。贝里（Berry, 1969, *Proceedings of the Physical Society*）用艾里函数（Airy function）$\text{Ai}(t)$ 精确描述彩虹附近的光强分布：

$$I(\theta) \propto \left[\text{Ai}\!\left(-\frac{\theta - \theta_{\text{rainbow}}}{w}\right)\right]^2$$

其中 $w$ 是与波长相关的特征角宽度（约为 $0.5°$ 量级）。艾里函数本身是折叠突变相积分的精确解，这一联系正是突变论统一几何光学奇点的核心数学机制。

### 弹性屈曲与折叠、尖点突变

弹性屈曲是结构力学中突变论应用最直接的领域。以欧拉柱（Euler column）为例：长度为 $L$、弯曲刚度为 $EI$ 的细长杆受轴向压力 $P$ 时，其挠度 $w(x)$ 的势能为：

$$V[w] = \int_0^L \left[\frac{EI}{2}\left(\frac{d^2w}{dx^2}\right)^2 - \frac{P}{2}\left(\frac{dw}{dx}\right)^2\right] dx$$

对第一模态 $w(x) = q\sin(\pi x/L)$ 投影，势能化为单变量形式：

$$V(q; P) = \frac{EI\pi^4}{4L^3}q^2 - \frac{P\pi^2}{4L}q^2 + \frac{EI\pi^4}{8L^5}q^4 + \cdots = \frac{1}{2}(P_{cr} - P)\frac{\pi^2}{2L}q^2 + Cq^4$$

其中欧拉临界载荷 $P_{cr} = \pi^2 EI / L^2$。当 $P < P_{cr}$ 时，$q=0$（直立）为唯一稳定平衡；当 $P > P_{cr}$ 时，$q=0$ 变为不稳定，出现两个对称的弯曲平衡态 $q = \pm q_0$。这是一个**叉形分歧**（pitchfork bifurcation），在对称扰动 $F$（横向力）存在时，势函数变为：

$$V(q; a, F) = \frac{a}{2}q^2 + Cq^4 - Fq, \quad a \propto (P_{cr} - P)$$

这正是尖点突变的标准型。突变集（分歧集）给出了 $(a, F)$ 空间中屈曲跳跃的临界条件：当 $4a^3 + 27C^2F^2 = 0$ 时（对应尖点突变判别式），结构发生**突跳**（snap-through）——即弹性跳跃现象。

扎曼（Zeeman, 1977）在《科学美国人》（*Scientific American*）发表的文章中以拱形薄壳的突然凹陷为例，直观展示了这种弹性突跳。扎曼的机械模型——一块连接弹簧的弹性板——成为演示尖点突变的标志性实验装置，实验中观察到的迟滞回路（hysteresis loop）与尖点突变控制空间中的"折叠区域"在几何上完全吻合。

## 关键公式与模型

**尖点突变分歧集判别式**（适用于相变、屈曲突跳）：

$$\Delta = 8a^3 + 27b^2 = 0$$

（势函数 $V = x^4/4 + ax^2/2 + bx$ 对应的分歧集边界方程，$\Delta < 0$ 区域为双稳态区域）

**彩虹角公式**（折叠突变在几何光学中的物理实现）：

$$\theta_{\text{rainbow}} = 4\arcsin\!\left(\sqrt{\frac{4-n^2}{3}}\right) - \pi$$

对 $n=1.332$（红光），$\theta_{\text{rainbow}} \approx 137.9°$（即观测者视角约 $42.1°$）。

**朗道展开与尖点突变对应的序参量方程**（相变均场理论）：

$$\frac{\partial F}{\partial M} = 2\alpha\tau M + 4\beta M^3 - H = 0$$

这与尖点突变平衡曲面方程 $x^3 + ax + b = 0$（令 $x \propto M$，$a \propto \tau$，$b \propto -H$）完全同构。

## 实际应用

**案例一：虹与霓的突变分析**。普通彩虹（主虹）对应折叠突变，而双虹（secondary rainbow）中的霓（副虹）在主虹外侧约 $8°$ 处出现，对应两次内反射的另一个折叠突变点。两虹之间的暗带称为"亚历山大暗带"（Alexander's dark band），正是两个突变点之间的影区，亮度在几何光学意义下为零。贝里的艾里函数计算预测了这一暗带中的"多余虹"（supernumerary rainbows）条纹，与观测完全吻合。

**案例二：海底焦散光斑的实时演化**。晴天时海底出现的闪烁网状光斑是水面波纹充当聚焦透镜时形成的焦散线网络。其瞬时形态中既有折叠型焦散（光斑边缘），也有尖点型焦散（两段折叠焦散线的汇合点）。Nye（1999，《自然光学》）系统测量了这些焦散线的拓扑，验证了贝里-阿普拉尼斯理论的预测。
