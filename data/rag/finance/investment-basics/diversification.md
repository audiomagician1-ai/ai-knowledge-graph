# 分散投资

## 概述

分散投资是指将资金同时投入多种不同资产或资产类别，通过降低单一资产亏损对整体组合的冲击，从而在不牺牲预期收益的前提下削减总体风险的投资策略。其核心逻辑在于：不同资产的价格涨跌往往不同步，当某只股票下跌20%时，其他持仓未必同样下跌，组合整体的波动因此被"平滑"。

分散投资的思想可追溯至1952年，经济学家哈里·马科维茨（Harry Markowitz）在《投资组合选择》论文中首次用数学方法证明：只要两种资产的相关系数小于1，将它们组合后的风险就必然低于两者按权重计算的加权平均风险。马科维茨也因此荣获1990年诺贝尔经济学奖。在此之前，"不把鸡蛋放一个篮子"虽是民间智慧，但缺乏定量支撑。

分散投资之所以重要，是因为投资者承担的风险分为两类：**系统性风险**（整个市场下跌，无法通过分散消除）和**非系统性风险**（个别公司或行业的特有风险，可以通过分散大幅降低）。实证研究表明，持有约20至30只相关性较低的股票，即可消除约90%的非系统性风险，剩余的主要是市场整体波动。

## 核心原理

### 相关系数：分散效果的数学基础

两种资产之间的**相关系数（ρ）**是衡量分散效果的关键指标，取值范围为 −1 到 +1。

- ρ = +1：两者完全正相关，同涨同跌，组合在一起毫无分散效果。
- ρ = 0：两者不相关，组合风险可以显著降低。
- ρ = −1：两者完全负相关，理论上可以构造出**零风险**组合。

两种资产组合的风险计算公式为：

$$\sigma_p = \sqrt{w_1^2\sigma_1^2 + w_2^2\sigma_2^2 + 2w_1w_2\sigma_1\sigma_2\rho_{12}}$$

其中 $\sigma_p$ 为组合标准差，$w_1、w_2$ 为两种资产的权重，$\sigma_1、\sigma_2$ 为各自标准差，$\rho_{12}$ 为相关系数。当 $\rho_{12} < 1$ 时，$\sigma_p$ 严格小于 $w_1\sigma_1 + w_2\sigma_2$，这就是分散化产生"1+1<2"效果的数学原因。

### 资产数量与边际递减效应

增加持仓数量对风险的降低并非线性，而是**边际效益递减**的。持仓从1只增至5只时，非系统性风险迅速下降；从20只增至100只时，额外的风险降低已非常有限。以美国股市历史数据为例：

| 持股数量 | 年化波动率（近似） |
|---------|----------------|
| 1只     | ~35%           |
| 5只     | ~20%           |
| 20只    | ~15%           |
| 100只   | ~13%           |
| 市场整体 | ~12%（系统性底线） |

这说明盲目堆砌更多资产并不能无限降低风险，过了某个数量阈值后，主要收益来自于选择**相关性更低**的资产类别，而非单纯增加数量。

### 跨类别分散：相关性才是关键

真正有效的分散，依赖的是资产之间相关性的差异，而非数量的堆砌。常见资产类别之间的历史相关系数大致如下：

- **A股与美股**（2010—2020年平均）：约 +0.3，相关性较低。
- **股票与债券**：在多数市场环境下约为 −0.1 至 +0.2，是经典的低相关配对。
- **同一行业内两只股票**：往往高达 +0.7 至 +0.9，分散效果极差。
- **黄金与股票**：历史上接近 0，危机时期甚至转负，常作为对冲工具。

这意味着持有10只银行股并不等于分散投资，因为它们的相关系数极高，银行业系统性风险发生时会同步下跌。

## 实际应用

**案例一：个人投资者的简单实践**
一位投资者将10万元全部买入某单一科技股，如果该公司出现财务造假，可能损失60%以上。若改为将资金分配至5只不同行业的股票（科技、医药、消费、能源、金融各20%），单一公司暴雷仅影响整体组合的20%，总损失从假设的60%降至约12%。

**案例二：股债组合在2008年金融危机中的表现**
2008年标普500指数全年下跌约38%，而同期美国10年期国债价格上涨约14%。持有60%股票+40%债券的经典组合全年亏损约20%，显著优于纯股票组合，印证了低相关资产混合的保护作用。

**案例三：指数基金作为低成本分散工具**
购买一只追踪沪深300指数的ETF，等同于一次性持有300只不同公司的股票，管理费通常低至0.1%—0.5%/年，是普通投资者实现充分分散的最经济路径，无需逐一研究每家公司。

## 常见误区

**误区一："买得越多越安全"**
许多人误以为持仓数量越多风险越低。实际上，若所购资产高度相关（例如20只A股科技股），再多的数量也无法摆脱行业集体下跌的风险。决定分散效果的核心变量是相关系数，而非数量本身。极端情形下，100只相关系数均为+0.95的股票组合，风险与单只股票相差无几。

**误区二："分散投资能消灭所有风险"**
分散只能消除**非系统性风险**，无法消除市场整体下行的系统性风险。2020年3月新冠疫情冲击期间，全球几乎所有股票类资产在一个月内同步下跌30%—50%，此时即便持有500只股票的组合也无法幸免。分散投资降低的是"某一只踩雷"的风险，而非"整个市场崩溃"的风险。

**误区三："相关性是固定不变的"**
历史相关性在危机时期往往会急剧上升——平时低相关的资产，在流动性危机中可能同步暴跌，因为投资者会不分好坏地抛售所有资产换取现金。这种现象被称为**相关性趋同**（correlation convergence），是分散投资在极端市场中保护效果打折的主要原因。因此，不能仅凭过去5年的相关系数来判断未来分散效果。

## 知识关联

分散投资建立在**风险收益权衡**的基础之上：已知单一资产的风险与预期收益后，分散化提供了一种在不主动降低预期收益的前提下减少风险的途径，体现了"免费午餐"的稀有性——它是少数几个被金融学理论证实、真实存在的免费午餐之一。

理解分散投资是迈向**资产配置**的直接前提。资产配置将分散化原理扩展到大类资产层面，明确回答"股票、债券、现金、房产各应占多大比例"，其背后的权重决策正是对相关系数和波动率的定量运用。

进一步深入则引出**投资组合理论**（现代投资组合理论，MPT），其核心概念"有效前沿"（Efficient Frontier）本质上是穷举所有可能的资产权重组合，找出在给定风险水平下预期收益最高的那条曲线——这是分散投资思想的数学极致表达。
