# 期权策略

## 概述

期权策略是指通过同时持有不同行权价、到期日或方向的期权合约组合，或将期权与标的资产搭配，以实现特定风险收益目标的操作方法。与单纯持有看涨或看跌期权相比，组合策略可以精确剪裁收益曲线，在牺牲部分上行空间的前提下大幅降低权利金成本，或在持有股票时提供下行保护。

期权组合策略的系统化研究随1973年布莱克-斯科尔斯定价模型的发表而快速发展，交易所标准化期权合约的推出（芝加哥期权交易所CBOE于同年成立）使策略的实际执行成为可能。机构投资者首先大规模运用备兑看涨策略管理股票持仓成本，随后价差策略因能控制最大亏损而在散户中广泛流行。

理解期权策略的核心价值在于：每种策略都对应一张明确的"损益图"（Payoff Diagram），横轴为到期日标的价格，纵轴为策略盈亏，投资者在入场前即可精确计算最大盈利、最大亏损和盈亏平衡点三个关键数字。这种可量化的风险边界是期权策略区别于股票或期货交易的根本特征。

## 核心原理

### 保护性看跌策略（Protective Put）

保护性看跌策略由**持有标的资产多头 + 购买等量看跌期权**构成。假设投资者以50元买入某股票，同时花费2元购买行权价为48元的看跌期权，则：

- **最大亏损** = 股价买入价 − 看跌行权价 + 权利金 = 50 − 48 + 2 = **4元**（无论股价跌至何处）
- **盈亏平衡点** = 股票买入价 + 权利金 = 50 + 2 = **52元**
- **上行收益** = 无限（股价涨越52元即开始盈利）

看跌期权充当保险，其权利金相当于"保费"，将无限下行风险截断为已知的固定亏损。此策略适合持有浮盈仓位担心回调，或重要事件（如财报）前临时对冲的场景。

### 备兑看涨策略（Covered Call）

备兑看涨策略由**持有标的资产多头 + 卖出等量看涨期权**构成。投资者卖出行权价高于当前股价的看涨期权，收取权利金，以此降低持仓成本。以持仓成本50元、卖出行权价55元看涨期权收入3元为例：

- **最大收益** = (行权价 − 持仓成本) + 收到权利金 = (55 − 50) + 3 = **8元**（股价涨过55元后收益封顶）
- **盈亏平衡点** = 持仓成本 − 权利金 = 50 − 3 = **47元**
- **最大亏损** = 持仓成本 − 权利金 = 47元（股价跌至0时亏损47元）

卖出看涨期权使上行潜力受限，适合判断标的短期横盘或温和上涨，想用时间价值衰减（Theta）产生稳定现金流的持仓者。CBOE的BXM指数长期跟踪备兑看涨策略表现，历史数据显示其波动率比标普500低约30%。

### 价差策略（Spread Strategies）

价差策略通过**同时买入和卖出同类期权（均为看涨或均为看跌），但行权价或到期日不同**，将最大盈亏双向锁定。

**牛市看涨价差（Bull Call Spread）**：买入低行权价看涨期权，卖出高行权价看涨期权。
- 净成本 = 买入权利金 − 卖出权利金（成本低于单买看涨）
- 最大收益 = 高行权价 − 低行权价 − 净成本
- 最大亏损 = 净成本（即初始支出）

**熊市看跌价差（Bear Put Spread）**：买入高行权价看跌期权，卖出低行权价看跌期权，适合温和看跌行情。

**铁鹰策略（Iron Condor）**是更进阶的价差组合，由一个牛市看跌价差 + 一个熊市看涨价差构成，共涉及4条腿，在标的价格区间震荡时盈利，收取净权利金，两侧各有固定止损位。

## 实际应用

**个人投资者对冲持仓风险**：A股投资者持有10万元某蓝筹股，临近年报披露期，担心利空消息导致大跌。购买1个月到期、虚值5%的看跌期权，权利金约为持仓市值的1.5%（即1500元），可将下行风险锁定在6.5%以内，同时保留上涨收益。

**机构投资者增厚收益**：基金公司持有大量中长期股票仓位，每月对持仓滚动卖出月度到期的备兑看涨期权，在股市横盘时每年可额外贡献3%−6%的年化收益（即权利金收入），这是保险资金和养老金最常用的期权增收手段之一。

**利用价差控制成本**：投资者看好某科技股从100元涨至115元，直接买看涨期权需支付8元权利金，而买100元看涨/卖115元看涨的牛市价差仅需3元净成本，最大亏损从8元降至3元，最大收益为12元，资金效率大幅提升。

## 常见误区

**误区一：认为备兑看涨策略"完全无风险"**。卖出看涨期权收到的权利金会降低盈亏平衡点，但标的股票大幅下跌时，权利金收入远不足以覆盖股票亏损。以上述例子计算，股价从50元跌至30元，亏损为17元（30元损失减去3元权利金），备兑策略仅是"降低成本"而非"消除风险"。

**误区二：价差策略的最大收益等于行权价之差**。牛市看涨价差的最大收益必须扣除建仓时支付的净权利金。若高低行权价相差10元，净成本3元，则最大收益仅为7元，混淆此点会导致收益预期严重高估。

**误区三：保护性看跌与止损单等效**。止损单在价格跳空时可能以远差于预设价格成交，而看跌期权赋予持有者以精确行权价卖出的权利，对跳空下跌提供刚性保护。但期权有到期日，需在到期前续仓，而止损单没有时间成本，两者适用场景不同。

## 知识关联

**前置概念衔接**：期权策略的构建完全依赖对**期权基础**的掌握——看涨期权（Call）的买卖权利与义务、看跌期权（Put）的权利金定价、内在价值与时间价值的区分、以及Delta表示的方向敞口。若不清楚卖出期权承担的义务，则无法理解备兑看涨中"备兑"（以持有的股票作为担保）的含义。

**横向概念联系**：期权策略中的Greeks（Delta、Theta、Vega）决定各策略对时间流逝和波动率变化的敏感性。备兑看涨策略是Theta正收益策略（时间流逝有利），而买入保护性看跌则是Theta负收益策略（持有越久时间价值损耗越多）。理解这一对立关系有助于根据市场环境选择合适的策略方向。

**延伸实践方向**：掌握基础三类策略后，可进一步学习跨式策略（Straddle，同时买入同行权价看涨和看跌）用于押注波动率而非方向，以及日历价差（Calendar Spread，不同到期日的相同行权价组合）用于捕捉时间价值差异，这两类策略将期权策略从"方向性工具"延伸至"波动率交易工具"。
