# 净现值分析

## 概述

净现值（Net Present Value，NPV）是将投资项目未来所有现金流量折现至当前时点后的加总，再减去初始投资成本所得的差值。其核心公式为：

**NPV = Σ [CFₜ / (1+r)ᵗ] − C₀**

其中 CFₜ 代表第 t 期的净现金流入，r 为折现率（通常取项目的加权平均资本成本 WACC），t 为期数，C₀ 为初始投资额。

NPV方法由经济学家欧文·费雪（Irving Fisher）在1930年的著作《利息理论》中奠定理论基础，后经约翰·伯尔·威廉姆斯于1938年进一步发展，最终成为现代公司金融评估资本投资的主流工具。与内部收益率（IRR）或回收期法相比，NPV直接以货币金额量化投资所创造的财富增量，与股东价值最大化目标完全契合。

NPV分析在公司金融中的独特价值在于：正NPV项目意味着投资回报超过资本成本，投资行为本身将为股东创造超额财富；负NPV项目则意味着资本被低效配置，应当拒绝。这一直接的货币量化特征，使决策者可以在不同规模、不同行业的项目之间进行横向比较与加总。

## 核心原理

### NPV计算的具体步骤

完整的NPV计算需要四个输入量：初始投资额、各期自由现金流（FCF）、项目寿命期限以及折现率。自由现金流的计算公式为：

**FCF = EBIT×(1−税率) + 折旧摊销 − 资本性支出 − 净营运资本增量**

以一个典型5年期制造业项目为例：初始投资1000万元，第1至5年预期FCF分别为200、250、300、280、260万元，WACC取10%。对每期现金流分别除以(1.10)¹至(1.10)⁵，求和后得到折现值约为987.5万元，减去1000万元初始成本，NPV≈−12.5万元，判定为拒绝投资。

### NPV决策规则

NPV的决策规则极为明确：**NPV > 0 则接受，NPV < 0 则拒绝，NPV = 0 则理论上无差异**。当面临多个互斥项目时（如两套不同设备方案），选择NPV绝对值最大的方案，而非NPV/投资比率（那是获利指数PI的逻辑）。对于独立项目组合，只要各项目NPV为正，均可同时接受，因为NPV具有**加法可加性**——组合项目的NPV等于各单项NPV之和，这是IRR不具备的性质。

### 敏感性分析

单点NPV计算依赖多个假设，敏感性分析用于识别哪些假设对最终结论影响最大。具体操作是：每次仅改变一个变量（如收入增长率从基准的5%调整至3%或7%），保持其他变量不变，重新计算NPV，观察变化幅度。

**龙卷风图（Tornado Chart）**是敏感性分析的标准可视化工具：将各变量对NPV的影响范围从大到小纵向排列，形似龙卷风形状。若折现率1个百分点的变动导致NPV变化超过15%，而销量变动同等幅度仅引起5%变化，则折现率是该项目的关键风险驱动因素，需要重点管控。

**情景分析**是敏感性分析的升级版，它同时改变多个变量，构建悲观（如经济衰退）、基准、乐观三种完整情景，分别计算对应NPV，从而评估项目在极端环境下的抗风险能力。

## 实际应用

**案例一：工厂扩建决策**
某快消品公司考虑投资8000万元新建一条生产线，预计10年内每年新增税后现金流1200万元。以WACC=9%折现，年金现值系数为6.418，折现现金流合计7701.6万元，NPV = 7701.6 − 8000 = −298.4万元。结论为拒绝，除非管理层能将成本压缩至7700万元以下，或找到提升年现金流至1250万元以上的途径。

**案例二：并购估值中的NPV应用**
在收购目标公司时，投资银行通常构建5年详细FCF预测，第6年起以"终值（Terminal Value）"替代，终值公式为：**TV = FCF₆ / (WACC − g)**，其中g为永续增长率（通常取2%~3%）。将TV折现至今，与前5年FCF折现值加总，得到目标公司的NPV（即内在价值），再与报价比较，正差值代表收购溢价空间。

**案例三：租赁还是购买的NPV比较**
企业面临购买设备（一次性支付500万元）与租赁设备（每年支付80万元，租期8年）的选择，可将两方案的现金流出分别折现，比较NPV绝对值，选取现值成本更低的方案，这是NPV方法在成本最小化决策中的直接应用。

## 常见误区

**误区一：将会计利润直接当作现金流折现**
净现值分析要求使用**现金流**而非会计利润。折旧是非现金费用，不应作为现金流出处理；相反，资本性支出在发生当期全额计入现金流出，不按折旧年限分摊。若错误地用净利润代替FCF，会因折旧的双重计算（已在税前利润中扣除，再次被视为现金流出）而低估NPV，可能导致放弃本应接受的正NPV项目。

**误区二：把沉没成本纳入NPV计算**
已经发生且无法追回的支出（如前期研发费用200万元）属于沉没成本，不应计入NPV计算的现金流序列。NPV分析只考虑**增量现金流**——即因做出本次投资决策而新增的现金流入与流出。将沉没成本纳入会人为压低NPV，可能导致对仍有正增量价值的项目作出错误的否决决策。

**误区三：使用单一折现率忽视项目风险差异**
很多初学者对公司所有项目统一使用公司层面的WACC作为折现率。然而，若某投资项目的系统风险显著高于公司整体业务（如进入全新行业），应使用反映该项目特定风险的折现率，而非公司WACC。使用过低的折现率会高估高风险项目的NPV，导致错误接受决策。
