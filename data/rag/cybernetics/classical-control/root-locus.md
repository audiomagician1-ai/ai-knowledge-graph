# 根轨迹法

## 概述

根轨迹法（Root Locus Method）由美国控制工程师沃尔特·埃文斯（Walter R. Evans）于1948年至1950年间系统提出，核心成果发表于论文《Graphical Analysis of Control Systems》（Evans, 1948）和《Control System Synthesis by Root Locus Method》（Evans, 1950）。该方法以图形化方式揭示：当开环增益 $K$ 从 $0$ 变化至 $+\infty$ 时，闭环系统特征方程的根（即闭环极点）在复平面上所描绘出的完整轨迹。

根轨迹法的核心价值在于将抽象的代数求根问题转化为几何图形问题。工程师通过观察轨迹是否进入左半复平面（LHP）、距离虚轴的远近、以及阻尼比对应的射线角度，可以直觉性地判断系统的稳定性与动态响应品质，而无需对每个增益值反复求解高阶多项式方程根。

## 核心原理

### 特征方程与相角条件

考虑标准反馈控制系统，开环传递函数为 $G(s)H(s)$，则闭环特征方程为：

$$1 + KG(s)H(s) = 0$$

等价改写为：

$$KG(s)H(s) = -1$$

由于 $-1$ 是模为 $1$、辐角为 $\pm 180°$ 的复数，该方程可拆解为两个独立条件：

- **相角条件（Angle Condition）**：$\angle G(s)H(s) = (2k+1)\cdot 180°$，$k = 0, \pm1, \pm2, \ldots$
- **幅值条件（Magnitude Condition）**：$K = \dfrac{1}{|G(s)H(s)|}$

其中**相角条件是判断某点是否位于根轨迹上的充要条件**；幅值条件仅用于在确认某点在轨迹上之后计算对应的增益 $K$ 值。这一区分是根轨迹法精髓所在——几何作图完全由相角条件驱动，增益的确定是后续步骤。

### 埃文斯八条绘制法则

Evans（1950）总结了从开环零极点信息直接推断根轨迹形态的八条规则，使得工程师无需逐点计算即可手工草绘轨迹：

1. **起点与终点**：根轨迹起始于开环极点（$K=0$），终止于开环零点（$K\to\infty$）；若零点数 $m$ 少于极点数 $n$，则有 $n-m$ 条轨迹趋向无穷远。

2. **对称性**：根轨迹关于实轴对称，因为特征多项式系数为实数，复数根必成共轭对出现。

3. **实轴上的轨迹段**：实轴上某段属于根轨迹，当且仅当该段右侧开环实数零极点的总数为奇数。

4. **渐近线**：趋向无穷远的 $n-m$ 条轨迹的渐近线角度为：
$$\theta_a = \frac{(2k+1)\cdot 180°}{n-m}, \quad k = 0, 1, \ldots, n-m-1$$

5. **渐近线交点（重心）**：渐近线汇交于实轴上的点：
$$\sigma_a = \frac{\sum \text{开环极点} - \sum \text{开环零点}}{n - m}$$

6. **分离点与汇合点**：两条轨迹在实轴上相遇后离开（分离）或重新汇合的点，由 $\dfrac{dK}{ds} = 0$ 求解，即对特征方程关于 $s$ 求导并令导数为零。

7. **与虚轴的交叉**：令特征方程中 $s = j\omega$，分别令实部与虚部为零，可求解交叉点坐标 $\omega$ 及对应临界增益 $K_{cr}$，此即劳斯-赫尔维茨（Routh-Hurwitz）判据的图形化对应。

8. **出射角与入射角**：从复数开环极点出发的轨迹出射角，以及进入复数开环零点的入射角，均由相角条件精确计算。

### 阻尼比与自然频率的几何意义

在复平面中，闭环极点 $s = -\sigma \pm j\omega_d$ 所在位置具有直接的时域响应意义：

$$\zeta = \cos\theta, \quad \omega_n = \sqrt{\sigma^2 + \omega_d^2}$$

其中 $\theta$ 是从原点到极点的连线与负实轴的夹角，$\omega_n$ 是从原点到极点的距离。因此，根轨迹与过原点、半角为 $\theta = \arccos\zeta$ 的射线的交点，即为满足特定阻尼比 $\zeta$ 要求的设计点。工程中常见要求 $\zeta \geq 0.707$，对应 $\theta \leq 45°$ 的区域。

## 关键公式与模型

### 增益-极点关系的完整表达式

设开环传递函数写为零极点形式：

$$G(s)H(s) = \frac{K\prod_{j=1}^{m}(s-z_j)}{\prod_{i=1}^{n}(s-p_i)}$$

则在根轨迹上任意一点 $s_0$，对应的增益由幅值条件给出：

$$K = \frac{\prod_{i=1}^{n}|s_0 - p_i|}{\prod_{j=1}^{m}|s_0 - z_j|}$$

即**所有开环极点到该点的距离之积，除以所有开环零点到该点的距离之积**。这一几何解释使得"用尺子量复平面上的距离"成为确定增益的实用操作。

### 二阶系统的典型根轨迹案例

考虑开环传递函数 $G(s) = \dfrac{K}{s(s+2)}$，其特征方程为 $s^2 + 2s + K = 0$。

两个开环极点位于 $s = 0$ 和 $s = -2$，无开环零点（$m=0, n=2$），因此有两条轨迹趋向无穷远，渐近线角度为 $\pm 90°$，渐近线交点为 $\sigma_a = (0 + (-2))/(2-0) = -1$。

分离点由 $\dfrac{d}{ds}\left[-s(s+2)\right] = 0$ 求解，得 $-(2s+2)=0$，即 $s = -1$，对应 $K = 1$。

- 当 $0 < K < 1$：两个实数极点，过阻尼响应；
- 当 $K = 1$：重极点 $s = -1$，临界阻尼；
- 当 $K > 1$：一对共轭复数极点，欠阻尼振荡响应，轨迹沿垂直线 $\text{Re}(s) = -1$ 向上下延伸。

由于轨迹始终在左半平面，该系统对所有 $K > 0$ 均稳定。

## 实际应用

### 飞行控制系统设计

在波音707等早期喷气式飞机的俯仰姿态控制中，工程师使用根轨迹法选择俯仰速率陀螺增益，使闭环极点位于 $\zeta \approx 0.5\text{～}0.7$、$\omega_n \approx 2\text{～}5$ rad/s 的区域，以满足MIL-F-8785飞行品质规范对短周期模态的要求。根轨迹法使工程师能直观判断增益裕度（Gain Margin），即轨迹在交叉虚轴之前允许的增益增大倍数。

### 直流电机速度控制

例如，对于直流电机开环传递函数 $G(s) = \dfrac{K_m}{s(\tau s+1)}$（其中 $K_m$ 为电机增益常数，$\tau$ 为电气时间常数），根轨迹分析显示：随比例增益 $K$ 增大，闭环极点从实轴上分离并向复平面延伸，振荡频率 $\omega_d$ 随 $K$ 增大而增大，而实部保持不变（对应固定的收敛速度 $1/\tau$）。工程师据此在速度响应的快速性与振荡可接受程度之间取得平衡。

### 与频域法的互补关系

根轨迹法与波德图（Bode Plot）、奈奎斯特图（Nyquist Plot）构成经典控制理论的三大图形工具，均在1940年代末至1950年代初密集涌现。根轨迹法直接给出极点位置（时域响应直接相关），而频域方法给出频率响应特性（易于处理系统不确定性）；两者互补使用才能对系统动态有完整把握。

## 常见误区

**误区一：将根轨迹的相角条件混淆为幅值条件**。
初学者常误以为"只要幅值等于 $1/K$，该点就在轨迹上"。实则幅值条件是确定增益的工具，轨迹的几何形状完全由相角条件 $\angle G(s)H(s) = \pm 180°$ 决定。相角条件对增益 $K$ 的具体数值没有任何约束。

**误区二：认为根轨迹仅适用于正反馈情形的变型**。
标准根轨迹（Evans根轨迹）针对负反馈系统，相角条件为奇数倍 $180°$。**互补根轨迹**（Complementary Root Locus）用于正反馈，相角条件为偶数倍 $180°$（即 $0°, \pm 360°, \ldots$），实轴上的轨迹段判断规则也随之改变（右侧零极点总数为偶数的段属于互补轨迹）。混淆两者会导致轨迹形状完全错误。

**误区三：根轨迹随 $K$ 单调向右移动意味着一定会不稳定**。
轨迹进入右半平面的临界增益 $K_{cr}$ 可以通过劳斯表或令 $s=j\omega$ 代入特征方程求得。对某些系统（如含不稳定开环极点的系统），根轨迹可能从右半平面出发，经过适当增益后反而进入左半平面，实现镇定，这与直觉相反。

**误区四：分离点总是位于两相邻实数极点之间**。
分离点的位置由 $dK/ds = 0$ 的实数解决定，对高阶系统，解可能位于两相邻实数极点之外，甚至位于零点附近（汇合点）。直接套用"两极点取中点"的直觉会产生严重误差。

## 知识关联

**前置知识——传递函数（Transfer Function）**：根轨迹的出发点是系统开环传递函数 $G(s)H(s)$ 的零极点分解，不理解拉普拉斯变换与传递函数的学习者无法理解相角条件的推导来源。

**后续应用——超前-滞后补偿器设计（Lead-Lag Compensator Design）**：根轨迹法的最直接工程延伸是补偿器设计。通过在开环函数中添加零点（超前补偿）可将轨迹拉向左半平面，改善稳定裕度；添加极点（滞后补偿）可在低频段增大增益，改善稳态误差。补偿器零极点的位置选取本质上是根轨迹图形的逆向设计问题（Evans, 1950）。

**横向联系——状态空间极点配置（Pole Placement）**：根轨迹法在单输入单输出（SISO）系统中通过调节单一增益移动极点；状态反馈极点配置（Ackermann公式）则通过多维状态反馈向量将极点任意指定到期望位置，是根轨迹思想在现代控制理论中的推广（Ogata, 1990，《Modern Control Engineering》）。

**理论基础——劳斯-赫尔维茨稳定判据**：根轨迹与虚轴的交点精确对应劳斯表出现全零行时的临界条件，两者从代数与几何两个角度描述同一稳定边界，互为验证手段。
