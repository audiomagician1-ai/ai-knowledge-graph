# 补偿器设计

## 概述

补偿器设计（Compensator Design）是经典控制理论中将系统性能规格转化为具体网络或算法参数的核心技术环节。其本质是在原有被控对象的开环传递函数 $G_p(s)$ 基础上，串联或并联一个附加传递函数 $G_c(s)$，使闭环系统同时满足稳态精度、瞬态响应速度和相对稳定裕度三方面指标。

历史上，超前-滞后补偿网络的系统化设计方法由 Hendrik Bode 在1945年出版的《Network Analysis and Feedback Amplifier Design》中奠定基础，Bode 图方法使工程师能够在频率域直接操纵相位和增益裕度。与此同时，Ziegler 与 Nichols 于1942年在《Transactions of the ASME》发表的论文中给出了 PID 参数整定的经验规则，这两条技术路线至今仍是工业控制中最广泛使用的补偿器设计框架（Bode, 1945；Ziegler & Nichols, 1942）。

补偿器设计的难度在于：稳态性能、带宽、相位裕度之间存在内在矛盾——提高开环增益可改善稳态误差但往往损害稳定裕度；增大带宽加快响应速度但放大高频噪声。三类经典补偿器——**超前补偿器（Lead Compensator）**、**滞后补偿器（Lag Compensator）** 和 **PID 补偿器**——分别从不同角度化解这一矛盾。

## 核心原理

### 超前补偿器（Lead Compensator）

超前补偿器的传递函数标准形式为：

$$G_c(s) = K_c \frac{s + z_c}{s + p_c}, \quad p_c > z_c > 0$$

其中极点 $p_c$ 大于零点 $z_c$，导致在零极点之间的频率段产生**正相位贡献**（相位超前）。最大超前相位角 $\phi_m$ 出现在几何中心频率 $\omega_m = \sqrt{z_c \cdot p_c}$ 处，由以下公式精确计算：

$$\phi_m = \arcsin\left(\frac{1 - \alpha}{1 + \alpha}\right), \quad \alpha = \frac{z_c}{p_c} < 1$$

单级超前补偿器能提供的最大相位超前约为 $55°$～$60°$（对应 $\alpha \approx 0.05$）；若需要更大相位补偿，须级联两级超前网络。

**设计步骤（频率域法）**：
1. 确定所需相位裕度增量 $\Delta\phi = PM_{desired} - PM_{uncompensated} + 5°$（5° 为安全余量）；
2. 由 $\Delta\phi$ 求 $\alpha$：$\alpha = \frac{1 - \sin\Delta\phi}{1 + \sin\Delta\phi}$；
3. 将 $\omega_m$ 选在未补偿系统的增益交叉频率处，据此确定 $z_c = \omega_m\sqrt{\alpha}$，$p_c = \omega_m / \sqrt{\alpha}$；
4. 调整 $K_c$ 以维持原增益交叉频率处的 $0\,\text{dB}$ 穿越（补偿超前网络带来的增益损失 $-20\log(1/\sqrt{\alpha})$\,dB）。

**案例**：被控对象 $G_p(s) = \frac{10}{s(s+2)}$，未补偿相位裕度约 $18°$，要求达到 $50°$。计算得 $\Delta\phi = 37°$，$\alpha \approx 0.25$，$\omega_m \approx 4.5\,\text{rad/s}$，设计出 $G_c(s) = 2\frac{s+2.25}{s+9}$，验证后相位裕度达 $51.3°$，满足规格。

超前补偿器的物理实现可用 RC 无源网络：一个电阻 $R_1$ 与电容 $C$ 串联后再并联电阻 $R_2$，其传递函数与上式完全对应，历史上大量模拟控制器即采用此结构（Ogata, 1970）。

### 滞后补偿器（Lag Compensator）

滞后补偿器的传递函数为：

$$G_c(s) = K_c \frac{s + z_c}{s + p_c}, \quad 0 < p_c < z_c$$

此时 $p_c < z_c$，网络在低频段提供**正增益提升**（约 $20\log(z_c/p_c)$\,dB），而在高频段几乎不改变相位（实际会引入少量相位滞后，因此在设计时将滞后补偿器的零极点放置在增益交叉频率 $\omega_{gc}$ 的 $1/10$ 以下，以防止损害相位裕度）。

滞后补偿器的核心作用是**在不改变高频特性（即不改变相位裕度和带宽）的前提下，通过提高低频增益来降低稳态误差**。令 $\beta = z_c / p_c > 1$，则低频增益提升为 $20\log\beta$\,dB。

$$\text{稳态速度误差系数提升比} = \beta = \frac{z_c}{p_c}$$

**设计步骤**：
1. 确定满足稳态误差要求所需的额外低频增益 $\beta$（例如要求 $K_v$ 提高5倍，则 $\beta = 5$）；
2. 选择滞后补偿器的零点 $z_c = 0.1\,\omega_{gc}$（放置在增益穿越频率十分之一处），则 $p_c = z_c / \beta$；
3. 验证补偿后在 $\omega_{gc}$ 处相位裕度减少不超过 $5°$。

**案例**：对象 $G_p(s) = \frac{K}{s(s+1)(s+4)}$，要求 $K_v \geq 5\,\text{s}^{-1}$ 且相位裕度 $\geq 40°$。在不补偿时 $K=5$ 使 $K_v=1.25$ 不达标；增大 $K$ 会损失相位裕度。设计滞后补偿器 $\beta=4$，$z_c=0.1$，$p_c=0.025$，得 $G_c(s) = \frac{s+0.1}{s+0.025}$，既满足 $K_v$，又保持相位裕度约 $42°$。

### PID 补偿器

PID（比例-积分-微分）补偿器的时域控制律为：

$$u(t) = K_p e(t) + K_i \int_0^t e(\tau)\,d\tau + K_d \frac{de(t)}{dt}$$

其拉普拉斯域传递函数为：

$$G_c(s) = K_p + \frac{K_i}{s} + K_d s = \frac{K_d s^2 + K_p s + K_i}{s}$$

- **积分环节 $K_i/s$**：引入一个原点处的开环极点，将系统型次提升1级，从根本上消除阶跃输入的稳态误差（对 0 型系统）；
- **微分环节 $K_d s$**：等效于相位超前补偿，提升高频相位，改善动态响应（减少超调、缩短调节时间）；但纯微分器对噪声极度敏感，工程实现通常加截止极点：$G_d(s) = \frac{K_d s}{s/N + 1}$，$N$ 一般取 5～20。

**Ziegler-Nichols 整定规则（1942）**：基于被控对象的临界增益 $K_u$ 和临界振荡周期 $T_u$，直接给出 PID 参数：

| 控制器类型 | $K_p$       | $T_i$ (= $K_p/K_i$) | $T_d$ (= $K_d/K_p$) |
|-----------|-------------|----------------------|----------------------|
| P         | $0.5K_u$    | $\infty$             | 0                    |
| PI        | $0.45K_u$   | $T_u/1.2$            | 0                    |
| PID       | $0.6K_u$    | $T_u/2$              | $T_u/8$              |

该规则设计的 PID 闭环系统具有约 $25\%$ 的衰减比（Decay Ratio = 0.25），即每次振荡峰值为前次的1/4，代表了响应速度与稳定性的经验平衡点。

PID 控制器也可从根轨迹视角理解：PD 控制器在原点附近引入一个零点，将根轨迹向左半平面吸引，改善阻尼；PI 控制器引入紧靠原点左侧的零点和原点极点，在几乎不影响主导极点的同时消除稳态误差。

## 关键公式与性能指标对比

| 补偿器类型   | 相位贡献         | 增益贡献        | 主要改善指标     | 主要代价           |
|-------------|-----------------|----------------|-----------------|-------------------|
| 超前补偿     | 正相位（超前）   | 高频增益上升     | 相位裕度、带宽   | 噪声敏感性增加     |
| 滞后补偿     | 轻微负相位（滞后）| 低频增益上升    | 稳态精度         | 响应速度轻微下降   |
| PID（完整）  | 中频正相位       | 低频无限增益     | 稳态精度+动态响应 | 需整定3个参数      |

对于一阶被控对象 $G_p(s) = K/(Ts+1)$，理想 PID 可实现**完全极点对消**（Pole-Zero Cancellation），使闭环变为纯一阶系统，调节时间仅由积分增益决定：

$$t_s \approx \frac{4T}{K \cdot K_i \cdot T}$$

但极点对消在实际中需谨慎——若被消极点位于右半平面（非最小相位系统），对消仅是"隐藏"而非真正消除不稳定模态，内部状态仍将发散（Ogata, 1970；Franklin et al., 1994）。

## 实际应用

**工业过程控制**：炼油厂温度控制回路中，被控对象通常为带纯滞后的一阶惯性系统（FOPDT），标准做法是使用带Smith预估器的PID，其中Smith预估器将纯滞后 $e^{-Ls}$ 从闭环中"移出"，使PID仅需对最小相位部分设计（Smith, 1957）。

**伺服电机驱动**：数控机床的位置环通常采用超前补偿器或PD控制器，将系统相位裕度提升至 $45°$～$60°$，确保定位精度在 $\pm 1\,\mu\text{m}$ 量级的同时响应频率达 $100\,\text{Hz}$ 以上。

**电力电子变换器**：DC-DC Buck变换器的电压控制中，由于LC输出滤波器在谐振频率 $\omega_n = 1/\sqrt{LC}$ 处产生 $-180°$ 的急剧相位跌落，通常采用 **Type-III 补偿器**（即两个超前零点加两个滞后极点）来在宽频段内同时修复相位和增益，实现带宽 $f_c > f_s/10$（$f_s$ 为开关频率）。

## 常见误区

**误区一：超前补偿器可以任意叠加提供相位**。实际上，每级超前网络的极点都会提升高频增益，级联过多将导致系统对传感器噪声极度敏感，且每级最大超前约55°，三级以上的设计应考虑改用状态反馈方法。

**误区二：PID 的微分项等同于超前补偿**。PID 微分项 $K_d s$ 在理论上是纯微分，相位贡献高达 $+90°$，而超前补偿器的零极点结构将相位贡献限制在特定频段。工程实现中的带限微分 $K_d s/(s/N+1)$ 在高于 $N\omega_0$ 后相位回落，与超前补偿器的频率响应曲线形状相近但参数含义不同。
