# 新柏拉图主义

## 概述

新柏拉图主义是公元3世纪兴起于罗马帝国的哲学流派，由普罗提诺（Plotinus，约205—270年）系统创立，其思想核心是以"太一"（The One）为绝对第一本原的等级性流溢宇宙论。普罗提诺的弟子波菲利（Porphyry，约234—305年）在其师去世后，将遗稿54篇整理为《九章集》（*Enneads*，希腊语 ἐννεάδες），按六组九篇的结构排列，并附上传记《普罗提诺传》（*Vita Plotini*），此书是迄今研究普罗提诺思想的第一手原典（Plotinus, *Enneads*, trans. Armstrong, Loeb Classical Library, 1966–1988）。

这一流派并非对柏拉图的简单注释。普罗提诺以柏拉图《巴门尼德篇》中"太一"的辩证分析、《蒂迈欧》中的宇宙生成论、以及《理想国》509b 中"善超越存在"的命题为思想资源，同时吸收了亚里士多德《形而上学》第十二卷的"不动的推动者"（自我思考的理智）和斯多葛的宇宙气息论（pneuma），综合构建出一套三本体（hypostases）层级结构：太一→努斯（Nous）→灵魂（Psyche）。

新柏拉图主义的影响跨越千余年：奥古斯丁（354—430年）在皈依基督教前大量研读普罗提诺，伊斯兰哲学家法拉比（al-Fārābī，约872—950年）和伊本·西拿（Ibn Sīnā，980—1037年）的存在论皆带有显著的流溢论色彩，中世纪神秘主义者迈斯特·艾克哈特（Meister Eckhart，约1260—1328年）的"神性根底"（Gottheit）概念直接继承太一的否定神学表述。1492年，意大利人文主义者费奇诺（Marsilio Ficino，1433—1499年）将《九章集》全部译成拉丁文，为文艺复兴的柏拉图主义复兴奠定了文本基础。

## 核心原理

### 太一（The One）：超越存在的第一本原

普罗提诺体系的绝对顶端是"太一"（τὸ ἕν，to hen）。太一超越一切谓词——不能说它"是"什么，不能说它"思考"，不能说它"存在"，甚至不能说它"是一"，因为一旦施加任何谓词，即意味着主谓之间的区分，而区分即多样性，多样性破坏绝对统一。这种进路被称为"否定神学"（apophatic theology）或"否定之路"（via negativa），在《九章集》V.3 和 VI.9 中有最集中的论述。

太一创生世界不出于意志，不出于计划，更不出于任何匮乏。普罗提诺在《九章集》V.1.6 中使用了一个精准的类比：正如太阳放光并非出于选择，完满的存在必然向外"溢出"。这一逻辑意味着太一的生产是非意志性的、必然性的，与后来基督教"上帝自由创造"的神学立场形成鲜明对比——这正是奥古斯丁改造新柏拉图主义时须处理的最大张力。

太一在《巴门尼德篇》137c—142a 的第一假设中已有预示：若"一"是严格意义上的一，它便不具有部分、整体、形状、位置、时间，甚至不能被命名，也不能被知晓。普罗提诺将此读作对第一本原本性的正面描述，而非单纯的逻辑练习。

### 流溢说（Emanation）：三本体的层级结构

普罗提诺的宇宙生成论是"流溢"（ἀπόρροια，aporrhoia；或 πρόοδος，prohodos，即"前进/展开"），而非犹太-基督教的"创造"（creatio ex nihilo）。流溢不意味着太一有所损失，正如蜡烛发光后并不减弱。三个本体层次（hypostases）由高至低依次为：

**第一本体：太一（The One）**
绝对统一，无思无欲，超越存在范畴，是一切的"无根之根"。

**第二本体：努斯（Νοῦς，Intellect / 理智）**
太一的第一流溢物。努斯是思维与被思维对象的同一（noesis noeseos，思想的思想），内含柏拉图全部的理念（εἴδη，eidē）世界。但努斯的这一反思性（"思考太一"）即引入了思者与被思者的二元性，是"多"的逻辑起点。对应亚里士多德《形而上学》1074b34 中"神圣理智自我思考"的命题，普罗提诺赋予了努斯更丰富的内容：它不仅自我思考，且其中充满无数理念，每一个理念既是独立的，又统摄于努斯的整体统一中（*Enneads* V.9）。

**第三本体：灵魂（Ψυχή，Psyche / Soul）**
努斯的流溢物，分为"世界灵魂"（World Soul，不降入质料）和"个体灵魂"（individual souls，可降入肉体）。世界灵魂向下俯望质料世界，生成时间（χρόνος，chronos，《九章集》III.7 对时间的著名定义：时间是"永恒在运动中的影像"）与感性宇宙。个体灵魂则处于努斯与质料之间的临界位置，这一位置赋予了人类独特的哲学使命：可以向上回归太一，也可以向下沉溺质料。

**质料（Ὕλη，Hyle）**
流溢链的最末端，是"非存在"（mē on）的极点，代表黑暗、恶与匮乏。质料并非独立的恶的本原（此处普罗提诺拒绝摩尼教式的二元论），而是流溢链在极度稀薄化后的边界状态，是"形式的绝对缺乏"（*Enneads* I.8）。

用结构示意，三本体的垂直秩序可表示为：

$$\text{太一（The One）} \xrightarrow{\text{流溢}} \text{努斯（Nous）} \xrightarrow{\text{流溢}} \text{灵魂（Psyche）} \xrightarrow{\text{俯视}} \text{质料（Hyle）}$$

每一层次对上一层次的"转向"（ἐπιστροφή，epistrophē）产生认知关系，而这种认知关系本身又使该层次"得以存在"——存在、生命、理智三者在努斯层次中合为一体（*Enneads* VI.6）。

### 回归与神秘合一（Epistrophe and Henosis）

流溢不是单向的宇宙展开，普罗提诺同时强调灵魂向上的"回归"（ἐπιστροφή）运动。《九章集》I.6（论美）给出了灵魂上升的完整路径：

1. **净化阶段**（Katharsis）：放弃对肉体欲望的执著，使灵魂"轻装"，恢复其本性的纯粹性。
2. **理智沉思阶段**（Theoria）：专注于努斯层次的理念世界，使灵魂完全与理智同化，达到"与努斯合一"。
3. **神秘合一阶段**（Henosis，字面义"统一化"）：超越理智的二元性，进入与太一的无言合一。这一状态不是认知关系——只要仍保有"我在合一"的意识，就仍是主客二分，尚未真正抵达太一。波菲利在《普罗提诺传》第23章记录：普罗提诺在生命的最后六年中共经历了**四次**与太一合一的直接体验（henosis），波菲利本人也曾有过一次。

这种神秘合一体验与同时代的基督教神秘主义、摩尼教光明神学形成有意思的对比：普罗提诺的合一是"无形象"（formless）的，拒绝一切感性内容，而摩尼教的光明体验往往伴随具体的光明形象。

## 关键概念与公式

普罗提诺对"永恒"与"时间"关系的界定是新柏拉图主义哲学中最精确的概念分析之一，出自《九章集》III.7.11：

$$\text{时间（χρόνος）} = \text{永恒（αἰών）在运动中的影像（εἰκών）}$$

这一公式直接呼应柏拉图《蒂迈欧》37d 的表述（"时间是永恒按数运行的动态影像"），但普罗提诺将其精确化：时间不是宇宙天球的旋转，而是世界灵魂在活动展开中对自身生命的"延伸"（diastasis）。时间属于灵魂本体，而非物理天体的属性。这一立场比奥古斯丁在《忏悔录》第十一卷中将时间定性为"心灵的延伸"（distentio animi）早约130年，并直接影响了后者。

三本体之间的关系可用以下逻辑结构来把握：每一本体对其来源的"凝视"（theoria）使它自身得以规定和存在：

```
太一（无规定性，超越存在）
  │ 流溢（prohodos）
  ▼
努斯（凝视太一 → 自我折回 → 产生二元性：思者/被思者）
  │ 流溢
  ▼
灵魂（凝视努斯 → 生成理念的时间影像）
  │ 俯视
  ▼
质料（流溢链末端，形式缺乏，"非存在"边界）
```

## 实际应用与历史影响

**对基督教神学的改造性吸收**
奥古斯丁在《忏悔录》（*Confessiones*，约397—400年）第七卷9节明确写道，是"某些柏拉图主义者的书"（libri Platonicorum，学界普遍认为即维多里努斯翻译的普罗提诺文本）使他理解了"非物质实体"的可能性，突破了摩尼教将善恶设想为两种物质力量的困境。奥古斯丁将努斯层次的理念世界内化为"上帝的永恒理念"，将灵魂的回归路径改写为基督教的"恩典与救赎"叙事，并将太一的无条件完满性转化为基督教的"上帝的善性"。

**伊斯兰哲学中的流溢论**
中世纪伊斯兰哲学家法拉比在《完善城邦居民的观点》（*Ārāʾ ahl al-madīna al-fāḍila*，约940年）中，将普罗提诺的流溢结构与亚里士多德的天球智识层叠相结合，构建出从"第一存在"（al-mawjūd al-awwal，对应太一）向下流溢出十个智识（Intellects）的宇宙论。伊本·西拿（阿维森纳）进一步将流溢论发展为存在论：他区分"本质"（māhiyya）与"存在"（wujūd），认为万物的存在皆是从"必然存在者"（Wājib al-Wujūd，即上帝）流溢而来的"偶然存在"——这一区分在13世纪经过阿奎那的拉丁转译，成为经院哲学"本质与存在之区分"的核心命题（Gilson, *Being and Some Philosophers*, Pontifical Institute, 1949）。

**文艺复兴新柏拉图主义的复活**
费奇诺在1492年完成《九章集》的拉丁文全译，并在其《柏拉图神学》（*Theologia Platonica*，1482年）中将普罗提诺的灵魂不朽论与基督教复活教义相结合，使"灵魂作为宇宙中心"的命题成为文艺复兴人文主义的精神地标。
