# 爱情心理学

## 概述

爱情心理学是社会心理学中专门研究浪漫爱情的情感结构、形成机制与维持条件的分支领域。与泛泛的"喜欢"不同，爱情涉及特定的神经化学基础——恋爱初期大脑会大量分泌多巴胺、去甲肾上腺素和苯乙胺（PEA），产生类似成瘾的兴奋体验；而长期伴侣关系则更多依赖催产素和加压素维系情感纽带。

罗伯特·斯滕伯格（Robert Sternberg）于1986年在《心理学评论》（Psychological Review）上正式提出爱情三角理论，将人类爱情系统性地拆解为三个独立但可交叠的成分。此前，鲁宾（Zick Rubin）于1970年代区分了"喜欢"与"爱"的量表维度，李·约翰（John Lee）在1977年提出爱情六色彩类型学，这些研究奠定了爱情心理学的实证基础。斯滕伯格的贡献在于提供了一套可以预测恋爱满意度与稳定性的几何化模型。

爱情三角理论与依恋风格理论共同构成了当代爱情心理学的两大支柱：前者描述爱情的结构是什么，后者解释人们为何以特定方式去爱。两套框架结合使用，可以解释为什么同样相爱的两人，关系结局却截然不同。

## 核心原理

### 斯滕伯格爱情三角理论的三个成分

斯滕伯格将爱情分解为**亲密（Intimacy）、激情（Passion）、承诺（Commitment）**三个顶点，形成一个三角形。

- **亲密**指情感上的联结、温暖与分享欲，是关系中缓慢建立、难以快速消退的成分。
- **激情**指生理唤醒、性吸引与强烈的迷恋状态，上升迅速但下降同样迅速——研究显示激情的高峰期平均持续约6至30个月，随后因"习惯化"（habituation）而衰减。
- **承诺**分为短期（决定爱对方）和长期（维持这段关系）两个层面，是三者中最受意志力控制的成分。

三角形的**面积**代表爱情的总量，三角形的**形状**代表爱情的类型。当三个成分大小相近时，三角形趋于等边，斯滕伯格将此定义为"完美之爱"（Consummate Love），是最完整的爱情形式，也是最难长期维持的形式。

三角理论还引入了"现实三角"与"理想三角"的比较：若双方心中对关系的期待三角形与实际感知三角形高度重合，关系满意度就高；反之则产生不满。斯滕伯格以此解释了为何有些伴侣客观条件相似，主观幸福感却差异悬殊。

### 七种爱情类型

根据三个成分的有无组合，斯滕伯格推导出七种爱情类型：

| 类型 | 亲密 | 激情 | 承诺 |
|------|------|------|------|
| 喜欢（Liking） | ✓ | — | — |
| 迷恋（Infatuation） | — | ✓ | — |
| 空洞之爱（Empty Love） | — | — | ✓ |
| 浪漫之爱（Romantic Love） | ✓ | ✓ | — |
| 伴侣之爱（Companionate Love） | ✓ | — | ✓ |
| 愚昧之爱（Fatuous Love） | — | ✓ | ✓ |
| 完美之爱（Consummate Love） | ✓ | ✓ | ✓ |

"空洞之爱"常见于仅剩形式婚姻的关系；"伴侣之爱"则是激情消退后长期婚姻的典型形态；"愚昧之爱"的危险在于双方在缺乏情感亲密的基础上做出了婚姻承诺。

### 依恋风格对爱情模式的塑造

哈赞（Cindy Hazan）与谢弗（Phillip Shaver）于1987年将鲍尔比的依恋理论延伸至成人浪漫关系，提出成人依恋三分法：**安全型（Secure）、焦虑-矛盾型（Anxious-Ambivalent）、回避型（Avoidant）**，后来巴塞洛缪（Bartholomew）于1990年扩展为四象限模型，加入**恐惧-回避型（Fearful-Avoidant）**。

- **安全型**（约占成人人口55%）：对亲密感到自在，能有效调节分离焦虑，倾向于建立高亲密、高承诺的关系。
- **焦虑型**：对被抛弃的恐惧驱动其过度寻求亲密认可，表现为"爱情成瘾"式行为，往往将激情成分放大，误将焦虑激活解读为强烈爱情。
- **回避型**：压抑亲密需求，对承诺产生阻抗，在关系中表现为情感距离感，倾向于形成低亲密度的伴侣之爱形态。

依恋风格并非完全固定。研究显示，与安全型伴侣长期相处可以将不安全依恋者的内部工作模型向安全方向修正，这一过程被称为"关系依恋的习得安全感"（earned security）。

## 实际应用

**关系咨询中的评估工具**：治疗师常使用斯滕伯格1997年开发的《爱情量表》（Triangular Love Scale，共45题）分别测量伴侣双方对亲密、激情、承诺三个维度的评分，将各自的"实际三角"与"理想三角"进行视觉化比对，帮助来访者识别关系中的具体缺口。例如，一方承诺分高而激情分低，另一方恰好相反，则需针对性地讨论双方期待不一致的根源。

**依恋风格的匹配问题**：焦虑型与回避型伴侣的组合在临床上极为常见，被称为"追逐-疏远"循环（pursue-withdraw pattern）——焦虑型越求近，回避型越退缩，形成强化彼此不安全感的负循环。认知行为伴侣治疗（CBCT）和情绪聚焦伴侣治疗（EFT）均将打破这一循环作为干预核心目标。

**激情衰退的正常化教育**：预防性婚姻教育项目（如美国的PREP项目）会告知伴侣，激情在婚后2至4年内下降属于生理正常现象，真正决定婚姻稳定性的是亲密与承诺的维护。这一基于三角理论的心理教育能有效降低伴侣因误判激情衰退为"不爱了"而提出分手的比例。

## 常见误区

**误区一：激情强烈就代表爱得深**
激情成分本质上是生理唤醒的认知归因。达顿（Dutton）与阿龙（Aron）1974年的吊桥实验证明，由恐惧引发的心跳加速也会被错误标记为对邻近异性的爱慕。焦虑型依恋者尤其容易将分离焦虑带来的躯体激活误读为激情强度，从而高估关系的爱情质量。

**误区二：完美之爱是所有关系的目标**
斯滕伯格明确指出，完美之爱难以长期维持，而非每段关系都需要三成分均衡。对于老年伴侣而言，以亲密和承诺为主的"伴侣之爱"在双方都能接受的前提下，完全可以是高满意度的爱情形式。强行追求三成分均等反而可能破坏关系内在的稳定动态。

**误区三：依恋风格是永久不变的宿命**
不少人因了解自己属于回避型或焦虑型后产生宿命感，认为自己"天生不适合爱情"。但哈赞与谢弗的跟踪研究显示，约20%的成人在4年内经历了依恋风格的稳定转变，其中关系质量和个体心理治疗是最强的预测因素。依恋风格是可塑的内部工作模型，而非人格特质。
