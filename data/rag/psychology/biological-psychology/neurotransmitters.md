# 神经递质

## 概述

神经递质（Neurotransmitter）是由神经元合成并储存在突触小泡中的化学物质，当动作电位到达突触前膜时，这些物质被释放到宽约20–40纳米的突触间隙，与突触后膜上的特异性受体结合，将电信号转换为化学信号再转换回电信号，完成神经元之间毫秒级别的信息传递。

神经递质的发现始于1921年，奥地利生理学家奥托·勒维（Otto Loewi）在深夜梦醒后设计了著名的"青蛙双心脏实验"：他用盐溶液灌流刺激迷走神经的心脏，再将灌流液转移到第二颗心脏，发现第二颗心脏随即减慢跳动，证明化学物质（后确认为乙酰胆碱）而非电信号在神经间传递指令。勒维因此与亨利·戴尔（Henry Dale）共同获得1936年诺贝尔生理学或医学奖。这一实验宣告了化学突触传递理论的诞生，并为此后一个世纪的精神药理学研究奠定了根基（参见《生理心理学》，Carlson, 2013，Pearson出版社）。

目前已知的神经递质超过100种，但在心理学与精神医学中最受关注的不超过10种。本文聚焦于多巴胺（DA）、5-羟色胺（5-HT）、γ-氨基丁酸（GABA）、谷氨酸（Glu）和乙酰胆碱（ACh）这五类核心递质，阐明其合成路径、受体类型、功能通路及临床意义。

## 核心原理

### 神经递质的判定标准与化学分类

一种物质被学界正式确认为神经递质，必须同时满足以下五项经典标准（Dale's Principle的延伸版本）：
1. 在突触前神经元内部合成；
2. 储存于突触小泡（直径40–100纳米的膜性结构）中；
3. 动作电位触发突触小泡与前膜融合后被钙离子依赖性地释放；
4. 作用于突触后特异性受体，产生可测量的电位变化或生化反应；
5. 存在明确的失活机制（再摄取、酶解或扩散）。

按化学结构分类，神经递质主要包括四大类：

| 类别 | 代表递质 | 主要功能性质 |
|------|----------|-------------|
| 氨基酸类 | 谷氨酸、GABA、甘氨酸 | 兴奋性或抑制性 |
| 单胺类 | 多巴胺、5-HT、去甲肾上腺素 | 调节性为主 |
| 胆碱类 | 乙酰胆碱 | 兴奋性或抑制性 |
| 肽类 | 内啡肽、P物质、神经肽Y | 调节性、镇痛性 |

氨基酸类递质（谷氨酸和GABA）负责脑内绝大多数"快速"突触传递，约80%以上的突触属于这两类；单胺类递质则更多以"调制"方式作用于广泛脑区，影响全局兴奋水平与行为状态。

### 五种核心递质的功能精解

**① 多巴胺（Dopamine，DA）**

多巴胺由酪氨酸经两步酶促反应合成：酪氨酸在酪氨酸羟化酶（TH）作用下生成左旋多巴（L-DOPA），再经芳香族氨基酸脱羧酶（AADC）脱羧生成多巴胺。帕金森病的标准药物治疗正是补充L-DOPA（而非直接补充多巴胺，因多巴胺无法穿越血脑屏障）。

多巴胺的四条主要投射通路各司其职：
- **中脑边缘通路**（VTA→伏隔核）：编码奖赏预测误差，可卡因、海洛因等成瘾物质均通过此通路使DA浓度提升300%–800%；
- **中脑皮质通路**（VTA→前额叶皮质）：调控工作记忆与执行功能，精神分裂症阴性症状与此通路DA不足相关；
- **黑质纹状体通路**（黑质→纹状体）：控制运动协调，帕金森病患者该通路神经元死亡超过80%才出现明显震颤症状；
- **结节漏斗通路**（下丘脑→垂体）：抑制催乳素分泌，抗精神病药阻断此通路会导致高催乳素血症副作用。

值得注意的是，神经科学家沃尔夫拉姆·舒尔茨（Wolfram Schultz）在1990年代通过猴子单细胞电生理实验揭示：多巴胺神经元并非在"感受快乐"时激活，而是在"实际奖赏与预测奖赏的差值为正"时激活（Schultz, 1997, Science）。这彻底改变了"多巴胺=快乐"的通俗误解。

**② 5-羟色胺（Serotonin，5-HT）**

5-HT由必需氨基酸色氨酸经色氨酸羟化酶-2（TPH2）催化合成，这一步骤是限速步骤，意味着色氨酸的饮食摄入量会影响脑内5-HT合成速率。人体约90%的5-HT存在于消化道肠嗜铬细胞中，脑内的5-HT神经元几乎全部集中于脑干的缝际核（Raphe nuclei），但其轴突分支投射极为广泛，覆盖大脑皮层、边缘系统、小脑等几乎全部脑区。

5-HT受体共有7大家族（5-HT₁至5-HT₇），超过15种亚型，其中5-HT₃受体是唯一的离子通道型受体，其余均为G蛋白偶联受体。选择性5-HT再摄取抑制剂（SSRIs，如氟西汀/百忧解，1987年上市）通过阻断突触前膜上的SERT（5-HT转运体），使突触间隙5-HT浓度升高，是目前全球最广泛使用的抗抑郁药类别。然而，SSRIs起效通常需要2–4周，提示单纯提升5-HT水平并非其完整作用机制，突触后受体的脱敏下调可能是延迟起效的关键原因之一。

**③ GABA（γ-氨基丁酸）**

GABA是中枢神经系统最主要的抑制性递质，约20%–40%的突触使用GABA。它由谷氨酸经谷氨酸脱羧酶（GAD）催化脱羧生成，维生素B₆（吡哆醛磷酸）是GAD的必需辅因子，这解释了严重维生素B₆缺乏为何可能导致惊厥发作。

GABA受体分为两型：
- **GABA-A受体**：配体门控氯离子通道，GABA结合后Cl⁻内流，使细胞膜从静息电位（约–70 mV）进一步超极化至约–75至–80 mV，抑制神经元放电；
- **GABA-B受体**：G蛋白偶联受体，通过减少cAMP及开放钾离子通道产生较慢的抑制效应。

苯二氮䓬类药物（如地西泮，即"安定"）结合于GABA-A受体上α亚基与γ亚基之间的变构位点，自身不能直接开放氯离子通道，而是将GABA的结合亲和力提升2–3倍，属于"正变构调节剂"。巴比妥类药物则结合于不同位点，在高剂量时可直接开放通道，因此安全窗口远窄于苯二氮䓬类，过量风险更高。

**④ 谷氨酸（Glutamate，Glu）**

谷氨酸是脑内占主导地位的兴奋性递质，参与几乎所有的快速兴奋性突触传递。其NMDA受体（N-甲基-D-天冬氨酸受体）因同时具备"电压依赖性"和"配体依赖性"双重门控特性而成为突触可塑性的核心分子：在静息电位下，Mg²⁺堵塞通道孔；只有当突触后膜去极化（通常由AMPA受体介导）且谷氨酸与甘氨酸（共激活配体）同时存在时，Mg²⁺才释放，Ca²⁺大量内流，激活下游CaMKII等信号通路，实现长时程增强（LTP）。LTP被认为是学习与记忆的细胞学基础（Bliss & Lømo, 1973, Journal of Physiology）。

氯胺酮（Ketamine）是NMDA受体非竞争性拮抗剂，在亚麻醉剂量下（0.5 mg/kg静脉注射）可在数小时内产生快速抗抑郁效果，这一发现挑战了以单胺假说为核心的传统抑郁症理论。

**⑤ 乙酰胆碱（Acetylcholine，ACh）**

ACh由胆碱与乙酰辅酶A在胆碱乙酰转移酶（ChAT）催化下合成，释放后被突触间隙中的乙酰胆碱酯酶（AChE）迅速水解为胆碱和乙酸（半衰期不足1毫秒）。阿尔茨海默病的核心病理之一是基底前脑胆碱能神经元（Meynert基底核）的显著丢失，导致皮质ACh水平下降，这正是胆碱酯酶抑制剂（如多奈哌齐）治疗阿尔茨海默病的生化依据。

## 关键公式与突触传递模型

突触后电位的幅度可以用以下简化模型描述：

$$
\Delta V = \frac{N \cdot g_{单个通道} \cdot (V_m - E_{rev})}{C_m}
$$

其中：
- $N$ = 开放的离子通道数量（取决于递质浓度与受体亲和力）
- $g_{单个通道}$ = 单个通道电导（pS级别，约10–50 pS）
- $V_m$ = 当前膜电位（mV）
- $E_{rev}$ = 该离子的反转电位（如Cl⁻约为–70 mV，Na⁺约为+60 mV）
- $C_m$ = 膜电容

以GABA-A受体为例：当 $V_m = -65\text{ mV}$，$E_{rev}(\text{Cl}^-) = -70\text{ mV}$ 时，$V_m - E_{rev} = +5\text{ mV}$，驱动力较小，Cl⁻内流有限，产生轻度超极化；若初始膜电位已因兴奋性输入升高至 $-55\text{ mV}$，则 $V_m - E_{rev} = +15\text{ mV}$，抑制效果更强。这解释了为何GABA的抑制效果具有"状态依赖性"。

```python
# 简单模拟突触后电位叠加（EPSP vs IPSP）
import numpy as np

def psp(t, t0, tau_rise=1, tau_decay=10, amplitude=1):
    """
    双指数模型模拟突触后电位
    t: 时间数组 (ms)
    t0: 递质释放时刻 (ms)
    amplitude: 正值=EPSP（谷氨酸），负值=IPSP（GABA）
    """
    dt = t - t0
    response = np.where(dt > 0,
                        amplitude * (np.exp(-dt/tau_decay) - np.exp(-dt/tau_rise)),
                        0)
    return response

t = np.linspace(0, 100, 1000)
epsp = psp(t, t0=10, tau_rise=0.5, tau_decay=5, amplitude=5)    # 谷氨酸EPSP
ipsp = psp(t, t0=15, tau_rise=1, tau_decay=15, amplitude=-3)    # GABA IPSP
net = epsp + ipsp  # 实际膜电位变化 = 两者叠加
```
