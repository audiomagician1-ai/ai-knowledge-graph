# 神经元结构

## 概述

神经元（neuron）是神经系统的基本功能单位，负责接收、整合和传递电化学信号。人体中约有860亿个神经元（Azevedo et al., 2009），每个神经元可与数千乃至数万个其他神经元形成突触连接，大脑皮层单个锥体细胞（pyramidal cell）平均接收约10,000个突触输入。神经元与肝细胞、肌肉细胞的根本区别在于其高度特化的极性形态：树突负责接收信号、细胞体负责整合信号、轴突负责单向输出信号，三者分工严格，不可互换。

神经元研究史上最关键的转折发生在19世纪末。1891年，德国解剖学家Wilhelm von Waldeyer-Hartz正式提出"神经元"（Neuron）这一术语。西班牙神经科学家Santiago Ramón y Cajal改良应用高尔基银染法（Golgi stain），系统描绘了神经元的精细形态，证明神经系统由独立、不连续的细胞构成，而非当时主流观点所认为的"合胞体网络"，这一"神经元学说"（Neuron Doctrine）奠定了现代神经科学的基础，Cajal因此与Camillo Golgi共同获得1906年诺贝尔生理学或医学奖。

轴突有无髓鞘对信号传导速度影响巨大：有髓鞘纤维的传导速度可达70～120 m/s，而无髓鞘C类纤维仅为0.5～2 m/s，相差60倍以上。这一结构差异直接决定了人体痛觉、触觉、本体感觉的反应速度差异，也是理解多发性硬化症等脱髓鞘疾病机制的关键。

## 神经元的三部分结构

### 细胞体（Soma）

细胞体是神经元的代谢与整合中枢，直径约10～80微米，含细胞核、线粒体、粗面内质网（又称尼氏体，Nissl bodies）等细胞器。尼氏体负责合成轴突末梢所需的神经递质合成酶及膜蛋白，当轴突受损时尼氏体溶解（染色质溶解反应，chromatolysis），是神经损伤的早期组织学标志。细胞体还承担**信号整合**功能：来自数千个树突棘的兴奋性和抑制性突触后电位在此进行时间总和（temporal summation）与空间总和（spatial summation），决定轴突起始段是否达到放电阈值。

### 树突（Dendrites）

树突是从细胞体发出的多个树状分支，其表面密布**树突棘**（dendritic spines），每个树突棘头部直径约0.5～2微米，是兴奋性突触的主要接收位点。大脑皮层锥体细胞的树突总长度可超过10毫米，单个细胞可携带1,000～10,000个树突棘。树突棘的密度和形态（细长型、蘑菇型、短粗型）具有活动依赖性可塑性：长时程增强（LTP）发生后，树突棘体积在30分钟内可增大约200%，这一结构变化被认为是突触级别记忆储存的形态学基础（Bhatt et al., 2009）。

### 轴突（Axon）

轴突是从细胞体轴丘（axon hillock）延伸出的单一突起，长度从不足1毫米（皮层中间神经元）到超过1米（坐骨神经运动轴突）不等。轴丘处的电压门控Na⁺通道密度远高于细胞体其他区域，是动作电位的**起始发生点**。轴突内部由微管（microtubules）和神经丝（neurofilaments）支撑，快速顺向轴浆运输（kinesin驱动，速度约200～400 mm/day）将囊泡、线粒体等物质从细胞体运向末梢；慢速逆向运输（dynein驱动，约50～100 mm/day）则将代谢废物和营养因子信号运回细胞体，破伤风毒素和狂犬病毒均利用这一逆向运输途径侵入中枢神经系统。

## 髓鞘与郎飞结的跳跃式传导

许多轴突外包裹**髓鞘**（myelin sheath），在周围神经系统由施旺细胞（Schwann cells）形成，每个施旺细胞包裹约1毫米轴突；在中枢神经系统由少突胶质细胞（oligodendrocytes）形成，单个少突胶质细胞可同时包裹多达50条轴突片段。髓鞘由约70%脂质和30%蛋白质组成，绝缘性极高，使跨膜离子电流几乎不能在髓鞘覆盖处通过。

每隔约1～2毫米，髓鞘中断处形成**郎飞结**（nodes of Ranvier，由法国解剖学家Louis-Antoine Ranvier于1878年命名），结处宽度仅约1微米，但Na⁺通道密度高达约1,000～2,000个/μm²，远高于无髓鞘轴突膜的约10个/μm²。动作电位以"**跳跃式传导**"（saltatory conduction，源自拉丁语saltare，意为"跳跃"）方式在郎飞结间传递，不仅速度快（最高120 m/s），且能量消耗仅为连续传导的约1/100，因为Na⁺-K⁺泵只需在郎飞结处恢复离子梯度。

多发性硬化症（multiple sclerosis, MS）是自身免疫系统攻击中枢髓鞘碱性蛋白（MBP）和蛋白脂蛋白（PLP），导致脱髓鞘斑块形成，使受累轴突传导速度骤降甚至传导阻滞，临床表现为视力障碍、肢体无力和感觉异常。

## 动作电位的产生机制与关键公式

### 静息电位与离子分布

静息状态下，神经元膜内外存在约 $-70\ \text{mV}$ 的电位差（膜内相对膜外为负），称为**静息电位**（resting potential）。这一电位由三个因素共同维持：

1. K⁺的平衡电位约为 $-90\ \text{mV}$（K⁺倾向外流）；
2. Na⁺的平衡电位约为 $+60\ \text{mV}$（Na⁺倾向内流）；
3. Na⁺-K⁺泵每次消耗1个ATP，泵出3个Na⁺、泵入2个K⁺，维持离子浓度梯度。

离子平衡电位可由**能斯特方程**（Nernst equation）计算：

$$E_{ion} = \frac{RT}{zF} \ln \frac{[ion]_{out}}{[ion]_{in}}$$

其中 $R$ 为气体常数（8.314 J·mol⁻¹·K⁻¹），$T$ 为绝对温度（体温约310 K），$z$ 为离子价数，$F$ 为法拉第常数（96,485 C·mol⁻¹）。在37°C下，对于K⁺（$z=+1$，$[\text{K}^+]_{out}/[\text{K}^+]_{in} \approx 1/30$），计算结果约为 $-91\ \text{mV}$，与实测值吻合良好。

### 动作电位的时相

当轴丘膜电位由 $-70\ \text{mV}$ 上升至**阈值**（约 $-55\ \text{mV}$）时，电压门控Na⁺通道（voltage-gated Na⁺ channels）大量开放，引发正反馈的"全或无"（all-or-none）去极化：

- **去极化相**（0～1 ms）：Na⁺大量内流，膜电位在约0.5 ms内冲至约 $+40\ \text{mV}$；
- **复极化相**（1～2 ms）：Na⁺通道失活（inactivation gate关闭），电压门控K⁺通道开放，K⁺外流使膜电位迅速回落；
- **超极化相/后电位**（2～3 ms）：K⁺通道缓慢关闭，膜电位短暂降至约 $-80\ \text{mV}$，随后恢复至 $-70\ \text{mV}$。

**绝对不应期**（absolute refractory period）约为1～2 ms，此时Na⁺通道处于失活状态，任何刺激均无法触发新的动作电位，这一特性保证了动作电位的单向传导，不会向已激活区域回流。

```python
# 简化版 Hodgkin-Huxley 动作电位模拟（概念示意）
import numpy as np

dt = 0.01  # 时间步长 (ms)
t = np.arange(0, 40, dt)
V = -70.0   # 初始膜电位 (mV)
threshold = -55.0  # 阈值

for i, time in enumerate(t):
    if time == 5.0:   # 在5ms施加刺激
        V += 20       # 模拟外部电流注入
    if V >= threshold:
        print(f"t={time:.2f}ms: 动作电位触发！膜电位={V:.1f}mV")
        V = 40        # 峰值
        # 后续复极化由K⁺通道开放完成
        break
```

## 突触传递：从电信号到化学信号

动作电位到达轴突终扣后，触发**突触传递**（synaptic transmission）过程，完成从电信号到化学信号再回到电信号的转换：

1. 动作电位去极化激活终扣膜上的**电压门控Ca²⁺通道**，Ca²⁺从胞外（浓度约2 mM）流入胞内（约100 nM），浓度梯度高达20,000倍；
2. 升高的胞内Ca²⁺与突触囊泡膜上的**突触结合蛋白**（synaptotagmin）结合，触发SNARE蛋白复合体介导的囊泡与突触前膜融合，以**胞吐**（exocytosis）方式将神经递质释放入突触间隙（宽度约20～50 nm）；
3. 神经递质扩散并与**突触后受体**结合，引发突触后膜的兴奋性（EPSP，幅度约0.5～1 mV）或抑制性（IPSP，幅度约−0.5 mV）突触后电位；
4. 递质通过**再摄取**（reuptake，如多巴胺转运体DAT）、酶解（如乙酰胆碱酯酶在0.5 ms内水解乙酰胆碱）或扩散失活。

例如，在神经肌肉接头处，一次动作电位可导致约200个突触囊泡释放，每个囊泡含约5,000～10,000个乙酰胆碱分子，总计约100万个分子同时涌入突触间隙，使终板电位幅度高达约70 mV，远超动作电位阈值，确保每次神经冲动都能引发肌肉收缩。

## 神经元的分类

按形态，神经元可分为：**多极神经元**（multipolar，如脊髓运动神经元，有多个树突和一个轴突）、**双极神经元**（bipolar，如视网膜双极细胞）、**假单极神经元**（pseudounipolar，如背根神经节感觉神经元，轴突呈T形分叉）。

按功能，神经元分为：**感觉神经元**（将外周刺激传入中枢）、**运动神经元**（将中枢指令传出至效应器）、**中间神经元/联络神经元**（占大脑神经元总数的99%以上，负责皮层内信息整合）。

按递质类型，神经元可分为谷氨酸能（兴奋性，约75%皮层突触）、GABA能（抑制性，约25%皮层突触）、多巴胺能、血清素能、乙酰胆碱能等类型，不同递质类型与不同的神经回路功能直接对应。
