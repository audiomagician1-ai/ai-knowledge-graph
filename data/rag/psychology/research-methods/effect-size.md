# 效应量

## 概述

效应量（Effect Size）是量化两组差异或两变量关联**实际大小**的统计指标，独立于样本量存在。与p值只能回答"差异是否存在"不同，效应量回答的是"差异有多大"——即使p < 0.001，差异也可能小到毫无实践意义；而效应量能直接告诉研究者这个差异值不值得重视。

效应量概念的系统化源于Jacob Cohen在1962年对心理学期刊的检验力分析。他发现当时大量研究的统计检验力严重不足（中位数约0.46），部分原因正是研究者从未估计期望效应量的大小。Cohen随后在1969年出版《社会行为科学的统计检验力分析》，正式提出了d、f、w等一系列效应量指标及其"小、中、大"的参考基准，奠定了现代效应量研究的基础。

2010年后，美国心理学会（APA）第六版出版手册明确要求所有实证研究**必须报告效应量**，这使效应量从可选统计量变为强制报告项目。其重要性在于：样本量越大，越微小的真实差异都能达到统计显著，此时只有效应量才能区分"统计上显著"与"实际上有意义"的差别。

## 核心原理

### Cohen's d：均值差异的标准化度量

Cohen's d 专用于比较两组均值差异，公式为：

$$d = \frac{\mu_1 - \mu_2}{\sigma_{pooled}}$$

其中 $\mu_1$、$\mu_2$ 为两组均值，$\sigma_{pooled}$ 为合并标准差（$\sigma_{pooled} = \sqrt{\frac{(n_1-1)s_1^2 + (n_2-1)s_2^2}{n_1+n_2-2}}$）。d值表示两组均值相差多少个标准差单位。Cohen（1988）提出的经验基准为：d = 0.2 为小效应，d = 0.5 为中效应，d = 0.8 为大效应。以新药研究为例，若治疗组与对照组疼痛评分差异的d = 0.15，即使n = 2000时p < 0.001，临床医生仍可判断该药效微乎其微。

### 相关系数r作为效应量

皮尔逊相关系数r本身就是一种效应量，直接反映两连续变量线性关联的强度。Cohen的基准：r = 0.1（小），r = 0.3（中），r = 0.5（大）。r²（决定系数）表示一个变量能解释另一变量方差的百分比——r = 0.3意味着仅解释9%的方差，这提醒研究者即便相关显著，实际预测力可能极为有限。r与d之间可以相互转换，公式为：$r = \frac{d}{\sqrt{d^2 + 4}}$，当d = 0.5时对应r ≈ 0.243。

### η²与ω²：方差分析中的效应量

在单因素或多因素方差分析（ANOVA）中，η²（eta-squared）= SS效应 / SS总，表示自变量解释因变量方差的比例。Cohen基准：η² = 0.01（小），0.06（中），0.14（大）。但η²存在高估偏差，尤其在小样本中，因此推荐使用偏差校正版ω²（omega-squared）：$\omega^2 = \frac{SS_{效应} - df_{效应} \cdot MS_{误差}}{SS_{总} + MS_{误差}}$，ω²的值通常比η²低0.01–0.05，更准确反映总体真实效应。

### 效应量与统计检验力的关系

检验力（1-β）随效应量增大而提升。在显著性水平α = 0.05、双尾检验、中效应d = 0.5条件下，达到0.80检验力需要每组约64名被试；若期望效应为小效应d = 0.2，则需要每组约394名被试。这说明效应量估计是样本量规划（power analysis）的必要前提——在收集数据前预估效应量，避免研究因样本不足而错过真实效应。

## 实际应用

**教育心理学干预评估**：一项元分析综合了180项课堂干预研究，平均效应量d = 0.40，属于中小效应。研究者据此结论：该干预有统计支持但实践价值有限，每花费1000美元只能将学生成绩提升约0.4个标准差，与其他已知有效干预（如反馈练习d ≈ 0.73）相比性价比偏低。

**临床心理学中的诊断意义**：抑郁症CBT治疗的效应量d通常在0.6–0.8之间（中到大效应）。研究者若仅报告p < 0.05，无法判断治疗是否达到临床意义的改善门槛（通常定义为症状量表改分≥50%）。效应量配合可靠改变指数（RCI）才能完整描述干预效果。

**人格心理学中的r值解读**：外向性与工作绩效的元分析相关约r = 0.15（即r² = 0.0225），仅解释约2.25%的绩效方差。尽管此相关在大样本（n > 1000）时高度显著（p < 0.001），但对于实际人才选拔，单独使用外向性测试的预测价值极为有限，效应量揭示了统计显著与实用价值之间的鸿沟。

## 常见误区

**误区一：以为效应量越大越好，忽视研究背景**。Cohen自己明确指出，0.2/0.5/0.8的基准仅是"没有更好参考时的最后手段"，不同研究领域的合理效应量差异巨大。在基因与复杂行为的关联研究中，单一SNP位点解释行为方差常低于0.5%（r < 0.07），但这仍是该领域有意义的发现。盲目套用"小效应无意义"会错误否定重要结果。

**误区二：把效应量等同于p值的补充，仍以显著性为主**。部分研究者计算效应量只为满足期刊要求，仍以"p < 0.05且d = 0.22，说明效果显著"来下结论。实际上，d = 0.22属于小效应，应反过来讨论：虽然统计显著，但实际差异微小，是否值得推广该干预需要成本效益分析。效应量应成为结论的**主要依据**而非注脚。

**误区三：独立样本d公式直接用于配对设计**。配对t检验中的效应量应使用 $d_z = \frac{\bar{D}}{s_D}$（差值均值除以差值标准差），而非独立样本的合并标准差公式。若错用独立样本d公式，配对设计的效应量会被严重低估——因为配对设计消除了个体差异，其实际效率远高于等量独立样本设计。

## 知识关联

效应量建立在推断统计的基础上——理解p值的局限性（p值受样本量影响但效应量不受）才能体会效应量存在的必要性。掌握标准差和方差分析中的SS分解是计算Cohen's d和η²的前提，因为这两个效应量指标本质上都是对变异来源的标准化。

效应量是元分析（meta-analysis）的核心输入量：不同研究使用不同量表测量同一构念，只有将各研究结果转化为统一的效应量（通常是d或r），才能跨研究进行量化综合。此外，效应量直接决定研究设计阶段的样本量计算（power analysis），在收集数据前通过文献确定期望效应量大小，是规避I型和II型错误的关键步骤。
