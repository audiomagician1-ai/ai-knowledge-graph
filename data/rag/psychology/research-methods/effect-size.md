# 效应量

## 概述

效应量（Effect Size）是量化两组数据差异程度或变量间关联强度的统计指标，它回答的问题是"差异有多大"，而非"差异是否存在"。与p值只告诉我们某个结果偶然出现的概率不同，效应量提供了一个独立于样本量的、反映现象本身大小的数值。

效应量的系统化应用始于1969年，统计学家雅各布·科恩（Jacob Cohen）在其著作《行为科学的统计功效分析》中首次提出了 Cohen's d 的标准化解释框架，并在1988年的第二版中进一步完善。此前，研究者长期依赖p值判断结论，科恩指出这导致大量研究混淆了"统计显著"与"实际有意义"这两个根本不同的概念。

效应量在心理学研究中至关重要，因为心理学实验常常面临样本量较大时p值几乎必然显著的困境。例如，一项n=10,000的调查发现某干预使抑郁分数降低0.3分（满分100），p<0.001，但效应量d=0.02，意味着该差异在实践中几乎无意义。效应量帮助研究者和读者识别这类情况。

## 核心原理

### Cohen's d：均值差异的标准化测量

Cohen's d 是最常用于比较两组均值的效应量指标，其计算公式为：

$$d = \frac{\mu_1 - \mu_2}{SD_{pooled}}$$

其中 $\mu_1$ 与 $\mu_2$ 分别为两组均值，$SD_{pooled}$ 为合并标准差（pooled standard deviation），计算方式为：

$$SD_{pooled} = \sqrt{\frac{(n_1-1)SD_1^2 + (n_2-1)SD_2^2}{n_1+n_2-2}}$$

科恩在1988年为 d 提出了被广泛引用的经验性判断标准：**d = 0.2 为小效应，d = 0.5 为中等效应，d = 0.8 为大效应**。以认知训练研究为例，如果训练组的智力测验均分比控制组高出0.8个标准差，即 d = 0.8，说明训练效果相当显著。值得注意的是，科恩本人强调这些阈值仅为参考，不同研究领域应建立自己的基准。

### 相关系数 r 作为效应量

皮尔逊相关系数 r 本身即为一种效应量，用于描述两个连续变量之间线性关系的强度与方向，取值范围为 −1 到 +1。科恩同样提供了 r 的判断标准：**r = 0.1 为小效应，r = 0.3 为中等效应，r = 0.5 为大效应**。

r 与 Cohen's d 之间存在数学转换关系：

$$r = \frac{d}{\sqrt{d^2 + 4}}$$

这个公式使研究者能够在元分析中统一比较使用不同设计的研究结果。r² 又称决定系数，代表一个变量能被另一个变量解释的方差比例——例如 r = 0.3 意味着 X 能解释 Y 约9%的方差（0.3² = 0.09）。

### η²（Eta Squared）与方差分析中的效应量

当研究设计包含两个以上组别（即使用方差分析ANOVA时），效应量通常用 η²（eta squared）表示，计算公式为：

$$\eta^2 = \frac{SS_{between}}{SS_{total}}$$

其中 $SS_{between}$ 为组间平方和，$SS_{total}$ 为总平方和。科恩的标准为：**η² = 0.01 为小效应，η² = 0.06 为中效应，η² = 0.14 为大效应**。η² 存在高估总体效应量的倾向（尤其在小样本时），因此更推荐使用偏η²（partial η²）或ω²（omega squared）作为校正版本。

## 实际应用

**临床心理学疗效评估**：在对认知行为疗法（CBT）治疗抑郁症的元分析中，Cuijpers等人（2019年）报告的平均 d 约为0.56，属于中等效应，说明CBT疗效在统计上可靠且临床上有实际意义。这一结论无法仅从p值中得出。

**教育心理学中的学习干预**：John Hattie在其2009年的综合研究《可见学习》中汇集了超过800个元分析，以效应量d为统一标准比较不同教学策略。他发现反馈（Feedback）的效应量 d = 0.73，属大效应；而班级规模缩减的效应量仅 d = 0.21，属小效应。这为教育政策提供了基于数据的优先级排序依据。

**测量敏感性场景**：在人格心理学研究中，测量神经质与焦虑的相关性时，r 通常在0.4至0.6之间。研究者在报告结果时同时呈现 r 值与95%置信区间，使读者既能了解效应大小，又能了解估计的精确程度。

## 常见误区

**误区一：认为效应量越大越好，或认为小效应无意义**
效应量的"大"与"小"必须结合研究背景判断。在公共卫生领域，一种预防措施的效应量 d = 0.1 看似微小，但若该措施面向全国数亿人口，其实际影响极为巨大。阿司匹林预防心肌梗死的效应量仅 r = 0.034，却被视为医学上的重要发现。效应量判断标准不能脱离具体情境。

**误区二：混淆统计显著性与效应量**
p < 0.05 只表明在假设零效应的情况下，观察到当前或更极端结果的概率低于5%，它并不反映差异的大小。样本量加倍可以使同一个 d = 0.1 的微小效应从 p = 0.20 变为 p = 0.001。换句话说，p值是样本量与效应量共同作用的结果，不能单独代表效应量的信息。

**误区三：将 Cohen's d 的固定阈值机械套用**
许多学生认为只要 d ≥ 0.8 就可以声称"大效应"。然而科恩明确指出，这些数值基于行为科学研究的历史均值，不适用于所有领域。医学研究中 d = 0.3 可能已是重大突破，而感知心理物理实验中 d = 2.0 才算普通结果。不同子领域应优先参考本领域已发表研究的效应量分布。

## 知识关联

效应量的理解建立在**推断统计**基础上：只有理解了p值、抽样分布与假设检验逻辑，才能准确理解为何效应量能弥补p值的局限性——p值受制于样本量，效应量则不受其影响。

效应量还是**统计功效（Statistical Power）**分析的核心输入变量。计算研究所需样本量时，研究者需要预先设定期望效应量（通常用先前文献中的 d 或 r 估计）、显著性水平α（通常0.05）和功效（通常0.80），四者之间存在固定的数学关系：缩小期望效应量就必须增大样本量来维持相同功效。

在**元分析**（Meta-Analysis）领域，效应量是整合不同研究结果的通用货币。不同研究使用的测量工具、量表和样本各异，但转换为共同的效应量指标（如 d 或 r）后，即可在同一框架下加权平均，得出跨研究的综合估计。这正是效应量报告成为APA（美国心理学会）自2001年起要求的统计报告规范的根本原因。
