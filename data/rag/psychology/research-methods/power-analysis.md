# 统计功效分析

## 概述

统计功效分析（Statistical Power Analysis）是研究者在设计实验或评估已有研究时，用于确定样本量与检验灵敏度之间定量关系的方法体系。其核心参数包括四个相互制约的量：统计功效（Power，记作1−β）、显著性水平（α）、效应量（Effect Size，如Cohen's d）、以及样本量（N）。这四个量中任意三个已知，即可推算第四个。

该方法由雅各布·科恩（Jacob Cohen）于1962年系统化，他在分析《异常与社会心理学杂志》发表的70余篇论文时，发现当时心理学研究的平均统计功效仅约0.46，即超过一半的真实效应无法被检测到。1988年，科恩出版了奠基性著作《社会科学行为研究的统计功效分析》（第二版），为心理学、教育学、医学等领域提供了系统的功效计算工具与效应量参考标准。

功效分析的实践意义在于两个方向的应用：**前瞻性分析**（a priori analysis）在研究开始前确定所需样本量，避免检验力不足（underpowered）或浪费资源的过大样本；**事后分析**（post hoc analysis）则解释已有研究"未发现显著结果"究竟是真正的零效应，还是样本不足导致的假阴性。APA出版手册从第6版起明确要求研究者报告效应量，正是功效分析理念推动的规范变革。

## 核心原理

### 四参数与两类错误的关系

统计功效分析建立在对两类错误的精确控制上。**Type I错误（α错误）**指虚无假设H₀为真时却拒绝它，概率为α，通常设为0.05。**Type II错误（β错误）**指H₀为假时却未能拒绝它，概率为β。统计功效定义为**1−β**，即在真实效应存在时正确拒绝H₀的概率。

科恩建议将心理学研究的功效标准设为**0.80**，即β≤0.20，意味着研究者接受"错过真实效应的风险"是"错误宣称存在效应风险"的4倍（0.20 vs 0.05）。这一比值并非凭空规定，而是科恩基于"发现真实效应比避免误报重要约4倍"的实践判断所设。若研究者要求更严格的功效（如0.90或0.95），所需样本量将显著增加。

四参数之间存在单调关系：固定α和效应量不变，增大N将提升功效；固定N和效应量，降低α（如从0.05改为0.01）将降低功效；效应量越大，达到相同功效所需的N越小。

### 效应量：功效计算的核心输入

效应量是功效分析中最容易被忽视却最关键的参数。对于独立样本t检验，Cohen's d定义为：

$$d = \frac{\mu_1 - \mu_2}{\sigma_{pooled}}$$

其中σ_pooled为两组合并标准差。科恩制定了约定俗成的参考标准：**d = 0.2**为小效应，**d = 0.5**为中等效应，**d = 0.8**为大效应。

对于方差分析（ANOVA），常用效应量为**η²**或**f**，科恩定义f = 0.10（小）、f = 0.25（中）、f = 0.40（大）。对于相关分析，效应量直接使用皮尔逊r，临界值为r = 0.10、0.30、0.50。

研究者确定效应量的三种途径是：参考既往元分析报告的平均效应量、使用领域内"可接受的最小效应量"（smallest effect size of interest, SESOI）、或基于理论推导的期望差值。随意套用"中等效应量"而不给出依据，是当前功效分析报告中最常见的方法论缺陷。

### 样本量计算的实操逻辑

以双样本t检验为例，在α = 0.05（双侧）、功效 = 0.80、d = 0.50（中等效应）的标准条件下，每组所需样本量约为**64人**（共128人）。若将效应量缩小至d = 0.20，同等条件下每组需要**197人**，样本量需求增至约3倍。

G\*Power是心理学领域最广泛使用的免费功效分析软件，由Franz Faul等人开发，支持超过50种统计检验的功效计算，包括t检验、F检验、χ²检验、回归分析等。使用时需选择"Test family"（如t tests）和"Statistical test"（如Means: Difference between two independent means），输入效应量d、α及功效目标，软件即输出每组所需N及对应的功效曲线。

## 实际应用

**临床心理学干预研究**：研究者计划评估认知行为疗法（CBT）对抑郁症的疗效，参考已有元分析（如Hofmann等2012年综述）得知CBT的效应量约为d = 0.67。设α = 0.05（双侧）、功效 = 0.80，通过G\*Power计算每组需36人，双组共72人。若预计脱落率为15%，实际招募应为每组约42人。

**发表偏倚的诊断**：研究者重新分析某研究领域的50篇文章，发现其中40篇报告了显著结果（p < 0.05），而基于该领域典型效应量计算的平均功效仅为0.35。若功效仅0.35，理论上50篇中预期只有约17.5篇（50×0.35）能出现显著结果，而实际有40篇，这一显著差异提示存在严重发表偏倚或p-hacking。

**课程教学研究中的小效应**：教育心理学研究常发现d ≈ 0.20的干预效应。若不进行功效分析而随意用30人/组，此时功效仅约0.12，即88%的概率会"发现"不了真实存在的效应，导致错误结论认为干预无效。

## 常见误区

**误区一：事后功效分析可以解释"不显著"结果**
许多研究者在得到p > 0.05后，用观测到的效应量进行事后功效计算，并声称"功效不足，结论待定"。然而"观测效应量"并不等于"真实效应量"，用它计算的事后功效与p值存在数学上的一一对应关系（功效高恰好对应p值低），因此这种做法是循环论证，毫无增量信息。正确做法是报告置信区间，判断置信区间是否排除了有意义的效应。

**误区二：功效0.80是必须达到的固定标准**
科恩的0.80建议是历史经验性准则，并非统计定律。在探索性研究中，研究者可能接受更低的功效（如0.60）以节省资源，在医疗决策或政策性研究中则可能要求功效达0.90甚至0.95。功效目标应与研究的决策后果挂钩：错过一个真实效应的代价越大，要求的功效应越高。

**误区三：样本量越大功效一定越好**
统计功效随样本量增大单调递增是正确的，但极大样本会使微小、无实践意义的效应达到统计显著。例如N = 10000时，d = 0.05（极小效应）的功效可能超过0.99，此时p < 0.05的结论在统计上成立但实践上毫无价值。因此，功效分析必须与效应量的实践意义判断结合使用，而非单纯追求p值的显著性。

## 知识关联

统计功效分析以**推断统计**为基础，直接运用假设检验的零假设逻辑、p值计算和置信区间概念。理解α的含义（显著性水平即允许的Type I错误率）是进入功效分析的前提，同时需要掌握不同检验统计量（t、F、χ²）的分布特性，因为非中心参数（non-centrality parameter）是各类检验功效计算的数学核心。

功效分析与**效应量**的概念密不可分：Cohen's d、η²、f、r这些效应量不仅是功效计算的输入，也是近年来心理学开放科学运动（如预注册、复现危机讨论）中强调"超越p值"的核心指标。许多研究者将功效分析与**贝叶斯因子分析**相对比，后者不区分α和β，而是直接量化证据支持H₀还是H₁的相对强度，是频率论功效分析的重要替代框架。
