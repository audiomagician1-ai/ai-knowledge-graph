# 双相障碍

## 概述

双相障碍（Bipolar Disorder，BD）是一种以躁狂发作或轻躁狂发作与重性抑郁发作交替出现为核心特征的慢性、复发性精神疾病。全球终生患病率约为2.4%，其中双相I型约为0.6%，双相II型约为0.4%（Merikangas et al., 2011，《美国精神病学杂志》）。该病的首次发作平均年龄为20岁，从首次出现症状到获得准确诊断，患者平均需要等待**8至10年**——这段漫长的延误，主要源于双相II型患者的轻躁狂期往往被当事人体验为"状态极好"，而非病态。

历史溯源上，德国精神病学家卡尔·路德维希·卡尔鲍姆（Karl Ludwig Kahlbaum）于1882年首次描述"循环性精神病"（Zyklisches Irresein），埃米尔·克雷佩林（Emil Kraepelin）在1899年出版的《精神病学》第六版中将其系统化为"躁狂-抑郁性精神病"，并首次提出情绪发作与间歇期的周期性概念。1980年，《精神障碍诊断与统计手册》第三版（DSM-III）将双相I型与双相II型正式分离，确立了现代诊断框架。

双相障碍的误诊后果具有直接的临床危险性：若患者被错误诊断为单纯抑郁症并单独使用抗抑郁药（如SSRIs），有约20%至40%的概率诱发躁狂发作或加速情绪循环（转相风险），使病情变得更难控制。这一数字赋予了精确鉴别诊断极高的医疗优先级。

## 核心原理

### 发作类型与DSM-5诊断标准

双相障碍依据发作严重程度分为以下亚型：

**双相I型**：要求至少一次完整躁狂发作，持续时间≥7天（若因严重到需住院则不限天数）。躁狂发作的核心症状包括：情感高涨或易激惹、夸大感、睡眠需求减少（区别于失眠——患者不觉得疲劳）、思维奔逸（flight of ideas）、注意力分散、目标导向活动增加、冲动行为（如挥霍消费、性行为不检点）。严重时可出现与心境协调或不协调的精神病性症状（妄想、幻觉）。

**双相II型**：以轻躁狂发作（持续≥4天，功能损害相对轻微，未达到需住院的程度）加上至少一次重性抑郁发作为诊断标准。其误诊率高达**40%以上**，因为患者主动就诊时几乎都处于抑郁期，对轻躁狂期的记忆往往描述为"那段时间效率特别高"。

**环性心境障碍**：在连续2年内（儿童青少年1年）存在多次轻躁狂症状与抑郁症状，但任何单次发作均未达到完整的轻躁狂或重性抑郁的标准。

**混合特征**：DSM-5中以"混合特征标注"替代了DSM-IV的"混合发作"，指在一次主要为躁狂或抑郁的发作中，同时存在对立相位的至少3条症状。混合状态下患者的**自杀风险达到所有发作类型中的最高点**，临床需高度警惕。

### 神经生物学基础

**神经递质层面**：双相障碍的发作相位与不同神经递质系统的功能状态密切对应。躁狂发作期间，多巴胺（DA）系统呈现过度激活状态——纹状体DA释放增加，D2/D3受体的占用率上升；这与成瘾和奖赏寻求行为的神经机制高度重叠，解释了躁狂期患者的冲动性和快感驱动行为。抑郁发作期间，5-羟色胺（5-HT）与去甲肾上腺素（NE）系统的功能性不足更为突出。谷氨酸/GABA的失衡在两相转换的动态过程中被认为起到"开关"作用。

**神经回路层面**：前额叶皮质（PFC）对杏仁核的自上而下抑制控制在双相障碍患者中存在结构和功能双重损害。功能性磁共振成像（fMRI）研究发现，躁狂期患者杏仁核对情绪图片的激活幅度比健康对照高出约**30%至50%**，而左侧背外侧前额叶皮质（dlPFC）的灰质体积比对照组平均减少**5%至10%**（Phillips & Swartz, 2014，《神经科学自然评论》）。这一结构性损伤在确诊前的高风险个体中已可检测到，提示其为特质性标志而非状态性改变。

**遗传与分子层面**：双相障碍的遗传度估计高达**60%至80%**，一级亲属（父母、兄弟姐妹）的终生患病风险是普通人群的**8至10倍**。大规模全基因组关联研究（GWAS）已识别出多个与双相障碍显著相关的基因变异，其中最具重复性的包括：

- **ANK3**（锚蛋白3基因）：参与神经元轴突初始段的电压门控钠通道组装，调控动作电位的发放阈值。
- **CACNA1C**（L型电压门控钙通道α1C亚基基因）：直接影响神经元内Ca²⁺内流量，与情绪周期性调节密切相关。
- **ODZ4**、**SYNE1**：参与神经元发育与突触可塑性。

这些基因均指向神经元兴奋性调控这一核心病理通路，与锂盐等情绪稳定剂的作用靶点在机制上互相呼应。

### 昼夜节律紊乱与生物钟假说

昼夜节律（Circadian Rhythm）紊乱是双相障碍中被最广泛复制的生物标志之一。患者存在CLOCK基因（Circadian Locomotor Output Cycles Kaput）表达异常，褪黑素分泌高峰延迟（相位延迟型更为常见），睡眠-觉醒周期不稳定。

关键临床证据：**睡眠剥夺本身可以在约60%的双相I型易感个体中诱发躁狂发作**，且这一过程可在24至48小时内完成。跨时区飞行（尤其是向东飞行导致昼夜节律提前）、换班工作和产后睡眠剥夺均是临床记录在案的躁狂发作触发因素。

锂盐（Li⁺）的部分作用机制正是通过抑制**GSK-3β**（糖原合成酶激酶-3β）来稳定生物节律。GSK-3β在昼夜节律分子环路中磷酸化CLOCK蛋白的降解靶点，Li⁺的抑制作用使PERIOD蛋白的降解减慢，从而延长并稳定昼夜节律周期。这一分子通路连接了锂盐的情绪稳定效果与生物钟调节功能，形成了"双相障碍本质上是昼夜节律病"这一理论框架的核心证据。

## 关键公式与量化评估工具

在临床评估中，躁狂严重程度最常使用**Young躁狂评定量表（YMRS）**，共11个条目，总分0至60分，≥20分提示有临床意义的躁狂发作。抑郁相评估则常用**汉密顿抑郁量表（HDRS-17）**或**蒙哥马利-阿斯伯格抑郁评定量表（MADRS）**。

锂盐的治疗窗以血清锂浓度（$[Li^+]_{serum}$）为核心监测指标：

$$
[Li^+]_{治疗范围} = \begin{cases} 0.8 \sim 1.2 \text{ mmol/L} & \text{（急性躁狂发作期）} \\ 0.6 \sim 0.8 \text{ mmol/L} & \text{（维持/预防期）} \\ > 1.5 \text{ mmol/L} & \text{（中毒警戒线）} \end{cases}
$$

锂盐中毒的早期症状包括细颤、多尿、恶心；血锂浓度超过2.0 mmol/L时出现粗颤、共济失调、意识模糊；超过3.0 mmol/L可危及生命，需紧急血液透析。治疗窗如此狭窄，要求临床每3至6个月监测一次血锂浓度，并同步评估肾小球滤过率（eGFR）和促甲状腺激素（TSH），因为长期使用锂盐有20%至40%的患者会出现亚临床甲状腺功能减退。

```python
# 示例：根据血锂浓度判断临床状态的简单分类函数
def lithium_status(serum_li: float, phase: str = "maintenance") -> str:
    """
    serum_li: 血清锂浓度 (mmol/L)
    phase: 'acute' (急性躁狂) 或 'maintenance' (维持期)
    """
    if serum_li < 0.4:
        return "浓度不足，治疗效果可能不佳"
    elif phase == "acute" and 0.8 <= serum_li <= 1.2:
        return "急性期治疗窗内，继续当前剂量"
    elif phase == "maintenance" and 0.6 <= serum_li <= 0.8:
        return "维持期治疗窗内，继续当前剂量"
    elif 1.2 < serum_li <= 1.5:
        return "偏高，建议复查并评估肾功能"
    elif serum_li > 1.5:
        return "⚠️ 中毒警戒——立即停药并就医"
    else:
        return "浓度偏低，需结合临床症状评估是否调整剂量"
```

## 实际应用

### 药物治疗

**情绪稳定剂**是双相障碍药物治疗的核心支柱。锂盐（碳酸锂）自1949年澳大利亚精神病学家约翰·凯德（John Cade）首次报告其抗躁狂效果（Cade, 1949，《澳大利亚医学杂志》）以来，历经75年仍是一线选择，也是目前唯一具有确切**抗自杀**循证证据的情绪稳定剂——荟萃分析显示，长期锂盐治疗使双相患者的自杀及自杀未遂风险降低约**60%**。

对于不耐受锂盐的患者，**丙戊酸盐**（valproate，血药浓度目标50至100 μg/mL）和**拉莫三嗪**（lamotrigine，对双相抑郁相预防效果尤为突出）是主要替代选择。拉莫三嗪通过阻断电压门控Na⁺通道，抑制谷氨酸过度释放，在双相II型抑郁的长期预防中优于安慰剂，但其起效需要缓慢滴定（初始剂量25 mg/日，每2周递增，避免严重皮疹Stevens-Johnson综合征，发生率约0.1%）。

**非典型抗精神病药**（如喹硫平、奥氮平、阿立哌唑）在急性躁狂和双相抑郁的短期控制中均有FDA批准适应症，喹硫平是目前唯一对双相抑郁急性发作具有A级推荐证据的非情绪稳定剂类药物。

### 心理社会治疗

药物之外，**人际与社会节律治疗（Interpersonal and Social Rhythm Therapy，IPSRT）**是双相障碍循证等级最高的心理治疗方法之一（Frank et al., 2005）。其理论基础正来自昼夜节律失调假说：治疗师帮助患者通过"社会节律图"（Social Rhythm Metric）追踪每日起床时间、首次社交互动、正餐、工作/活动和就寝时间，目标是将这5项行为的标准差压缩至最小，以稳定内在生物钟、预防情绪发作。

此外，**心理教育**（psychoeducation）作为团体或个体形式的干预，已被随机对照试验证实可将1年内的复发率降低约**30%至40%**（
