# 进食障碍

## 概述

进食障碍（Eating Disorders）是一组以异常进食行为和对体重、体形的过度关注为核心特征的精神障碍，主要包括神经性厌食症（Anorexia Nervosa, AN）、神经性贪食症（Bulimia Nervosa, BN）与暴食障碍（Binge Eating Disorder, BED）三种主要类型。这些障碍不仅造成严重的心理痛苦，还引发多系统躯体并发症。神经性厌食症的标准化死亡率（SMR）约为5.86，在所有精神障碍中位居首位，主要死因为心脏骤停（源于低钾血症诱发的QTc间期延长）和自杀（约占死亡案例的20%）（Arcelus et al., 2011）。

进食障碍的系统性临床描述始于19世纪。英国医生威廉·古尔（William Gull）于1873年在伦敦临床协会的演讲中首次以"神经性厌食症"命名极度限制进食的临床现象，同年法国医生拉塞格（Charles Lasègue）也独立描述了相同病例，两人几乎同时奠定了AN的临床概念。神经性贪食症则直到1979年才由英国精神科医生杰拉尔德·拉塞尔（Gerald Russell）正式发表于《心理医学》（Psychological Medicine）期刊，并将其命名，他同时警示这是"AN的不祥变体"。暴食障碍在2013年出版的《精神障碍诊断与统计手册第五版》（DSM-5，American Psychiatric Association）中才首次作为独立诊断类别被纳入，结束了此前数十年将其归入"未特定进食障碍（EDNOS）"的模糊处理方式。

进食障碍在青少年女性中尤为高发：AN终生患病率约为0.5%–1%，BN约为1%–3%，BED约为3.5%（Hudson et al., 2007），男女患病比例约为1:10。由于病程迁延、复发率高，约50%的患者在治疗5年后仍有残留症状，AN的完全康复率仅约46%。

## 核心原理

### 三大亚型的诊断标准

**神经性厌食症（AN）** 的DSM-5核心诊断特征包括三点：①持续限制能量摄入导致体重明显低于正常最低值（成年人BMI < 17.5 kg/m²，或低于正常预期体重的85%）；②对体重增加的强烈恐惧，或持续干扰体重增加的行为；③对自身体形、体重的认知扭曲（体像障碍），或对低体重严重性的持续性否认。AN进一步分为**限制型**（过去3个月内无反复暴食或清除行为，仅靠节食和过度运动维持低体重）和**暴食/清除型**两个亚型。

**神经性贪食症（BN）** 的核心特征是反复发作的暴食（在任意2小时内摄入量明显多于大多数人在相似情境下摄入的量，同时伴有失控感）与不恰当的补偿行为（自我引吐、滥用泻药或利尿剂、灌肠、禁食或过度运动）。DSM-5要求上述行为平均每周至少发作1次，持续3个月以上。患者体重通常处于正常或超重范围（BMI 18.5–29.9），这与AN形成明显区别，也导致BN患者更难被周围人察觉。

**暴食障碍（BED）** 同样具有反复暴食发作，但**不伴有规律性补偿行为**。DSM-5要求暴食发作平均每周至少1次，持续至少3个月，且符合以下至少3项：进食速度异常快；直到感到不舒适的饱胀感才停止；不感到饥饿时仍进食大量食物；因尴尬而独自进食；事后对自己感到厌恶、抑郁或极度愧疚。三种亚型中BED患病率最高，且与肥胖、2型糖尿病的共病率显著高于AN和BN。

### 病因的生物-心理-社会模型

进食障碍的发生遵循多因素交互模型，无单一因素可独立解释其发病机制。

**生物因素**：5-羟色胺（5-HT）系统功能异常在AN和BN中均有神经影像学证据。正电子发射断层扫描（PET）研究显示，AN患者5-HT2A受体结合力降低约10%–20%，而5-HT1A受体活性升高；BN患者则在额叶和顶叶区域呈现5-HT2A受体结合力下降。这些异常与患者的强迫性限食行为、冲动性暴食行为直接相关（Kaye et al., 2009）。双生子研究显示AN的遗传度约为50%–58%，BN约为55%–62%，说明遗传因素贡献超过环境因素。

**心理因素**：完美主义、低自尊、负性情绪调节困难是三大亚型共有的核心心理特征。克里斯托弗·费尔本（Christopher Fairburn）于1997年提出的**跨诊断认知行为模型**认为，BN患者以"通过控制体形和体重来提升自我价值"的过度评价图式（overconcern schema）为核心：严格节食 → 节食失败触发暴食 → 暴食后罪恶感触发清除行为 → 短暂缓解焦虑 → 重新节食，形成封闭的恶性循环。

**社会文化因素**：特里西·罗丁（Traci Rodin）等学者提出的"瘦理想内化"（thin-ideal internalization）理论指出，媒体持续输出以极度消瘦为美的形象，通过社会比较机制使个体产生身体不满意感（body dissatisfaction），后者是三大亚型共同的近端风险因素。跨文化研究发现，进食障碍在西化程度较高的亚裔移民女性中患病率明显高于其原籍国，支持了文化因素的致病作用。

### 躯体并发症机制

**神经性厌食症的躯体损害**覆盖几乎所有系统：长期低体重导致低钾血症（血钾 < 3.5 mmol/L），钾离子失衡引发心肌电活动异常，心电图出现QTc间期延长（> 450 ms视为危险阈值），是猝死的直接机制。骨骼方面，雌激素分泌减少导致骨密度下降（Z值 < -2.0 SD），约38%的AN患者在患病2年内发生骨量减少。内分泌方面，下丘脑-垂体-性腺轴（HPG轴）受抑，导致女性患者闭经（AN诊断的传统标准之一，DSM-5已删除此条，因其排斥了尚在月经初潮前的患者和男性患者）。

**神经性贪食症的躯体损害**主要由反复清除行为引起：自我引吐导致胃酸反复接触牙釉质，造成牙齿侵蚀（dental erosion），尤其影响上颌切牙的腭面；同时引发腮腺肥大（Russell征，即双侧腮腺无痛性肿大）；长期低钾可导致肾功能损害。泻药滥用则引发代谢性碱中毒和慢性便秘（停药后肠道反射减弱）。

## 关键评估工具与诊断公式

临床诊断AN时，体重状态评估采用体质指数（BMI）公式：

$$BMI = \frac{\text{体重（kg）}}{\text{身高（m）}^2}$$

DSM-5将AN严重程度按BMI分为四级：
- **轻度**：BMI ≥ 17 kg/m²
- **中度**：BMI 16–16.99 kg/m²
- **重度**：BMI 15–15.99 kg/m²
- **极重度**：BMI < 15 kg/m²

标准化临床筛查工具包括：**进食障碍检查量表（EDE-Q，Fairburn & Beglin, 1994）**，共28题，测量过去28天内的饮食限制、体形关注、体重关注和进食关注四个维度；**进食态度测验（EAT-26，Garner et al., 1982）**，以总分≥20为临床关注切分值，灵敏度约77%。

```python
# 简易BMI计算与AN严重程度分级示例
def classify_AN_severity(weight_kg, height_m):
    bmi = weight_kg / (height_m ** 2)
    if bmi >= 17:
        severity = "轻度"
    elif bmi >= 16:
        severity = "中度"
    elif bmi >= 15:
        severity = "重度"
    else:
        severity = "极重度"
    return round(bmi, 2), severity

# 案例：体重38kg，身高1.62m的AN患者
bmi, severity = classify_AN_severity(38, 1.62)
print(f"BMI = {bmi} kg/m²，AN严重程度分级：{severity}")
# 输出：BMI = 14.49 kg/m²，AN严重程度分级：极重度
```

## 实际应用：循证治疗方案

### 心理治疗

**认知行为疗法（CBT-E）** 是BN和BED的一线心理治疗，由费尔本（Fairburn, 2008）开发的增强版CBT（Enhanced CBT）疗程为20次（BN）至40次（复杂案例），聚焦打破"节食→暴食→清除"的恶性循环，重构以体重体形为核心的过度评价图式。随机对照试验（RCT）数据显示，CBT-E使BN患者的暴食/清除行为减少约60%，完全缓解率约45%。

**家庭治疗（Maudsley方法，FBT）** 是青少年AN（18岁以下）的一线治疗，由英国莫兹利医院（Maudsley Hospital）团队开发，强调将父母纳为体重恢复的主要执行者，分三阶段：第一阶段（约10次）父母全面掌控患者饮食；第二阶段逐步将控制权返还青少年；第三阶段聚焦青少年期发展任务和家庭关系。RCT数据显示FBT的5年完全缓解率达49%，显著优于个体疗法的23%（Lock et al., 2010）。

### 药物治疗

氟西汀（Fluoxetine，SSRI类药物）是目前FDA批准用于**BN**治疗的唯一药物，推荐剂量60 mg/d（高于治疗抑郁症的20 mg/d），可将暴食和清除频率降低约50%，但停药后复发率高，通常与CBT联合使用。**AN**目前无FDA批准的特异性药物，奥氮平（Olanzapine，2.5–10 mg/d）在部分RCT中显示可改善体重恢复和强迫性思维，但证据强度有限。**BED**的一线药物为托吡酯（Topiramate）或利斯德安非他命（Lisdexamfetamine，Vyvanse），后者于2015年获FDA批准，是首个获批用于BED的药物，可将每周暴食发作次数从约5次降低至1.5次。

## 常见误区

**误区一："AN患者不可能超重"**。DSM-5已于2013年删除了AN诊断中的"体重低于预期85%"的绝对体重标准，转而使用"显著低于正常最低值"的相对表述。这意味着对于原本肥胖的个体，即使其BMI仍在超重范围，若体重快速下降且伴有AN的认知特征，也可诊断为**非典型神经性厌食症（Atypical AN）**，其躯体风险与经典AN同样严重，却因体重"看似正常"而常被临床忽视。

**误区二："暴食=进食量客观很大"**。DSM-5在BN和BED的暴食定义中使用了"客观暴食发作"（objective binge）与"主观暴食发作"（subjective binge）的区分：客观暴食指在有限时间内摄入量确实明显超过常人；主观暴食则指患者主观感到失控，但实际摄入量并不异常（如仅吃了两块饼干却报告"狂吃了一整天"）。两者的情绪体验相似，但临床意义不同，治疗焦点也有所差异。
