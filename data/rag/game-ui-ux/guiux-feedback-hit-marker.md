# 命中标记反馈

> **领域**: 游戏UI/UX > 交互反馈 | **难度**: ⭐⭐⭐ | **预计学习时间**: 29分钟

## 核心概念

击中敌人时的准星反馈、伤害数字和击杀确认

## 反馈类型

1. **视觉反馈** — 颜色变化、动画、粒子效果
2. **听觉反馈** — 操作音效、确认音、错误音
3. **触觉反馈** — 手柄震动、HD Rumble
4. **系统反馈** — 数值变化、状态更新

## 设计原则

- **即时性** — 反馈应在16ms内响应用户操作
- **比例性** — 反馈强度应与操作重要性成正比
- **一致性** — 相同操作应产生相同类型的反馈

## 实践要点

- 将反馈分层: 主反馈(视觉) + 增强反馈(音效) + 可选反馈(震动)
- 避免反馈过度(每个操作都震动会导致疲劳)
- 为关键操作设计独特的反馈组合

## 经典案例

- **Hades** — 每次攻击的屏幕震动+粒子+音效协同
- **Destiny 2** — 射击手感的反馈设计标杆
- **Celeste** — 平台跳跃中精确的输入反馈

## 关键术语

- **命中标记反馈**: 击中敌人时的准星反馈、伤害数字和击杀确认

## 学习建议

1. 先理解基本概念和设计原则
2. 分析优秀游戏中的实际案例
3. 尝试在自己的项目中实践
4. 收集用户反馈并迭代优化
