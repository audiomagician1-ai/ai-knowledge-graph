# 图标风格指南

> **领域**: 游戏UI/UX > 图标系统 | **难度**: ⭐⭐ | **预计学习时间**: 21分钟

## 核心概念

线性/面性/混合风格的选择和一致性标准

## 设计原则

1. **一致性** — 所有图标使用统一的视觉风格和网格系统
2. **辨识度** — 图标应在小尺寸下依然清晰可辨
3. **语义清晰** — 图标含义应直观，不需要文字标签辅助

## 图标制作流程

1. 定义图标网格(如32x32, 4px内边距)
2. 建立风格指南(线宽、圆角、填充规则)
3. 按功能分类设计(物品/技能/状态/导航)
4. 测试不同尺寸和背景下的辨识度
5. 导出为Sprite Atlas优化性能

## 实践要点

- 使用剪影测试法验证辨识度(去掉颜色只看轮廓)
- 物品品质用统一的边框/光效体系而非仅颜色
- 为色盲用户提供颜色以外的区分方式(形状/纹理)

## 工具链

- **Figma/Sketch** — 矢量图标设计
- **TexturePacker** — Sprite Atlas打包
- **IconJar** — 图标资产管理
