# 水体渲染

## 概述

水体渲染是游戏引擎渲染管线中专门处理水面、河流、海洋等液态表面视觉效果的技术集合。其核心挑战在于水具有三种物理上相互交织的光学特性：反射（水面镜像环境）、折射（光线穿过水体发生偏折）以及次表面散射（光在浅水区域的散射导致蓝绿色调）。这三种特性需要在每帧16毫秒内完成近似计算，因此发展出了多套针对不同水体类型的特化算法。

水体渲染的早期方案可追溯至1990年代，彼时多采用静态贴图加顶点动画的方式模拟水波。2001年前后，基于正弦波叠加的Gerstner波模型成为主流，它能以解析方式计算水面顶点位移，公式为 `x(t) = x₀ + Σ(Aᵢ · sin(kᵢ·x - ωᵢ·t + φᵢ))` 其中 A 为振幅、k 为波数、ω 为角频率。2010年代后，GPU计算能力的提升使FFT海洋方案在主机和PC平台得到普及，显著提升了大面积海洋的真实感。

水体渲染的技术选型直接影响游戏的运行时开销和视觉风格。开放世界海洋场景通常需要FFT方案，而室内喷泉或小型池塘则可以用简单的法线贴图滚动替代。选错方案会导致GPU占用率暴涨或视觉效果失真，因此理解各算法的适用范围具有很强的工程价值。

## 核心原理

### FFT海洋（快速傅里叶变换海洋）

FFT海洋方案由Jerry Tessendorf于2001年在论文《Simulating Ocean Water》中系统化提出，至今仍是大面积海洋渲染的工业标准。其原理分为三步：首先在频域中用Phillips谱函数描述海洋能量分布，Phillips谱公式为 `P(k) = A · exp(-1/(kL)²) / k⁴ · |k̂·ŵ|²`，其中 L = V²/g 表示由风速V和重力加速度g决定的特征长度；然后将频域信号乘以时间相位因子 `e^(iω(k)t)` 使波浪随时间演化；最后通过GPU上的2D IFFT将频域数据转换回空间域，生成512×512或1024×1024分辨率的位移贴图（Displacement Map），驱动网格顶点的XYZ偏移。

实际实现时，通常叠加三张不同分辨率的FFT贴图（尺度分别对应数百米、数十米、数米的波浪），以覆盖从洋涌到涟漪的多尺度细节。在Unreal Engine 5的Water插件中，此方案默认使用蝶形FFT算法将计算复杂度从O(N²)降低到O(N log N)。

### Flowmap河流

Flowmap技术使用一张预烘焙的2D速度场贴图（RG通道分别存储U和V方向的流速），在运行时驱动法线贴图和颜色贴图在流场方向上进行UV偏移，从而模拟水流沿河道流动的效果。核心着色器逻辑使用两个相位偏移0.5的时间周期交替采样并线性混合，消除因UV循环导致的跳变瑕疵：

```
float phase0 = frac(time * speed);
float phase1 = frac(time * speed + 0.5);
float blend = abs(phase0 * 2.0 - 1.0);
color = lerp(tex2D(normalMap, uv + flowDir * phase0),
             tex2D(normalMap, uv + flowDir * phase1), blend);
```

Flowmap的优势在于零运行时物理计算，全部开销集中在离线烘焙阶段。美术人员可以使用Houdini或专用工具绘制流向，精确控制急流、漩涡和瀑布区域的视觉表现。

### 焦散（Caustics）

焦散是光线通过水面折射后在水底投射出的动态光斑网格，是浅水区域视觉真实感的重要组成部分。实时焦散通常有两种实现路径：**贴图投影法**使用一张预生成或程序化生成的焦散动画贴图，沿光源方向投影到水底几何体上，成本极低但难以响应波浪形状变化；**物理模拟法**基于光子映射或屏幕空间方法追踪折射光路，计算量大但效果准确，多见于光线追踪管线（如DXR扩展中的焦散Pass）。

焦散贴图的动态版本可通过对两张静态焦散贴图做时间偏移叠加来近似，配合扰动的UV偏移，能在较低开销下实现可接受的动态效果。焦散的衰减深度通常控制在水面以下0.5至3米范围内，超过此深度因水体吸收光能，焦散强度应快速归零。

## 实际应用

**《刺客信条：黑旗》（2013）**率先在主机游戏中大规模部署FFT海洋，将512×512的位移贴图与实时泡沫遮罩结合，船只行进时的尾浪基于Gerstner波手动叠加在FFT基础波形之上。

**Unity HDRP的Water System**（Unity 2022.2引入）采用分层FFT方案：远景海洋使用低分辨率128×128 FFT，近景切换到高精度512×512，并通过岸线距离场自动计算浪花破碎强度，破碎阈值与雅可比行列式 `J = (1 + ∂D_x/∂x)(1 + ∂D_z/∂z) - (∂D_x/∂z)²` 的负值区域绑定。

**Flowmap在《荒野大镖客：救赎2》**中被广泛用于河流系统，美术团队为游戏内数十条河道手动绘制了流向场，结合深度相关的折射偏移（折射强度与水深成反比），实现了清澈溪流到浑浊急流的多样化表现。

## 常见误区

**误区一：FFT海洋适用于所有水体场景**。FFT方案的最低有效尺度约为数米量级，对于宽度不足5米的小溪或室内水池，FFT的周期性波浪会产生明显的视觉不匹配。这类场景应改用Flowmap或简单法线贴图滚动，并配合动态顶点扰动模拟涟漪。

**误区二：折射和反射可以用同一套屏幕空间采样实现**。水面反射依赖摄像机视线的镜面反射向量采样，而折射依赖折射向量（由Snell定律计算，水的折射率约为1.33）。两者采样的屏幕空间方向相反，混用会导致折射区域出现不合理的镜像图像。在前向渲染管线中，反射通常使用平面反射或SSR，折射则抓取当前帧颜色缓冲并施加扰动偏移。

**误区三：焦散只需要投影一张固定贴图**。静态焦散贴图在水面静止时看起来正确，但当波浪运动时，水面曲率变化会导致焦散形状和亮度都发生剧烈变化。正确的实现应让焦散贴图的UV扰动与水面法线贴图的扰动保持同步，使光斑随波浪形状动态收缩和扩张。

## 知识关联

水体渲染建立在**渲染管线概述**所讲解的顶点着色器、片元着色器和后处理Pass概念之上：FFT位移贴图的结果在顶点着色阶段写入网格坐标，而焦散的屏幕空间投影属于后处理阶段的操作。掌握法线贴图的原理是理解Flowmap技术的前提，因为Flowmap本质上是对法线贴图UV采样的动态控制。水体渲染还与**PBR材质系统**高度相关——水面的菲涅尔反射率计算依赖材质的折射率参数（IOR），浅水区域的透射颜色由Beer-Lambert定律 `T = e^(-σ·d)` 决定，其中σ为吸收系数、d为水体深度，这些参数共同构成完整的物理基础水体材质模型。
